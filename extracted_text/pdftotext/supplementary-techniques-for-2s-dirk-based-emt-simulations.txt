                                                                                           It should be noted that both the ﬁrst-stage and the second-stage
                                                                                           formula, respectively shown in (3) and (5), are in the same form
                                                                                           as the backward Euler formula. This simpliﬁes the coding of the
                                                                                           2S-DIRK method, since the code can be written in a way that the
                                                                                           backward Euler method is applied twice. Another point to note is
                                                                                           that solving a set of n circuit equations using a general two-stage
                                                                                           implicit method in which two stages are coupled each other results
                                                                                           in solving a set of 2n simultaneous equations. The 2S-DIRK method,
                                                                                           however, is designed so as to decouple the two stages, and thus we
                                                                                           can sequentially solve two sets of n simultaneous equations. This
                                                                                           leads to a faster calculation.
