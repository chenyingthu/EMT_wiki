            This paper proposes a universal-blocking-module (UBM)
         based AVM, which can accurately represent the MMCs under
         blocking mode operations. The contributions of this paper are
         summarized as follows:
         1) The proposed AVM uses a generalized model structure that
             can simulate the MMCs with HB, FB, and MB SMs in
             both de-blocking and blocking mode operations.
         2) The proposed model can accurately represent the dynamics
             of the converter arms in blocking mode by emulating the
             equivalent arm capacitor voltages using controlled voltage
             sources and the arm inductors’ initial currents using
             controlled current sources.
