# Time Domain Transformation Method for Accelerating EMTP Simulation of Power System Dynamics
Shengtao Fan, Member, IEEE, and Hui Ding, Member, IEEE

**Abstract**—Electromagnetic Transients Program (EMTP) is widely used in power system dynamics studies. However, in most cases EMTP is found inadequate for dealing with the realistic size power systems due to the small time step resulting in relatively slow simulation speeds. To accelerate the EMTP simulations of power system dynamics, a novel frequency-adaptive methodology is proposed. In this method, a new transformation is presented. The properties of the transformation and the numerical solutions based on the transformation are discussed. The component models obtained by discretizing the differential equations at the branch level are also given which are convenient for EMTP implementation. The proposed method allows using large time steps without sacrificing accuracy, which greatly improves the simulation speed. To validate the proposed methodology, a general program has been developed. Furthermore, various case studies substantiate the claims and demonstrate the application of the method.

**Index Terms**—Algorithms, companion model, Electromagnetic Transients Program (EMTP), power system dynamic, power system simulation, power system transient stability, shifted frequency analysis (SFA).

## I. INTRODUCTION
ELECTROMAGNETIC Transients Program (EMTP) was originally developed for calculating the electromagnetic transients in arbitrary single- or multiphase networks with lumped and distributed parameters. Now it is seeing broader applications in power system steady state as well as dynamic studies and will remain one of the mainstreams in power system research. For example, EMTP can be used in the simulation of transient states of large-scale power systems including power electronic apparatuses and control devices associated with FACTS and HVDC, which requires a simulator as fast as the transient stability (TS) and as detailed as the electromagnetic transients (EMT) simulator. However, in most cases EMTP is found inadequate for dealing with the large size of realistic power systems due to the small time step resulting in relatively slow simulation speeds [1]–[4]. Many attempts have been made to expand the scales of the simulated systems, which can be divided into four categories.

### A. System Equivalent
The detailed EMT modeling is only used for the part of the system which is of great interest (referred to as the internal system), while the remaining part of the system (external system) is reduced to an equivalent system. In order to precisely reflect the wide band frequency dynamic characteristics of the external system, the frequency dependent network equivalent (FDNE) was proposed to represent the high frequency behavior of the external system [5]–[7]. However, the system equivalent method should have limitations when the external systems contain nonlinear components.

### B. Hybrid Simulation
The basic idea of the hybrid simulation is to divide the system network into two parts: one is for the TS simulator which can represent very large systems without dynamic equivalents, and the other for the EMT simulator which needs detailed representation at the device level [8]. The idea of combining TS and EMT simulation was first proposed and implemented by Heffernan in 1981 [9]. Since then, considerable research efforts have been put to improve its performance [4]. However, the lack of concrete theory foundation prevents it from being a perfect solution. The underlying problem is that the TS program is based on the fundamental frequency, single-phase, phasor-type data, while the EMT program is based on the three-phase instantaneous waveform data which includes wide-band frequency components.

### C. Parallel Processing
Parallel processing is another important technique to expand the scale of the simulated power system. The basic network partitioning technique for parallel processing is based on the natural decoupling introduced by traveling wave propagation on transmission lines. The time delay produced by the effect of traveling waves in the long transmission lines is large enough to completely decouple the partitioned subsystems [10]. Some more flexible methods for partitioning the network were proposed in [11] and [12]. Splitting the original system is independent of transmission lines, and arbitrary buses in the network can be assumed as boundary buses. Moreover, partitioning the original network into subsystems allows using different time steps for different subsystems. A smaller time-step is used for components with fast dynamics while a larger time-step can be used for slower dynamic subsystems [13], [14]. However, parallel processing alone has practical limitations when used for realistic-size power systems.
