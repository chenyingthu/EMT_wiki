# Multi-scale Modeling and Simulation of Diverse Transients in VSC-HVDC Transmission Systems

YE Hua1,2, AN Ting3 , TANG Yanan1,2, PEI Wei1,2, QI Zhiping1,2, HAN Congda3

(1. Institute of Electrical Engineering, Chinese Academy of Sciences, Haidian District, Beijing 100190, China;

ABSTRACT: Multi-scale modeling for accurate, fast and flexible simulation of electromagnetic and electromechanical transients in voltage source converter-based high voltage direct current (VSC-HVDC) transmission systems was presented and validated. It makes use of the methodologies such as shifted frequency analysis (SFA). A novel analytical relationship between the shifted frequency (SF)-domain and dq-domain was derived leading to a system-level SFA-based VSC model. Additionally, a multi-scale model of DC transmission lines was developed with selective insertion of a -segment. By adjusting the simulation parameters such as time-step size, diverse transients with multiple time-scales and in different network-positions can be simulated. Simulation results are carried out on two-terminal and five-terminal VSC-HVDC transmission systems. It is proved that high-frequency transients are simulated accurately, and slow-changing transients are modulated with the computational costs reduced significantly. In addition, the flexibility of hybrid simulation of the electromagnetic and electromechanical transients in VSC-HVDC systems is improved.

KEY WORDS: electromagnetic transients; electromechanical transients; high voltage direct current (HVDC); multi-scale modeling; shifted-frequency analysis (SFA); voltage source converter (VSC)

摘要：针对基于电压源转换器的高压直流(high voltage directcurrent based voltage source converter，VSC-HVDC)输电系统，该文提出电磁暂态–机电暂态多尺度建模方法并验证多尺度仿真算法的准确、快速和灵活性。该模型采用频移分析(shifted-frequency analysis，SFA)等多尺度建模方法。首先推导移频(shifted frequency，SF)域和 dq 域之间的解析变换关系，并提出用于系统特性分析的 VSC 移频多尺度模型；其次，选择性插入模型获得直流输电线的多尺度暂态模型；最后，通过调整仿真参数，如仿真时间步长和移动频率，使得建立的模型能够灵活仿真具有多个时间尺度和不同网络位置的各种暂态。采用两端和五端VSC-HVDC输电系统作为算例，仿真结果对比表明，提出的模型和算法能够精确模拟高频暂态，快速仿真慢变化暂态，降低计算成本。此外，多尺度模型还可提高 VSC-HVDC 输电系统电磁–机电暂态混合仿真的灵活性。

## 0 INTRODUCTION

The voltage source converter-based high voltage direct current (VSC-HVDC) technologies are widely used in the fields of large-scale wind power integration, multi-terminal direct current (MTDC) power transmission and so on. To ensure an improved planning design and operation of related applications, modeling and simulation of diverse physical phenomena of VSC-HVDC system is essential [1]. Including all time constants of the VSC-HVDC and associated AC systems, however, would lead to complex equations to be resolved, and thus multi-scale modeling would be a concern to simulate electromagnetic and electromechanical transients within a same simulation run.

Currently, modeling requirements of a VSC-HVDC system mainly depend on the nature and scope of the physical phenomena to be simulated [2]. For the studies involving the power electronic device designing, electromagnetic transient (EMT) models of two/three-level converter and even modular multilevel converter (MMC) technologies are respected for the accuracy in representing physical behaviors [3-5]. A detailed representation of high-frequency transients for HVDC grids may lead to increasing computation burden when the number of the sub-modules (SMs) in MMCs increases significantly. To speed up the computational time of VSC-HVDC system, dynamic average-value models (AVMs) that only modulate the average quantities of switching events were addressed in [5-7]. The AVMs were developed in the timedomain EMT simulation environment using a time-step size of microseconds.

A detailed phasor model of MMC considering the circulating current control was derived in [8] where the MMC model is a two-level VSC model added by an equivalent series capacitance. The latter is also concluded in [9] where the detailed and simplified transient stability (TS) models were derived, separately. As addressed in [9], both models have their relative limitations in the simulation of slow or fast-changing transients, and thus further studies are required. For large-scale VSC-HVDC transmission systems, the dynamic-phasor model of the VSC may be suitable for studying slow-changing dynamics within the time-frame of interest [10]. However, the solution method for phasor-based models has a low degree of the accuracy in modulating EMT phenomena because of simplifications and neglecting fast-changing events [10-12].

An alternative approach is to combing the EMT simulator for modulating high-frequency transients and TS programs to capture snapshots of the system operation [13]. Considerable research efforts were addressed for hybrid simulation of electromagnetic and electromechanical transients of line-commutated converter-based HVDC (LCC-HVDC), for example, [14-15]. Reference [16] offers algorithms of hybrid EMT and TS simulations for VSC-HVDCs and also gives the interfacing techniques. The interface-based hybrid simulation using EMT and TS models that are radically different may lead to the inaccuracy and low-level flexibility in terms of data exchange and subsystem equivalence [17].

Power system blackouts such as the 2003 North America blackout [18] have created requirements for profound and through understanding on wide-frequency transients within the same study. The shifted frequency analysis (SFA) method [19-21] removes out fundamental frequency carrier in AC quantities and develops the SFA models in electromagnetic transient program (EMTP) simulation framework. These models can be used to efficiently simulate power system slow dynamics, and also are capable of simulating fast transients. By introducing a new simulation parameter, namely shift frequency, in addition to the time-step size, the methodology frequency-adaptive simulation of transients (FAST) was created [22-23]. Herein, the adaptive frequency shifting of the Fourier spectra of analytic signals that describe diverse transients is achieved within the same study. In this context, the FAST-based multi-scale transient models of frequency-dependent transmission line and induction machinery were developed respectively in [24-25], respectively. The matrix exponential function method was proposed to accelerate the simulation of slow-changing transients, while fast transients are accurately reconstructed through efficient dense output mechanism [26]. The algorithm was validated by performing diverse transients in a wind farm consisting of 661.5MW fixed-speed induction generators. Currently the multi-scale methods above were well applied in modeling the multi-scale transients in pure ac power electric systems. Recently, the shifted-frequency model of MMC was proposed but it is difficult for modeling electromechanical transients with a time-step size of milliseconds [27]. Therefore, it is still a challenge within the same study to model multiple time-scale transients from high-frequency transients to system-level slow-changing dynamics in VSC-HVDC systems.

In this paper, multi-scale modeling of diverse electromagnetic and electromechanical transients for VSC-HVDC transmission systems is presented. For VSC-HVDC transmission system-level modeling, this paper gives reconfiguration and extension to authors’ former work [28]. A SFA-based VSC model is firstly established, mathematical relationship of transforming the SF-domain and dq-domain quantities is derived, and then selective insertion technique of a-segment leads to a multi-scale model of dc transmission lines. By adjusting the simulation parameters such as time-step size, the proposed model is used to simulate diverse transients with multiple time-scales and in different network-positions. Finally, the multi-scale model and simulation algorithm are verified and applied in diverse studies that include phase-to-ground faults, wind power fluctuations, and transient stability in two test systems.

Section II is intended to provide the reader with background knowledge on the SFA. In Section III, multi-scale transient model of the VSC-HVDC including components of the VSC, dc line, and associated controllers is developed. In Section IV, network-position-based simulation algorithms of simulating multi-scale transients are proposed. Verification and application of the multi-scale model are studied in Section V. Conclusions are remarked in Section VI.

## 1 SHIFTED FREQUENCY ANALYSIS

In an AC electric power system, electrical signals of the voltages and currents have a fundamentalfrequency carrier that is denoted by $f _ { \mathrm { c } } { = } 5 0 \mathrm { H z }$ or $f _ { \mathrm { c } } { = } 6 0 \mathrm { H z } .$ . By considering time-varying waveforms with the fundamental-frequency component, the electrical signal can be expressed as:

$$
x ( t ) = A ( t ) \cos [ ( \omega _ { \mathrm { c } } + \Delta \omega ( t ) ) t + \phi _ { 0 } ]\tag{1}
$$

where the angular carrier frequency $\omega _ { \mathrm { c } } = 2 \pi f _ { \mathrm { c } } ,$ , and $\phi _ { 0 }$ is a constant depending on initial state of the system. The amplitude A(t) and angular frequency deviation (t) are variables of the time. Applying the trigonometric identity, (1) is rewritten as:

$$
x ( t ) = x _ { \mathrm { R } } \left( t \right) \cos ( \omega _ { \mathrm { c } } t ) - x _ { \mathrm { I } } ( t ) \sin ( \omega _ { \mathrm { c } } t )\tag{2}
$$

with:

$$
\begin{array} { r } { \left\{ \boldsymbol { x } _ { \mathrm { R } } ( t ) = \boldsymbol { A } ( t ) \cos ( \Delta \omega ( t ) t + \phi _ { 0 } ) \right. } \\ { \left. \boldsymbol { x } _ { \mathrm { I } } ( t ) = \boldsymbol { A } ( t ) \sin ( \Delta \omega ( t ) t + \phi _ { 0 } ) \right. } \end{array}\tag{3}
$$

When the power system is subjected to lowfrequency dynamics, for example, electromechanical oscillations, xR(t) and xI(t) behave as two low-pass signals which contain all information except for the fundamental-frequency component. To extract the low-pass signals, application of the Hilbert transform as given in Appendix A into (2) yields:

$$
\begin{array} { r l r } { \underline { { x } } ( t ) = x _ { \mathrm { R } } ( t ) \cos ( \omega _ { \mathrm { c } } t ) - x _ { \mathrm { I } } ( t ) \sin ( \omega _ { \mathrm { c } } t ) + } & { } & \\ { \mathrm { j } [ x _ { \mathrm { R } } ( t ) \sin ( \omega _ { \mathrm { c } } t ) + x _ { \mathrm { I } } ( t ) \cos ( \omega _ { \mathrm { c } } t ) ] = } & { } & \\ { \quad \quad ( x _ { \mathrm { R } } ( t ) + \mathrm { j } x _ { \mathrm { I } } ( t ) ) [ \cos ( \omega _ { \mathrm { c } } t ) + \mathrm { j } \sin ( \omega _ { \mathrm { c } } t ) ] } & { } & \end{array}\tag{4}
$$

where the analytic signal x( )t is structured by adding to x(t) an imaginary part that is calculated using the Hilbert transform.

It is visible from (4) that the carrier signal is $\mathrm { e } ^ { \mathrm { j } \omega _ { \mathrm { c } } t } { = } \cos ( \omega _ { \mathrm { c } } t ) + \mathrm { j } \sin ( \omega _ { \mathrm { c } } t )$ . The signal $x _ { \mathrm { R } } ( t ) + \mathrm { j } x _ { \mathrm { I } } ( t )$ is the complex envelope of x(t), which corresponds to a dynamic phasor. The latter is readily obtained, and multiplication of ${ \mathrm { e } } ^ { - { \mathrm { j } } \omega _ { \mathrm { c } } t }$ on both sides of (4) leads to:

$$
S ( \underline { { x } } ( t ) ) = x _ { \mathrm { R } } \left( t \right) + \mathrm { j } x _ { \mathrm { I } } ( t ) = \underline { { x } } ( t ) \mathrm { e } ^ { - \mathrm { j } \omega _ { \mathrm { c } } t }\tag{5}
$$

The real and imaginary parts of (5) can be decomposed and then written in a vector-matrix form as:

$$
\begin{array} { r } { \Big [ \boldsymbol { x } _ { \mathrm { R } } ( t ) \Big ] = \left[ \begin{array} { l l } { \cos ( \omega _ { \mathrm { c } } t ) } & { \sin ( \omega _ { \mathrm { c } } t ) } \\ { - \sin ( \omega _ { \mathrm { c } } t ) } & { \cos ( \omega _ { \mathrm { c } } t ) } \end{array} \right] \left[ \begin{array} { l } { \boldsymbol { x } ( t ) } \\ { \boldsymbol { H } ( \boldsymbol { x } ( t ) ) } \end{array} \right] } \end{array}\tag{6}
$$

where H(x(t)) is the Hilbert transform of x(t).

The effect of creating an analytic signal from the real signal with bandpass characteristics about the carrier frequency is that Fourier spectra of the analytic signal only contain positive frequencies. Hereby, the frequency shifting is available to shift the Fourier spectra of the simulated quantities, such as voltage and current, and so influence the maximal frequencies involved. When simulating slow-changing transients, the Fourier spectra of analytic signals concentrating around the carrier frequency are shifted as shown from the center to the right of Fig. 1. ## 2.3 Control System Modeling Interfacing Shifted-Frequency and DQ Domains

With a high degree of controllability, the VSC often uses the symmetric-sequence-based vector control scheme to deal efficiently with unbalance multi-phase quantities at its AC-side. To ensure a seamless interface of control components to the multi-scale model of electrical systems, it is essential to derive an analytical relationship between the SF-domain and dq-domain quantities. Considering the transformations of the Park, symmetric-sequence, and shifted-frequency quantities, an alternative relationship between $d q$ and SF components of three-phase AC voltages, for example, can be obtained. The positive-sequence $d q$ voltages are expressed as functions of the shifted-frequency voltages in a vector-matrix form.

$$
\begin{array} { r } { \bigg [ \pmb { u } _ { d } ^ { + } ( t ) \bigg ] = \frac { 1 } { 3 } \bigg [ \cos ( \Delta \theta ) \quad \sin ( \Delta \theta ) \bigg ] \Bigg [ \pmb { T } ( 0 ) \bigg ] ^ { \mathrm { T } } \bigg [ \pmb { u } _ { \mathrm { R } } ( t ) \bigg ] } \\ { \bigg [ \pmb { u } _ { q } ^ { + } ( t ) \bigg ] ^ { \mathrm { d } } - \sin ( \Delta \theta ) \quad \cos ( \Delta \theta ) \bigg ] \Bigg [ \pmb { T } ( - \frac { \pi } { 2 } ) \Bigg ] \bigg [ \pmb { u } _ { \mathrm { I } } ( t ) \bigg ] } \end{array}\tag{21}
$$

where T(0) and $T ( - \pi / 2 )$ are identical to those of the Park transformation with replacements  by 0 and $\pi / 2 .$ , respectively.

Following the same procedure as is done from (21), the negative-sequence $u _ { d } ^ { - }$ and $\bar { u _ { q } }$ can be calculated from $\pmb { u } _ { \mathrm { R } }$ and $\pmb { u } _ { \mathrm { I } } .$ . Defining $\pmb { u } _ { d q } ^ { + } = [ u _ { d } ^ { + } , \ u _ { q } ^ { + } ] ^ { \mathrm { T } }$ and $\pmb { u } _ { d q } ^ { - } =$ $\left[ u _ { d } ^ { - } , \ u _ { q } ^ { - } \right] ^ { \mathrm { T } }$ , a mathematical relationship between the dq-domain and SF-domain voltages can be established.

$$
\begin{array} { r } { [ \pmb { u } _ { d q } ^ { + } ( t ) ] = \pmb { T } _ { \mathrm { S F } _ { - } d q } ( \Delta \theta ) \overset { \sim } { \underbrace { \pmb { u } _ { \mathrm { R } } ( t ) } } ] } \\ { \pmb { u } _ { d q } ^ { - } ( t ) \rfloor } \end{array}\tag{22}
$$

with:

$$
T _ { \mathrm { S F } _ { - } d q } ( \Delta \theta ) = \frac { 1 } { 3 } \left[ \begin{array} { l l } { T ( \Delta \theta ) ^ { \mathrm { T } } } & { T ( \Delta \theta - \frac { \pi } { 2 } ) ^ { \mathrm { T } } } \\ { T ( - \Delta \theta ) ^ { \mathrm { T } } } & { - T ( \frac { \pi } { 2 } - \Delta \theta ) ^ { \mathrm { T } } } \end{array} \right]\tag{23}
$$

Note that $\pmb { T } _ { \mathrm { S F } \_ d q }$ and $T _ { d q \_ \mathrm { S F } }$ would be matrices of size $6 \times 6$ if the zero-sequence of the variables are taken into account. In this case, $\pmb { T } _ { \mathrm { S F } \_ d q }$ and ${ \pmb T } _ { d q \_ \mathrm { S F } }$ are reciprocal matrices.

The shifted-frequency-based multi-scale model of the VSC and its controller is established as depicted in Fig. 3. Using the transformation matrix $\pmb { T } _ { \mathrm { S F } \_ d q : }$ , the shifted-frequency quantities are transformed into dq-coordinate quantities and the latter are as inputs to the control systems in terms of positive and negative sequences. With outer and inner vector-control actions, the modulation signals $\pmb { m } _ { d q } ^ { + }$ and $\pmb { m } _ { d q } ^ { - }$ are formed as the control signals of the controlled voltage sources and current sources using the matrix $\pmb { T } _ { d q \_ \mathrm { S F } }$ .

## 2.4 Multi-Scale Transient Model of DC Transmission Line

In order to modulate multiple time-scale transients of HVDC grids within the same study, it is essential to develop a multi-scale model of DC transmission lines to simulate slow-changing oscillations up to high-frequency traveling wave effects. The line losses are considered by means of adding lumped resistances to both ends of a single-phase lossless distributed-parameter line. The relationship of currents and voltages at both ends 1 and 2 of the lossless line can be rewritten as:

$$
{ i _ { \mathrm { 1 1 } } } ( t ) = \frac { 1 } { z _ { 0 } } u _ { \mathrm { 1 1 } } ( t ) - i _ { \mathrm { 1 2 } } ( t - T _ { \mathrm { w p } } ) - \frac { 1 } { z _ { 0 } } u _ { \mathrm { 1 2 } } ( t - T _ { \mathrm { w p } } )\tag{24}
$$

$$
i _ { 1 2 } ( t ) = \frac { 1 } { z _ { 0 } } u _ { 1 2 } ( t ) - i _ { 1 1 } ( t - T _ { \mathrm { w p } } ) - \frac { 1 } { z _ { 0 } } u _ { 1 1 } ( t - T _ { \mathrm { w p } } )\tag{25}
$$

where the characteristic impedance is $z _ { 0 } = \sqrt { L ^ { \prime } / C ^ { \prime } }$ ， $T _ { \mathrm { w p } } = l / \sqrt { L ^ { \prime } C ^ { \prime } }$ , and $T _ { \mathrm { w p } }$ is the wave propagation time from one end of the line to the opposite side. The line has a length of l with distributed inductance $L ^ { \prime }$ and capacitance C'.

The wave propagation time $T _ { \mathrm { w p } }$ is usually not an integer multiple of the time-step size $\Delta t ,$ and thus the linear interpolation in [23] is used for instantaneous voltages and currents. It is performed on $u _ { \mathrm { l 1 } } ( t - T _ { \mathrm { w p } } )$ as an example as:

$$
u _ { \mathrm { 1 1 } } ( t - T _ { \mathrm { w p } } ) = \rho u _ { \mathrm { 1 1 } } [ t - ( \gamma - 1 ) \Delta t ] + \sigma u _ { \mathrm { 1 1 } } ( t - \gamma \Delta t )\tag{26}
$$

where $\rho { = } \gamma { - } T _ { \mathrm { w p } } / \Delta t$ and $\sigma { = } 1 - \gamma { + } T _ { \mathrm { w p } } / \Delta t .$ It is assumed that ceil rounds the variables to the nearest integers towards infinity, i. e. $\gamma { = } \mathrm { c e i l } ( T _ { \mathrm { w p } } / \Delta t )$ .  1 corresponds to the case when $\Delta t > T _ { \mathrm { w p } } .$ , and $\gamma > 1$ corresponds to the case when $\Delta t < T _ { \mathrm { w p } } .$ The interpolations for $u _ { 1 2 } ( t - T _ { \mathrm { w p } } ) , i _ { 1 1 } ( t - T _ { \mathrm { w p } } )$ and $i _ { 1 2 } ( t - T _ { \mathrm { w p } } )$ are carried out in the same way.

By inserting the interpolation results into (24) and (25) and rearranging the equations, a multi-scale DC line model with distributed parameters is established, and the equivalent circuit model is shown in Fig. 4. $$

## 3 NETWORK POSITION-BASED SIMUL-ATION OF MULTIPLE TIME-SCALE TRA-NSIENTS

With the developed multi-scale models of the components including the VSC, DC transmission line and associated controllers, a system-level multi-scale transient model of the VSC-HVDC system is constructed using the stamping method [22]. When electromechanical transients are of particular interest, a time-step size in the range of milliseconds is set, and the switches as depicted in Fig. 4 are closed. When simulating electromagnetic phenomena, highfrequency information is modulated with a time-step size of microseconds, and in Fig. 5 the switches are open. With an increase of HVDC size, large-scale MTDC or DC grid poses a challenge to multi-scale simulation of EMT and TS exhibiting in different network positions. In this paper, network position-based multi-scale transient simulation is proposed in which the simulation parameters such as time-step size can be adjusted in different network positions.

As a special multi-rate simulation, different time-steps/sampling times are used to compute different state variables within a system. In order to facilitate data exchange, the time-step size of TS simulation is often made as an integer multiple of that in the EMT simulation.

## 5 CONCLUSIONS

This paper presented the multi-scale modeling of VSC-HVDC transmission systems for simulating electromagnetic and electromechanical transients within the same study. Firstly, a SFA-based multi-scale transient model of the VSC was developed, and then the analytical relationship between the SF-domain and dq-domain quantities was derived. Additionally, selective insertion of a -segment in both ends of DC line is proposed to allow multi-scale simulation of both traveling wave effects and slow-changing transients. Secondly, application of the multi-scale modeling method into hybrid simulation of EMT and TS responses was investigated, highlighting the conclusion that the interfacing approach is relatively simple because of same modeling methods for both EMT and TS zones. Thirdly, comparisons of the simulation results between the proposed model and EMT-AVM prove that the multi-scale model accurately simulates the transients caused by phase-to-ground faults, and on the other hand modulates wind power fluctuations significantly reducing computational cost. Moreover, the flexibility of hybrid simulation on electromagnetic and electromechanical transients at different network positions of AC/DC systems is enhanced using the multi-scale model.

The ongoing work is aimed at applying this modeling method for multi-rate real-time simulation of slow and fast changing transients. The time-step sizes may vary between different states and at different instances, which pose great challenge in the real-time simulation of large-scale hybrid MTDC and AC systems. It is another challenge to develop a novel multi-scale model in future works to accurately account for detailed power electronic effects during electromagnetic transients.

# Multi-scale Modeling and Simulation of Diverse Transients in VSC-HVDC Transmission Systems

YE Hua1,2, AN Ting3 , TANG Yanan1,2, PEI Wei1,2, QI Zhiping1,2, HAN Congda3

(1. Institute of Electrical Engineering, Chinese Academy of Sciences; 2. University of Chinese Academy of Sciences; 3. State Grid Global Energy Interconnection Research Institute)

KEY WORDS: electromagnetic transients; electromechanical transients; high voltage direct current (HVDC); multi-scale modeling; shifted-frequency analysis (SFA); voltage source converter (VSC)

The voltage source converter-based high voltage direct current (VSC-HVDC) technologies are widely employed in the fields of large-scale wind power integration, multi-terminal direct current (MTDC) power transmission and etc. Including all time constants of the VSC-HVDC and associated AC systems, however, would lead to complex transients to be analyzed, and thus multi-scale modeling would be a concern to simulate electromagnetic and electromechanical transients within the same simulation run.

In this paper, multi-scale modeling of diverse electromagnetic and electromechanical transients for VSC-HVDC transmission systems is presented. A shifted frequency analysis (SFA)-based VSC model is firstly established, and a mathematical relationship of transforming the SF-domain and dq-domain quantities is derived. Then, using selective insertion technique of -segments, a multi-scale model of the DC transmission lines is developed. By adjusting the simulation parameters such as time-step size, the proposed model is used to simulate diverse transients with multiple time-scales and in different network-positions. As the salient feature of this model, the SF-domain controlled voltage sources can be expressed as a function of the VSC DC-side voltage $u _ { \mathrm { d c } } ,$ the modulation signals $\pmb { m } _ { d q } ^ { + }$ and $\begin{array} { r } { \pmb { m } _ { d q } ^ { - } , } \end{array}$ and the transformation matrices as follows:

$$
\begin{array} { r } { \left[ { \pmb u } _ { \mathrm { N } _ { - } \mathrm { R } } \left( t \right) \right] = 0 . 5 u _ { \mathrm { d c } } ( t ) { \pmb T } _ { d q _ { - } \mathrm { S F } } ( \Delta \theta ) \left[ { \pmb m } _ { d q } ^ { + } ( t ) \right] } \\ { { \pmb u } _ { \mathrm { N } _ { - } \mathrm { I } } ( t ) } \end{array}\tag{1}
$$

with:

$$
T _ { d q _ { \mathrm { - } } \mathrm { S F } } ( \Delta \theta ) = \left[ \begin{array} { c c } { { T ( \Delta \theta ) } } & { { T ( - \Delta \theta ) } } \\ { { T ( \Delta \theta - \frac { \pi } { 2 } ) } } & { { T ( \frac { \pi } { 2 } - \Delta \theta ) } } \end{array} \right]\tag{2}
$$

where $\pmb { T } ( \Delta \theta )$ is in fact the Park transformation as a function of the angular deviation, and in a special case of  0, T(0) is the Clake transformation matrix.

The multi-scale model and simulation algorithm are verified through comparison with the EMT-type simulator (Matlab/SimPowerSystem) results. Diverse transients are performed, as affected by phase-to-ground faults in Fig. 1, and under wind power fluctuations in Fig. 2.
