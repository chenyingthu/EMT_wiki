# 多绕组变压器模型 (Multi-winding Transformer Model)

## 定义与边界

多绕组变压器模型（Multi-winding Transformer Model）描述三个及以上绕组通过共同磁路、漏磁路径和端口连接耦合的变压器 EMT 等效。物理对象是铁芯、绕组、绝缘、分接开关、接地和寄生电容；EMT 等效对象可以是多端口漏感矩阵、励磁支路、理想变压器约束、零序网络、饱和磁链状态和端口导纳历史源。

本页是 [[transformer-model|变压器模型]] 的多端口扩展，不替代换流变压器、特殊移相变压器或高频白盒模型页面。多绕组模型的难点不是"绕组数更多"本身，而是端口参数、磁路路径、零序通道和饱和状态在多个绕组之间相互约束。

## EMT 建模对象

对 $m$ 个绕组，电压和磁链关系的一般形式为：

$$\mathbf{v}=\mathbf{R}\mathbf{i}+\frac{d\boldsymbol{\psi}}{dt},\qquad
\boldsymbol{\psi}=\mathbf{L}(\boldsymbol{\psi},\mathbf{i})\mathbf{i}.$$

其中 $\mathbf{v}=[v_1,\dots,v_m]^\top$ 为端口电压矢量，$\mathbf{i}=[i_1,\dots,i_m]^\top$ 为端口电流矢量，$\boldsymbol{\psi}$ 为各绕组磁链矢量。在线性模型中 $\mathbf{L}$ 是对称电感矩阵，非对角项 $L_{ij}\,(i\neq j)$ 表示绕组间互感；非线性模型中 $\mathbf{L}$ 随磁链和饱和程度变化。

进入 EMT 节点法后，离散伴随电路形式为：

$$\mathbf{i}_{n+1}=\mathbf{Y}_{eq}\mathbf{v}_{n+1}+\mathbf{i}_{hist,n}.$$

其中 $\mathbf{Y}_{eq}$ 是多端口等效导纳矩阵，$\mathbf{i}_{hist,n}$ 包含上一时间步的磁链历史项、饱和状态和励磁贡献。

## 对偶原理等效电路建模

### 核心思想

对偶原理将磁路离散为电路等效：铁芯的绕组柱、轭部和外柱（五铁芯）分别映射为非线性磁化电感与线性损耗电阻并联支路。通过渐近光滑（asymptotic）函数构建磁滞回线，避免分段线性近似的数值振荡。

**几何比例定义**（Shafieipour 2020）：

$$r_s = \frac{s_y}{s_w},\quad r_l = \frac{l_y}{l_w},\quad r_{s0} = \frac{s_y}{s_o},\quad r_{l0} = \frac{l_y}{l_o}$$

其中 $s_w, s_y, s_o$ 分别是绕组柱、轭部、外柱截面积；$l_w, l_y, l_o$ 是对应磁路长度。该比例用于将平均磁化参数分配至各铁芯段。

**漏感计算**（短路试验参数化）：

$$L_{i,j} = \frac{1}{j\omega}\frac{V_i}{I_i},\quad i eq j$$

即通过短路试验测得的电压和电流计算绕组 $i$ 与 $j$ 间的等效漏感。**互感系数**为：

$$M_{i,j} = \frac{1}{2}(L_{i,j+1}+L_{i+1,j}-L_{i,j}-L_{i+1,j+1})$$

该关系利用相邻漏感的差值构建多绕组间互感网络，支持任意数量绕组（含分接头）的建模。

### 饱和与磁滞方程

**饱和渐近方程**（Shafieipour 2020 公式(4)）：

$$\tilde{i}(\lambda) = \frac{\sqrt{(\lambda-\lambda_0)^2+4dL_{air}}+\lambda-\lambda_0}{2L_{air}}-\frac{d}{\lambda_0}$$

其中 $\lambda$ 是磁链，$\lambda_0$ 是膝点磁链，$d$ 是控制曲线弯曲度的参数，$L_{air}$ 是空气芯电感（合理区间 $0.1sim0.3$ pu，偏离此范围通常指示参数提取错误）。

**四象限磁滞方程**（公式(5)）：

$$\tilde{i}_q(\lambda) = \pm\left(\frac{\sqrt{(-\lambda-\lambda_0)^2+4dL_{air}}-\lambda-\lambda_0}{2L_{air}}-\frac{d}{\lambda_0}\right)\pm\frac{D}{2}$$

其中 $D$ 是磁滞回线宽度，控制回线的路径依赖宽度。当 $D=0$ 时退化为纯饱和渐进方程。

**各段磁滞电感**（公式(6)）：

$$\tilde{L}^q_\alpha(\lambda) = \frac{V_c}{j\omega}\left[\pm\left(\frac{\sqrt{(-\lambda-\lambda_{0\alpha})^2+4d_\alpha L_{\alpha air}}-\lambda-\lambda_{0\alpha}}{2L_{\alpha air}}-\frac{d_\alpha}{\lambda_{0\alpha}}\right)\pm\frac{D_\alpha}{2}\right]^{-1}$$

其中 $\alpha\in\{w,y,o\}$ 分别代表绕组柱（winding limb）、轭部（yoke）和外柱（outer limb）；$q=1,2,3,4$ 为四象限标志。膝点磁链 $\lambda_{0\alpha}=k_\alpha\cdot V_c/\omega$。

**参数约束**（公式(7)）：

$$c_\alpha = \frac{1}{L_{air}}\cdot\frac{V_c}{\omega}-\lambda_c+\lambda_{0\alpha},\quad \alpha=w,y,o.$$

各铁芯段的 $L_{\alpha air}$、$L_\alpha$ 和 $D_\alpha$ 可由厂家铭牌数据或常规试验确定，不需要铁芯内部几何和材料信息。

## 多端口导纳矩阵建模

### 高频端口模型

Morched 等（1993）提出基于端口节点导纳矩阵有理函数逼近的变压器高频等效模型。**等效导纳函数**为：

$$\mathbf{Y}(s) = \sum_{k=1}^{n_r}\frac{R_k}{s-p_k} + \sum_{k=1}^{n_c}\left(\frac{R_k}{s-p_k}+\frac{R_k^*}{s-p_k^*}\right)$$

其中 $n_r$ 个实极点和 $n_c$ 对复共轭极点描述集肤效应、铁芯涡流损耗和绕组杂散电容。**矩阵形式**为：

$$[I_{port}] = [Y_{eq}(s)]\cdot[V_{port}]$$

有效频带覆盖 **60 Hz ~ 200 kHz**。三相双绕组变压器经模态分解后，需独立拟合的导纳函数数量从 21 个减少至 6 个，**计算效率提升约 71%**。

### 多柱变压器零序阻抗闭式公式

Ahmadi 等（2026）为三柱变压器首次推导了零序路径电感的闭式公式：

$$L_{0,closed-form} = f(Z_{HM}, Z_{HL}, Z_{ML}, geometry)$$

该公式使三柱变压器开路零序阻抗匹配**误差降至 0%**，消除了传统数值迭代的收敛不确定性。

## 模型层级

| 层级 | 保留内容 | 适合用途 | 边界 |
|------|----------|----------|------|
| 理想多绕组变压器 + 漏阻抗 | 匝比、漏阻抗和端口耦合 | 潮流初始化、常规暂态 | 不描述饱和、涌流和高频电容 |
| 多端口电感矩阵 | 绕组间互感和耦合 | EMT 端口耦合、故障暂态 | 参数必须正定或物理可解释 |
| 饱和励磁支路 | 磁链-电流非线性、剩磁 | 合闸涌流、过励磁、铁磁谐振 | 需要磁化曲线和初始磁链 |
| 磁路/对偶电路模型 | 铁芯柱、轭、漏磁路径 | 多芯柱、零序和非对称饱和 | 参数获取和验证成本高 |
| 高频端口模型 | 绕组电容、频变损耗、端口阻抗 | 雷电、操作冲击、绕组谐振 | 不能由低频短路试验单独确定 |

## EMT 建模方法分类

### 方法 1：Terminal Duality（端对偶法，TDM）

**原理**：将磁路对偶映射为电路中的非线性磁化支路。铁芯几何尺寸用铭牌参数替代（短路阻抗、开路阻抗、额定电压频率）。

**关键公式**：
- 漏感矩阵：$L_{i,j} = V_i/(j\omega I_i)$（短路试验）
- 互感：$M_{i,j} = (L_{i,j+1}+L_{i+1,j}-L_{i,j}-L_{i+1,j+1})/2$
- 饱和电感：$\tilde{L}^q_\alpha(\lambda)$ 由公式(6)给出

**特点**：不需要铁芯内部设计参数；支持任意数量绕组（含分接头）；3柱和5柱结构均适用。

**典型精度**：50 MVA 三铁芯三绕组变压器（138/13.8/6.972 kV）和 390 MVA 五铁芯双绕组变压器（238 kV侧）在 PSCAD/EMTDC 中验证，短路漏抗仿真误差 **< 0.1%**。

### 方法 2：高频端口导纳有理函数逼近（Morched 1993）

**原理**：对每个端口的输入导纳做有理函数拟合（实极点 + 复共轭极点对），捕捉宽频阻抗特性。

**关键公式**：$Y(s) = \sum R_k/(s-p_k)$，极点配置典型为 6 个实极点 + 15 对复共轭极点。

**特点**：有效频带 60 Hz ~ 200 kHz；减少需拟合的导纳函数数量（21→6，效率提升约 71%）；参数来自阶跃电压注入测试。

**典型精度**：125 MVA、215/44 kV 三柱变压器上，EMTP 仿真波形与现场录波在幅值、谐振频率和衰减时间常数上高度一致。

### 方法 3：BCTRAN / Geometrical Model

**原理**：基于变压器几何参数（绕组布置、铁芯尺寸）计算电阻和电感矩阵，用 EMTP 节点导纳矩阵实现。

**特点**：参数来源于变压器几何和材料特性；需要厂家设计资料或反向计算。

**典型精度**：400/120/21 kV、400/400/125 MVA 三柱自耦变压器验证，BCTRAN 与 Geometrical Model 计算结果**偏差 < 5%**。

### 方法 4：磁路对偶电路（Shafieipour 2020）

**原理**：用对偶原理将多铁芯（三柱/五柱）磁路映射为等效电路，用非线性磁滞电感和线性电阻并联支路表征饱和、磁滞、深饱和和涡流损耗。

**关键约束**：
- 空气芯电感合理区间：$L_{air}\in[0.1,0.3]$ pu
- 膝点电压合理区间：$k_w\in[1.0,1.25]$ pu（超出易引发数值发散）
- 涌流峰值典型为额定电流的 **6~12 倍**

**特点**：不依赖铁芯几何和材料 B-H 曲线；用常规试验可确定的替代参数驱动；渐近方程保证数值稳定。

### 方法 5：Z 域传递函数模型

**原理**：在 Z 域建立变压器的离散传递函数模型，用于数值阻尼分析和稳定性评估。

**特点**：极点位置（$z=-1$ 振荡型 vs $z=1$ 耗散型）决定数值阻尼特性；可通过滤波或隐式积分改善阻尼。

## 关键技术挑战

### 挑战 1：零序漏感约束强制匹配

多绕组变压器的任意绕组零序漏感约束是尚未解决的难题（Shafieipour 2020 明确指出被忽略）。零序阻抗取决于绕组接线、中性点接地和铁芯结构，不能只由正序漏抗推断。

**应对方向**：三柱变压器需特别注意外部零序返回路径建模（Ahmadi 2026）；五柱变压器轭部电感按绕组堆叠比例分布可改善不对称磁路精度。

### 挑战 2：饱和与磁滞的数值稳定性

分段线性磁化曲线在饱和拐点处会导致数值振荡。高频暂态仿真中 $L_{air}$ 和 $k_w$ 参数的微小偏差会显著影响深饱和区行为。

**应对方向**：使用 Shafieipour 2020 的渐近光滑函数（公式(4)(5)）避免分段线性近似的数值不稳定性；$L_{air}$ 控制在 0.1~0.3 pu 区间内。

### 挑战 3：多端口导纳有理函数拟合的自动化

Morched 1993 的高频模型需要手动配置极点阶数和初始参数估计（优化后变化幅度 < 5%）。缺乏自动化的误差—复杂度权衡准则。

**应对方向**：参见 Gustavsen 等的矢量拟合（Vector Fitting）方法作为自动化工具。

### 挑战 4：参数可得性与物理一致性

短路阻抗折算需要统一基准、绕组连接和测量条件；未说明基准的参数不能直接比较。多铁芯变压器的零序路径参数和轭部分布电感在实际工程中难以直接测量。

**应对方向**：TDM 方法使用替代参数（铭牌数据 + 常规试验）而非设计参数；参数物理一致性通过开路/短路试验验证。

### 挑战 5：高频绕组电容与寄生网络确定

绕组电容、套管电容和接地电容缺失时，冲击电压、绕组谐振和局部过电压结果不可靠。高频端口模型的有效频带受限于测量频率范围。

**应对方向**：用阶跃电压注入测试或频率扫描测量端口阻抗；参考 Morched 1993 的多对复共轭极点配置（15 对）。

## 适用边界与选择指南

| 应用场景 | 推荐模型 | 理由 |
|----------|----------|------|
| 合闸涌流、剩磁敏感性分析 | TDM/NCC（Shafieipour 2020） | 支持磁滞回线和初始磁链；不依赖铁芯内部参数 |
| 零序故障和接地分析 | TDM/NCC（Ahmadi 2026）闭式零序公式 | 零序阻抗误差 0%；明确零序返回路径 |
| 雷电/操作冲击、高频谐振 | 高频端口导纳模型（Morched 1993） | 有效频带 60 Hz ~ 200 kHz；包含寄生电容 |
| 快速暂态（开断过电压） | BCTRAN / Geometrical Model（Papadias 2004） | 偏差 < 5%；计算效率高 |
| 宽频阻抗扫描、谐波分析 | 多端口导纳矩阵（Morched 1993）+ 矢量拟合 | 有理函数极点多对复共轭极点覆盖宽频带 |
| 工业级多柱变压器饱和涌流 | 磁路对偶电路（Shafieipour 2020） | 50/390 MVA 实测验证；涌流峰值 6~12 倍额定电流 |
| 实时仿真（RSCAD-RTDS） | TDM/NCC（Ahmadi 2026） | 单步计算耗时增加 < 2%；数值振荡完全消除 |

**模型不适用场景**：
- 把多绕组模型简化为多个独立双绕组变压器 → 丢失绕组间功率分配和磁耦合
- 零序阻抗仅由正序漏抗推断 → 三柱变压器误差大（需外部零序返回路径）
- 饱和模型缺剩磁和磁化曲线 → 合闸涌流结果仅为敏感性分析
- 缺高频寄生网络时做冲击电压仿真 → 结果不可靠

## 验证方法

多绕组变压器的 EMT 验证应绑定目标层级：

1. **低频端口**：开路试验、短路试验、负载试验和成对短路阻抗是否复现
2. **零序与接地**：不同接线组、中性点接地和铁芯结构下的零序响应是否合理
3. **饱和暂态**：合闸角、剩磁、故障清除和过励磁工况是否与试验或详细模型一致
4. **高频暂态**：端口频响、冲击试验、绕组电压分布或宽频测量是否支撑所用寄生网络
