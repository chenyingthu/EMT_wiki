# 差动保护 (Differential Protection)

## 概述

差动保护(Differential Protection)是基于基尔霍夫电流定律(KCL)的一种选择性保护原理，通过比较被保护设备两侧（或多侧）的电流大小和相位，判断故障是否发生在被保护区域内。当区内故障时，两侧电流差值显著增大，保护动作；区外故障时，两侧电流基本平衡，保护不动作。差动保护因其原理简单、动作快速、选择性好，广泛应用于变压器、发电机、母线、电动机和线路等重要设备的保护。

## 基本原理

### 基尔霍夫电流定律

**KCL表述**：流入节点的电流之和等于流出节点的电流之和
$$\sum I_{in} = \sum I_{out}$$

**正常运行/区外故障**：
$$I_1 = I_2 \Rightarrow I_d = |I_1 - I_2| \approx 0$$

**区内故障**：
$$I_1 + I_2 = I_{fault} \Rightarrow I_d \gg 0$$

### 差动电流定义

**动作量（差动电流）**：
$$I_d = \left|\sum_{i=1}^{n} I_i\right|$$

**制动量（穿越电流）**：
$$I_r = \frac{1}{2}\sum_{i=1}^{n} |I_i|$$

或：
$$I_r = \max(|I_1|, |I_2|)$$

### 动作判据

**绝对值比较**：
$$I_d > I_{set}$$

**比率制动特性**：
$$I_d > k \cdot I_r + I_{pickup}$$

其中：
- $k$: 制动系数(0.2-0.6)
- $I_{pickup}$: 最小动作电流

## 比率制动特性

### 双折线特性

```
Id
↑
│           ╱
│          ╱
│         ╱ ← 制动段 (斜率k)
│    ────╱
│    │  ↑
│    │  Ipickup
│    │
└────┴───────────→ Ir
```

**动作方程**：
$$I_d > \begin{cases} I_{pickup} & I_r \leq I_{r0} \\ I_{pickup} + k(I_r - I_{r0}) & I_r > I_{r0} \end{cases}$$

### 三折线特性

**适用场景**：
n- 高穿越电流下的CT饱和
- 严重外部故障

**参数设置**：
| 参数 | 典型值 | 说明 |
|-----|-------|------|
| $I_{pickup}$ | 0.2-0.5 $I_n$ | 最小动作电流 |
| $k_1$ | 0.2-0.4 | 低段制动系数 |
| $k_2$ | 0.5-0.8 | 高段制动系数 |
| $I_{r1}$ | 1.0-2.0 $I_n$ | 拐点电流 |

## 变压器差动保护

### 接线补偿

**Y/△-11变压器**：
- Y侧电流相位校正
- 幅值补偿

**补偿公式**：
$$I_{Y}^{\prime} = \frac{I_A - I_B}{\sqrt{3}n_{CT}}$$

**电流互感器配置**：
| 变压器侧 | 接线方式 | CT接线 |
|---------|---------|--------|
| Y侧 | Y形 | △形 |
| △侧 | △形 | Y形 |

### 励磁涌流

**产生原因**：变压器空载合闸

**特征**：
- 幅值大（6-8倍额定电流）
- 偏于时间轴一侧
- 含有大量二次谐波

**二次谐波制动**：
$$K_{2nd} = \frac{I_2}{I_1} > 15\% - 20\%$$

**波形识别**：
- 间断角识别
- 波形对称识别

### 过励磁保护

**五倍频制动**：
$$K_{5th} = \frac{I_5}{I_1} > 35\%$$

**过励磁倍数**：
$$V/f > 1.1$$

## 母线差动保护

### 完全电流差动

**单母线**：
$$I_d = \left|\sum_{i=1}^{n} I_i\right|$$

**动作判据**：
$$I_d > I_{set}$$

### 比率制动母线保护

**大差动（总差动）**：
$$I_{d,total} = \left|\sum_{k=1}^{m} I_k\right|$$

**小差动（间隔差动）**：
$$I_{d,j} = \left|I_j - \frac{I_{d,total}}{m}\right|$$

**选择逻辑**：
- 大差动启动
- 小差动选择故障间隔

### 断路器失灵保护

**启动条件**：
1. 保护动作
2. 故障电流持续存在

**延时跳闸**：
$$t_{trip} = t_{breaker} + t_{margin}$$

## 发电机差动保护

### 纵联差动保护

**保护范围**：发电机定子绕组及引出线

**整定计算**：
$$I_{set} = K_{rel} \cdot I_{unb,max}$$

其中$K_{rel}$为可靠系数(1.3-1.5)。

### 不平衡电流来源

| 来源 | 原因 | 抑制措施 |
|-----|------|---------|
| CT误差 | 10%误差曲线 | 选高精度CT |
| 饱和 | 大穿越电流 | 高拐点CT |
| 变比差异 | 选型不当 | 匹配变比 |
| 调压分接头 | ±10%调节 | 中间档位整定 |

## 线路差动保护

### 光纤差动保护

**通道类型**：
- 专用光纤
- 复用光纤通道

**同步方法**：
- GPS同步
- 乒乓法同步

**动作方程**：
$$|I_M + I_N| > k|I_M - I_N| + I_{pickup}$$

### 电容电流补偿

**分布电容影响**：
$$I_C = j\omega C_L V$$

**补偿方法**：
- 半补偿
- 全补偿
- 自适应补偿

## 电动机差动保护

### 不平衡分析

**启动电流影响**：
- 启动电流大(5-7$I_n$)
- 衰减慢

**整定原则**：
$$I_{set} > 1.3I_{start,unb}$$

## 微机保护实现

### 采样与计算

**采样频率**：
- 基波：每周波24-64点
- 谐波分析：每周波≥64点

**滤波算法**：
- 傅氏算法
- 半波积分
- 全波傅氏

### 比率制动实现

**制动特性选择**：
```
if Ir < Ir0:
    I_operate = Ipickup
else:
    I_operate = Ipickup + k*(Ir - Ir0)

if Id > I_operate:
    trip()
```

### 谐波制动

**二次谐波提取**：
$$I_2 = \sqrt{a_2^2 + b_2^2}$$

其中：
$$a_2 = \frac{2}{N}\sum_{k=0}^{N-1} i_k\cos(2\cdot\frac{2\pi k}{N})$$
$$b_2 = \frac{2}{N}\sum_{k=0}^{N-1} i_k\sin(2\cdot\frac{2\pi k}{N})$$

## 整定计算

### 变压器差动整定

**最小动作电流**：
$$I_{pickup} = (0.2 \sim 0.5)I_{n,CT}$$

**制动系数**：
$$k = 0.3 \sim 0.5$$

**二次谐波制动比**：
$$K_{2nd} = 15\% \sim 20\%$$

### 灵敏度校验

**最小灵敏系数**：
$$K_{sen} = \frac{I_{k,min}}{I_{set}} \geq 2$$

其中$I_{k,min}$为最小运行方式下区内故障电流。
