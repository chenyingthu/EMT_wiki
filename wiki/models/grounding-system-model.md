# 接地系统 (Grounding System)

## 1. 物理对象概述

### 1.1 功能与作用

接地系统是电力系统安全运行的重要保障：

**核心功能**：
- **故障电流疏散**：提供低阻抗路径，快速清除故障
- **过电压保护**：限制雷电和操作过电压
- **人身安全**：限制接触电压和跨步电压
- **设备保护**：提供等电位连接，防止地电位升高损坏设备
- **参考电位**：为控制和测量系统提供零电位参考

### 1.2 接地类型

| 类型 | 定义 | 应用场景 |
|------|------|---------|
| **工作接地** | 系统中性点接地 | 变压器中性点、发电机中性点 |
| **保护接地** | 设备外壳接地 | 开关柜、变压器外壳、杆塔 |
| **防雷接地** | 避雷针/线接地 | 变电站、线路杆塔 |
| **信号接地** | 弱电系统接地 | 通信、控制、测量系统 |

### 1.3 接地装置类型

**自然接地体**：
- 金属管道（水管、油管）
- 电缆金属外皮
- 建筑物钢筋

**人工接地体**：
- **垂直接地极**：钢管/角钢/铜棒，长度2.5-3m
- **水平接地极**：扁钢/圆钢，埋深0.6-0.8m
- **接地网**：变电站主接地网，网格5-10m
- **放射形接地**：线路杆塔，3-8根放射线

## 2. 物理模型与数学描述

### 2.1 土壤电阻模型

#### 2.1.1 均匀土壤模型

**土壤电阻率**：
$$\rho = \frac{R \cdot A}{L}$$

其中：
- $\rho$：土壤电阻率（Ω·m）
- $R$：测量电阻
- $A$：电极面积
- $L$：电极间距

**典型土壤电阻率**：
| 土壤类型 | 电阻率 (Ω·m) | 说明 |
|---------|-------------|------|
| 沼泽地 | 5-30 | 低电阻率 |
| 粘土 | 20-100 | 良好 |
| 砂质粘土 | 50-250 | 一般 |
| 砂土 | 250-500 | 较差 |
| 岩石 | 1000-10000 | 高电阻率 |

#### 2.1.2 分层土壤模型

双层土壤结构：
$$\rho_{app}(L) = \rho_1 \left[1 + 4 \sum_{n=1}^{\infty} K^n \left(\frac{1}{\sqrt{1+(2n\frac{h}{L})^2}} - ...\right)\right]$$

其中：
- $\rho_1, \rho_2$：上层/下层电阻率
- $h$：上层厚度
- $K = (\rho_2 - \rho_1)/(\rho_2 + \rho_1)$：反射系数

### 2.2 频变土壤特性

#### 2.2.1 频率对土壤电阻率的影响

高频下土壤呈现频变特性：
$$\rho(f) = \rho_0 \left[1 + \left(\frac{f}{f_c}\right)^\alpha\right]^{-1/2}$$

其中：
- $\rho_0$：直流电阻率
- $f_c$：临界频率（通常为kHz-MHz）
- $\alpha$：衰减系数（0.5-1.0）

**物理机制**：
- 低频：离子导电主导
- 高频：极化效应、电容效应显现
- 雷电频率（MHz级）：电阻率可能降低30-50%

#### 2.2.2 复介电常数模型

$$\epsilon^*(f) = \epsilon' - j\left(\epsilon'' + \frac{\sigma}{\omega}\right)$$

**Debye模型**：
$$\epsilon^*(f) = \epsilon_\infty + \frac{\epsilon_s - \epsilon_\infty}{1 + j2\pi f \tau}$$

### 2.3 接地电阻计算

#### 2.3.1 单根垂直接地极

**接地电阻**：
$$R_g = \frac{\rho}{2\pi L} \left[\ln\left(\frac{4L}{d}\right) - 1\right]$$

其中：
- $L$：接地极长度（m）
- $d$：接地极直径（m）
- $\rho$：土壤电阻率（Ω·m）

**简化公式**（L=2.5m, d=0.05m）：
$$R_g \approx 0.3 \rho$$

#### 2.3.2 水平接地极

**单根水平接地体**：
$$R_g = \frac{\rho}{2\pi L} \left[\ln\left(\frac{2L}{d}\right) + \ln\left(\frac{L}{h}\right) - 2\right]$$

其中：
- $L$：长度（m）
- $h$：埋深（m）

#### 2.3.3 接地网

**方形接地网**：
$$R_g = \frac{\rho}{4r} + \frac{\rho}{L_{total}}$$

其中：
- $r = \sqrt{A/\pi}$：等效半径
- $A$：接地网面积
- $L_{total}$：导体总长度

**简化公式**（网格n×n）：
$$R_g \approx 0.5 \frac{\rho}{\sqrt{A}}$$

## 3. EMT仿真模型

### 3.1 集中参数模型

**单根接地极等效**：
- 电阻：$R = R_g$
- 电感：$L = \frac{\mu_0 L}{2\pi} \ln\frac{2L}{d}$
- 电容：$C = \frac{2\pi \epsilon_0 \epsilon_r L}{\ln\frac{2L}{d}}$

**频变等效电路**：
```
        L_lead
    ━━━━/\/\/━━━━┳━━━━/\/\/━━━━┓
                ┃            ┫
              ┌─┴─┐          ═ C_g
              │ R │          ┫
              │(f)│          ┻━━━━┛
              └───┘
```

### 3.2 分布参数模型

**多段π型电路**：
将接地极分段，每段：
$$R_i = \frac{\rho \Delta L}{A_{eq}}, \quad L_i = L_{unit} \Delta L, \quad C_i = C_{unit} \Delta L$$

### 3.3 接地网频变模型

**矢量拟合FDNE**：
$$Z_{ground}(s) = \sum_{k=1}^{N} \frac{R_k}{s - p_k} + R_0$$

适用于变电站接地网宽频建模（工频-1MHz）。

## 4. 典型参数

### 4.1 接地电阻要求

| 接地类型 | 电阻要求 | 说明 |
|---------|---------|------|
| 变压器中性点 | ≤ 0.5Ω | 大电流接地系统 |
| 变电站接地网 | ≤ 0.5Ω | 一般要求 |
| 线路杆塔 | ≤ 10Ω | 架空线路 |
| 配电变压器 | ≤ 4Ω | 100kVA及以下 |
| 建筑物防雷 | ≤ 10Ω | 第三类防雷 |

### 4.2 人身安全限值

**接触电压限值**：
$$V_{touch,max} = \frac{174 + 0.17\rho_s}{\sqrt{t_s}}$$

**跨步电压限值**：
$$V_{step,max} = \frac{174 + 0.7\rho_s}{\sqrt{t_s}}$$

其中：
- $\rho_s$：表层土壤电阻率（Ω·m）
- $t_s$：故障持续时间（s）

**典型值**（$t_s = 0.5s, \rho_s = 3000$Ω·m）：
- $V_{touch,max} \approx 500V$
- $V_{step,max} \approx 1000V$

| 相对介电常数 | 10-20 | 15-25 | 20-40 |
