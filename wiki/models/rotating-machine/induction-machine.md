# 感应电机 (Induction Machine)

## 定义与边界

感应电机（又称异步电机）是转子转速低于定子旋转磁场转速的交流旋转电机，依靠电磁感应在转子绕组中产生电流和转矩。根据转子结构分为**鼠笼式**（转子导条短接，无外接电阻）和**绕线式**（转子绕组通过滑环外接电阻，可调节转子回路）两种。在EMT仿真中，感应电机模型需要考虑机电耦合、饱和特性和启动过程。

**边界限定**：本页面聚焦于三相感应电机的EMT建模，不包括单相感应电机或特殊结构电机。

## EMT中的作用

感应电机是电力系统的重要动态负荷：

- **负荷建模**：工业和民用负荷的主要成分（占工业总负荷的60-80%）
- **启动分析**：大电机启动的电压暂降
- **稳定性分析**：负荷特性对系统稳定的影响
- **故障分析**：不对称故障下的电机响应
- **保护整定**：电机保护的协调配合
- **风电并网**：双馈感应发电机（DFIG）是风电场主流机型

## EMT建模方法

感应电机的EMT建模根据接口形式和精度层次可分为五类方法：dq坐标系模型、电压后电抗（VBR）模型、近似电压后电抗（AVBR）模型、节点缩减（NR）模型，以及含饱和的VBR扩展模型。

### 1. dq坐标系模型

#### 坐标变换与电压方程

感应电机在同步旋转dq坐标系下的电压方程为：

$$v_{ds} = R_s i_{ds} + \frac{d\psi_{ds}}{dt} - \omega_s \psi_{qs}$$

$$v_{qs} = R_s i_{qs} + \frac{d\psi_{qs}}{dt} + \omega_s \psi_{ds}$$

$$v_{dr} = R_r i_{dr} + \frac{d\psi_{dr}}{dt} - (\omega_s - \omega_r)\psi_{qr} = 0$$

$$v_{qr} = R_r i_{qr} + \frac{d\psi_{qr}}{dt} + (\omega_s - \omega_r)\psi_{dr} = 0$$

其中 $R_s$、$R_r$ 分别为定子和转子电阻，$\omega_s$ 为同步电角速度，$\omega_r$ 为转子电角速度。

#### 磁链方程

$$\psi_{ds} = L_s i_{ds} + L_m i_{dr}$$

$$\psi_{qs} = L_s i_{qs} + L_m i_{qr}$$

$$\psi_{dr} = L_m i_{ds} + L_r i_{dr}$$

$$\psi_{qr} = L_m i_{qs} + L_r i_{qr}$$

其中 $L_s = L_{ls} + L_m$、$L_r = L_{lr} + L_m$ 分别为定子和转子全电感，$L_m$ 为互感（励磁电感）。

#### 电磁转矩

$$T_e = \frac{3}{2}p \frac{L_m}{L_r}(\psi_{dr} i_{qs} - \psi_{qr} i_{ds})$$

其中 $p$ 为极对数。

#### 运动方程

$$J \frac{d\omega_r}{dt} = T_e - T_L$$

转差率定义：

$$s = \frac{\omega_s - \omega_r}{\omega_s}$$

#### 特点

dq模型的核心优势在于将三相耦合系统解耦为两个独立的旋转坐标系方程组，显著降低了方程求解复杂度。转子参考系下的电感矩阵为常数对角阵（对称正弦分布绕组假设），避免了时变电感带来的数值困难。但dq模型需要进行abc-dq坐标变换（每步需12次三角函数计算），且当使用预测接口与外部网络连接时，会引入接口误差并破坏网络导纳矩阵的常数特性。



### 2. 电压后电抗（VBR）模型

VBR模型将定子放在abc相坐标、转子放在dq参考系，以实现与外部物理网络的直接接口，同时利用dq坐标简化转子动态处理。

#### 定子电压方程（相域）

$$v_{abc} = R_s i_{abc} + \frac{d\lambda_{abc}}{dt}$$

其中 $v_{abc}$、$i_{abc}$、$\lambda_{abc}$ 分别为定子电压、电流和磁链向量。

#### 离散化接口形式

梯形积分法离散化后得到通用形式：

$$i(t) = Y(t) v(t) + h(t - \Delta t)$$

其中 $Y(t)$ 为时变导纳矩阵，$h$ 为历史电流项。

#### 转子电流消去

通过转子参考系变换消去转子变量，得到仅含定子端口的等效电路。转子电流的递推形式为：

$$i_{qr}(t) = \frac{1}{\frac{2L_{lr}}{\Delta t} + R_r + \frac{2L_m}{\Delta t}} \left[ \lambda_{qr}(t) + \frac{2L_m}{\Delta t} i_{abc,r}(t) - h_{qr}(t - \Delta t) \right]$$

其中 $i_{abc,r}$ 为定子电流在转子参考系的分量。

#### 特点

VBR模型的外部接口量是定子三相端口电压和电流，可直接作为网络节点导纳与历史电流源接入EMTP节点方程。相比dq模型，VBR避免了每步的旋转变换，但精确VBR模型的等效电阻矩阵的非对角元素与转速相关，导致等效电导子矩阵随时间变化，使EMTP网络导纳矩阵需每步重新LU分解。



### 3. 近似电压后电抗（AVBR）模型

AVBR模型（Wang & Jatskevich, 2010）通过忽略VBR模型中转速相关的非对角系数，使等效电阻矩阵退化为常数对角阵，从而获得恒定电导子矩阵。

#### 核心近似

精确VBR模型的等效电阻矩阵结构为：

$$R_{eq}^{VBR} = \text{diag}(d_j) + \text{off-diag}(k_j)$$

其中非对角元素 $k_j$ 含转速依赖系数。理论分析表明 $|k_j / d_j| = O(\Delta t^2)$，在典型EMTP步长下该比值降至 $10^{-3}$ 量级，矩阵呈强对角占优。AVBR模型将非对角元素 $k_j$ 置零：

$$R_{eq}^{AVBR} \approx \text{diag}(d_j)$$

#### EMTP全局网络方程

$$G_{total} \mathbf{V} = \mathbf{I}_{hist}$$

其中 $G_{total}$ 包含电机恒定电导子矩阵，$\mathbf{V}$ 为节点电压向量，$\mathbf{I}_{hist}$ 为历史电流源。将 $R_{eq}^{AVBR}$ 求逆得恒定电导矩阵 $G_{eq}^{AVBR}$，在初始化时一次性完成LU分解，每步只需更新历史项并同步求解端电压和电流。

#### 近似误差上界

$$\frac{\|\mathbf{x} - \tilde{\mathbf{x}}\|}{\|\mathbf{x}\|} \leq \frac{\|R_{eq}^{-1}\| \|R_{eq} - \tilde{R}_{eq}\|}{1 - \|R_{eq}^{-1}\| \|R_{eq} - \tilde{R}_{eq}\|}$$

该界在 $\Delta t = 500 \, \mu s$ 时，3 HP电机最大误差界仅为 $1.2\%$，50 HP及以上电机误差界均低于 $0.5\%$。

#### 量化性能（Wang & Jatskevich, 2010）

| 指标 | 数值 |
|------|------|
| AVBR单步耗时 | $1.9 \, \mu s$ |
| PD模型单步耗时 | $4.3 \, \mu s$ |
| 精确VBR单步耗时 | $2.2 \, \mu s$ |
| AVBR相对PD加速比 | $126\%$ |
| AVBR相对VBR加速比 | $13.6\%$ |
| 500 μs步长下最大误差界 | $<1.2\%$（3 HP）|
| 非对角/对角系数比值衰减阶数 | $O(\Delta t^2)$ |

#### 特点

AVBR模型实现了电机电气/机械变量与EMTP网络的直接、非迭代同步求解，彻底避免了网络矩阵的重复LU分解。在500 μs大时间步长下仍保持高精度，综合精度与效率显著优于PD、VBR及传统dq接口模型。精度排序：精确VBR > AVBR > PD > 传统dq预测接口模型。



### 4. 节点缩减（NR）模型

NR模型（Vilchis-Rodriguez & Acha, 2012）通过节点缩减技术消去内部转子节点，将相域模型的$6 \times 6$导纳满阵缩减为$3 \times 3$对角阵。

#### NR-CC模型最终形式

$$i_{abc}(t) = Y_{nrcc} v_{abc}(t) + h_{nrcc}(t - \Delta t)$$

其中 $Y_{nrcc}$ 为 $3 \times 3$ 恒定对角导纳矩阵，实现时不变参数建模。

#### 恒定等效导纳矩阵

$$Y_{nrcc} = \left[ R_s + \frac{2L_{ls}}{\Delta t} + \frac{3}{2} \frac{(2L_m / \Delta t)^2}{(2L_{lr} / \Delta t + R_r) + (2L_m / \Delta t)} \right]^{-1}$$

该式利用梯形积分参数和电机参数构成对角阵，证明在对称正弦分布假设下导纳矩阵时不变。

#### 算法步骤

1. 建立三相感应电机原始相域方程（定子电压方程 + 磁链方程）
2. 应用梯形积分法离散化，得到 $i(t) = G v(t) + h(t - \Delta t)$
3. 通过转子参考系变换矩阵 $C_r(\theta)$ 将转子侧变量变换到转子参考系，使电感矩阵对角化
4. 在转子参考系中表达转子电流（对角阵形式），计算复杂度从满阵运算降为对角元素运算
5. 节点缩减消元：消去转子节点变量，仅保留定子端节点
6. 构建NR-CC模型：得到恒定对角导纳矩阵 $Y_{nrcc}$，无需每步更新

#### 量化性能（Vilchis-Rodriguez & Acha, 2012）

| 指标 | 数值 |
|------|------|
| 导纳矩阵维度缩减 | $6 \times 6$ 满阵 → $3 \times 3$ 对角阵 |
| 每步矩阵运算次数 | $O(n^3) = 216$ 次乘法 → $O(n) = 3$ 次乘法 |
| 运算效率提升 | $\approx 98\%$ |
| 允许大步长 | $1$–$5$ ms（PD模型需 $< 0.5$ ms）|
| 步长灵活性提升 | $3$–$10$ 倍 |
| 定子电流瞬态误差 | $< 0.5\%$（相比详细PD模型）|
| 稳态误差 | $< 0.1\%$ |
| 历史项计算量降低 | $\approx 85\%$（对角阵乘法 vs 满阵乘法）|
| 内存访问次数减少 | $\approx 70\%$ |
| 每步节省三角函数计算 | $12$ 次（避免abc-dq旋转变换）|

#### 特点

NR模型导纳矩阵对角化使矩阵求逆计算量降低约98%，允许使用1-5 ms的大步长而保持数值稳定性。由于使用纯电流变量集，消除了磁通-电流转换的计算开销，且无需abc-dq旋转变换。



### 5. 含饱和的VBR扩展模型

Wang & Jatskevich（2010）提出含磁饱和的VBR模型，基于主磁通路径饱和假设，采用分段线性（PWL）近似表征非线性磁化特性。

#### 分段线性饱和关系

在每个积分步长内，将饱和曲线线性化为增量电感与剩余磁链的组合：

$$\lambda_m = \lambda_{res} + L_{inc} i_m$$

其中 $i_m$ 为磁化电流，$L_{inc} = \frac{\partial \lambda_m}{\partial i_m} \approx \frac{\Delta \lambda_m}{\Delta i_m}$ 为动态增量电感。

#### dq轴交叉饱和投影

主磁链矢量在转子dq坐标系中的投影方程为：

$$\lambda_{md} = \lambda_m \cos\theta, \quad \lambda_{mq} = \lambda_m \sin\theta$$

其中 $\theta = \arctan(\lambda_{mq} / \lambda_{md})$ 为主磁链矢量在dq坐标系中的空间角度。该投影实现了dq轴静态和动态交叉饱和。

#### 量化性能（Wang & Jatskevich, 2010）

| 指标 | 数值 |
|------|------|
| 50-100 μs步长下稳态/暂态误差 | $< 1.0\%$ |
| 传统d轴饱和模型误差 | $> 3\%$ |
| 单步计算耗时降低 | $30\%$–$40\%$ |
| q轴磁链动态响应偏差（交叉饱和后） | 从 $2.5\%$ 收敛至 $< 0.4\%$ |
| 10段分段线性拟合精度 | $< 0.5\%$ |

#### 特点

含饱和VBR模型将饱和特性直接嵌入伴随电路的导纳矩阵与历史电流源中，无需迭代求解。通过分段线性化将非线性饱和特性嵌入EMTP离散求解过程，同时保留VBR模型与外部网络直接接口的特性。适用于主磁化通道饱和建模，对漏磁饱和的扩展未在当前验证证据中覆盖。

## 量化性能边界

### 启动特性与暂态响应（Krause 2013, Fitzgerald 2003）

- **直接启动电流**：$5$–$7 \, I_N$（额定电流倍数），全压启动时电压暂降可达 $10$–$30\%$
- **直接启动转矩**：$1.5$–$2.5 \, T_N$（额定转矩倍数），加速过程中转矩振荡逐渐衰减
- **启动时间**：小电机（$<100$ kW）$0.5$–$2$ s，大电机（$>1$ MW）$5$–$20$ s，取决于负载惯量
- **星三角启动**：启动电流降至直接启动的 $1/3$，启动转矩同步降至 $1/3$
- **软启动器**：$2$–$10$ s 内平滑加速，启动电流降至 $2$–$3 \, I_N$

### EMT模型精度（dq0电压方程与伴随电路离散化）

- **梯形积分法（$\Delta t = 10$–$50 \, \mu s$）**下感应电机EMT模型稳态误差 $< 0.1\%$
- **电磁时间常数**（定子时间常数 $T_s = L_s/R_s$）：$0.01$–$0.1$ s
- **机电时间常数**（转子时间常数 $T_r = L_r/R_r$）：$0.1$–$2$ s（大电机更长）
- 步长 $\Delta t < T_s/10$ 时电磁暂态分量可精确捕捉；步长 $> T_s/2$ 时可能出现数值振荡
- 步长 $> 0.5$ ms 时PD模型可能出现数值不稳定；NR模型可扩展至 $1$–$5$ ms

### 典型测试数据

| 功率等级 | 启动电流 | 启动转矩 | 启动时间 | 转差率 |
|----------|----------|----------|----------|--------|
| $< 100$ kW | $5$–$7 \, I_N$ | $1.5$–$2.5 \, T_N$ | $0.5$–$2$ s | $2\%$–$5\%$ |
| $100$–$1000$ kW | $5$–$6 \, I_N$ | $1.8$–$2.5 \, T_N$ | $2$–$5$ s | $1\%$–$3\%$ |
| $> 1$ MW | $4$–$6 \, I_N$ | $2.0$–$2.5 \, T_N$ | $5$–$20$ s | $0.5\%$–$2\%$ |



## 关键技术挑战

### 挑战1：机网接口导纳矩阵更新

dq模型和PD模型与EMTP网络接口时，若使用随转速/时间变化的电导子矩阵，每步都需要重新分解网络导纳矩阵，计算代价极高。VBR模型的精确形式存在转速相关非对角系数，导致每步更新。

**解决方向**：AVBR近似忽略非对角转速相关系数（误差$O(\Delta t^2)$）；NR模型通过节点缩减将对角化电感矩阵常值化。

### 挑战2：磁饱和与交叉饱和

感应电机在启动过程和大负荷工况下存在显著的磁饱和，导致$d$轴和$q$轴磁路互相耦合（交叉饱和）。饱和特性使等效电感参数随工作点变化，增加了模型非线性。

**解决方向**：分段线性（PWL）方法将饱和曲线分区线性化；凸极系数法实现交叉饱和的统一描述。

### 挑战3：电磁暂态与机电暂态多速率耦合

电磁暂态时间常数（$T_s = 0.01$–$0.1$ s）远小于机电暂态时间常数（$T_r = 0.1$–$2$ s），两类动态在同一个时间框架内耦合求解，需要合理的步长选择和数值阻尼。

**解决方向**：大步长EMT方法（NR模型支持1-5 ms）；多速率仿真架构。

### 挑战4：参数辨识不确定性

感应电机等效电路参数（$R_s, R_r, X_s, X_r, X_m$）的精确获取依赖测试数据，不同测试方法（堵转试验、开路试验、频率响应测试）得到的参数可能存在偏差，直接影响EMT模型精度。

**解决方向**：综合多种测试方法交叉验证；参数灵敏度分析指导关键参数优先辨识。

### 挑战5：不对称故障与谐波含量

感应电机在不对称故障工况下产生较大的谐波分量，特别是2倍频谐波（$100$ Hz在$50$ Hz系统中），对电机发热和转矩脉动有显著影响。

**解决方向**：完整的相域建模捕捉谐波；dq模型需额外注入负序分量和谐波分量。



## 适用边界与选择指南

### 方法选择决策表

| 应用场景 | 推荐模型 | 步长范围 | 精度特点 |
|----------|----------|----------|----------|
| 高精度需求（误差 < 0.5%）| NR-CC / NR-CF | $0.1$–$1$ ms | 精度最高，允许大步长 |
| 大规模系统实时仿真 | AVBR / NR | $0.5$–$5$ ms | 恒定导纳矩阵，避免每步重LU |
| 含饱和的暂态分析 | 含饱和VBR | $50$–$100 \, \mu s$ | 包含dq轴交叉饱和 |
| 一般性暂态仿真 | dq模型（梯形积分）| $10$–$100 \, \mu s$ | 实现简单，精度一般 |
| 大规模系统参数扫描 | NR（对角阵）| $1$–$5$ ms | 每步计算量最小 |

### 失效边界

- **启动失败**：电压过低或负载过大时，电机无法建立足够转矩
- **失速**：转矩不足导致转速下降直至停机
- **过热**：长期过载导致温升超标
- **电压暂降**：大电机直接启动引起系统电压跌落 $10$–$30\%$
- **深度饱和**：饱和VBR模型对主磁化通道饱和的假设在深度饱和区误差增大
- **高频开关谐波**：VBR/AVBR/NR模型不包含逆变器PWM高频开关细节，需要器件级模型
