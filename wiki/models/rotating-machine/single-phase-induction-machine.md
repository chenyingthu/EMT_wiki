# 单相感应电机 (Single-Phase Induction Machine)

## 定义与边界

单相感应电机是由单相交流电源供电的感应电机，依靠辅助绕组和启动装置（电容、电阻或罩极）产生旋转磁场实现启动。与三相感应电机不同，单相感应电机本质上是不对称电机——单相电源本身不能产生旋转磁场，必须依靠辅助手段打破对称性才能产生启动转矩。在EMT仿真中，单相感应电机模型需要精确描述主绕组（m）和辅助绕组（a）之间的不对称、启动装置的切换过程，以及电容对稳态运行特性的影响。

**边界限定**：本页面聚焦于单相感应电机的EMT建模，不包括三相感应电机（见[[induction-machine]]）或同步电机（见[[induction-machine-model]]）。

## EMT中的作用

单相感应电机是配电系统的重要负荷，其EMT建模需求来自多个工程场景：

- **居民负荷**：家用电器（空调、冰箱、洗衣机）的主要动力，占Hydro-Québec基荷的2/3以上
- **商业负荷**：小型水泵、风机、压缩机
- **农村电网**：灌溉泵、农用机械
- **微网分析**：孤岛系统的电机启动暂态问题
- **电压稳定性**：大规模电机同时启动引起的电压跌落和恢复动态
- **大扰动仿真**：故障期间电压跌落高达50%时的电机响应

在EMT仿真中，单相电机与三相电机面临不同的建模挑战：故障导致的大转差率（s接近1）会使传统单笼模型失效，表现为零起动转矩特性无法复现。此外，单相电机的辅助绕组只在启动阶段工作，离心开关断开后电机进入单绕组运行状态，这种拓扑切换增加了EMT接口的复杂性。

## EMT建模方法

### 1. 双轴理论模型（d-q Reference Frame）

主绕组（m）和辅助绕组（a）在静止坐标系下的电压方程：

$$v_m = R_m i_m + \frac{d\psi_m}{dt}$$

$$v_a = R_a i_a + \frac{d\psi_a}{dt}$$

磁链方程：

$$\begin{bmatrix} \psi_m \\ \psi_a \end{bmatrix} = \begin{bmatrix} L_m & L_{ma} \\ L_{am} & L_a \end{bmatrix} \begin{bmatrix} i_m \\ i_a \end{bmatrix}$$

互感 $L_{ma} = L_{am}$ 是转子位置的函数，与三相电机的Park变换类似，需要将双绕组电压方程变换到转子旋转坐标系（d-q坐标系）消去时变电感。定子电压方程在d-q坐标系下：

$$v_{ds} = R_s i_{ds} + \frac{d\psi_{ds}}{dt} - \omega_r \psi_{qs}$$

$$v_{qs} = R_s i_{qs} + \frac{d\psi_{qs}}{dt} + \omega_r \psi_{ds}$$

转子电压方程：

$$v_{dr} = R_r i_{dr} + \frac{d\psi_{dr}}{dt} - (\omega_s - \omega_r)\psi_{qr} = 0$$

$$v_{qr} = R_r i_{qr} + \frac{d\psi_{qr}}{dt} + (\omega_s - \omega_r)\psi_{dr} = 0$$

电磁转矩：

$$T_e = \frac{3}{2}p\left(\psi_{ds} i_{qs} - \psi_{qs} i_{ds}\right)$$

### 2. 正序/负序等效电路

单相电机的运行特性可分解为正序和负序两个对称分量系统。定义转差率 $s = (n_s - n_r)/n_s$，其中 $n_s$ 为同步转速，$n_r$ 为转子实际转速。正序和负序转差率分别为 $s$ 和 $2-s$：

$$Z_+ = R_1 + jX_1 + \frac{(R_2'/s + jX_2')(jX_m)}{R_2'/s + j(X_2'+X_m)}$$

$$Z_- = R_1 + jX_1 + \frac{(R_2'/(2-s) + jX_2')(jX_m)}{R_2'/(2-s) + j(X_2'+X_m)}$$

合成电磁转矩：

$$T_e = T_+ + T_-$$

其中 $T_+ = |I_+|^2 R_2'/s/\omega_s$ 为正序转矩（驱动），$T_- = |I_-|^2 R_2'/(2-s)/\omega_s$ 为负序转矩（制动）。启动时 $s=1$，正负序转矩方向相反，合成转矩可能为零——这正是传统单笼模型的失效点。

### 3. 启动方式与等效电路

**电容启动（Capacitor-Start）**：
- 启动电容与辅助绕组串联，容抗补偿主辅绕组的相位差
- 离心开关在转速达到75%额定转速时断开，切除辅助回路
- 启动转矩大（可达额定转矩的2-3倍），启动电流小
- 适用于重载启动场景（压缩机、水泵）

**电容运行（Capacitor-Run）**：
- 小电容永久连接，不使用离心开关
- 运行性能改善，功率因数可达0.8-0.9
- 启动转矩较小（额定转矩的0.5-1倍）
- 适用于长期运行场景（风扇、空调室内机）

**电阻启动（Split-Phase）**：
- 辅助绕组电阻较大（通过匝数减少或线径细化实现）
- 结构简单，成本低，无电容失效风险
- 启动转矩较小（额定转矩的0.5-1倍）
- 适用于轻载启动场景（搅拌机、小风机）

### 4. 电压-频率相关静态特性

电压和频率变化时，有功和无功功率的变化规律为：

$$P = P_0 \left(\frac{V}{V_0}\right)^{n_p} \left[1 + K_p\frac{f-f_0}{f_0}\right]$$

$$Q = Q_0 \left(\frac{V}{V_0}\right)^{n_q} \left[1 + K_q\frac{f-f_0}{f_0}\right]$$

典型值：$n_p = 0.5 \sim 1.5$（有功电压指数），$n_q = 1.5 \sim 4.0$（无功电压指数）。单相电机的静态特性在宽电压范围（0.5-1.5 pu）内仍能保持物理一致性，是负荷聚合模型的重要组成部分。

### 5. 二次方转矩机械负载模型

居民负荷中的单相感应电机通常驱动二次方转矩机械负载（如风扇、泵）：

$$T_m = T_0 \left(\frac{n_r}{n_{r0}}\right)^2 = T_0(1-s)^2$$

该模型比恒转矩模型更接近实际，因为转矩与转速的平方成正比，模拟了流体惯性和系统阻尼。仿真研究表明，在电压暂降期间，采用二次方转矩机械负载模型的单相感应电机动态响应与现场录波数据匹配度显著优于恒转矩模型。

## 量化性能边界

### 模型有效电压范围

| 研究来源 | 模型方法 | 有效电压范围 | LOADSYN原有范围 | 范围扩展 |
|---------|---------|------------|----------------|---------|
| Khodabakhchian 1997 | EMTP UM-7 + 变压器饱和模型 | **0.5-1.5 pu** | 0.85-1.15 pu | **+233%** |
| 梯形法常规模型 | 静态指数模型 | 0.85-1.15 pu | — | 基准 |

### 故障动态量化数据（Khodabakhchian 1997, Hydro-Québec Brossard变电站验证）

- **故障类型**：单相接地故障（SLG）
- **正序电压跌落**：17.5%（1.0 pu → 0.825 pu）
- **故障相电压跌落**：36%（1.0 pu → 0.64 pu）
- **健康相电压跌落**：8%
- **故障持续时间**：2.5周波（约41.67 ms，60 Hz系统）
- **有功功率变化**：约300 MW → 240 MW（故障期间）→ 恢复至300 MW
- **无功功率变化**：约50 MVAr → -50 MVAr（故障后电机转变为无功来源）→ 恢复
- **电机接触器脱扣阈值**：额定电压的45%-55%区间，持续1-3个工频周波
- **电压恢复至65%额定值后**：70%的电机负荷在20 ms内完成重合闸

### 启动电流特性

$$I_{start} = (5-7) I_N$$

单相电机直接启动电流比三相电机更大，原因是缺少两相电流的抵消效应。在EMT仿真中，启动电流的峰值和持续时间直接影响配电变压器饱和和母线电压暂降的幅度。

### 额定转差率

$$s_N = \frac{n_s - n_N}{n_s}$$

典型值：0.02-0.06（电容运行电机），0.04-0.08（电容启动电机）。

## 适用边界与失效模式

### 适用条件

- 额定电压和频率范围已知（通常220 V/50 Hz或110 V/60 Hz）
- 负载转矩特性已知（恒转矩、二次方、还是混合型）
- 启动装置参数（电容值、离心开关动作转速和时间）明确
- 电网短路容量足够（避免过度电压跌落导致电机停转）
- 电压变化范围在0.5-1.5 pu内

### 失效边界

| 问题场景 | 表现 | 原因 | 缓解办法 |
|---------|------|------|---------|
| 电压过低导致启动失败 | 电机无法启动，转矩不足 | 转矩与电压平方成正比 | 提高启动电压或使用变频器 |
| 电容老化 | 启动困难或停转 | 电容容量下降改变相位角 | 定期更换电容或使用固态启动器 |
| 离心开关故障 | 辅助绕组过热或烧毁 | 开关无法断开导致长期通电 | 使用无电容方案或固态开关 |
| 过载过热 | 绝缘老化或停转 | 通风不良或负载过大 | 加装热继电器或温度监测 |
| 单相运行（辅绕组开路） | 电机振动、电流不平衡 | 一相断开后的不对称运行 | 电机保护继电器 |

### 关键假设

1. 气隙磁场正弦分布（忽略空间谐波）
2. 铁损可忽略或等效到定子电阻
3. 绕组参数对称，除匝数比外忽略其他不对称
4. 启动开关（离心开关或固态开关）动作理想，无弹跳或延迟
5. 机械负载转矩特性已知且可建模

## 相关模型与建模体系

### 电机模型层级

```
感应电机建模
├── 三相感应电机
│   ├── 经典 dq0 模型（转子速dq坐标系）
│   ├── 相域模型（abc坐标系）
│   ├── 电压源后阻抗（VBR）模型
│   └── 平均值模型（AVM）
└── 单相感应电机
    ├── 双绕组双轴模型（本文）
    ├── 电压源后阻抗模型
    └── 等效电路模型（正序/负序）
```

### 相关的EMT建模技术

- **多速率仿真**（[[multirate-method]]）：单相电机与系统网络的接口稳定性
- **动态相量法**（[[dynamic-phasor]]）：慢动态下使用相量表示加速仿真
- **变压器饱和模型**（EMTP模型98）：配电变压器饱和影响电机端电压
- **平均值模型**（[[average-value-model]]）：变流器供电单相电机的降阶等效
