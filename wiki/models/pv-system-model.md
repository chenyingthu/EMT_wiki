# 光伏系统 (Photovoltaic System)

## 1. 物理对象概述

### 1.1 功能与结构

光伏发电系统将太阳能转换为电能，主要组件包括：

**光伏阵列 (PV Array)**：
- 光伏组件串并联组成
- 单晶硅/多晶硅/薄膜技术
- 额定功率：kW级（分布式）至GW级（集中式）

**DC/AC变换器 (逆变器)**：
- 组串式逆变器（10-100kW）
- 集中式逆变器（>1MW）
- 微型逆变器（组件级）

**并网接口**：
- 升压变压器（10kV/35kV）
- 滤波器（L/LCL）
- 保护设备

### 1.2 工作原理

**光伏效应**：
光子激发电子-空穴对 → 内建电场分离 → 产生光生电流

**I-V特性**：
$$I = I_{ph} - I_0 \left[\exp\left(\frac{q(V+IR_s)}{nkT}\right) - 1\right] - \frac{V+IR_s}{R_{sh}}$$

其中：
- $I_{ph}$：光生电流（与辐照度成正比）
- $I_0$：反向饱和电流
- $n$：理想因子（1-2）
- $R_s$：串联电阻
- $R_{sh}$：并联电阻

### 1.3 运行激励

**环境条件**：
- 太阳辐照度：0-1000 W/m²
- 环境温度：-20°C至+50°C
- 光谱分布：AM1.5标准光谱

**电气激励**：
- 直流电压：200-1500V（取决于串联数量）
- 交流电压：220V/380V/10kV/35kV
- 频率：50Hz/60Hz

**故障激励**：
- 低电压穿越（LVRT）
- 高电压穿越（HVRT）
- 频率扰动

## 2. 物理模型与数学描述

### 2.1 光伏电池模型

#### 2.1.1 单二极管模型

等效电路：电流源并联二极管和并联电阻，再串联电阻

**输出特性方程**：
$$I = I_{ph} - I_0 \left[\exp\left(\frac{V+IR_s}{V_t}\right) - 1\right] - \frac{V+IR_s}{R_{sh}}$$

其中热电压：
$$V_t = \frac{N_s k T A}{q}$$

**参数温度依赖**：
- $I_{ph}(T) = I_{ph,ref} \left[1 + \alpha_{sc}(T - T_{ref})\right] \frac{G}{G_{ref}}$
- $I_0(T) = I_{0,ref} \left(\frac{T}{T_{ref}}\right)^3 \exp\left[\frac{qE_g}{nk}\left(\frac{1}{T_{ref}} - \frac{1}{T}\right)\right]$
- $V_{oc}(T) = V_{oc,ref} \left[1 + \beta_{oc}(T - T_{ref})\right]$

#### 2.1.2 最大功率点

最大功率点电压/电流：
$$V_{mpp} \approx V_{oc} - V_t \ln\left(1 + \frac{V_{mpp}}{V_t}\right)$$
$$P_{max} = V_{mpp} \cdot I_{mpp}$$

典型值：$V_{mpp}/V_{oc} \approx 0.75-0.8$，$I_{mpp}/I_{sc} \approx 0.9$

### 2.2 逆变器模型

#### 2.2.1 平均模型

**直流侧**：
$$C_{dc} \frac{dV_{dc}}{dt} = I_{pv} - I_{dc}$$

**交流侧**：
$$v_{abc} = m_{abc} \cdot \frac{V_{dc}}{2}$$
$$I_{dc} = \frac{3}{4} \sum_{k=a,b,c} m_k i_k$$

#### 2.2.2 控制策略

**MPPT控制**：
- 扰动观察法（P&O）
- 增量电导法（IncCond）
- 参考电压：$V_{ref} = V_{mpp}$

**电流控制环**：
$$i_{d,ref} = k_p(V_{dc,ref} - V_{dc}) + \int k_i(V_{dc,ref} - V_{dc})dt$$
$$i_{q,ref} = -Q_{ref}/(1.5V_d)$$

**PLL同步**：
$$\omega = \omega_0 + k_{p,pll} \cdot v_q + k_{i,pll} \int v_q dt$$

## 3. EMT仿真模型

### 3.1 详细模型

**光伏阵列**：
- 单二极管模型
- 考虑温度和辐照度变化
- 串联/并联组件配置

**逆变器**：
- IGBT开关模型
- PWM调制（载波频率2-20kHz）
- LCL滤波器

### 3.2 平均值模型

**光伏源**：
- 受控电流源：$I_{pv} = f(V_{dc}, G, T)$
- 查表法或解析计算

**逆变器**：
- 受控电压源：$v_{abc} = m_{abc} V_{dc}/2$
- 控制延时模型（1-2个开关周期）

### 3.3 大规模等值

**光伏电站聚合**：
- 等效光伏特性曲线
- 等效逆变器容量
- 集电线路阻抗
