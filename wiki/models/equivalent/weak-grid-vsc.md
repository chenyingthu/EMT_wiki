# 弱电网VSC模型

## 定义与概述

弱电网VSC模型用于描述电压源换流器（VSC）在接入弱交流电网（低短路比SCR条件）时的电磁暂态行为。电网强度通常用短路比（Short Circuit Ratio, SCR）量化：SCR < 3定义为弱电网，SCR < 2定义为极弱电网（IEEE Std 1204-1997）。在弱电网条件下，VSC的锁相环（PLL）稳定性、功率传输能力、控制动态特性与强网条件下有本质差异，需采用专门建模方法和控制策略（如构网控制GFM）以保证仿真精度和系统稳定性。

## 电网强度分类与指标

### 短路比（SCR）

$$\text{SCR} = \frac{S_{sc}}{P_n}$$

其中 $S_{sc}$ 为公共连接点（PCC）的三相短路容量（MVA），$P_n$ 为VSC额定功率（MW）：
- **强电网**：SCR > 5
- **中等电网**：3 ≤ SCR ≤ 5
- **弱电网**：SCR < 3
- **极弱电网**：SCR < 2

### 临界短路比（CSCR）

系统保持稳定运行的最小SCR值，取决于VSC控制架构和参数：
- 传统跟网型（GFL）VSC：CSCR ≈ 1.5-2.5
- 构网型（GFM）VSC：CSCR可低至1.0-1.5
- Jiang (2025) 基于dq域多频率扫描的CSCR判别方法：在CSCR = 3.7时检测到控制交互风险

## 弱电网VSC的EMT建模关键问题

### PLL稳定性

弱电网下电网阻抗增大，PLL与电网阻抗形成负阻效应，可能导致失稳。Ranasinghe (2024) 提出的改进DSOGI-PLL（自适应带宽+暂态频率冻结）在SCR ≈ 1.8的弱电网中，不对称故障下调节时间缩短60%（0.040 s → 0.016 s），频率跟踪RMSE降至0.001 Hz（降幅99.95%）。

### 功率传输极限

弱电网下VSC有功功率传输受PCC电压波动限制，当SCR降低时等效电网阻抗增大，相同功角下传输功率下降且电压灵敏度升高。

### 控制交互

弱电网中VSC电流环、PLL和外环控制之间形成多时间尺度交互。Li (2024) 指出弱电网下VSC外环控制带宽需限制在 < 30 Hz以避免与PLL交互失稳。跟网型控制与电网阻抗的交互可通过阻抗分析法（奈奎斯特判据）评估。

## 弱电网VSC建模方法

### 跟网型（GFL）模型

适用于SCR ≥ 2的电网条件，采用PLL同步+矢量控制。PLL小信号模型描述dq坐标系下的同步动态。

### 构网型（GFM）模型

适用于SCR < 2的极弱电网或孤岛供电，无需PLL，通过有功-频率和无功-电压下垂模拟同步机外特性。Allabadi (2024) 提出了MTDC系统中GFM VSC的两种EMT初始化方法（CISS和DI），在CIGRE BM4基准系统中将初始化时间缩短6.9倍，适用于GFM VSC接入弱网的EMT仿真启动。

## 量化性能边界

**SCR稳定边界**：
- 跟网型VSC（SRF-PLL，带宽30-50 Hz）：稳定运行下限SCR ≈ 2.0-2.5
- 跟网型VSC（DSOGI-PLL）：稳定运行下限SCR可扩展至≈ 1.8（Ranasinghe 2024）
- 构网型VSC（下垂控制）：稳定运行下限SCR可低至≈ 1.0-1.5

**PLL暂态性能（SCR ≈ 1.8，IEEE 9节点，Ranasinghe 2024）**：
- 不对称故障：调节时间0.016 s（对比传统PLL 0.040 s，缩短60%）
- 相位跳变（+60°）：调节时间0.03 s（对比传统PLL 0.15 s，缩短80%）
- 频率跟踪RMSE：0.001 Hz（对比传统PLL 2.16 Hz，降幅99.95%）

**控制参数边界**：
- Li (2024)：弱电网下VSC外环带宽建议 < 30 Hz
- 电流内环带宽通常为外环5-10倍，弱电网下受LCL谐振频率限制

**初始化性能**：
- Allabadi (2024)：CISS/DI方法相比传统潮流初始化，GFM VSC在MTDC系统中初始化时间缩短6.9倍

**数据缺口声明**：不同电网强度（SCR从5降至1）下VSC控制参数的自适应调节规律缺乏统一量化框架。GFM与GFL控制在相同弱电网条件下的稳态/暂态性能边界缺乏系统对比数据。弱电网下VSC的EMT模型验证多基于特定测试系统，不同文献的结论可比性有限。

## 适用边界

**适用条件**：
- VSC接入弱电网（SCR < 3）的EMT暂态研究
- 跟网型/构网型控制在弱电网条件下的稳定性评估
- 新能源并网系统的PLL参数设计与优化

**失效边界**：
- SCR < 1.0时多数跟网型控制失效，需采用GFM或混合控制
- 极弱电网下线性化小信号模型不适用于大扰动暂态分析
- 不同PLL结构（SRF/DSOGI/延时信号消除）在相同SCR下的稳定性边界不同，不可泛化
