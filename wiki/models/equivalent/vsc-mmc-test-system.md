# VSC-MMC测试系统

## 定义与概述

VSC-MMC测试系统是用于验证VSC和MMC EMT模型精度与效率的标准化基准系统。该类系统通常包含两端或多端VSC/MMC换流站、交直流网络和控制保护系统，广泛应用于模型降阶、平均值模型验证、控制策略开发和硬件在环测试等研究。

## 典型配置

### 两电平VSC测试系统

**基本参数**（参考2012年DAVM标准测试系统）：
- 额定容量：10-1000 MVA
- AC电压：12.5-525 kV
- DC电压：±5-±800 kV
- 开关频率：1980-2000 Hz
- DC电容：200-5000 μF

**控制模式**：
- 整流站：定DC电压 + 定无功功率
- 逆变站：定有功功率 + 定无功功率

### MMC测试系统

**401电平MMC系统**（Peralta 2012, Saad 2013）：
- 额定容量：1000 MW
- 子模块数：400/桥臂（401电平）
- DC电压：±320 kV
- AC电压：220 kV
- 总IGBT数：4812
- 子模块电容：10 mF
- 桥臂电感：100 mH

**41电平MMC系统**（IEEE标准测试系统）：
- 额定容量：640 MW
- 子模块数：40/桥臂
- DC电压：±320 kV
- 仿真步长：1-5 μs（详细模型），20-100 μs（AVM）

## 量化性能边界

**仿真步长范围**：
- 详细开关模型：1-5 μs
- 戴维南等效模型：5-20 μs
- 改进AVM：20-50 μs
- 简化AVM（DI-AVM）：可达1000 μs（Ebrahimi 2023）

**精度对比（401电平MMC）**：
- 稳态：AVM的直流电压误差 < 3%，功率分配误差 < 5%
- 直流故障：详细模型峰值电流约17 p.u.，AVM约73 p.u.（IEEE Task Force 2013）

**计算加速比**：
- Gnanarathna (2011) 戴维南等效：310倍加速（120子模块/桥臂）
- DAVM (2012)：CPU减少50-70%（5-40 μs步长）
- IEEE Task Force (2013) AVM：10-16倍加速（401电平系统）

**数据缺口声明**：不同测试系统配置（两电平VSC vs. MMC，不同电平数）之间的精度-加速比系统对比数据不足。测试系统参数在不同文献中缺乏一致性，难以直接比较不同建模方法。

## 适用边界

**适用条件**：
- VSC/MMC建模方法的精度验证与对比
- 控制策略的离线开发与测试
- 模型降阶和平均值模型的基准测试

**失效边界**：
- 测试系统参数与实际工程的差异可能导致验证结果偏差
- 小规模测试系统（低电平数）的结论不可直接推广至大规模系统
- 不同文献的测试系统配置不一致，影响方法间公平对比

## EMT中的作用

VSC-MMC测试系统 在EMT仿真中主要用于：

- **建模对象**：描述VSC-MMC测试系统在电力系统中的物理角色和电气特性
- **仿真场景**：适用于VSC-MMC测试系统相关的电磁暂态分析、故障响应、控制交互等场景
- **模型接口**：提供VSC-MMC测试系统的端口变量、状态方程和边界条件
- **验证基准**：可作为VSC-MMC测试系统仿真模型正确性的验证基准
