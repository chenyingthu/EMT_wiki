# Grid Connected Inverter

## 定义与概述

并网逆变器（Grid-Connected Inverter）是将可再生能源（光伏、风电）或储能系统的直流电转换为交流电并馈入电网的电力电子装置，是新能源并网系统的核心接口设备。本模型涵盖并网逆变器的主电路拓扑、控制策略、滤波器设计及 EMT 仿真建模方法，适用于光伏并网、风电并网、储能系统等应用场景的电磁暂态仿真。

### 核心功能
- 直流-交流功率变换
- 最大功率点跟踪（新能源）
- 并网电流/电压控制
- 电网同步与锁相
- 故障穿越与电网支撑

### 运行激励

**输入信号**：
- 直流侧电压 $V_{dc}$
- 并网点三相电压 $v_a, v_b, v_c$
- 功率给定 $P_{ref}, Q_{ref}$
- 电网频率/相位参考

**输出信号**：
- 三相并网电流 $i_a, i_b, i_c$
- 开关管 PWM 驱动信号
- 功率注入 $P, Q$

## 拓扑结构

### 两电平电压源逆变器

两电平 VSI 是最基本的并网逆变器拓扑，由六个开关管（IGBT/MOSFET）组成三相桥臂。

**主电路方程**（dq 旋转坐标系）：
$$L\frac{di_d}{dt} = v_d - Ri_d + \omega Li_q - v_{dc}s_d$$
$$L\frac{di_q}{dt} = v_q - Ri_q - \omega Li_d - v_{dc}s_q$$

### 三电平中点箝位（NPC）逆变器

NPC 逆变器每个桥臂输出三种电平，适用于中高压并网应用。

### 多电平逆变器

包括级联 H 桥（CHB）、模块化多电平变换器（MMC）等拓扑，适用于高压大功率并网场景。

## 控制策略

### 跟网型控制（GFL）

通过锁相环（PLL）跟踪电网相位，以电流源模式运行。

**电流环控制**：
$$v_d^* = (K_p + K_i/s)(i_d^* - i_d) - \omega L i_q + v_d$$
$$v_q^* = (K_p + K_i/s)(i_q^* - i_q) + \omega L i_d + v_q$$

### 构网型控制（GFM）

模拟同步发电机特性，自主建立电压和频率，以电压源模式运行。

**下垂控制**：
$$f = f_0 - k_p(P - P_0)$$
$$V = V_0 - k_q(Q - Q_0)$$

## LCL 滤波器设计

并网逆变器通常采用 L 或 LCL 滤波器抑制开关谐波。

**谐振频率**：
$$f_{res} = \frac{1}{2\pi}\sqrt{\frac{L_1 + L_2}{L_1 L_2 C_f}}$$

**设计约束**：
- $10f_{grid} < f_{res} < 0.5f_{sw}$
- 电容无功功率 < 5% 额定功率
- 总电感压降 < 10%

## EMT 仿真建模

### 详细开关模型

保留所有开关器件的导通/关断动态，适用于开关谐波分析和损耗评估，计算量大。

### 平均值模型（AVM）

将逆变器等效为受控电压/电流源，忽略开关纹波，适用于系统级稳定性分析。

### 适用边界

**适用条件**：
- 新能源并网系统的电磁暂态分析
- 并网逆变器控制参数设计和优化
- 电网故障穿越和支撑能力评估

**失效边界**：
- 短路比（SCR）低于临界值时 PLL 失稳
- 详细开关模型在大规模系统中计算负担过大
- 平均值模型在开关频率以下信息丢失

### 量化性能边界

并网逆变器 EMT 建模的精度取决于模型保真度选择（详细开关 vs 平均值）、离散化步长和控制参数。以下汇总可引用的量化数据：

**平均值模型精度**：DAVM (2012) 验证了 VSC-HVDC 动态平均值模型在 5 μs 步长下 CPU 减少 50-54%，≥40 μs 步长下减少 60-70%。Luchini (2023) 的 GFL 等效模型与全开关基准相比故障电流平均误差约 2.33%，仿真时间减少约 70%。

**模型选择对比**：Misyris (2021) 证明了构网型变流器在满足时间尺度分离条件下 RMS 模型与 EMT 误差小于 3%，RMS 计算时间缩短 80-90%。

**PLL 同步性能**：Ranasinghe (2024) 的改进型 DSOGI-PLL 在不对称故障下调节时间从 0.040 s 缩短至 0.016 s，频率跟踪 RMSE 从 2.16 Hz 降至 0.001 Hz。

**数据缺口声明**：并网逆变器建模在不同电网强度（SCR）、不同拓扑和不同控制策略下的系统量化对比在公开文献中缺乏统一基准。不同滤波器配置（L vs LCL）对 EMT 仿真精度的影响未见独立评估。建议用户根据具体应用选择合适的模型保真度。
