# Grid-Forming Inverter

## 定义

Grid-Forming Inverter（构网型变流器，GFM）是新型电力系统中能够**独立建立并维持电网电压和频率**的逆变器技术。与跟网型（Grid-Following Inverter, GFL）依赖锁相环（PLL）跟踪电网相位不同，GFM 通过自身控制生成电压参考，为弱电网、孤岛系统和高比例新能源电力系统提供惯量支撑。

从电磁暂态（EMT）仿真角度，GFM 变流器的核心挑战在于：如何在无外部电网参考的条件下正确初始化控制器状态、如何在限幅非线性工况下保持电压源特性、如何实现多机并联时的自主功率均分。

**核心功能**：
- 独立建立电压幅值和频率（自主电压源特性）
- 提供虚拟惯量支撑（模拟同步发电机转子运动方程）
- 实现多机并联功率均分（下垂控制无需通信）
- 黑启动能力（可作为孤岛系统的一次电源）

**运行工况**：

输入信号：直流侧电压 $V_{dc}$、并网点三相电压/电流、有功/无功功率给定 $P_{ref}, Q_{ref}$、额定频率/电压参考

输出信号：三相交流电压（电压源特性）、PWM 开关驱动信号、实际功率注入 $P, Q$

## EMT 中的角色

在 EMT 仿真中，GFM 变流器承担**电压源构建者**的角色，其核心在于模拟同步发电机的外特性。GFM 的 EMT 建模面临以下核心挑战：

1. **控制器初始化**：GFM 采用 V/f 控制，外环积分器初值若与潮流工况不匹配，切除辅助电压源后会产生长时间暂态振荡
2. **电压源特性维护**：GFM 在稳态时表现为电压源，但过载或限幅时退化为电流源，模型需要正确反映这种模式切换
3. **多机并联同步**：多台 GFM 并联时，下垂系数设计不当会导致机间同步振荡
4. **弱电网适应性**：短路比（SCR）低至 1~3 时，GFM 与电网的交互动态与常规并网场景差异显著

**关键需求**：
- 精确的初始条件计算（避免启动暂态耗时）
- 能够在 1μs~100μs 步长下准确捕捉控制响应
- 支持 V/f 控制、功率同步环（PSL）、虚拟同步机（VSG）等多种构网策略

## EMT 建模方法

### 方法 1：详细开关模型（Detailed Switch Model, DSM）

保留所有 IGBT/MOSFET 开关器件的导通/关断动态，在每个仿真时步精确计算器件换相状态。

**原理**：三相桥式逆变器电路包含 6 个开关器件，每个开关以二进制状态 $S_k \in \{0,1\}$ 表示导通/关断。开关状态由 PWM 调制信号 $d_k$ 决定：

$$S_k = \begin{cases} 1 & \text{if } d_k > 0.5 \\ 0 & \text{if } d_k \le 0.5 \end{cases}$$

直流侧电压 $V_{dc}$ 经桥臂换相后在交流侧产生 PWM 波形，与 L/LC/LCL 滤波器耦合后注入 PCC 点。

**特点**：
- 最高保真度，可捕捉开关纹波、换相暂态、谐波特性
- 计算量最大，实时仿真步长通常 ≤ 1μs（FPGA 实现）或 ≤ 10μs（CPU 多核）
- 适用于故障穿越分析、谐波评估、开关器件应力分析

**适用场景**：故障穿越仿真、谐波分析、开关器件损耗评估

### 方法 2：平均值模型（Average Value Model, AVM）

忽略开关纹波，用占空比 $d_k$ 的平均值替代离散开关状态，等效为一个受控电压源。

**原理**：三相桥臂等效输出电压为：

$$\begin{bmatrix} e_a \\ e_b \\ e_c \end{bmatrix} = \frac{V_{dc}}{2} \begin{bmatrix} d_a \\ d_b \\ d_c \end{bmatrix}$$

其中占空比 $d_k$ 由 dq 坐标系下的电压外环和电流内环确定：

$$d_d^{ref} = K_{pvd} (V_{dc}^{ref} - V_{dc}) + \frac{1}{T_{iv}} \int (V_{dc}^{ref} - V_{dc}) dt$$

$$d_q^{ref} = K_{pvi} (i_q^{ref} - i_q) + \frac{1}{T_{ii}} \int (i_q^{ref} - i_q) dt$$

**特点**：
- 计算效率比 DSM 高 10~100 倍
- 保留 GFM 控制外特性（下垂响应、惯量支撑）
- 无法捕捉开关谐波和高频暂态
- 限幅非线性会导致模型精度下降

**适用场景**：系统级稳定性分析、多机并联仿真、实时仿真加速

### 方法 3：虚拟同步机模型（Virtual Synchronous Machine, VSM）

在平均值模型基础上，增加同步发电机转子运动方程的物理模拟，实现虚拟惯量响应。

**原理**：转子运动方程（ swing equation）为：

$$J \frac{d\omega}{dt} = P_{ref} - P_e - D(\omega - \omega_0)$$

$$\frac{d\theta}{dt} = \omega$$

其中 $J$ 为虚拟惯量（kg·m²），$D$ 为阻尼系数（N·m·s/rad），$P_e$ 为电磁功率，$\omega_0$ 为额定角频率。

电磁功率由 dq 坐标系下计算：

$$P_e = \frac{3}{2} (V_d i_d + V_q i_q)$$

**特点**：
- 物理意义清晰，惯量响应可量化
- 惯量系数 $J$ 影响系统等效惯性时间常数 $H_{eq} = J\omega_0^2 / (2S_{rated})$
- 阻尼 $D$ 可抑制低频振荡
- 适用于惯量评估、黑启动、并联稳定性分析

**适用场景**：惯量支撑分析、黑启动、含多台 GFM 的系统稳定性分析

### 方法 4：稳态控制初始化（Controller Integration Steady-state Initialization, CISS）

针对 GFM 变流器的 EMT 仿真初始化问题，利用潮流计算结果和已知控制器参数，解析计算外环 PI 积分器初值。

**原理**：给定潮流计算得到的 PCC 电压相量 $\dot{V}_{pcc}$、交流电流相量 $\dot{I}_{ac}$、直流电压 $V_{dc}$，以及换流器阻抗参数（变压器阻抗 $Z_t$、桥臂电抗 $X_{arm}$）和控制参数（电压环积分增益 $K_i$），通过交流侧 KVL 反推参考电压：

$$E_d^{ref} = K_i h + |V_{ac}^{set}|$$

其中 $h$ 为 PI 积分器状态，$V_{ac}^{set}$ 为电压设定值。

外环积分器初值计算流程：
1. 由 $\dot{V}_{pcc}$ 和 $\dot{I}_{ac}$ 计算 PCC 点复功率 $S_{pcc} = \dot{V}_{pcc} \dot{I}_{ac}^*$
2. 计算内部参考电压 $\dot{E}_{inv} = \dot{V}_{pcc} + (R_t + jX_{arm}) \dot{I}_{ac}$
3. 由 $V_{dc}$ 标幺化关系转成 d 轴参考量
4. 由电压环方程反解积分状态 $h$

**特点**：
- 消除辅助电压源与 GFM 控制器之间的逻辑冲突
- 初始化时间缩短 6.9 倍（Allabadi 2024）
- 需要模型内部参数可读（适合研究型模型）

**适用场景**：大规模 MTDC 系统初始化、多端柔性直流输电

### 方法 5：解耦接口法（Decoupling Interface, DI）

通过在 GFM 并网点插入接口辅助源（Interface Auxiliary Source, IAS），将孤岛电网子系统（Island Grid Subsystem, IGS）与 GFM 电气解耦，使两侧子系统独立收敛后再恢复原始拓扑。

**原理**：
1. 在 GFM 并网点注入 IAS，等效为电流源维持潮流功率交换
2. 同时在 IGS 侧注入等效电压源复制端口电压条件
3. 两侧子系统分别进行时域仿真至稳态（独立收敛）
4. 断开 IAS，恢复原始拓扑，此时系统已处于稳态可直接进入研究场景

**特点**：
- 无需访问 GFM 内部控制参数（适用于黑盒/供应商模型）
- 初始化时间同样缩短 6.9 倍
- 需要正确识别外部控制模式（V/f 控制）

**适用场景**：工程供应商黑盒模型接入、OEM 换流器初始化

### 方法 6：PQ 能力边界模型

针对 GFM 逆变器的运行域约束，综合考虑 PWM 饱和约束和额定电流约束，在 EMT 仿真中实时限制 PQ 指令。

**原理**：PCC 点有功/无功功率传输方程为：

$$P_{PCC} \approx V_{PCC} \frac{E_{inv}}{X_F} \delta_{inv}$$

$$Q_{PCC} \approx V_{PCC} \frac{E_{inv} - V_{PCC\_F}}{X_F}$$

双重约束条件：
- **额定电流约束**：$\sqrt{(P^*_{PCC})^2 + (Q^*_{PCC})^2} \le S_{rated}$
- **PWM 饱和约束**：$| \vec{E}_{inv} | \le E_{max}$

其中 $E_{max}$ 取决于调制策略（SVPWM: 1.53 p.u.; SPWM: 1.33 p.u.，Nurunnabi 2025）。

下垂控制方程：
$$f_{ik} = f_{set} + m_{pi}(P_{kset\_i} - P_{ki}) \quad \text{(P-f 下垂)}$$

$$V_{ik} = V_{set} + m_{qi}(Q_{kset\_i} - Q_{ki}) \quad \text{(Q-V 下垂)}$$

**特点**：
- 防止低电压工况下过流和 PWM 调制饱和
- 运行域随 PCC 电压、直流电压动态更新
- 区分 L/LC/LCL 不同滤波器拓扑的影响

**适用场景**：弱电网并网、多机并联功率分配、IEEE 1547 合规性评估

## 形式化表达

### PCC 点功率传输方程

$$P_{PCC} \approx V_{PCC} \frac{E_{inv}}{X_F} \delta_{inv}, \quad Q_{PCC} \approx V_{PCC} \frac{E_{inv} - V_{PCC\_F}}{X_F}$$

*PCC 点有功/无功功率近似方程，用于分析相角差对有功传输、电压幅值差对无功传输的影响*

### 虚拟同步机转子运动方程

$$J\frac{d\omega}{dt} = P_{ref} - P_e - D(\omega - \omega_0), \quad \frac{d\theta}{dt} = \omega$$

*GFM 惯量响应的核心方程，$J$ 决定虚拟惯性时间常数 $H_{eq} = J\omega_0^2 / (2S_{rated})$*

### PQ 能力双重约束

$$\sqrt{(P^*_{PCC})^2 + (Q^*_{PCC})^2} \le S_{rated} \quad \text{且} \quad |\vec{E}_{inv}| \le E_{max}$$

*额定电流约束（RIC）与 PWM 饱和约束的交集构成实际 PQ 运行域*

### 下垂控制律

$$f = f_0 - m_p (P - P_0) \quad \text{(P-f 下垂)}$$

$$V = V_0 - m_q (Q - Q_0) \quad \text{(Q-V 下垂)}$$

*GFM 无通信自主功率分配的核心控制律*

### 功率同步环频率方程（PSL）

$$\omega_i = \omega_1 + K_p(P_{ref}-P)$$

*PSL 通过有功偏差直接调节输出频率，决定外环主导时间常数（Misyris 2021）*

### 含阻尼与下垂的电压矢量控制方程

$$U = U_{set} - \frac{k_v s}{s+\omega_v} i + K_q(Q_{set}-Q)$$

*用于量化无功控制对相角裕度的耦合影响，含高通滤波阻尼项抑制高频谐振*

### 线路电磁暂态电压-电流方程

$$U = U_{sc} e^{-j(\theta_0+\Delta\theta)} + [R_g + (s+j\omega_i)L_g] i$$

*EMT 模型的核心微分环节，RMS 模型将其简化为代数方程*

## 关键技术挑战

### 挑战 1：过载模式切换导致电压源特性退化

**问题描述**：当 GFM 变流器输出电流超过额定值时，电流限幅环节将强制控制器进入限流模式，变流器从电压源退化为电流源，失去维持 PCC 电压的能力。

**机制**：限幅非线性的本质是 PWM 占空比饱和，此时输出电压幅值不再跟随电压环参考，而是被直流母线和调制策略钳位。EMT 仿真中若采用 AVM，这一非线性可能被低估，导致故障后电压恢复预测偏高。

**应对策略**：在 EMT 模型中嵌入电流限幅检测模块，当 $|i| > I_{rated}$ 时切换为电流源等效模型；同时在 AVM 中引入限幅非线性补偿（参见 Nurunnabi 2025 的 CPID 抗积分饱和机制）。

### 挑战 2：功角失稳与多机同步振荡

**问题描述**：多台 GFM 并联时，下垂系数设计不当会导致机间同步振荡，类似于同步发电机的低频振荡问题，但阻尼机制完全不同。

**机制**：GFM 的同步机制依赖有功-频率下垂（$f = f_0 - m_p \Delta P$），缺乏同步发电机固有的转子惯量阻尼。多机并联时，若下垂系数 $m_p$ 过大或虚拟惯量 $J$ 过小，系统对功率扰动的响应过快，可能引发振荡。

**量化边界**：Nurunnabi 2025 指出，多机并联时 Q-V 下垂系数 3% 在恢复速度与稳态精度间取得最优平衡；频率波动始终 < 0.1 Hz。

**应对策略**：合理选择 $J$ 和 $D$ 参数，确保虚拟惯量与系统惯性时间常数匹配；采用自适应惯量控制（根据 $\Delta\omega$ 动态调整 $J$）。

### 挑战 3：GFM 与 GFL 并联时的交互稳定性

**问题描述**：当系统中同时存在 GFM 和 GFL 变流器时，GFL 的锁相环（PLL）与 GFM 的电压源特性之间可能存在负阻尼交互，引发弱电网下的宽频域振荡（1~50 Hz）。

**机制**：GFL 通过 PLL 跟踪 PCC 电压相位，当 SCR 较低时 PCC 电压相位受 GFM 控制影响较大，PLL 可能追踪到被 GFM 动态污染的相位信号，导致 GFL 输出功率振荡。GFM 则不受 PLL 控制，其电压源特性不受 PLL 带宽影响。

**量化边界**：Jiang 2025 通过 dq0 动态频率扫描得到电压源型 VSG 的临界短路比（CSCR）约 3.7，振荡频率 1.15 Hz。

### 挑战 4：限幅非线性导致平均值模型精度下降

**问题描述**：AVM 在正常运行条件下精度高，但当 GFM 进入限幅模式（过载、故障穿越）时，忽略开关纹波的平均化处理与实际开关行为差异显著增大。

**机制**：AVM 假设 PWM 占空比可以连续调节，但限幅时占空比被钳位在实际边界值，控制环的积分器可能进入饱和状态。此时 AVM 的输出电压与 DSM 的实际输出电压存在显著偏差，故障后恢复时间预测可能偏短。

**应对策略**：在 EMT 仿真中实时监测 PI 积分器状态，当积分器饱和时切换为 DSM 或采用抗积分饱和机制（如 Nurunnabi 2025 的 CPID）。

### 挑战 5：V/f 控制初始化与辅助电压源冲突

**问题描述**：传统基于潮流的初始化方法（LFSI）在 GFM 端口放置辅助电压源强制节点电压等于设定值，导致 V/f 控制器的电压误差信号被强制归零，外环 PI 积分器初值计算错误，切除辅助源后产生长时间暂态振荡。

**机制**：潮流初始化假设网络节点电压由潮流解决定，但 GFM 的 V/f 控制通过积分器主动调节输出电压以跟踪设定值。辅助电压源的存在使积分器感知不到任何误差，因此初值为零或随机值，而非潮流工况对应的稳态值。

**应对策略**：采用 CISS（需要内部参数）或 DI（无需内部参数），Allabadi 2024 证明这两种方法可将初始化时间缩短 6.9 倍。

| 系统稳定性分析（稳态） | AVM 或 VSM | 50~100 μs | 保留控制外特性 |
| 惯量响应评估 | VSM | 50~100 μs | 包含转子运动方程 |
| 黑启动过程仿真 | VSM + DSM 混合 | 10~50 μs | 启动阶段用 VSM，故障时切换 DSM |
| 多机并联功率均分 | AVM + PQ 边界 | 50~100 μs | 加入 PQ 限幅模块 |
| 谐波分析 | DSM | 1~2 μs | 捕捉开关谐波 |
| 实时仿真加速 | AVM + CISS/DI | 10~50 μs | CISS/DI 加速初始化 |
| 弱电网（SCR ≤ 2） | AVM + 限幅补偿 | 10~50 μs | 需考虑限幅非线性 |
| 大规模 MTDC 初始化 | CISS 或 DI | 10 μs | 6.9× 初始化加速 |

### GFM vs GFL 选择指南

| 特征 | GFM（构网型） | GFL（跟网型） |
|-----|------------|-------------|
| 控制目标 | 建立电压/频率 | 跟踪电网相位 |
| PLL 需求 | 无 | 需要 |
| 惯量支撑 | 具备（通过 VSM） | 不具备 |
| 孤岛运行 | 支持 | 不支持 |
| 弱电网适应性 | 强（SCR ≥ 1） | 弱（SCR ≥ 2~3） |
| 多机并联 | 自然均分（下垂控制） | 需要同步机制 |
| 黑启动能力 | 具备 | 不具备 |
| 故障电流贡献 | 高（电压源特性） | 低（电流源特性） |
| 适用场景 | 弱电网、微电网、黑启动 | 强电网并网 |

### GFM 控制策略对比

| 控制策略 | 核心方程 | 特点 | 适用场景 |
|---------|---------|------|---------|
| 基本下垂（FDR） | $f = f_0 - m_p \Delta P$ | 无通信、简单可靠 | 多机并联功率均分 |
| 增强型电压调节（EVR） | FDR + 抗积分饱和 | 动态响应优、THD 低 | 动态负载、故障恢复 |
| 虚拟同步机（VSM） | $J\dot{\omega} = P_{ref} - P_e - D\Delta\omega$ | 物理意义清晰、惯量可量化 | 惯量评估、黑启动 |
| 功率同步环（PSL） | $\omega_i = \omega_1 + K_p(P_{ref}-P)$ | 响应速度快 | SCR 1~2 弱电网 |
| 匹配控制（MCC） | 基于瞬时功率匹配 | 切换平滑 | GFM/GFL 混联 |
| 虚拟振荡器控制（VOC） | 自激振荡耦合 | 无需 PLL | 多逆变器并列 |
