# PWM调制器 (PWM Modulator)

## 定义与概述

PWM（脉冲宽度调制）调制器是电力电子变换器EMT仿真的核心控制环节，通过调制载波与参考波的比较产生开关信号，控制功率器件的通断。本模型涵盖正弦PWM（SPWM）、空间矢量PWM（SVPWM）、特定谐波消除PWM（SHEPWM）、三次谐波注入PWM（THIPWM）、不连续PWM（DPWM）等调制策略，适用于两电平/多电平变换器的EMT仿真。

在EMT仿真中，PWM调制器是连接控制算法与功率电路的桥梁。控制算法计算得到参考电压/电流，经调制器生成开关信号作用于功率器件，产生期望的输出波形。EMT仿真的精度和效率在很大程度上取决于PWM调制环节的建模粒度——从详细的逐开关仿真到平均值等效，多种建模精度层次对应不同的计算代价和适用场景。

## EMT建模中的核心角色

### 为什么需要PWM调制器模型

PWM调制器在EMT仿真中有三个核心功能：

1. **控制-电路接口**：将控制器输出的调制信号（占空比、电压矢量）转换为开关状态，注入电路方程
2. **谐波生成机理**：PWM的开关动作产生与载波频率相关的谐波，这些谐波影响滤波器设计、谐振稳定性分析
3. **精度-效率折中**：不同建模粒度决定仿真步长选择，进而决定计算效率

### 核心挑战

- **开关时刻量化误差**：固定步长仿真中，开关时刻被强制到时间网格上，导致占空比估计偏差
- **谐波保留与计算效率的矛盾**：保留所有开关谐波需要极小步长（~开关周期的1/100），大步长下则丢失谐波信息
- **多电平变流器的调制等效**：MMC等模块化变流器的子模块数量巨大，每个子模块的开关状态独立跟踪导致计算爆炸
- **死区效应的精确建模**：死区时间导致的输出电压损失和波形畸变需要精确捕捉

### 关键需求

根据Sano等(2021)和Horiuchi等(2020)的研究，EMT仿真对PWM调制器的关键需求是：**在同一仿真框架下，实现精度与效率的解析可控折中**，具体表现为：

- 能在较大步长下保留开关谐波信息
- 调制比、载波相位等参数量化可调
- 与节点分析、状态空间等主流EMT求解器兼容

## 物理对象概述

### 功能与分类

**基本功能**：
- 将连续参考信号转换为离散开关信号
- 控制输出电压/电流的平均值
- 决定输出波形质量和谐波特性
- 影响变换器损耗和效率

**PWM类型对比表**：

| 类型 | 调制方式 | 优点 | 缺点 | 应用场景 |
|------|----------|------|------|----------|
| SPWM | 正弦波调制 | 实现简单，谐波可控 | 电压利用率低（0.5Vdc） | 通用逆变器 |
| SVPWM | 空间矢量 | 电压利用率高15%，THD低 | 计算复杂，需坐标变换 | 电机驱动 |
| THIPWM | 三次谐波注入 | 电压利用率提升至1.15倍 | 谐波略有增加 | 三相系统 |
| SHEPWM | 特定谐波消除 | 消除指定次谐波，开关损耗低 | 需离线优化计算 | 大功率变流器 |
| DPWM | 不连续PWM | 减少1/3开关损耗 | 谐波稍大 | 高效率应用 |

### 调制原理

**基本结构**：
```
  参考波 v_ref
       │
       ▼
    ┌──┴──┐    比较    ┌──────┐
    │     ├─⊕──────→│ 开关 │──→ 输出
    │载波 │         │ 信号 │
    │ v_c│←────────┘      └──────┘
    └──┬──┘
       │
    三角波/锯齿波
```

**调制比定义**：
- 幅值调制比：$m_a = \frac{V_{ref,peak}}{V_{c,peak}}$
- 频率调制比：$m_f = \frac{f_c}{f_{ref}}$

### 运行激励

**输入信号**：
- 参考电压：三相或单相正弦参考
- 直流母线电压：$V_{dc}$
- 调制比：$0 \leq m_a \leq 1.15$（取决于调制方式）
- 载波频率：$f_c = 1-20$ kHz

**输出信号**：
- 开关信号：$S \in \{0, 1\}$ 或 $\{-1, 0, 1\}$
- 占空比：$d = \frac{t_{on}}{T_s}$
- 输出电压：$v_{out} = S \cdot V_{dc}$

**性能指标**：
- THD：总谐波失真
- WTHD：加权谐波失真
- 开关损耗：与开关频率和电流相关

## 物理模型与数学描述

### 正弦PWM（SPWM）

#### 双极性SPWM

**两电平变换器**：
$$v_{ao} = \begin{cases} +\frac{V_{dc}}{2}, & v_{ref} > v_c \\ -\frac{V_{dc}}{2}, & v_{ref} < v_c \end{cases}$$

**占空比计算**：
$$d = \frac{1 + m_a \sin(\omega t)}{2}$$

**输出电压基波**：
$$V_{1,rms} = \frac{m_a V_{dc}}{2\sqrt{2}}$$

**线性调制范围**：$0 \leq m_a \leq 1$

#### 单极性SPWM

**半桥变换器**：
$$v_{ao} = \begin{cases} +V_{dc}, & v_{ref} > v_c \text{ 且 } v_{ref} > 0 \\ 0, & v_{ref} < v_c \text{ 且 } v_{ref} > 0 \\ 0, & v_{ref} > -v_c \text{ 且 } v_{ref} < 0 \\ -V_{dc}, & v_{ref} < -v_c \text{ 且 } v_{ref} < 0 \end{cases}$$

**优点**：等效开关频率加倍，谐波特性更好。

### 三次谐波注入PWM（THIPWM）

**注入三次谐波**：
$$v_{ref}' = v_{ref} - \frac{\max(v_a, v_b, v_c) + \min(v_a, v_b, v_c)}{2}$$

或注入15%三次谐波：
$$v_{ref}' = m_a V_c \left[\sin(\omega t) + \frac{1}{6}\sin(3\omega t)\right]$$

**电压利用率**：
$$V_{1,rms} = \frac{1.15 V_{dc}}{2\sqrt{2}}$$

**线性范围**：$0 \leq m_a \leq 1.15$

### 空间矢量PWM（SVPWM）

#### 电压矢量定义

**基本电压矢量**：
$$\vec{V} = \frac{2}{3}(v_a + v_b e^{j120°} + v_c e^{j240°})$$

**两电平变换器开关状态表**：

| 状态 | $S_a$ | $S_b$ | $S_c$ | 矢量 | 幅值 |
|------|-------|-------|-------|------|------|
| V0 | 0 | 0 | 0 | $\vec{V}_0$ | 0 |
| V1 | 1 | 0 | 0 | $\vec{V}_1$ | $2V_{dc}/3$ |
| V2 | 1 | 1 | 0 | $\vec{V}_2$ | $2V_{dc}/3$ |
| V3 | 0 | 1 | 0 | $\vec{V}_3$ | $2V_{dc}/3$ |
| V4 | 0 | 1 | 1 | $\vec{V}_4$ | $2V_{dc}/3$ |
| V5 | 0 | 0 | 1 | $\vec{V}_5$ | $2V_{dc}/3$ |
| V6 | 1 | 0 | 1 | $\vec{V}_6$ | $2V_{dc}/3$ |
| V7 | 1 | 1 | 1 | $\vec{V}_7$ | 0 |

**空间矢量图**：
```
          V3(010)
            │
    V2(110)─┼─V4(011)
           /│\
          / │ \
    V1(101)│V5(101)
            │
          V6(100)
```

#### 矢量作用时间计算

**扇区判断**：根据参考电压角度确定所在扇区。

**作用时间计算**（以第一扇区为例）：
$$\begin{cases} T_1 = \frac{\sqrt{3} T_s}{V_{dc}} \left(\frac{\sqrt{3}}{2}v_\alpha - \frac{1}{2}v_\beta\right) \\ T_2 = \frac{\sqrt{3} T_s}{V_{dc}} v_\beta \\ T_0 = T_s - T_1 - T_2 \end{cases}$$

**过调制处理**：
- 线性区：$T_1 + T_2 \leq T_s$
- 过调制：按比例缩小作用时间

**电压利用率**：
$$V_{1,rms} = \frac{V_{dc}}{\sqrt{6}} \approx \frac{1.15 V_{dc}}{2\sqrt{2}}$$

### 载波波形分析

#### 三角载波

**双沿调制**：
$$v_c(t) = \frac{2}{T_s}(t \mod T_s) - 1$$

**谐波特性**：
- 载波频率倍数处的谐波
- 边带谐波：$f_{side} = mf_c \pm nf_{ref}$

#### 锯齿载波

**单沿调制**：
$$v_c(t) = \frac{t \mod T_s}{T_s}$$

**特点**：
- 非对称调制
- 偶次谐波增加
- 较少使用

### 谐波分析

#### 双极性SPWM谐波

**Bessel函数描述**：
$$v_{ao}(t) = \frac{V_{dc}}{2} m_a \sin(\omega_{ref} t) + \text{谐波项}$$

**主要谐波频率**：
- $f_c \pm 2f_{ref}$
- $2f_c \pm f_{ref}$
- $2f_c \pm 3f_{ref}$

**THD近似**：
$$THD \approx \sqrt{\sum_{n=2}^{\infty} \left(\frac{V_n}{V_1}\right)^2} \approx 80\% \text{（单相）}$$

#### SVPWM谐波优势

相比SPWM：
- 相同开关频率下THD降低约30%
- 电压利用率提高15%
- 更适合电机驱动

### 双重傅里叶级数建模

对于SPWM调制的输出电压，可利用双重傅里叶级数（Double Fourier Series）建立解析谐波模型，这是分析并联变流器载波相位差效应的核心理论工具。

**三角载波表达式**（Hu等2016）：
$$u_{car} = \begin{cases} \frac{2\omega_c t}{\pi} - 2k\pi, & -\frac{\pi}{2} + 2k\pi < \omega_c t + \theta_c < \frac{\pi}{2} + 2k\pi \\ -\frac{2\omega_c t}{\pi} + 2k\pi, & \frac{\pi}{2} + 2k\pi < \omega_c t + \theta_c < \frac{3\pi}{2} + 2k\pi \end{cases}$$

其中$\omega_c$为载波角频率，$\theta_c$为载波初始相位。

**SPWM输出电压双重傅里叶展开**：
$$u_o(t) = \sum_{n=1}^{\infty} [A_{0n}\cos(ny) + B_{0n}\sin(ny)] + \sum_{m=1}^{\infty} [A_{m0}\cos(mx) + B_{m0}\sin(mx)] + \sum_{m=1}^{\infty} \sum_{n=\pm 1}^{\pm\infty} [A_{mn}\cos(mx+ny) + B_{mn}\sin(mx+ny)]$$

其中$x = \omega_c t + \theta_c$，$y = \omega_r t + \theta_r$。

**谐波系数计算**：
$$A_{mn} + jB_{mn} = \frac{1}{2\pi^2} \int_{0}^{2\pi} \int_{0}^{2\pi} u_p(t) e^{j(m\omega_c t + n\omega_r t)} d(\omega_c t) d(\omega_r t)$$

该模型揭示了载波相位差$\theta_c$如何影响各次谐波的幅值与相位分布，从而为并联变流器的环流分析提供理论依据。

## EMT建模方法

### 五模型精度-效率映射

根据Sano等(2021)的研究，并网逆变器PWM调制器在EMT仿真中有五种建模精度层次，从最高精度到最低复杂度：

| 模型类型 | 精度 | 计算效率 | 步长要求 | 适用场景 |
|----------|------|----------|----------|----------|
| SW（开关模型） | 最高 | 最低 | ≤2 μs | 谐波分析、故障穿越、直流短路 |
| VI（电压插值模型） | 高 | 较高 | ≤10 μs | 开关频率谐波、死区效应 |
| HP-AVM（保留谐波平均值模型） | 较高 | 中等 | ≤100 μs | 系统级谐波动态 |
| AV（平均值模型） | 中 | 高 | ≤100 μs | 基波动态、控制验证 |
| CCI/SCI（电流注入模型） | 低 | 最高 | ≤600 μs | 大系统功率注入 |

#### 全开关模型（SW）

完整模拟每个IGBT/二极管的开通关断过程，保留所有开关瞬态。

**开关状态判定**：
$$S(t) = \begin{cases} 1, & v_{ref}(t) > v_c(t) \\ 0, & v_{ref}(t) < v_c(t) \end{cases}$$

**步长约束**：开关频率2-20 kHz时，步长需≤1 μs（Nyquist约束的2-10倍）。

#### 电压插值模型（VI）

Horiuchi等(2020)提出，通过解析公式计算插值系数，在较大步长下保留开关谐波。

**精确开关时刻计算**：
$$t_{fall} = t_{n-1} + \frac{v^* - v_{carrier}}{h}$$
$$t_{rise} = t_n - \frac{v^* - v_{carrier}}{h}$$

**插值系数**：
$$s = \frac{1}{2} + \frac{v^* - v_{carrier}}{h T_s}$$

该方法将时间平均法（TAM）从后向欧拉积分推广至梯形积分法，兼容主流离线EMT程序。

**量化性能**：步长扩大5倍，计算时间降低至1/3，3次、5次、7次及开关频率谐波幅值误差<1%。

#### 保留谐波的平均值模型（HP-AVM）

Cao等(2026)提出"平均值+谐波"统一框架，用占空比$D$替代开关状态$S$推进平均状态，同时同步计算开关谐波分量。

**HP-AVM核心状态方程**：
$$\dot{\bar{x}} = \bar{A} \cdot \bar{x} + \bar{b}, \quad \bar{A} = A_0 + \sum_{k=1}^N (D_{up,k} \cdot A_{up,k} + D_{low,k} \cdot A_{low,k})$$

**梯形积分离散化**：
$$x^n = \left(I - \frac{h}{2}A\right)^{-1} \cdot \left[ \left(I + \frac{h}{2}A\right) \cdot x^{n-1} + \frac{h}{2}(b^n + b^{n-1}) \right]$$

**半载波周期（HCP）占空比预测**：将状态矩阵求逆频率降至每半载波周期一次，计算速度提升5-6倍，基波动态误差<0.5%，关键开关谐波幅值误差<1.5%。

#### 平均值模型（AV）

忽略开关细节，直接使用占空比：

$$\bar{v}_{ao} = d \cdot V_{dc} = m_a V_{dc} \sin(\omega t)$$

**适用场景**：系统级仿真、控制策略验证、稳态分析。

**限制**：无法分析开关谐波、无法分析开关损耗。

#### 电流注入模型（CCI/SCI）

将变流器交流侧等效为受控电流源，输入为有功/无功参考或电流参考。

**限制**：弱化滤波器细节、无法反映电网谐波交互、控制失稳和部分故障暂态。

### 死区效应建模

死区时间是防止桥臂上下开关器件同时导通而设置的安全时间，其导致的输出电压损失和波形畸变是PWM调制器EMT建模必须处理的关键问题。

**等效输出电压损失**（据工程经验）：
$$V_{loss} \approx \frac{2T_{dead}}{T_s} \cdot V_{dc}$$

**死区补偿方法**：

| 补偿方法 | 原理 | 精度 | 计算代价 |
|----------|------|------|----------|
| 电压损失预估补偿 | 根据死区时间和电流极性预估电压损失 | 中 | 低 |
| 占空比修正补偿 | 在占空比计算中加入死区补偿项 | 高 | 中 |
| 重复预测补偿 | 利用前一周期信息预测死区效应 | 高 | 高 |

### 并联变流器的载波相位差效应

当多台变流器并联运行时，载波相位差对谐波环流有显著影响（Hu等2016）。

**物理机理**：不同变流器的载波相位差导致PWM输出电压谐波的相位差异，在并联连接点形成谐波电压差，驱动谐波环流在变流器之间闭合。

**环流抑制策略量化数据**（Hu等2016）：

| 策略 | 环流抑制效果 | 代价 |
|------|-------------|------|
| 增加连接电抗（3mH→5mH） | 环流幅值下降30%-40% | 无功容量占用增加 |
| 提高开关频率（6.4kHz→12.8kHz） | 环流幅值下降50% | 开关损耗增加约100% |
| 差异化开关频率（6.45/6.35kHz） | 中等抑制 | 控制器一致性变差 |
| PLL载波相位同步 | 接近0环流 | 依赖理想通信条件 |

## 仿真软件实现

### PSCAD/EMTDC实现

**SPWM调制器**：
```fortran
! SPWM调制器
REAL Vref(3)      ! 三相参考电压
REAL Vc           ! 载波幅值
REAL Vtri         ! 三角波值
INTEGER S(3)      ! 开关信号

! 生成三角载波（0到1）
Vtri = ABS(2.0 * MOD(TIME * FC, 1.0) - 1.0)

! 比较产生开关信号
DO I = 1, 3
  IF (Vref(I) > Vtri) THEN
    S(I) = 1
  ELSE
    S(I) = 0
  END IF
END DO
```

**SVPWM实现**：
```fortran
! SVPWM调制
REAL Valpha, Vbeta    ! 参考电压在alpha-beta坐标系
REAL Vdc              ! 直流电压
REAL Ts               ! 开关周期
REAL T1, T2, T0       ! 矢量作用时间
INTEGER Sector        ! 扇区

! Clarke变换
Valpha = (2.0*Va - Vb - Vc) / 3.0
Vbeta = (Vb - Vc) / SQRT(3.0)

! 扇区判断
Angle = ATAN2(Vbeta, Valpha)
Sector = INT((Angle + PI) / (PI/3.0)) + 1

! 计算作用时间
X = SQRT(3.0) * Ts * Vbeta / Vdc
Y = Ts * (1.5*Valpha - SQRT(3.0)*Vbeta/2.0) / Vdc
Z = Ts * (-1.5*Valpha - SQRT(3.0)*Vbeta/2.0) / Vdc

! 根据扇区选择
SELECT CASE (Sector)
  CASE (1)
    T1 = -Z
    T2 = X
  CASE (2)
    T1 = Z
    T2 = Y
  ...
END SELECT

! 过调制处理
IF (T1 + T2 > Ts) THEN
  T1 = T1 * Ts / (T1 + T2)
  T2 = T2 * Ts / (T1 + T2)
END IF

T0 = Ts - T1 - T2
```

### MATLAB/Simulink实现

**SPWM模块**：
```matlab
% PWM Generator
pwm = pwmGenerator('CarrierFreq', 10e3, 'SampleTime', 1e-6);

% 或自定义实现
fc = 10e3;  % 载波频率
fs = 100e6; % 仿真步长对应的采样率
t = 0:1/fs:0.1;

% 三角载波
carrier = sawtooth(2*pi*fc*t, 0.5);

% 参考波
Vref = 0.8 * sin(2*pi*50*t);

% PWM信号
pwm_signal = double(Vref > carrier);
```

**SVPWM实现**：
```matlab
function [Sa, Sb, Sc] = svpwm(Vabc, Vdc, Ts)
    % Clarke变换
    Valpha = (2*Vabc(1) - Vabc(2) - Vabc(3)) / 3;
    Vbeta = (Vabc(2) - Vabc(3)) / sqrt(3);
    
    % 扇区判断
    Vref1 = Vbeta;
    Vref2 = sqrt(3)/2 * Valpha - 0.5 * Vbeta;
    Vref3 = -sqrt(3)/2 * Valpha - 0.5 * Vbeta;
    
    N = sign(Vref1) + 2*sign(Vref2) + 4*sign(Vref3);
    sector = [0; 4; 6; 2; 3; 1; 5]; % 扇区映射
    Sector = sector(N + 4);
    
    % 计算作用时间
    X = sqrt(3) * Ts * Vbeta / Vdc;
    Y = Ts * (1.5*Valpha + sqrt(3)*Vbeta/2) / Vdc;
    Z = Ts * (-1.5*Valpha + sqrt(3)*Vbeta/2) / Vdc;
    
    % 根据扇区选择
    switch Sector
        case 1, T1 = -Z; T2 = X;
        case 2, T1 = Z; T2 = Y;
        case 3, T1 = X; T2 = -Y;
        case 4, T1 = -X; T2 = Z;
        case 5, T1 = -Y; T2 = -Z;
        case 6, T1 = Y; T2 = -X;
    end
    
    % 过调制处理
    if T1 + T2 > Ts
        T1 = T1 * Ts / (T1 + T2);
        T2 = T2 * Ts / (T1 + T2);
    end
    T0 = Ts - T1 - T2;
    
    % 生成开关信号（七段式）
    Ta = (T1 + T2 + T0/2) / Ts;
    Tb = (T2 + T0/2) / Ts;
    Tc = T0/2 / Ts;
    
    % 输出开关状态
    Sa = Ta > 0.5;
    Sb = Tb > 0.5;
    Sc = Tc > 0.5;
end
```

### FPGA实现

**Verilog代码**：
```verilog
module pwm_generator (
    input clk,
    input reset,
    input [15:0] duty,      // 占空比 (0-65535)
    input [15:0] period,    // 周期计数
    output reg pwm_out
);

    reg [15:0] counter;
    
    always @(posedge clk or posedge reset) begin
        if (reset) begin
            counter <= 16'd0;
            pwm_out <= 1'b0;
        end else begin
            if (counter >= period - 1)
                counter <= 16'd0;
            else
                counter <= counter + 1'b1;
                
            pwm_out <= (counter < duty);
        end
    end
endmodule
```

## 量化性能边界

PWM调制器的EMT仿真精度取决于载波频率、调制策略和离散化方法。以下汇总可引用的量化数据：

**全开关模型（Detailed Switching Model）**：保留每个IGBT/二极管的开通关断过程，时间步长需≤1 μs。开关频率2-20 kHz时，以401电平MMC为例，每个桥臂含200个子模块，全开关模型每秒需模拟401×6×f_sw ≈ 4.8-48亿次开关事件。

**电压插值模型（VI）**（Horiuchi等2020）：步长扩大5倍，计算时间降低至原来的1/3，3次、5次、7次及开关频率谐波幅值误差<1%。

**保留谐波平均值模型（HP-AVM）**（Cao等2026）：计算速度提升5-6倍，基波动态误差<0.5%，关键开关谐波幅值误差<1.5%，状态矩阵求逆频率降至每半载波周期1次。

**SVPWM性能量化**：相比SPWM电压利用率提高15%（从0.5Vdc提升至0.577Vdc基波幅值），THD降低约30%（SPWM ~80% → SVPWM ~55-60%），扇区判断和矢量时间计算需12-20次三角函数运算。

**MMC载波移相（CPS-PWM）**：对N个子模块，载波相位差360°/N导致等效开关频率提升N倍。N=200时，等效开关频率为200×f_c，THD从单载波的~50%降至<5%。

## 关键技术挑战

### 开关时刻量化误差

固定步长仿真中，开关时刻被强制到时间网格上，导致：
- 占空比估计偏差（实际占空比与网格量化值的差异）
- 开关谐波相位偏移
- 误差随步长增大而加剧

**解决方向**：插值校正（Horiuchi VI方法）、变步长检测

### 多电平变流器的计算爆炸

MMC等模块化变流器含大量子模块，每个子模块独立跟踪开关状态导致：
- 全开关模型计算量与子模块数量成正比
- 状态矩阵维度爆炸
- 实时仿真步长约束趋严

**解决方向**：NLM等效（Cao 2026）、调制指数等效（Bahirat 2015）

### 死区效应的精确捕捉

死区时间导致的：
- 输出电压基波损失（与死区时间和电流极性相关）
- 波形畸变和额外谐波
- 在低电流区间尤为显著

**解决方向**：死区补偿算法、基于电流极性的修正占空比

### 并联变流器的载波同步问题

多台变流器并联时：
- 载波相位差导致非特征谐波环流
- 720Hz附近环流可在系统侧不表现、仅在设备侧显现
- 级联H桥拓扑因直流电压不平衡更易受影响

**解决方向**：载波相位同步、差异化开关频率、高通滤波

## 适用边界与选择指南

### 调制策略选择

| 应用场景 | 推荐调制策略 | 原因 |
|----------|-------------|------|
| 大功率变流器 | SHEPWM | 消除特定低次谐波，降低开关损耗 |
| 电机驱动 | SVPWM | 高电压利用率，低THD |
| 并网逆变器 | SPWM/THIPWM | 实现简单，谐波可控 |
| 高效率应用 | DPWM | 减少1/3开关损耗 |

### EMT建模粒度选择

| 研究问题 | 推荐模型 | 步长 |
|----------|----------|------|
| 开关频率谐波分析 | SW或VI | ≤10 μs |
| 谐波稳定性分析 | VI或HP-AVM | ≤100 μs |
| 基波动态研究 | AV | ≤100 μs |
| 大系统功率注入 | CCI/SCI | ≤600 μs |

### 模型限制

- **死区效应**：未建模死区引起的电压损失（需单独处理）
- **器件非线性**：未考虑开关瞬态和器件寄生参数
- **最小脉宽**：极短脉宽可能无法实现
- **共模电压**：PWM产生高频共模电压
