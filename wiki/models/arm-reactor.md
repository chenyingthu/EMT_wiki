# 桥臂电抗器 (Arm Reactor)

## 概述

桥臂电抗器是模块化多电平换流器([[mmc-model]])的关键元件，串联连接在每个桥臂中，主要作用是限制环流、抑制桥臂间的不平衡电流、在直流故障时限制故障电流上升率，并参与换流器的动态电压分配。作为MMC换流阀([[design-of-hybrid-series-converter-valve-considering-device-switching-characteris]])的核心组成部分，桥臂电抗器的设计直接影响换流器的运行性能、可靠性和经济性。

桥臂电抗器在MMC中的独特之处在于需要同时承受交流分量和直流偏置电流，其工作条件比传统交流电抗器更为复杂。电抗器的设计需要综合考虑电磁性能、热性能、机械强度和绝缘水平等多个方面。

## 基本结构

### 结构形式

#### 干式空心电抗器
干式空心电抗器是MMC桥臂电抗器最常见的形式，具有以下特点：

- **结构特点**：采用多层并联绕组结构，由铝或铜导线绕制而成，层间设置绝缘垫块保证散热通道
- **无铁心设计**：消除了铁心饱和和非线性特性，电感值线性度好
- **散热方式**：自然冷却或强制风冷，绕组温升需满足IEC标准
- **电磁特性**：磁场向外扩散，需要考虑对周围设备的电磁干扰
- **维护需求**：免维护或低维护，适合长期运行
- **典型应用**：陆上柔直换流站、容量中等(50-500MVA)的MMC工程

干式空心电抗器的电感值计算公式：
$$L = \frac{\mu_0 N^2 h}{2\pi} \ln\frac{D_o}{D_i}$$

其中：
- $\mu_0$: 真空磁导率 ($4\pi \times 10^{-7}$ H/m)
- $N$: 绕组匝数
- $h$: 绕组高度
- $D_o$: 绕组外径
- $D_i$: 绕组内径

#### 油浸式电抗器
大容量MMC工程(>1000MVA)常采用油浸式电抗器：

- **冷却优势**：变压器油提供优异的散热性能，允许更高的电流密度
- **绝缘增强**：油浸绝缘系统提供更高的绝缘强度，适用于高电压等级
- **结构紧凑**：有铁心设计使体积比同容量空心电抗器小30-40%
- **磁屏蔽**：铁心结构本身具有磁屏蔽作用，减少外部电磁干扰
- **维护要求**：需要定期油色谱分析和绝缘监测
- **应用场景**：大容量海上柔直送出、城市中心地下换流站

油浸式电抗器的等效电路参数：
$$L_{eq} = L_{air} + \frac{L_{iron}}{1 + \frac{L_{iron}}{L_{gap}}}$$

其中气隙长度设计为：
$$l_g = \frac{\mu_0 N I_{max}}{B_{max}}$$

#### 磁屏蔽式电抗器
为减少电磁干扰而设计的特殊结构：

- **磁屏蔽设计**：在电抗器外部设置磁屏蔽壳体，引导磁场闭合路径
- **屏蔽材料**：采用硅钢片或软磁复合材料，降低屏蔽损耗
- **结构形式**：可集成于干式或油浸式电抗器中
- **电磁兼容**：满足IEC 61000系列EMC标准
- **损耗权衡**：屏蔽结构会引入额外损耗，需优化设计

磁屏蔽 effectiveness 计算：
$$SE = 20\log_{10}\left(\frac{H_{without}}{H_{with}}\right)$$

典型屏蔽效能要求：SE > 40 dB

### 设计参数

| 参数 | 典型值 | 说明 |
|------|--------|------|
| 电感值 | 30-100 mH | 根据容量选择，大容量取小值 |
| 额定电流 | 500-3000 A | 桥臂电流额定值，含直流偏置 |
| 短时电流 | 2-3 pu | 故障电流耐受，持续时间2s |
| 直流电阻 | <10 mΩ | 降低稳态损耗 |
| 温升限值 | 65-80 K | 取决于绝缘等级 |
| 绝缘水平 | 50-150 kV | 由换流器电压等级决定 |

## 数学模型

### 基本方程

桥臂电抗器电压-电流关系由电磁感应定律描述：
$$v_L = L\frac{di}{dt} + Ri$$

其中：
- $L$: 电感值 (H)
- $R$: 绕组直流电阻 ($\Omega$)
- $i$: 桥臂电流 (A)
- $v_L$: 电抗器电压降 (V)

对于交直流混合电流，桥臂电流可表示为：
$$i_{arm}(t) = I_{dc}/3 + \frac{I_{ac}}{2}\sin(\omega t) + \sum_{h}I_h\sin(h\omega t + \phi_h)$$

其中：
- $I_{dc}$: 直流母线电流
- $I_{ac}$: 交流侧电流峰值
- $I_h$: 第h次谐波电流幅值

电抗器感应电压：
$$v_L(t) = L\frac{d}{dt}\left[\frac{I_{ac}}{2}\sin(\omega t) + \sum_{h}I_h\sin(h\omega t + \phi_h)\right] + Ri_{arm}(t)$$

展开得：
$$v_L(t) = \frac{\omega L I_{ac}}{2}\cos(\omega t) + \sum_{h}h\omega L I_h\cos(h\omega t + \phi_h) + Ri_{arm}(t)$$

### 二阶效应

#### 磁饱和效应
对于油浸式电抗器，大电流时铁心可能饱和：

饱和电感模型：
$$L(i) = L_0 \cdot \left[1 + \left(\frac{|i|}{I_{sat}}\right)^n\right]^{-1}$$

其中：
- $L_0$: 未饱和电感值
- $I_{sat}$: 饱和电流阈值
- $n$: 饱和系数(通常n=3-5)

磁化曲线采用双曲正切函数拟合：
$$B(H) = B_{sat}\tanh\left(\frac{\mu_0 \mu_r H}{B_{sat}}\right)$$

- [[magnetic-saturation-modeling]] - 磁饱和详细分析

#### 涡流损耗
高频分量在导体中产生涡流损耗：
$$P_{eddy} = \frac{\pi^2 f^2 B_{max}^2 d^2}{6\rho} V$$

其中：
- $f$: 频率 (Hz)
- $d$: 导体厚度 (m)
- $\rho$: 电阻率 ($\Omega\cdot m$)
- $V$: 导体体积 ($m^3$)

#### 集肤效应
高频时电流分布不均匀，等效交流电阻增加：
$$R_{ac}(f) = R_{dc} \cdot \frac{\xi}{2} \cdot \frac{\sinh\xi + \sin\xi}{\cosh\xi - \cos\xi}$$

其中：
$$\xi = d\sqrt{\frac{\pi f \mu}{\rho}}$$

集肤深度：
$$\delta = \sqrt{\frac{\rho}{\pi f \mu}}$$

#### 邻近效应
多层绕组中相邻导体的电磁耦合导致附加损耗：
$$P_{prox} = P_{eddy} \cdot k_{prox}(N, D, f)$$

邻近效应系数与绕组层数N、线径D和频率f相关。

## 功能作用

### 环流抑制原理

#### 环流产生机理
MMC内部存在二倍频环流([[analysis-on-non-characteristic-harmonic-circulating-current-in-parallel-inverter]])，主要由以下因素引起：

1. **电压不平衡**：上下桥臂子模块投入数量瞬时差异
2. **参数不对称**：桥臂电抗器参数不一致
3. **控制延迟**：调制信号与实际投入的时延

环流数学表达式：
$$i_{circ,j} = \frac{I_{dc}}{3} + i_{2nd,j}$$

其中二倍频环流分量：
$$i_{2nd,j} = I_{2nd}\cos(2\omega t + \phi_j), \quad j \in \{a, b, c\}$$

三相环流之和为零：
$$i_{circ,a} + i_{circ,b} + i_{circ,c} = I_{dc}$$

#### 环流阻抗特性
桥臂电抗器提供环流阻抗：
$$Z_{circ} = 2\omega L$$

对于第h次谐波环流：
$$Z_{circ}(h) = 2h\omega L$$

环流幅值与电感值的关系：
$$I_{2nd} \approx \frac{V_{2nd}}{2\omega L}$$

其中$V_{2nd}$为二倍频电压分量，约等于$0.1-0.2$ pu。

#### 环流抑制控制(`circulating-current-suppression`)

基于电抗器的环流抑制控制策略：

1. **无源抑制**：仅依靠桥臂电抗器的阻抗限制环流
   - 电感值越大，抑制效果越好
   - 但会增加成本和体积

2. **有源抑制**：配合控制系统主动抑制环流
   环流抑制控制电压：
   $$v_{cir,j} = -2\omega L \cdot i_{2nd,j}$$

3. **混合抑制**：无源+有源相结合
   总环流抑制效果：
   $$\eta = \frac{I_{2nd,without}}{I_{2nd,with}} = \frac{\sqrt{(2\omega L)^2 + K_c^2}}{2\omega L}$$

其中$K_c$为控制器增益。

### 故障电流限制机制

#### 直流故障特性
直流故障时，MMC通过桥臂电抗器限制故障电流上升率：
$$\frac{di}{dt} = \frac{V_{dc} - V_{fault}}{2L}$$

最严重工况(端部短路)：
$$\frac{di}{dt}_{max} = \frac{V_{dc}}{2L}$$

#### 故障电流计算
故障电流峰值估算：
$$I_{peak} = \frac{V_{dc}}{2L} t_{block} + I_{initial}$$

其中：
- $t_{block}$: 保护闭锁时间(通常1-5 ms)
- $I_{initial}$: 故障前初始电流

#### 保护配合设计
电抗器参数与保护系统的配合：

1. **子模块保护**：限制故障电流峰值，避免IGBT过流损坏
   $$I_{peak} < I_{IGBT,max}$$

2. **故障穿越**：为保护动作争取时间
   $$t_{available} = \frac{L(I_{limit} - I_{initial})}{V_{dc}/2}$$

3. **限流效果评估**：
   限流率：
   $$\eta_{limit} = \frac{I_{peak,withL} - I_{peak,withoutL}}{I_{peak,withoutL}} \times 100\%$$

- `dc-fault-protection` - 直流故障保护详细设计

### 动态均压原理

#### 电压分配机制
桥臂电抗器在子模块投切过程中参与动态电压分配：

子模块投切瞬态过程：
$$v_{SM} = V_{cap} - L\frac{di}{dt} - Ri$$

电压分配系数：
$$k_v = \frac{L_{arm}}{L_{arm} + L_{stray}}$$

其中$L_{stray}$为杂散电感。

#### 电流均衡作用
抑制桥臂间电流不均衡：
$$\Delta i = i_p - i_n$$
$$\frac{d(\Delta i)}{dt} = \frac{V_{dc} - 2v_{arm}}{2L}$$

电抗器提供电流均衡阻抗：
$$Z_{balance} = 2\omega L$$

#### 开关过电压限制
限制IGBT关断过电压：
$$V_{ce,max} = V_{dc} + L_{stray}\frac{di_{c}}{dt} < V_{ce,rating}$$

桥臂电抗器减小di/dt，降低过电压幅值。

## 在MMC中的配置

### 每相结构

每相上下两个桥臂对称配置：
```
        上桥臂电抗器 L_p
DC+ ----[L]----+---- AC相输出
               |
        下桥臂电抗器 L_n
DC- ----[L]----+----
```

桥臂等效电路：
$$v_{arm} = v_{SM,sum} + L\frac{di_{arm}}{dt} + Ri_{arm}$$

上下桥臂电压关系：
$$v_{p} + v_{n} = V_{dc}$$

### 参数设计方法

#### 电感值选择约束

电感值设计需要满足多重约束：

1. **环流抑制约束**：
   $$L \geq \frac{V_{2nd,max}}{2\omega I_{2nd,limit}}$$

2. **故障限流约束**：
   $$L \geq \frac{V_{dc} \cdot t_{block}}{2(I_{limit} - I_{initial})}$$

3. **动态响应约束**：
   $$L \leq \frac{\Delta t_{max} \cdot V_{dc}}{4\Delta I_{max}}$$

4. **损耗约束**：
   $$L \leq \frac{P_{loss,limit} - P_{base}}{\omega I_{2nd}^2}$$

#### 优化设计流程

电抗器参数优化目标函数：
$$\min f(L) = w_1 \cdot C_{cost}(L) + w_2 \cdot P_{loss}(L) + w_3 \cdot V_{volume}(L)$$

约束条件：
$$\text{s.t.} \quad L_{min} \leq L \leq L_{max}$$

其中权重系数满足：$w_1 + w_2 + w_3 = 1$

#### 典型取值公式

工程经验公式：
$$L = (0.1 \sim 0.3) \frac{V_{dc}}{\omega I_{N}}$$

或基于换流器容量：
$$L = k_L \cdot \frac{S_{N}}{3\omega I_{N}^2}$$

其中$k_L$为设计系数，典型值0.05-0.15。

### 参数不对称影响

#### 三相不对称
三相电抗器参数不一致导致：
$$\Delta L_{max} = \max|L_a - L_b|, |L_b - L_c|, |L_c - L_a|$$

环流不平衡度：
$$\epsilon_{circ} = \frac{\Delta L_{max}}{\bar{L}} \times 100\%$$

通常要求：$\epsilon_{circ} < 5\%$

#### 上下桥臂不对称
上下桥臂电感差引起零序环流：
$$i_0 = \frac{\Delta L}{2L} \cdot I_{dc}$$

设计要求：$|L_p - L_n| < 2\% \cdot L_{nominal}$

## EMT仿真建模

### 集中参数模型

#### 离散化方法
采用[[trapezoidal-rule]]进行离散：
$$i_{n+1} = \frac{v_{n+1} + v_n}{R_{eq}} + I_{history}$$

其中等值电阻：
$$R_{eq} = \frac{2L}{\Delta t} + R$$

历史电流：
$$I_{history} = \frac{v_n}{R_{eq}} + \frac{2L/\Delta t - R}{2L/\Delta t + R}i_n$$

#### 伴随模型实现
[[companion-model]]电路：
```
        R_eq
    +---[===]---+----
    |           |
   +|-         +|
  v_{n+1}    I_history
   -|          -|
    |           |
    +-----------+----
```

电磁暂态仿真中的实现要点：
1. 每个时间步更新等值电阻
2. 历史电流项作为等值电流源
3. 与换流器其他元件形成节点导纳矩阵

### 饱和模型

#### 分段线性化方法
考虑大电流饱和的`nonlinear-inductor`：

电感值随电流变化：
$$L(i) = \begin{cases}
L_0 & |i| \leq I_{sat1} \\
L_0 \cdot \frac{I_{sat2} - |i|}{I_{sat2} - I_{sat1}} & I_{sat1} < |i| < I_{sat2} \\
L_{sat} & |i| \geq I_{sat2}
\end{cases}$$

#### 查表法实现
基于实测磁化曲线：

| 电流i (pu) | 电感L (pu) | 磁链ψ (Wb) |
|------------|-----------|-----------|
| 0.0 | 1.00 | 0.00 |
| 0.5 | 0.98 | 0.49 |
| 1.0 | 0.90 | 0.95 |
| 1.5 | 0.65 | 1.25 |
| 2.0 | 0.40 | 1.45 |

插值计算：
$$L(i) = L_k + \frac{L_{k+1} - L_k}{i_{k+1} - i_k}(i - i_k)$$

### 高频模型

#### 等效电路
考虑寄生参数的`high-frequency-model`：
```
    R_s    L_s
    +--[===]--[000]--+----
    |                |
    +---[  C_p  ]----+----
    |                |
    +------[R_p]-----+----
```

参数定义：
- $R_s$: 绕组高频等效电阻
- $L_s$: 等效电感
- $C_p$: 匝间寄生电容
- $R_p$: 铁心损耗等效电阻

#### 频率特性
阻抗频率响应：
$$Z(f) = \frac{(R_s + j2\pi f L_s) \cdot R_p}{R_s + R_p + j2\pi f L_s - (2\pi f)^2 L_s C_p R_p} \cdot \frac{1}{1 + j2\pi f C_p R_p}$$

自谐振频率：
$$f_{sr} = \frac{1}{2\pi\sqrt{L_s C_p}}$$

设计要求：$f_{sr} > 10f_{switching}$

## 损耗计算详细方法

### 损耗组成分析

总损耗分解：
$$P_{total} = P_{cu} + P_{core} + P_{stray} + P_{harm}$$

#### 铜损计算
直流铜损：
$$P_{cu,dc} = I_{dc}^2 \cdot R_{dc}$$

交流铜损(有效值法)：
$$P_{cu,ac} = I_{rms}^2 \cdot R_{ac} - I_{dc}^2 \cdot R_{dc}$$

等效电阻计算：
$$R_{ac} = R_{dc} \cdot k_{skin} \cdot k_{prox}$$

其中$k_{skin}$为集肤效应系数，$k_{prox}$为邻近效应系数。

#### 铁损计算
铁心损耗采用Steinmetz方程：
$$P_{core} = k_h f B_{max}^\alpha + k_e (f B_{max})^2 + k_a (f B_{max})^{1.5}$$

其中：
- $k_h$: 磁滞损耗系数
- $k_e$: 涡流损耗系数  
- $k_a$: 异常损耗系数
- $\alpha$: Steinmetz系数(通常1.6-2.2)

#### 杂散损耗
漏磁引起的结构件损耗：
$$P_{stray} = k_{stray} \cdot I_{rms}^2 \cdot f^{0.8}$$

### 谐波损耗分析

考虑环流谐波的`harmonic-loss`：
$$P_{harm} = \sum_{h} I_h^2 \cdot R_{ac}(h)$$

各次谐波贡献：
| 谐波次数h | 电流幅值I_h(%) | 频率(Hz) | 损耗贡献(%) |
|-----------|---------------|---------|------------|
| 1 | 100 | 50/60 | 60-70 |
| 2 | 5-15 | 100/120 | 15-25 |
| 3 | 2-5 | 150/180 | 3-8 |
| 4+ | <2 | >200 | 2-5 |

谐波损耗系数：
$$k_{harm} = \frac{\sum_{h} I_h^2 R_{ac}(h)}{I_{fund}^2 R_{ac}(1)}$$

### 热计算与温升

#### 热路模型
电抗器热路等效电路：
$$\theta_{rise} = P_{total} \cdot (R_{th,ins} + R_{th,surf})$$

其中：
- $R_{th,ins}$: 绝缘热阻
- $R_{th,surf}$: 表面对流热阻

#### 热点温度
绕组热点温度：
$$T_{hot} = T_{ambient} + \theta_{rise} \cdot k_{hot}$$

热点系数$k_{hot}$通常取1.1-1.3。

## 保护设计

### 过电压保护

#### 避雷器配置(`lightning-arrester`)
并联金属氧化物避雷器(MOA)配置：

保护水平：
$$V_{protection} = V_{ref} + L\frac{di}{dt}_{max}$$

其中$V_{ref}$为避雷器参考电压。

能量吸收能力：
$$E_{MOV} = \int_0^{t_{fault}} v_{MOV}(t) \cdot i_{fault}(t) dt$$

#### RC吸收回路
过电压缓冲设计：
$$v_{peak} = V_{dc} + I_{arm} \sqrt{\frac{L}{C_{snubber}}}$$

RC参数选择：
$$R_{snubber} = \sqrt{\frac{L}{C_{snubber}}}$$
$$\tau = R_{snubber}C_{snubber} > t_{fall}$$

### 过流保护

#### 热保护
基于绕组温度的热模型保护：
$$\theta(t) = \theta_{final}(1 - e^{-t/\tau_{th}}) + \theta_{initial}e^{-t/\tau_{th}}$$

热时间常数：
$$\tau_{th} = m_{copper} \cdot c_{copper} \cdot R_{th}$$

#### 电流保护
桥臂过流保护整定：
$$I_{pickup} = 1.2 \sim 1.5 \cdot I_{rated}$$
$$t_{delay} = f(I_{actual} / I_{pickup})$$

反时限特性：
$$t_{trip} = \frac{k}{(I/I_{pickup})^\alpha - 1}$$

#### 不平衡保护
桥臂间电流差保护：
$$\Delta I = |i_p - i_n|$$
$$\Delta I_{set} = 0.1 \sim 0.2 \cdot I_{rated}$$

三相不平衡保护：
$$I_{neg} = \frac{1}{3}|I_a + a^2 I_b + a I_c|$$
$$I_{zero} = \frac{1}{3}|I_a + I_b + I_c|$$

### 绝缘配合

#### 绝缘水平设计([[insulation-coordination]])
基本绝缘水平(BIL)：
$$BIL = k_{ins} \cdot V_{peak,normal}$$

其中$k_{ins}$为绝缘配合系数(通常2.0-2.5)。

开关冲击绝缘水平(BSL)：
$$BSL = 0.83 \cdot BIL$$

#### 局部放电监测
局放监测设置：
$$PD_{limit} = 100 \sim 500 pC$$

局放模式识别：
- 电晕放电：低幅值，均匀分布
- 表面放电：中幅值，相位集中
- 内部放电：高幅值，随机分布

## 工程应用案例

### 南网直流工程

**工程概况**：
- 额定容量：1000 MW
- 直流电压：±350 kV
- 桥臂电抗器：干式空心，每相2组

**电抗器参数**：
| 参数 | 数值 |
|------|------|
| 电感值 | 55 mH |
| 额定电流 | 1500 A |
| 短时电流 | 3000 A/2s |
| 直流电阻 | 4.2 mΩ |
| 绝缘水平 | 105 kV |

**运行特性**：
- 环流抑制效果：二倍频环流<8%额定电流
- 故障限流：直流故障电流上升率<1.5 kA/ms
- 温升：热点温度<105°C(额定工况)

### 张北柔直工程

**工程概况**：
- 世界首个四端柔直环网
- 额定容量：3000 MW
- 直流电压：±500 kV

**设计特点**：
1. **高海拔适应**：海拔>1000m，绝缘加强设计
2. **低温运行**：-40°C环境温度，材料低温选型
3. **大容量**：单桥臂电感值40mH，额定电流2000A

**创新技术**：
- 磁屏蔽优化：电磁干扰降低30dB
- 均流设计：三相不平衡度<2%
- 在线监测：温度、局放实时监测

### 海上风电柔直送出

**应用场景**：
- 海上换流平台空间受限
- 盐雾腐蚀环境
- 高可靠性要求

**设计优化**：
1. **紧凑结构**：油浸式设计，体积减少35%
2. **防腐处理**：C5-M防腐等级
3. **模块化**：便于海上平台安装维护

**性能指标**：
- 可用率：>99.5%
- 维护周期：5年以上
- 故障穿越：满足电网规程要求

### 城市中心地下换流站

**特殊要求**：
- 空间极度受限
- 电磁兼容严格
- 噪声限制

**解决方案**：
1. **高阻抗设计**：增大电感值至80mH，减少子模块数量
2. **主动降噪**：声屏障+消声器，噪声<55dB
3. **全屏蔽设计**：磁屏蔽+电磁屏蔽，满足城市EMC标准

## 试验标准与方法

### 型式试验
- **温升试验**：IEC 60076-6，额定工况至稳态温升
- **绝缘试验**：雷电冲击、操作冲击、工频耐压
- **短路承受能力**：2s短时电流，热稳定和动稳定验证

### 例行试验
- **电感测量**：额定电流和饱和电流下的电感值
- **电阻测量**：绕组直流电阻，计算75°C折算值
- **损耗测量**：负载损耗、空载损耗
- **局放测量**：施加1.5倍额定电压，测量局放量

### 特殊试验
- **声级测量**：声压级和声功率级
- **振动测量**：运行振动特性
- **频率响应分析**：高频阻抗特性

## 发展趋势

### 新材料应用
1. **高温超导**：零电阻、高电流密度
2. **非晶合金**：低铁损、高磁导率
3. **纳米晶材料**：高频低损耗

### 智能化发展
1. **在线监测**：温度、局放、振动实时监测
2. **状态评估**：基于多源数据的健康状态评估
3. **故障预警**：机器学习辅助的故障预测

### 紧凑化设计
1. **集成化**：与换流阀集成设计
2. **模块化**：标准化、模块化生产
3. **轻量化**：新材料、新结构减重
