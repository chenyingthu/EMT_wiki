# 子模块 (Submodule)

## 概述

子模块(Submodule, SM)是模块化多电平换流器(MMC)的基本构建单元，通过级联大量子模块实现高电压等级、高质量输出波形和模块化设计。每个子模块由储能电容、功率开关器件和控制电路组成，通过控制开关状态实现电容的投切，从而输出多级电平。子模块的设计直接影响MMC的电压等级、功率容量、输出波形质量和故障处理能力。

## 基本结构

### 典型子模块组成

```
         ┌── T1 (上开关)
    上端 ─┤
         └──┬── 电容 C ──┬── 下端
            │            │
           ─┴─           ─┴─
         ┌── T2 (下开关)
    下端 ─┘
```

**核心元件**：
| 元件 | 功能 | 典型参数 |
|-----|------|---------|
| IGBT | 功率开关 | 1.7kV/1kA - 6.5kV/750A |
| 反并联二极管 | 续流 | 与IGBT匹配 |
| 直流电容 | 储能 | 1-10 mF, 1-3 kV |
| 均压电阻 | 静态均压 | 10-100 kΩ |
| 旁路开关 | 故障处理 | 晶闸管或IGBT |

### 开关状态

**半桥子模块**：
| T1 | T2 | 输出电压 | 状态 | 电流路径 |
|----|----|---------|------|---------|
| 导通 | 关断 | $V_C$ | 投入 | 经T1充电/放电 |
| 关断 | 导通 | 0 | 切除 | 经T2旁路 |
| 关断 | 关断 | - | 闭锁 | 二极管续流 |

**禁止状态**：T1和T2同时导通（直通短路）

## 子模块类型

### 半桥子模块(HBSM)

**电路结构**：
```
          T1      D1
        ┌─/\/\─┬──►|───┐
        │      │      │
SM+ ────┤      C      ├─── SM-
        │      │      │
        └─/\/\─┴──|◄───┘
          T2     D2
```

**输出电平**：
- 投入状态：$V_{SM} = V_C$
- 切除状态：$V_{SM} = 0$

**特点**：
- 结构简单，成本低
- 效率高（2个器件导通）
- 不能阻断直流故障
- 应用最广泛

**适用场景**：
- VSC-HVDC
- STATCOM
- 电机驱动

### 全桥子模块(FBSM)

**电路结构**：
```
        T1   D1      T3   D3
      ┌/\/\┬►|┐    ┌|◄┬/\/\┐
      │    │  │    │  │    │
SM+ ──┤    C  ├────┤  C   ├── SM-
      │    │  │    │  │    │
      └/\/\┴|◄┘    └►|┴/\/\┘
        T2   D2      T4   D4
```

**输出电平**：
- 正投入：$V_{SM} = +V_C$
- 切除：$V_{SM} = 0$
- 负投入：$V_{SM} = -V_C$

**特点**：
- 可输出负电平
- 直流故障自清除能力
- 成本高（4个器件）
- 损耗较大

**开关状态**：
| T1 | T2 | T3 | T4 | 输出电压 | 状态 |
|----|----|----|----|---------|------|
| 导 | 关 | 关 | 导 | +Vc | 正投入 |
| 关 | 导 | 导 | 关 | -Vc | 负投入 |
| 关 | 导 | 关 | 导 | 0 | 正极旁路 |
| 导 | 关 | 导 | 关 | 0 | 负极旁路 |

### 钳位双子模块(CDSM)

**电路结构**：
- 两个半桥串联
- 公共钳位二极管

**输出电平**：
- 0, $V_C$, $2V_C$

**特点**：
- 器件数量适中
- 直流故障阻断能力
- 适合直流电网应用

### 其他拓扑

**交叉连接子模块**：
- 双向功率流动
- 高效率

**混合拓扑**：
- 半桥+全桥混合
- 成本与性能平衡

**级联拓扑**：
- 多级串联
- 更高电压等级

## 数学模型

### 开关函数模型

**半桥子模块**：
$$V_{SM} = S \cdot V_C$$
$$I_C = S \cdot I_{SM}$$

其中开关函数$S \in \{0, 1\}$。

**全桥子模块**：
$$V_{SM} = S \cdot V_C, \quad S \in \{-1, 0, 1\}$$

### 电容动态方程

**电容电压变化**：
$$\frac{dV_C}{dt} = \frac{1}{C}I_C = \frac{S \cdot I_{SM}}{C}$$

**能量方程**：
$$E = \frac{1}{2}CV_C^2$$
$$\frac{dE}{dt} = V_C \cdot I_C$$

### 电容电压纹波

**纹波电压**：
$$\Delta V_C = \frac{1}{C}\int_{t_0}^{t_0+T}I_C dt$$

**工频纹波**（MMC整流/逆变）：
$$\Delta V_C \approx \frac{I_{arm}}{2\omega C}$$

**纹波抑制**：
- 增大电容
- 提高开关频率
- 并联均压控制

## 控制策略

### 电容电压平衡控制

**目标**：
- 所有子模块电压相等
- 电压波动在允许范围内

**方法**：

**1. 排序法(CPS)**：
- 实时检测各SM电压
- 按电压排序
- 充电时投入电压低的SM
- 放电时投入电压高的SM

**算法流程**：
```
1. 测量所有SM电压
2. 按电压排序
3. 根据桥臂电流方向：
   - I > 0（充电）：投入电压最低的N个SM
   - I < 0（放电）：投入电压最高的N个SM
4. 输出开关信号
```

**2. 轮换法**：
- 周期性轮换投入顺序
- 均衡开关损耗

**3. 预测控制**：
- 预测未来电压变化
- 优化投切策略

### 调制策略

**载波移相PWM(CPS-PWM)**：
- 每个SM独立调制
- 载波相位错开
- 等效开关频率提高

**最近电平调制(NLM)**：
- 选择最接近的电压电平
- 低开关频率
- 高效率

**特定谐波消除(SHE-PWM)**：
- 优化开关角
- 消除特定谐波

## 保护设计

### 过电压保护

**触发条件**：
$$V_C > V_{C,max}$$

**保护措施**：
- 快速切除SM
- 能量回馈
- 过压吸收

### 过电流保护

**检测**：
- 桥臂电流
- IGBT电流

**动作**：
- 快速闭锁
- 旁路开关动作

### 故障处理

#### 短路故障

**故障电流路径**：
- 电容放电
- 大电流冲击

**保护策略**：
- 快速检测（<10μs）
- 立即闭锁
- 旁路开关动作

#### 开路故障

**影响**：
- 电平数减少
- 输出波形畸变

**容错运行**：
- 冗余设计
- 剩余SM重新分配
- 降额运行

#### 电容故障

**类型**：
- 电容短路
- 电容开路
- 容量衰减

**诊断**：
- 电压异常检测
- 谐波分析

### 旁路系统

**机械旁路**：
- 接触器
- 适用于慢速保护

**固态旁路**：
- 晶闸管
- 快速响应(<1ms)

**混合旁路**：
- 固态快速动作
- 机械长期承载

## 损耗分析

### IGBT损耗

**导通损耗**：
$$P_{cond} = I_{rms}^2R_{on} + V_{ce0}I_{avg}$$

**开关损耗**：
$$P_{sw} = f_{sw}(E_{on} + E_{off})\frac{I}{I_{rated}}$$

### 子模块总损耗

**半桥**：
$$P_{SM} = 2P_{cond} + P_{sw}$$

**全桥**：
$$P_{SM} = 4P_{cond} + 2P_{sw}$$

### 效率对比

| 类型 | 器件数 | 导通损耗 | 开关损耗 | 总损耗 |
|-----|-------|---------|---------|-------|
| 半桥 | 2 | 低 | 低 | 最低 |
| 全桥 | 4 | 高 | 高 | 高 |
| CDSM | 3 | 中 | 中 | 中等 |

## 热设计

### 散热要求

**热阻**：
$$T_j = T_a + P_{loss}(R_{th,jc} + R_{th,ch} + R_{th,ha})$$

**冷却方式**：
- 风冷
- 水冷
- 热管

### 热管理

**温度监测**：
- NTC热敏电阻
- 光纤测温

**降额运行**：
- 高温限制功率
- 保护器件安全

## 应用参数

### 典型MMC参数

| 参数 | 低压 | 中压 | 高压 | 特高压 |
|-----|------|------|------|-------|
| 直流电压 | 10kV | 50kV | 200kV | 800kV |
| SM数量/臂 | 10 | 50 | 200 | 400 |
| SM电压 | 1kV | 1kV | 1kV | 2kV |
| 电容容量 | 2mF | 4mF | 10mF | 15mF |
| 功率等级 | 5MW | 50MW | 500MW | 2000MW |

### 选型指南

**电压选择**：
$$V_{C,rated} = \frac{V_{dc}}{N} \times k_{margin}$$

$k_{margin}$ = 1.1-1.2（电压裕度）

**电容选择**：
$$C = \frac{I_{arm}}{2\omega \Delta V_{C,max}}$$
