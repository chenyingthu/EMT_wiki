# 无功补偿装置 (Reactive Compensation Device)

## 概述

无功补偿装置(Reactive Compensation Device)是电力系统中用于动态调节无功功率、维持电压稳定的关键设备。随着电力系统规模扩大和新能源大量接入，电压控制问题日益突出，无功补偿装置的作用愈发重要。从传统的并联电容器、电抗器到现代的静止无功补偿器(SVC)、静止同步补偿器(STATCOM)，无功补偿技术经历了从机械式到电力电子式的革命性发展。现代无功补偿装置响应速度快、控制精度高，能够实现无功的连续调节，是保障电网电压质量和稳定运行的重要技术手段。

## 无功补偿原理

### 无功功率特性

**定义**:
$$Q = \sqrt{S^2 - P^2} = VI\sin\varphi$$

**功率因数**:
$$\cos\varphi = \frac{P}{S}$$

**无功流动**:
- 无功功率不能远距离传输
- 就地补偿最为经济
- 电压与无功密切相关

### 电压调节原理

**电压降落**:
$$\Delta V \approx \frac{PR + QX}{V}$$

**无功-电压关系**:
$$\frac{dV}{dQ} < 0$$
注入无功可提升电压

**无功平衡**:
$$Q_{gen} = Q_{load} + Q_{loss} + Q_{comp}$$

## 机械式补偿装置

### 并联电容器

**补偿原理**:
提供容性无功，抵消感性无功

**分组投切**:
- 机械开关控制
- 分组容量: 1-10Mvar
- 响应时间: 秒级

**串联电抗器**:
- 限制合闸涌流
- 抑制谐波放大
- 典型电抗率: 5-12%

**电压特性**:
$$Q_C = \frac{V^2}{X_C} = \omega C V^2$$
电压下降时输出无功减少

### 并联电抗器

**应用场景**:
- 超高压线路容性无功过剩
- 轻载运行电压升高
- 电缆线路充电功率

**补偿容量**:
- 线路充电功率的50-100%
- 分级投切
- 高压侧接入

### 变压器分接头

**调压原理**:
改变变压器变比调节二次侧电压

**调压范围**:
- ±8×1.25%
- 共17档
- 有载调压

**响应时间**:
- 每档3-10秒
- 全范围60-100秒

## 静止无功补偿器(SVC)

### 基本结构

**TCR型SVC**:
- 晶闸管控制电抗器(TCR)
- 固定电容器(FC)
- 滤波器组

**TSC型SVC**:
- 晶闸管投切电容器(TSC)
- 无级调节
- 响应快

**混合型**:
- TCR+TSC组合
- 双向连续调节
- 范围宽

### TCR工作原理

**控制方式**:
$$B_{TCR} = \frac{2\pi - 2\alpha + \sin 2\alpha}{\pi X_L}$$
其中α为触发角(90°-180°)

**触发角范围**:
- 90°: 完全导通
- 180°: 完全关断
- 90°-180°: 连续调节

**谐波产生**:
- 奇次谐波
- 5次、7次显著
- 滤波器配置

### SVC控制系统

**电压闭环控制**:
$$B_{ref} = K_p(V_{ref} - V) + K_i\int(V_{ref} - V)dt$$

**斜率特性**:
$$V = V_{ref} + X_{sl} \cdot Q$$
典型斜率: 1-5%

**响应时间**:
- 晶闸管导通: 1-10ms
- 闭环响应: 1-2周期

## 静止同步补偿器(STATCOM)

### 工作原理

**电压源换流器**:
- 三相全桥逆变
- 直流电容储能
- PWM控制

**等效电路**:
$$\dot{V}_{STATCOM} = \dot{V}_{system} + jX_s \cdot \dot{I}$$

**无功调节**:
- $V_{STATCOM} > V_{system}$: 发出无功
- $V_{STATCOM} < V_{system}$: 吸收无功

### 拓扑结构

**两电平**:
- 结构简单
- 器件耐压要求高
- 适用于中低压

**三电平(NPC)**:
- 输出波形好
- 损耗较低
- 中高压应用

**MMC多电平**:
- 模块化设计
- 可扩展性强
- 高压大容量

### 控制策略

**电流解耦控制**:
$$\begin{cases}
v_d = v_d^* - \omega L i_q \\
v_q = v_q^* + \omega L i_d
\end{cases}$$

**无功电流指令**:
$$i_q^* = K_p(V_{ref} - V) + K_i\int(V_{ref} - V)dt$$

**直流电压控制**:
$$i_d^* = K_{pdc}(V_{dc}^* - V_{dc})$$

### STATCOM优势

**响应速度**:
- 响应时间: 1-2周期
- 比SVC更快

**运行范围**:
- 容性到感性连续
- 额定电流恒定

**电压特性**:
- 低电压时仍能输出额定无功
- 优于SVC

**占地面积**:
- 约为SVC的50%
- 无大型电抗器

## 同步调相机

### 工作原理

**同步电机**:
- 空载运行
- 过励磁: 发出无功
- 欠励磁: 吸收无功

**无功调节**:
$$Q = \frac{EV}{X_s}\cos\delta - \frac{V^2}{X_s}$$

**惯性响应**:
- 提供短路容量
- 增加系统惯量
- 支撑电压恢复

### 应用场景

**高压直流送端**:
- 提供短路容量
- 电压支撑
- 滤除谐波

**新能源汇集**:
- 无功储备
- 电压稳定
- 故障穿越

**负载中心**:
- 动态电压支撑
- 功率因数校正
- 电能质量改善

## 建模方法

### 稳态模型

**SVC稳态**:
$$Q_{SVC} = B_{SVC} \cdot V^2$$
$$B_{min} \leq B_{SVC} \leq B_{max}$$

**STATCOM稳态**:
$$Q_{STATCOM} = \frac{V_s \cdot V_c \cdot \sin\delta}{X}$$

**约束条件**:
- 容量限制
- 电流限制
- 电压限制

### 暂态模型

**控制系统**:
- 测量环节
- PI调节器
- 触发/调制环节

**延时环节**:
- 测量延时: 10-20ms
- 控制延时: 5-10ms
- 执行延时: 1-10ms

**饱和与限幅**:
- 无功输出限幅
- 电流限幅
- 电压限幅

### 电磁暂态模型

**详细模型**:
- 阀级详细模型
- 开关过程
- 谐波分析

**平均值模型**:
- 忽略开关过程
- 基频等效
- 计算效率高

**等效模型**:
- 电压源等效
- 电流源等效
- 简化潮流计算
