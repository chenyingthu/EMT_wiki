# 故障阻抗模型 (Fault Impedance Model)

## 概述

故障阻抗模型(Fault Impedance Model)是描述电力系统故障点阻抗特性的数学模型，用于模拟短路故障时故障点的电气行为。故障阻抗主要包括电弧电阻、接地电阻、接触电阻和树木电阻等，这些阻抗会显著影响故障电流大小、故障测距精度和保护装置的动作特性。准确的故障阻抗模型对于保护整定计算、故障分析和继电保护设计具有重要意义。

## 故障阻抗组成

### 阻抗类型

| 类型 | 描述 | 典型值 | 影响因素 |
|-----|------|--------|---------|
| 电弧电阻 | 电弧放电形成的电阻 | 1-100Ω | 电流、距离、介质 |
| 接地电阻 | 接地装置阻抗 | 0.1-100Ω | 土壤电阻率、接地方式 |
| 接触电阻 | 导体接触电阻 | 0.01-10Ω | 接触面积、压力 |
| 树木电阻 | 树木导电电阻 | 100-10000Ω | 树种、湿度 |
| 冰雪电阻 | 冰雪覆盖电阻 | 可变 | 温度、湿度 |

### 物理意义

**故障回路**：
```
电源 ───┬─── 线路阻抗 ───┬─── 故障点
        │                │
        │               [Zf] 故障阻抗
        │                │
       负载              └─── 地/中性点
```

**故障电流**：
$$I_f = \frac{V_{source}}{Z_{source} + Z_{line} + Z_f}$$

## 电弧阻抗模型

### 电弧物理特性

**电弧形成**：
1. 绝缘击穿
2. 热电离
3. 电弧稳定燃烧

**电弧电压特性**：
```
电弧电压
    │
 Uc ├──────────────┐
    │              │
    │    ┌────┐    │
    │    │    │    │
    └────┴────┴────┴──→ 电流
        低   高
```

### 经典电弧模型

#### Warrington公式

$$R_{arc} = \frac{28700 \cdot L_{arc}}{I_f^{1.4}}$$

其中：
- $R_{arc}$: 电弧电阻 (Ω)
- $L_{arc}$: 电弧长度 (m)
- $I_f$: 故障电流有效值 (A)

**特点**：电弧电阻与电流成反比

#### Van Warrington公式（改进）

$$R_{arc} = \frac{440 \cdot L_{arc}}{I_f^{0.73}}$$

**适用范围**：高压输电线路

### 动态电弧模型

#### Cassie-Mayr模型

**Cassie方程**（高电流区）：
$$\frac{1}{g}\frac{dg}{dt} = \frac{1}{\tau}\left(\frac{u^2}{U_c^2} - 1\right)$$

**Mayr方程**（低电流区）：
$$\frac{1}{g}\frac{dg}{dt} = \frac{1}{\tau}\left(\frac{ui}{P_0} - 1\right)$$

其中：
- $g$: 电弧电导
- $\tau$: 时间常数
- $U_c$: 电弧特征电压
- $P_0$: 散热功率

### 电弧电阻时变特性

**工频电弧**：
$$R_{arc}(t) = R_0 + R_1\sin(2\omega t + \phi)$$

**非线性特性**：
- 电压过零时电阻增大
- 电流峰值时电阻减小
- 呈现非线性V-I特性

## 接地阻抗模型

### 土壤电阻率影响

**接地电阻计算**：
$$R_g = \frac{\rho}{2\pi L}\ln\frac{4L}{d}$$

其中：
- $\rho$: 土壤电阻率 (Ω·m)
- $L$: 接地极长度 (m)
- $d$: 接地极直径 (m)

**典型土壤电阻率**：
| 土壤类型 | 电阻率(Ω·m) |
|---------|------------|
| 沼泽地 | 5-50 |
| 农田 | 50-200 |
| 砂砾 | 200-500 |
| 岩石 | 500-10000 |

### 接地方式影响

**有效接地**：
- 中性点直接接地
- 接地电阻小
- 故障电流大

**非有效接地**：
- 中性点经消弧线圈/电阻接地
- 接地电阻大
- 故障电流小

### 季节变化

**影响因素**：
- 土壤湿度
- 温度
- 冻土深度

**修正系数**：
$$R_{g,winter} = K \cdot R_{g,summer}, \quad K = 1.2-3.0$$

## 过渡电阻对保护的影响

### 距离保护影响

**测量阻抗**：
$$Z_m = Z_{line} + \frac{I_f}{I_m} \cdot R_f$$

其中：
- $I_f$: 故障点总电流
- $I_m$: 保护测量电流

**超越现象**：
- 对端电源助增
- 测量阻抗增大
- 保护范围缩小

### 故障测距影响

**测距误差**：
$$\%Error = \frac{R_f \cdot \sin(\phi_{line} - \phi_{fault})}{X_{line}} \times 100\%$$

**改善方法**：
- 双端测距
- 故障分量法
- 零序补偿

### 零序保护影响

**零序电流**：
$$3I_0 = \frac{3V_0}{Z_0 + 3R_f}$$

**灵敏度降低**：
- 高阻接地故障
- 零序电流减小
- 保护可能拒动

## 建模方法

### 恒定阻抗模型

**简化模型**：
$$Z_f = R_f = \text{const}$$

**适用场景**：
- 稳态故障分析
- 保护整定计算
- 简单短路计算

### 时变阻抗模型

**指数衰减**：
$$R_f(t) = R_0 + (R_\infty - R_0)(1 - e^{-t/\tau})$$

**电弧拉长**：
$$L_{arc}(t) = L_0 + v_{wind} \cdot t$$

### 非线性模型

**分段线性**：
$$R_{arc}(I) = \begin{cases} R_1 & I < I_{th} \\ R_2 & I \geq I_{th} \end{cases}$$

**多项式拟合**：
$$V_{arc} = a + bI + cI^2$$

## 不同故障类型

### 单相接地故障

**零序网络**：
```
    ┌─── Z₁ ───┐
    │          │
   (+)        (+)
    V          Zf
    │          │
    └─── Z₀ ───┘
```

**故障阻抗**：
$$Z_f = R_f + \frac{1}{j\omega C_0}$$

### 相间短路

**正序网络**：
$$I_f = \frac{E}{Z_1 + Z_f}$$

**电弧特性**：
- 电压较低
- 电弧长度短
- 电阻较小

### 三相短路

**对称故障**：
$$I_f = \frac{V_{LL}}{\sqrt{3}(Z_1 + Z_f)}$$

**金属性故障**：
$$Z_f \approx 0$$

## EMT仿真实现

### 电弧模型实现

**非线性电阻**：
```
function R_arc = Arc_Model(I, L_arc)
    % Warrington公式
    R_arc = 28700 * L_arc / abs(I)^1.4;
    
    % 限幅
    R_arc = min(R_arc, R_max);
    R_arc = max(R_arc, R_min);
end
```

**动态方程**：
- 状态空间表示
- 微分方程求解
- 迭代计算

### 接地阻抗

**频变特性**：
$$Z_g(f) = R_g + j\omega L_g$$

**冲击接地**：
- 火花效应
- 土壤电离
- 非线性特性

## 应用案例

### 高阻接地故障检测

**问题**：
- 故障电流小于负荷电流
- 传统保护难以检测
- 持续电弧危险

**解决方案**：
- 零序电压保护
- 谐波检测
- 暂态量保护

### 距离保护整定

**过渡电阻覆盖**：
$$R_{set} = k \cdot R_{f,max}$$

其中$k$为可靠系数(1.2-1.5)。
