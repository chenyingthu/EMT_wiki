# 二极管 (Diode)



## 定义与概述

二极管是单向导电的半导体器件，在EMT仿真中用于整流、续流、保护和开关等应用。实际二极管具有正向压降、反向恢复特性、结电容等非理想效应，这些特性在电力电子开关暂态仿真中尤为重要。本模型涵盖理想二极管、详细半导体物理模型、开关模型，适用于整流器、逆变器和保护电路的EMT分析。

## 1. 物理对象概述

### 1.1 功能与分类

**基本功能**：
- 单向导电（整流）
- 反向电压阻断
- 续流保护
- 电压钳位

**二极管类型**:
| 类型 | 结构 | 电压/电流 | 开关速度 | 应用 |
|------|------|-----------|----------|------|
| 整流二极管 | 硅PN结 | 1-10kV/1-10kA | 慢(μs) | 工频整流 |
| 快恢复二极管(FRD) | 硅PN结+金掺杂 | 0.6-3kV/1-5kA | 快(50-200ns) | 高频整流 |
| 肖特基二极管 | 金属-半导体 | 20-200V/1-500A | 极快(<50ns) | 低压高频 |
| 功率二极管 | 扩散PN结 | 1-8kV/0.1-5kA | 中等 | 电力电子 |

### 1.2 物理结构

**PN结结构**：
- **阳极(P区)**：掺杂受主杂质
- **阴极(N区)**：掺杂施主杂质
- **耗尽层**：空间电荷区

**载流子运动**：
- **正向偏置**：多数载流子扩散，形成导通电流
- **反向偏置**：少数载流子漂移，形成漏电流
- **反向恢复**：存储电荷清除过程

**寄生参数**：
- 结电容 $C_j$：随反向电压变化
- 扩散电容 $C_d$：与正向电流相关
- 体电阻 $R_b$：半导体材料电阻
- 引线电感 $L_s$：封装引线电感

### 1.3 运行激励

**电压激励**：
- 正向导通电压：0.3-1.2V（取决于材料）
- 反向阻断电压：额定电压的2/3（安全工作区）
- 浪涌电压：瞬时过压能力

**电流激励**：
- 正向额定电流：由热限制决定
- 浪涌电流：10ms半周波最大电流
- 反向恢复电流：关断瞬间的反向电流

**dI/dt限制**：
- 正向dI/dt：由封装电感限制
- 反向dI/dt：影响反向恢复特性

## 2. 物理模型与数学描述

### 2.1 理想二极管模型

**开关特性**：
$$i_D = \begin{cases}
0, & v_D < 0 \quad \text{(截止)} \\
\infty, & v_D \geq 0 \quad \text{(导通)}
\end{cases}$$

或电压源形式：
$$v_D = \begin{cases}
0, & i_D > 0 \quad \text{(导通)} \\
-\infty, & i_D \leq 0 \quad \text{(截止)}
\end{cases}$$

### 2.2 肖克利二极管方程

**基本方程**：
$$i_D = I_S \left(e^{\frac{v_D}{nV_T}} - 1\right)$$

其中：
- $I_S$：反向饱和电流（10⁻¹⁵至10⁻¹² A）
- $V_T = kT/q$：热电压（室温约25.9mV）
- $n$：理想因子（1-2）

**修正模型**（考虑体电阻）：
$$i_D = I_S \left(e^{\frac{v_D - i_D R_b}{nV_T}} - 1\right)$$

### 2.3 正向导通模型

#### 2.3.1 恒定压降模型

$$v_D = V_{on}, \quad i_D > 0$$

典型值：
- 硅二极管：0.6-0.7V
- 肖特基二极管：0.3-0.5V

#### 2.3.2 分段线性模型

$$v_D = \begin{cases}
0, & i_D = 0 \quad \text{(截止)} \\
V_{on} + R_{on} i_D, & i_D > 0 \quad \text{(导通)}
\end{cases}$$

其中：
- $V_{on}$：开启电压（0.5-1.0V）
- $R_{on}$：导通电阻（1-100mΩ）

### 2.4 反向恢复模型

#### 2.4.1 存储电荷模型

反向恢复过程由存储电荷 $Q_s$ 决定：
$$\frac{dQ_s}{dt} = i_D - \frac{Q_s}{\tau}$$

其中 $\tau$ 为少数载流子寿命。

**反向恢复时间**：
$$t_{rr} = \tau \ln\left(1 + \frac{I_F}{I_R}\right)$$

其中：
- $I_F$：正向导通电流
- $I_R$：反向恢复电流峰值

#### 2.4.2 双指数模型

反向恢复电流：
$$i_R(t) = -I_{RM} \left( e^{-t/\tau_1} - e^{-t/\tau_2} \right)$$

其中：
- $I_{RM}$：反向恢复电流峰值
- $\tau_1$：慢衰减时间常数
- $\tau_2$：快衰减时间常数

**软度因子**：
$$S = \frac{t_{rrs}}{t_{rrf}} = \frac{\text{软恢复时间}}{\text{快恢复时间}}$$

- $S < 1$：硬恢复（snappy）
- $S > 1$：软恢复

### 2.5 结电容模型

#### 2.5.1 耗尽层电容

$$C_j = \frac{C_{j0}}{\left(1 - \frac{v_D}{V_{bi}}\right)^M}$$

其中：
- $C_{j0}$：零偏置结电容
- $V_{bi}$：内建电势（0.5-0.9V）
- $M$：梯度系数（0.33-0.5）

#### 2.5.2 扩散电容

正向偏置时的扩散电容：
$$C_d = \frac{\tau I_D}{nV_T}$$

**总电容**：
$$C_{tot} = C_j + C_d$$

## 3. EMT仿真模型

### 3.1 理想开关模型

**导纳表示**：
$$G_D = \begin{cases}
G_{on} \approx \infty, & \text{导通} \\
G_{off} \approx 0, & \text{截止}
\end{cases}$$

**EMT实现**:
使用可变导纳：
$$G_{eq} = \begin{cases}
10^6 S, & v_D > 0 \\
10^{-9} S, & v_D \leq 0
\end{cases}$$

### 3.2 分段线性电阻模型

**状态判断**：
```
if v_D > V_on:
    G_eq = 1/R_on
    I_hist = V_on / R_on
else:
    G_eq = G_off = 1e-9
    I_hist = 0
```

**节点导纳**：
$$Y_{nn} = \begin{bmatrix} G_{eq} & -G_{eq} \\ -G_{eq} & G_{eq} \end{bmatrix}$$

### 3.3 指数特性模型

**迭代求解**（牛顿法）：
$$f(v_D) = i_D - I_S\left(e^{v_D/nV_T} - 1\right) = 0$$

**雅可比矩阵元素**：
$$\frac{\partial i_D}{\partial v_D} = \frac{I_S}{nV_T} e^{v_D/nV_T}$$

**离散化导纳**：
$$G_{eq} = \frac{\partial i_D}{\partial v_D}$$

### 3.4 反向恢复模型

#### 3.4.1 电荷控制模型

状态方程：
$$\frac{dQ}{dt} = i_D - \frac{Q}{\tau}$$

**电导率调制**：
$$G_{on} = G_{min} + (G_{max} - G_{min}) \frac{Q}{Q_{ss}}$$

其中 $Q_{ss} = \tau I_F$ 为稳态存储电荷。

#### 3.4.2 查表模型

使用实测的反向恢复波形建立查表：
- $t_{rr}$ vs $I_F$
- $I_{RM}$ vs $di/dt$
- $Q_{rr}$ vs $I_F$ 和 $di/dt$

### 3.5 详细暂态模型

**完整等效电路**:
```
    Ls      Rs
━━━////━━━━/\/\/\━┓
                 ┃
    ━━━━━━━━━━━━━┫
    ┃     Cj     ┃
  /-/\/\/\       ┫
  ┃    Rd        ┃
  ┃   (可变)     ┃
  ┃              ┃
━━┻━━━━━━━━━━━━━━┻━
```

**EMT离散化**:
各元件分别离散化后组合到导纳矩阵。

## 4. 仿真软件实现

### 4.1 PSCAD/EMTDC实现

**理想二极管**:
```fortran
DIODE
  VFWD = 0.0      ! 正向压降
  VREV = 0.0      ! 反向阻断电压
```

**详细模型**:
```fortran
DIODE_DETAILED
  IS = 1.0e-12    ! 饱和电流(A)
  N = 1.0         ! 理想因子
  RS = 1.0e-3     ! 串联电阻(Ohm)
  CJO = 100.0e-12 ! 零偏结电容(F)
  VJ = 0.75       ! 内建电势(V)
  TT = 50.0e-9    ! 渡越时间(s)
```

**自定义模型**:
```fortran
! 反向恢复模型
IF (STATE .EQ. ON) THEN
  IF (V_D .LT. 0.0) THEN
    ! 开始反向恢复
    Q_STORED = TAU * I_FORWARD
    STATE = RECOVERY
  END IF
ELSE IF (STATE .EQ. RECOVERY) THEN
  DQ/DT = I_D - Q/TAU
  IF (Q .LT. 0.0) STATE = OFF
END IF
```

### 4.2 MATLAB/Simulink实现

**Simscape模型**:
```matlab
% 二极管
D = diode('D');

% 设置参数
D.Vf = 0.7;      % 正向压降
D.Ron = 0.01;    % 导通电阻
D.Roff = 1e6;    % 关断电阻
```

**自定义模型**:
```matlab
function [i_d, state] = diode_model(v_d, state_prev, dt, params)
    V_on = params.V_on;
    R_on = params.R_on;
    R_off = params.R_off;
    
    % 状态判断
    if state_prev == 1 || v_d > V_on  % 导通
        i_d = (v_d - V_on) / R_on;
        state = 1;
    else  % 关断
        i_d = v_d / R_off;
        state = 0;
    end
end
```

**反向恢复模型**:
```matlab
function didt = diode_reverse_recovery(t, Q, i_F, params)
    tau = params.tau;
    dQdt = i_F - Q/tau;  % 存储电荷变化率
    
    % 电流与电荷关系
    i_D = params.I_S * (exp(Q/params.Q_ref) - 1);
end
```

### 4.4 实时仿真考虑

**步长限制**:
- 反向恢复过程：$\Delta t < t_{rr}/10$
- 快速二极管（$t_{rr}$ < 100ns）：需步长 < 10ns
- 可使用子步长处理反向恢复

**数值稳定性**:
- 导通/关断切换：使用插值避免振荡
- 状态改变时：重新计算导纳矩阵

**模型简化**:
对于系统级仿真，可使用：
- 恒定压降模型
- 忽略反向恢复
- 固定导通电阻

- **热耦合**：电-热耦合需单独建模

## 量化性能边界

二极管作为开关器件的 EMT 仿真性能已有可核验的量化结果，但以下数据均绑定具体建模方法和验证条件，不能外推为通用能力：

- **Na (2023)** 提出了结合半步长后向欧拉法与插值/外推策略的改进算法，用于固定步长 EMT 仿真中开关器件（含二极管、IGBT、晶闸管）的暂态模拟。在 IGBT/二极管/电感电路测试中，虚假开关损耗从普通插值法的 **212.6 μJ** 大幅降低至约 **2 μJ**，消除率超过 **99%**。该方法彻底消除了传统瞬时插值法固有的"提前一个时间步"数值误差，电感电流/电压波形与 0.1 μs 参考解的相位偏差趋近于零。验证基于自研 EMT 仿真程序和特定电力电子拓扑，不自动适用于大规模网络或多开关频繁动作场景（Na 2023）。

这些量化数据不构成对二极管建模方法的全面性能评价，只说明在特定测试条件下可获得的能力边界。
