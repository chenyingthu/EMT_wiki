# 负荷模型 (Load Model)

## 定义与概述

电力系统负荷是消耗电能的设备总和，包括恒阻抗（Z）负荷、恒电流（I）负荷、恒功率（P）负荷、感应电机负荷等多种类型。负荷模型用于描述负荷的电压特性、频率特性和动态响应，对电力系统潮流计算、暂态稳定和电压稳定分析至关重要。

从电磁暂态（EMT）仿真角度看，负荷模型的核心挑战在于**如何用精确且可计算的等效电路再现实际负荷在电压/频率扰动下的动态行为**。传统静态ZIP模型将大量终端设备等效为静态阻抗、导纳或恒功率，难以再现电机转速/转矩动态、电力电子整流前端、照明驱动和变频负荷在电压变化下的非线性行为（Milani 2021）。

本模型涵盖ZIP负荷模型、频率依赖模型、感应电机动态模型、温控负荷模型，适用于配电网仿真、电压稳定分析和负荷建模。

## EMT中的角色

### 为什么需要负荷模型

EMT仿真中，负荷模型决定了仿真系统对电压/频率扰动的响应精度。主要挑战包括：

1. **终端设备类型多样**：现代配电网负荷包含开关电源（SMPS）、LED驱动、变频电机、恒温控制负荷等，聚合特性差异大
2. **非线性动态行为**：电机脱扣/重合闸、电力电子整流前端在电压暂降/恢复过程中呈现复杂的非线性响应
3. **频率依赖特性**：感应电机转速随频率变化，进而影响有功功率，在低惯量系统中频率相关负荷对UFLS整定有重要影响（Fini等 2026）
4. **电压敏感非线性**：恒功率负荷在电压跌落时的电流特性与恒阻抗完全不同

### 关键需求

| 需求 | 描述 |
|------|------|
| 动态响应精度 | 在电压暂降/恢复过程中准确复现电机脱扣、负荷恢复等动态过程 |
| 频率敏感性 | 在低惯量系统频率摆动中反映负荷频率依赖 |
| 计算效率 | 在EMT步长下可求解，支持长时间尺度仿真 |
| 参数可辨识 | 负荷组成比例可从现场录波数据中提取 |

## EMT建模方法

### 方法1：ZIP静态负荷模型

ZIP负荷模型是最广泛使用的静态负荷模型，通过三个多项式项描述有功和无功功率随电压的变化关系：

$$P = P_0 \left[a_p\left(\frac{V}{V_0}\right)^2 + b_p\left(\frac{V}{V_0}\right) + c_p\right]$$

$$Q = Q_0 \left[a_q\left(\frac{V}{V_0}\right)^2 + b_q\left(\frac{V}{V_0}\right) + c_q\right]$$

其中归一化系数满足 $a + b + c = 1$：
- $a$：恒阻抗（Z）比例 → $P \propto V^2$
- $b$：恒电流（I）比例 → $P \propto V$
- $c$：恒功率（P）比例 → $P = \text{const}$

**EMT离散实现（恒导纳伴随电路）**：

$$G_{eq} = G_0 \left[a\left(\frac{V}{V_0}\right) + b + c\left(\frac{V_0}{V}\right)\right]$$

历史电流源用于保持非线性特性：
$$I_{hist} = I_{ZIP}(V(t-\Delta t), f)$$

**典型参数**（IEEE Std. 3002.2）：

| 负荷类型 | $a_p$ | $b_p$ | $c_p$ | $a_q$ | $b_q$ | $c_q$ |
|---------|-------|-------|-------|-------|-------|-------|
| 居民 | 0.5 | 0.3 | 0.2 | 0.0 | 0.0 | 1.0 |
| 商业 | 0.4 | 0.3 | 0.3 | 0.3 | 0.3 | 0.4 |
| 工业 | 0.2 | 0.2 | 0.6 | 0.1 | 0.2 | 0.7 |
| 农业 | 0.7 | 0.2 | 0.1 | 0.6 | 0.3 | 0.1 |

### 方法2：频率相关负荷模型（fdLoad）

频率相关负荷在低惯量系统中对频率稳定性有重要影响。fdLoad模型（Fini等 2026）将指数型频率相关负荷转化为等效导纳网络，使其可直接接入节点方程：

$$P(f,V) = P_0 \left(\frac{f}{f_0}\right)^{\alpha_f} \left(\frac{V}{V_0}\right)^{\alpha_v}$$

$$Q(f,V) = Q_0 \left(\frac{f}{f_0}\right)^{\beta_f} \left(\frac{V}{V_0}\right)^{\beta_v}$$

**向等效导纳的转化**（在 $V \approx V_0$ 条件下）：
$$Y(f) = \frac{P(f) - jQ(f)}{V^2}$$

通过Vector Fitting在目标频带（如57–63 Hz）内拟合为有理函数，再展开为常数R、L、C并联/串联支路网络：

| 频率指数 | 感应电机 | 综合负荷 | 照明 | 加热 |
|---------|---------|---------|------|------|
| $\alpha_f$（有功） | 2.0–3.0 | 1.0–2.0 | 0–0.5 | 0 |
| $\beta_f$（无功） | 0–1.0 | -1.0–0 | 0 | 0 |

**SFA-EMT结合**：SFA-EMT将60 Hz基波频移至0 Hz附近，使每一步的频率隐含在网络代数方程解中。对于0.9 Hz频偏，传统EMT需步长约0.8 ms，而SFA-EMT可使用约55 ms，在接近相量仿真成本下捕捉频率相关负荷对UFLS的影响。

### 方法3：感应电机动态模型

感应电机是配电网中最重要的动态负荷组成部分，其EMT模型需要保留电磁与机械动态的耦合。

**dq0坐标下的动态方程**：

电压方程：
$$v_{ds} = R_s i_{ds} + \frac{d\psi_{ds}}{dt} - \omega_s \psi_{qs}$$
$$v_{qs} = R_s i_{qs} + \frac{d\psi_{qs}}{dt} + \omega_s \psi_{ds}$$

磁链方程（线性磁路假设）：
$$\psi_{ds} = L_s i_{ds} + L_m i_{dr}$$
$$\psi_{qs} = L_s i_{qs} + L_m i_{qr}$$
$$\psi_{dr} = L_m i_{ds} + L_r i_{dr}$$
$$\psi_{qr} = L_m i_{qs} + L_r i_{qr}$$

转子运动方程：
$$T_m = T_{load} + 2H \frac{ds}{dt}$$

其中 $s = (\omega_s - \omega_r)/\omega_s$ 为滑差，$H$ 为惯性时间常数。

**电压稳定性临界条件**：
$$V_{crit} = 0.6 - 0.7 \text{ p.u.}, \quad s_{stall} = 0.1 - 0.2$$

**实用稳态等效电路**（用于潮流计算）：
$$P = \frac{V^2 R_r/s}{(R_s + R_r/s)^2 + (X_s + X_r)^2}$$

### 方法4：详细元件级综合负荷模型（Component-based）

Milani（2021）提出的详细元件级负荷模型将配电网母线级负荷分解为若干可辨识终端设备，在EMTP中用子电路层级架构进行聚合：

**元件组成**：
- 单相/三相感应电机（直接驱动机械负载）
- 开关电源（SMPS）：等效电阻 $R_{eq} = v_{dc}^2 / P_{rated}$
- LED驱动器：阈值电压 $V_{thr} = 2.7$ V，动态电阻 $R_d = 0.3$ Ω
- 紧凑型荧光灯（CFL）：充电态 $R_{eq,charging} = 11v_{dc} + 0.021v_{dc}^2 + 1300$ Ω
- 变频驱动器（VSI）：PWM矢量控制
- 静态电阻负荷

**聚合等效参数**：
$$P_{eq} = \sum_i P_i, \quad Q_{eq} = \sum_i Q_i$$
$$Z_{eq} = \frac{V^2}{P_{eq} - jQ_{eq}}$$
$$H_{eq} = \frac{\sum_i H_i S_i}{\sum_i S_i}$$

**电机保护逻辑（现场校准参数）**：
- 脱扣电压阈值：额定电压的45%–55%
- 脱扣持续时间：1–3个工频周波
- 重合闸延时：电压恢复至~65%后20 ms
- 重合闸成功率：70%的电机负荷在重合闸后20 ms内完成重连

### 方法5：温控负荷聚合模型

温控负荷（空调、热水器等）通过热平衡方程建模，在配电网中大量聚合后可等效为一阶传递函数：

**单体温控负荷热平衡方程**：
$$C_{th} \frac{dT_{in}}{dt} = P_{heat} - \frac{T_{in} - T_{out}}{R_{th}}$$

**控制滞后特性**：
$$
P_{heat} = \begin{cases}
P_{rated}, & T_{in} < T_{set} - \Delta T/2 \\
0, & T_{in} > T_{set} + \Delta T/2 \\
P_{prev}, & \text{otherwise}
\end{cases}
$$

**大规模聚合等效**（Fini等 2026）：
$$G_{agg}(s) = \frac{P_{agg}}{\Delta f} = \frac{K}{1 + sT_{eq}}$$

## 形式化表达

### ZIP负荷模型

$$\begin{bmatrix} P \\ Q \end{bmatrix} = \begin{bmatrix} P_0 & 0 \\ 0 & Q_0 \end{bmatrix} \begin{bmatrix} a_p & b_p & c_p \\ a_q & b_q & c_q \end{bmatrix} \begin{bmatrix} (V/V_0)^2 \\ (V/V_0) \\ 1 \end{bmatrix}$$

### 频率相关负荷模型

$$Y(f) = \frac{P_0}{V_0^2} \left(\frac{f}{f_0}\right)^{\alpha_f} - j \frac{Q_0}{V_0^2} \left(\frac{f}{f_0}\right)^{\beta_f}$$

### 感应电机动态模型

$$\mathbf{v}_s = \mathbf{R}_s \mathbf{i}_s + \frac{d\boldsymbol{\psi}_s}{dt} + \omega_s \mathbf{J} \boldsymbol{\psi}_s$$
$$\mathbf{v}_r = \mathbf{R}_r \mathbf{i}_r + \frac{d\boldsymbol{\psi}_r}{dt} + (s\omega_s) \mathbf{J} \boldsymbol{\psi}_r$$
$$\mathbf{v}_s = \boldsymbol{\psi}_s \mathbf{L}_s^{-1} + \boldsymbol{\psi}_r \mathbf{L}_m^{-1}$$

## 关键技术挑战

### 挑战1：电机脱扣与重合闸的精确模拟

电机接触器在电压暂降时的脱扣阈值（45%–55%额定电压）和重合闸延时（20 ms）是影响负荷恢复动态的关键参数。需要根据现场录波数据对每个地区的具体阈值进行校准，而非使用通用整定值。

**现场校准方法**（Milani 2021）：
1. 施加实测电压暂降波形到聚合负荷模型
2. 比较仿真与实测的三相电流、有功/无功功率波形
3. 迭代调整脱扣阈值、延时和负荷组成比例
4. 验证电机接触器脱扣和重合闸产生的暂态冲击电流

### 挑战2：电力电子负荷的等效处理

传统模型将电力电子负荷简化为恒功率（CMLD方法），但实际开关电源在电压暂降时从直流侧电容汲取的功率是电压依赖的。详细元件级模型通过等效电阻公式捕捉这种非线性：

$$R_{eq} = \frac{v_{dc}^2}{P_{rated}} \quad \text{(SMPS)}$$

### 挑战3：大规模温控负荷的聚合动态

大量温控负荷聚合后，其总功率对频率扰动的响应可等效为一阶传递函数，但等效时间常数 $T_{eq}$ 取决于负荷组成和气候条件，需要从实测数据中辨识。

### 挑战4：低惯量系统中频率相关负荷对UFLS的影响

在IBR高渗透率系统中，负荷频率依赖会影响切负荷后最大频率偏差，可能导致UFLS过早或过晚动作。fdLoad模型将频率相关负荷综合为网络支路，使当前步频率在网络方程中隐式求解，避免了后处理相角差估计的延迟问题。

### 挑战5：负荷组成比例的自适应辨识

实际配电网负荷组成比例随时间、季节和天气变化。详细元件级模型需要定期用现场录波数据更新各元件占比，而非使用固定参数。

## 适用边界与选择指南

### 模型选择决策表

| 研究类型 | 推荐模型 | 步长 | 关键参数 |
|---------|---------|------|---------|
| 潮流计算 | ZIP静态 | 稳态 | $a_p, b_p, c_p$ |
| 暂态稳定 | 电机+ZIP | 1–5 ms | 电机惯性、负荷比例 |
| 电压稳定 | 详细元件级 | 0.1–1 ms | 电机动态、SMPS等效 |
| EMT仿真 | 恒导纳/详细元件级 | 10–100 μs | 电机保护逻辑 |
| 频率摆动（SFA-EMT） | fdLoad | 1–10 ms | $\alpha_f, \beta_f$ |
| 谐波分析 | 详细元件级 | <50 μs | 整流器谐波模型 |

### 适用条件

- **电压范围**：0.7–1.1 p.u.（超出可能导致电机失速或电力电子保护动作）
- **频率范围**：48–52 Hz（大范围频率偏移需用fdLoad）
- **时间尺度**：周波级至分钟级
- **负荷类型**：静态负荷+感应电机为主，电力电子占比<50%

### 失效边界

1. **聚合简化丢失个体特性**：多台电机聚合为等效参数后，无法再现单台电机的保护动作差异
2. **温控负荷**：简化为等效热模型后，无法捕捉温控器相位差造成的功率波动
3. **电子负荷恒功率假设**：电压暂降期间整流前端功率不可视为恒定
4. **新能源渗透**：分布式光伏/储能的反向功率流未纳入传统负荷模型框架

### 关键假设

1. 负荷组成比例已知且在仿真时段内相对固定
2. 各负荷元件独立运行，忽略相互耦合（忽略配电网络阻抗）
3. 负荷电压为理想正弦，忽略谐波畸变（除非使用详细元件级模型）
4. 电机转矩特性为已知（恒转矩或二次方转矩）
