# 电阻 (Resistor)

## 定义与概述

电阻是EMT仿真中最基础的电路元件，表示对电流的阻碍作用。在理想情况下，电阻遵循欧姆定律，电压与电流呈线性关系。然而，在高频电磁暂态仿真中，实际电阻器表现出频率相关特性、寄生电感和寄生电容效应。本模型涵盖理想电阻、频率相关电阻、分布参数电阻，适用于从直流到高频EMT仿真的各种场景。

## 1. 物理对象概述

### 1.1 功能与分类

**基本功能**：
- 限制电流流动
- 分压和分流
- 能量耗散（转化为热能）
- 信号衰减和匹配

**电阻类型**:
| 类型 | 材料 | 功率范围 | 特点 | 应用 |
|------|------|----------|------|------|
| 碳膜电阻 | 碳膜 | 0.125-2W | 成本低、噪声大 | 一般电路 |
| 金属膜电阻 | 金属膜 | 0.125-1W | 精度高、稳定性好 | 精密电路 |
| 线绕电阻 | 合金线 | 1-500W | 功率大、有电感 | 功率应用 |
| 水泥电阻 | 绕线封装 | 2-50W | 耐脉冲、散热好 | 电源电路 |
| 贴片电阻 | 厚膜/薄膜 | 0.05-1W | 体积小、高频好 | 电子产品 |

### 1.2 物理结构

**线绕电阻寄生效应**：
- 绕线结构引入寄生电感（1-100 μH）
- 层间电容形成寄生电容（0.1-10 pF）
- 高频时呈现阻抗特性而非纯电阻

**薄膜电阻高频特性**：
- 寄生电感较小（nH级）
- 分布电容影响高频响应
- 趋肤效应在高频显著

### 1.3 运行激励

**电压激励**：
- 工作电压：0-数千伏（取决于规格）
- 脉冲电压：可达额定电压的2-3倍（短时）
- 过电压：可能引发电弧或损坏

**电流激励**：
- 额定电流：由功率和阻值决定 $I_{rated} = \sqrt{P/R}$
- 脉冲电流：取决于热容量和脉冲宽度
- 过电流：导致温升和阻值变化

**频率范围**：
- DC至数GHz（取决于类型和封装）

## 2. 物理模型与数学描述

### 2.1 理想电阻模型

**欧姆定律**:
$$v(t) = R \cdot i(t)$$
$$i(t) = G \cdot v(t), \quad G = 1/R$$

**功率耗散**:
$$p(t) = v(t) \cdot i(t) = R \cdot i^2(t) = \frac{v^2(t)}{R}$$

**能量耗散**:
$$W = \int_0^t p(\tau) d\tau = R \int_0^t i^2(\tau) d\tau$$

### 2.2 频率相关电阻模型

#### 2.2.1 趋肤效应模型

高频时导体内部电流分布不均匀，有效电阻增加：

$$R(f) = R_{DC} \cdot k_{skin}(f)$$

**趋肤深度**:
$$\delta = \sqrt{\frac{\rho}{\pi f \mu}}$$

**圆柱导体的趋肤效应系数**（近似）：
$$k_{skin} \approx 1 + \frac{1}{3}\left(\frac{r}{\delta}\right)^4, \quad \text{当} \frac{r}{\delta} < 1$$

其中：
- $r$：导体半径
- $\rho$：电阻率
- $\mu$：磁导率

#### 2.2.2 寄生参数模型

**等效电路**（考虑寄生L和C）：

```
        Ls
    ━━━━////━━━━
    ┃          ┃
──┬┴┬──────┬┴┬──
  │ R      │ Cp
──┴┬┴──────┴┬┴──
    ━━━━━━━━━━
```

**阻抗表达式**:
$$Z(s) = \frac{(R + sL_s) \cdot \frac{1}{sC_p}}{R + sL_s + \frac{1}{sC_p}}$$

简化形式（高频近似）：
$$Z(j\omega) \approx R + j\omega L_s - j\frac{1}{\omega C_p}$$

### 2.3 温度相关模型

**线性温度系数模型**:
$$R(T) = R_{ref} [1 + \alpha (T - T_{ref})]$$

其中：
- $\alpha$：电阻温度系数（1/°C）
- $T_{ref}$：参考温度（通常25°C）

**常见材料的温度系数**:
| 材料 | 温度系数 α (ppm/°C) | 备注 |
|------|---------------------|------|
| 铜 | +3900 | 正温度系数 |
| 铝 | +4200 | 正温度系数 |
| 康铜 | ±40 | 近零温度系数 |
| 锰铜 | ±15 | 精密电阻 |

### 2.4 非线性电阻模型

#### 2.4.1 压敏电阻 (Varistor)

$$I = k \cdot V^\alpha$$

其中 $\alpha$ 为非线性系数（通常20-50）。

#### 2.4.2 热敏电阻 (NTC/PTC)

**NTC（负温度系数）**:
$$R(T) = R_0 \exp\left[B\left(\frac{1}{T} - \frac{1}{T_0}\right)\right]$$

## 3. EMT仿真模型

### 3.1 理想电阻模型

EMT仿真中最简单的元件，代数关系：

**电压源形式**:
$$v_k = R \cdot i_k$$

**电流源形式**:
$$i_k = G \cdot v_k$$

**节点导纳矩阵贡献**:
$$Y_{nn} = \begin{bmatrix} G & -G \\ -G & G \end{bmatrix}$$

### 3.2 频率相关电阻模型

#### 3.2.1 矢量拟合法

使用有理函数逼近频率响应：
$$Z(s) = R_{DC} + \sum_{i=1}^{n} \frac{c_i}{s - a_i}$$

转换为时域状态空间形式用于EMT仿真。

#### 3.2.2 分段线性化

在多个频点测量阻抗，分段线性插值：
$$R_{eff}(f) = R_i + \frac{R_{i+1} - R_i}{f_{i+1} - f_i}(f - f_i), \quad f_i \leq f < f_{i+1}$$

### 3.3 含寄生参数模型

#### 3.3.1 集总参数模型

在EMT中用RLC串联或并联表示：

**串联RLC**:
$$v = R \cdot i + L \frac{\mathrm{d}i}{\mathrm{d}t} + \frac{1}{C} \int i \, dt$$

**离散化（梯形法）**:
$$i_{n+1} = \frac{\Delta t}{2L + R\Delta t} (v_{n+1} + v_n) + \text{历史项}$$

#### 3.3.2 分布参数模型

长电阻器用分布参数传输线模型：
$$\frac{\partial v}{\partial x} = -R' i - L' \frac{\partial i}{\partial t}$$
$$\frac{\partial i}{\partial x} = -G' v - C' \frac{\partial v}{\partial t}$$

### 3.4 故障限流电阻模型

在 EMT 仿真中，故障限流电阻（Fault Current Limiting Resistor, FCLR）用于限制短路电流峰值，其等效电路为串联电阻与受控开关的并联组合。当故障检测到时，开关闭合将电阻切入电路，其等效阻抗在基频下的电流限制效果为：

$$R_{FCLR} = \frac{V_{rated}}{\sqrt{3} \cdot I_{target}}$$

其中 $I_{target}$ 为目标限制电流有效值。对于 $10\ \text{kV}$ 系统，若限制到 $2\ \text{kA}$，所需限流电阻约 $2.9\ \Omega$（Saldaña & Calzolari 2022）。

**EMTP节点方程接入**：限流电阻在时域仿真中通过诺顿等效电路接入节点方程：

$$i_k^{(n)} = G_{FCLR} \cdot v_k^{(n)} + I_{hist,FCLR}^{(n)}$$

其中 $G_{FCLR} = 1/R_{FCLR}$，历史电流项 $I_{hist,FCLR}$ 取决于开关动作时刻的电流初始值，用于保证开关闭合瞬间的电流连续性。

**量化性能边界**：实验表明，FCLR 在投入后 1 个工频周期内将短路电流峰值限制到原来的 $35\%$–$50\%$，等效衰减时间常数从纯电感系统的 $\tau_0 = L/R_{fault}$ 延长至 $\tau_{lim} = L/(R_{fault} + R_{FCLR})$，文中验证系统在 8 条出线的 10 kV 配电网中低阻故障衰减误差小于 $\pm10\%$（Saldaña & Calzolari 2022）。

### 3.5 阻尼电阻模型

阻尼电阻（Snubber / Damping Resistor）常用于电力电子开关换流过程和输电线路操作过电压的阻尼吸收。其作用是消耗LC振荡能量、抑制电压尖峰。

**阻尼效果量化**：

$$\zeta = \frac{R_{snubber}}{2} \sqrt{\frac{C}{L}}$$

当 $\zeta \geq 1$ 时为过阻尼，振荡完全抑制；当 $\zeta = 0.5$–$0.7$ 时为临界阻尼，既能有效抑制振荡又不引入过大的功率损耗（IEEE Std 399-1997）。

**EMTP等效实现**：阻尼电阻通常以并联形式接在开关或线路两端，其数值选择应满足：

$$R_{snubber} \approx 2\zeta \sqrt{\frac{L}{C}}$$

典型取值为 $10\ \Omega$–$100\ \Omega$（取决于 L 和 C 的量级），在 EMTP 中通过 `RESISTOR` 元件直接建模。

## 4. 仿真软件实现

### 4.1 PSCAD/EMTDC实现

**理想电阻**:
```fortran
! 直接电阻元件
RESISTOR
  R = 10.0     ! 电阻值(Ohm)
```

**非线性电阻**:
```fortran
! 温度相关电阻
RT = R25 * (1.0 + ALPHA * (T - 25.0))
I = V / RT
```

### 4.2 MATLAB/Simulink实现

**Simscape模型**:
```matlab
% 理想电阻
R = resistor('R', 10);  % 10 Ohm

% 频率相关电阻（使用查表）
f_data = [1e3, 1e4, 1e5, 1e6];  % Hz
R_data = [10, 10.1, 11, 15];    % Ohm
R_freq = griddedInterpolant(f_data, R_data);
R_eff = R_freq(f_operating);
```

**自定义实现**:
```matlab
function v = resistor_model(i, R, T, alpha)
    R_eff = R * (1 + alpha * (T - 25));
    v = R_eff * i;
end
```

### 4.3 SPICE实现

**基本电阻**:
```spice
R1 node1 node2 10k
```

**温度相关**:
```spice
R1 node1 node2 10k TC1=0.01 TC2=0.0001
```

**行为模型**:
```spice
B1 node1 node2 V=I(B1)*10*(1+0.004*(TEMP-25))
```

### 4.4 实时仿真考虑

**步长限制**：
- 理想电阻：无时域限制
- 含L/C寄生参数：需满足 $\Delta t < \sqrt{LC}$

**数值稳定性**：
- 极小电阻（$R < 0.001 \Omega$）：可能导致数值问题
- 极大电阻（$R > 1 G\Omega$）：等效为开路

## 7. 适用边界与限制

### 7.1 适用条件
- **阻值范围**：$10^{-6} \Omega$ 至 $10^{12} \Omega$
- **频率范围**：DC 至数GHz（取决于模型精度要求）
- **温度范围**：-65°C 至 +275°C（取决于电阻类型）
- **功率范围**：不超过额定功率的2倍（短时脉冲）

### 7.2 模型限制
- **趋肤效应模型**：仅适用于圆柱导体，薄膜电阻需修正
- **温度模型**：线性模型仅在有限温度范围内准确
- **非线性效应**：电弧、击穿等非线性现象需特殊模型
- **辐射效应**：极高频时的电磁辐射效应未考虑

### 7.3 量化性能边界

电阻模型的 EMT 仿真精度高度依赖于对寄生参数和频率相关效应的处理。现有的量化性能数据主要来自阻尼电阻和故障限流电阻的应用场景：

- **阻尼电阻**（IEEE Std 399-1997）：$\zeta = 0.5$–$0.7$ 临界阻尼条件下，开关电压尖峰抑制率可达 $60\%$–$80\%$，阻尼电阻功耗 $P_d = V_{peak}^2 / R_{snubber}$，典型值 $100\ \text{W}$–$500\ \text{W}$（对应 $10\ \text{kV}$ 系统）
- **故障限流电阻**（Saldaña & Calzolari 2022）：10 kV 配电网验证，短路电流峰值限制到原来的 $35\%$–$50\%$，低阻故障衰减误差 $< \pm10\%$，高阻故障因忽略接地支路误差可达 $34\%$
- **线路集中损耗**（Mathur 1989）：扩展 Bergeron 模型中集中电阻近似精度：端部集中模型误差 $10\%$–$20\%$，三点集中模型误差 $1\%$–$2\%$（500 kV 平武线路测试）

**数据缺口声明**：纯电阻元件的独立 EMT 建模精度系统数据尚未见报道。上述数据均来自特定应用场景，不能外推为通用精度边界。

### 7.4 关键假设
1. 电阻材料均匀、各向同性
2. 温度分布均匀（忽略局部热点）
3. 寄生参数为集总常数（实际分布参数）
4. 电阻值不随电压变化（忽略非线性）
