# 理想开关模型 (Ideal Switch Model)

## 概述

理想开关模型(Ideal Switch Model)是电力电子电路仿真中最基本的开关建模方法，通过将实际开关器件抽象为理想元件（导通时电阻为零、关断时电阻为无穷大），大幅简化电路分析和数值计算。该模型适用于拓扑验证、控制策略开发和快速仿真分析，是EMT仿真、平均值模型和状态空间分析的基础。虽然理想开关忽略了实际器件的导通压降、开关损耗和暂态过程，但其简洁性使其成为电力电子系统初步设计阶段的重要工具。

## 基本特性

### 理想开关定义

**理想条件**：
- 导通状态：零阻抗($R_{on} = 0$)
- 关断状态：无穷大阻抗($R_{off} = \infty$)
- 瞬时切换：零开关时间
- 无损耗：导通和开关过程均无能量损失

### 开关状态

| 状态 | 电阻 | 电压 | 电流 | 约束条件 |
|-----|------|------|------|---------|
| 导通 | $R_{on} = 0$ | $V_{sw} = 0$ | 由外电路决定 | 电压约束 |
| 关断 | $R_{off} = \infty$ | 由外电路决定 | $I_{sw} = 0$ | 电流约束 |

### 开关函数表示

**电阻形式**：
$$R_{sw}(t) = \begin{cases} 0 & \text{导通} \\ \infty & \text{关断} \end{cases}$$

**开关函数形式**：
$$S(t) = \begin{cases} 1 & \text{导通} \\ 0 & \text{关断} \end{cases}$$

## 数学模型

### 代数约束方程

**导通约束**（电压源型）：
$$V_{sw} = 0$$

**关断约束**（电流源型）：
$$I_{sw} = 0$$

### 混合方程系统

**一般形式**：
$$g(x, S) = 0$$

其中：
- $x$: 系统状态变量
- $S$: 开关状态向量

### 状态空间表示

**开关网络状态方程**：
$$\dot{x} = A_S x + B_S u$$

其中矩阵$A_S$和$B_S$依赖于开关状态$S$。

## EMT仿真实现

### 伴随电路模型

**导通状态等效**：
- 理想电压源：$V_{eq} = 0$
- 等效电导：$G_{eq} = \infty$

**关断状态等效**：
- 理想电流源：$I_{eq} = 0$
- 等效电导：$G_{eq} = 0$

### 节点分析处理

**导通处理**：
- 将开关两端节点短路
- 合并节点导纳矩阵
- 节点数减少

**关断处理**：
- 开关支路电流为零
- 移除该支路
- 等效为开路

### 开关事件处理

**事件驱动仿真**：
```
1. 在当前开关状态下求解电路
2. 检测开关触发条件：
   - 二极管：电流过零
   - 晶闸管：触发脉冲+正向电压
   - IGBT：门极信号
3. 确定精确开关时刻
4. 更新开关状态
5. 重初始化求解
```

**插值开关时刻**：
$$t_{sw} = t_n + \Delta t \cdot \frac{x_{ref} - x_n}{x_{n+1} - x_n}$$

## 拓扑分析应用

### 状态空间平均

**占空比定义**：
$$d = \frac{T_{on}}{T_{sw}}$$

**平均状态方程**：
$$\dot{x} = [dA_1 + (1-d)A_2]x + [dB_1 + (1-d)B_2]u$$

[[state-space-average-method]] - 状态空间平均法

### 小信号分析

**扰动方程**：
$$\dot{\hat{x}} = A_{avg}\hat{x} + B_{avg}\hat{u} + E\hat{d}$$

**传递函数**：
$$G(s) = C(sI - A_{avg})^{-1}E$$

## 实际开关模型对比

### 导通特性对比

| 特性 | 理想开关 | 实际IGBT | 实际二极管 |
|-----|---------|---------|-----------|
| 导通电阻 | 0 | 1-10 mΩ | 0.5-5 mΩ |
| 导通压降 | 0 | 1.5-3 V | 0.7-1.5 V |
| 开关时间 | 0 | 0.1-2 μs | 反向恢复时间 |
| 开关损耗 | 0 | 显著 | 反向恢复损耗 |

### 误差分析

**导通损耗误差**：
$$\Delta P = I^2R_{on,actual} + V_{ce}I$$

**相对误差**：
$$\epsilon = \frac{\Delta P}{P_{out}} \times 100\%$$

**典型值**：
- 低压大电流：5-10%
- 高压小电流：<2%

## 改进模型

### 导通电阻模型

**非理想导通**：
$$R_{sw} = \begin{cases} R_{on} & \text{导通} \\ R_{off} & \text{关断} \end{cases}$$

其中：
- $R_{on}$ = 1-100 mΩ
- $R_{off}$ = 1-100 MΩ

### 分段线性模型

**过渡特性**：
```
电压
  │
  │    ┌───────────
  │   ╱
  │  ╱ 过渡区
  │ ╱
──┴/───────────────
          电流
```

**开关过程**：
- 导通延迟：$t_{d,on}$
- 上升时间：$t_r$
- 关断延迟：$t_{d,off}$
- 下降时间：$t_f$

[[detailed-switch-model]] - 详细开关模型

### 平滑开关模型

**连续函数**：
$$R_{sw}(t) = R_{on}\frac{1+\tanh(k(t-t_{sw}))}{2} + R_{off}\frac{1-\tanh(k(t-t_{sw}))}{2}$$

**优点**：
- 避免数值振荡
- 连续导纳变化
- 适合变步长算法

## 数值问题与对策

### 开关时刻振荡

**问题**：
- 理想开关切换导致导纳矩阵突变
- 数值振荡（虚假振荡）

**解决**：
- 临界阻尼调整(CDA)
- 后退欧拉法
- 插值开关时刻

[[numerical-oscillation-suppression]] - 数值振荡抑制

### 电感电流突变

**问题**：
- 开关关断时电感电流无处流通
- 违反物理定律

**处理**：
- 续流二极管
- 缓冲电路
- 死区时间

### 电容电压突变

**问题**：
- 开关导通时电容短路
- 无穷大电流

**处理**：
- 串联电阻
- 实际ESR
- 软开关

## 典型电路分析

### Buck变换器

**导通状态**：
$$L\frac{di_L}{dt} = V_{in} - V_{out}$$
$$C\frac{dv_C}{dt} = i_L - \frac{V_{out}}{R}$$

**关断状态**：
$$L\frac{di_L}{dt} = -V_{out}$$
$$C\frac{dv_C}{dt} = i_L - \frac{V_{out}}{R}$$

### 逆变器

**桥臂开关约束**：
$$S_1 + S_2 = 1$$
- 上下桥臂互补
- 防止直通

**死区**：
- 引入延迟
- 防止短路
