# 仿真实践与工程指南 (Simulation Practice and Engineering Guide)

## 概述

EMT仿真实践与工程指南是面向电力工程师和研究人员的实用操作手册，涵盖仿真方法选择、步长选择、收敛性保障、数值振荡抑制、结果分析等工程实践要点。良好的仿真实践习惯是获得可靠结果、提高工作效率、避免常见陷阱的关键。本指南汇总EMT仿真的最佳实践、常见问题诊断和解决方案。

仿真实践的核心目标：
- **结果可靠**：数值误差可控，结果物理合理
- **效率优化**：在合理时间内完成仿真
- **问题可诊断**：能快速定位和解决问题
- **流程标准化**：可重复、可追溯

## 作用机制

### 26.1 仿真方法选择

**模型复杂度选择**

| 研究目标 | 推荐模型 | 步长 | 计算时间 |
|---------|---------|------|---------|
| 概念验证 | 平均值模型 | 100-500μs | 秒级 |
| 控制策略 | 详细开关模型 | 20-50μs | 分钟级 |
| 保护整定 | 详细模型+故障 | 10-20μs | 小时级 |
| 设备应力 | 详细器件模型 | 1-10μs | 小时级 |
| 实时测试 | 恒导纳模型 | 1-10μs | 实时 |

**求解器选择**

梯形法（Trapezoidal）：
- 适用：大多数场景
- 优势：A-稳定、精度高
- 注意：可能产生数值振荡

后向欧拉（Backward Euler）：
- 适用：开关时刻、刚性系统
- 优势：L-稳定、强阻尼
- 注意：精度较低、数值阻尼

混合积分：
- 策略：正常用梯形，开关时刻切换至BE
- 实现：CDA（Critical Damping Adjustment）

### 26.2 步长选择准则

**理论约束**

最高频率限制：
$$h_{max} = \frac{1}{10 f_{max}}$$

最小时间常数：
$$h_{max} = \frac{\tau_{min}}{10}$$

**经验准则**

| 系统类型 | 推荐步长 | 说明 |
|---------|---------|------|
| 输电网 | 50-100μs | 线路行波传播 |
| 配电网 | 20-50μs | 配电变压器、负荷 |
| 两电平VSC | 10-20μs | 开关频率2kHz |
| MMC | 10-50μs | 子模块电容动态 |
| 详细IGBT | 0.1-1μs | 开关暂态 |

**自适应步长**

变步长策略：
- 稳态：大步长
- 暂态：减小步长
- 开关时刻：最小步长

注意：变步长可能影响实时性。

### 26.3 初始条件设置

**稳态初始化**

加载潮流结果：
- 电压幅值和相角
- 发电机有功/无功
- 负荷功率

**动态初始化**

状态变量初始值：
- 电感电流：$i_L(0) = i_{steady}$
- 电容电压：$v_C(0) = v_{steady}$
- 控制器积分器：清零或预置

**MMC初始化**

子模块电容电压：
$$v_{C,init} = \frac{V_{dc}}{N}$$

桥臂电流：
$$i_{arm,init} = \frac{I_{dc}}{3} + \frac{I_{ac}}{2}$$

### 26.4 收敛性问题诊断

**非线性不收敛**

症状：
- 迭代次数超限
- 误差持续增大
- 仿真报错停止

原因：
- 步长过大
- 非线性太强
- 初值不合理

解决：
- 减小步长
- 改进初值
- 使用阻尼迭代
- 分段线性化

**开关抖动**

症状：
- 开关状态频繁切换
- 计算时间剧增
- 结果不稳定

原因：
- 开关阈值设置不当
- 数值振荡
- 控制逻辑问题

解决：
- 设置开关死区（hysteresis）
- 使用插值开关
- 改进控制算法

**刚性问题**

症状：
- 小步长仍不稳定
- 快动态与慢动态共存
- 数值发散

原因：
- 时间尺度分离大
- 刚性比 > 1000

解决：
- 使用隐式积分（Gear、BDF）
- 多速率方法
- 模型降阶

### 26.5 数值振荡识别与抑制

**数值振荡特征**

梯形法振荡：
- 频率：$f_{osc} \approx 1/(2h)$
- 幅值：不衰减或缓慢衰减
- 触发：突变后（故障、开关）

物理振荡 vs 数值振荡：
| 特征 | 物理振荡 | 数值振荡 |
|-----|---------|---------|
| 频率 | 与系统参数相关 | ≈1/(2h) |
| 阻尼 | 由系统决定 | 梯形法无衰减 |
| 触发 | 扰动后 | 突变后 |

**抑制方法**

CDA（临界阻尼调整）：
- 检测突变
- 后续两步使用BE
- 恢复梯形法

人工阻尼：
- 梯形法+小阻尼
- 损失部分精度
- 换取稳定性

减小步长：
- 最直接有效
- 增加计算成本

**实践建议**

- 始终启用CDA
- 设置合理的开关插值
- 避免理想开关（使用小电阻/大电阻）
- 必要时切换至BE

### 26.6 结果验证与分析

**正确性验证**

能量守恒检查：
$$\Delta E = \int P_{gen} dt - \int P_{load} dt - \int P_{loss} dt \approx 0$$

稳态验证：
- 与潮流结果对比
- 功率平衡检查
- 电压电流合理性

**精度评估**

步长收敛性：
- 步长减半，结果变化<1%
- 证明步长足够小

特征量对比：
- 峰值误差<5%
- 时间常数误差<10%
- 频率误差<1%

**结果分析**

时域分析：
- 波形特征（峰值、超调、稳定时间）
- 频谱分析（FFT、谐波含量）
- 模态分析（Prony、阻尼比）

统计指标：
- THD（总谐波畸变率）
- RMS值
- 峰值系数

### 26.7 仿真优化技巧

**模型简化**

合理简化：
- 远方系统用等值
- 非关注设备用平均模型
- 短线路用集中参数

避免过度简化：
- 关键设备保留详细
- 关注频段保留
- 故障路径完整

**计算加速**

并行计算：
- 多核CPU并行
- GPU加速
- 分网并行

模型优化：
- 避免频繁重分解
- 使用恒导纳模型
- 减少不必要输出

**脚本自动化**

批处理仿真：
- 参数扫描
- 蒙特卡洛分析
- 优化寻优

结果后处理：
- 自动提取特征量
- 生成报告
- 绘制对比图

## 适用边界

- **EMT仿真**适用于电磁暂态（μs-ms级），机电暂态（s级）应使用专用软件
- **详细模型**计算成本高，大规模系统需配合模型降阶
- **数值稳定性**与模型、步长、算法均相关，需综合考虑
- **结果可信度**需通过验证确认，不能盲目相信
- **工程实践**需平衡精度与效率，过度追求精度可能不经济

## 典型问题速查

| 问题 | 可能原因 | 解决方案 |
|-----|---------|---------|
| 不收敛 | 步长过大、初值差 | 减小步长、改进初值 |
| 数值振荡 | 梯形法、突变后 | 启用CDA、减小步长 |
| 开关抖动 | 阈值不当、振荡 | 设置死区、插值开关 |
| 速度慢 | 模型复杂、步长小 | 模型简化、并行计算 |
| 内存不足 | 规模大、输出多 | 分网、减少输出 |
| 结果不合理 | 参数错、拓扑错 | 检查输入、简化验证 |

## 深度增强内容

### 1. 仿真故障诊断流程

```
仿真报错
  ↓
识别错误类型
  ├── 不收敛 → 检查步长、初值、非线性
  ├── 数值振荡 → 启用CDA、减小步长
  ├── 内存溢出 → 减小规模、减少输出
  └── 其他错误 → 查阅文档、寻求帮助
  ↓
定位问题
  ├── 简化模型测试
  ├── 分步调试
  └── 对比成功案例
  ↓
解决问题
  ↓
验证修复
```

### 2. 步长选择决策树

```
开始
  ↓
确定关注现象
  ├── 工频稳态 → 50-100μs
  ├── 开关暂态 → 10-20μs
  ├── 雷电/VFTO → 1-10μs
  └── 机电暂态 → EMT不适用
  ↓
检查稳定性
  ├── 不稳定 → 减小步长
  └── 稳定 → 继续
  ↓
检查精度
  ├── 步长减半，变化>1% → 继续减小
  └── 步长减半，变化<1% → 当前步长合适
```

### 3. 模型验证层次

| 层次 | 验证内容 | 通过标准 |
|-----|---------|---------|
| 单元 | 单个元件 | 解析解误差<0.1% |
| 模块 | 子系统 | 与参考对比<1% |
| 系统 | 完整系统 | 特征量误差<5% |
| 工程 | 实际对比 | 与实测<10% |

### 4. 常见陷阱与避免

| 陷阱 | 后果 | 避免方法 |
|-----|------|---------|
| 理想开关 | 虚假振荡 | 使用R-on/R-off |
| 理想电源 | 不收敛 | 加小阻抗 |
| 零初始条件 | 启动暂态 | 稳态初始化 |
| 过大步长 | 不稳定/不准 | 按准则选择 |
| 过度输出 | 内存溢出 | 只输出关注量 |
| 忽略CDA | 数值振荡 | 始终启用 |

### 5. 工程经验参数

| 场景 | 经验参数 | 说明 |
|-----|---------|------|
| 开关R-on | 0.001-0.01Ω | 避免理想开关 |
| 开关R-off | 1e6Ω | 避免理想开关 |
| 死区时间 | 1-10μs | 避免抖动 |
| 仿真裕度 | 20-30% | 实时仿真 |
| 收敛容差 | 1e-4-1e-6 | 相对误差 |
