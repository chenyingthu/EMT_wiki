# 数值稳定性 (Numerical Stability)

## 概述

数值稳定性(Numerical Stability)是电磁暂态仿真中的核心问题，涉及数值积分方法在求解微分方程时的误差累积、传播和收敛特性。稳定的数值方法能够确保仿真结果随时间推进保持有界，而不稳定的方法则会导致解发散或产生虚假振荡。数值稳定性分析是选择和设计仿真算法的基础，直接影响仿真结果的可靠性和计算效率。

在电力系统仿真中，数值稳定性问题尤为重要，因为电力系统具有刚性(Stiff)特性，即系统包含多个时间尺度差异巨大的动态过程。从微秒级的开关暂态到秒级的机电摇摆，时间常数比可达 $10^9$ 量级，这对数值积分方法提出了严峻挑战。

## 稳定性基本概念

### 测试方程

**标准测试方程**:
$$\dot{x} = \lambda x$$
其中 $\lambda \in \mathbb{C}$ 为复数。

**解析解**:
$$x(t) = x(0)e^{\lambda t}$$

**稳定性条件**:
- $Re(\lambda) < 0$: 稳定(解指数衰减)
- $Re(\lambda) > 0$: 不稳定(解指数增长)
- $Re(\lambda) = 0$: 临界稳定(等幅振荡)

### 数值解

**一般形式**:
$$x_{n+1} = R(z)x_n$$
其中：
- $z = \lambda \Delta t$: 无量纲参数
- $R(z)$: 稳定性函数

**稳定性条件**:
数值解稳定当且仅当：
$$|R(z)| \leq 1$$

**稳定性区域**:
复平面上使 $|R(z)| \leq 1$ 的区域：
$$S = \{z \in \mathbb{C} : |R(z)| \leq 1\}$$

### 稳定性分类

**零稳定性**:
当 $\Delta t \to 0$ 时方法的稳定性，与一致性相关。

**绝对稳定性**:
固定步长下的稳定性，由稳定性区域描述。

**相对稳定性**:
数值解与解析解的相对行为一致性。

## 稳定性类型

### A稳定性

**定义**:
数值方法对所有 $Re(z) < 0$ (即 $Re(\lambda) < 0$) 都稳定：
$$S \supseteq \mathbb{C}^-$$
其中 $\mathbb{C}^-$ 为左半平面。

**物理意义**:
- 无条件稳定
- 对任何稳定系统都适用
- 步长不受稳定性限制

**典型方法**:
- 后向欧拉法
- 梯形法
- Gear法(隐式多步法)

**优势**:
- 适合刚性系统
- 可采用大步长
- 计算效率高

### L稳定性

**定义**:
方法既是A稳定，又满足：
$$\lim_{z \to -\infty} |R(z)| = 0$$

**物理意义**:
- 高频模态快速衰减
- 无数值振荡
- 适合开关仿真

**典型方法**:
- 后向欧拉法
- 隐式Runge-Kutta方法

**与A稳定对比**:
梯形法是A稳定但不是L稳定：
$$\lim_{z \to -\infty} R_{trap}(z) = -1$$
后向欧拉是L稳定：
$$\lim_{z \to -\infty} R_{BE}(z) = 0$$

### 刚性稳定性

**刚性定义**:
系统同时包含快变和慢变模态：
$$\kappa = \frac{\max|Re(\lambda_i)|}{\min|Re(\lambda_i)|} \gg 1$$

**刚性稳定性要求**:
- 在左半平面大部分区域稳定
- 允许在正实轴附近有界不稳定区域
- 适合特定类型刚性问题

### 条件稳定性

**定义**:
稳定性依赖于步长大小：
$$\Delta t < \Delta t_{max}$$

**典型方法**:
- 前向欧拉法
- 显式Runge-Kutta法
- 显式多步法

**步长限制**:
$$\Delta t_{max} = \frac{c}{|Re(\lambda)|_{max}}$$
其中 $c$ 为与方法相关的常数。

## 数值积分方法稳定性

### 前向欧拉法

**稳定性函数**:
$$R(z) = 1 + z$$

**稳定区域**:
单位圆内：
$$|1 + z| \leq 1$$

**稳定性条件**:
$$\Delta t \leq \frac{2}{|\lambda|_{max}}$$

**特点**:
- 显式方法
- 一阶精度
- 条件稳定
- 适合非刚性系统

### 后向欧拉法

**稳定性函数**:
$$R(z) = \frac{1}{1 - z}$$

**稳定区域**:
整个左半平面，以(1,0)为圆心的单位圆外。

**特点**:
- 隐式方法
- 一阶精度
- L稳定
- 适合刚性系统

**数值阻尼**:
高频模态被强阻尼：
$$|R(-\infty)| = 0$$

### 梯形法

**稳定性函数**:
$$R(z) = \frac{1 + z/2}{1 - z/2}$$

**稳定区域**:
整个左半平面。

**特点**:
- 隐式方法
- 二阶精度
- A稳定但不是L稳定
- 无数值阻尼

**临界稳定问题**:
$$|R(-\infty)| = 1$$
高频模态不衰减，可能导致数值振荡。

### Gear法

**稳定性函数**:
多步法的稳定性函数较为复杂。

**阶数与稳定性**:
- 1-2阶：A稳定
- 3-6阶：刚性稳定
- 更高阶：不稳定

**特点**:
- 多步法
- 可变阶数
- 适合大规模刚性系统
- 需要启动过程

### Runge-Kutta法

**显式RK**:
- 条件稳定
- 稳定性区域有限
- 适合非刚性系统

**隐式RK**:
- A稳定或L稳定
- 计算量大
- 高精度

## 误差来源与传播

### 截断误差

**局部截断误差**:
单步引入的误差：
$$\tau_n = x(t_{n+1}) - x_{n+1}$$

**阶数与误差**:
- $p$ 阶方法：$\tau_n = O(\Delta t^{p+1})$
- 梯形法($p=2$)：$O(\Delta t^3)$
- 后向欧拉($p=1$)：$O(\Delta t^2)$

**全局误差**:
多步累积误差：
$$e_n = O(\Delta t^p)$$

### 舍入误差

**来源**:
- 浮点数表示精度限制
- 运算舍入
- 数据存储

**影响**:
- 误差累积
- 数值噪声
- 解的精度限制

**控制**:
- 高精度运算
- 数值稳定性好的算法
- 误差监控

### 误差传播

**误差方程**:
$$e_{n+1} = R(z)e_n + \tau_n$$

**放大因子**:
$$|e_{n+1}| \leq |R(z)||e_n| + |\tau_n|$$

**稳定性要求**:
$$|R(z)| < 1$$
确保误差不放大。

## 刚性系统

### 刚性来源

**多时间尺度**:
电力系统同时存在：
- 快模态：开关暂态($\tau \sim \mu s$)
- 中模态：电磁过程($\tau \sim ms$)
- 慢模态：机械摇摆($\tau \sim s$)

**刚性比**:
$$\kappa = \frac{\tau_{max}}{\tau_{min}} = \frac{L_{max}/R_{min}}{L_{min}/R_{max}}$$
电力系统 $\kappa$ 可达 $10^9$。

### 刚性对数值方法的要求

**显式方法的限制**:
步长受最快模态限制：
$$\Delta t < \tau_{min}$$
计算量巨大。

**隐式方法的优势**:
- 无条件稳定
- 步长仅受精度限制
- 可采用大步长

### 刚性求解策略

**隐式方法**:
- 后向欧拉
- 梯形法
- Gear法

**多速率仿真**:
不同区域采用不同步长。

**模型降阶**:
消除快模态，降低刚性。

## 数值振荡问题

### 振荡类型

**物理振荡**:
系统真实的动态响应。

**数值振荡**:
数值方法引入的虚假振荡。

### 梯形法数值振荡

**产生原因**:
- 临界阻尼为零
- 高频模态不衰减
- 开关引起的高频分量持续振荡

**表现形式**:
- 开关后电压/电流振荡
- 振荡幅度不衰减
- 频率与步长相关

**影响**:
- 虚假结果
- 收敛困难
- 精度下降

### 振荡解决方案

**1. 插值法**:
精确确定开关时刻：
$$t_{sw} = t_n + \frac{v_{th} - v_n}{v_{n+1} - v_n}\Delta t$$

**2. 临界阻尼调整(CDA)**:
开关步切换至后向欧拉。

**3. 变步长**:
开关附近使用小步长。

**4. 人工阻尼**:
添加小电阻或采用L稳定方法。

## 稳定性分析工具

### 特征值分析

**系统矩阵特征值**:
$$\dot{x} = Ax$$
特征值 $\lambda_i$ 决定系统模态。

**稳定性判据**:
$$Re(\lambda_i) < 0, \quad \forall i$$

**数值方法特征值**:
离散化后系统的特征值。

### 根轨迹

**参数变化影响**:
分析参数变化对稳定性的影响。

**应用**:
- 控制器设计
- 参数优化
- 稳定裕度分析

### 李雅普诺夫方法

**非线性稳定性**:
构造李雅普诺夫函数 $V(x)$：
- $V(x) > 0$ (正定)
- $\dot{V}(x) < 0$ (负定)

**应用**:
- 非线性系统稳定性
- 大范围稳定性
- 吸引域估计

## 最佳实践

### 方法选择

**刚性系统**:
- 后向欧拉(开关)
- 梯形法(一般)
- Gear法(大规模)

**非刚性系统**:
- 显式RK(高精度)
- 前向欧拉(简单)

**混合系统**:
- CDA方法
- 自适应方法
- 多速率仿真

### 步长选择

**稳定性限制**:
条件稳定方法需满足：
$$\Delta t < \Delta t_{stab}$$

**精度限制**:
由截断误差决定：
$$\Delta t < \Delta t_{acc}$$

**实际选择**:
$$\Delta t = \min(\Delta t_{stab}, \Delta t_{acc})$$

### 验证方法

**能量守恒**:
保守系统能量应守恒。

**稳态检查**:
稳态解应与理论一致。

**收敛性测试**:
减小步长，检查收敛。
