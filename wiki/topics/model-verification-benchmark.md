# 模型验证与基准测试 (Model Verification and Benchmarking)

## 概述

模型验证与基准测试是EMT仿真质量保证的核心环节，通过标准测试系统、跨工具对比和误差评估，确保仿真模型的正确性和结果的可信度。验证回答"模型是否正确"，基准测试回答"结果是否可比"。在电力系统EMT仿真中，验证和基准测试是模型工程化应用、学术论文发表、标准制定的必备步骤。

验证与基准测试的核心问题：
- **正确性验证**：模型数学推导、实现代码是否正确
- **精度验证**：数值误差是否在可接受范围
- **一致性验证**：跨工具结果是否一致
- **收敛性验证**：步长减小结果是否收敛
- **物理合理性**：结果是否符合物理规律

## 作用机制

### 20.1 验证层次体系

**Level 1: 单元测试（Unit Testing）**

单个元件验证：
- **R/L/C元件**：稳态响应、时间常数
- **开关**：导通/关断特性、时序
- **电源**：幅值、频率、相位

验证方法：
$$\text{误差} = \frac{|V_{sim} - V_{analytical}|}{|V_{analytical}|} \times 100\%$$

目标：误差<0.1%

**Level 2: 子系统测试（Subsystem Testing）**

功能模块验证：
- **变压器**：变比、饱和特性
- **线路**：波阻抗、传播时间
- **换流器**：开关函数、谐波含量

验证方法：频域对比、稳态潮流、暂态响应

**Level 3: 系统测试（System Testing）**

完整系统验证：
- **小型测试系统**：IEEE 14节点、39节点
- **标准测试案例**：CIGRE基准、IEEE标准
- **实际系统对比**：现场故障录波

### 20.2 CIGRE标准测试系统

**CIGRE 850MHz基准系统**

目的：验证电力电子模型和开关控制

拓扑：
- 12脉波换流器
- 交流滤波器
- 直流线路

测试场景：
- 整流侧交流故障
- 逆变侧交流故障
- 直流线路故障
- 换相失败

验收标准：
| 特征量 | 允许误差 |
|-------|---------|
| 直流电压峰值 | ±5% |
| 故障电流峰值 | ±10% |
| 恢复时间 | ±20% |

**CIGRE WG B4.57 MMC测试系统**

目的：验证MMC模型和控制系统

拓扑：
- 401电平MMC-HVDC
- 两端系统
- 架空线/电缆混合

测试场景：
- 单相接地故障
- 三相短路故障
- 直流侧故障
- 闭锁/解锁过程

**CIGRE 2025新型电力系统基准**

目的：高比例新能源接入验证

特征：
- 100%电力电子电源
- 构网型/跟网型混合
- 宽频振荡风险

### 20.3 IEEE标准验证流程

**IEEE Std 1204-1997**

指导文件：EMTP应用指南

验证要求：
- 与解析解对比
- 与现场测量对比
- 收敛性分析

**IEEE PES技术报告**

基准测试系列：
- 变压器饱和模型
- 线路参数模型
- 电弧模型
- 负荷模型

**验证流程**

```
1. 建立标准测试系统
2. 设定参考基准（解析解或实测）
3. 运行仿真并记录结果
4. 计算误差指标
5. 误差分析
6. 模型修正（如有需要）
7. 文档化验证报告
```

### 20.4 跨工具对比方法

**对比流程**

1. **系统建模**：各工具建立相同拓扑
2. **参数统一**：确保参数一致
3. **求解器设置**：步长、算法统一
4. **仿真运行**：执行相同工况
5. **结果提取**：统一格式导出
6. **对比分析**：误差计算和波形对比

**误差评估指标**

波形误差：
$$\epsilon_{rms} = \sqrt{\frac{\sum_{i=1}^{N}(y_i^{ref} - y_i^{sim})^2}{\sum_{i=1}^{N}(y_i^{ref})^2}} \times 100\%$$

特征量误差：
$$\epsilon_{feature} = \frac{|F_{ref} - F_{sim}|}{|F_{ref}|} \times 100\%$$

其中 $F$ 为特征量（峰值、时间常数、频率等）。

**接受准则**

| 应用场景 | RMS误差 | 特征量误差 |
|---------|---------|-----------|
| 概念研究 | <10% | <15% |
| 工程分析 | <5% | <10% |
| 保护整定 | <2% | <5% |
| 学术研究 | <3% | <5% |

### 20.5 收敛性验证

**步长收敛测试**

测试步长序列：
$$h_n = \frac{h_0}{2^n}, \quad n = 0, 1, 2, 3, 4$$

收敛准则：
$$|y_{h_n} - y_{h_{n+1}}| < \epsilon_{tol}$$

**时间常数测试**

激励信号：
- 阶跃响应
- 脉冲响应
- 正弦扫频

验证时间常数：
$$\tau_{sim} \approx \tau_{analytical}$$

**数值稳定性测试**

刚性系统测试：
- 长输电线（L/C比大）
- 高开关频率系统
- 快动态与慢动态共存

判据：
- 无虚假振荡
- 无数值发散
- 能量守恒

### 20.6 物理合理性检验

**能量守恒检验**

总能量变化：
$$\Delta E = \int_0^T P_{gen}(t)dt - \int_0^T P_{load}(t)dt - \int_0^T P_{loss}(t)dt$$

判据：$|\Delta E| < \epsilon_{tol}$

**无源性检验**

端口功率：
$$P(t) = v(t) \cdot i(t) \geq 0, \quad \forall t$$

负功率出现可能预示数值不稳定。

**因果性检验**

输入-输出时延：
$$y(t) \text{ 不应超前于 } u(t)$$

非因果响应指示模型或求解器问题。

## 适用边界

- **解析解对比**仅适用于简单线性系统，复杂非线性系统需数值基准
- **跨工具对比**假设至少一个工具结果是正确的，可能共同错误无法发现
- **实测对比**受测量误差影响，高频分量可能无法准确记录
- **收敛性测试**计算成本高，全系统测试可能不实际
- **标准测试系统**不能覆盖所有应用场景，特定问题需自定义验证

## 深度增强内容

### 1. 验证检查清单

**建模阶段**：
- [ ] 拓扑与原理图一致
- [ ] 参数与铭牌/设计值一致
- [ ] 单位统一（p.u.或实际值）
- [ ] 基准值明确

**仿真阶段**：
- [ ] 步长满足稳定性要求
- [ ] 总仿真时间覆盖关注暂态
- [ ] 输出变量完整
- [ ] 初始条件正确

**验证阶段**：
- [ ] 与解析解/基准对比
- [ ] 误差在允许范围
- [ ] 收敛性验证通过
- [ ] 物理合理性检查通过

### 2. 误差来源分析

| 误差来源 | 影响 | 控制方法 |
|---------|------|---------|
| 截断误差 | 与步长相关 | 减小步长/高阶算法 |
| 舍入误差 | 浮点精度 | 双精度/高精度算法 |
| 模型误差 | 简化假设 | 改进模型/基准对比 |
| 参数误差 | 输入不准确 | 参数辨识/灵敏度分析 |
| 数值振荡 | 算法不稳定 | CDA/阻尼算法 |

### 3. 基准测试库建议

**必备测试集**：
| 测试名称 | 目的 | 通过标准 |
|---------|------|---------|
| 简单RL电路 | 基础验证 | 误差<0.1% |
| 单相线路 | 行波模型 | 传播时间误差<1% |
| 变压器饱和 | 非线性 | 饱和特性匹配 |
| 三相短路 | 故障暂态 | 峰值误差<5% |
