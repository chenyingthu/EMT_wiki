# 电磁暂态 (Electromagnetic Transient)

## 概述

电磁暂态(Electromagnetic Transient, EMT)是电力系统中由开关操作、故障、雷击等扰动引起的快速电磁过程，时间尺度通常在微秒到毫秒级。与机电暂态不同，电磁暂态关注电压、电流的快速变化过程，涉及行波传播、高频振荡、过电压等复杂现象。准确分析电磁暂态对于设备绝缘配合、保护装置设计、系统安全运行具有重要意义。随着电力电子设备大量应用和电网复杂度提高，电磁暂态现象更加复杂，需要借助高精度的数值仿真工具进行深入研究。

## 暂态类型与时间尺度

### 雷电暂态

**时间尺度**:
- 波头时间: 1.2μs
- 波尾时间: 50μs
- 频率范围: 10kHz-10MHz

**雷电流波形**:
$$i(t) = I_m(e^{-\alpha t} - e^{-\beta t})$$

典型参数:
- 幅值: 10-200kA
- 波头陡度: 10-100kA/μs

**过电压机理**:
- 直击雷过电压
- 感应雷过电压
- 侵入波过电压

### 开关暂态

**时间尺度**:
- 断路器操作: 0.1-10ms
- 隔离开关: 0.1-1ms
- 频率范围: 100Hz-100kHz

**主要现象**:
- 截流现象
- 重燃过电压
- 预击穿

**典型过电压**:
- 切除空载变压器过电压
- 切除空载线路过电压
- 合闸过电压

### 故障暂态

**时间尺度**:
- 故障发生: 0-1ms
- 直流分量衰减: 10-100ms
- 频率范围: 工频-1kHz

**故障类型**:
- 单相接地
- 相间短路
- 三相短路

**暂态分量**:
- 工频稳态分量
- 直流衰减分量
- 高频振荡分量

## 数学描述

### 基本方程

**麦克斯韦方程组**:
$$\nabla \times \mathbf{E} = -\frac{\partial \mathbf{B}}{\partial t}$$
$$\nabla \times \mathbf{H} = \mathbf{J} + \frac{\partial \mathbf{D}}{\partial t}$$

**电报方程**:
$$\frac{\partial V}{\partial x} = -L\frac{\partial I}{\partial t} - RI$$
$$\frac{\partial I}{\partial x} = -C\frac{\partial V}{\partial t} - GV$$

### 行波理论

**波动方程**:
$$\frac{\partial^2 V}{\partial x^2} = LC\frac{\partial^2 V}{\partial t^2}$$

**行波速度**:
$$v = \frac{1}{\sqrt{LC}} \approx \frac{c}{\sqrt{\varepsilon_r \mu_r}}$$

架空线典型值: 接近光速(300m/μs)
电缆典型值: 150m/μs

**波阻抗**:
$$Z_c = \sqrt{\frac{L}{C}}$$
架空线: 300-500Ω
电缆: 20-75Ω

**波的反射与折射**:
$$\Gamma = \frac{Z_2 - Z_1}{Z_2 + Z_1}$$
$$T = \frac{2Z_2}{Z_2 + Z_1}$$

## 主要暂态现象

### 过电压

**内部过电压**:
- 操作过电压
- 故障过电压
- 谐振过电压

**外部过电压**:
- 直击雷
- 感应雷
- 地闪

**过电压倍数**:
$$K = \frac{V_{max}}{V_{normal}}$$
典型值: 2.0-3.5 p.u.

### 过电流

**短路电流**:
$$i(t) = I_m\sin(\omega t + \alpha - \varphi) - I_m\sin(\alpha - \varphi)e^{-t/\tau}$$

**冲击电流**:
- 变压器励磁涌流
- 电容器合闸涌流
- 电机启动电流

### 振荡现象

**铁磁谐振**:
$$V = I \cdot \omega L(I)$$
非线性电感与电容谐振

**次同步谐振(SSR)**:
- 串补线路与发电机组相互作用
- 轴系扭振
- 电气谐振频率: 15-45Hz

**高频振荡**:
- 电力电子开关
- 分布参数谐振
- EMI/EMC问题

## 数值仿真方法

### 时域仿真

**集中参数模型**:
- R、L、C元件
- 状态空间方程
- 常微分方程求解

**分布参数模型**:
- Bergeron模型
- 频率相关模型
- 行波模型

**数值积分方法**:
- 梯形法(Trapezoidal)
- 后向欧拉法
- Gear法

### 伴随电路法

**电感伴随模型**:
$$i_L(t) = G_{eq}v_L(t) + I_{eq}$$
$$G_{eq} = \frac{\Delta t}{2L}, \quad I_{eq} = i_L(t-\Delta t) + G_{eq}v_L(t-\Delta t)$$

**电容伴随模型**:
$$i_C(t) = G_{eq}v_C(t) + I_{eq}$$
$$G_{eq} = \frac{2C}{\Delta t}, \quad I_{eq} = -i_C(t-\Delta t) - G_{eq}v_C(t-\Delta t)$$

### 开关建模

**理想开关**:
- 导通: 零电阻

- 插值处理

**实际开关**:

- 暂态恢复电压(TRV)
- 熄弧能力
