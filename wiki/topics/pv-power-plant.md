# 光伏电站 (PV Power Plant)

## 概述

光伏电站(Photovoltaic Power Plant)是利用光伏效应将太阳能直接转换为电能的发电设施，是可再生能源发电的重要组成部分。光伏电站具有模块化、分布式、运维简单、无燃料成本等显著优势，近年来在全球范围内快速发展。根据规模和并网方式，光伏电站可分为集中式大型电站和分布式电站两大类。随着光伏技术进步和成本下降，光伏已成为许多地区最经济的新增电源。然而，光伏发电的间歇性和波动性也给电网运行带来了挑战，需要通过先进的控制技术、储能系统和智能调度来应对。

## 系统组成

### 光伏阵列

**光伏组件**:
- 单晶硅: 效率20-22%，成本高
- 多晶硅: 效率16-18%，成本适中
- 薄膜: 效率10-12%，成本低，弱光性好

**组件连接**:
- 串联: 提高电压
- 并联: 提高电流
- 串并联组合形成阵列

**阵列配置**:
- 组串式: 10-20块组件串联
- 阵列规模: kW-MW级
- 容量配比: 1.0-1.3倍逆变器容量

**跟踪系统**:
- 固定式: 最佳倾角安装
- 单轴跟踪: 东西向跟踪，增益10-15%
- 双轴跟踪: 全向跟踪，增益20-30%

### 逆变器系统

**集中式逆变器**:
- 容量: 500kW-8MW
- 集中控制，管理简单
- 大型电站首选

**组串式逆变器**:
- 容量: 50-250kW
- 每串独立MPPT
- 适合复杂地形

**集散式逆变器**:
- 容量: 1-3MW
- 多路MPPT
- 集中逆变+分散优化

**微型逆变器**:
- 容量: 0.25-2kW
- 组件级控制
- 适合分布式

### 升压变压器

**箱式变电站**:
- 逆变器输出: 270-800V AC
- 升压至: 10-35kV
- 集成变压器、开关、保护

**主变压器**:
- 电压等级: 35kV/110kV/220kV
- 容量: 与电站规模匹配
- 有载调压: ±8×1.25%

### 汇流与配电

**直流汇流箱**:
- 组串汇流: 12-24路
- 防反二极管
- 防雷保护

**交流配电**:
- 开关柜
- 无功补偿
- 滤波装置

## 控制策略

### MPPT控制

**扰动观察法(P&O)**:
$$\Delta P > 0, \Delta V > 0 \Rightarrow \text{继续增加电压}$$
$$\Delta P < 0 \Rightarrow \text{反向扰动}$$

**电导增量法(IncCond)**:
$$\frac{dI}{dV} = -\frac{I}{V} \Rightarrow \text{最大功率点}$$

**改进算法**:
- 变步长
- 模糊控制
- 神经网络

### 并网控制

**电流源控制**:
- 锁相环同步
- 电流内环
- 功率外环

**有功控制**:

**无功控制**:

## 建模与仿真

### 光伏电池模型

**单二极管模型**:
$$I = I_{ph} - I_0\left(e^{\frac{V+IR_s}{aV_t}} - 1\right) - \frac{V+IR_s}{R_{sh}}$$

**简化模型**:
$$I = I_{sc}\left(1 - e^{\frac{V-V_{oc}}{V_t}}\right)$$

**参数特性**:
- $I_{sc}$: 短路电流(与辐照度成正比)
- $V_{oc}$: 开路电压(受温度影响)
- $P_{max}$: 最大功率点
