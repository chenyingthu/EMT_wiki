# 负荷与分布式电源建模 (Load and Distributed Generation Modeling)

## 定义与概述

电力系统负荷是消耗电能的设备总和，分布式电源（DG）是接入配电网的小型发电单元。准确的负荷与DG模型对电压稳定分析、电能质量评估、保护配合和系统规划至关重要。在EMT仿真中，负荷与DG模型需要同时考虑基频响应和宽频暂态特性，涵盖从毫秒级开关暂态到秒级机电暂态的多时间尺度动态。

**负荷与DG建模的核心挑战**：
- **负荷多样性**：居民/商业/工业负荷特性差异巨大，配电网负荷组成随时间、季节和地域变化
- **电压频率特性**：负荷功率随电压和频率非线性变化，恒功率负荷在电压跌落时的电流特性与恒阻抗完全不同
- **DG控制复杂**：逆变器控制、MPPT、低电压穿越等动态，以及电力电子接口引入的宽频谐波特性
- **集群效应**：大规模DG并网的交互影响，多个逆变器之间的谐波耦合与交互稳定性
- **多时间尺度耦合**：从微秒级开关动作到分钟级调度指令，各时间尺度动态相互耦合

## 在EMT仿真中的角色

### 为什么需要负荷与DG综合建模

EMT仿真中，负荷与DG模型决定了仿真系统对电压/频率扰动的响应精度。主要挑战包括：

1. **终端设备类型多样**：现代配电网负荷包含开关电源（SMPS）、LED驱动、变频电机、恒温控制负荷等，聚合特性差异大
2. **非线性动态行为**：电机脱扣/重合闸、电力电子整流前端在电压暂降/恢复过程中呈现复杂的非线性响应
3. **频率依赖特性**：感应电机转速随频率变化，进而影响有功功率，在低惯量系统中频率相关负荷对UFLS整定有重要影响
4. **电压敏感非线性**：恒功率负荷在电压跌落时的电流特性与恒阻抗完全不同，需要精确的等效电路建模
5. **IBR宽频响应**：光伏/风电/储能的逆变器接口会引入数百Hz至数kHz的谐波交互，与传统电机负荷的动态截然不同

### 关键需求

| 需求 | 描述 |
|------|------|
| 动态响应精度 | 在电压暂降/恢复过程中准确复现电机脱扣、负荷恢复等动态过程 |
| 频率敏感性 | 在低惯量系统频率摆动中反映负荷频率依赖 |
| 多时间尺度 | 覆盖开关级暂态（μs）、电磁暂态（ms）、机电暂态（s） |
| 计算效率 | 在EMT步长下可求解，支持长时间尺度仿真 |
| 参数可辨识 | 负荷组成比例可从现场录波数据中提取 |
| DG控制保真度 | 保留逆变器PLL、电流环、LVRT等控制动态 |

## 负荷模型体系

### 静态负荷模型（ZIP模型）

ZIP负荷模型通过三个多项式项描述有功和无功功率随电压的变化关系，是最广泛使用的静态负荷模型（见[[load-model]]）：

$$P = P_0 \left[a_p\left(\frac{V}{V_0}\right)^2 + b_p\left(\frac{V}{V_0}\right) + c_p\right]$$

$$Q = Q_0 \left[a_q\left(\frac{V}{V_0}\right)^2 + b_q\left(\frac{V}{V_0}\right) + c_q\right]$$

其中归一化系数满足 $a + b + c = 1$：$a$ 为恒阻抗（Z）比例，$b$ 为恒电流（I）比例，$c$ 为恒功率（P）比例。

**EMT离散实现（恒导纳伴随电路）**：
$$G_{eq} = G_0 \left[a\left(\frac{V}{V_0}\right) + b + c\left(\frac{V_0}{V}\right)\right]$$

历史电流源用于保持非线性特性：
$$I_{hist} = I_{ZIP}(V(t-\Delta t), f)$$

**典型参数**（IEEE Std. 3002.2）：

| 负荷类型 | $a_p$ | $b_p$ | $c_p$ | $a_q$ | $b_q$ | $c_q$ |
|---------|-------|-------|-------|-------|-------|-------|
| 居民 | 0.5 | 0.3 | 0.2 | 0.0 | 0.0 | 1.0 |
| 商业 | 0.4 | 0.3 | 0.3 | 0.3 | 0.3 | 0.4 |
| 工业 | 0.2 | 0.2 | 0.6 | 0.1 | 0.2 | 0.7 |
| 农业 | 0.7 | 0.2 | 0.1 | 0.6 | 0.3 | 0.1 |

### 频率依赖负荷模型（fdLoad）

频率相关负荷在低惯量系统中对频率稳定性有重要影响。fdLoad模型（Fini等 2026）将指数型频率相关负荷转化为等效导纳网络，使其可直接接入节点方程：

$$P(f,V) = P_0 \left(\frac{f}{f_0}\right)^{\alpha_f} \left(\frac{V}{V_0}\right)^{\alpha_v}$$

$$Q(f,V) = Q_0 \left(\frac{f}{f_0}\right)^{\beta_f} \left(\frac{V}{V_0}\right)^{\beta_v}$$

**向等效导纳的转化**（在 $V \approx V_0$ 条件下）：
$$Y(f) = \frac{P(f) - jQ(f)}{V^2}$$

| 频率指数 | 感应电机 | 综合负荷 | 照明 | 加热 |
|---------|---------|---------|------|------|
| $\alpha_f$（有功） | 2.0–3.0 | 1.0–2.0 | 0–0.5 | 0 |
| $\beta_f$（无功） | 0–1.0 | -1.0–0 | 0 | 0 |

### 动态感应电机负荷模型

感应电机是配电网中最重要的动态负荷组成部分，其EMT模型需要保留电磁与机械动态的耦合（见[[induction-machine-model]]）：

**dq0坐标下的动态方程**：

电压方程：
$$v_{ds} = R_s i_{ds} + \frac{d\psi_{ds}}{dt} - \omega_s \psi_{qs}$$
$$v_{qs} = R_s i_{qs} + \frac{d\psi_{qs}}{dt} + \omega_s \psi_{ds}$$

磁链方程（线性磁路假设）：
$$\psi_{ds} = L_s i_{ds} + L_m i_{dr}$$
$$\psi_{qs} = L_s i_{qs} + L_m i_{qr}$$
$$\psi_{dr} = L_m i_{ds} + L_r i_{dr}$$
$$\psi_{qr} = L_m i_{qs} + L_r i_{qr}$$

转子运动方程：
$$T_m = T_{load} + 2H \frac{ds}{dt}$$

其中 $s = (\omega_s - \omega_r)/\omega_s$ 为滑差，$H$ 为惯性时间常数。

**电压稳定性临界条件**：
$$V_{crit} = 0.6 - 0.7 \text{ p.u.}, \quad s_{stall} = 0.1 - 0.2$$

### 详细元件级综合负荷模型

Milani（2021）提出的详细元件级负荷模型将配电网母线级负荷分解为若干可辨识终端设备，在EMTP中用子电路层级架构进行聚合（见[[load-model]]）：

**元件组成**：
- 单相/三相感应电机（直接驱动机械负载）
- 开关电源（SMPS）：等效电阻 $R_{eq} = v_{dc}^2 / P_{rated}$
- LED驱动器：阈值电压 $V_{thr} = 2.7$ V，动态电阻 $R_d = 0.3$ Ω
- 紧凑型荧光灯（CFL）：充电态 $R_{eq,charging} = 11v_{dc} + 0.021v_{dc}^2 + 1300$ Ω
- 变频驱动器（VSI）：PWM矢量控制
- 静态电阻负荷

**聚合等效参数**：
$$P_{eq} = \sum_i P_i, \quad Q_{eq} = \sum_i Q_i$$
$$Z_{eq} = \frac{V^2}{P_{eq} - jQ_{eq}}$$
$$H_{eq} = \frac{\sum_i H_i S_i}{\sum_i S_i}$$

**电机保护逻辑（现场校准参数）**：
- 脱扣电压阈值：额定电压的45%–55%
- 脱扣持续时间：1–3个工频周波
- 重合闸延时：电压恢复至~65%后20 ms
- 重合闸成功率：70%的电机负荷在重合闸后20 ms内完成重连

## 分布式电源EMT建模

### 光伏系统EMT建模

光伏系统EMT建模涵盖光伏电池物理模型和逆变器控制模型（见[[pv-system-model]]）：

**光伏电池单二极管模型**：

$$I = I_{ph} - I_0 \left[\exp\left(\frac{q(V+IR_s)}{nkT}\right) - 1\right] - \frac{V+IR_s}{R_{sh}}$$

参数温度依赖：
- $I_{ph}(T) = I_{ph,ref} [1 + \alpha_{sc}(T - T_{ref})] \frac{G}{G_{ref}}$
- $I_0(T) = I_{0,ref} (\frac{T}{T_{ref}})^3 \exp[\frac{qE_g}{nk}(\frac{1}{T_{ref}} - \frac{1}{T})]$

**逆变器控制模型（双环控制）**：

直流侧动态：
$$C_{dc} \frac{dV_{dc}}{dt} = I_{pv} - I_{dc}$$

交流侧PWM调制：
$$v_{abc} = m_{abc} \cdot \frac{V_{dc}}{2}$$
$$I_{dc} = \frac{3}{4} \sum_{k=a,b,c} m_k i_k$$

外环功率/直流电压控制：
$$i_{d,ref} = k_p(V_{dc,ref} - V_{dc}) + \int k_i(V_{dc,ref} - V_{dc})dt$$
$$i_{q,ref} = -Q_{ref}/(1.5V_d)$$

PLL同步：
$$\omega = \omega_0 + k_{p,pll} \cdot v_q + k_{i,pll} \int v_q dt$$

**LVRT要求**：

| 电压 | 要求 |
|------|------|
| V≥90% | 正常运行 |
| 90%>V≥20% | 低电压穿越，不脱网 |
| 20%>V | 允许脱网 |

无功支撑：电压跌落期间提供无功电流，$I_q \geq 1.5(0.9 - V_{pu})I_N$

### 双馈风机（DFIG）EMT建模

DFIG定子直接并网，转子通过背靠背换流器励磁（见[[dfig-model]]）：

**DFIG控制结构**：

转子侧换流器（RSC）控制：
- 定子磁链定向矢量控制
- 有功/无功解耦控制
- 最大功率点跟踪（MPPT）
- Crowbar保护（故障时旁路转子侧换流器）

网侧换流器（GSC）控制：
- 直流电压控制
- 电网电压支撑
- 功率因数控制

**Crowbar保护动作时间**：10–50 ms，决定故障穿越能力

**次同步振荡风险**：DFIG经串补电网存在次同步相互作用风险，需在EMT中保留详细控制动态进行分析

### 永磁同步风机（PMSG）EMT建模

PMSG通过全功率变换器并网，无齿轮箱（见[[pmsm-model]]）：

**全功率变换器结构**：
- 机侧换流器：发电机转矩控制
- 网侧换流器：直流电压控制 + 电网支撑
- 直流母线电容：稳定直流电压

**构网型控制**（Grid-Forming）：
- 虚拟同步机（VSG）控制
- 下垂控制
- 提供电网电压和频率支撑

### 电池储能系统（BESS）EMT建模

**电池储能（BESS）电化学模型**：

开路电压与SOC关系：
$$V_{oc}(SOC) = V_{oc,0} + k_1 \cdot SOC + k_2 \cdot \ln(SOC) + k_3 \cdot \ln(1-SOC)$$

等效内阻模型：
$$R_{int}(SOC, T) = R_0 + R_1/SOC + R_2(1-SOC)$$

充放电效率：
$$\eta_{ch} = 0.85-0.95, \quad \eta_{dis} = 0.90-0.98$$

**变流器模型**（见[[average-value-model]]）：
- 双向DC/DC变换器
- 并网逆变器（PWM或SVPWM）
- SOC管理控制（防止过充过放）

**超级电容**：

$$V_{SC}(t) = V_0 + \frac{1}{C_{SC}} \int_0^t I_{SC}(\tau) d\tau$$

特点：大功率密度（5–10 kW/kg）、快速响应（<0.1 s）、循环寿命长（>100000次）

### 风电场等值聚合

多机风电场聚合方法（见[[wind-farm-modeling]]）：

**聚合维度**：
- 按风速分群（风速差异导致出力不同）
- 按发电机类型分群（DFIG/PMSG/异步机）
- 保留关键动态特性（Crowbar动作、MPPT响应）

**等值模型参数**：
$$S_{eq} = \sum_{i=1}^{N} S_i, \quad H_{eq} = \frac{\sum_{i=1}^{N} H_i S_i}{S_{eq}}$$

**聚合误差控制**：误差<5%需按风速分群至少3–5群；集群光伏并网的交互效应需在等值模型中保留至少10%的详细机

## 关键技术挑战

### 挑战1：电机脱扣与重合闸的精确模拟

电机接触器在电压暂降时的脱扣阈值（45%–55%额定电压）和重合闸延时（20 ms）是影响负荷恢复动态的关键参数。需要根据现场录波数据对每个地区的具体阈值进行校准，而非使用通用整定值。

**现场校准方法**（Milani 2021）：
1. 施加实测电压暂降波形到聚合负荷模型
2. 比较仿真与实测的三相电流、有功/无功功率波形
3. 迭代调整脱扣阈值、延时和负荷组成比例
4. 验证电机接触器脱扣和重合闸产生的暂态冲击电流

### 挑战2：电力电子负荷的等效处理

传统模型将电力电子负荷简化为恒功率（CMLD方法），但实际开关电源在电压暂降时从直流侧电容汲取的功率是电压依赖的。详细元件级模型通过等效电阻公式捕捉这种非线性：

$$R_{eq} = \frac{v_{dc}^2}{P_{rated}} \quad \text{(SMPS)}$$

### 挑战3：大规模DG并网的交互稳定性

大规模光伏/风电并网时，多个逆变器之间的交互可能引发谐波振荡（数百Hz至数kHz）。需要保留逆变器详细控制动态（PLL带宽、电流环响应）进行EMT宽频扫描分析。

**PLL带宽影响**：PLL带宽过宽可能引发振荡，建议PLL带宽设置在10–50 Hz范围。

### 挑战4：IBR频率支撑特性

光伏/风电逆变器在低惯量系统中可提供虚拟惯量支撑（Virtual Synchronous Control），但其频率响应特性与传统电机完全不同，需要构网型控制（Grid-Forming）而非跟网型控制（Grid-Following）。

## 量化性能边界

### 负荷模型精度对比

| 模型类型 | 电压精度 | 频率精度 | 计算效率 | 适用场景 |
|---------|---------|---------|---------|---------|
| ZIP静态 | ±5% | 不支持 | 极高 | 潮流计算、稳态分析 |
| fdLoad | ±3% | 支持 | 高 | 低惯量系统频率分析 |
| 感应电机 | ±1% | 支持 | 中 | 电压稳定、暂态分析 |
| 详细元件级 | ±0.5% | 支持 | 低 | 大扰动分析、负荷恢复 |

### DG模型效率对比

| DG类型 | 详细模型步长 | 等值模型步长 | 加速比 | 精度损失 |
|-------|------------|------------|--------|---------|
| 单机光伏 | 1–10 μs | 50–100 μs | 5–10× | <2% |
| 单机DFIG | 10–50 μs | 100–500 μs | 5–10× | <5% |
| 风电场(10台) | 20–100 μs | 200–1000 μs | 10–50× | 5–10% |
| BESS(储能) | 10–50 μs | 100–500 μs | 5–10× | <3% |

### 负荷特性量化数据

- 综合负荷频率敏感系数 $k_{pf}$ 在1.0–2.0范围，夏季空调负荷可达3.0
- ZIP模型参数随电压等级变化，配电网比输电网的恒阻抗比例更高
- 电力电子负荷比例增加导致电压特性硬化（更趋近恒功率）
- 光伏逆变器控制带宽影响电网稳定性，PLL带宽过宽可能引发振荡
- DFIG Crowbar保护动作时间在10–50 ms范围
- 风电场等值聚合误差<5%需按风速分群至少3–5群
