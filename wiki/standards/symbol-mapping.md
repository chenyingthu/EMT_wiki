## 标准公式模板

### 电感电压-电流关系

$$v = L \frac{\mathrm{d}i}{\mathrm{d}t}$$

### 电容电流-电压关系

$$i = C \frac{\mathrm{d}v}{\mathrm{d}t}$$

### 状态空间标准形式

$$\frac{\mathrm{d}\mathbf{x}}{\mathrm{d}t} = \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{u}$$

### 节点电压方程

$$\mathbf{Y}\mathbf{v} = \mathbf{i}$$

## 排版规范

1. **微分符号**: 使用 $\mathrm{d}$ 而非 $d$ (正体)
2. **虚数单位**: 使用 $\mathrm{j}$ 而非 $j$ (正体)
3. **向量表示**: 使用 $\mathbf{x}$ 而非 $\vec{x}$ (粗体)
4. **矩阵表示**: 使用 $\mathbf{A}$ (粗体大写)
5. **单位间距**: 数字与单位之间加 $\,$ 间距
