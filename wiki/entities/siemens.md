# Siemens

## 概述

Siemens（西门子）是工业制造与技术服务公司，在电力系统领域拥有超过170年的历史。在高压直流输电（HVDC）和柔性交流输电（FACTS）技术领域有长期积累，开发了Siemens HVDC技术和SIPLINK®等产品。在EMT仿真领域，Siemens提供详细的设备模型和工程验证数据，与PSCAD/EMTDC、EMTP-RV等仿真工具有合作。

## 核心贡献

### 1. HVDC技术

**Siemens HVDC技术体系**
- **HVDC Classic**: 传统LCC-HVDC技术
- **HVDC Plus**: VSC-HVDC技术
- **HVDC MaxSine**: 下一代VSC技术

**工程成就**
- 全球超过100个HVDC工程
- 总输送容量超过100,000MW
- 覆盖欧洲、亚洲、美洲、非洲

### 2. LCC-HVDC技术

**晶闸管换流器系统**
- SIEMENS技术平台：SINCAL、NETOMAC
- 双12脉波换流器配置
- 高可靠性控制系统

**典型工程**
- **Nelson River Bipole I/II (加拿大)**
  - ±450kV/1620MW + 1800MW
  - 1970s投运，多次扩容升级
  - 与曼尼托巴大学密切合作

- **Pacific Intertie (美国)**
  - ±500kV/3100MW
  - 1970年投运，2000s升级
  - 美国西海岸电力输送主动脉

- **高岭背靠背 (中国)**
  - ±125kV/1500MW
  - 东北-华北异步联网
  - 世界上容量最大的背靠背工程

### 3. VSC-HVDC技术 (HVDC Plus)

**HVDC Plus®技术**
- 模块化多电平换流器(MMC)技术
- 2010年首次应用
- 品牌名称：HVDC Plus

**技术特点**
- 黑启动能力
- 弱系统连接
- 无功功率独立控制
- 支持多端口直流电网

**代表性工程**
- **Trans Bay Cable (美国)**
  - ±200kV/400MW
  - 2010年投运
  - 首个MMC-HVDC商业化应用

- **Nordlink (德国-挪威)**
  - ±525kV/1400MW
  - 2020年投运
  - 连接德国风电与挪威水电

- **Ultranet (德国)**
  - ±525kV/2000MW
  - 与ABB合作
  - 架空线柔直

### 4. FACTS与电力电子

**SVC技术**
- SVC Light®
- 晶闸管控制电抗器(TCR)
- 静止同步补偿器(STATCOM)

**电力电子变压器 (SST)**
- SST (Solid State Transformer)
- 中压直流配电网应用
- EMT仿真验证平台

## 技术演进脉络

### 1847-1950年 (电气工业先驱)
- **Siemens创立 (1847)**
  - Werner von Siemens创办
  - 电气工程领域多项发明

- **发电机与电动机**
  - 世界第一台实用发电机
  - 电气铁路技术

### 1950-1970年 (HVDC探索)
- **HVDC研究起步**
  - 汞弧阀换流器研究
  - 1967年首个HVDC工程（德国本土）

### 1970-1990年 (LCC-HVDC成熟)
- **晶闸管时代 (1970s)**
  - 取代汞弧阀
  - 更高可靠性和效率

- **Nelson River工程 (1970s)**
  - 与加拿大合作
  - 奠定Siemens HVDC技术基础

- **中国早期合作 (1980s)**
  - 舟山直流工程
  - 技术引进与培训

### 1990-2010年 (技术转型)
- **NETOMAC软件 (1990s)**
  - 电力系统仿真软件
  - 电磁暂态仿真功能
  - 后并入PowerFactory

- **SVC技术发展**
  - SVC Light®
  - FACTS商业化

- **VSC技术探索 (2000s)**
  - IGBT换流器研发
  - HVDC Plus®技术开发

### 2010-2020年 (MMC时代)
- **HVDC Plus®商业化 (2010)**
  - Trans Bay Cable投运
  - MMC技术成熟

- **欧洲电网互联**
  - Nordlink (德-挪)
  - ALEGrO (德-比)
  - 海上风电送出项目

### 2020年至今 (直流电网与新能源)
- **海上风电主导**
  - 欧洲海上风电柔直送出
  - 亚洲海上风电市场

- **直流电网技术**
  - 多端直流保护
  - 直流断路器研发

- **固态变压器 (SST)**
  - 电力电子变压器
  - 中压直流配电网

### 2. HVDC Plus技术详解

#### 2.1 MMC拓扑

**子模块配置**
- 半桥子模块（标准配置）
- 全桥子模块（可选）
- 混合拓扑（特定应用）

**技术参数**
- 电压等级：±320kV至±525kV
- 容量：500MW至2000MW
- 损耗：<1%（满载）

#### 2.2 控制策略

**分层控制**
```
系统级：功率调度
  ↓
换流站级：有功/无功控制
  ↓
换流器级：电压平衡控制
  ↓
阀级：PWM调制
```

**EMT仿真模型**
- 详细开关模型
- 平均值模型
- 控制系统详细逻辑
