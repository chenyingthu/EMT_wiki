# ABB

## 概述

ABB（Asea Brown Boveri）是全球领先的电力与自动化技术公司，在高压直流输电（HVDC）、电力电子设备制造领域拥有百年历史。作为HVDC技术的开创者之一，ABB参与了全球超过一半的HVDC工程建设，其换流器模型是EMT仿真的重要验证基准。在699篇EMT论文中，ABB作为HVDC设备制造商和技术标准制定参与者被多次提及。

## 核心贡献

### 1. HVDC技术发展

**HVDC技术先驱**
- **商业HVDC开创者 (1954)**
  - 瑞典Gotland岛HVDC工程（世界上第一个商业HVDC工程）
  - 20MW, 90kV, 海底电缆

- **技术演进领导者**
  - 汞弧阀时代 → 晶闸管时代 → IGBT时代
  - LCC-HVDC → VSC-HVDC → MMC-HVDC

**工程成就**
- 全球超过120个HVDC工程
- 总输送容量超过130,000MW
- 覆盖所有电压等级（最高±1100kV）

### 2. LCC-HVDC技术

**晶闸管换流器**
- 12脉波、双12脉波换流器设计
- 换流变设计与制造
- 换流阀技术

**控制保护系统**
- ABB MACH2控制系统
- 换相失败预测控制
- 直流功率调制

**EMT模型贡献**
- 提供标准LCC-HVDC仿真模型
- 换流阀详细开关模型
- 控制系统仿真模块

### 3. VSC/MMC-HVDC技术

**HVDC Light®技术**
- ABB的VSC-HVDC品牌
- 1997年首次应用（瑞典Hellsjön）
- 采用两电平/三电平换流器

**HVDC Light® Evolution**
- 模块化多电平换流器(MMC)技术
- 支持±320kV至±525kV
- 容量可达1000MW+

**EMT建模贡献**
- 子模块详细模型
- 换流器平均值模型
- 控制保护系统模型

### 4. 电力电子与FACTS

**SVC技术**
- 世界上第一台商用SVC（1971年）
- TCR (晶闸管控制电抗器)
- TSC (晶闸管投切电容器)

**STATCOM**
- 电压源型静止无功补偿器
- 基于VSC技术
- 快速无功调节

**EMT模型**
- 提供标准FACTS设备模型
- 与PSCAD/EMTDC、EMTP-RV集成

## 技术演进脉络

### 1890-1950年 (前身公司发展)
- **ASEA (瑞典)**
  - 高压输电技术先驱
  - 1893年：第一台三相发电机
  
- **BBC (Brown Boveri, 瑞士)**
  - 电气设备制造
  - 高压开关技术

### 1950-1970年 (HVDC开创)
- **Gotland HVDC (1954)**
  - 世界首个商业HVDC
  - 汞弧阀技术
  - 奠定HVDC工程基础

- **英吉利海峡 (1961)**
  - 英法国际联网
  - 160MW, ±100kV

### 1970-1990年 (晶闸管时代)
- **晶闸管换流器 (1970s)**
  - 取代汞弧阀
  - 可靠性大幅提升
  - 容量和电压等级提高

- **中国早期HVDC**
  - 葛洲坝-南桥（1989）
  - 舟山直流（1986）
  - 与中方技术合作

### 1990-2010年 (VSC革命)
- **HVDC Light® (1997)**
  - VSC-HVDC商业化
  - 两电平换流器
  - 适用于弱系统连接

- **Shetland联网 (1999)**
  - 海上风电送出
  - 弱系统联网

- **Estlink (2006)**
  - 爱沙尼亚-芬兰
  - 330MW, ±150kV

### 2010-2020年 (MMC时代)
- **MMC技术成熟 (2010+)**
  - 模块化多电平技术
  - 容量和电压大幅提升

- **Ultranet (2019)**
  - 德国±525kV/2000MW
  - 架空线柔直

- **张北柔直 (2020)**
  - 中国四端柔直电网
  - ABB提供核心设备

### 2020年至今 (直流电网)
- **直流电网技术**
  - 多端直流保护
  - 直流断路器
  - 直流电网控制

- **海上风电送出**
  - Dogger Bank项目
  - 全球首个±525kV海上柔直

### 2. EMT仿真模型体系

#### 2.1 LCC-HVDC模型

**详细模型**
- 换流阀详细开关模型
- 换流变饱和特性
- 控制系统详细逻辑
- 保护系统详细模型

**平均值模型**
- 换流器平均值等效
- 控制系统简化
- 适用于系统级仿真

**验证基准**
- 基于实际工程参数
- 与现场测试数据对比验证
- 误差<2%（典型值）

#### 2.2 VSC/MMC模型

**子模块模型**
- 两电平子模块（早期HVDC Light）
- 半桥子模块（HVDC Light Evo）
- 全桥子模块（可选）

**控制模型**
- 矢量控制（dq0框架）
- 电压控制环
- 电流控制环
- 环流抑制控制
