# 离散化方法 (Discretization Methods)

## 定义与概述

离散化方法（Discretization Methods）是将连续时间动态系统转换为离散时间代数方程的数学技术，是电磁暂态（EMT）仿真的核心理论基础。在EMT仿真中，所有动态元件（电感、电容、线路、变压器、电机）的微分方程必须通过离散化转化为等效的代数方程，才能在每个时间步长内求解。离散化方法的选择直接决定了仿真的数值稳定性、精度阶数、计算效率和适用场景，是连接连续物理世界与离散数字仿真的关键桥梁。

## 1. 理论基础

### 1.1 连续到离散的转换

**基本问题**：给定连续时间微分方程
$$\frac{dx(t)}{dt} = f(x(t), u(t))$$

将其转换为离散形式：
$$x_{k+1} = F(x_k, u_k, u_{k+1}, h)$$

其中$h$为步长，$x_k$为第$k$步的状态。

**泰勒展开基础**：
$$x(t+h) = x(t) + h\dot{x}(t) + \frac{h^2}{2}\ddot{x}(t) + \frac{h^3}{6}\dddot{x}(t) + O(h^4)$$

**局部截断误差（LTE）**：
$$\text{LTE} = x(t_{k+1}) - x_{k+1} = O(h^{p+1})$$

其中$p$为方法阶数。

### 1.2 EMT常用离散化方法对比

| 方法 | 离散公式 | 精度 | 稳定性 | 数值阻尼 | 典型应用 |
|------|----------|------|--------|----------|----------|
| **前向欧拉（FE）** | $x_{k+1} = x_k + hf(t_k, x_k)$ | 1阶 | 条件稳定 | 无 | 简单预估，非刚性系统 |
| **后向欧拉（BE）** | $x_{k+1} = x_k + hf(t_{k+1}, x_{k+1})$ | 1阶 | L稳定 | 强 | 刚性系统、开关事件后 |
| **梯形法（TR）** | $x_{k+1} = x_k + \frac{h}{2}[f_k + f_{k+1}]$ | 2阶 | A稳定 | 无 | 标准EMT仿真 |
| **Gear法（BDF）** | $\sum_{j=0}^{k}\alpha_j x_{n+j} = h\beta_k f_{n+k}$ | 2-6阶 | 刚性稳定 | 可调 | 多步长、刚性系统 |
| **2S-DIRK** | 分阶段隐式 | 2阶 | L稳定 | 可控 | 电力电子、实时仿真 |

**稳定性区域**：
- **A稳定**：对所有$\text{Re}(\lambda) < 0$稳定，允许任意大步长
- **L稳定**：A稳定 + $\lim_{\lambda \to -\infty} R(\lambda h) = 0$，完全阻尼高频

### 1.3 伴随电路模型（Companion Model）

**统一形式**：
$$i_{k+1} = G_{eq} v_{k+1} + I_{hist}$$

其中：
- $G_{eq}$：等效导纳（由元件参数和积分方法决定）
- $I_{hist}$：历史电流源（汇总前一时步信息）

| 元件 | 梯形法离散化 | $G_{eq}$ | $I_{hist}$ |
|------|-------------|----------|-----------|
| **电感L** | $i_{k+1} = i_k + \frac{h}{2L}(v_k + v_{k+1})$ | $\frac{h}{2L}$ | $i_k + \frac{h}{2L}v_k$ |
| **电容C** | $v_{k+1} = v_k + \frac{h}{2C}(i_k + i_{k+1})$ | $\frac{2C}{h}$ | $-i_k - \frac{2C}{h}v_k$ |
| **电阻R** | 静态 | $\frac{1}{R}$ | $0$ |

## 2. EMT仿真应用

### 2.1 电感元件离散化

**梯形法伴随模型**：
$$i_L(t) = \frac{h}{2L}v_L(t) + I_{hist}$$

其中历史电流源：
$$I_{hist} = i_L(t-h) + \frac{h}{2L}v_L(t-h)$$

**后向欧拉伴随模型**（更强稳定性）：
$$i_L(t) = \frac{h}{L}v_L(t) + I_{hist}, \quad I_{hist} = i_L(t-h)$$

**物理意义**：电感离散化为等效电阻$R_{eq} = \frac{2L}{h}$（梯形法）与电流源并联。

**典型参数**：
- $L = 0.1$ H，$h = 50$ μs
- 梯形法：$R_{eq} = 4$ kΩ
- 后向欧拉：$R_{eq} = 2$ kΩ

### 2.2 电容元件离散化

**梯形法伴随模型**：
$$i_C(t) = \frac{2C}{h}v_C(t) + I_{hist}$$

其中历史电流源：
$$I_{hist} = -i_C(t-h) - \frac{2C}{h}v_C(t-h)$$

**物理意义**：电容离散化为等效电阻$R_{eq} = \frac{h}{2C}$与电流源并联。

**MMC子模块电容应用**：
- 子模块电容$C_{SM} = 10$ mF
- 步长$h = 20$ μs
- 等效电阻：$R_{eq} = 1$ Ω

### 2.3 输电线路离散化

**Bergeron模型**（无损线）：

特征线方程离散化：
$$i_k(t) = \frac{1}{Z_c}v_k(t) + I_{k,hist}$$

其中$Z_c = \sqrt{L'/C'}$为波阻抗，历史电流源：
$$I_{k,hist} = -\frac{1}{Z_c}v_m(t-\tau) - i_m(t-\tau)$$

**传播延时**：
$$\tau = l\sqrt{L'C'} = \frac{l}{v_{wave}}$$

| 线路类型 | 长度 | 波速 | 延时 | 离散处理 |
|----------|------|------|------|----------|
| 架空线 | 100 km | 300 km/ms | 0.33 ms | 长延时独立存储 |
| 电缆 | 10 km | 150 km/ms | 0.067 ms | 短延时插值处理 |

### 2.4 变压器离散化

**集总参数模型**：
- 漏感：按电感离散化
- 励磁电感：非线性需分段线性化
- 绕组电阻：静态

**移频仿真变压器离散化**（2022 高仕林）：
复包络域离散化：
$$\frac{dx_E(t)}{dt} = f_E(t) - j\omega_c x_E(t)$$

梯形法离散：
$$x_{E,k+1} = x_{E,k} + \frac{h}{2}[f_{E,k} + f_{E,k+1}] - j\omega_c \frac{h}{2}[x_{E,k} + x_{E,k+1}]$$

**优势**：步长可增大5-10倍（复包络频率远低于原信号）。

### 2.5 开关器件离散化

**分段线性化处理**（2006 GENE方法）：

将非线性V-I特性分段线性近似：
- 导通状态：$R_{on} \approx 0.01$ Ω
- 关断状态：$R_{off} \approx 10^6$ Ω

**预计算策略**：
枚举所有开关状态组合（$n$个开关有$2^n$种状态），预先计算每种状态对应的导纳矩阵。

**12脉动换流器示例**：
- 6个晶闸管 × 2状态 × 3个电抗器状态 = 192种组合
- 运行时直接查表，避免在线LU分解

## 3. 实现技术

### 3.1 数值积分方法选择

**刚性检测与自适应切换**：
```
非刚性区域 → 梯形法（2阶精度）
       ↓ 检测到刚性（步长受限）
刚性区域 → 后向欧拉/2S-DIRK（L稳定）
       ↓ 开关事件
事件后 → 两步BE（抑制振荡）→ 恢复TR
```

**变步长控制**：
$$h_{new} = h_{old} \cdot \left(\frac{\epsilon}{\text{LTE}}\right)^{1/(p+1)}$$

其中$\epsilon$为容差，$p$为方法阶数。

### 3.2 多速率离散化

**分区策略**：
```
快变子系统（电力电子）：小步长 h_f = 10μs
                      方法：后向欧拉/DIRK
慢变子系统（交流网络）：大步长 h_s = 100μs  
                      方法：梯形法
接口插值：线性/二次插值
```

**插值公式**（慢→快）：
$$u_f(t) = u_s(t_n) + \frac{t-t_n}{h_s}[u_s(t_{n+1}) - u_s(t_n)]$$

### 3.3 移频离散化

**复数信号构造**：
$$x_S(t) = x(t) + j\mathcal{H}\{x(t)\}$$

**移频变换**：
$$x_E(t) = x_S(t)e^{-j\omega_c t}$$

**离散化优势**：
| 参数 | 传统EMT | 移频EMT | 改善 |
|------|---------|---------|------|
| 信号最高频率 | 1-10 kHz | <10 Hz | 100-1000倍 |
| 典型步长 | 10-50 μs | 100-500 μs | 5-10倍 |
| 求解复杂度 | $O(n^3)$ | $O(n^3)$ | 复数运算 |

**IEEE 9241节点测试**：复数形式8.15s vs 矩阵形式21.59s（2.65倍加速）。

### 3.4 伴随模型实现优化

**恒导纳矩阵策略（ADC）**：
约束开关等效电感电容乘积：
$$L_{sw}C_{sw} = h^2$$

确保开关状态变化时系统导纳矩阵保持不变，避免每步重分解。

**预计算存储**（GENE方法）：
```c
// 预计算阶段
for each switch_state_combination:
    Y_matrix[state_id] = build_admittance_matrix(state)
    LU_factor[state_id] = lu_decompose(Y_matrix[state_id])

// 运行阶段
current_state = detect_switch_state()
Y = Y_matrix[current_state]  // 直接查表
```

## 4. 仿真软件实现

### 4.1 EMTP/PSCAD实现

**标准离散化流程**：
```fortran
! 电感离散化（梯形法）
G_EQ = DT / (2.0 * L)
I_HIST = I_L_OLD + G_EQ * V_L_OLD
I_L_NEW = G_EQ * V_L_NEW + I_HIST

! 电容离散化（梯形法）
G_EQ = 2.0 * C / DT
I_HIST = -I_C_OLD - G_EQ * V_C_OLD
I_C_NEW = G_EQ * V_C_NEW + I_HIST

! 组装节点导纳矩阵
Y(NODE_I, NODE_I) = Y(NODE_I, NODE_I) + G_EQ
J(NODE_I) = J(NODE_I) - I_HIST
```

**开关处理**：
- 导通：$R_{sw} = R_{on}$
- 关断：$R_{sw} = R_{off}$
- 状态变化时更新导纳矩阵并重分解

### 4.2 MATLAB/Simulink实现

**ode15s（刚性系统求解器）**：
```matlab
% 定义DAE系统
function dx = power_system_dae(t, x)
    % 状态方程离散化
    dx = A*x + B*u(t);
end

% 配置求解器（变阶BDF）
options = odeset('RelTol', 1e-4, 'AbsTol', 1e-6, ...
                 'MaxStep', 1e-4, 'BDF', 'on');
[t, x] = ode15s(@power_system_dae, [0 0.1], x0, options);
```

### 4.3 实时仿真器实现

**RTDS固定步长离散化**：
- 预计算所有可能导纳矩阵
- 查表获取LU因子（<1μs）
- 固定步长50μs，单步计算<40μs

**FPGA离散化**（RTL级）：
```verilog
// 电感离散化流水线
always @(posedge clk) begin
    // 第1级：读取历史值
    i_hist <= i_reg + (h/(2*L)) * v_reg;
    // 第2级：计算新电流
    i_new <= (h/(2*L)) * v_new + i_hist;
    // 第3级：更新寄存器
    i_reg <= i_new;
    v_reg <= v_new;
end
```

## 7. 适用边界与限制

### 7.1 适用条件

**离散化方法有效场景**：
- **连续动态系统**：可用微分方程描述的元件和网络
- **步长约束满足**：$h < h_{max}$（稳定性限制）
- **精度要求匹配**：LTE在可接受范围内

**方法选择指南**：
| 场景 | 推荐离散化方法 | 理由 |
|------|---------------|------|
| 非刚性系统 | 梯形法 | 2阶精度，无数值阻尼 |
| 刚性系统 | 后向欧拉/DIRK | L稳定，大步长稳定 |
| 开关事件 | CDA（两步BE） | 抑制数值振荡 |
| 实时仿真 | 固定步长BE | 确定性计算时间 |
| 多时间尺度 | 多速率/移频 | 效率优化 |

### 7.2 失效边界

**数值不稳定信号**：
- 结果发散或振幅指数增长
- 开关事件后持续高频振荡（梯形法非L稳定导致）
- 步长减小后误差反而增大

**刚性限制**：
- 刚性比$S > 10^6$：即使L稳定方法也需极小步长
- 开关频率>10kHz：离散化误差累积

### 7.3 精度边界

| 离散化方法 | 稳态误差 | 暂态误差 | 数值阻尼 |
|------------|----------|----------|----------|
| 前向欧拉 | $O(h)$ | $O(h)$ | 无 |
| 后向欧拉 | $O(h)$ | $O(h)$ | 强 |
| 梯形法 | $O(h^2)$ | $O(h^2)$ | 无（振荡风险） |
| Gear(2阶) | $O(h^2)$ | $O(h^2)$ | 中等 |
| 2S-DIRK | $O(h^2)$ | $O(h^2)$ | 可控 |

**误差控制建议**：
- 保护整定研究：使用梯形法，误差<1%
- 系统级暂态：CDA处理开关，误差<5%
- 实时仿真：牺牲精度保稳定性，误差<10%
