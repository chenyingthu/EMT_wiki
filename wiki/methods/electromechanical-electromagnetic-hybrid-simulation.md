# 机电-电磁暂态混合仿真 (Electromechanical-Electromagnetic Hybrid Simulation)

## 定义与概述

机电-电磁暂态混合仿真（简称机电-电磁混合仿真）是将大规模电力系统划分为机电暂态侧（基频相量域）和电磁暂态侧（时域瞬时域），通过接口技术实现两侧协同仿真的技术。机电侧采用准稳态相量模型和大步长（10ms级）仿真大规模交流电网的动态稳定性，电磁侧采用瞬时-value模型和小步长（50μs级）详细模拟电力电子设备（HVDC、FACTS、MMC等）的开关暂态过程。混合仿真技术有效解决了大规模交直流混联电网仿真中"规模-速度-精度"三者不可兼得的矛盾，是目前含高比例电力电子设备电网仿真分析的主流方案。

## 1. 理论基础

### 1.1 混合仿真的基本原理

**替代定理与系统分网**:
根据替代定理，任何网络都可以在其端口处用等效电源替代而不影响外部电路的电压电流分布。混合仿真将系统划分为：
- **机电暂态侧**：大规模交流电网，采用基频相量模型，求解发电机转子运动方程和网络代数方程
- **电磁暂态侧**：需要详细研究的电力电子装置及其邻近网络，采用三相瞬时值模型，求解微分-代数方程组

**核心矛盾**:
| 特性 | 机电暂态侧 | 电磁暂态侧 |
|------|-----------|-----------|
| 时间尺度 | 10-100 ms | 10-100 μs |
| 步长 | 1-10 ms | 20-50 μs |
| 模型形式 | 基频相量 | 三相瞬时值 |
| 适用场景 | 大规模系统稳定性 | 局部设备开关暂态 |
| 网络规模 | 10,000+节点 | 100-500节点 |

**时间尺度差异**: 机电侧步长与电磁侧步长之比通常为 200:1（如10ms vs 50μs）

### 1.2 接口技术的四大核心问题

**1. 等值形式选择**:
机电侧对电磁侧的等值模型决定了外部网络在边界处的电气特性表示：

- **戴维南等值（Thevenin Equivalent）**:
  $$\dot{U}_{eq} = \dot{U}_{bus} + Z_{eq}\dot{I}_{bus}$$
  适用于电压源型接口，等值阻抗$Z_{eq}$需考虑正序、负序、零序不对称特性

- **诺顿等值（Norton Equivalent）**:
  $$\dot{I}_{eq} = \dot{I}_{bus} + Y_{eq}\dot{U}_{bus}$$
  适用于电流源型接口，电磁侧将机电侧等值为电流源并联导纳

- **频率相关网络等值（FDNE）**:
  $$Z_{FDNE}(s) = \sum_{i=1}^{n} \frac{c_i}{s-a_i} + d + se$$
  通过有理函数拟合表征宽频特性（0.1Hz-2kHz），克服基频等值对谐波响应的不足

- **多端口等值**:
  $$Y_{red} = Y_{11} - Y_{12}Y_{22}^{-1}Y_{21}$$
  通过高斯消去将外部网络降阶至接口母线，保留端口间耦合关系

**2. 数据转换（相量提取）**:
电磁侧时域波形需转换为机电侧可用的基频相量：

- **离散傅里叶变换（DFT）**:
  $$X(k) = \sum_{n=0}^{N-1} x(n)e^{-j2\pi kn/N}$$
  基频相量提取，数据窗长1-2个周期（20-40ms）

- **dq-120变换（改进型）**:
  $$\begin{bmatrix} v_d \\ v_q \end{bmatrix} = \frac{2}{3} \begin{bmatrix} \cos\theta & \cos(\theta-120°) & \cos(\theta+120°) \\ -\sin\theta & -\sin(\theta-120°) & -\sin(\theta+120°) \end{bmatrix} \begin{bmatrix} v_a \\ v_b \\ v_c \end{bmatrix}$$
  实时提取基波正序分量，结合Prony算法可在故障后5个采样点（0.1ms）内精确提取

- **矩阵束方法（Matrix Pencil）**:
  $$x(t) = \sum_{i=1}^{M} A_i e^{(\sigma_i + j\omega_i)t} + B$$
  适用于含高谐波畸变（40% THD）和直流偏移的工况，基波幅值误差<0.3%

**3. 接口位置选择**:

| 接口位置 | 优势 | 劣势 | 适用场景 |
|---------|------|------|----------|
| 换流母线处 | 简单直接，端口数少 | 谐波畸变严重，波形不对称 | 简单HVDC接入 |
| 向交流侧延伸1-3条母线 | 畸变衰减，精度提高 | 端口数增加，等值复杂 | 多馈入直流系统 |
| 电气距离>3-5节点处 | 波形质量高，误差<2% | 电磁侧网络扩大 | 非对称故障仿真 |

**4. 交互时序设计**:

- **串行时序（Serial）**:
  ```
  机电侧计算 → 数据交互 → 电磁侧计算 → 数据交互 → 下一周期
  ```
  无动态时延误差，但实时性差，难以满足实时仿真要求

- **并行时序（Parallel）**:
  ```
  机电侧迭代计算 ←→ 电磁侧独立计算
       ↓                    ↓
     非整数周期信息交换（每交互周期3-5次迭代）
  ```
  满足实时性要求（每步长计算时间<步长值），接口功率偏差可收敛至<0.1%

### 1.3 多速率仿真机制

**时间尺度协调**:
机电侧步长$T_{me}$与电磁侧步长$T_{em}$的比例通常为 200:1：
- 机电侧：$T_{me}$ = 10 ms（典型值）
- 电磁侧：$T_{em}$ = 50 μs（典型值）

**插值与保持**:
- **电磁侧→机电侧**: 在一个机电侧步长内，电磁侧计算200个步长，需提取等效相量
- **机电侧→电磁侧**: 机电侧等值电源在电磁侧多个步长内保持，或采用线性插值

**稳定性条件**:
接口时延导致的相位误差需控制在可接受范围：
- 串行时序相位滞后：$\Delta\phi = 2\pi f \Delta t$，10ms步长约18°
- 并行时序通过预测校正可将相位误差降至<2°

## 2. EMT仿真应用

### 2.1 HVDC与FACTS装置仿真

**应用场景**:
- 高压直流输电系统故障恢复过程仿真
- 换流器换相失败分析
- FACTS装置（SVC、STATCOM、TCSC）动态响应测试
- VSC-HVDC控制策略验证

**接口等值设计**:
- 机电侧：大规模交流电网的FDNE等值，覆盖0.1Hz-2kHz频带
- 电磁侧：12脉波/6脉波换流器详细模型，含阀组、触发控制、保护逻辑

**量化成果**:
- ADPSS平台支持1,000台发电机、10,000节点规模
- 相比全电磁暂态仿真，计算效率提升100-1000倍
- 仿真精度误差<2%（与全EMT对比）

### 2.2 MMC-HVDC大规模接入仿真

**技术挑战**:
MMC子模块数量庞大（单端400+子模块），全电磁暂态仿真规模受限（RTDS约100节点）

**混合仿真架构**:
- 电磁侧：MMC阀组详细模型，含子模块电容电压均衡控制
- 机电侧：大规模交流电网，含新能源汇集站

**平台实现**:
- RT-LAB平台：单核CPU实现10,000节点机电-电磁混合实时仿真
- RTDS+FPGA：电磁侧采用CBuilder自定义建模，FPGA实现数字化交互

### 2.3 新能源大规模并网仿真

**应用需求**:
风电场（1000+台DFIG/PMSG）、光伏电站详细建模与电网稳定性协同分析

**分网策略**:
- 电磁侧：风电场详细模型（含变流器控制、保护）
- 机电侧：区域电网及远端大系统

**FDNE优势**:
- 传统诺顿等值功率计算误差：8-12%（多馈入场景）
- 多端口FDNE等值误差：<1%

### 2.4 非对称故障仿真

**技术难点**:
非对称故障时接口处存在负序、零序分量，基频等值无法准确表征

**解决方案**:
- 多端口戴维南等值考虑正负零序阻抗不对称
- 节点分裂法或电流补偿法处理导纳矩阵非对称
- 接口位置选择：故障点与接口母线电气距离>3-5节点时误差<2%

### 3.2 FDNE实现流程

**算法步骤**:
```
1. 频率范围选取: 1-2 kHz（覆盖12脉波HVDC特征谐波）
2. 采样点设置: N0=500个频点
3. 元件频率特性计算: 发电机/负荷/线路/变压器简化模型
4. 网络导纳矩阵形成: 各频率下组装并收缩至边界节点
5. 整体矢量拟合: 公共极点策略，迭代3-4次收敛
6. 无源性校正: 摄动法保证G(s)矩阵正定
7. 时域实现: 状态空间或递推卷积形式
```

**无源性约束**:
$$P(s) = u^T G(s) u > 0$$
保证FDNE在任何频率下只吸收能量而不产生能量，是时域仿真稳定性的数学保证

### 3.3 相量提取算法优化

**改进dq-120算法**:

```python
## 伪代码：结合Prony算法的改进dq-120相量提取
def improved_dq120_extract(v_abc, theta, sampling_rate):
    # 1. 标准dq变换
    v_dq = park_transform(v_abc, theta)
    
    # 2. Prony分析提取基波分量（故障后快速收敛）
    if fault_detected:
        prony_result = prony_decompose(v_dq, order=2)
        v_fundamental = extract_fundamental(prony_result)
    else:
        v_fundamental = moving_average(v_dq, window=1_cycle)
    
    return v_fundamental
```

## 7. 适用边界与限制

### 7.1 适用条件
- **大规模交直流混联电网**: 需同时考虑电网稳定性与电力电子装置动态
- **电力电子设备接入分析**: HVDC、FACTS、大规模新能源并网
- **故障仿真**: 直流故障、换相失败、非对称故障
- **控制策略验证**: 需要详细电力电子控制模型与电网动态联合验证

### 7.2 失效边界
- **极高频率现象**: 频带超过FDNE范围（>2-3kHz）的谐波难以准确表征
- **强非线性控制**: 某些复杂非线性控制策略可能导致接口不稳定
- **接口位置不当**: 电气距离过近（<3-5节点）导致等值误差>2%
- **实时性不足**: 串行时序无法满足实时仿真要求

### 7.3 精度边界
| 误差来源 | 典型误差 | 缓解方法 |
|---------|---------|----------|
| 基频等值谐波忽略 | 8-12%（多馈入） | 采用FDNE |
| 相量提取时延 | 18°（串行时序） | 并行时序+预测校正 |
| 非对称故障 | >5%（接口处） | 接口向交流侧延伸 |
| 数值稳定性 | 可能发散 | 无源性强制 |
