# 电力系统稳定器 (PSS - Power System Stabilizer)

## 概述

电力系统稳定器(Power System Stabilizer, PSS)是同步发电机励磁控制系统的重要组成部分，通过向励磁系统注入辅助控制信号来提供额外的正阻尼，有效抑制电力系统的低频振荡，提高系统的小信号稳定性和动态性能。

### PSS的核心作用

在现代电力系统中，同步发电机通过励磁系统调节端电压。然而，高增益的自动电压调节器(AVR)虽然能快速响应电压变化，但会引入负阻尼转矩分量，导致系统在受到扰动后产生持续甚至增幅的低频振荡(0.1-2.0 Hz)。PSS通过引入与转子速度偏差相关的补偿信号，抵消励磁系统产生的负阻尼，使系统获得足够的正阻尼。

### 发展历史

- **1960s**: 早期PSS概念提出，基于比例-微分控制
- **1970s**: IEEE制定PSS标准模型(PSS1A, PSS2A)
- **1990s**: 多频段PSS(PSS4B)问世，适应更宽频率范围
- **2000s至今**: 智能PSS、自适应PSS和广域PSS(Wide-Area PSS)研究

## PSS类型详解

### IEEE PSS1A - 单输入型PSS

PSS1A是早期广泛使用的标准模型，采用单一输入信号（通常为转速偏差）和单级超前-滞后补偿结构。

#### 传递函数

```
        sT_w          1 + sT_1     1 + sT_3
V_PSS = ──── × K_PSS × ────────── × ────────── × Δω
       1 + sT_w       1 + sT_2     1 + sT_4
```

其中：
- $K_{PSS}$: PSS增益
- $T_w$: 隔直时间常数（通常5-10秒）
- $T_1, T_2, T_3, T_4$: 超前-滞后时间常数
- $T_1/T_2$ 和 $T_3/T_4$: 超前-滞后比

#### 特点
- **结构简单**: 参数整定直观
- **单一输入**: 仅依赖转速或功率信号
- **适用性**: 本地模式振荡抑制效果良好
- **局限**: 对区域间振荡效果有限

#### 框图表示

```
Δω ──>[Washout]──>[K_PSS]──>[Lead-Lag 1]──>[Lead-Lag 2]──>[Limiter]──>V_PSS
          │           │            │              │            │
         sTw        增益       (1+sT1)/(1+sT2) (1+sT3)/(1+sT4)  ±Vmax
        ─────                    相位超前       相位超前
        1+sTw
```

---

### IEEE PSS2B - 双输入型PSS

PSS2B采用双输入信号（转速和功率），通过信号合成产生等效加速功率信号，具有更好的阻尼特性。

#### 信号合成原理

PSS2B的核心创新在于使用机械功率和电功率信号合成加速功率信号：

$$P_{accel} = \frac{M}{\omega_0} \frac{d\omega}{dt} = P_{mech} - P_{elec}$$

其中：
- $M$: 惯性常数
- $\omega_0$: 额定角频率
- $P_{mech}$: 机械功率
- $P_{elec}$: 电磁功率

#### 传递函数

```
                    ┌─(1-sT_6)/(1+sT_6)─┐
Δω ──>[K_ω]────┬───>│    (陷波滤波器)    │──┬──>[Washout]──>[Ks2]──>[Lead-Lags]──>V_PSS
               │   └───────────────────┘  │
               │                          │
Pe ──>[K_P]───[+]──(合成加速功率信号)───────┘
```

#### 参数说明

| 参数 | 典型值 | 功能 |
|------|--------|------|
| $K_\omega$ | 10-20 | 转速增益 |
| $K_P$ | 1-2 | 功率增益 |
| $T_6$ | 0.01-0.05s | 陷波时间常数 |
| $T_w$ | 5-10s | 隔直时间常数 |

#### 优势
- **双信号输入**: 转速+功率，信息更丰富
- **加速功率**: 物理意义明确的合成信号
- **抑制噪声**: 陷波滤波器消除扭振模式影响
- **宽频适应**: 对不同振荡模式均有良好效果

---

### IEEE PSS4B - 多频段PSS

PSS4B是现代电力系统中应用的多频段PSS，采用三条并联支路分别针对低频、中频和高频振荡模式进行优化。

#### 三频段结构

```
输入信号 ──┬──>[低频支路 L]──┐
          ├──>[中频支路 I]──┼──>[合成]──>[限幅]──>V_PSS
          └──>[高频支路 H]──┘
```

#### 各频段支路特性

**低频支路 (Low Frequency)**
- 目标频率: 0.1-0.5 Hz
- 应用场景: 区域间低频振荡
- 典型时间常数: $T_{L1}, T_{L2}$ 较大

**中频支路 (Intermediate Frequency)**
- 目标频率: 0.5-1.5 Hz
- 应用场景: 本地-区域间耦合振荡
- 典型参数: 中等时间常数

**高频支路 (High Frequency)**
- 目标频率: 1.5-2.5 Hz
- 应用场景: 本地振荡模式
- 典型时间常数: $T_{H1}, T_{H2}$ 较小

#### 传递函数

$$V_{PSS} = V_L + V_I + V_H$$

其中各支路：

$$V_L = K_{L} \frac{sT_{wL}}{1+sT_{wL}} \frac{1+sT_{L1}}{1+sT_{L2}} \frac{1+sT_{L3}}{1+sT_{L4}} \Delta\omega$$

$$V_I = K_{I} \frac{sT_{wI}}{1+sT_{wI}} \frac{1+sT_{I1}}{1+sT_{I2}} \frac{1+sT_{I3}}{1+sT_{I4}} \Delta\omega$$

$$V_H = K_{H} \frac{sT_{wH}}{1+sT_{wH}} \frac{1+sT_{H1}}{1+sT_{H2}} \frac{1+sT_{H3}}{1+sT_{H4}} \Delta\omega$$

#### 特点
- **多频段优化**: 各支路独立设计，覆盖全频段
- **灵活整定**: 可根据系统特性调整各支路增益
- **鲁棒性**: 适应不同运行工况
- **复杂度高**: 参数较多，整定相对复杂

## PSS结构组件详解

### 隔直环节 (Washout Filter)

隔直环节是PSS的首要环节，用于阻断直流分量和极低频信号通过，只允许振荡信号进入控制系统。

#### 传递函数

$$G_{wash}(s) = \frac{sT_w}{1 + sT_w}$$

#### 频率特性

- **低频特性** ($f \ll 1/T_w$): 增益接近0，阻断直流
- **高频特性** ($f \gg 1/T_w$): 增益接近1，通过振荡信号
- **截止频率**: $f_c = 1/(2\pi T_w)$

#### 参数选择

- $T_w$ 通常选择5-10秒
- 过小: 隔直不充分，影响稳态运行
- 过大: 响应慢，对低频振荡效果差

### 超前-滞后环节 (Lead-Lag Compensator)

超前-滞后环节是PSS的核心相位补偿环节，用于补偿励磁系统和发电机的相位滞后，提供必要的阻尼转矩。

#### 单级超前-滞后

$$G_{lead-lag}(s) = \frac{1 + sT_1}{1 + sT_2}$$

#### 频率特性分析

在中心频率 $\omega_m$ 处提供最大相位超前：

$$\omega_m = \frac{1}{\sqrt{T_1 T_2}}$$

最大相位超前角：

$$\phi_{max} = \arcsin\left(\frac{T_1/T_2 - 1}{T_1/T_2 + 1}\right)$$

#### 参数设计原则

1. **确定补偿目标频率**: 通常为本地振荡模式频率
2. **计算所需相位补偿**: 补偿励磁系统相位滞后
3. **选择超前-滞后比**: 通常2-5之间
4. **计算时间常数**: $T_1 = \frac{\sqrt{\alpha}}{\omega_m}$, $T_2 = \frac{1}{\sqrt{\alpha}\omega_m}$

#### 多级串联

PSS通常采用两级超前-滞后串联，提供更大的相位补偿范围：

$$G_{comp}(s) = \frac{1+sT_1}{1+sT_2} \times \frac{1+sT_3}{1+sT_4}$$

### 增益环节 (Gain Stage)

增益$K_{PSS}$决定PSS的输出强度，是整定中的关键参数。

#### 增益对系统的影响

| 增益范围 | 系统响应 | 阻尼特性 |
|----------|----------|----------|
| $K_{PSS}$ 过小 | 响应微弱 | 阻尼不足 |
| $K_{PSS}$ 适中 | 响应良好 | 最佳阻尼 |
| $K_{PSS}$ 过大 | 超调大 | 可能失稳 |

#### 临界增益

临界增益是指使系统处于临界稳定状态的PSS增益：

$$K_{crit} = \frac{1}{|G_{loop}(j\omega_{osc})|}$$

实际整定增益通常取临界增益的1/3到1/2：

$$K_{PSS} = (0.3 \sim 0.5) \times K_{crit}$$

### 输出限幅 (Output Limiter)

输出限幅器限制PSS输出信号的幅度，防止过度调节影响电压控制。

#### 典型限幅值

- **正常运行**: ±0.05 pu 至 ±0.10 pu
- **极端工况**: ±0.15 pu（需特殊许可）

#### 限幅的作用

1. **保护励磁系统**: 防止过调制
2. **维持电压控制**: 保证AVR主导电压调节
3. **避免失稳**: 防止PSS负阻尼作用

## 输入信号选择

### 转速偏差 ($\Delta\omega$)

转速偏差是最常用的PSS输入信号，直接反映转子振荡。

#### 优点
- 物理意义明确
- 与阻尼转矩直接相关
- 易于测量

#### 缺点
- 对扭振模式敏感
- 需要轴系速度传感器

#### 信号处理

$$
\Delta\omega = \omega - \omega_0$$

其中：
- $\omega$: 实际转速 (pu)
- $\omega_0$: 额定转速 (1.0 pu)

### 有功功率偏差 ($\Delta P$)

有功功率偏差作为PSS输入，反映发电机输出变化。

#### 优点
- 易于测量
- 信号可靠
- 无扭振干扰

#### 缺点
- 相位关系复杂
- 受原动机响应影响

#### 信号合成

$$
\Delta P = P_{elec} - P_{ref}$$

### 频率偏差 ($\Delta f$)

频率偏差用于广域PSS或远离发电机的控制点。

#### 应用场景
- 广域阻尼控制
- 区域间振荡抑制
- 多点协调控制

#### 传递关系

频率偏差与转速偏差的关系：

$$\Delta f = \frac{\Delta\omega}{2\pi} \times f_0$$

### 多信号合成

现代PSS通常采用多信号合成策略：

$$V_{PSS} = K_1 \cdot \Delta\omega + K_2 \cdot \Delta P + K_3 \cdot \Delta f$$

## 阻尼转矩分析

### 同步转矩与阻尼转矩

同步发电机的电磁转矩可分解为：

$$\Delta T_e = K_S \Delta\delta + K_D \Delta\omega$$

其中：
- $K_S$: 同步转矩系数
- $K_D$: 阻尼转矩系数
- $\Delta\delta$: 功角偏差
- $\Delta\omega$: 转速偏差

#### 转矩系数对稳定性的影响

| 转矩系数 | 条件 | 稳定性 |
|----------|------|--------|
| $K_S > 0$ | 正同步转矩 | 静态稳定 |
| $K_D > 0$ | 正阻尼 | 动态稳定 |
| $K_D < 0$ | 负阻尼 | 振荡失稳 |

### 励磁系统的负阻尼效应

高增益AVR会产生负阻尼分量：

$$K_{D,AVR} = -K_{AVR} \cdot K_2 \cdot \sin(\phi_{AVR})$$

其中：
- $K_{AVR}$: AVR增益
- $K_2$: 发电机参数
- $\phi_{AVR}$: 励磁系统相位滞后

### PSS的正阻尼补偿

PSS引入的正阻尼转矩：

$$K_{D,PSS} = K_{PSS} \cdot K_2 \cdot |G_{PSS}(j\omega)| \cdot \cos(\phi_{PSS} - \phi_{exc})$$

理想情况下，应使：

$$K_{D,total} = K_{D,mech} + K_{D,AVR} + K_{D,PSS} > 0$$

### 阻尼转矩系数图 (Damping Torque Coefficient)

阻尼转矩系数随频率变化的曲线称为D-曲线：

```
KD
│
│  ──────────────── AVR单独作用 (负阻尼)
│      ╲
│       ╲
│        ╲  ─────── PSS+AVR (正阻尼)
│         ╲________
│
└──────────────────────→ f
   0.1   0.5   1.0   2.0 Hz
```

## PSS参数整定方法

### 频率响应法

频率响应法通过测量励磁系统的频率特性来设计PSS参数。

#### 步骤

1. **测量励磁系统相位**: 在目标振荡频率$f_{osc}$处测量励磁系统的相位滞后$\phi_{exc}$

2. **计算所需补偿**: 需要的相位超前：
   $$\phi_{comp} = -\phi_{exc}$$

3. **设计超前-滞后环节**: 
   $$\alpha = \frac{1 + \sin(\phi_{comp}/n)}{1 - \sin(\phi_{comp}/n)}$$
   其中$n$为超前-滞后级数

4. **计算时间常数**:
   $$T_{lead} = \frac{\sqrt{\alpha}}{2\pi f_{osc}}$$
   $$T_{lag} = \frac{1}{\sqrt{\alpha} \cdot 2\pi f_{osc}}$$

5. **整定增益**: 逐步增加增益至临界值的30-50%

### 特征值分析法

基于状态空间模型和小信号稳定性分析进行PSS设计。

#### 状态空间表示

$$
\Delta\dot{x} = A \Delta x + B \Delta u$$
$$
\Delta y = C \Delta x + D \Delta u$$

#### 特征值计算

求解特征方程：

$$\det(A - \lambda I) = 0$$

得到特征值$\lambda_i = \sigma_i \pm j\omega_i$，其中：
- $\sigma_i$: 阻尼实部
- $\omega_i$: 振荡频率
- 阻尼比: $\xi_i = -\sigma_i / \sqrt{\sigma_i^2 + \omega_i^2}$

#### PSS设计目标

调整PSS参数使机电振荡模式的阻尼比满足：

$$\xi_i \geq 0.05 \sim 0.10$$

### 优化方法

基于优化算法自动搜索最优PSS参数。

#### 目标函数

$$J = \sum_{i} w_i (\xi_i^{target} - \xi_i)^2 + w_g (K_{PSS} - K_{PSS}^{nom})^2$$

其中：
- $w_i$: 模式权重
- $\xi_i^{target}$: 目标阻尼比
- $w_g$: 增益权重

#### 约束条件

- 参数范围: $T_{min} \leq T_i \leq T_{max}$
- 增益限制: $K_{PSS} \leq K_{PSS,max}$
- 稳定约束: 所有特征值实部为负

## 与励磁系统的协调

### 励磁系统类型

PSS需要与不同类型的励磁系统协调工作。

#### 直流励磁系统

- **特点**: 较慢响应(0.5-1.0s)
- **PSS需求**: 较大相位补偿
- **典型模型**: IEEE DC1A

#### 交流励磁系统

- **特点**: 中等响应(0.1-0.5s)
- **PSS需求**: 适中相位补偿
- **典型模型**: IEEE AC1A, AC4A

#### 静止励磁系统

- **特点**: 快速响应(0.01-0.1s)
- **PSS需求**: 较小相位补偿
- **典型模型**: IEEE ST1A, ST4B

### 协调设计原则

1. **响应速度匹配**: PSS不应比励磁系统更快
2. **相位补偿**: 补偿励磁系统相位滞后
3. **增益协调**: PSS增益与AVR增益相匹配
4. **限幅设置**: 保证AVR主导作用

### 整体传递函数

```
AVR + PSS 组合传递函数:

        ┌──────────────────────────────────────┐
        │   sTw      1+sT1   1+sT3           │
V_ref ──┼──[1 + Kpss ───── ────── ──────]─────┼──> Efd
        │   1+sTw    1+sT2   1+sT4           │
        └──────────────────────────────────────┘
                    AVR核心
```

## 现场测试与投运

### 出厂测试

#### 静态测试

1. **环节特性测试**: 验证各环节的传递函数
2. **限幅器校验**: 确认限幅值正确
3. **信号极性**: 验证输入输出极性

#### 动态测试

1. **阶跃响应**: 输入阶跃信号观测输出
2. **频率扫描**: 测量实际频率特性
3. **噪声测试**: 验证抗干扰能力

### 现场投运步骤

#### 步骤1: 信号检查

- 测量输入信号质量
- 检查信号极性
- 验证隔直效果

#### 步骤2: 开环测试

- 断开PSS至励磁的连接
- 施加小扰动观测响应
- 验证相位补偿效果

#### 步骤3: 闭环投运

- 逐步投入PSS
- 观察系统响应
- 监测振荡阻尼

#### 步骤4: 负载试验

- 不同有功负载下测试
- 记录阻尼特性
- 验证整定有效性

### 性能评估指标

| 指标 | 要求 | 说明 |
|------|------|------|
| 阻尼比 | >5% | 机电振荡模式 |
| 调节时间 | <10s | 扰动后恢复 |
| 超调量 | <20% | 电压恢复过程 |
| 稳态偏差 | <0.5% | 无PSS影响 |

## 先进PSS技术

### 自适应PSS

根据系统运行工况自动调整参数。

#### 增益自适应

$$K_{PSS} = K_0 + \alpha \cdot (P_{gen} - P_0)$$

#### 频率自适应

根据在线辨识的振荡频率调整相位补偿：

$$T_1^{new} = T_1^{base} \times \frac{f_{base}}{f_{measured}}$$

### 广域PSS (Wide-Area PSS)

利用PMU广域测量信息进行协调控制。

#### 结构

```
PMU测量 ──>[通信]──>[WAMS]──>[广域PSS算法]──>[控制指令]──>[励磁]
  │                                               │
  └────── 其他机组/线路 ────────────────────────────┘
```

#### 优势
- **区域间振荡**: 有效抑制区域间模式
- **协调控制**: 多机协调优化
- **全局可观**: 全局系统状态感知

### 智能PSS

基于人工智能的PSS控制策略。

#### 模糊PSS

利用模糊逻辑处理不确定性：

```
Δω, Δω̇ ──>[模糊化]──>[模糊规则]──>[解模糊]──>VPSS
```

#### 神经网络PSS

通过神经网络学习最优控制策略：

$$V_{PSS} = f_{NN}(\Delta\omega, \Delta P, \Delta\delta; W)$$

## EMT仿真建模要点

### 详细模型

在电磁暂态仿真中，PSS应采用详细模型：

1. **连续传递函数**: 使用微分方程实现
2. **离散化方法**: 采用梯形积分或后向欧拉
3. **时间步长**: 与励磁系统一致(0.001-0.01s)

### 离散化实现

超前-滞后环节的离散化：

$$G(z) = \frac{Y(z)}{U(z)} = \frac{b_0 + b_1 z^{-1}}{1 + a_1 z^{-1}}$$

系数计算：

$$b_0 = \frac{T + 2T_1}{T + 2T_2}, \quad b_1 = \frac{T - 2T_1}{T + 2T_2}$$

$$a_1 = \frac{T - 2T_2}{T + 2T_2}$$

### 与发电机模型接口

```
发电机模型 ──> Δω, ΔP ──> PSS模型 ──> V_PSS ──> 励磁系统 ──> Efd ──> 发电机
   │                                                 │
   └─────────────────────────────────────────────────┘
```

## 常见问题与解决

### 问题1: 反调现象

**现象**: 负载变化时电压反向调节

**原因**: 隔直时间常数过小或功率信号使用不当

**解决**: 
- 增大$T_w$至10秒以上
- 使用加速功率信号(PSS2B)
- 调整功率信号系数

### 问题2: 高频噪声放大

**现象**: 引入高频振荡

**原因**: 超前-滞后环节高频增益过高

**解决**:
- 增加滤波环节
- 限制超前-滞后比
- 使用陷波滤波器

### 问题3: 多机负交互

**现象**: 多台PSS相互影响导致失稳

**原因**: PSS之间缺乏协调

**解决**:
- 协调整定各机PSS参数
- 使用广域PSS
- 采用多机协调控制策略
