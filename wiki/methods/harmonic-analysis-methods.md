# 谐波分析方法 (Harmonic Analysis Methods)

## 定义与概述

谐波分析方法（Harmonic Analysis Methods）是电力系统电磁暂态（EMT）仿真中用于分析、计算和控制非正弦波形中谐波分量的技术。现代电力系统中，电力电子设备（变流器、变频器、开关电源）、电弧炉、变压器饱和等是主要谐波源。谐波会导致设备过热、继电保护误动、通信干扰、电能质量下降等问题。谐波分析方法包括频域谐波潮流、时域FFT分析、谐波阻抗计算、谐振分析、滤波器设计等，是电能质量评估和治理的关键技术。

## 1. 理论基础

### 1.1 谐波基本概念

**傅里叶级数分解**：
$$v(t) = V_0 + \sum_{h=1}^{\infty} V_h \sin(h\omega t + \phi_h)$$

其中：
- $h$：谐波次数（基波$h=1$）
- $V_h$：第$h$次谐波电压幅值
- $\phi_h$：第$h$次谐波相位

**谐波分量提取**：
$$V_h = \sqrt{a_h^2 + b_h^2}, \quad \phi_h = \arctan\left(\frac{a_h}{b_h}\right)$$

$$a_h = \frac{2}{T}\int_0^T v(t)\cos(h\omega t)dt$$
$$b_h = \frac{2}{T}\int_0^T v(t)\sin(h\omega t)dt$$

**特征谐波**（整流器）：
$$h = kp \pm 1$$

其中$p$为脉波数，$k=1,2,3...$

| 换流器类型 | 脉波数 | 特征谐波 |
|------------|--------|----------|
| 单相全桥 | 2 | 3, 5, 7, 9... |
| 三相六脉动 | 6 | 5, 7, 11, 13... |
| 三相十二脉动 | 12 | 11, 13, 23, 25... |
| 二十四脉动 | 24 | 23, 25, 47, 49... |

### 1.2 谐波指标

**总谐波畸变率（THD）**：
$$\text{THD}_V = \frac{\sqrt{\sum_{h=2}^{H}V_h^2}}{V_1} \times 100\%$$

$$\text{THD}_I = \frac{\sqrt{\sum_{h=2}^{H}I_h^2}}{I_1} \times 100\%$$

**谐波含有率（HR）**：
$$\text{HR}_h = \frac{V_h}{V_1} \times 100\%$$

**国际标准限值**：

| 标准 | 电压等级 | THD_V限值 | 单次谐波限值 |
|------|----------|-----------|--------------|
| **IEEE 519** | <69kV | 5.0% | 3.0% |
| | 69-161kV | 3.0% | 1.5% |
| | >161kV | 1.5% | 1.0% |
| **GB/T 14549** | 0.38kV | 5.0% | 4.0% |
| | 6-10kV | 4.0% | 3.2% |
| | 35-66kV | 3.0% | 2.4% |
| | 110kV | 2.0% | 1.6% |

**TDD（总需求畸变）**：
$$\text{TDD} = \frac{\sqrt{\sum_{h=2}^{H}I_h^2}}{I_{load}} \times 100\%$$

### 1.3 谐波阻抗

**频域阻抗**：
$$Z(h) = R(h) + jX(h) = R(h) + jhX_1$$

**设备谐波阻抗特性**：

| 设备 | 谐波阻抗 | 特点 |
|------|----------|------|
| **变压器** | $Z_T(h) = R_T + jhX_T$ | 漏抗线性增加 |
| **线路** | $Z_L(h) = R_L(h) + jhX_L$ | 集肤效应使R增加 |
| **电容器** | $X_C(h) = X_{C1}/h$ | 容抗与h成反比 |
| **电抗器** | $X_L(h) = hX_{L1}$ | 感抗与h成正比 |
| **负载** | $Z_{load}(h)$ | 与负载类型相关 |

**谐波阻抗测量**：
$$Z(h) = \frac{V(h)}{I(h)}$$

通过向系统注入谐波电流，测量响应电压。

## 2. EMT仿真应用

### 2.1 频域谐波潮流

**谐波潮流方程**：
$$Y(h)V(h) = I(h)$$

其中：
- $Y(h)$：h次谐波导纳矩阵
- $V(h)$：h次谐波节点电压
- $I(h)$：h次谐波注入电流源

**迭代求解流程**：
```
1. 基波潮流计算（h=1）
2. 确定非线性设备工作状态
3. for h = 2, 3, ..., H:
     计算谐波电流源 I(h)
     求解 Y(h)V(h) = I(h)
4. 叠加各次谐波
5. 更新非线性设备状态
6. 重复3-5直到收敛
```

**非线性设备建模**：

**变流器谐波电流源**：
$$I_h = \frac{V_{dc}}{\pi Z_{sys}(h)} \cdot f(\alpha, \mu)$$

其中：
- $\alpha$：触发角
- $\mu$：换相重叠角
- $f(\alpha, \mu)$：与工况相关的函数

**变压器饱和**：
$$i_m = k_1\Phi + k_3\Phi^3 + k_5\Phi^5 + ...$$

产生奇次谐波。

### 2.2 时域FFT分析

**采样要求**：
- 采样频率：$f_s > 2 \cdot h_{max} \cdot f_1$
- 采样点数：$N = 2^m$（便于FFT）
- 窗函数：Hanning/Hamming减少泄漏

**FFT计算**：
$$V(k) = \sum_{n=0}^{N-1} v(n)e^{-j2\pi kn/N}$$

**谐波幅值**：
$$V_h = \frac{2|V(k)|}{N}, \quad k = h \cdot \frac{f_1}{\Delta f}$$

**频率分辨率**：
$$\Delta f = \frac{f_s}{N} = \frac{1}{T_{window}}$$

**窗函数选择**：

| 窗函数 | 主瓣宽度 | 旁瓣衰减 | 适用场景 |
|--------|----------|----------|----------|
| 矩形 | 2 bins | -13 dB | 瞬态分析 |
| Hanning | 4 bins | -31 dB | 一般谐波分析 |
| Hamming | 4 bins | -41 dB | 高精度幅值 |
| Blackman | 6 bins | -57 dB | 邻近频率分辨 |

### 2.3 谐振分析

**串联谐振**：
$$h_{res} = \sqrt{\frac{X_C}{X_L}} = \sqrt{\frac{1}{\omega_1^2 LC}}$$

**并联谐振**：
$$h_{res} = \sqrt{\frac{X_{C,eq}}{X_{L,eq}}}$$

**谐振放大因子**：
$$Q = \frac{X_L}{R} = \frac{h_{res}X_{L1}}{R}$$

**典型谐振场景**：

| 场景 | 谐振元件 | 谐振频率 | 风险 |
|------|----------|----------|------|
| 电容器组投切 | 变压器漏抗+电容器 | 3-15次 | 谐波放大 |
| HVDC滤波器 | 滤波电感+电容 | 11/13/24次 | 谐波注入 |
| 长线路充电 | 线路电感+电容 | 0.5-2次 | 铁磁谐振 |
| 风电场汇集 | 汇集线路+箱变 | 5-13次 | 谐振过压 |

**谐波阻抗扫描**：
$$Z_{sys}(h) = \frac{V_{inj}(h)}{I_{inj}(h)}$$

绘制$|Z_{sys}(h)|$随$h$变化的曲线，识别谐振点。

### 2.4 谐波滤波器设计

**单调谐滤波器**：
$$h_n = \sqrt{\frac{X_C}{X_L}}$$

品质因数：
$$Q = \frac{X_L}{R} = \frac{\omega_n L}{R}$$

典型$Q$值：30-100

**双调谐滤波器**：
两个谐振频率$h_1$和$h_2$：
$$\omega_{n1} = \frac{1}{\sqrt{L_1 C_1}}, \quad \omega_{n2} = \frac{1}{\sqrt{L_2 C_2}}$$

**高通滤波器**：
截止频率：
$$f_c = \frac{1}{2\pi RC}$$

**滤波器设计步骤**：
```
1. 确定谐波源特性（次数、幅值）
2. 计算系统谐波阻抗
3. 选择滤波器类型和调谐次数
4. 计算LC参数
5. 校验无功补偿容量
6. 校验谐振点避开特征谐波
7. 热稳定和过压校验
```

**滤波器容量校验**：
$$Q_{filter} = \sum_{h} I_h^2 X_{L,h} < Q_{rated}$$

## 3. 实现技术

### 3.1 谐波测量与监测

**在线监测装置**：
- 采样率：≥256点/周波
- 谐波次数：2-50次
- 测量精度：0.2级

**谐波数据分析**：
```python
def harmonic_analysis(v, fs, f1=50):
    N = len(v)
    V_fft = np.fft.fft(v)
    
    # 谐波幅值
    V_h = 2 * np.abs(V_fft) / N
    
    # 谐波次数索引
    h_index = np.arange(2, 51) * f1 / (fs/N)
    
    # 提取谐波
    harmonics = V_h[h_index.astype(int)]
    
    # THD计算
    THD = np.sqrt(np.sum(harmonics[1:]**2)) / harmonics[0] * 100
    
    return harmonics, THD
```

### 3.2 谐波潮流并行计算

并行for h = 2 to H:

## 7. 适用边界与限制

### 7.1 适用条件

**频域谐波分析适用**：
- 稳态谐波分析
- 线性网络
- 已知谐波源特性

**时域FFT分析适用**：
- 暂态谐波分析
- 非线性系统
- 时变谐波源

### 7.2 失效边界

**频域方法局限**：
- 不能处理谐波间的相互作用
- 忽略相位信息
- 假设线性叠加

**时域方法局限**：
- 计算量大
- 频谱泄漏
- 频率分辨率受限

### 7.3 精度边界

| 分析方法 | 频率精度 | 幅值精度 | 适用场景 |
|----------|----------|----------|----------|
| 频域潮流 | - | 5-10% | 规划设计 |
| 时域FFT | $\Delta f$ | 2-5% | 实测分析 |
| EMT仿真 | 高 | <5% | 精确分析 |
