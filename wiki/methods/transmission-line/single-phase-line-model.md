# 单相线路模型 (Single-Phase Line Model)

## 定义

单相线路模型是将一条输电线路简化为单一导体及其返回路径的电磁暂态（EMT）等效电路表示，是EMT仿真中最基础的线路模型类型。该模型的核心价值在于降低问题维度——将三相耦合线路解耦为独立的单相回路，适合教学演示、配电馈线分析、单相等效回路，或经[[modal-transformation]]解耦后的单一传播模态。

从物理本质上讲，单相线路模型的有效性依赖于明确的返回路径定义。根据应用场景不同，返回路径可以是：相-地回路、相-相回路、芯线-护套回路，或模态等效回路。不同返回路径对应的参数含义和量级存在显著差异，在模型构建时必须明确界定。

单相线路模型的局限性同样清晰：简化为单相会丢失相间互阻抗、零序耦合、不平衡传播以及平行线路感应等问题。因此，[[protection-relay-modeling]]、故障定位和并行线路感应分析不能随意使用单相简化模型。

## EMT中的角色

单相线路模型在EMT仿真中承担多重角色：

**1. 入门级建模工具**：作为理解[[bergeron-line-model]]和[[transmission-line-theory]]的起点，单相Bergeron模型将分布参数线路的波动方程转化为集总参数电路形式，是初学者掌握线路时域建模的最佳切入点。

**2. 配电系统等效**：在配电系统或低压网络中，单相线路模型用于构建简单的故障算例、开关动作分析和负荷接入研究。此时需要明确返回导体的接地路径和等效阻抗。

**3. 模态域表示**：经[[modal-transformation]]后的多相线路，每个独立模态可作为单相线路处理。这意味着单相模型有时是"模态等效"而非"物理单根导线"，两者参数含义不同，需在文档中明确说明。

**4. 参数提取验证**：Kurokawa等提出的频域参数反演方法，通过开路/短路测试提取单相等效阻抗和导纳，用于校准理论计算值与实际线路参数的偏差。

## EMT建模方法

单相线路模型在EMT中有四种主要建模形式，各有其适用场景和精度边界。

### 方法一：集中参数R-L（或R-L-C）模型

这是最简单的线路模型，将线路用串联阻抗$R_\ell + j\omega L_\ell$和并联导纳$G_\ell + j\omega C_\ell$近似表示。该模型的数学基础是忽略线路的波动效应，假设电压电流在全线均匀分布。

对于长度为$\ell$的线路，端口阻抗可表示为：

$$Z_\ell(\omega) = R_\ell + j\omega L_\ell, \quad Y_\ell(\omega) = G_\ell + j\omega C_\ell$$

其中$R_\ell = r_0 \ell$、$L_\ell = l_0 \ell$、$C_\ell = c_0 \ell$、$G_\ell = g_0 \ell$，$r_0, l_0, c_0, g_0$为单位长度参数。

**EMT离散化实现**：采用梯形积分法或后向欧拉法对RL支路和GC支路进行时域离散。以梯形法为例，电阻和电感的串联网格可写为：

$$i_k(t) = \frac{v_k(t)}{R} + I_{hist,k}(t)$$

其中历史电流源项$I_{hist,k}$携带上一时刻的信息，实现无矩阵迭代求解。

**适用场景**：短线低频近似、潮流计算、机电暂态等值、故障电流计算

**精度边界**：当线路电气长度$\ell / \lambda$（线路长度与波长之比）大于0.05时，分布效应开始显著，集中参数模型误差增大。对于60Hz、100km架空线路，$\lambda \approx 5000$km，$\ell/\lambda \approx 0.02$，尚可接受；但对于高频暂态（数十kHz），$\ell/\lambda$可能达到0.5以上，集中模型完全失效。

### 方法二：π型等值模型

π型等值是工程中最广泛使用的集中参数模型变体，将线路表示为两端并联导纳和中央串联阻抗的π形网络。该模型比简单R-L串联储提供更精确的电压分布近似。

**双端π型等值电路的节点导纳矩阵**：

对于线路两端节点k和m，π型等值的导纳矩阵为：

$$\mathbf{Y}_{km} = \begin{bmatrix} Y_k + Y_c & -Y_c \\ -Y_c & Y_m + Y_c \end{bmatrix}$$

其中$Y_c = 1/Z_c$为特征导纳，$Z_c = \sqrt{(R+j\omega L)/(G+j\omega C)}$为特征阻抗。

**等值参数计算**：当已知单位长度参数$r_0, l_0, g_0, c_0$和线路长度$\ell$时：

$$Z'_c = \sqrt{\frac{r_0 + j\omega l_0}{g_0 + j\omega c_0}}$$

对于无损或低损线路，简化特征阻抗为$Z'_c \approx \sqrt{l_0/c_0}$。

**EMT求解**：构建系统节点导纳矩阵后，采用稀疏矩阵技术求解线性代数方程组。π型等值的优势在于其矩阵结构稀疏，便于EMT程序的数值求解。

**适用场景**：低频暂态近似、故障分析、开关操作仿真

**精度边界**：π型模型本质上是集中参数模型，对高频分量和长线距离效应（如行波反射）的捕捉能力有限。对于包含上升沿小于1μs的雷电暂态，π型模型误差可达15%以上。

### 方法三：单相Bergeron模型

Bergeron模型是基于行波理论的时域等效电路，通过"历史电流源"将分布参数线路的波动效应嵌入集总参数电路。该模型是EMT程序中最广泛使用的线路模型之一。

**电报方程**（分布参数形式）：

$$-\frac{\partial v}{\partial x} = Z'(\omega) i, \quad -\frac{\partial i}{\partial x} = Y'(\omega) v$$

其中$Z'(\omega) = R' + j\omega L'$为单位长度串联阻抗，$Y'(\omega) = G' + j\omega C'$为单位长度并联导纳。

**特征阻抗与传播常数**：

$$Z_c(\omega) = \sqrt{\frac{Z'(\omega)}{Y'(\omega)}}, \quad \gamma(\omega) = \sqrt{Z'(\omega) Y'(\omega)}$$

**行波时延**：

$$\tau = \ell \sqrt{L' C'}$$

对于无损线路，$\gamma = j\omega\sqrt{L'C'}$，$Z_c = \sqrt{L'/C'}$为实数。

**Bergeron电流递推公式**（时域离散形式）：

$$I_k(t) = \frac{V_k(t)}{Z_c} + I_{hist,k}(t)$$

其中历史电流源项为：

$$I_{hist,k}(t) = -\frac{V_m(t-\tau)}{Z_c} + I_m(t-\tau)$$

该公式表明：当前时刻的线路注入电流由当前电压和历史电压电流共同决定，实现了对波传播效应的时域等效。

**矢量拟合扩展**（频率相关Bergeron模型）：

Torrez Caballero 2014提出将频率相关效应引入Bergeron纵向参数。采用矢量拟合（Vector Fitting）技术将频变纵向阻抗$Z(\omega)$逼近为有理函数：

$$Z_{fit}(s) = d + sh + \sum_{n=1}^{N} \frac{r_n}{s - p_n}$$

通过极点-留数映射，将$Z_{fit}(s)$综合为等效RL梯形网络，嵌入Bergeron等效电路。量化结果：在0.1Hz~10kHz宽频范围内阻抗幅值误差<1.5%，相位误差<2.0°；相比传统π型级联模型，内存占用降低约55%，仿真速度提升约1.8倍。

**适用场景**：教学、单模态行波、简单EMT仿真、实时仿真

**精度边界**：常参数Bergeron模型忽略纵向参数的频率依赖性，在高频暂态或涉及集肤效应、大地回路的场景中精度下降。频变Bergeron模型通过参数拟合扩展了频带覆盖，但需要额外的拟合计算。

### 方法四：频变单相模型

频变模型在宽频范围内精确考虑参数随频率的变化，是处理电缆暂态、高频谐波和大地回路效应的必要工具。

**单位长度阻抗的频变特性来源**：

导体内部阻抗受集肤效应影响：

$$Z_i(\omega) = \frac{j\omega\mu_0}{2\pi r} \frac{I_0(kr)}{I_1(kr)}, \quad k = \sqrt{j\omega\mu_0/\sigma}$$

其中$I_0, I_1$为修正贝塞尔函数，$r$为导体半径，$\sigma$为电导率。

大地返回阻抗（Carson公式近似）：

$$Z_g(\omega) = \frac{j\omega\mu_0}{\pi} \int_0^{\infty} \frac{e^{-2hy}}{\sqrt{y^2 + j\omega\mu_0/\rho}} \, dy$$

其中$h$为导线高度，$\rho$为土壤电阻率，$y$为积分变量。

**频变导纳的建模**：

对地电容的频率效应相对较弱，但在高频时需考虑接地返回导纳。Duarte 2023指出：在1000Ωm高电阻率土壤中，忽略接地返回导纳将导致高频暂态仿真出现显著误差（幅值偏差可达15%以上）。

**参数提取的反演方法**（Kurokawa 2006）：

对于单相线路，通过远端开路和短路测试，可直接由端口阻抗反演单位长度参数：

$$Z_{c,m} = \sqrt{Z_{eq,open} \cdot Z_{eq,short}}$$
$$\gamma_m = \frac{1}{\ell} \ln\left(\frac{1+X}{1-X}\right), \quad X = \sqrt{\frac{Z_{eq,short}}{Z_{eq,open}}}$$

其中$Z_{eq,open}$和$Z_{eq,short}$为开路和短路输入阻抗的模态值。

量化结果：10Hz~10kHz频段内，新方法提取的频变参数与Carson/Pollaczek公式法计算结果高度吻合，全频段相对误差<0.5%。

**FDTD验证基准**（Duarte 2023）：

Duarte采用全波FDTD方法作为独立基准，验证传输线理论在地下电缆快速暂态中的精度。结果表明：精确计算接地返回参数时，传输线理论可准确模拟上升沿低至0.2μs的快速暂态，峰值误差普遍<0.5%。

该验证的重要认知是：传输线理论本身并非主要误差来源，关键在于是否将接地返回阻抗和导纳作为频变参数认真处理。

## 形式化表达

### 核心参数定义

**单位长度阻抗矩阵（单相）**：
$$Z'(\omega) = R' + j\omega L' + Z_g'(\omega)$$

其中$R'$为导体直流电阻，$j\omega L'$为外部电感引起的感抗，$Z_g'(\omega)$为大地返回阻抗（频率相关）。

**单位长度导纳矩阵（单相）**：
$$Y'(\omega) = G' + j\omega C' + Y_g'(\omega)$$

其中$G'$为绝缘泄漏电导，$j\omega C'$为对地电容引起的容纳，$Y_g'(\omega)$为接地返回导纳。

**特性阻抗**：
$$Z_c(\omega) = \sqrt{\frac{Z'(\omega)}{Y'(\omega)}} = \sqrt{\frac{R' + j\omega L' + Z_g'}{G' + j\omega C' + Y_g'}}$$

**传播常数**：
$$\gamma(\omega) = \sqrt{Z'(\omega) Y'(\omega)}$$

**行波时延**（无损近似）：
$$\tau = \ell \sqrt{L' C'}$$

### Bergeron模型时域递推

**线路端电流**（时域离散）：
$$I_k(t_n) = \frac{V_k(t_n)}{Z_c} + I_{hist,k}(t_n)$$

**历史电流源更新**：
$$I_{hist,k}(t_n) = -\frac{V_m(t_{n-m})}{Z_c} + I_m(t_{n-m})$$

其中$m = \text{round}(\tau / \Delta t)$为时延对应的步数，$\Delta t$为仿真步长。

### π型等值参数

**等效串联阻抗**：
$$Z_{series} = Z_c \sinh(\gamma \ell) \approx Z_c \gamma \ell \quad (\gamma\ell \ll 1)$$

**等效并联导纳**：
$$Y_{shunt} = \frac{2}{Z_c} \tanh\left(\frac{\gamma\ell}{2}\right) \approx Y_c \ell \quad (\gamma\ell \ll 1)$$

对于短线近似（$\gamma\ell < 0.1$）：
$$Z_{series} \approx (R' + j\omega L')\ell, \quad Y_{shunt} \approx (G' + j\omega C')\ell$$

## 关键技术挑战

### 挑战一：返回路径建模的物理不确定性

单相线路模型的有效性高度依赖返回路径的明确定义。在相-地回路中，大地作为返回导体，其等效阻抗随频率、土壤电阻率和大地电流分布方式变化。Carson公式假设均匀平坦大地、无限大导电平面，而实际地形起伏、土壤分层、有色金属管线都会改变返回路径特性。

**量化影响**：Duarte 2023的FDTD验证表明，在1000Ωm高电阻率土壤中，忽略接地返回导纳导致高频暂态幅值偏差达15%以上。

### 挑战二：集肤效应和邻近效应的精确建模

导体内部阻抗随频率变化的集肤效应在高频暂态中不可忽略。修正贝塞尔函数表达式虽然精确，但计算复杂。工程中常采用近似公式：

$$R'(\omega) \approx R'_{dc} \cdot \begin{cases} 1 & f < f_c \\ \sqrt{f/f_c} & f > f_c \end{cases}$$

其中$f_c$为集肤效应临界频率，与导体半径和材料属性相关。

### 挑战三：行波数值稳定性条件

Bergeron模型要求仿真步长$\Delta t$满足$\Delta t \leq \tau$以保证数值稳定性。对于长线路和大系统，减小$\Delta t$直接导致计算量增加。实时仿真要求$\Delta t \leq 50\mu s$，这对长线路行波效应是挑战。

**解决方向**：古斯塔夫森提出的矩阵对角化技术通过预计算传播特性，将时延$\tau$精确嵌入历史源，避免了对步长的严格限制。

### 挑战四：参数频率依赖性的高效处理

频变参数模型需要大量频率点采样、矢量拟合和时域卷积计算。Kurokawa 2006指出，传统的Carson/Pollaczek积分公式在大地不均匀、导体非平行时误差显著；参数反演方法虽可规避几何假设，但需要精确的模态变换矩阵。

**计算效率对比**：Duarte 2023报告，传输线理论结合闭式近似公式相比全波FDTD加速超过10000倍，将单位长度参数计算耗时从分钟级降至毫秒级。

## 适用边界与选择指南

### 何时使用单相线路模型

**适用条件**：

- 配电系统或低压网络（电压等级<35kV）局部等效
- 教学演示和算法验证
- 单相故障分析（如单相接地故障）
- 多相系统经模态分解后的独立模态表示
- 已知返回路径明确的简单网络

**不适用条件**：

- 三相不平衡线路（应使用完整三相模型）
- 平行线路耦合问题（需[[mutual-impedance]]）
- 非换位线路的零序特性分析
- 高频暂态（>10kHz）的精确模拟
- 地线/护套回流不可忽略的输电线路

### 建模方法选择决策表

| 应用场景 | 推荐模型 | 备选模型 | 注意事项 |
|---------|---------|---------|---------|
| 潮流计算 | π型等值 | 集中R-L | 忽略对地导纳在高压系统中可接受 |
| 故障电流计算 | π型等值 | Bergeron | 故障初始瞬态可能需要Bergeron |
| 雷电暂态 | 频变模型 | 频变Bergeron | 必须考虑接地返回导纳 |
| 操作过电压 | Bergeron | 频变π型 | 波过程是核心，需Bergeron |
| 谐波分析 | 频变模型 | 分段π型 | 需覆盖全频段参数 |
| 实时仿真 | Bergeron | 集中R-L | 步长限制是关键约束 |
| 教学演示 | 集中R-L | π型等值 | 简化模型易于理解 |

### 单相与多相模型的选择判断

**可用单相模型的场景**：

1. 三相对称且完全换位的线路，模态分析后可独立处理每模态
2. 配电网络单相支路，返回路径明确
3. 故障分析中，将三相故障分解为三个单相问题（假设故障阻抗相同）

**必须用多相模型的场景**：

1. 不平衡故障（单相接地、两相短路、两相接地）
2. 平行线路感应（需要互阻抗模型）
3. 同塔多回线路的相间耦合
4. 电缆三相铠装感应
