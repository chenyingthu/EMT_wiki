# 集总参数模型 (Lumped Parameter Model)

## 定义与边界

集总参数模型（Lumped Parameter Model）将空间分布的电阻、电感、电导和电容替换为有限个集中元件。在输电线路、电缆、变压器漏抗、滤波器阻抗和等效支路中，它把连续场问题或传输线问题转化为可由 [[nodal-analysis]] 或 [[state-space-method]] 求 解的电路模型。

本页讨论 EMT 方法中的集总化原则，不提供固定工程长度阈值表。集总近似是否成立取决于线路**电气长度**（$d/\lambda$，其中 $d$ 为物理长度，$\lambda$ 为目标频带内波长）、**目标频带**、**波头陡度**、**参数频变性**和研究指标。不能把某一电压等级或某一长度阈值当作全局适用规则。

## EMT 中的作用

集总参数模型在 EMT 中承担以下任务：

- 将短线路、滤波器、支路阻抗和变压器漏抗转化为 R、L、C、G 元件，直接装配进节点导纳矩阵；
- 作为多段 π 型线路或 Bergeron 线的**基本单元**；
- 为控制器、设备端口模型和网络等值提供**低阶端口电路**；
- 通过 [[companion-model]] 把电感、电容离散为当前等效导纳和历史电流源，使梯形积分可直接应用于集总 RLC 网络。

主要收益是**模型结构简单**、可解析推导端口关系、实现成本低；主要风险是**丢失传播延时**、**分布谐振**和**参数频变效应**。

## 核心机制

均匀传输线的分布参数形式（电报员方程）为：

$$\frac{\partial v}{\partial x} = -(r + l\frac{\partial}{\partial t})i$$

$$\frac{\partial i}{\partial x} = -(g + c\frac{\partial}{\partial t})v$$

若研究频带内线路电气长度较小（$d/\lambda \ll 1$），则总参数近似为：

$$R = r\ell,\quad L = l\ell,\quad G = g\ell,\quad C = c\ell$$

其中 $\ell$ 为线路长度。

### π 型等效电路

经典 π 型等效将串联阻抗 $Z$ 置于线路中间，两端各并联一半导纳 $Y/2$：

$$Z = (r + j\omega l)\ell$$

$$Y = (g + j\omega c)\ell$$

精确 π 型等效（来自双曲方程精确解）为：

$$Z_{c} = \sqrt{\frac{Z(\omega)}{Y(\omega)}} = \sqrt{\frac{r + j\omega l}{g + j\omega c}}$$

$$\gamma = \sqrt{Z(\omega)Y(\omega)}$$

$$Y_{z\pi} = \frac{1}{Z_{c}\sinh(\gamma\ell)} \quad \text{（串联支路导纳）}$$

$$Y_{\pi} = \frac{\cosh(\gamma\ell) - 1}{Z_{c}\sinh(\gamma\ell)} \quad \text{（每端对地导纳）}$$

精确 π 型的关键优势是：**无需分段**即可用单一 π 支路表示任意长度线路，通过双曲函数修正消除标称 π 型的近似误差。

### 梯形法离散化

在 EMT 时域中，$R$ 和 $G$ 直接进入导纳矩阵，$L$ 和 $C$ 通过梯形积分转化为伴随支路。电容的梯形离散：

$$i_{n+1} = \frac{2C}{\Delta t}v_{n+1} + \underbrace{\left[-\frac{2C}{\Delta t}v_n - i_n\right]}_{I_{\text{hist}}^n}$$

等效电导：$G_{C} = \frac{2C}{\Delta t}$；电感的梯形离散：

$$i_{n+1} = \frac{\Delta t}{2L}v_{n+1} + \underbrace{\left[i_n + \frac{\Delta t}{2L}v_n\right]}_{I_{\text{hist}}^n}$$

等效电导：$G_{L} = \frac{\Delta t}{2L}$。

**临界阻尼调整（CDA）**（Marti 2004）在检测到电感电流开断或电容电压突加等不连续点时，临时将梯形法则切换为后向欧拉，执行两个 $\Delta t/2$ 半步，利用后向欧拉在恰好一整个步长内提供完全临界阻尼的特性消除数值振荡。

## EMT 建模方法

### 方法 1：串联 RL 模型

仅保留串联电阻和电感，忽略并联电容和电导。

$$Z = (r + j\omega l)\ell, \quad Y \approx 0$$

适用于：**短支路**（$d/\lambda < 0.01$）、**低频阻抗计算**（工频端口阻抗）、变压器漏抗等集总串联阻抗建模。

失效场景：无法表示充电电流、行波反射和高频谐振。

### 方法 2：标称 π 型等效

标准 π 型结构，串联 $Z$，两端并联 $Y/2$。

$$Z = (r + j\omega l)\ell, \quad Y_1 = Y_2 = \frac{Y}{2}$$

适用于：**中等长度线路**（$d/\lambda \approx 0.01\text{--}0.05$）的工频或较低频分析。

失效场景：高频传播（>10 kHz）、行波效应、分布谐振。长线路需要分段级联多个 π 段，分段过多时（>10 段）可能引入虚假振荡（Lessa 2012）。

### 方法 3：精确 π 型电路（Vector Fitting + RLC 综合）

由双曲方程推导精确 $Y_{z\pi}$ 和 $Y_\pi$，通过矢量拟合（Vector Fitting）将有理导纳近似为极点-留数形式：

$$F(s) = \sum_{k=1}^{N_p} \frac{r_k}{s - p_k} + d + es$$

再综合为离散 RLC 支路。Balestero 2022 证明该方法可避免卷积和逆变换，在 ATP/PSCAD/EMTP 中直接实现，支持非线性元件接入。

适用于：**需要宽频精度的任意长度线路**，尤其是频变参数线路和非线性元件共存的场景。

### 方法 4：多段 π 型级联

将线路分为 $N$ 个 π 型单元级联，每个单元采用标称 π 结构。段数选择需满足：

$$\Delta x \leq \frac{\lambda_{\min}}{10} = \frac{v_{\min}}{10f_{\max}}$$

其中 $\Delta x = \ell/N$ 为每段长度，$v_{\min}$ 为最低波速，$f_{\max}$ 为目标最高频率。

Costa 2011 采用修正 Clarke 矩阵处理三相不对称耦合，并通过特征系统解析积分允许使用 $50\text{--}100\,\mu\text{s}$ 的大步长（比传统 EMT 步长大 $5\text{--}20$ 倍）。

适用于：**长线路的时域直接仿真**，需要有限传播近似的 EMT 场景。

### 方法 5：频率相关 π 型（FD-π）

将纵向阻抗 $Z(\omega)$ 替换为频变参数（通过 Gustavsen 宽频无源拟合或 JMarti 标幺化），每段 π 型的串联支路保留频率依赖特性。

适用于：**宽频暂态分析**（雷电、开关操作），参数频变效应不可忽略的场景。

## 形式化表达

### 集总参数边界判据

线路可用集总参数近似的电气长度判据：

$$\frac{d}{\lambda} = \frac{d\cdot f\sqrt{LC}}{v} \ll 1$$

其中 $v = 1/\sqrt{LC}$ 为波速。工程经验阈值：

| 场景 | 近似成立条件 |
|------|------------|
| 工频稳态阻抗 | 任意长度均可用集总参数（幅值误差 < 5%） |
| 低频暂态（< 1 kHz） | $d/\lambda < 0.05$ |
| 开关暂态（~kHz） | $d/\lambda < 0.01$ |
| 雷电/高频（> 100 kHz） | 通常需要分布参数模型 |

### 标称 π vs 精确 π 误差

精确 π 型的误差来源是双曲函数近似本身（理论上零误差），而标称 π 型的相对误差约为：

$$\epsilon_{\text{nominal}} \approx \frac{(\gamma\ell)^2}{24} \quad (\gamma\ell \ll 1)$$

对于 $\gamma\ell = 0.5$（中等电气长度），误差约 1%；对于 $\gamma\ell = 1.0$，误差约 4%。

## 关键技术挑战

### 挑战 1：数值振荡抑制
当梯形法在电感电流开断或电容电压突加处表现为微分器时，会产生持续数值振荡。**CDA 算法**（Marti 2004）通过两个 $\Delta t/2$ 后向欧拉半步提供完全临界阻尼，幅值误差 < 5%，相位误差 < 2°（$f < 0.1/\Delta t$）。

### 挑战 2：参数频变处理
纵向参数 $R(\omega)$、$L(\omega)$ 随频率变化（集肤效应、邻近效应、大地返回阻抗）。解法：矢量拟合（VF）将频变导纳近似为有理函数，综合为 RLC 网络；或采用 Gustavsen 宽频无源拟合。

### 挑战 3：段数与步长选择
多段 π 型需要满足 $\Delta x \leq \lambda_{\min}/10$，段数过多增加计算量，段数过少引入近似误差。对于 $50\,\mu\text{s}$ 步长、$f_{\max}=100\,\text{kHz}$，要求每段电气长度 $\Delta x/\lambda < 0.1$，对应最大段间距约 600 m（架空线）。

### 挑战 4：长线路累积误差
长线路多段级联时，误差逐段累积，虚假振荡可能掩盖物理暂态。精确 π 型通过单支路代替多段标称 π 消除此问题，但需要高质量矢量拟合。

### 挑战 5：三相不对称耦合
非换位三相线路需要模态解耦。Costa 2011 的修正 Clarke 矩阵将标准矩阵在高频段（10–100 kHz）的相位误差从 > 15° 降至 < 2°。

## 量化性能边界

| 建模方法 | 适用频带 | 典型步长 | 加速比（vs 分布参数） | 精度 |
|---------|---------|---------|-------------------|------|
| 串联 RL | < 100 Hz | 任意 | 1.0×（无收益） | 仅工频阻抗 |
| 标称 π 型 | < 1 kHz | 50–100 μs | 1.2–2× | 中等电气长度误差 ~1% |
| 精确 π 型（VF） | 全频段 | 10–50 μs | 3–5× | 理论上精确 |
| 多段 π 型 | < 100 kHz | 1–10 μs | 0.8–1.5×（开销） | 依赖段数 |
| FD-π | < 1 MHz | 0.1–1 μs | 2–4× | 高频精度最高 |

注：加速比数据来源于 Costa 2011（50–100 μs 步长 vs 传统 1–10 μs）和 Balestero 2022（精确 π 型在 ATP 中的实现），原文未报告归一化误差百分比。

## 适用边界与选择指南

| 场景 | 推荐模型 | 关键判据 |
|------|---------|---------|
| 工频阻抗计算、短支路 | 串联 RL | $d/\lambda < 0.01$，只需端口响应 |
| 低频潮流、故障分析 | 标称 π 型 | $d < 100\,\text{km}$，$f < 1\,\text{kHz}$ |
| 任意长度线路宽频仿真 | 精确 π 型（VF） | 需要高精度且 $f$ 覆盖宽频带 |
| 长线路时域直接仿真 | 多段 π 型 | 有现成 EMT 工具，需要传播近似 |
| 雷电、开关、频变参数 | FD-π | $R(\omega)$、$L(\omega)$ 频变不可忽略 |

常见失败模式：
- 用单个 π 型支路分析雷电、陡波开关或电缆高频振荡；
- 只根据物理长度选择模型，不检查波速、频率范围和目标误差；
- 把集总线路端口电压外推为沿线过电压分布；
- 对长电缆忽略频率相关参数和多模传播；
- 用过多 π 段模拟行波却不检查数值步长和矩阵条件数。
