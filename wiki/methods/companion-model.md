# 伴随模型 (Companion Model)

## 概述

伴随模型(Companion Model)是电磁暂态(EMT)仿真中将动态元件（电感、电容）离散化为等效电阻和电流源的数值方法。基于梯形法等数值积分规则，伴随模型使得每个仿真步长内的非时变网络可以使用线性代数方法求解。

伴随模型的核心思想是将连续时间的微分方程转化为离散时间的代数方程，在每个时间步将动态元件替换为等效电路（等效电导并联历史电流源），从而将整个网络转化为纯电阻网络进行求解。

## 基本原理

### 数值积分基础

对于一般的一阶微分方程：
$$\frac{dy}{dt} = f(y, t)$$

数值积分的目标是在离散时间点 $t_n$ 上近似求解 $y_n = y(t_n)$。

#### 梯形法（Trapezoidal Rule）

梯形法基于梯形面积近似积分：
$$\int_{t_n}^{t_{n+1}} f(y, t) dt \approx \frac{\Delta t}{2}[f(y_n, t_n) + f(y_{n+1}, t_{n+1})]$$

由此得到梯形法递推公式：
$$y_{n+1} = y_n + \frac{\Delta t}{2}[f(y_n, t_n) + f(y_{n+1}, t_{n+1})]$$

整理为标准形式：
$$\frac{y_{n+1} - y_n}{\Delta t} = \frac{f(y_{n+1}, t_{n+1}) + f(y_n, t_n)}{2}$$

**精度分析**：梯形法是二阶精度方法，局部截断误差为 $O(\Delta t^3)$，全局误差为 $O(\Delta t^2)$。

**稳定性**：梯形法是A-稳定的，即对于所有 $Re(\lambda) < 0$ 的测试方程 $y' = \lambda y$，数值解都衰减到零。

#### 后向欧拉法（Backward Euler）

后向欧拉法采用当前时刻的导数值：
$$y_{n+1} = y_n + \Delta t \cdot f(y_{n+1}, t_{n+1})$$

标准形式：
$$\frac{y_{n+1} - y_n}{\Delta t} = f(y_{n+1}, t_{n+1})$$

**精度分析**：后向欧拉是一阶精度方法，局部截断误差为 $O(\Delta t^2)$，全局误差为 $O(\Delta t)$。

**稳定性**：后向欧拉是L-稳定的（比A-稳定更强），对于刚性系统有很好的数值阻尼特性。

#### Gear方法（向后微分公式）

Gear方法是多步法，k阶Gear公式：
$$\sum_{j=0}^{k} \alpha_j y_{n+1-j} = \Delta t \cdot \beta_0 f(y_{n+1}, t_{n+1})$$

常用的是二阶Gear方法（BDF2）：
$$\frac{3y_{n+1} - 4y_n + y_{n-1}}{2\Delta t} = f(y_{n+1}, t_{n+1})$$

**精度分析**：k阶Gear方法具有k阶精度。

**稳定性**：Gear方法在一定阶数内是A-稳定的（一阶和二阶），高阶时稳定性区域减小。

### 电感伴随模型详细推导

电感元件的电压-电流关系：
$$v = L\frac{di}{dt}$$

**使用梯形法离散**：

将微分方程改写为：
$$\frac{di}{dt} = \frac{v}{L}$$

应用梯形法：
$$\frac{i_{n+1} - i_n}{\Delta t} = \frac{1}{2}\left(\frac{v_{n+1}}{L} + \frac{v_n}{L}\right)$$

整理：
$$i_{n+1} = i_n + \frac{\Delta t}{2L}(v_{n+1} + v_n)$$

将含 $i_{n+1}$ 和 $v_{n+1}$ 的项分离：
$$i_{n+1} = \frac{\Delta t}{2L}v_{n+1} + \left(i_n + \frac{\Delta t}{2L}v_n\right)$$

定义等效电导和历史电流源：
- **等效电导**：$G_{eq} = \frac{\Delta t}{2L}$
- **历史电流源**：$I_{history} = i_n + G_{eq}v_n = i_{n-1} + 2G_{eq}v_n$

最终伴随模型形式：
$$i_{n+1} = G_{eq}v_{n+1} + I_{history}$$

**电路等效**：电感等效为一个电导 $G_{eq}$ 并联一个电流源 $I_{history}$。

**使用不同积分方法的电感模型**：

| 积分方法 | 等效电导 $G_{eq}$ | 历史电流源 $I_{history}$ |
|---------|------------------|------------------------|
| 梯形法 | $\frac{\Delta t}{2L}$ | $i_n + G_{eq}v_n$ |
| 后向欧拉 | $\frac{\Delta t}{L}$ | $i_n$ |
| Gear-2 | $\frac{2\Delta t}{3L}$ | $\frac{4i_n - i_{n-1}}{3} + \frac{2\Delta t}{3L}v_n$ |

### 电容伴随模型详细推导

电容元件的电流-电压关系：
$$i = C\frac{dv}{dt}$$

**使用梯形法离散**：

将微分方程改写为：
$$\frac{dv}{dt} = \frac{i}{C}$$

应用梯形法：
$$\frac{v_{n+1} - v_n}{\Delta t} = \frac{1}{2}\left(\frac{i_{n+1}}{C} + \frac{i_n}{C}\right)$$

整理求解 $i_{n+1}$：
$$v_{n+1} - v_n = \frac{\Delta t}{2C}(i_{n+1} + i_n)$$

$$i_{n+1} = \frac{2C}{\Delta t}(v_{n+1} - v_n) - i_n$$

分离为伴随模型形式：
$$i_{n+1} = \frac{2C}{\Delta t}v_{n+1} + \left(-i_n - \frac{2C}{\Delta t}v_n\right)$$

定义：
- **等效电导**：$G_{eq} = \frac{2C}{\Delta t}$
- **历史电流源**：$I_{history} = -i_n - G_{eq}v_n = -i_{n-1} - 2G_{eq}v_n$

最终形式：
$$i_{n+1} = G_{eq}v_{n+1} + I_{history}$$

**使用不同积分方法的电容模型**：

| 积分方法 | 等效电导 $G_{eq}$ | 历史电流源 $I_{history}$ |
|---------|------------------|------------------------|
| 梯形法 | $\frac{2C}{\Delta t}$ | $-i_n - G_{eq}v_n$ |
| 后向欧拉 | $\frac{C}{\Delta t}$ | $-\frac{C}{\Delta t}v_n$ |
| Gear-2 | $\frac{3C}{2\Delta t}$ | $-\frac{4i_n - i_{n-1}}{3} - \frac{2C}{\Delta t}v_n$ |

### 电阻伴随模型

电阻是即时元件，不需要离散化：
$$v = R \cdot i \quad \text{或} \quad i = G \cdot v$$

其中 $G = 1/R$ 为电导。

在伴随模型框架下，电阻直接贡献其电导到节点导纳矩阵：
$$G_{eq} = \frac{1}{R}$$

### 电导伴随模型

电导元件同样保持不变：
$$i = G \cdot v$$

等效电导即为电导值本身：
$$G_{eq} = G$$

## 模型类型详解

### 电感伴随模型电路图

```
        i(t)
         ↓
    ┌─────────┐
    │    L    │
    │         │
    └────┬────┘
         │
        v(t)

离散化后等效为:

        i(t)
         ↓
    ┌────┴────┐
    │         │
   ┌┴┐       ┌┐
   │ │ Geq   ││ Ihist
   │ │       ││ ↑
   └┬┘       └┘
    │         │
    └────┬────┘
         │
        v(t)

参数:
  Geq = Δt/(2L)               [S]
  Ihist = i(t-Δt) + Geq·v(t-Δt) [A]
```

### 电容伴随模型电路图

```
        i(t)
         ↓
    ┌─────────┐
    │    C    │
    │         │
    └────┬────┘
         │
        v(t)

离散化后等效为:

        i(t)
         ↓
    ┌────┴────┐
    │         │
   ┌┴┐       ┌┐
   │ │ Geq   ││ Ihist (方向向下)
   │ │       ││ ↓
   └┬┘       └┘
    │         │
    └────┬────┘
         │
        v(t)

参数:
  Geq = 2C/Δt                   [S]
  Ihist = -i(t-Δt) - Geq·v(t-Δt) [A]
```

### 耦合电感伴随模型

对于耦合电感组，电压-电流关系：
$$\mathbf{v} = \mathbf{L}\frac{d\mathbf{i}}{dt}$$

其中 $\mathbf{L}$ 是电感矩阵。

使用梯形法离散：
$$\mathbf{v}_{n+1} = \frac{2}{\Delta t}\mathbf{L}(\mathbf{i}_{n+1} - \mathbf{i}_n) - \mathbf{v}_n$$

求解电流：
$$\mathbf{i}_{n+1} = \frac{\Delta t}{2}\mathbf{L}^{-1}\mathbf{v}_{n+1} + \left(\mathbf{i}_n + \frac{\Delta t}{2}\mathbf{L}^{-1}\mathbf{v}_n\right)$$

等效导纳矩阵：
$$\mathbf{G}_{eq} = \frac{\Delta t}{2}\mathbf{L}^{-1}$$

历史电流源向量：
$$\mathbf{I}_{history} = \mathbf{i}_n + \mathbf{G}_{eq}\mathbf{v}_n$$

## 节点导纳矩阵构建方法

### 基本原理

使用伴随模型后，网络中的所有元件都转化为等效电导和电流源，整个网络变为纯电阻网络。

节点电压方程的标准形式：
$$\mathbf{Y}\mathbf{v} = \mathbf{i}_{src} + \mathbf{i}_{hist}$$

其中：
- $\mathbf{Y} \in \mathbb{R}^{n \times n}$: 节点导纳矩阵（对称、稀疏）
- $\mathbf{v} \in \mathbb{R}^{n}$: 节点电压向量（待求解）
- $\mathbf{i}_{src} \in \mathbb{R}^{n}$: 独立电流源注入向量
- $\mathbf{i}_{hist} \in \mathbb{R}^{n}$: 历史电流源注入向量

### 导纳矩阵构建算法

对于网络中的每个元件，按照以下规则贡献到导纳矩阵：

**1. 连接节点 $k$ 和 $m$ 的电导 $G$**：
- $Y_{kk} \leftarrow Y_{kk} + G$
- $Y_{mm} \leftarrow Y_{mm} + G$
- $Y_{km} \leftarrow Y_{km} - G$
- $Y_{mk} \leftarrow Y_{mk} - G$

**2. 接地电导 $G$（连接节点 $k$ 和地）**：
- $Y_{kk} \leftarrow Y_{kk} + G$

**3. 节点 $k$ 的电流源 $I$**：
- $i_{k} \leftarrow i_{k} + I$

### 元件对导纳矩阵的贡献

| 元件类型 | 连接节点 | 导纳矩阵贡献 | 电流源贡献 |
|---------|---------|-------------|-----------|
| 电阻 $R$ | $k$-$m$ | $Y_{kk}, Y_{mm} += 1/R$; $Y_{km}, Y_{mk} -= 1/R$ | 无 |
| 电感 $L$ | $k$-$m$ | $Y_{kk}, Y_{mm} += \Delta t/(2L)$; $Y_{km}, Y_{mk} -= \Delta t/(2L)$ | $i_k += I_{hist}$; $i_m -= I_{hist}$ |
| 电容 $C$ | $k$-$m$ | $Y_{kk}, Y_{mm} += 2C/\Delta t$; $Y_{km}, Y_{mk} -= 2C/\Delta t$ | $i_k += I_{hist}$; $i_m -= I_{hist}$ |
| 电流源 | $k$→$m$ | 无 | $i_k += I$; $i_m -= I$ |
| 电压源 | $k$-地 | 需要伴随模型或修改节点法 | 诺顿等效 |

### 求解步骤

**每个时间步的求解流程**：

1. **初始化导纳矩阵**（首次或拓扑变化时）
   - 遍历所有元件，按规则填充 $\mathbf{Y}$
   - 对 $\mathbf{Y}$ 进行符号分析（可选，用于优化求解）

2. **构建右端向量**
   - 初始化 $\mathbf{i} = \mathbf{0}$
   - 添加独立电流源贡献
   - 计算各动态元件的历史电流源
   - 添加历史电流源贡献

3. **求解线性系统**
   - 求解 $\mathbf{Y}\mathbf{v} = \mathbf{i}$
   - 可采用直接法（LU分解）或迭代法

4. **更新历史值**
   - 根据求解得到的电压，计算各支路电流
   - 更新各动态元件的历史状态

5. **时间推进**
   - $t \leftarrow t + \Delta t$
   - 返回步骤2

## 数值积分方法对比分析

### 梯形法则特性

**数学形式**：
$$y_{n+1} = y_n + \frac{\Delta t}{2}[f(t_n, y_n) + f(t_{n+1}, y_{n+1})]$$

**特性分析**：
- **精度**: 二阶（$O(\Delta t^2)$）
- **稳定性**: A-稳定（无条件稳定）
- **数值振荡**: 对于突变，会产生无阻尼振荡（虚假振荡）
- **能量守恒**: 对于LC电路，保持能量守恒

**虚假振荡现象**：
当电路发生开关动作或故障时，梯形法会产生数值振荡。例如RL电路中电流突变后，梯形法计算结果会在真实值附近振荡而不衰减。

**临界阻尼修正（CDA）**：
在开关动作时刻临时切换到后向欧拉法一个步长，消除振荡。

### 后向欧拉特性

**数学形式**：
$$y_{n+1} = y_n + \Delta t \cdot f(t_{n+1}, y_{n+1})$$

**特性分析**：
- **精度**: 一阶（$O(\Delta t)$）
- **稳定性**: L-稳定（强稳定）
- **数值阻尼**: 大，快速抑制振荡
- **能量耗散**: 会引入人工能量损耗

**适用场景**：
- 含有快速瞬态或突变的电路
- 刚性系统
- 作为梯形法的振荡抑制手段

### Gear方法特性

**数学形式**（二阶）：
$$\frac{3y_{n+1} - 4y_n + y_{n-1}}{2\Delta t} = f(t_{n+1}, y_{n+1})$$

**特性分析**：
- **精度**: k阶可调
- **稳定性**: 一阶和二阶A-稳定，高阶稳定性减弱
- **启动**: 需要多步启动（使用单步法初始化前几步）
- **变步长**: 天然支持变阶变步长

**适用场景**：
- 刚性微分方程组
- 需要自动阶数和步长控制的场合
- 长时间仿真

### 方法选择指南

| 应用场景 | 推荐方法 | 理由 |
|---------|---------|------|
| 一般EMT仿真 | 梯形法 | 精度高，计算效率高 |
| 含频繁开关 | 后向欧拉或CDA | 抑制振荡 |
| 刚性系统 | Gear方法 | 稳定性好 |
| 高精度需求 | 梯形法+小步长 | 二阶精度 |
| 长时仿真 | Gear方法 | 可变步长节省计算 |

## 开关处理技术

### 理想开关模型

理想开关的两种状态：
- **导通状态**：$R_{on} \approx 0$（如 $10^{-6} \Omega$）
- **关断状态**：$R_{off} \approx \infty$（如 $10^6 \Omega$）

在伴随模型中，开关状态变化通过修改导纳矩阵实现：
- 导通时：添加大电导 $G_{on} = 1/R_{on}$
- 关断时：添加小电导 $G_{off} = 1/R_{off}$

### 开关时刻处理

**插值法确定精确开关时刻**：

当检测到开关状态应在步长内变化时：
1. 线性插值计算精确的开关时刻 $t_{sw}$
2. 以较小的步长 $\Delta t_1 = t_{sw} - t_n$ 积分到开关时刻
3. 改变开关状态，修改导纳矩阵
4. 以步长 $\Delta t_2 = t_{n+1} - t_{sw}$ 积分完成本步

**插值公式**：
$$t_{sw} = t_n + \Delta t \cdot \frac{v_{threshold} - v_n}{v_{n+1} - v_n}$$

其中 $v_{threshold}$ 是开关触发的阈值电压/电流。

### 开关诱导的数值振荡抑制

**临界阻尼调整法（CDA）**：

在检测到开关动作时：
1. 在当前步使用梯形法计算
2. 发现开关动作后，以半步长 $\Delta t/2$ 使用后向欧拉法重算
3. 后续步恢复梯形法

**理论基础**：
后向欧拉的L-稳定性保证它能快速阻尼突变引起的高频分量。

### 多开关协调

当多个开关在同一时间窗内动作时：
1. 按时间顺序排序开关事件
2. 依次处理每个开关事件（可能涉及多个小步长）
3. 确保所有开关状态更新后才进行下一步主积分

## 非线性元件处理

### 非线性电阻的线性化

对于非线性电阻 $i = f(v)$，在每个迭代步使用牛顿-拉夫逊法线性化：

**等效电导**（动态电导）：
$$G_{eq}^{(k)} = \left.\frac{df}{dv}\right|_{v^{(k)}}$$

**等效电流源**：
$$I_{eq}^{(k)} = f(v^{(k)}) - G_{eq}^{(k)}v^{(k)}$$

**迭代过程**：
1. 假设初始电压 $v^{(0)}$
2. 计算 $G_{eq}^{(k)}$ 和 $I_{eq}^{(k)}$
3. 求解线性系统得到 $v^{(k+1)}$
4. 检查收敛：$|v^{(k+1)} - v^{(k)}| < \epsilon$
5. 若未收敛，返回步骤2

### 二极管模型

**指数模型**：
$$i = I_s(e^{v/V_T} - 1)$$

**动态电导**：
$$G_{eq} = \frac{I_s}{V_T}e^{v/V_T}$$

**伴随模型**：
$$i_{n+1} = G_{eq}v_{n+1} + (i_n - G_{eq}v_n)$$

### 磁饱和电感

磁链-电流关系 $\psi = f(i)$ 是非线性的。

**增量电感**：
$$L_{inc} = \frac{d\psi}{di}$$

**分段线性化**：
将磁化曲线分为若干段，每段使用线性电感近似：
$$L_k = \frac{\Delta \psi_k}{\Delta i_k}$$

在伴随模型中，根据当前电流选择对应的线性段参数。

**查表法**：
使用实测的 $\psi$-$i$ 曲线数据，通过插值确定当前工作点的等效电感。

### 电弧模型

电弧的伏安特性是高度非线性的：
$$v_{arc} = f(i_{arc}, \text{arc length}, \text{gas pressure}, ...)$$

通常需要与外部电路方程联立求解，使用迭代法处理耦合。

## 多相系统伴随模型

### 三相电感伴随模型

对于三相耦合电感，电感矩阵：
$$\mathbf{L} = \begin{bmatrix} L_{aa} & L_{ab} & L_{ac} \\ L_{ba} & L_{bb} & L_{bc} \\ L_{ca} & L_{cb} & L_{cc} \end{bmatrix}$$

对称情况下：
$$\mathbf{L} = \begin{bmatrix} L_s & L_m & L_m \\ L_m & L_s & L_m \\ L_m & L_m & L_s \end{bmatrix}$$

伴随模型的导纳矩阵：
$$\mathbf{G}_{eq} = \frac{\Delta t}{2}\mathbf{L}^{-1}$$

**逆矩阵计算**：
对于对称三相电感矩阵：
$$\mathbf{L}^{-1} = \frac{1}{(L_s - L_m)(L_s + 2L_m)}\begin{bmatrix} L_s + L_m & -L_m & -L_m \\ -L_m & L_s + L_m & -L_m \\ -L_m & -L_m & L_s + L_m \end{bmatrix}$$

### 三相电容伴随模型

三相耦合电容的电容矩阵 $\mathbf{C}$，伴随模型导纳：
$$\mathbf{G}_{eq} = \frac{2}{\Delta t}\mathbf{C}$$

### 序分量变换

使用对称分量变换将三相量分解为正序、负序、零序：
$$\begin{bmatrix} I_0 \\ I_1 \\ I_2 \end{bmatrix} = \frac{1}{3}\begin{bmatrix} 1 & 1 & 1 \\ 1 & a & a^2 \\ 1 & a^2 & a \end{bmatrix}\begin{bmatrix} I_a \\ I_b \\ I_c \end{bmatrix}$$

其中 $a = e^{j2\pi/3}$。

在序分量域中，正序、负序、零序网络通常解耦，可以独立求解。

### 多相输电线路

对于分布参数的多相输电线路，需要使用相域模型或模域模型。

**Bergeron模型（单相）扩展到多相**：
使用特征线法，多相线路的伴随模型需要考虑波阻抗矩阵和传播时间矩阵。

## 数值稳定性分析

### A-稳定性分析

测试方程：
$$y' = \lambda y, \quad Re(\lambda) < 0$$

对于梯形法：
$$y_{n+1} = \frac{1 + \frac{\Delta t\lambda}{2}}{1 - \frac{\Delta t\lambda}{2}}y_n$$

放大因子：
$$\rho = \frac{1 + z/2}{1 - z/2}, \quad z = \Delta t\lambda$$

对于所有 $Re(z) < 0$，有 $|\rho| \leq 1$，故梯形法是A-稳定的。

### L-稳定性分析

L-稳定性要求：
1. A-稳定
2. $\lim_{|z| \to \infty}|\rho(z)| = 0$

后向欧拉法的放大因子：
$$\rho = \frac{1}{1 - z}$$

满足 $\lim_{|z| \to \infty}|\rho| = 0$，故是L-稳定的。

梯形法的放大因子在 $|z| \to \infty$ 时趋近于 $-1$，不是L-稳定的。

### 刚性系统稳定性

刚性系统的特征值分布广泛：
$$|\lambda_{max}| / |\lambda_{min}| \gg 1$$

对于刚性系统，需要使用A-稳定或L-稳定的方法，否则步长受稳定性限制过小。

### 步长选择准则

**精度限制**：
$$\Delta t_{acc} \leq \sqrt{\frac{12\epsilon}{|y''|}}$$

**稳定性限制**：
对于显式方法：
$$\Delta t_{stab} \leq \frac{2}{|\lambda_{max}|}$$

对于A-稳定隐式方法（如梯形法），稳定性不限制步长。

## 误差分析

### 局部截断误差

梯形法的局部截断误差：
$$LTE = \frac{\Delta t^3}{12}y'''(t_n) + O(\Delta t^4)$$

后向欧拉的局部截断误差：
$$LTE = \frac{\Delta t^2}{2}y''(t_n) + O(\Delta t^3)$$

### 全局误差累积

全局误差是各步局部误差的累积。对于稳定的数值方法，全局误差与局部截断误差同阶。

梯形法全局误差：$O(\Delta t^2)$
后向欧拉全局误差：$O(\Delta t)$

### 误差估计与步长控制

**嵌入式方法**：
使用不同阶数的方法计算，比较结果估计误差：
$$\epsilon_{est} = |y_{n+1}^{(p)} - y_{n+1}^{(p+1)}|$$

**步长调整**：
$$\Delta t_{new} = \Delta t_{old}\left(\frac{\epsilon_{tol}}{\epsilon_{est}}\right)^{1/(p+1)}$$

其中 $p$ 是方法阶数，$\epsilon_{tol}$ 是容许误差。

### 数值振荡误差

梯形法在开关动作后产生的数值振荡是一种特殊的误差形式，表现为围绕真实值的周期性偏差。

**振荡幅度**：
$$|v_{num} - v_{true}| \propto e^{-\alpha t}$$

对于纯梯形法 $\alpha = 0$（无阻尼），使用CDA后 $\alpha > 0$（有阻尼）。

## 与其他方法的对比

### 与状态空间法的对比

| 特性 | 伴随模型法 | 状态空间法 |
|-----|-----------|-----------|
| 建模方式 | 元件级等效 | 系统级方程 |
| 方程形式 | 节点导纳方程 | 状态方程 $\dot{x} = Ax + Bu$ |
| 求解器 | 线性代数求解器 | 积分器+矩阵指数 |
| 开关处理 | 直接修改导纳矩阵 | 需重新计算系统矩阵 |
| 非线性处理 | 迭代线性化 | 同样需迭代 |
| 稀疏性利用 | 天然支持 | 需特殊处理 |
| 适用场景 | 大规模网络 | 控制分析、小系统 |

**互补使用**：
- 电磁暂态仿真主要使用伴随模型法
- 控制系统设计常使用状态空间法

### 与电磁场法的对比

| 特性 | 伴随模型法（电路级） | 电磁场法 |
|-----|-------------------|---------|
| 建模维度 | 集总参数 | 分布参数 |
| 计算复杂度 | 低（毫秒级） | 高（分钟到小时） |
| 精度 | 电路行为 | 场分布细节 |
| 适用频率 | 低频到中频 | 全频段 |
| 适用场景 | 系统级仿真 | 设备级分析 |

### 与频域法的对比

| 特性 | 时域伴随模型 | 频域分析 |
|-----|------------|---------|
| 分析类型 | 时域波形 | 频域响应 |
| 非线性处理 | 直接支持 | 需线性化或谐波平衡 |
| 暂态分析 | 直接支持 | 需逆变换 |
| 计算效率 | 时间步迭代 | 单次求解 |
| 信息类型 | 全波形 | 稳态响应 |

## 优势与局限

### 优势

1. **简单高效**
   - 每步求解线性系统，计算效率高
   - 利用稀疏矩阵技术，大规模网络也可高效求解
   - [[sparse-matrix-techniques]] - 稀疏矩阵技术

2. **通用性强**
   - 适用于各种元件类型
   - 易于扩展到新元件模型
   - 统一框架处理线性和非线性元件

3. **数值稳定性**
   - 隐式方法无条件稳定
   - 可采用变步长策略

4. **工业验证**
   - 在EMTP、PSCAD等商业软件中广泛应用
   - 经过大量工程实践验证

### 局限

1. **数值误差**
   - 离散化引入截断误差
   - [[numerical-integration-error]] - 数值积分误差
   - 累积误差影响长时间仿真精度

2. **数值振荡**
   - 梯形法在突变后产生虚假振荡
   - [[numerical-oscillation-suppression]] - 数值振荡
   - 需要额外的振荡抑制措施（如CDA）

3. **步长限制**
   - 虽然稳定性不限制步长，但精度要求限制步长
   - 含快速动态时需要小步长

4. **刚性问题**
   - 刚性系统可能需要特殊处理方法
   - 多时间尺度问题效率受限

## 软件实现

### 商业软件

- **EMTP（Electromagnetic Transients Program）**
  - 最早的伴随模型实现
  - 基于梯形法和节点分析法
  - 广泛应用于电力系统暂态分析

- **PSCAD/EMTDC**
  - 图形化界面的EMT仿真工具
  - 采用伴随模型和Dommel算法
  - 支持自定义元件模型

- **ATP（Alternative Transients Program）**
  - EMTP的免费版本
  - 同样基于伴随模型

### 自定义实现

MATLAB/Python实现伴随模型的基本框架：

```matlab
% 伴随模型仿真框架伪代码
for n = 1:N_steps
    % 1. 构建导纳矩阵
    Y = build_admittance_matrix(components, dt);
    
    % 2. 计算历史电流源
    I_hist = compute_history_currents(components, v_prev, i_prev);
    
    % 3. 添加独立源
    I_src = get_source_currents(t(n));
    
    % 4. 求解节点电压
    v = Y \ (I_src + I_hist);
    
    % 5. 更新历史值
    update_component_states(components, v, v_prev);
    
    % 6. 保存结果
    v_history(n,:) = v';
end
```

## 实际应用案例

### 断路器操作过电压分析

使用伴随模型仿真断路器开断感性负载产生的过电压：
- 电感用伴随模型表示
- 断路器用理想开关模型
- 使用CDA抑制开关振荡
- 分析过电压幅值和频率

### 变频器驱动系统

变频器等电力电子装置含有大量开关元件：
- IGBT用理想开关+反并联二极管
- 直流母线电容用伴随模型
- 电机绕组用耦合电感伴随模型

### 雷电冲击分析

雷电波是高频暂态过程：
- 输电线路用分布参数模型或级联伴随模型
- 避雷器用非线性电阻伴随模型
- 需要小步长捕捉波过程
