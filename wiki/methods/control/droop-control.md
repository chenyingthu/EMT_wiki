# 下垂控制 (Droop Control)

## 1. 物理背景与工程需求

下垂控制是在不依赖高速通信的条件下，通过频率-有功（$P$-$f$）和电压-无功（$Q$-$V$）静态关系实现并联系统功率共享的控制方法。在传统电力系统中，同步发电机的调速器自然提供下垂特性；在逆变器主导的电力系统中，下垂控制被重新实现为构网型（GFM）逆变器的一次控制层，用于电压源特性的构建和功率分配。

下垂控制在 EMT 中的工程需求包括：

1. **GFM 逆变器一次控制**：黑启动、孤岛供电和弱网支撑场景中，逆变器需在不依赖外部电网的情况下建立电压和频率（Nguyen 2021）
2. **并联系统功率共享**：多逆变器并联时下垂控制近似分配有功/无功，无需高速通信
3. **构网型逆变器 PQ 能力边界**：下垂控制需在额定电流和 PWM 饱和约束下生成物理可实现的 PQ 运行点（Nurunnabi 2025）
4. **层级控制底座**：下垂作为一次控制与二次恢复、能量管理配合，恢复稳态偏差

## 2. 数学描述

### 2.1 基本下垂关系

标准有功-频率和电压-无功下垂关系：

$$\omega = \omega_0 - m_p (P - P^\star)$$

$$V = V_0 - n_q (Q - Q^\star)$$

其中 $\omega_0$、$V_0$ 为额定频率和电压，$m_p$、$n_q$ 为下垂系数。$m_p$ 越大，频率对有功偏差越敏感，稳态频率偏移也越大。

等效功率共享关系：当多台 GFM 并联时，各单元的频率在稳态相同，因此有功分配近似与下垂系数倒数成正比：

$$m_{p1} P_1 \approx m_{p2} P_2 \approx \dots$$

### 2.2 下垂控制的物理背景

下垂控制模拟同步发电机的自然频率-功率调节特性。同步机转子运动方程中，频率变化率与功率不平衡量的关系可以近似映射到 GFM 逆变器的 $P$-$\omega$ 下垂环，使逆变器在暂态和稳态中表现出类似同步机的频率响应特征。

## 3. EMT 建模方法

### 3.1 基本下垂 + dq 双环（Nguyen 2021）

Nguyen (2021) 在 PSCAD 中构建光储混合 GFM 黑启动模型，控制架构分两级工作：

**一次控制（下垂环）**：根据 PCC 点注入的有功和无功实时调节电压幅值和频率参考：

$$f_g = f_{rated} - m_p (P_{ac} - P_{ref})$$

$$V_g = V_{rated} - n_q (Q_{ac} - Q_{ref})$$

**二次控制（PI 恢复环）**：用 PI 对频率和电压误差积分补偿，消除一次下垂造成的稳态偏差：

$$f_{ref} = f_g + K_{pf} (f_{rated} - f_g) + K_{if} \int (f_{rated} - f_g) dt$$

二次控制的时间尺度比一次慢一个数量级，避免两个层级相互干扰。

**dq 双环内环**：下垂环输出的电压幅值和频率参考经 dq 坐标系下的电压/电流双环嵌套处理后生成调制参考，最终通过 PWM 驱动功率开关：

```
每个控制周期:
1. 测量 P_ac, Q_ac
2. 下垂环: f_g = f_rated - m_p*(P_ac - P_ref), V_g = V_rated - n_q*(Q_ac - Q_ref)
3. 二次环 (慢): f_ref = f_g + PI(f_error), V_ref = V_g + PI(V_error)
4. 电压环 (dq): 生成 dq 电流参考
5. 电流环 (dq): 生成调制电压参考
6. PWM: 驱动功率开关
```

### 3.2 带虚拟阻抗的下垂

在基本下垂基础上串联虚拟阻抗，补偿线路阻抗不一致导致的功率分配偏差：

$$v_{ref} = V_g - Z_v \cdot i_{abc}$$

其中 $Z_v = R_v + j\omega L_v$ 为虚拟阻抗。虚拟阻抗的引入使等效输出阻抗显式化，降低线路阻抗差异对功率共享精度的影响，同时引入电压降落需在设计中补偿。

### 3.3 CPID（受控比例-积分-微分）下垂（Nurunnabi 2025）

Nurunnabi 2025 提出的 CPID 方法在 FDR（基础下垂）基础上加入微分项以改善动态响应：

$$f_g = f_{rated} - m_p (P_{ac} - P_{ref}) - m_d \frac{d(P_{ac} - P_{ref})}{dt}$$

其中 $m_d$ 为微分增益，用于加速对有功突变的响应。实验表明 CPID 在动态负载下的频率偏差 <0.37%，优于 FDR。

### 3.4 EVR（增强型电压调节）下垂（Nurunnabi 2025）

EVR 策略的核心是：在 RIC（额定电流约束）与 PWM 饱和约束交集形成的 PQ 运行域内，优先保证电压支撑能力而非有功输出。具体实现为在下垂限幅时保留 $V_g$ 调节余量，使系统在故障穿越期间仍能提供无功支撑。

**PQ 能力边界数学描述**（Nurunnabi 2025）：

1. **RIC 约束**：PCC 电流幅值不超过额定值 $I_{rated}$，形成 P-Q 平面的电流圆边界：

$$\vec{I}_{PCC} \leq I_{rated}$$

在标幺制下，基于额定 PCC 电压 $V_{rated}$ 和额定功率 $S_{rated}$，得：

$$\frac{\sqrt{P^2 + Q^2}}{|V_{PCC}|} \leq I_{rated}$$

2. **PWM 饱和约束**：逆变器输出端电压受直流母线电压和调制方式限制。对于 SPWM，最大相电压幅值为 $E_{max} = V_{dc}/2$；对于 SVPWM，$E_{max} = V_{dc}/\sqrt{3}$。合成电压需满足：

$$\vec{E}_{inv} \leq E_{max}$$

3. **PQ 运行域**：联立 RIC 与 PWM 约束，得下垂控制在 P-Q 平面内的可行域交集。EVR 通过在限幅时优先保留 $Q$ 而非 $P$ 的可调度余量，确保 $V_g$ 仍有调节空间。

### 3.5 自适应下垂

$m_p$、$n_q$ 根据运行点在线调整：

$$m_p(t) = m_{p0} + k_m \cdot \frac{|P - P^\star|}{S_{rated}}$$

$$n_q(t) = n_{q0} + k_n \cdot \frac{|Q - Q^\star|}{S_{rated}}$$

其中 $k_m$、$k_n$ 为自适应增益。自适应下垂在负载变化范围宽的微电网中能兼顾稳态精度和动态响应，但需状态估计环节，增加了实现复杂度。

## 4. 关键技术挑战

### 4.1 PQ 能力边界约束与下垂指令的冲突

当下垂环根据 $P$、$Q$ 偏差计算出参考点超出 RIC 与 PWM 饱和约束交集时，内环限幅激活导致外环下垂指令被覆盖，GFM 暂态行为偏离下垂预期。EVR 策略通过优先保留电压支撑能力部分缓解此问题，但无法完全消除约束冲突。

### 4.2 虚拟阻抗设计与电压降落的矛盾

虚拟阻抗增大可改善功率共享精度，但同时引入更大电压降落，在弱网场景中可能导致 PCC 电压进一步跌落。设计时需在共享精度和电压维持之间做权衡。

### 4.3 时间尺度分离被破坏导致环路交互失稳

下垂环带宽必须远低于电压环和电流环带宽。若下垂系数 $m_p$ 过大，频率对有功偏差的响应速度与内环带宽重叠，引起环路交互振荡。

### 4.4 多逆变器并联的功率共享精度

线路阻抗 $(R+jX)$ 的不一致性（尤其是 $R/X$ 较大时）、滤波器参数差异和测量位置偏差均导致实际功率分配偏离下垂系数预设值。倒下垂（$P$-$V$、$Q$-$f$）可部分补偿高 $R/X$ 场景的共享误差，但增加了控制复杂度。

### 4.5 故障期间下垂与 IEEE 1547 无功支撑要求的冲突

IEEE 1547 标准要求 GFM 在故障期间提供无功电流支撑，优先级高于有功下垂。此时下垂关系被让位于电压穿越逻辑，需要在控制架构中显式处理模式切换。

## 6. 适用边界与选择指南

### 场景-方法选择

| 应用场景 | 推荐方法 | 原因 |
|----------|----------|------|
| 微电网并联逆变器（无高速通信） | 基本下垂 + 虚拟阻抗 | 实现简单，近似共享 |
| 黑启动/孤岛供电 | FDR + 二次 PI | 稳态精度由二次环保证 |
| 动态负载（频繁功率变化） | CPID | 微分项加速动态响应 |
| 故障穿越（需电压支撑） | EVR | 限幅时保留无功支撑能力 |
| 线路阻抗差异大（高 $R/X$） | 倒下垂 + 虚拟阻抗 | 改善高 $R/X$ 场景功率分配精度 |
| 弱网/远距离并联 | 自适应下垂 | 运行点变化时参数在线调整 |

### 失效模式

| 失效场景 | 原因 | 后果 |
|----------|------|------|
| 稳态频率/电压偏移 | 下垂固有比例特性 | 需二次控制器补偿 |
| 线路阻抗不一致导致共享误差 | $R/X$ 显著、滤波器差异、测量偏差 | 功率分配偏离下垂预设 |
| 下垂参考超出 PQ 能力边界 | 额定电流/PWM 饱和约束 | 内环限幅，外环下垂失效 |
| 限流/保护覆盖下垂 | 下垂参考被故障旁路 | GFM 暂态偏离下垂预期 |
| 时间尺度分离被破坏 | 下垂带宽接近内环带宽 | 环路交互失稳 |

## 量化性能边界

**Nguyen 2021 光储GFM黑启动下垂控制（改进IEEE 9节点）**:
- 提出一次下垂（P-f/Q-V）+ 二次PI恢复的两级控制架构
- 下垂环根据PCC有功/无功实时调节电压幅值和频率参考
- 二次PI积分补偿消除一次下垂造成的稳态频率/电压偏移
- 黑启动完成时稳态电压与优化解误差 <1%，频率精确收敛至 60 Hz
- 覆盖变压器励磁、线路充电和负荷阶跃等多种工况
- 验证平台：PSCAD 高保真 EMT 模型，光储混合电站

**Nurunnabi 2025 GFM逆变器PQ能力与下垂边界**:
- 系统界定 GFM 逆变器下垂控制受额定电流约束（RIC）和 PWM 饱和约束的双重限制
- CPID（受控比例-积分-微分）方法引入微分项改善动态响应，频率偏差 <0.37%
- EVR（增强型电压调节）在限幅时保留电压支撑能力，优于 FDR
- 动态负载时电压偏差 <2.9%，满足 IEEE 1547 标准
- 含 L、LC、LCL 三种滤波器拓扑的 PQ 运行域对比（SVPWM 增强 PQ 容量）
- 验证方式：EMT 仿真 + 实时 HIL

**设计参数约束（据方法推断）**:
- 下垂系数 $m_p$ 决定频率对有功偏差的灵敏度：$m_p$ 越大稳态偏差越大，但功率共享响应越快
- 多机并联时功率分配误差与线路阻抗不一致性正相关：$R/X$ 显著时需改用 P-V/Q-f 倒下垂
- 时间尺度分离是下垂与内环稳定的前提：下垂环带宽必须远低于电压环和电流环
- 下垂参考点超出 PQ 能力边界时需启用限流或抗饱和机制，此时理想下垂关系被覆盖
- IEEE 1547 要求故障期间无功电流支撑优先，下垂关系需让位于电压穿越逻辑
