# 双闭环PI控制器 (Dual-Loop PI Controller)

## 定义

双闭环PI控制器（Dual-Loop PI Controller）是电力电子变流器中采用的级联控制结构，由外环（慢速）和内环（快速）两级PI控制器串级组成。外环根据功率、电压或频率等稳态目标生成电流参考；内环以更高带宽跟踪该电流参考并输出调制电压或等效控制量。

在EMT仿真中，双闭环PI控制器的核心挑战在于：**控制器参数的离散化实现、采样延迟的等效建模、以及限幅和抗饱和策略的时域表示**，这些因素共同决定了控制器在EMT中的精度和数值稳定性。

## EMT中的作用

双闭环PI控制器在EMT仿真中承担两类任务：

1. **并网变流器控制接口**：作为VSC/逆变器/MMC与外部电路的连接桥梁，外环跟踪功率或电压目标，内环实现电流跟踪和调制输出
2. **构网型控制骨架**：在构网型（GFM）变流器中，双闭环PI结构可与下垂控制、虚拟同步控制（VSG）等协同工作

关键EMT建模问题包括：
- 采样—计算—输出的链路延迟（通常1.5~2个步长）
- dq坐标系下的交叉耦合项补偿精度
- 积分饱和导致的 windup 效应及其对故障穿越的影响
- PLL动态与内环控制的交互（在弱电网条件下尤为显著）

## 核心机制：级联控制结构

### 控制器架构

```
         ┌─────────────────────────────────────────────────────────┐
         │                    双闭环PI控制器                       │
         │                                                         │
         │  ┌──────────┐      ┌──────────┐      ┌──────────────┐   │
         │  │  外环PI  │ ──►  │  内环PI  │ ──►  │ PWM/调制接口  │   │
         │  │ (慢速)   │      │ (快速)   │      │              │   │
         │  └──────────┘      └──────────┘      └──────────────┘   │
         │       ▲                ▲                                 │
         │       │                │                                 │
         │  参考/测量差        电流参考与实测                      │
         └─────────────────────────────────────────────────────────┘
```

**外环输入**：直流电压、有功功率、无功功率、交流电压或频率附加信号  
**内环输入**：外环输出的 $i_d^*$、$i_q^*$ 电流参考 + 实测电流  
**同步参考**：PLL相位或内部振荡器相位 $\theta$  
**执行输出**：PWM脉冲、开关状态或等效电压源

### 电流内环数学表达

在同步旋转坐标系（dq系）下，对于L滤波器并网模型，电流状态方程为：

$$L\frac{di_d}{dt} = v_d - v_{gd} - Ri_d + \omega L i_q \tag{1}$$

$$L\frac{di_q}{dt} = v_q - v_{gq} - Ri_q - \omega L i_d \tag{2}$$

式(1)(2)中，$v_d$、$v_q$ 为换流器输出电压的d/q轴分量，$v_{gd}$、$v_{gq}$ 为电网电压的d/q轴分量，$i_d$、$i_q$ 为并网电流，$\omega L i_q$ 和 $-\omega L i_d$ 为交叉耦合项。

电流内环PI控制器的输出为：

$$v_d^* = K_{p_i}\left(i_d^* - i_d\right) + K_{i_i}\int\left(i_d^* - i_d\right)dt + \omega L i_q + v_{gd} \tag{3}$$

$$v_q^* = K_{p_i}\left(i_q^* - i_q\right) + K_{i_i}\int\left(i_q^* - i_q\right)dt - \omega L i_d + v_{gq} \tag{4}$$

其中 $K_{p_i}$、$K_{i_i}$ 为内环PI参数，$v_d^*$、$v_q^*$ 为调制电压参考。

### 外环数学表达

外环PI控制器根据功率或电压目标生成电流参考：

$$i_d^* = K_{p_o}\left(P_{ref} - P\right) + K_{i_o}\int\left(P_{ref} - P\right)dt \tag{5}$$

$$i_q^* = K_{p_o}\left(V_{dc,ref} - V_{dc}\right) + K_{i_o}\int\left(V_{dc,ref} - V_{dc}\right)dt \tag{6}$$

其中式(5)为有功功率外环，式(6)为直流电压外环（常用于VSC-HVDC）。对于交流电压控制，$i_q^*$ 由交流电压误差生成。

### PI控制器标准形式

PI控制器时域表达式为：

$$u(t) = K_p e(t) + K_i \int_0^t e(\tau)d\tau \tag{7}$$

离散化（后向欧拉）形式为：

$$u_k = u_{k-1} + K_p(e_k - e_{k-1}) + K_i T_s e_k \tag{8}$$

其中 $T_s$ 为采样步长。

## 解耦项与补偿机制

### dq交叉耦合解耦

式(3)(4)中的 $\omega L i_q$ 和 $-\omega L i_d$ 项实现d/q轴解耦。解耦后的等效d轴电流环为：

$$L\frac{di_d}{dt} + Ri_d = v_d^* - v_{gd} \tag{9}$$

当滤波器参数准确、采样延迟较小、运行点接近设计点时，解耦项可有效减少轴间耦合。

### 电网电压前馈

电网电压前馈项 $v_{gd}$、$v_{gq}$ 补偿电网扰动对电流跟踪的影响。在强电网条件下，前馈可显著改善电流环动态响应。

### 抗积分饱和（Anti-windup）

当控制输出达到限幅值时，积分项继续累积会导致恢复时过冲。常用策略包括：

**条件积分法**：当输出饱和时，停止积分更新

$$e_{sat} = u_{sat} - u_{free}$$
$$\text{if } |e_{sat}| > \epsilon: \text{ freeze integral}$$

**反算抗饱和（Back-calculation）**：将饱和误差反馈到积分项

$$u = u_{sat} + K_{aw}(u_{sat} - u_{free})$$

其中 $K_{aw} > 0$ 为抗饱和增益。

## EMT建模方法

### 方法1：离散PI + 固定步长（基础）

将PI控制器按后向欧拉法离散化，在EMT求解器中逐步更新：

```
for each time step:
    e = ref - measured
    integral = integral + Ki * Ts * e
    u = Kp * e + integral
    if |u| > u_lim:  # saturation
        u = sign(u) * u_lim
        integral = u - Kp * e  # anti-windup
```

**特点**：实现简单，计算量小，但忽略采样延迟和PWM占空比量化效应

**适用场景**：稳态运行、控制器参数设计初步验证

### 方法2：采样延迟等效 + 修正PI参数

采样-计算-输出链路引入约1.5~2个步长的延迟，在离散PI中建模为：

$$u_k = K_p e_k + K_i T_s \sum_{j=0}^{k} e_j - \alpha_d u_{k-1} \tag{10}$$

其中 $\alpha_d$ 表征延迟效应。参数需根据实际延迟时间修正。

**特点**：保留延迟影响，参数需结合具体硬件平台整定

**适用场景**：需要精确模拟控制器动态的实时仿真

### 方法3：dq解耦 + 电网阻抗建模

将电网等效阻抗（含串抗）纳入电流环设计：

$$L\frac{di_d}{dt} = v_d - (R_g + R)i_d - (\omega L_g + \omega L)i_q \tag{11}$$

在弱电网条件下（短路比SCR < 3），电网阻抗变化显著，解耦项需包含电网分量。

**特点**：适用于弱电网场景，但需准确获取电网等效参数

**适用场景**：SCR较低的并网变流器分析

### 方法4：平均值模型接口

在平均值模型（AVM）中，双闭环PI控制器不产生PWM脉冲，而是输出等效电压：

$$v_{dq}^{av} = f(i_{dq}^{ref}, \text{PI参数}, v_{g}) \tag{12}$$

电流内环被吸收进受控电压源模型，仅外环作为控制接口保留。

**特点**：大步长（100~500 μs）可行，忽略开关纹波

**适用场景**：需要长时间仿真的功率调度分析

### 方法5：阻抗模型中的等效控制延迟

在阻抗稳定性分析中，控制器的采样延迟等效为相位滞后：

$$G_{delay}(j\omega) = e^{-j\omega T_d} \approx 1 - j\omega T_d \tag{13}$$

该延迟与PLL、内环带宽交互，共同决定高频阻抗特性。

**特点**：保留相位信息，适合稳定性分析

**适用场景**：并网变流器小信号稳定性评估

## 量化性能边界

| 指标 | 典型范围 | 说明 |
|------|----------|------|
| 内环带宽 | 100~500 Hz | 取决于滤波器参数、采样步长和控制器硬件 |
| 外环带宽 | 10~50 Hz | 通常为内环的1/5~1/10 |
| 采样延迟 | 1.5~2.0 Ts | Ts为EMT仿真步长 |
| dq解耦精度要求 | ±5% 参数偏差 | 解耦效果对参数敏感 |
| 抗饱和恢复时间 | 5~20 ms | 与限幅深度和Ki相关 |
| 弱网稳定边界 | SCR > 2.5 | 低于此值需特殊控制策略 |

**来自 [[characteristic-analysis-of-high-frequency-resonance-of-flexible-high-voltage-dir]] 的证据**：MMC-HVDC中PLL、内外环、环流抑制和延时共同影响高频阻抗特性，在550 Hz~2 kHz频段可能诱发谐振失稳。

**来自 [[dynamic-performance-of-embedded-hvdc-with-13&14]] 的证据**：在PSCAD/EMTDC仿真中，叠加频率控制的双环PI结构可使故障后临界切除时间（CCT）提高约22.93%。

## 关键技术挑战

### 挑战1：采样延迟与离散化精度

采样-计算-输出的链路延迟（约1.5~2 Ts）导致相位滞后，在高频段与滤波器谐振交互可能引发不稳定。传统连续PI参数在离散实现中可能失效。

**缓解策略**：离散PI参数需根据延迟时间修正；高频场景建议使用双线性变换（Tustin）并加入延迟补偿。

### 挑战2：dq解耦项的参数敏感性

交叉耦合项 $\omega L i_q$ 和 $-\omega L i_d$ 假设滤波器参数准确。在参数不确定性较大（如LCL滤波器谐振）或运行点变化时，解耦项可能不再等效于理想补偿。

**缓解策略**：弱网场景下将电网阻抗纳入解耦计算；或采用自适应解耦项。

### 挑战3：弱电网条件下的PLL交互

跟网型控制在弱电网中依赖PLL跟踪电网相位。PLL动态与内环控制争夺带宽，可能引发低频振荡（10~50 Hz）。

**缓解策略**：降低PLL带宽或采用无PLL的虚拟同步控制（VSG）。

### 挑战4：积分饱和与故障穿越

限流模式下，PI积分项持续累积可能导致恢复时过冲，影响故障穿越能力。

**缓解策略**：显式抗积分饱和（Back-calculation或条件积分）；限幅与模式切换协同设计。

### 挑战5：多速率接口的一致性保持

外环（慢速）与内环（快速）在不同时间尺度更新，在多速率EMT仿真中需要保持接口一致性。

**缓解策略**：外环输出经保持器（Zero-order Hold）传递给内环；内环状态在接口时刻重置。
