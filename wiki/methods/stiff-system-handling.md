# 刚性系统处理 (Stiff System Handling)

## 定义与边界

刚性系统处理是针对同一 EMT 模型中快衰减模态、慢动态和离散事件共存时的数值策略。刚性不是单纯“系统很复杂”，而是指稳定性要求的时间步长远小于精度要求的时间步长，或某些高频数值模态在常用积分器下难以衰减。

本页关注处理策略：如何识别刚性、选择 L 稳定积分、处理事件后历史项和协调多时间尺度。具体积分公式见 [[numerical-integration]]；多速率分区见 [[multirate-method]]。

## EMT 中的作用

电力电子开关、故障切除、限幅控制、饱和磁化曲线和长短时间常数并存都会使 EMT 离散系统表现出刚性。若处理不足，仿真可能出现：

- 显式方法必须使用远小于研究目标频带所需的步长。
- 梯形法在不连续事件后保留持续数值振荡。
- 非线性迭代收敛困难或历史源与新拓扑不一致。
- 实时仿真为了满足 deadline 被迫使用过度简化的阻尼策略。

## 核心机制

对线性化系统 $\dot{x}=Ax$，若 $A$ 的特征值实部跨度很大，快模态会限制显式步长。常用刚性指标是特征值实部或时间常数的跨度，但在实际 EMT 中还应结合开关事件和控制限幅判断。

处理刚性的核心工具包括：

- 使用后向欧拉、DIRK、BDF 等隐式或 L 稳定方法阻尼高频数值模态。
- 在事件后临时切换到阻尼较强的积分步骤，再恢复常规积分。
- 对快慢子系统采用 [[multirate-method]] 或移频/动态相量模型。
- 对非线性元件使用一致的线性化、牛顿迭代和历史源重构。

以梯形法为例，其稳定函数在刚性极限趋近 $-1$，因此突变后误差可能步间换号而不衰减。后向欧拉和合适的 DIRK 方法在刚性极限趋近 $0$，可作为事件后阻尼机制。

## 分类与变体

| 策略 | 机制 | 适用边界 |
|------|------|----------|
| 后向欧拉阻尼 | 用一阶 L 稳定步骤衰减高频误差 | 可能局部降阶和过度阻尼 |
| 临界阻尼调整 | 事件后插入阻尼步骤，再恢复梯形法 | 需要可靠事件检测和历史项重构 |
| 2S-DIRK/3S-SDIRK | 用多阶段隐式法兼顾阶数和刚性阻尼 | 阶段数增加；非线性仍需迭代 |
| BDF/Gear | 用多步隐式公式处理平滑刚性动态 | 频繁事件会破坏多步历史 |
| 多速率处理 | 快动态小步长、慢动态大步长 | 接口稳定性和采样规则成为关键 |
| 固定导纳等效 | 降低开关导致的矩阵结构突变 | 需检查虚拟损耗和初始误差 |

## 适用边界与失败模式

- 刚性处理不是把所有高频分量都滤掉；若研究目标包括宽频暂态，阻尼策略必须与目标频带匹配。
- 事件检测不完整时，CDA 类策略可能漏掉线路行波、控制限幅或非线性工作点突变。
- BDF 和 Gear 方法在频繁开关场景中需要重启动；直接保留旧历史可能产生伪误差。
- 简化牛顿法和固定雅可比只适合工作点变化较小的非线性区间。
- 多速率分区若把强耦合刚性模态切在接口上，可能比单速率更不稳定。

## 与相关页面的关系

- [[numerical-integration]] 提供 A 稳定、L 稳定、DIRK、BDF 等基础。
- [[interpolation-method]] 用于事件时刻定位，但不单独保证刚性稳定。
- [[multirate-method]] 是多时间尺度刚性的一种分区处理。
- [[fixed-admittance]] 缓解频繁开关导致的矩阵重构，但不自动保证精度。
- [[trapezoidal-rule]] 是常见基线；其非 L 稳定性解释了许多事件后振荡。
- [[backward-euler]] 是常见阻尼步骤；其边界是低阶和数值耗散。
