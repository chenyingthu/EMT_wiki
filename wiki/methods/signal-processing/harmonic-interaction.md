# 谐波交互机制 (Harmonic Interaction Mechanism)

## 定义

谐波交互机制（Harmonic Interaction Mechanism）描述电力系统中不同设备、网络阻抗、控制环节和频率分量之间通过电压、电流或控制变量形成的耦合关系。它关注的不只是"系统中存在谐波"，而是**谐波分量如何传播、叠加、调制、反馈**，并在特定条件下影响换相、保护、振荡或稳定裕度。

在电磁暂态（EMT）仿真领域，谐波交互分析的核心挑战在于：**多频率分量同时存在且相互耦合**，设备侧的谐波电流受谐波电压影响，网络侧的谐波电压又受谐波电流调制，形成闭环交互关系。这种交互在以下场景中尤为突出：

- 多馈入 LCC-HVDC 系统中，交流故障激发的谐波经耦合网络传递至远端换流站
- 并联逆变器或 MMC 系统中，控制延迟、PLL、滤波器和网络阻抗形成宽频振荡
- 谐波源、滤波器和频率相关线路/电缆模型之间形成放大路径
- 频域诊断结果与 EMT 波形中的失稳、过电压或保护误动之间的可解释联系

## EMT 中的角色

EMT 仿真能保留开关、控制和网络暂态，因此常用于观察谐波交互的时域过程。具体而言：

1. **多频率耦合捕获**：EMT 可同时追踪基波、谐波和边带分量的时域演化，无需事后 FFT
2. **非线性闭环验证**：限幅、保护动作、离散控制延迟等非线性环节的谐波响应只能在时域验证
3. **交叉频率交互**：开关频率与基波频率的相互作用（如载波边带）在 EMT 中自然保留
4. **宽频阻抗交互**：PLL 控制延迟与长线路分布电容的交互在 500Hz~2kHz 频段呈现负阻感特性

谐波交互分析则帮助解释 EMT 仿真中观察到的以下问题：

- 多馈入 LCC-HVDC 中，交流故障和换流器谐波如何影响远端逆变器关断角
- 并联逆变器或 MMC 系统中，控制延迟、PLL、滤波器和网络阻抗如何形成宽频振荡
- 谐波源、滤波器和频率相关线路/电缆模型之间是否可能形成放大路径
- 频域诊断结果与 EMT 波形中的失稳、过电压或保护误动之间是否存在可解释联系

这些分析都应绑定拓扑、控制模型、故障类型和运行点。单个算例中的交互路径不能直接写成一般规律。

## EMT 建模方法与核心机制

### 机制一：多馈入 LCC-HVDC 谐波传递与关断角偏移

多馈入 LCC-HVDC 系统中，交流短路故障激发谐波，经耦合网络传递至远端换流站，改变逆变器关断角，诱发并发换相失败（CCF）。

**核心方程**：谐波电压传递方程描述故障母线至远端母线的 $h$ 次谐波传递关系：

$$\dot{U}_{2h} = \frac{Z_{2h}}{Z_{12h} + Z_{2h}} \dot{U}_{1h}$$

其中 $Z_{12h}$ 为母线间耦合通道 $h$ 次阻抗，$Z_{1h}$、$Z_{2h}$ 为受端系统 $h$ 次等效阻抗。传递系数幅值稳定在 0.35~0.62 区间。

**关断角修正公式**（Yao 等 2023）：

$$\gamma' = \arccos\left(\frac{\sqrt{2} U_{ac} \cos \alpha}{U_{dc}}\right) - \phi - \mu'$$

其中 $\phi$ 为谐波引起的换相电压过零点偏移角，$\mu'$ 为修正后的重叠角。计及谐波影响后，逆变器实际关断角平均减小 2.8°~4.5°，CCF 概率提升约 35%。

### 机制二：并联变流器非特征谐波环流

多台并列运行变流器之间形成的非特征次谐波环流（720Hz 附近），由载波相位差引发，级联 H 桥拓扑因直流侧电压不平衡更易受影响。

**SPWM 双重傅里叶级数模型**（Hu 等 2016）：

$$u_o(t) = \sum_{n=1}^{\infty} [A_{0n} \cos(ny) + B_{0n} \sin(ny)] + \sum_{m=1}^{\infty} [A_{m0} \cos(mx) + B_{m0} \sin(mx)] + \sum_{m=1}^{\infty} \sum_{n=\pm 1}^{\pm \infty} [A_{mn} \cos(mx+ny) + B_{mn} \sin(mx+ny)]$$

其中 $x = \omega_c t + \theta_c$（载波相位），$y = \omega_r t + \theta_r$（调制波相位）。当两台变流器载波相位差为 180° 时，720Hz 谐波在变流器侧占比高达 60%，而系统侧含量极小。

**抑制策略对比**：

| 策略 | 环流抑制效果 | 副作用 |
|------|------------|--------|
| 高通滤波器 | 最彻底 | 无 |
| 增大连接电抗（3mH→5mH） | 降低 30%~40% | 无功占用增加，直流电压利用率下降 |
| 提高开关频率（6.4kHz→12.8kHz） | 降低 50% | 开关损耗增加 100% |
| 差异化开关频率 | 中等 | 控制器一致性变差 |
| PLL 载波同步 | 理论最优 | 依赖理想通信 |

### 机制三：MMC-HVDC 链路延时高频振荡

柔性直流工程中，链路延时（350μs~550μs）使 MMC 高频阻抗呈现"负电阻电感"特性，与长交流线路分布电容交互导致 700Hz~1.8kHz 振荡。

**时滞状态空间模型**（Zhang 等 2022）：

$$\dot{\Delta x} = A_0 \Delta x + A_1 \Delta x(t - \tau)$$

特征方程 $\det(sI - A_0 - A_1 e^{-\tau s}) = 0$ 含有超越项 $e^{-\tau s}$，传统特征值法难以直接求解。采用广义特征根法构造矩阵对 $(V, U)$，通过求解广义特征值直接获取时滞稳定裕度：

$$\tau_{wd} = \min_{1 \le i \le 2mn} \frac{\theta_i}{\omega_i}$$

**量化边界**：电流内环比例系数 $k_p$ 从 5.0 增至 7.5 时，时滞稳定裕度从 347.90μs 下降至 240.25μs，临界振荡频率从 927.63Hz 上升至 1147.11Hz。

**MMC 高频阻抗模型**（Guo 等 2020）：

$$Y_{mmc} = T_{p1}^{-1} (E_{2×2} - G_{de} N_1 M_8)^{-1} N_2 \cdot (T_{p1} + T_{p2} T_{PLL}) + T_{p3} T_{PLL}$$

其中 $G_{de} = e^{-T_{de}s}$ 为链路延时环节。链路延时临界失稳阈值为 165μs，超过该值即呈现多频段负阻感特性。

**三阶阻尼控制器**：

$$F_{damp} = \frac{k_s s}{s + 2\pi f_{HPF}} \cdot \frac{2\pi f_{LPF1}}{s + 2\pi f_{LPF1}} \cdot \frac{2\pi f_{LPF2}}{s + 2\pi f_{LPF2}}$$

投入阻尼控制器后，1.8kHz 处相位差从 185° 降至 177°，0.65kHz 处从 181° 降至 164°。

### 机制四：跟网型换流器 PLL 同步失稳

传统 RMS 模型（恒定导纳矩阵代数模型）无法捕捉跟网型换流器（GFL）中 PLL 与网络交互导致的失稳。电感 "di/dt" 效应是引发 PLL 相关小信号失稳的关键物理机制。

**RMS+ 增强电路模型**（Carreño 等 2026）：

$$u_{dq}^{RMS+} = i_{dq}(jX) + L \frac{di_{dq}}{dt}\bigg|_{PLL} + V$$

通过慢快系统理论分离时间尺度：电磁暂态（快动态，毫秒级）视为边界层，PLL 动态（慢动态，百毫秒级）视为简化流形。

**分岔稳定判据**：

- **跨临界分岔**：$K_p > \frac{1}{i_P L}$ 时发生
- **Hopf 分岔**：$K_p \sqrt{V^2 - (i_P X)^2} - K_i i_P L > 0$ 时稳定

短路比 SCR < 2.0 时，Hopf 分岔临界功率下降约 40%。状态变量降维效率：对于含 N 节点和 M 个 GFL 换流器的系统，EMT 状态数 ≈ 2N+M，RMS+ 状态数 ≈ M，计算复杂度从 $O((2N+M)^3)$ 降至 $O(M^3)$。

### 机制五：频变网络等值的动态谐波域分析

在 EMT 仿真中用网络等值替代外部复杂电网时，需保留线路、电缆、变压器等线性网络的频率相关端口行为，并能在谐波畸变和暂态过程中分析各次谐波动态。

**动态谐波域（DHD）方程**（Karami 等 2021）：

$$x(\tau) = \sum_{h=-\infty}^{\infty} X_h(t) e^{jh\omega\tau}$$

$$X_k^d = \alpha_d X_{k-1}^d + B_d V_{k-1}^d, \quad I_k^d = \tilde{C}_d X_k^d + G_d V_k^d$$

**矢量拟合（VF）极点-留数模型**：

$$f(s) = \sum_{m=1}^{N} \frac{r_m}{s - a_m} + d + se$$

Y 矩阵 VF 拟合精度极高：幅值最大偏差低于 $10^{-14}$。平衡实现降阶后，系统从全阶降至 96 阶，计算负担降低约 58%，加性误差严格控制在 <0.05。

### 机制六：谐波相量域协同仿真

**谐波相量域（HPD）状态空间方程**（Shu 等 2021）：

$$\frac{d \langle x \rangle_k^{HD}(t)}{dt} = A \langle x \rangle_k^{HD}(t) + B \langle u \rangle_k^{HD}(t) - \langle x \rangle_k^{HD}(t) T\left(k, -\frac{\pi}{2k\omega_s}\right)$$

HPD 协同仿真核心优势：仿真步长从传统 EMT 的微秒级扩展至 500μs，系统方程维数仅与网络节点数成正比（从状态变量数×谐波阶数降至约 80%），同时输出瞬时波形与谐波相量波形。

## 形式化表达

### 谐波交互通用框架

在任意 EMT 网络中，设备谐波电流受谐波电压影响时，可写成局部 Norton 或导纳型关系：

$$\Delta I_h = Y_{dev}(h) \Delta V_h + \Delta I_{src}(h)$$

代入网络方程后得到闭环关系：

$$\Delta V_h = (I - Z_h Y_{dev}(h))^{-1} Z_h \Delta I_{src}(h)$$

## 关键技术挑战

1. **频率耦合非单调性**：开关频率、基波和边带分量的交互不能简单按固定特征次分析，不同载波相位差会激活不同的频率分量组合

2. **控制-网络动态耦合**：PLL 带宽、电流环响应速度和链路延时的交互在高频段呈现强耦合，弱电网条件下更容易激发宽频振荡

3. **责任归因困难**：多设备系统中，THD 测量只能反映总体畸变，无法区分各设备对谐波的贡献比例

4. **实时仿真步长约束**：高频振荡分析（700Hz~2kHz）要求 EMT 步长 ≤ 10μs，与大规模系统实时仿真构成矛盾

5. **等值模型保真度**：频变网络等值在谐波域中的行为与原始网络可能存在偏差，特别是在故障暂态和换相期间

**失效模式**：

- 频域闭环判据基于线性化和固定运行点；大扰动故障、限幅和保护逻辑需要 EMT 时域验证
- 若只给 FFT 频谱而没有网络阻抗、控制模型或注入响应，不能直接断言存在谐波不稳定
- 多设备系统中的责任归因需要相位、阻抗和源模型支撑；简单比较各节点 THD 容易误导
