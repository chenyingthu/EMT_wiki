# 宽频建模方法 (Wideband Modeling)

## 定义与概述

宽频建模方法是电力系统电磁暂态仿真中用于准确表征设备与网络在宽频率范围（从直流到MHz级）内电气特性的建模技术。与传统仅考虑工频（50/60Hz）的准稳态模型不同，宽频模型能够描述设备在谐波、暂态、开关频率乃至雷击等高频工况下的动态响应。宽频建模是频变网络等值（FDNE）、电缆暂态分析、变压器高频建模、接地系统分析等领域的基础，对于准确预测电力电子装置引起的谐波传播、雷电过电压、VFTO（特快速瞬态过电压）等宽频现象至关重要。

## 1. 理论基础

### 1.1 频变参数的产生机理

**集肤效应（Skin Effect）**:
高频电流趋于导体表面分布，导致有效截面积减小、交流电阻增大、内电感减小。

$$R_{ac}(f) = R_{dc} \cdot k_{skin}(f)$$
$$L_{int}(f) = L_{int,dc} \cdot k_{ind}(f)$$

其中集肤深度：
$$\delta = \sqrt{\frac{1}{\pi f \mu \sigma}}$$

| 频率 | 铜导体集肤深度 | 影响 |
|------|---------------|------|
| 50 Hz | 9.4 mm | 可忽略 |
| 1 kHz | 2.1 mm | 中等截面导体受影响 |
| 10 kHz | 0.67 mm | 明显集肤效应 |
| 1 MHz | 0.067 mm | 强烈表面集中 |

**邻近效应（Proximity Effect）**:
相邻导体电流的磁场相互作用导致电流分布不均匀，增加等效电阻。

**大地回流效应（Ground Return）**:
地中回流路径随频率变化，卡松（Carson）公式或纳卡川（Nakagawa）方法用于计算频变地阻抗。

### 1.2 宽频模型的数学表示

**频域导纳/阻抗矩阵**:
$$Y(s) = G(s) + sC(s) + \frac{1}{s}K$$

其中频变电导$G(s)$和频变电纳$C(s)$需要通过有理函数拟合：

**有理函数近似**:
$$Y(s) \approx \sum_{i=1}^{n} \frac{r_i}{s-p_i} + d + sh$$

或状态空间形式：
$$\begin{cases}
\dot{x} = Ax + Bu \\
y = Cx + Du
\end{cases}$$

**极点-留数模型的物理意义**:
- 极点$p_i$：系统的固有模态（谐振频率与阻尼）
- 留数$r_i$：各模态的激励强度
- 常数项$d$：高频渐近特性
- 微分项$h$：瞬态响应初始值

### 1.3 频域到时许的转换方法

**递推卷积法**:
$$i(t) = \int_{0}^{t} y(t-\tau) v(\tau) d\tau$$

在有理函数近似下可转化为递推形式：
$$i_k = g_0 v_k + \sum_{j=1}^{n} h_{k,j}$$

其中历史项$h_{k,j}$通过极点留数参数递推更新，避免全历史卷积计算。

**伴随电路法**:
将有理函数导纳等效为RLC并联电路：
$$Y_{FDNE}(s) = \frac{1}{R_0} + \frac{1}{sL_\infty} + \sum_{i=1}^{m} \frac{1}{R_i + sL_i}$$

## 2. 宽频建模方法分类

### 2.1 参数化模型法

**基于物理的模型**:
从设备几何结构和材料特性出发，建立电磁场方程：
- **电缆**：考虑导体、绝缘层、屏蔽层、铠装的频变阻抗
- **变压器**：考虑绕组漏磁、铁心非线性、匝间电容
- **输电线路**：考虑相地回路、相相间耦合的频变特性

**解析公式**:
| 设备类型 | 关键参数 | 解析表达式 |
|---------|---------|-----------|
| 单根导线 | 内阻抗 | $Z_i = \frac{j\omega\mu}{2\pi r} \frac{J_0(\gamma r)}{J_1(\gamma r)}$ |
| 同轴电缆 | 特征阻抗 | $Z_c = \frac{1}{2\pi}\sqrt{\frac{\mu}{\varepsilon}} \ln\frac{b}{a}$ |
| 变压器绕组 | 波阻抗 | 基于传输线理论的分布参数模型 |

### 2.2 数值拟合法

**矢量拟合（Vector Fitting, VF）**:
由Bjørn Gustavsen于1999年提出，是目前应用最广的频域拟合算法。

**算法流程**:
```
1. 设定初始极点（对数均匀分布或线性分布）
2. 构造线性方程组，求解留数和权重函数零点
3. 权重函数零点作为新极点，迭代直至收敛
4. 最终极点-留数用于构建状态空间或伴随电路
```

**改进算法**:
| 算法 | 特点 | 适用场景 |
|------|------|----------|
| VF | 标准矢量拟合 | 单端口/多端口 |
| MVF | 模态矢量拟合 | 高阻抗终端 |
| CVF | 复数矢量拟合 | 复共轭极点约束放宽 |
| RVF | 松弛矢量拟合 | 非对称导纳矩阵 |
| MPM | 矩阵束方法 | 大系统快速拟合 |
| LM | Levy方法 | 最小二乘拟合 |

**拟合阶数选择**:
- 典型范围：8-20阶（输电线路）
- 频带越宽、谐振峰越多，所需阶数越高
- 模型降阶（MOR）技术可在保持精度下减少阶数

### 2.3 混合建模法

**FDNE（频率相关网络等值）**:
将大规模外部网络等值为多端口宽频导纳矩阵：

$$I_{port}(s) = Y_{FDNE}(s) \cdot V_{port}(s)$$

**构建流程**:
1. 在10-1000个频点进行频率扫描
2. 对每个频点形成全导纳矩阵并消去内部节点
3. 多端口导纳矩阵元素整体矢量拟合
4. 无源性校正（摄动法或半定规划）
5. 时域实现（状态空间或伴随电路）

**应用场景**: 混合仿真接口、大规模电网简化、谐波分析

## 3. EMT仿真应用

### 3.1 电缆宽频建模

**应用场景**:
- 地下/海底电缆暂态过电压分析
- VSC-HVDC电缆系统谐波传播
- 海上风电场集电电缆高频振荡

**模型要素**:
- 导体频变电阻（考虑集肤效应和邻近效应）
- 绝缘层频率相关介电损耗
- 金属护套/铠装的接地方式
- 土壤频变电导率（Carson/Nakagawa模型）

**典型频带**:
| 应用 | 频带范围 | 建模重点 |
|------|---------|----------|
| 工频潮流 | DC-60Hz | 直流电阻、稳态电容 |
| 谐波分析 | 60Hz-2kHz | 集肤效应、介质损耗 |
| 开关暂态 | 2kHz-100kHz | 波过程、行波反射 |
| 雷击分析 | 100kHz-10MHz | 高频波阻抗、接地 |

### 3.2 变压器宽频建模

**应用场景**:
- 变压器雷击过电压分析
- 特快速瞬态过电压（VFTO）传播
- 高频谐波负载效应

**模型结构**:
```
┌─────────────────────────────────────┐
│  绕组电阻（频变）                      │
│    R(f) = R_dc * k_skin(f)           │
├─────────────────────────────────────┤
│  漏电感（频变）                        │
│    L(f) = L_dc * k_ind(f)            │
├─────────────────────────────────────┤
│  匝间电容/对地电容（频变）              │
│    C(f) = C_0 * (1 + tanδ(f))        │
├─────────────────────────────────────┤
│  铁心非线性（磁滞+涡流）               │
│    考虑频变涡流损耗                   │
└─────────────────────────────────────┘
```

**黑盒建模**:
通过扫频测量（FRA）获取端口阻抗，矢量拟合构建宽频等值电路。

### 3.3 输电线路宽频建模

**J. Marti模型**:
采用频变特征阻抗和传播常数：

$$Z_c(s) = \sqrt{\frac{R(s) + sL(s)}{G(s) + sC(s)}}$$
$$\gamma(s) = \sqrt{(R(s) + sL(s))(G(s) + sC(s))}$$

通过矢量拟合分别对$Z_c(s)$和$e^{-\gamma(s)l}$进行有理函数逼近。

**Noda模型**:
直接对导纳矩阵进行拟合：
$$Y_{line}(s) = \frac{1}{Z_c(s)\sinh(\gamma(s)l)}$$

**适用边界**:
- Marti模型：适用于架空线，计算效率高
- Noda模型：适用于电缆和复杂地回路，精度更高

### 3.4 接地系统宽频建模

**应用场景**:
- 变电站接地网暂态电位升高
- 雷电流入地时的接地阻抗频变特性
- 跨步电压/接触电压计算

**频变土壤电阻率**:
土壤电阻率随频率增加而减小：
$$\rho(f) = \frac{\rho_0}{1 + (2\pi f \varepsilon_r \varepsilon_0 / \sigma)^2}$$

**接地网等值**:
基于电磁场数值计算（矩量法或有限元）获取频域接地阻抗，再矢量拟合为时域模型。

## 4. 实现技术

### 4.1 矢量拟合算法实现

**MATLAB实现**:
```matlab
% 矢量拟合示例
s = 2j*pi*f;                    % 复频率向量
f = Y_measured;                 % 测得的频域响应
poles = logspace(2, 6, 20);     % 初始极点（100Hz-1MHz）

% 调用矢量拟合
[poles, residues, d, h] = vector_fitting(s, f, poles);

% 重构拟合结果
Y_fitted = zeros(size(s));
for i = 1:length(poles)
    Y_fitted = Y_fitted + residues(i)./(s - poles(i));
end
Y_fitted = Y_fitted + d + s*h;
```

**C/C++并行实现**:
```c
// 并行矢量拟合核心步骤
void parallel_vf(int n_ports, int n_poles, int n_freq) {
    // 1. 并行构建线性方程组
    #pragma omp parallel for
    for (int p = 0; p < n_ports; p++) {
        for (int q = 0; q < n_ports; q++) {
            build_least_squares(&A[p][q], &b[p][q], 
                               freq_data, Y_data[p][q], 
                               poles, n_freq, n_poles);
        }
    }
    
    // 2. 并行QR分解求解
    #pragma omp parallel for
    for (int p = 0; p < n_ports; p++) {
        for (int q = 0; q < n_ports; q++) {
            solve_qr(A[p][q], b[p][q], &residues[p][q]);
        }
    }
    
    // 3. 极点更新（串行，需全局协调）
    update_poles(&poles, zeros, n_poles);
}
```

### 4.2 无源性强制

**无源性条件**:
导纳矩阵$Y(s)$满足：
$$\text{Re}\{u^* Y(s) u\} > 0, \quad \forall \text{Re}\{s\} > 0, \forall u \neq 0$$

**摄动校正法**:
1. 检测无源越界频率（奇异测试矩阵特征值）
2. 对非无源极点/留数进行摄动调整
3. 迭代直至所有频率满足无源性条件

**半定规划法**:
将无源性约束转化为凸优化问题，求解满足无源性的最小摄动。

### 4.3 模型降阶

**POD（本征正交分解）**:
通过奇异值分解提取主导模态：
$$Y(s) \approx U_k \Sigma_k V_k^T$$

**平衡截断**:
保留可控性和可观性最强的状态，截除弱影响模态。

**Krylov子空间**:
在指定频点或展开点匹配原始系统响应。

## 7. 适用边界与限制

### 7.1 适用条件
- **宽频暂态分析**: 雷击、操作过电压、谐波传播
- **电力电子装置**: 开关频率及其倍频分析
- **电磁兼容(EMC)**: 传导干扰与辐射分析
- **设备绝缘配合**: VFTO、快速暂态分析

### 7.2 失效边界
- **极高频率**: >10MHz时，电磁场分布复杂，集总参数模型失效
- **非线性主导**: 强非线性设备（铁心饱和）需结合时域非线性模型
- **温度效应**: 宽频模型通常不考虑温度对参数的影响
- **多物理耦合**: 热-电-机械耦合需多物理场联合建模

### 7.3 精度边界
| 限制因素 | 典型误差 | 缓解方法 |
|---------|---------|----------|
| 拟合阶数不足 | 5-15% | 增加阶数或分段拟合 |
| 无源性损失 | 数值发散 | 强制无源校正 |
| 高频渐近特性 | >10MHz误差增大 | 采用分布参数模型 |
| 测量噪声 | 拟合振荡 | 正则化或加权拟合 |
