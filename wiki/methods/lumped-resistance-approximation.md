# 集中电阻近似 (Lumped Resistance Approximation)

## 概述

集中电阻近似(Lumped Resistance Approximation)是一种简化输电线路建模的方法，将沿线路分布的电阻集中处理为若干离散电阻元件，而非连续分布参数。这种近似方法可以显著降低计算复杂度，同时在中低频暂态分析中保持足够的精度。集中电阻近似常与无损Bergeron模型结合使用，是电磁暂态仿真中重要的线路建模简化技术。

## 基本原理

### 分布参数与集中参数

**分布参数线路方程**:
$$-\frac{\partial v}{\partial x} = Ri + L\frac{\partial i}{\partial t}$$
$$-\frac{\partial i}{\partial x} = Gv + C\frac{\partial v}{\partial t}$$

**集中近似思想**:
- 将总电阻$R_{total}$分解为若干集中电阻
- 电阻之间用无损线段连接
- 保持总参数不变

### 电阻分布方式

**两端集中**:
```
R/2 ──无损线── R/2
```

**中点集中**:
```
无损线 ── R ── 无损线
```

**多段集中**:
```
R/n ──无损线── R/n ── ... ── R/n
```

## 近似模型类型

### 单π模型

**结构**:
```
R ─┬─ C/2 ─┬─
   │       │
   L       L
   │       │
   ┴─ C/2 ─┴─
```

**参数**:
- 总电阻: $R = r \cdot l$
- 总电感: $L = l_{unit} \cdot l$
- 总电容: $C = c \cdot l$

**适用**: 短线路($l < 80$ km)

### 双π模型

**结构**:
```
R/2 ──无损线── R/2 ──无损线── R/2
```

**参数计算**:
- 每段电阻: $R/2$
- 每段无损线长度: $l/2$
- 波阻抗: $Z_c = \sqrt{L/C}$
- 传播时间: $\tau/2$

### 多段级联模型

**n段模型**:
```
R/n ──无损1── R/n ──无损2── ... ──无损n── R/n
```

**段数选择**:
| 线路长度 | 推荐段数 | 频率上限 |
|---------|---------|---------|
| < 100 km | 1 | 1 kHz |
| 100-300 km | 2 | 2 kHz |
| 300-600 km | 3-4 | 5 kHz |
| > 600 km | 5+ | 10 kHz |

## 精度分析

### 频率响应比较

**精确分布参数**:
$$Z_{exact}(\omega) = \sqrt{\frac{R+j\omega L}{G+j\omega C}}$$

**集中近似**:
$$Z_{lumped}(\omega) = R + j\omega L \| \frac{1}{j\omega C}$$

**误差分析**:
$$\epsilon = \frac{|Z_{exact} - Z_{lumped}|}{|Z_{exact}|} \times 100\%$$

### 适用频率范围

**低频段**($f < 1$ kHz):
- 单π模型: 误差 < 5%
- 双π模型: 误差 < 2%

**中频段**($1 < f < 10$ kHz):
- 需多段模型
- 每段长度$< 100$ km

**高频段**($f > 10$ kHz):
- 分布效应显著
- 建议用频域模型

### 误差来源

**集肤效应**:
$$R_{ac} = R_{dc}(1 + k\sqrt{f})$$
- 高频电阻增大
- 集中模型难以精确描述

**地线返回**:
- 地阻抗频率相关
- 低频近似较好

## 与Bergeron模型结合

### Bergeron无损模型

**历史电流源**:
$$i_k(t) = \frac{1}{Z_c}v_k(t) + I_k(t-\tau)$$

**无损假设**:
- 忽略电阻
- 恒波阻抗
- 恒定传播速度

### 有损Bergeron模型

**电阻集中处理**:
1. 将总电阻$R$分成$n$份
2. 每份与无损线段串联
3. 用伴随电路表示电阻

**伴随电路**:
$$I_R(t) = G_R v_R(t) - J_R(t-h)$$

其中$G_R = 1/R_{segment}$, $J_R$为历史项

### 实现步骤

```
1. 计算总参数
   R = r × l, L = l × l, C = c × l

2. 选择分段数n
   (基于频率要求和精度需求)

3. 计算段参数
   R_seg = R/n, τ_seg = τ/n

4. 构建等效电路
   - n-1个无损Bergeron段
   - n个集中电阻

5. 求解网络方程
   - 每时步更新历史电流源
   - 节点导纳矩阵求解
```

## 参数计算

### 典型线路参数

**架空线路(500kV)**:
| 参数 | 单位 | 典型值 |
|-----|------|-------|
| r | Ω/km | 0.02 |
| l | mH/km | 0.89 |
| c | nF/km | 12.7 |

**电缆线路(220kV)**:
| 参数 | 单位 | 典型值 |
|-----|------|-------|
| r | Ω/km | 0.05 |
| l | mH/km | 0.35 |
| c | nF/km | 200 |

### 波阻抗与传播速度

**无损近似**:
$$Z_c = \sqrt{\frac{L}{C}}$$

**传播速度**:
$$v = \frac{1}{\sqrt{LC}} \approx \frac{c_{light}}{\sqrt{\epsilon_r}}$$

架空线: $v \approx 3 \times 10^8$ m/s
电缆: $v \approx 1.5 \times 10^8$ m/s

## 应用案例

### 短路电流计算

**故障点电压**:
$$V_f = \frac{Z_{eq}}{Z_{eq} + Z_s}V_{pre}$$

**近似误差**:
- 工频短路: < 1%
- 高频暂态: 5-10%

### 雷电过电压分析

**波头时间**: $1.2$ μs
**等效频率**: $\sim 300$ kHz

**适用性**:
- 集中模型: 不适用
- 需分布参数或频域模型

### 开关暂态

**操作频率范围**:
- 工频: 集中模型适用
- 重燃频率: 需多段模型

## 优缺点分析

### 优点

**计算效率**:
- 节点数减少
- 矩阵维数降低
- 仿真速度提升

**实现简单**:
- 直接利用Bergeron模型
- 无需复杂卷积
- 适合实时仿真

**数值稳定**:
- 无条件稳定(梯形法)
- 大步长可行

### 缺点

**精度限制**:
- 高频误差大
- 地线效应难以精确
- 集肤效应忽略

**适用范围**:
- 不适合雷电分析
- 长线路需多段

## 模型选择指南

### 选择流程

```
线路长度?
├─ < 80 km → 单π模型
├─ 80-300 km → 多段Bergeron
└─ > 300 km → 频域模型或多段

暂态类型?
├─ 工频过电压 → 集中模型
├─ 开关暂态 → 多段模型
└─ 雷电过电压 → 频域模型

频率范围?
├─ < 1 kHz → 1-2段
├─ 1-10 kHz → 3-5段
└─ > 10 kHz → 频域模型
```
