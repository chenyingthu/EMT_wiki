# 开关建模方法 (Switch Modeling)

## 定义与概述

开关建模方法（Switch Modeling Methods）是电力电子电磁暂态（EMT）仿真中处理功率半导体器件（IGBT、MOSFET、GTO、晶闸管、二极管）开关特性的建模技术。由于电力电子开关具有高频动作（kHz-MHz级）、强非线性、状态突变等特性，其建模方法直接影响EMT仿真的精度、数值稳定性和计算效率。从最简单的理想开关到考虑结电容、反向恢复、热特性的详细模型，不同建模层级适用于不同的仿真需求，是电力电子系统EMT仿真的核心技术。

## 1. 理论基础

### 1.1 开关器件的物理特性

**半导体开关类型**：
| 器件 | 导通机制 | 关断特性 | 典型应用 | 开关频率 |
|------|----------|----------|----------|----------|
| **二极管** | 正向偏置自动导通 | 反向阻断 | 整流、续流 | 工频-数百Hz |
| **晶闸管（SCR）** | 门极触发 | 电流过零关断 | LCC-HVDC | 50-300 Hz |
| **GTO/GCT** | 门极触发 | 门极负脉冲关断 | 中压传动 | 100-500 Hz |
| **IGBT** | 栅极电压控制 | 栅极电压控制 | VSC、MMC | 1-20 kHz |
| **MOSFET** | 栅极电压控制 | 栅极电压控制 | 低压高频 | 10-500 kHz |
| **SiC/GaN** | 宽禁带材料 | 高速开关 | 新一代高效变流 | 50-500 kHz |

**关键物理参数**：
- 导通电阻$R_{on}$：IGBT典型1-10 mΩ，MOSFET更低
- 关断电阻$R_{off}$：理想无穷大，实际>1 MΩ
- 开关时间：开通$t_{on}$、关断$t_{off}$，影响损耗和EMI
- 结电容：$C_{gs}$、$C_{gd}$、$C_{ds}$，影响开关暂态
- 反向恢复：二极管反向恢复电荷$Q_{rr}$

### 1.2 开关建模层级

| 建模层级 | 复杂度 | 精度 | 计算成本 | 适用场景 |
|----------|--------|------|----------|----------|
| **理想开关** | 极低 | 概念级 | 极低 | 原理分析、拓扑验证 |
| **二值电阻** | 低 | 中等 | 低 | 系统级EMT仿真 |
| **分段线性** | 中 | 中高 | 中 | 常规EMT、损耗计算 |
| **详细物理** | 高 | 高 | 高 | 开关暂态、EMI分析 |
| **热-电耦合** | 极高 | 极高 | 极高 | 可靠性、寿命评估 |

**模型选择原则**：
- 系统级仿真（>1000开关）：二值电阻或平均值模型
- 变流器级仿真（10-1000开关）：分段线性模型
- 器件级分析（<10开关）：详细物理模型

## 2. EMT仿真应用

### 2.1 理想开关模型

**数学描述**：
$$R_{sw}(t) = \begin{cases} 0 & \text{ON} \\ \infty & \text{OFF} \end{cases}$$

**节点导纳方程**：
```
导通：V_anode = V_cathode（强制等电位）
关断：I_switch = 0（强制零电流）
```

**实现方式**：
- **导通**：合并节点（节点数减1）
- **关断**：删除支路（导纳矩阵降阶）

**优点**：
- 数值稳定性好（无阻尼振荡）
- 计算效率高（无迭代）
- 物理意义清晰

**缺点**：
- 无法计算开关损耗
- 无法仿真开关暂态
- 导纳矩阵每步需重构

### 2.2 二值电阻模型

**梯形法离散化**（EMTP标准）：
$$R_{sw} = \begin{cases} R_{on} \approx 0.001 \, \Omega & \text{ON} \\ R_{off} \approx 10^6 \, \Omega & \text{OFF} \end{cases}$$

**伴随电路**：
- 导通：等效电阻$R_{on}$并联历史电流源
- 关断：等效电阻$R_{off}$并联历史电流源（近似为零）

**导纳矩阵影响**：
$$Y_{ii} = Y_{ii} + \frac{1}{R_{sw}}, \quad Y_{ij} = Y_{ij} - \frac{1}{R_{sw}}$$

**开关状态切换**：
- 状态变化时需更新导纳矩阵
- 触发LU分解重分解（计算瓶颈）

**量化参数**（典型IGBT）：
| 参数 | 数值 | 影响 |
|------|------|------|
| $R_{on}$ | 1-10 mΩ | 导通损耗 |
| $R_{off}$ | 1-10 MΩ | 漏电流、关断损耗误差 |
| 比值 | $10^6$-$10^9$ | 矩阵条件数、数值精度 |

### 2.3 分段线性开关模型

**三段线性化**（GENE方法，2006）：
```
导通区（V<V_th）：  R_sw = R_on
线性区（V≈V_th）：  I = g_m(V - V_th)
关断区（V>V_th）：  R_sw = R_off
```

**预计算策略**：
- 枚举所有开关状态组合（$2^n$种）
- 每种状态预计算LU分解结果
- 运行时查表，避免在线分解

**12脉动换流器示例**：
- 6个晶闸管 × 2状态 × 3个电抗器状态 = 192种组合
- 预计算192个LU因子
- 状态切换时直接查表（<1μs）

**优势**：
- 避免非线性迭代
- 计算时间确定（适合实时）
- 精度可接受

### 2.4 IGBT详细模型

**等效电路**（双极型+ MOS）：
```
        Cgc
    G ---||--- C
      /       \
     Rg       Cce
    Gate      / \
      \      E   C
       \    /     \
        MOS      BJT
```

**关键方程**：
- MOS沟道电流：$I_{MOS} = \frac{K_p}{2}(V_{GS} - V_{TH})^2$
- BJT集电极电流：$I_C = \beta I_B$
- 结电容：$C_{gc} = f(V_{gc})$

**开关暂态过程**：
1. 开通延迟：$t_{d(on)}$（栅极充电至阈值）
2. 电流上升：$t_{ri}$（MOS导通，BJT开启）
3. 电压下降：$t_{fv}$（$V_{CE}$跌落）
4. 关断延迟：$t_{d(off)}$
5. 电压上升：$t_{rv}$
6. 电流下降：$t_{fi}$

**量化参数**（典型1200V IGBT）：
| 参数 | 典型值 | 物理意义 |
|------|--------|----------|
| $V_{CE(sat)}$ | 2-3 V | 导通压降 |
| $t_{on}$ | 50-200 ns | 开通时间 |
| $t_{off}$ | 100-400 ns | 关断时间 |
| $E_{on}$ | 5-20 mJ | 开通能量损耗 |
| $E_{off}$ | 5-30 mJ | 关断能量损耗 |

### 2.5 开关函数法

**占空比定义**：
$$d(t) = \frac{t_{on}}{T_s}, \quad 0 \leq d \leq 1$$

**状态空间平均**：
$$\dot{x} = [d A_1 + (1-d) A_2]x + [d B_1 + (1-d) B_2]u$$

**等效受控源**：
- 开关网络等效为受占空比控制的电压/电流源
- 开关频率分量被平均化
- 保留低频动态

**适用条件**：
- 开关频率 >> 系统带宽
- 不关注开关纹波
- 控制设计和系统分析

**MMC平均值模型**：
$$S = \frac{1}{N}\sum_{i=1}^{N} S_i, \quad C_{eq} = \frac{C_{SM}}{N}$$

降阶比>99%，提速10-100倍。

### 2.6 二极管反向恢复

**反向恢复现象**：
- 二极管从导通到关断，存在反向恢复时间$t_{rr}$
- 反向恢复电荷$Q_{rr}$需释放

**反向恢复模型**：
$$I_R(t) = I_{RM}(1 - \frac{t}{t_{rr}})^\alpha$$

**软度因子**：
$$S = \frac{t_s}{t_f}$$

- $S>1$：软恢复（低EMI）
- $S<1$：硬恢复（高EMI，高电压尖峰）

**对EMT仿真的影响**：
- 需极小步长（<1ns）捕捉恢复过程
- 可能引起数值振荡
- 影响换流阀串联均压

## 3. 实现技术

### 3.1 开关事件处理

**事件检测**：
```python
def detect_switching(v_sw, i_sw, v_threshold, i_threshold):
    for switch in switches:
        # 电压过零检测（晶闸管）
        if switch.type == 'thyristor':
            if cross_zero(v_sw, switch.v_prev):
                trigger_event(switch, 'natural_commutation')
        
        # 电流过零检测（二极管）
        elif switch.type == 'diode':
            if i_sw < 0 and switch.i_prev > 0:
                trigger_event(switch, 'turn_off')
        
        # 门极触发（IGBT）
        elif switch.type == 'igbt':
            if gate_signal > v_threshold and not switch.state:
                trigger_event(switch, 'turn_on')
```

**插值找精确开关时刻**：
$$t_{exact} = t_n + \frac{v_{threshold} - v_n}{v_{n+1} - v_n}h$$

**临界阻尼调整（CDA）**：
- 开关事件后使用两步后向欧拉
- 消除梯形法数值振荡
- 后续恢复梯形法

### 3.2 恒导纳矩阵技术（ADC）

**问题**：开关状态变化导致导纳矩阵变化，每步需重分解。

**ADC策略**：
约束开关等效电感电容：
$$L_{sw}C_{sw} = h^2$$

使得：
$$G_{eq,on} + G_{eq,off} = \text{const}$$

**实现效果**：
- 导纳矩阵与开关状态无关
- 仅需一次LU分解
- 适合FPGA实时仿真

### 3.3 多速率开关处理

**分区策略**：
```
电力电子子系统：步长 h_f = 1-10 μs
                  详细开关模型
交流网络子系统：步长 h_s = 50-100 μs
                  等效模型/无开关
```

**接口处理**：
- 慢系统输出插值到快系统
- 快系统等值回慢系统
- 避免全局小步长

### 3.4 实时仿真优化

**预计算策略**（RTDS）：
```
离线阶段：
1. 枚举所有开关状态组合
2. 预计算每种状态的LU分解
3. 存储到查找表

实时阶段：
1. 检测当前开关状态
2. 查表获取LU因子
3. 前代回代求解（<1μs）
```

**FPGA实现**：
- 流水线并行处理多开关
- 固定延时（确定性）
- 典型：100开关@50μs步长

## 4. 仿真软件实现

### 4.1 PSCAD/EMTDC实现

**标准二值电阻模型**：
```fortran
! 开关模型参数
RON = 0.001   ! 导通电阻
ROFF = 1.0E6  ! 关断电阻

! 状态判断
IF (GATE_SIGNAL .GT. VTH .OR. V_ANODE .GT. V_CATHODE) THEN
    RSW = RON
ELSE
    RSW = ROFF
ENDIF

! 伴随电路参数
GEQ = 1.0 / RSW
IHIST = 0.0  ! 纯电阻无历史源

! 组装导纳矩阵
Y(I,I) = Y(I,I) + GEQ
Y(J,J) = Y(J,J) + GEQ
Y(I,J) = Y(I,J) - GEQ
Y(J,I) = Y(J,I) - GEQ
```

**IGBT详细模型**：
- 内置双极型-MOS混合模型
- 用户可调整结电容、载流子寿命
- 支持损耗计算

## 7. 适用边界与限制

### 7.1 适用条件

**模型选择指南**：
| 目标 | 推荐模型 | 理由 |
|------|----------|------|
| 系统级控制设计 | 平均值模型 | 保留低频动态，计算高效 |
| 保护整定 | 二值电阻 | 中等精度，开关细节可忽略 |
| 损耗评估 | 分段线性 | 可计算导通/开关损耗 |
| EMI/EMC分析 | 详细物理 | 捕捉开关暂态振荡 |
| 器件选型 | 热-电耦合 | 结温影响开关特性 |

**开关频率影响**：
- $f_{sw}$ < 1 kHz：平均值模型适用
- 1 kHz < $f_{sw}$ < 20 kHz：二值电阻或分段线性
- $f_{sw}$ > 20 kHz：详细模型（EMI分析）

### 7.2 失效边界

**数值问题**：
- 刚性比过高（$R_{off}/R_{on} > 10^9$）：矩阵病态
- 开关频率过高（>100 kHz）：步长过小，效率低
- 多开关同时动作：状态空间爆炸

**模型局限**：
- 二值电阻：无法仿真开关暂态
- 平均值模型：丢失谐波和内部暂态
- 详细模型：计算量巨大，不适合大系统

### 7.3 精度边界

| 模型类型 | 导通损耗误差 | 开关损耗误差 | 暂态精度 | 计算速度 |
|----------|--------------|--------------|----------|----------|
| 理想开关 | 不适用 | 不适用 | 概念级 | 极快 |
| 二值电阻 | <5% | 无法计算 | 低 | 快 |
| 分段线性 | <2% | <20% | 中 | 中 |
| 详细物理 | <1% | <5% | 高 | 慢 |
| 热-电耦合 | <1% | <3% | 极高 | 极慢 |

**验证要求**：详细模型需与器件手册双脉冲测试对比验证。
