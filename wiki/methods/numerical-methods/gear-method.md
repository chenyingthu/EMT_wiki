# 吉尔方法 (Gear Method)

## 1. 物理背景与工程需求

### 为什么需要 Gear 方法？

Gear 方法在 EMT 语境中通常指后向微分公式（Backward Differentiation Formula, BDF）族——一类隐式线性多步积分法。BDF 家族从 BDF1（即 [[backward-euler]]）到 BDF2 再到更高阶，提供了精度和阻尼的灵活选择。

Gear 方法的核心优势：
- BDF1 和 BDF2 同时满足 A-稳定和 L-稳定
- BDF2 在保持 L-稳定的前提下达到二阶精度（梯形法是 A-稳定但非 L-稳定）
- 多步历史信息可以提高效率

Gear 方法的核心代价：
- 多步历史依赖——开关事件后需要重启
- BDF3+ 不再 A-稳定（稳定域大幅缩小）
- 步长变化后历史值与新步长不匹配

## 2. 数学描述

### BDF 通用形式

对初值问题 $\dot{x} = f(t, x)$，$k$ 步 BDF：

$$$
\sum_{j=0}^{k}\alpha_j x_{n+1-j} = \Delta t\,\beta_0 f(t_{n+1}, x_{n+1})
$$$

### 低阶形式

**BDF1（后向欧拉）**：

$$$
x_{n+1} - x_n = \Delta t f(t_{n+1}, x_{n+1})
$$$

L-稳定，一阶精度。$R(z) = 1/(1-z)$，$z \to -\infty \Rightarrow R(z) \to 0$。

**BDF2**：

$$$
\frac{3}{2}x_{n+1} - 2x_n + \frac{1}{2}x_{n-1} = \Delta t f(t_{n+1}, x_{n+1})
$$$

L-稳定，二阶精度。$R(z)$ 在 $z \to -\infty$ 时趋于 0。

---

## 3. 计算模型与离散化

### 伴随电路形式

BDF 在 EMT 中仍可形成伴随电路，但历史源依赖多步历史。以电感 $v = L\,di/dt$ 的 BDF2 为例：

电压项需要 $i_{n+1}$、$i_n$ 和 $i_{n-1}$ 三个时刻的电流值，不能仅用 $i_n$。这使历史电流源的计算更复杂。

### BDF 与 TR 的稳定性对比

| 方法 | 精度阶数 | A-稳定 | L-稳定 | 历史依赖性 |
|------|----------|--------|--------|-----------|
| BDF1 (BE) | 一阶 | 是 | 是 | 1 步 |
| BDF2 | 二阶 | 是 | 是 | 2 步 |
| BDF3 | 三阶 | 否 | - | 3 步 |
| BDF4-6 | 更高阶 | 否 | - | 多步 |
| 梯形法 | 二阶 | 是 | 否 | 1 步 |

BDF3+ 不再 A-稳定，不适合 EMT 中的刚性系统，因此实际 EMT 中使用的主要是 BDF2。

## 4. 实现方法与算法细节

### TR-BDF2 复合方法

TR-BDF2 是 BDF2 在 EMT 中最实用的变体。它在一个时步内先用梯形法做中间步，再用 BDF2 完成：

1. 用 TR 从 $t_n$ 推进到 $t_{n+\gamma}$（$\gamma$ 通常取 $1/2$）
2. 用 BDF2 从 $t_{n+\gamma}$ 推进到 $t_{n+1}$

这种复合方法兼得 TR 的二阶精度和 BDF2 的 L-稳定阻尼，且作为单步方法不存在 BDF2 的历史项重启问题。

### 多步法的启动与重启

BDF2 启动时需要 $x_0$ 和 $x_1$ 两个初始值：
1. $x_0$ 由初始条件给出
2. $x_1$ 用单步法（如 BE 或 TR）计算
3. 从 $x_2$ 开始使用 BDF2

开关事件后，历史步 $x_{n-1}$ 可能携带旧拓扑信息：
- 选择一：完全重启，降级回 BDF1 再逐步升阶
- 选择二：TR-BDF2 单步过渡，避免多步历史问题

### 变阶变步长 Gear

通用 ODE/DAE 求解器中常实现变阶变步长 Gear（如 DASSL、SUNDIALS）：
- 根据局部误差估计调整阶数（1-5 阶）
- 步长增大时升阶提高精度，步长减小时降阶保证稳定
- 在固定步长 EMT 中实现成本较高：矩阵结构频繁变化

## 5. 适用边界与失效模式

### 什么条件下好用？

- 刚性但事件不频繁的电磁-机电混合模型
- 平滑区间内的大步长包络仿真或动态相量模型
- 需要比梯形法更强数值阻尼、但又希望高于一阶精度的场景

### 什么条件下会出问题？

| 问题场景 | 表现 | 原因 | 缓解办法 |
|----------|------|------|----------|
| 开关事件密集 | 波形发散 | 多步历史与旧拓扑不一致 | 降级到 BDF1 或 TR |
| BDF3+ 用于刚性 | 不稳定 | 高阶 BDF 非 A-稳定 | 限制到 BDF2 |
| 变步长 | 精度骤降 | 历史值与新步长不匹配 | 重启积分器 |
| 真实高频暂态研究 | 高频分量被衰减 | BDF 的 L-稳定阻尼 | 改用 TR 或无阻尼方法 |

### 工程经验值

- BDF2 是 EMT 中唯一实用的高阶 BDF（BDF3+ 不再 A-稳定）
- TR-BDF2 作为单步复合方法比纯 BDF2 更适合开关场景
- Gear 在实时仿真中不常用，因变阶变步长与固定步长相矛盾

## 6. 应用案例

### BDF2 的递推

取 $h = 0.5$ ms：

$$$
i_{n+1} = \frac{2}{3 + 10h/L}\left[2i_n - \frac{1}{2}i_{n-1} + \frac{h}{L}(v_{n+1})\right]
$$$

数值试验请使用 EMTP 类软件或自行编程实现。

## 量化性能边界

**Gao 2021 三种积分方法在移频EMT中的对比（3S-SDIRK vs TR-BDF2 vs TR）**:
- 系统比较TR、2S-DIRK、TR-BDF2和3S-SDIRK在移频EMT仿真中的性能
- TR-BDF2作为BDF类复合方法，兼顾二阶精度和L-稳定阻尼
- TR-BDF2在开关事件后作为单步方法不存在多步历史重启问题
- 数据缺口：原文未报告在标准EMT测试系统上的具体误差指标和加速比

**Parvari 2025 频移大步长仿真的精度约束**:
- 频移不会消除系统固有暂态模态，步长仍受特征值约束
- 动态相量法的步长上限由最快非移频模态决定
- 这对Gear方法的大步长应用构成重要限制
- 数据缺口：不同BDF阶次在频移框架下的具体精度损失未系统量化

**BDF族固有数值特性（数值分析经典结论）**:
- BDF1（后向欧拉）：L-稳定，一阶精度，R(z)=1/(1-z)，z→-∞ ⇒ R(z)→0
- BDF2：L-稳定，二阶精度，z→-∞时趋于0，是EMT中唯一实用的高阶BDF
- BDF3+：不再A-稳定，BDF3稳定域大幅缩小，不适合EMT刚性系统
- TR（梯形法）：A-稳定非L-稳定，高频分量数值振荡时需阻尼
- TR-BDF2兼得TR的二阶精度和BDF2的L-稳定阻尼，且为单步法

**设计参数约束（据方法推断）**:
- BDF2需要2步历史（x_n, x_{n-1}），开关事件后必须重启或降级为BDF1
- TR-BDF2的γ参数通常取1/2，兼顾稳定性和精度
- 变步长时多步历史与新区间不匹配，需要重启积分器或插值
- BDF2在实时仿真中不常用，因变阶变步长与固定步长相矛盾
- BDF的L-稳定特性会衰减真实高频暂态分量，不适合高频细节研究

**数据缺口声明**：Gear方法的量化性能主要来自数值分析经典结论而非特定EMT论文的系统验证。Gao 2021和Parvari 2025提供了移频EMT中的参考但缺乏标准测试系统上的多方法横向对比。BDF2与TR在典型EMT算例（如MMC开关瞬态、变压器励磁涌流）中的精度-效率量化对比数据缺乏。
