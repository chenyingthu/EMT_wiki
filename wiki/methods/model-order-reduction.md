# 模型降阶方法 (Model Order Reduction)

## 定义与概述

模型降阶（Model Order Reduction, MOR）是将高维复杂系统模型转化为低阶等效模型，同时保持关键动态特性的数学技术。在EMT仿真中，模型降阶解决大规模电力电子设备（如MMC、SST）导致的维数灾难问题：含数千子模块的换流器若逐器件建模，导纳矩阵维度可达$10^4$以上，单步求解计算量巨大。模型降阶通过电路等效、数值综合、模态截断等策略，将系统阶数从$N$降至$n$（$n \ll N$），实现仿真速度1-2个数量级提升，是含电力电子设备大电网EMT仿真的核心技术。

## 1. 理论基础

### 1.1 模型降阶的数学本质

**状态空间降阶问题**：
原始高阶系统：
$$\dot{x} = Ax + Bu, \quad y = Cx + Du, \quad x \in \mathbb{R}^N$$

降阶目标：构造低阶系统
$$\dot{\hat{x}} = \hat{A}\hat{x} + \hat{B}u, \quad \hat{y} = \hat{C}\hat{x} + \hat{D}u, \quad \hat{x} \in \mathbb{R}^n$$

使得输出误差$||y - \hat{y}||$在关注频段内最小化。

**降阶比定义**：
$$\eta = \frac{N - n}{N} \times 100\%$$

| 降阶等级 | 降阶比 | 典型应用 |
|----------|--------|----------|
| 轻度降阶 | $\eta < 50\%$ | 变压器多绕组等效 |
| 中度降阶 | $50\% \leq \eta < 90\%$ | FDNE有理函数拟合 |
| 深度降阶 | $90\% \leq \eta < 99\%$ | MMC桥臂整体等效 |
| 极限降阶 | $\eta \geq 99\%$ | 平均值模型（开关细节全忽略） |

### 1.2 主要降阶方法分类

**基于投影的方法**：
- **Krylov子空间法**：匹配传递函数矩（moment matching），适合单端口网络
- **平衡截断法（Balanced Truncation）**：基于可控/可观Gramian矩阵，保证全局误差界
- **模态分析法**：保留主导特征值，忽略快衰减模态

**基于电路综合的方法**：
- **Brune综合法**：从阻抗频响表格直接综合RLC网络，保证无源性
- **Tellegen扩展**：将Brune法推广至多端口系统
- **节点消去法（Kron Reduction）**：消去内部节点，保留端口特性

**基于物理近似的方法**：
- **平均值模型**：开关周期平均，忽略高频纹波
- **多时间尺度分解**：快慢变量分离，分别处理

### 1.3 频域降阶指标

**传递函数近似误差**：
$$\epsilon(\omega) = \frac{|H(j\omega) - \hat{H}(j\omega)|}{|H(j\omega)|} \times 100\%$$

**常用范数**：
| 范数类型 | 定义 | 适用场景 |
|----------|------|----------|
| $H_\infty$范数 | $\sup_{\omega} \sigma_{max}(H(j\omega))$ | 鲁棒稳定性分析 |
| $H_2$范数 | $\sqrt{\frac{1}{2\pi}\int_{-\infty}^{\infty} ||H(j\omega)||_F^2 d\omega}$ | 随机输入响应 |
| 频域加权范数 | 特定频段加权积分 | EMT暂态仿真 |

## 2. EMT仿真应用

### 2.1 FDNE频域降阶

**问题描述**：外部网络频响$Z(j\omega)$采样点密集，直接时域实现阶数过高。

**矢量拟合降阶**：
原始频响数据：$Z(j\omega_k), k=1,2,...,M$（$M$可达数千）

有理函数拟合：
$$Z(s) \approx \sum_{m=1}^{n} \frac{R_m}{s - p_m} + D + sE$$

降阶效果：$M$个采样点 → $n$个极点（$n \ll M$，典型$n=10$-$30$）

**量化成果**：
- 230kV三相输电网络：原始640ms仿真 → FDNE降阶后194ms（3.3倍加速）
- 三端口网络仅需80个RLCM模块等效

### 2.2 MMC高效建模降阶

**戴维南等效整体模型**：
将桥臂$N$个子模块的离散电容状态聚合为单一等效电阻与电压源：

$$R_{arm}^{eq} = \sum_{i=1}^{N} R_{SM,i}, \quad V_{arm}^{eq} = \sum_{i=1}^{N} V_{SM,i}^{eq}$$

降阶效果：$N$个电容状态 → 1个等效源（降阶比$>99\%$）

**受控源解耦模型**：
桥臂与子模块电气解耦，子模块电容电压更新与网络求解并行：
- 网络求解维度：仅含桥臂电流（与$N$无关）
- 子模块更新：$N$个独立一阶方程（可并行）

**量化成果**（2015 Xu综述）：
- 戴维南模型计算复杂度：$O(N^2) \to O(N)$（优化排序后）
- 48电平MMC：详细模型指数增长 vs 等效模型线性增长
- 200子模块时提速15-20倍

### 2.3 固态变压器高频链路降阶

**DAB模块端口等效**：
通过节点导纳方程预处理与Kron消去，将DAB内部节点消除：

$$Y_{EX} = A - B \cdot C^{-1} \cdot B^T$$

降阶效果：内部$k$个节点 → 仅保留输入/输出2端口

**多模块聚合**：
ISOP/ISOS/IPOP/IPOS连接方式下，短路导纳参数直接代数叠加：
$$Y_{HFL} = \sum_{m=1}^{M} Y_{DAB,m}$$

**量化成果**（2022 Gao）：
- 仿真速度提升1-2个数量级（10-100倍）
- 单步矩阵求逆运算量降为0

### 2.4 变压器宽频模型降阶

**矢量拟合阶数选择**：
| 变压器类型 | 原始阶数 | 降阶后 | 频段覆盖 |
|------------|----------|--------|----------|
| 电力变压器 | 50-100 | 10-20 | 1Hz-10kHz |
| 换流变压器 | 80-150 | 15-25 | 0.1Hz-100kHz |
| 高频变压器 | 30-60 | 8-15 | 1kHz-1MHz |

## 3. 实现技术

### 3.1 Brune网络综合法

**核心思想**：直接从阻抗频响表格综合物理可实现的RLCM网络，无需有理函数拟合。

**四步递归降阶**：
```
步骤1：移除虚轴极点（s=0, s=∞, 谐振点）
       ↓ 识别相位角突变（阈值0.5°）
步骤2：移除虚轴零点（导纳域极点）
       ↓ 求逆后重复步骤1
步骤3：提取最小实部电阻
       ↓ 使剩余阻抗在某频率秩亏
步骤4：Tellegen-Brune循环
       ↓ 提取耦合电感与变压器
       ↓ 降阶后继续迭代
```

**收敛准则**：剩余阻抗退化为常数矩阵$R_{end}$

**优势**：物理RLCM网络天然无源，无需后处理无源性校正。

### 3.2 Krylov子空间矩匹配

**矩展开**：
传递函数在$s=0$处泰勒展开：
$$H(s) = m_0 + m_1 s + m_2 s^2 + ... + m_k s^k + ...$$

其中$m_k = (-1)^k C A^{-(k+1)} B$为第$k$阶矩。

**Arnoldi算法**：
构造Krylov子空间$\mathcal{K}_n(A^{-1}, A^{-1}B) = \text{span}\{A^{-1}B, A^{-2}B, ..., A^{-n}B\}$

投影后低阶系统匹配前$2n$个矩。

**适用场景**：单端口/少端口网络，宽频阻抗拟合。

### 3.3 Kron节点消去法

**分块导纳方程**：
$$\begin{bmatrix} I_E \\ I_I \end{bmatrix} = \begin{bmatrix} Y_{EE} & Y_{EI} \\ Y_{IE} & Y_{II} \end{bmatrix} \begin{bmatrix} V_E \\ V_I \end{bmatrix}$$

其中$E$为外部节点，$I$为内部节点。

**消去公式**：
令$I_I = 0$（内部无源），消去$V_I$得：
$$I_E = Y_{EX} V_E, \quad Y_{EX} = Y_{EE} - Y_{EI} Y_{II}^{-1} Y_{IE}$$

**预处理优化**：
利用开关导纳互补特性$G_{ON} + G_{OFF} = G_x$（常数），使$Y_{II}^{-1}$解析简化为常数矩阵，避免每步求逆。

### 3.4 多时间尺度分解

**快慢变量分离**：
识别快动态（开关纹波）与慢动态（包络/控制）：
$$x = \begin{bmatrix} x_f \\ x_s \end{bmatrix}, \quad \dot{x}_f = A_f x_f + B_f u, \quad \dot{x}_s = A_s x_s + B_s u$$

**准稳态近似**：
快动态达稳态：$\dot{x}_f \approx 0 \Rightarrow x_f \approx -A_f^{-1}B_f u$

降阶后仅求解慢动态：$\dot{x}_s = A_s x_s + B_s u$

**适用条件**：快慢时间尺度分离度$>100$。

## 4. 仿真软件实现

### 4.1 PSCAD/EMTDC实现

**FDNE等值模块**：
- 输入：外部网络频域扫描数据
- 处理：矢量拟合（VF/MVF）降阶
- 输出：状态空间模型或RLC等效电路

**戴维南MMC模型**：
```fortran
! 桥臂等效构建
DO i = 1, N_arm
  R_eq(i) = R_SM * N_insert(i)
  V_eq(i) = SUM(V_cap(1:N_insert(i)))
END DO

! 网络求解（与N无关）
CALL SOLVE_NETWORK(R_eq, V_eq, I_arm)

! 子模块更新（并行）
!$OMP PARALLEL DO
DO i = 1, N_SM
  V_cap(i) = V_cap(i) + I_arm * duty(i) * dt / C
END DO
```

### 4.2 MATLAB实现

**基于Control System Toolbox**：
```matlab
% 高阶系统（示例：100阶）
sys = ss(A, B, C, D);

% 平衡截断降阶至10阶
[sysb, g] = balreal(sys);  % 平衡实现
elim = (g < 1e-6);         % 识别弱可控/可观状态
sys_red = modred(sysb, elim);  % 降阶

% 频域误差验证
figure;
bode(sys, sys_red);
title('降阶前后频响对比');
```

## 7. 适用边界与限制

### 7.1 适用条件

**模型降阶有效场景**：
- **线性/弱非线性系统**：RLC网络、线性化控制系统
- **频段限制明确**：关注频段外动态可忽略
- **端口数量远小于内部节点**：适合FDNE、模块化设备
- **时间尺度分离**：快慢动态可解耦

**各类方法适用性**：
| 方法 | 最佳适用 | 避免使用 |
|------|----------|----------|
| Brune综合 | 多端口FDNE、无源网络 | 强非线性、时变网络 |
| Krylov矩匹配 | 单端口阻抗、宽频拟合 | 多端口耦合严重 |
| Kron消去 | 模块化内部节点消去 | 拓扑频繁切换 |
| 平均值模型 | 系统级暂态、控制研究 | 开关级细节分析 |

### 7.2 失效边界

**降阶失效信号**：
- 频域误差>10%且无法通过增加阶数改善
- 时域响应出现非物理振荡或发散
- 降阶后模型非无源（正实条件破坏）
- 拓扑切换导致降阶假设失效

**极限约束**：
- 极强非线性（电弧、铁磁饱和）：降阶误差难以控制
- 宽频带全细节需求：降阶比受限（通常$\eta<80\%$）
- 实时步长约束：降阶后计算量仍需<步长时限

### 7.3 精度边界

| 降阶等级 | 典型误差 | 适用研究 |
|----------|----------|----------|
| 轻度($\eta<50\%$) | 0.1-1% | 保护整定、设备应力 |
| 中度($\eta=50-90\%$) | 1-5% | 系统暂态、控制设计 |
| 深度($\eta=90-99\%$) | 5-10% | 潮流分析、规划研究 |
| 极限($\eta>99\%$) | 10-30% | 概念验证、初步筛选 |

**验证要求**：降阶模型必须通过时域仿真对比验证，误差指标需量化报告。
