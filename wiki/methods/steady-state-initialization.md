# 稳态初始化 (Steady-state Initialization)

## 概述

稳态初始化是电磁暂态(EMT)仿真开始前的关键步骤，其目标是为动态仿真提供一个与稳态潮流一致的初始条件。良好的初始化可以避免仿真开始时的暂态冲击，提高仿真效率和结果可信度。

## 初始化方法

### 基于潮流的初始化
最常用的初始化方法：
- **潮流计算**: 先进行稳态潮流
- **结果转换**: 转换为EMT模型初值
- **状态变量**: 电容电压、电感电流

步骤：
1. 建立潮流模型
2. 求解潮流方程
3. 提取节点电压、支路功率
4. 计算EMT状态变量初值

### 动态初始化
- **动态求解**: 让系统自然收敛
- **斜坡启动**: 逐步增加激励
- **缺点**: 耗时长，可能不稳定

### 混合初始化
- **部分潮流**: 关键节点用潮流
- **部分动态**: 复杂设备动态求解
- **迭代**: 联合求解

## 各元件初始化

### 同步发电机
- **功角**: 由有功功率和电压决定
- **转速**: 额定转速
- **内部电势**: 由励磁决定
- **阻尼绕组**: 零电流(稳态)

转子角度计算：
$$\delta = \angle(E) - \angle(V)$$

### 传输线
- **电压**: 沿线潮流计算
- **电流**: $I = S^*/V^*$
- [[transmission-line-model]] - 输电线路模型

分布参数线路：
- **行波**: 稳态行波解
- ** Bergeron模型**: 等值历史电流源

### 变压器
- **分接头**: 当前档位
- **磁通**: 额定电压对应
- [[transformer-model]] - 变压器模型

饱和考虑：
- **初始磁通**: 避免涌流
- **剩磁**: 考虑剩磁影响
- [[magnetic-saturation-modeling]] - 磁饱和建模

### 负荷
- **阻抗值**: 由功率和电压计算
$$Z = \frac{|V|^2}{S^*}$$

- **电动机**: 滑差=0初始化
- `induction-motor` - 感应电动机

## 数值方法

### 牛顿法
求解非线性初始化方程：
$$\mathbf{J}\Delta\mathbf{x} = -\mathbf{F}(\mathbf{x})$$

- **雅可比矩阵**: 迭代更新
- **收敛判据**: 残差足够小

### 不动点迭代
$$\mathbf{x}_{k+1} = \mathbf{g}(\mathbf{x}_k)$$

- **简单**: 实现容易
- **收敛慢**: 可能需要多次迭代

### 时域松弛
- **小步长**: 初始用小步长
- **逐步增大**: 稳定后增大
- **自适应**: 自动调整

## 验证方法

### 功率平衡
- **有功**: $P_{gen} = P_{load} + P_{loss}$
- **无功**: $Q_{gen} = Q_{load} + Q_{loss}$

### 状态检查
- **电压**: 在合理范围
- **电流**: 连续、合理
- **功率**: 方向正确

### 动态验证
- **初始扰动**: 应足够小
- **快速收敛**: 无持续振荡
- `dynamic-validation` - 动态验证
