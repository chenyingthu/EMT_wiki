# 最近电平调制 (Nearest Level Modulation)

## 定义

最近电平调制（Nearest Level Modulation, NLM），又称最近电平逼近控制（Nearest Level Control, NLC），是多电平换流器（尤其是模块化多电平换流器 MMC）中最核心的调制方法之一。其基本思想是：根据参考电压选择最接近的可用离散电平，确定当前步每个桥臂应投入的子模块数量，再结合电容电压排序或均衡逻辑决定具体哪些子模块投入/切除。

给定参考电压 $v_{ref}$ 和子模块电容电压平均值（或额定值 $V_{dc}/N$），投入数由取整函数确定：

$$n_{on} = \text{round}\left(\frac{v_{ref}}{V_c^{avg}}\right)$$

其中 $V_c^{avg}$ 为桥臂子模块电容电压平均值。NLM 的输出电平数等于 $N+1$（半桥子模块）或 $2N+1$（全桥子模块）。

**边界声明**：本页聚焦 NLM 的调制原理、等效建模方法及其与电容电压均衡的耦合。不讨论动态相量、多速率求解或频移方法等与 NLM 无直接关联的仿真技术。

## EMT 中的角色

在 EMT 仿真中，NLM 涉及三个相互关联的环节：

1. **电平映射**：将连续参考电压（由上层控制器生成）映射为离散的桥臂投入子模块数 $n_{on}(t)$。
2. **子模块选择**：根据 $n_{on}(t)$ 和电容电压均衡策略，从桥臂中选出应投入的具体子模块。
3. **等效建模**：在大规模系统中，NLM 的离散开关细节可通过连续等效模型替代，以换取仿真步长的放宽和计算效率的大幅提升。

NLM 的性能直接影响 MMC 的输出电压谐波含量、电容电压波动以及控制系统的动态响应时间。

## EMT 建模方法

### 方法 1：全量排序法（传统策略）

**原理**：每步对所有 $N$ 个子模块的电容电压进行排序，根据桥臂电流方向（充电/放电）决定优先投入或切除哪些子模块。

**数学表达**：设桥臂电容电压集合 $\{v_{c,1}, v_{c,2}, \cdots, v_{c,N}\}$，排序后得升序序列。设 $i_{arm} > 0$ 表示充电方向，则投入电压最低的 $n_{on}$ 个子模块；$i_{arm} < 0$ 表示放电方向，则投入电压最高的 $n_{on}$ 个子模块。

**计算复杂度**：$O(N \log N)$，在 $N$ 较大时（如 $N \geq 200$）计算负担显著。Xu 2015 指出，该方法在 $N=200$ 时可达 15-20 倍加速比，但排序开销随电平数增加而迅速上升。

### 方法 2：平均比较法（Yu 2014）

**原理**：计算桥臂电容电压算术平均值 $V_{avg} = \frac{1}{N}\sum_{i=1}^{N} v_{c,i}$，将各子模块电压与 $V_{avg}$ 逐一比较：

- 充电时（$i_{arm} > 0$）：优先投入低于平均值的子模块
- 放电时（$i_{arm} < 0$）：优先投入高于平均值的子模块

通过两个调节变量设置滞环带以控制等效开关频率。

**计算复杂度**：$O(N)$，Yu 2014 报道约 **5000 倍仿真加速**（5 秒暂态从 1 周压缩至分钟级）。

**等效电路**：子模块交流侧等效为受控电压源，直流侧等效为受控电流源与电容并联：

$$v_{sm,i} = S_i v_{c,i} + v_{sw,i}$$

$$v_{arm} = \sum_{i=1}^{N} v_{sm,i}$$

### 方法 3：双向堆排序法（Lian 2022）

**原理**：利用 NLM 只需确定 $n_{on}$ 个极值（最大或最小）的特点，构建规模为 $K = \min(n_{on}, N-n_{on})$ 的堆，而非对全部 $N$ 个子模块排序。

**计算复杂度**：$O(K \log K)$，最坏情况（$n_{on} = N/2$）不超过 $N/2$。导纳矩阵阶数从 11 降至 5 阶（降幅 54.5%）。

**灵活离散化**：引入参数 $\alpha$ 控制离散化方法——$\alpha = 0$ 为隐式梯形法（高精度），$\alpha = 1$ 为后退欧拉法（避免网络突变时的数值振荡）。当检测到桥臂投入子模块数变化时自动切换为后退欧拉法，结构稳定时恢复梯形法：

$$C \cdot \frac{u_C(t) - u_C(t-\Delta t)}{\Delta t} = \frac{(1+\alpha)i_C(t) + (1-\alpha)i_C(t-\Delta t)}{2}$$

### 方法 4：NLM 等效连续模型（Zhao 2023）

**原理**：将 NLM 的离散调制过程等效为连续模型，将每个子模块的等效占空比 $d_{eq}$ 分解为两部分：

$$d_{eq} = d_{stable} + d_{fluctuation}$$

- $d_{stable}$：由 MMC 主控制系统的调制波直接生成，负责形成桥臂基波电压
- $d_{fluctuation}$：通过为每个子模块独立配置直流电压闭环控制器自动产生，负责电容电压的动态均衡

该模型避免了 NLM 的载波比较与实时排序计算，可在 PSCAD/EMTDC 中用离散元件搭建，无需额外编程。

**电容电压更新（梯形法）**：

$$v_c(t) = v_c(t-\Delta t) + \frac{\Delta t}{2C}[i_c(t) + i_c(t-\Delta t)]$$

**后退欧拉电容增量**（当检测到子模块切换时）：

$$\Delta u_c(t) = \frac{i_{arm}(t) \cdot T_s}{C}$$

### 方法 5：戴维南等效模型系列（Xu 2015）

**等效模型 1**（传统戴维南）：将每个子模块等效为诺顿支路（电阻 $R_s$ 串联电压源 $v_{s,eq}$），桥臂 $N$ 个子模块叠加后等效为 2 节点支路。对高阶矩阵直接求逆，计算耗时指数增长。

**等效模型 2**（后退欧拉改进）：引入两项关键改进：①假设每个开关组关断电阻无穷大，使电容电压增量只与当前时刻状态有关；②分组比较替代全排序，最大计算复杂度降为 $N-1$。

**等效模型 3**（梯形法改进）：采用梯形积分法替代后退欧拉法，分组排序最大复杂度为 $2N-3$，精度优于等效模型 2。

**闭锁实现**：通过仿真软件中具备插值功能的二极管模型函数（Eqv.D1 和 Eqv.D2）与戴维南等效桥臂电容 Eqv.C 组合模拟 MMC 闭锁。

### A 相上桥臂电流表达式（Zhao 2023）

$$i_{pa} = -\frac{I_{dc}}{3} - \frac{I_{vm}}{2}\sin(\omega t + \theta_{a1} - \alpha) - I_{cirm}\cos(2\omega t + \theta_{a2} - \beta) - \frac{4U_{dc}}{\pi N(2L_{ac}+L_0)}\sum_{k=6i\pm1}^{\infty}\frac{f(k)}{k^2\omega}\cos(k\omega t + \theta_{ak})$$

其中 $I_{vm}$ 为交流电流幅值，$L_{ac}$ 为交流侧等效电感，$L_0$ 为桥臂电感，$f(k)$ 为与谐波阶数 $k$ 相关的波形系数。

## 关键技术挑战

### 挑战 1：大规模子模块排序的计算瓶颈

当 $N$ 达到数百时，全量排序法的 $O(N \log N)$ 复杂度导致仿真时间急剧增加。Xu 2015 指出，在 $N=200$ 时详细模型的仿真步长需限制在 20 μs 以内，仿真效率极低。

**解决路径**：Yu 2014 平均比较法（$O(N)$）→ Lian 2022 双向堆排序法（$O(K \log K)$）→ Zhao 2023 等效连续模型（跳过排序）。

### 挑战 2：离散化方法与数值稳定性

后退欧拉法数值稳定但精度较低，梯形法精度高但在网络突变时可能产生数值振荡。Lian 2022 的灵活切换策略（参数 $\alpha$）通过检测桥臂投入子模块数变化自动切换方法，在稳定性和精度之间取得平衡。

### 挑战 3：NLM 等效连续模型的精度边界

Zhao 2023 的等效连续模型将离散的开关动作与电平排序逻辑等效为连续电压闭环控制，在 0~500 Hz 低频范围或毫秒级步长下与传统 NLM 开关模型误差 < 0.5%。但该模型不适用于：开关应力分析、器件损耗评估、EMI 研究，以及需要精确开关时序的高频暂态场景。

### 挑战 4：电容电压均衡的收敛性

电容电压均衡是 NLM 稳定运行的前提。若散布过大，仅靠 NLM 的最近电平选择不足以维持平衡。Zhao 2023 通过为每个子模块配置独立直流电压闭环，使稳态波动幅值偏差控制在 ±2% 以内。

### 挑战 5：闭锁工况的等效建模

戴维南等效模型对 IGBT 和二极管不加区分，统一处理为开关组。当 MMC 闭锁时，子模块拓扑中只含二极管（不可控自然关断器件），需要插值处理以避免数值计算错误（Xu 2015）。

### 加速比与精度权衡

```
详细 NLM 开关模型：步长 ≤ 20 μs（极慢，精确）
  ↓ 戴维南等效（Xu 2015 等效模型 2/3）
等效模型加速：步长 ~20-100 μs（中速，高精度）
  ↓ 平均比较法（Yu 2014）
快速均衡算法：步长 ~100-500 μs（快速，中等精度）
  ↓ NLM 等效连续模型（Zhao 2023）
连续等效模型：步长 1-5 ms（极快，低频精度保证）
```

## 适用边界与选择指南

### 适用条件

| 方法 | 适用场景 | 推荐步长 | 推荐电平数 |
|------|---------|---------|-----------|
| 全量排序法 | 验证基准、电平数 ≤ 50 | 1-20 μs | N ≤ 50 |
| 平均比较法 | 大规模 MMC（HVDC、STATCOM）| 20-100 μs | N ≥ 100 |
| 双向堆排序 | 超大规模 MMC、需灵活切换 | 50-200 μs | N ≥ 200 |
| NLM 等效连续模型 | 系统级仿真、低频动态研究 | 1-5 ms | N ≥ 20 |
| 戴维南等效系列 | 需要开关细节但需加速 | 20-200 μs | N ≥ 20 |

### 失效边界

- **N < 10** 时电平数过少，输出电压谐波畸变严重，应使用 PWM 调制而非 NLM
- **NLM 等效连续模型**不适用于：开关应力分析、器件损耗评估、EMI 研究、高频谐波（> 500 Hz）场景
- 故障、闭锁或极端暂态中，等效模型的简化假设可能失效 [Zhao 2023]
- 纯 NLM 等效连续模型不适合需要精确开关时序的研究 [Yu 2014]
- 戴维南等效模型的闭锁功能需特殊插值处理 [Xu 2015]

### 选择决策表

**问题 1**：是否需要保留子模块电容动态？
- 是 → 选 NLM 等效连续模型（Zhao 2023）或平均比较法（Yu 2014）
- 否 → 选戴维南等效模型系列（Xu 2015）或平均值模型

**问题 2**：电平数 N 有多大？
- N ≤ 50 → 全量排序法或 CPS-PWM
- N ≥ 100 → 平均比较法或双向堆排序法
- N ≥ 200 → 双向堆排序法或 NLM 等效连续模型

**问题 3**：仿真时间尺度是多少？
- μs 级详细开关 → 戴维南等效 + 分组排序（Xu 2015）
- ms 级系统动态 → NLM 等效连续模型（Zhao 2023）

## 与载波调制的对比说明

NLM 与载波移相调制（CPS-PWM）的核心区别在于：NLM 在每个控制周期直接计算投入数量，不依赖高频载波比较，因而等效开关频率较低，适合子模块数量较多（通常 $N \geq 20$）的应用场景。当 $N$ 较小时（如 $N < 20$），NLM 的量化误差相对增大，CPS-PWM 的谐波性能更优。因此在工程实践中，$N = 20$ 通常被视为 NLM vs PWM 的分界点。
