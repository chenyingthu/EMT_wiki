# LCL 滤波器方法 (LCL Filter)

## 1. 物理背景与工程需求

LCL滤波器是并网电压源换流器（VSC）和全功率变流器风力发电机组交流侧滤波的常用拓扑结构。与单纯的L滤波器相比，LCL滤波器在相同电感量下具有更强的高频谐波衰减能力，能够以更低的总电感量实现相同的谐波抑制效果，从而降低系统成本和体积 (Nurunnabi 2025)。

**工程需求层面**，LCL滤波器在EMT仿真中面临以下核心挑战：

1. **谐振问题**：LCL滤波器在谐振频率处呈现高阻抗特性，若电流控制器带宽接近或超过谐振频率，可能导致系统不稳定。在弱网条件下（短路比SCR较低），电网等效电感 $L_g$ 较大，等效为LCL滤波器的第三电感，使实际谐振频率偏离设计值，严重时引发谐振发散 (Nurunnabi 2025)。

2. **数字延迟**：实际控制器存在采样和计算延迟（通常为1.5~2个开关周期），这些延迟会改变LCL滤波器的阻抗特性，在谐振频率处引入负阻尼效应 (Nurunnabi 2025)。

3. **阻尼设计**：为抑制LCL谐振，通常需要添加阻尼电阻或采用有源阻尼策略。阻尼电阻会增加功率损耗，而有源阻尼需要额外的控制算法设计和参数整定 (Luo 2022)。

4. **弱网适应**：在高阻抗并网场景下，等效电网电感会显著改变LCL滤波器的谐振特性，使原本稳定的系统在弱网条件下失稳 (Nurunnabi 2025)。

**典型应用场景**包括：风电场全功率变流器并网、光伏逆变器并网、储能系统PCS并网、MMC-HVDC系统的交流滤波器接口等。

## 2. 数学描述

### 2.1 LCL滤波器拓扑结构

LCL滤波器由三部分组成：
- **换流器侧电感** $L_1$：连接换流器输出端，限制开关频率电流纹波
- **滤波电容** $C_f$ ：提供高频电流通路，将高频电流导入接地支路
- **电网侧电感** $L_2$：连接PCC点，进一步衰减注入电网的高频谐波

### 2.2 谐振频率

LCL滤波器的固有谐振频率 $\omega_r$ 由三元件参数决定：

$$\omega_r = \sqrt{\frac{L_1 + L_2}{L_1 L_2 C_f}}$$

或等效表示为频率形式：

$$f_r = \frac{1}{2\pi} \sqrt{\frac{L_1 + L_2}{L_1 L_2 C_f}}$$

其中 $L_1$ 为换流器侧电感，$L_2$ 为电网侧电感，$C_f$ 为滤波电容。

### 2.3 PCC点功率传输方程

在EMT建模中，PCC点有功和无功功率的近似方程为 (Nurunnabi 2025)：

$$P_{PCC} \approx V_{PCC} \frac{E_{inv}}{X_F} \delta_{inv}$$

$$Q_{PCC} \approx V_{PCC} \frac{E_{inv} - V_{PCC\_F}}{X_F}$$

其中 $X_F$ 为滤波器的等效电抗，$E_{inv}$ 为逆变器输出电压，$V_{PCC}$ 为PCC电压，$\delta_{inv}$ 为功角。

### 2.4 LCL阻抗模型

LCL滤波器的输入阻抗（从换流器侧看入）为：

$$Z_{in}(s) = sL_1 + \frac{1}{sC_f} \parallel (sL_2 + Z_g)$$

在忽略电网阻抗 $Z_g$ 时，谐振频率由上述公式给出。当考虑弱网条件 $Z_g = sL_g$ 时，实际谐振频率将降低为：

$$f_{r,weak} = \frac{1}{2\pi} \sqrt{\frac{L_1 + L_2}{L_1 L_2 C_f} \cdot \frac{1}{1 + L_g/L_{total}}}$$

其中 $L_{total} = L_1 + L_2$。

### 2.5 有源阻尼控制

针对LCL谐振的有源阻尼通常采用电容电流反馈或并网电流反馈，其等效阻尼控制律为 (Luo 2022)：

$$F_d(s) = \frac{k_d s}{s + \omega_d}$$

其中 $k_d$ 为阻尼增益，$\omega_d$ 为高通滤波器的截止频率。该控制器在谐振频率处提供正阻尼，而在低频和高频处对系统动态影响较小。

## 3. 计算模型与离散化

### 3.1 等效电路模型

在EMT仿真中，LCL滤波器通常表示为三相π型等效电路：

```
L1          Cf          L2
┌───┐      ┌───┐      ┌───┐
│   │──────┤   │──────│   │
│ L1│      │ C │      │ L2│
│   │──────┤   │──────│   │
└───┘      └───┘      └───┘
```

每相由三个支路组成：电感 $L_1$ 支路、电容 $C_f$ 支路、电感 $L_2$ 支路。

### 3.2 数值离散化

电感元件采用梯形积分法的历史电流源等效：

$$i_{hist} = -\frac{2L}{\Delta t} i(t-\Delta t) - \frac{\Delta t}{2L} v(t-\Delta t)$$

电容元件的离散化等效为：

$$i_{hist} = \frac{2C}{\Delta t} v(t-\Delta t) + i_C(t-\Delta t)$$

### 3.3 阻抗扫描分析

在EMT软件中进行LCL滤波器设计时，通常需要进行阻抗扫描分析，绘制LCL滤波器的频率阻抗特性曲线：

$$|Z_{in}(j\omega)| = \sqrt{(\omega L_1 - \frac{1}{\omega C_f} + \omega L_2)^2}$$

相位特性为：

$$\angle Z_{in}(j\omega) = \arctan\left(\frac{\omega L_1 - \frac{1}{\omega C_f} + \omega L_2}{R_{eq}}\right)$$

其中 $R_{eq}$ 为考虑电感寄生电阻和阻尼电阻后的等效阻尼电阻。

## 4. 实现方法与算法细节

### 4.1 LCL滤波器参数设计流程

1. **确定开关频率谐波衰减目标**：根据IEEE 1547标准或电网公司要求，确定PCC点电流THD限值（如5%）。

2. **选择换流器侧电感 $L_1$**：限制开关频率电流纹波，通常取：

$$L_1 \geq \frac{V_{dc}}{2\Delta I_{max} f_{sw}}$$

其中 $\Delta I_{max}$ 为最大允许纹波电流，$f_{sw}$ 为开关频率。

3. **确定滤波电容 $C_f$**：在基波频率下，滤波电容的无功功率通常限制在额定功率的5~10%：

$$C_f \leq \frac{Q_{C,max}}{\omega_1 V_{grid}^2}$$

4. **计算电网侧电感 $L_2$**：根据目标谐振频率和已确定的 $L_1, C_f$ 计算：

$$L_2 = \frac{1}{\omega_r^2 C_f} - L_1$$

5. **阻尼设计**：选择有源阻尼或无源阻尼策略，计算阻尼电阻或阻尼控制器参数。

### 4.2 阻尼参数解析计算

基于电池储能装置直流侧阻抗建模的研究 (Luo 2022)，阻尼控制器参数需要满足以下约束条件：

**并联谐振约束（阻尼增益上限）**：

$$k_d \leq \frac{N^2\omega_{res1}^2 + (N^2 - E_N^2)\omega_d^2 - E_N\omega_{res1}}{P_N\omega_{res1}}$$

**串联谐振约束（阻尼带宽范围）**：

$$m_2\omega_d^2 - m_1\omega_d + m_0 < 0$$

其中系数由系统参数和目标阻抗特性确定。

### 4.3 数字延迟补偿

在EMT模型中，数字控制器的延迟通常等效为一个滞后环节：

$$G_{delay}(s) = e^{-sT_d}$$

其中 $T_d = 1.5 \sim 2 \times T_{sw}$，$T_{sw}$ 为开关周期。该延迟环节会影响LCL滤波器在谐振频率处的相位裕度。

补偿方法通常包括：
- **相位超前校正**：在电流控制器中加入相位超前环节
- **改进型PWM**：采用预测电流控制或滑模控制
- **谐振控制器**：在谐振频率处设置陷波滤波器

### 4.4 EMT仿真实现注意事项

1. **模型分离**：LCL滤波器应与控制系统模型解耦，确保数值稳定性
2. **步长选择**：开关频率谐波分析需要较小步长（如开关周期的1/20~1/50），但基波动态可用较大步长
3. **状态初始化**：确保滤波器初始状态与电网电压相位匹配，避免启动瞬态

## 5. 适用边界与失效模式

### 5.1 适用条件

- **PWM调制逆变器**：适用于SPWM、SVPWM等多种调制方式的并网变流器
- **中高压并网场景**：适用于10kV~500kV交流系统的LCL滤波设计
- **工频及以下频率分析**：适用于基波到开关频率二次谐波的频段分析
- **固定电网阻抗**：在电网阻抗变化不大的场景下可直接应用

### 5.2 失效边界与典型问题

1. **弱网失稳**
   - 表现：当短路比SCR<3时，电网等效电感显著增大，实际谐振频率降低，可能进入电流控制器带宽范围，引发谐振发散
   - 原因：弱网条件下 $L_g$ 增大，使 $f_r$ 降至低频段，与控制器带宽重叠
   - 原文未报告可核验的SCR失稳临界值

2. **数字延迟引发的不稳定**
   - 表现：数字控制器的计算和采样延迟使控制系统相位裕度减小，在谐振频率处可能变为负阻尼
   - 原因：延迟环节 $e^{-sT_d}$ 在谐振频率处引入约 $-\omega_r T_d$ 的相位滞后

3. **参数敏感性**
   - 表现：滤波电容偏差±10%可能导致谐振频率偏移±5%，可能使原设计的安全裕度消失
   - 原因：电容容值受温度、老化影响，且制造偏差通常在±10%左右

4. **阻尼参数越界**
   - 表现：阻尼增益过大时反而会在其他频段引入负阻尼，导致低频振荡 (Luo 2022)
   - 原因：阻尼控制器在增加谐振频率处阻尼的同时，可能改变其他频段的阻抗特性

5. **不对称故障**
   - 表现：单相接地故障时，LCL滤波器的不对称接地可能导致三相参数不一致，引发相间耦合振荡
   - 原因：故障期间PCC电压不平衡，导致滤波器支路电压电流不对称

### 5.3 设计建议

- **谐振频率设计**：建议将谐振频率设计在开关频率的1/5~1/3之外，同时远离电流控制器带宽（通常为开关频率的1/10~1/5）
- **阻尼验证**：在弱网条件（SCR=2~3）下进行EMT仿真验证，确保阻尼有效
- **参数裕度**：设计时考虑±15%的参数偏差，确保在最坏情况下系统仍稳定

## 6. 应用案例

### 案例1：100kVA GFM逆变器LCL滤波器设计 (Nurunnabi 2025)

**系统参数**：
- 额定容量：100 kVA
- 直流母线电压：1500 V
- PCC线电压：690 V / 60 Hz
- LCL滤波器参数：$L_1 = L_2 = 2$ mH, $R = 0.006$ $\Omega$, $C_f = 25$ $\mu$F

**关键结果**：
- SVPWM调制下最大允许输出电压达1.53 p.u.，较SPWM的1.33 p.u.提升约15%
- EVR策略在动态负载下将电压最大偏差限制在2.9%以内，频率最大偏差控制在0.37%以内，满足IEEE 1547标准
- 逆变器输出端短路容量计算值为316 kVA

### 案例2：2MW全功率变流器风机LCL滤波器 (Trevisan 2018)

**系统参数**：
- 额定功率：2 MW
- 网侧拓扑：两电平VSC
- 输出滤波：LCL滤波器
- 机械模型：单质量块模型

**关键结果**：
- 混合模型允许仿真步长增大至传统开关模型的5倍以上，计算效率提升约80%-85%
- 故障穿越期间网侧无功电流响应延迟<10ms，直流母线电压超调量<5%
- 现场实测与仿真波形在故障暂态期间的电压/电流峰值偏差<3%

### 案例3：直流配电网有源阻尼控制 (Luo 2022)

**系统参数**：
- 额定直流电压：20 kV
- 额定功率：10 MW
- VSC支撑电容：1 mF
- 直流限流电抗器：10 mH

**关键结果**：
- 系统自然串联谐振频率理论值为118 Hz，阻抗幅值交点频率为115.5 Hz，误差<2.5%
- 阻尼控制器增益在Buck模式下必须严格满足上限约束，否则会在50~100 Hz频段产生多段负电阻区域
- 采用解析参数后系统谐振振荡被完全抑制，直流电压稳态偏差控制在±0.5%以内
