# Small Time-Step FPGA-based Real-Time Simulation of Power Systems Including Multiple Converters

**作者**: Ramin Mirzahosseini, Reza Iravani
**年份**: 2019
**来源**: `35/TPWRD.2019.2933610.pdf.pdf`

## 摘要

本文提出一种基于FPGA的电磁暂态实时仿真平台，核心在于结合双值电阻开关模型、开关网络划分法（SNP）与重构改进节点分析法（RMNA）。传统ADC模型存在人工损耗与寄生振荡，而双值电阻模型虽精确但导致导纳矩阵时变，计算负担重。SNP方法在求解层将每个电力电子换流器开关网络（PECSN）划分为独立子系统，外部电路保持恒定导纳，避免使用存根线。RMNA通过重构节点注入向量，将独立源、支路历史电流与元件历史电流解耦，利用预计算的逆矩阵实现网络方程与元件模型求解的硬件级任务并行。该架构在FPGA上采用标准浮点核，实现固定硬件配置，支持亚微秒级步长实时求解大规模时变系统。

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求来自HIL/DRTS：含高频电力电子换流器的电力系统需要小于1 µs的EMT实时仿真，且多个PEC可能电气距离很近，不能靠真实传输线或人为stub line自然解耦。研究对象是“多个PEC及其外部电网”的FPGA实时求解，其中PEC开关采用双值电阻模型而非ADC模型。难点在于：ADC模型计算方便但会引入人工开关损耗和寄生振荡；双值电阻模型能更真实表示开关通断，却使系统导纳矩阵随开关状态变化，多个换流器叠加后矩阵大且时变，实时步内重构/求逆不可承受。已有建模级划分依赖stub line，会引入寄生元件；显式离散级划分可能因EMT刚性方程产生稳定性问题。本文贡献是把问题转到求解层：用SNP将每个PEC开关网络与外部恒定导纳网络分开求解，并用RMNA重构节点注入，使网络方程、历史源和元件模型可在FPGA上并行流水计算，从而在保留双值电阻开关模型的同时避免stub line解耦。

### 2. 模型、算法与实现技术

方法由三部分耦合工作。第一，PEC开关使用双值电阻模型：开通和关断分别对应不同电阻，因此开关支路仍作为电路元件进入EMT方程，避免ADC等效带来的人工损耗/振荡，但代价是开关网络导纳随状态变。第二，SNP把每个Power Electronic Converter Switching Network（PECSN）作为独立开关子网络，外部系统被视为恒定导纳网络，并在端口处形成戴维南/诺顿等效。机制上，外部网络先根据历史量和源量计算端口等效激励，PECSN再依据门极信号及当前端口电压/电流方向判定开关状态并求开关电流，最后把结果反馈到全网络变量更新。这样变化集中在小规模开关网络，而不是整个系统Y矩阵。第三，RMNA将传统MNA中的节点注入向量重构为独立源、支路历史电流、元件历史电流等分量；预先计算类似H_R=Y^{-1}K的固定矩阵后，每步只需矩阵-向量乘法。其关键作用不是改变EMT物理模型，而是解除网络求解与元件历史量更新之间的数据依赖，使FPGA上的浮点乘加、历史源更新、PECSN求解可并行展开。输入主要是上一时步状态、历史电流、独立源和门极信号；输出是当前时步节点电压、支路/开关电流和更新后的历史源。

### 3. 验证、优势与不足

原文摘要说明作者用一个study system评估平台，并与off-line simulation studies对比验证；引言和摘要还明确目标指标是sub-microsecond simulation time-step，以及面向含多个PEC的电力系统。现有证据没有给出可核验的测试拓扑、PEC数量、FPGA型号、资源占用、波形误差、实时裕度或离线软件名称，因此不能写具体误差百分比或加速比。可确认的优势主要来自方法结构：相对ADC模型，本文选择双值电阻开关模型以避免ADC已知的人工开关损耗和寄生振荡；相对建模级stub-line划分，SNP不要求用传输线/人为stub line把相邻PEC隔开，因此更适合电气距离很近的多个PEC；相对显式离散划分，本文属于基于隐式积分的solution-level partitioning，意图避免显式法稳定性限制。边界也应按验证范围理解：SNP依赖外部网络导纳在开关过程中保持恒定，并把时变部分限制在PECSN内；若外部网络本身含强时变元件、不同PECSN开关过程强耦合到无法独立划分，或需要详细器件级损耗/热模型，本文摘要和引言不能证明其仍然有效。原文未报告可核验的数值结果，因此“精度高、资源省、规模可扩展”的程度不能量化。

### 4. 价值、认知与可复用场景

这项工作的核心认知是：小步长实时EMT中，困难不只在开关模型本身，而在“开关导致全系统矩阵时变”与“实时步内计算依赖链过长”。本文用SNP把时变性局部化，用RMNA把依赖链拆成适合FPGA并行的固定矩阵-向量运算，为多个近距离PEC的HIL/DRTS提供了一种不靠stub line的建模入口。它适合被后续关于FPGA-EMT求解器、双值电阻开关模型实时化、换流器集群小步长仿真、solution-level partitioning的页面复用。不能外推为通用离线EMT求解器，也不能据此断言任意规模系统都能在亚微秒内实时运行；具体容量仍取决于FPGA资源、矩阵规模、PECSN数量和实现流水线。

## 方法细节

### 方法概述

本文提出一种基于FPGA的电磁暂态实时仿真平台，核心在于结合双值电阻开关模型、开关网络划分法（SNP）与重构改进节点分析法（RMNA）。传统ADC模型存在人工损耗与寄生振荡，而双值电阻模型虽精确但导致导纳矩阵时变，计算负担重。SNP方法在求解层将每个电力电子换流器开关网络（PECSN）划分为独立子系统，外部电路保持恒定导纳，避免使用存根线。RMNA通过重构节点注入向量，将独立源、支路历史电流与元件历史电流解耦，利用预计算的逆矩阵实现网络方程与元件模型求解的硬件级任务并行。该架构在FPGA上采用标准浮点核，实现固定硬件配置，支持亚微秒级步长实时求解大规模时变系统。

### 数学公式

**公式1**: $$$Y x_{node}^{n+1} = u_{node}^{n+1}$$$

*传统改进节点分析（MNA）网络方程，Y为导纳矩阵，x为未知量向量，u为注入电流向量。*

**公式2**: $$$b_{node}^{n+1} = [b_s^{n+1} \ b_h^{n+1} \ b_c^{n+1}]^T$$$

*RMNA重构的节点注入向量，将独立源、支路历史和元件历史电流解耦，为并行计算提供数据基础。*

**公式3**: $$$x_{node}^{n+1} = H_R b_{node}^{n+1}, \quad H_R = Y^{-1} K$$$

*RMNA求解公式，通过预计算逆矩阵与关联矩阵K的乘积$H_R$，实现矩阵-向量乘法并行化。*

**公式4**: $$$s_{SW}^{n+1} = g_{SW}^{n+1} + (\bar{s}_{SW}^n (v_{SW}^{n+1} < 0) + s_{SW}^n (i_{SW}^{n+1} < 0))$$$

*自然换流开关状态判定方程，结合门极信号与上一时刻状态及当前电压/电流方向确定开关通断。*

**公式5**: $$$\begin{bmatrix} v^{n+1} \\ i_o^{n+1} \end{bmatrix} = \begin{bmatrix} v_{eq}^{n+1} \\ i_{eq}^{n+1} \end{bmatrix} + Y_{N_{eq}}^{-1} \begin{bmatrix} i^{n+1} \\ v_o^{n+1} \end{bmatrix}$$$

*PECSN外部电路的戴维南/诺顿等效模型，用于SNP第一阶段求解。*

### 算法步骤

1. 离线预处理：针对恒定导纳的外部网络，计算并固化导纳矩阵逆矩阵$Y^{-1}$、关联矩阵$K$、RMNA核心矩阵$H_R$以及外部等效矩阵$H_{eq}$和$Y_{N_{eq}}^{-1}$，生成固定FPGA硬件架构。

2. 时间步$n$预测计算：利用上一时刻状态$x_{node}^n$和$b_{node}^n$，通过公式$b_{node_{ext}}^{n+1} = C_h b_{node}^n + C_v x_{node}^n$提前计算当前步外部电路注入向量，打破数据依赖，启动并行流水线。

3. 外部等效求解（SNP阶段1）：基于预测的$b_{node_{ext}}^{n+1}$，计算PECSN端口等效源向量$b_{eq}^{n+1} = H_{eq} b_{node_{ext}}^{n+1}$，获得外部电路对换流器的等效激励。

4. 开关状态判定与PECSN求解（SNP阶段2）：根据等效端口电压/电流，利用开关状态判定方程迭代或直接求解各开关通断状态$s^{n+1}$，并计算开关电流$i_{SW}^{n+1} = H_{SW}(s^{n+1}) b_{eq}^{n+1}$。

5. 历史电流更新与RMNA并行计算（SNP阶段3）：同步更新支路历史电流$b_h^{n+1}$与元件历史源$b_c^{n+1}$。由于RMNA向量解耦，独立源计算、历史电流更新与最终网络求解$x_{node}^{n+1} = H_R b_{node}^{n+1}$在FPGA上全并行执行，无需等待。

6. 输出与迭代：输出当前步节点电压与支路电流，作为下一时间步的初始条件，循环执行直至仿真结束。

### 关键参数

- **仿真步长**: < 1 µs (亚微秒级)

- **开关模型**: 双值电阻模型 (Two-value resistor model)

- **积分方法**: 隐式积分法 (梯形法TR或后向欧拉法BE)

- **矩阵处理**: 预计算逆矩阵 $Y^{-1}$ 与固定关联矩阵 $K$

- **硬件平台**: FPGA (采用标准浮点运算核)

- **划分策略**: 求解层划分 (Solution-level partitioning / SNP)

## 适用边界

- 适用于含多个PEC且PEC之间电气距离很近、不能用传输线或stub line自然解耦的实时EMT仿真。
- SNP假设PECSN外部电路导纳恒定，并假设不同PECSN的开关过程相互独立；强耦合开关网络或会改变外部导纳的模型需要额外验证。

- 该平台服务HIL/DRTS小步长实时性，不等同于通用离线EMT求解器。

## 关键公式

### RMNA核心求解方程

$$$x_{node}^{n+1} = H_R b_{node}^{n+1}$$$

*用于FPGA上并行计算节点电压与未知量，通过预计算矩阵$H_R$将复杂矩阵求逆转化为高效的矩阵-向量乘法。*

### 开关状态隐式判定方程

$$$s_{SW}^{n+1} = g_{SW}^{n+1} + (\bar{s}_{SW}^n (v_{SW}^{n+1} < 0) + s_{SW}^n (i_{SW}^{n+1} < 0))$$$

*在双值电阻模型下，用于确定自然换流开关的通断状态，避免传统迭代求解带来的计算延迟。*

### 外部电路等效源预测方程

$$$b_{eq}^{n+1} = H_{eq} (C_h + C_v H_R) b_{node}^n$$$

*SNP方法的核心，利用上一时刻数据提前计算当前步等效源，实现网络求解与元件模型更新的硬件级并行。*
