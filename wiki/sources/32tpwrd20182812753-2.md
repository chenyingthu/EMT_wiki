## 摘要

—The computation of the electromagnetic transients in a power transformer with nonlinear material using the ﬁnite element method (FEM) is so dense that traditional nonlinear solver employing the Newton-Raphson method can hardly exe- cute in real-time. In this paper, we emulate the ﬁnite-element computation of electromagnetic transients of a transformer in real-time for the ﬁrst time. The transmission line modeling (TLM) method employed in the FEM successfully decoupled the nonlin- ear elements from the linear network so the nonlinearities could be solved individually, which is perfect for parallel processing.
