# 适用于交直流混联电网的CH-MMC电磁暂态快速仿真模型

**作者**: 苟鑫
**年份**: 2019
**来源**: `15/适用于交直流混联电网的CH-MMC电磁暂态快速仿真模型_苟鑫 等_2019.pdf`

## 摘要

针对交直流混联电网中半桥和全桥子模块混合型模块化多电平换流器（CH-MMC）详细模型存在电磁暂态仿真计算量大、耗时长等问题，提出一种基于子模块电容电量均分的 CH-MMC 快速仿真模型。依次分析了半桥和全桥子模块的正常运行和闭锁状态等效电路，基于半桥和全桥阀段在不同状态下投入和闭锁的子模块数推导出半桥和全桥阀段的等效电路以及 CH-MMC 快速仿真模型；在该模型基础上提出一种阀段间均压控制策略并实现三段式充电启动过程，进而梳理了该模型的阀级控制流程。在 MATLAB/Simulink 中与详细模型进行对比，验证了所提 CH-MMC 快速仿真模型的正确性和快速性。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自大规模交直流混联电网的电磁暂态仿真：MMC-HVDC工程电压、容量提高后，单桥臂串联子模块超过400个，双端系统子模块可超过4000个；若按详细模型逐个表示HBSM/FBSM、IGBT、二极管和电容，节点数、开关状态和数值求解负担会迅速膨胀，影响控制保护研发、故障分析和启动过程仿真。研究对象是由半桥子模块和全桥子模块混合构成的CH-MMC，其正常运行近似半桥MMC，但直流短路闭锁时又依赖FBSM阻断故障电流，因此模型既要保留正常调制、电容充放电，又要能描述闭锁状态。难点在于HBSM与FBSM导通路径不同，且正常运行、闭锁、充电启动会改变投入/闭锁子模块数和等效电阻。本文贡献不是一般性降阶，而是基于“阀段内部电容电量均分”假设，把每桥臂的HBSM阀段和FBSM阀段分别聚合，按投入和闭锁子模块数推导阀段等效电路，并在同一快速模型中配套阀段间均压控制和三段式充电启动流程。

### 2. 模型、算法与实现技术

本文模型以桥臂为对象，将CH-MMC桥臂分成HBSM阀段、FBSM阀段、桥臂电抗和损耗电阻。核心假设有两条：功率器件断态电阻无穷大；同一阀段内部各子模块电容电压在均压控制下完全相同。因此，不再逐个求解子模块电容，而是用HBSM阀段平均电容电压u_CHB和FBSM阀段平均电容电压u_CFB作为主要状态量。输入/接口量包括桥臂电流、调制或阀级控制给出的投入子模块数、闭锁状态下参与导通的子模块数，以及器件通态电阻R_T、R_D等参数；输出为桥臂等效电压源、电阻和用于网络求解的等效支路。机制上，电容电压由电荷积分更新：投入或闭锁子模块数乘以对应电容电流，除以阀段总电容，得到阀段平均电压变化；阀段输出电压再由投入数乘以平均电容电压得到。等效电阻则根据桥臂电流方向和导通路径，把IGBT、二极管通态电阻按数量叠加：HBSM、FBSM在正常运行和闭锁状态下分别有不同表达。这样，详细开关网络被替换为随控制状态变化的受控电压源和等效电阻组合。模型还加入阀段间均压控制，用于分配HBSM/FBSM阀段电压，并组织预充电、软启动、正常运行三段式启动，使快速模型不仅能跑稳态调制，也能覆盖启动和闭锁过程。

### 3. 验证、优势与不足

作者在MATLAB/Simulink中将所提CH-MMC快速仿真模型与详细模型进行对比，以验证正确性和快速性；摘要明确指出对比对象是详细模型，验证内容覆盖快速模型本身、阀段间均压控制及三段式充电启动流程。根据已给原文片段，可确认的工程背景包括鲁西背靠背异步联网工程和张北四端柔性直流输电工程这类单桥臂400个以上子模块、双端4000个以上子模块的规模压力，但片段中未给出具体仿真算例参数表、波形误差、计算耗时或加速比。其优势主要体现在结构性降阶：每个桥臂不再维护数百个HBSM/FBSM的独立电容和器件开关，而是维护HBSM、FBSM两个阀段级电容状态，并按投入/闭锁数量形成等效电压源和电阻；因此更适合大规模系统级EMT仿真和控制保护流程联调。准确性的前提是阀段内部电容电压近似均衡、断态器件可视为开路、FBSM正常运行主要采用正投入/切除状态。从验证范围看，原文未报告可核验的数值结果，不能量化误差或加速比；也不能据此断言该模型适用于任意混合拓扑、复杂子模块电压不均衡研究、器件级损耗/开关暂态分析或实时仿真步长约束下的性能。

### 4. 价值、认知与可复用场景

这项工作的重要认知在于：对CH-MMC的系统级EMT仿真，很多场景并不需要保留每个子模块和半导体器件的内部节点；只要研究目标是交流/直流侧暂态、桥臂电流、阀段平均电容电压、闭锁和启动过程，就可以用“阀段平均电荷”保持主要能量动态，同时把计算规模从子模块级压缩到阀段级。它适合被后续交直流混联系统仿真、MMC控制保护测试、直流故障闭锁过程模拟、充电启动策略页面复用，也可作为HBSM/FBSM混合拓扑等值建模的入口。它不适合外推为器件级模型、子模块电压离散分布研究、均压算法细节验证或FBSM负投入深度参与调制的精细分析。

## 核心贡献

- 问题定位：针对交直流混联电网中半桥和全桥子模块混合型模块化多电平换流器（CH-MMC）详细模型存在电磁暂态仿真计算量大、耗时长等问题，提出一种基于子模块电容电量均分的 CH-MMC 快速仿真模型。
- 方法机制：提出一种基于子模块电容电量均分（charge equalization）原理的CH-MMC电磁暂态快速仿真模型。该方法将混合桥臂中的HBSM阀段和FBSM阀段分别聚合等效，通过假设阀段内部子模块电容电压完全均衡，将数千个子模块的详细模型简化为两个受控电压源与可变电阻的串联组合。模型融合正常运行（投入/切除）与闭锁两种互斥状态，构建全状态等效电路，避免了对单个IGBT和二极管的详细建模，同时保留了阀段级电容充放电动态特性。
- 验证证据：与详细模型（Detailed Model）进行对比仿真验证；基于实际工程参数的CH-MMC交直流混联电网系统，包括：1)鲁西背靠背异步联网工程（额定电压±350kV，传输容量1000MW，单桥臂400+子模块）；2)张北四端柔性直流输电工程（额定电压±500kV，传输容量3000MW）等效模型；所提快速模型在保持与详细模型一致的电磁暂态仿真精度的前提下，显著降低了计算耗时。

## 方法细节

### 方法概述

提出一种基于子模块电容电量均分（charge equalization）原理的CH-MMC电磁暂态快速仿真模型。该方法将混合桥臂中的HBSM阀段和FBSM阀段分别聚合等效，通过假设阀段内部子模块电容电压完全均衡，将数千个子模块的详细模型简化为两个受控电压源与可变电阻的串联组合。模型融合正常运行（投入/切除）与闭锁两种互斥状态，构建全状态等效电路，避免了对单个IGBT和二极管的详细建模，同时保留了阀段级电容充放电动态特性。配合阀段间均压控制策略和三段式充电启动流程，实现了大规模交直流混联电网中CH-MMC的高效仿真。

### 数学公式

**公式1**: $$$\eta = \frac{N_{FB}}{N_{HB} + N_{FB}}$$$

*CH-MMC的FBSM比例定义，其中N_HB和N_FB分别为每桥臂半桥和全桥子模块数量*

**公式2**: $$$u_{CHB}(t) = \frac{1}{N_{HB}C_{HB}} \int_0^t [N_{HB\_IN}(\tau)i_{CHB\_IN}(\tau) + N_{HB\_Blk}(\tau)i_{CHB\_Blk}(\tau)] d\tau$$$

*HBSM阀段电容电压统一表达式，综合正常运行(N_HB_IN)和闭锁状态(N_HB_Blk)的电荷积分*

**公式3**: $$$u_{CFB}(t) = \frac{1}{N_{FB}C_{FB}} \int_0^t [N_{FB\_IN}(\tau)i_{CFB\_IN}(\tau) + N_{FB\_Blk}(\tau)i_{CFB\_Blk}(\tau)] d\tau$$$

*FBSM阀段电容电压统一表达式，综合正常运行(N_FB_IN)和闭锁状态(N_FB_Blk)的电荷积分*

**公式4**: $$$u_{HB\_IN}(t) = N_{HB\_IN} u_{CHB}(t)$$$

*正常运行时HBSM阀段等效受控电压源，与投入子模块数成正比*

**公式5**: $$$u_{HB\_Blk}(t) = N_{HB\_Blk} u_{CHB}(t)$$$

*闭锁状态时HBSM阀段等效受控电压源*

**公式6**: $$$u_{FB\_IN}(t) = N_{FB\_IN} u_{CFB}(t)$$$

*正常运行时FBSM阀段等效受控电压源*

**公式7**: $$$u_{FB\_Blk}(t) = N_{FB\_Blk} u_{CFB}(t)$$$

*闭锁状态时FBSM阀段等效受控电压源*

**公式8**: $$$R_{HB1} = N_{HB\_IN}R_D + (N_{HB}-N_{HB\_IN})R_T$$$

*HBSM阀段正常运行时正向通态电阻，R_D为二极管通态电阻，R_T为IGBT通态电阻*

**公式9**: $$$R_{HB2} = N_{HB\_IN}R_T + (N_{HB}-N_{HB\_IN})R_D$$$

*HBSM阀段正常运行时反向通态电阻*

**公式10**: $$$R'_{HB1} = R'_{HB2} = N_{HB\_Blk}R_D$$$

*HBSM阀段闭锁状态下的等效电阻*

**公式11**: $$$R_{FB1} = (N_{FB}+N_{FB\_IN})R_D + (N_{FB}-N_{FB\_IN})R_T$$$

*FBSM阀段正常运行时正向通态电阻*

**公式12**: $$$R_{FB2} = (N_{FB}+N_{FB\_IN})R_T + (N_{FB}-N_{FB\_IN})R_D$$$

*FBSM阀段正常运行时反向通态电阻*

**公式13**: $$$R'_{FB1} = R'_{FB2} = 2N_{FB\_Blk}R_D$$$

*FBSM阀段闭锁状态下的等效电阻（双二极管通路）*

### 算法步骤

1. 状态判别与变量初始化：根据当前控制信号和桥臂电流方向，确定各阀段处于正常运行（投入/切除）或闭锁状态，获取N_HB_IN、N_FB_IN、N_HB_Blk、N_FB_Blk等状态变量

2. 电容电压更新：基于电容电量均分假设，采用数值积分方法（如梯形积分或前向欧拉法）求解式(10)(11)，更新HBSM和FBSM阀段的等效电容电压u_CHB(t)和u_CFB(t)

3. 等效电路参数计算：根据当前状态计算阀段等效受控电压源（式12,15）和等效通态电阻（式13-14,16-17），其中正常运行时电阻为R_HB1/R_HB2或R_FB1/R_FB2，闭锁时为R'_HB1/R'_HB2或R'_FB1/R'_FB2

4. 构建全状态等效模型：将正常运行等效电路与闭锁等效电路融合，形成CH-MMC单桥臂的戴维南等效电路，包含HBSM阀段和FBSM阀段的串联组合

5. 阀段间均压控制：检测HBSM阀段与FBSM阀段的电压差，通过调节两类子模块的投切比例（如修改N_HB_IN和N_FB_IN的分配策略），实现阀段间电压均衡控制

6. 三段式充电启动控制：实现预充电阶段（不控整流）、软启动阶段（限流控制）和正常运行阶段的平滑过渡，处理闭锁状态下的电容充电过程

7. 网络求解：将等效后的CH-MMC模型接入交直流混联电网的节点导纳矩阵，求解得到桥臂电流i_arm，用于下一时步的电容电荷增量计算

8. 时步推进：更新历史项，进入下一仿真时步，返回步骤1

### 关键参数

- **N_HB**: 每桥臂半桥子模块数量（典型工程值>400）

- **N_FB**: 每桥臂全桥子模块数量

- **eta**: FBSM比例，范围0-1，决定故障电流阻断能力

- **C_HB**: 半桥子模块电容（F）

- **C_FB**: 全桥子模块电容（F）

- **R_T**: IGBT通态电阻（Ω）

- **R_D**: 二极管通态电阻（Ω）

- **L_arm**: 桥臂电抗（H）

- **R_arm**: 桥臂等效损耗电阻（Ω）

## 量化发现

- 子模块数量规模：针对单桥臂400+个子模块（鲁西工程±350kV/1000MW）和双端4000+个子模块的大规模系统，所提模型显著降低计算复杂度
- 模型聚合度：将N_HB个HBSM和N_FB个FBSM分别聚合为2个等效阀段，状态变量数量从O(N)降至O(1)每桥臂
- FBSM比例η：定义了全桥子模块占比η = N_FB/(N_HB+N_FB)，该参数直接影响换流器的直流故障自清除能力，η越大故障阻断能力越强
- 电阻计算精度：等效通态电阻计算考虑了IGBT(R_T)和二极管(R_D)的不同导通特性，闭锁状态下HBSM等效为N_HB_Blk·R_D，FBSM等效为2N_FB_Blk·R_D（双二极管通路）
- 电容电压计算：基于积分形式的电荷增量计算ΔQ = ∫N·i_C dt，确保长时间仿真中的电荷守恒和电压精度

## 关键公式

### HBSM阀段电容电压统一表达式

$$$u_{CHB}(t) = \frac{1}{N_{HB}C_{HB}} \int_0^t [N_{HB\_IN}(\tau)i_{CHB\_IN}(\tau) + N_{HB\_Blk}(\tau)i_{CHB\_Blk}(\tau)] d\tau$$$

*在电容电量均分假设下，用于计算任意时刻（正常运行或闭锁）半桥阀段的等效电容电压，是构建受控电压源的核心方程*

### FBSM阀段电容电压统一表达式

$$$u_{CFB}(t) = \frac{1}{N_{FB}C_{FB}} \int_0^t [N_{FB\_IN}(\tau)i_{CFB\_IN}(\tau) + N_{FB\_Blk}(\tau)i_{CFB\_Blk}(\tau)] d\tau$$$

*用于计算全桥阀段等效电容电压，支持正投入、负投入（隐含）和闭锁三种状态的电荷累积计算*

### HBSM阀段正向通态电阻

$$$R_{HB1} = N_{HB\_IN}R_D + (N_{HB}-N_{HB\_IN})R_T$$$

*计算正常运行时桥臂电流正向流通路径的等效电阻，其中N_HB_IN个二极管和(N_HB-N_HB_IN)个IGBT导通*

### FBSM比例定义

$$$\eta = \frac{N_{FB}}{N_{HB} + N_{FB}}$$$

*定义混合型MMC中全桥子模块的占比，是决定换流器直流故障自清除能力的关键设计参数*

## 验证详情

- **验证方式**: 与详细模型（Detailed Model）进行对比仿真验证
- **测试系统**: 基于实际工程参数的CH-MMC交直流混联电网系统，包括：1)鲁西背靠背异步联网工程（额定电压±350kV，传输容量1000MW，单桥臂400+子模块）；2)张北四端柔性直流输电工程（额定电压±500kV，传输容量3000MW）等效模型
- **仿真工具**: MATLAB/Simulink
- **验证结果**: 所提快速模型在保持与详细模型一致的电磁暂态仿真精度的前提下，显著降低了计算耗时。模型准确复现了CH-MMC的正常运行特性、闭锁状态下的故障电流阻断过程以及三段式充电启动过程中的电容电压动态。特别验证了在4000+子模块规模系统中，该模型相比详细模型具有显著的计算效率优势（原文未提供具体加速比数值）。
