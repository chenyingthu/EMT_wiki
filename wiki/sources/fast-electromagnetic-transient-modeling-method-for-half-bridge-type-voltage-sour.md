# Fast Electromagnetic Transient Modeling Method for Half-bridge-Type Voltage Source Converter Based o

**作者**: Zhang 等
**年份**: 2023
**来源**: `18/Zhang 等 - 2021 - Fast Electromagnetic Transient Modeling Method for Half-bridge-Type Voltage Source Converter Based o.pdf`

## 摘要


现代电力系统存在大量的半桥型电压源变流器，采用传统电磁暂态仿真程序对其进行大规模仿真分析时，存在耗时高、效率低的问题。以半桥子电路为开关状态判断的基本单元，通过分析其开关状态变化时二极管的续流及关断过程，得出普适于半桥型电压源变流器的同步开关预判方法。该方法可在当前时步通过逻辑判断直接得出稳定的开关状态组合，消除了迭代计算。结合同步开关快速预判方法及内节点收缩方法构建了半桥型电压源变流器的快速仿真模型，通过对比仿真，验证了快速仿真模型具有与全详细化模型相当的仿真精度，且能有效减少仿真耗时，提高仿真效率。与全详细化模型相比，针对 80 模块固态变压器的快速仿真模型可加速 20 倍。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自含大量电力电子装置的电力系统暂态分析：半桥型VSC广泛用于新能源、微电网、直流输电等场景，研究者需要在正常、故障、闭锁和状态切换中保留开关级电磁暂态细节。研究对象不是单个器件，而是由半桥子电路构成的一类VSC拓扑族。难点在于全详细化模型需用百纳秒至微秒级步长捕获高频开关，且某个开关动作会触发二极管续流、关断等同步开关，传统EMT需反复重建导纳矩阵并全局因子分解直到状态稳定。本文的创新是把“半桥子电路”作为开关判断单元，利用门极信号、电流方向和二极管续流/反压机理，预判当前时步稳定开关组合，再结合内节点收缩形成端口等效模型，而不是依赖逐次迭代收敛。

### 2. 模型、算法与实现技术

方法包含两层：同步开关预判和内节点收缩。器件层面，IGBT和反并联二极管仍按开关级模型处理，导通/关断分别用小电阻和大电阻表示；每个IGBT/二极管组只允许3种状态：T_OFF/D_OFF、T_OFF/D_ON、T_ON/D_OFF。算法输入包括上一时步稳定状态、当前门极信号、支路电流方向及相关端电压极性；输出是当前时步可直接用于组网的稳定开关状态向量。其机制是：当门极变化或电流换向时，不先求一次电压再检查二极管，而是依据半桥上下管之间的续流路径和反向承压关系，判断哪些二极管必须导通、哪些器件必须关断，从而跳过同步开关迭代。网络层面，离散化后的节点方程为G(t)v(t)=i(t)+I_h(t)。对VSC内部节点做高斯消元/节点收缩，得到仅含外部端口的等效导纳Y_B1=Y_B−Y_BIY_I^{-1}Y_BI^T和等效历史电流源i_hB1=i_hB−Y_BIY_I^{-1}i_hI；全局网络只求外部端口电压，再由u_I=Y_I^{-1}(i_hI−Y_BI^Tu_B)恢复内部节点量并更新历史源。

### 3. 验证、优势与不足

作者采用快速模型与全详细化分立开关模型的对比仿真来验证。原文摘要明确给出的测试对象是80模块固态变压器，基线是全详细化模型，指标包括波形精度是否相当和仿真耗时；可核验的量化结论是该算例中快速模型相对全详细化模型可加速20倍。论文还声称该方法可仿真正常运行、短路、闭锁、状态切换等任意工况，但在当前提供的原文片段中，具体波形误差、时间步长设置、硬件平台、CPU耗时表、故障参数和控制参数未展示，因此不能把“误差小于某数值”作为已核验结论。优势主要来自两个机制：一是同步开关状态在当前时步由逻辑关系直接给出，减少反复因子分解；二是内节点收缩降低VSC接入全局网络时的矩阵规模。边界也相应清晰：该验证主要支持半桥型VSC及80模块SST算例，不能自动外推到全桥、混合子模块、详细器件非线性、热模型、死区/寄生参数主导的暂态或硬实时平台性能。

### 4. 价值、认知与可复用场景

这项工作的认知价值在于把EMT仿真慢的原因从“开关多”进一步定位为“同步开关导致稳定状态需迭代搜索”，并说明半桥结构中二极管续流和关断具有可预判的局部逻辑。它适合被后续半桥VSC、MMC半桥子模块、SST、多端口电力电子网络等页面复用，用于解释如何在保留开关级行为的同时减少全局矩阵更新和内部节点规模。工程上可服务于大规模离线EMT、控制保护策略校验和故障工况快速扫描。不适合外推为平均模型替代品，也不宜直接用于作者未验证的拓扑、器件非理想特性、极端高频寄生振荡或实时仿真加速结论。

## 方法细节

### 方法概述

本文提出一种基于同步开关预判与内节点收缩的半桥型VSC快速电磁暂态建模方法。首先，以半桥子电路为基本单元，定义IGBT/二极管组的三种开关状态。通过分析桥臂电流方向与门极信号，建立同步开关预判逻辑，在当前仿真步长内直接推导出上下管因续流或反压导致的稳定开关状态组合，彻底消除传统EMT仿真中因同步开关引发的全局矩阵反复迭代与因子分解。其次，结合内节点收缩技术，将包含大量内部节点的复杂VSC拓扑等效为仅含外部端口的诺顿等效电路，大幅缩减系统导纳矩阵规模。该方法支持正常运行、短路、闭锁及状态切换等全工况仿真，在保证与全详细模型精度一致的前提下，显著提升大规模电力电子网络的计算效率。

### 数学公式


**公式1**: $$$G(t)v(t) = i(t) + I_h(t)$$$

*网络节点电压方程，用于描述离散化后电力电子网络的电气关系*


**公式2**: $$$Y_{B1} = Y_B - Y_{BI}Y_I^{-1}Y_{BI}^T$$$

*内节点收缩后的等效导纳矩阵计算公式，用于消除内部节点*


**公式3**: $$$i_{hB1} = i_{hB} - Y_{BI}Y_I^{-1}i_{hI}$$$

*内节点收缩后的等效历史电流源计算公式*


**公式4**: $$$u_I = Y_I^{-1}(i_{hI} - Y_{BI}^T u_B)$$$

*内部节点电压恢复公式，用于求解外节点电压后反推内部状态*


### 算法步骤

1. 步骤1：网络元件离散化。将IGBT与二极管等效为两态可变电阻（导通0.01Ω，关断10^8Ω），利用Dommel算法将电感、电容离散为电阻与历史电流源并联的诺顿等效形式。

2. 步骤2：初步开关状态判断。根据上一时步稳定状态$x(t-\Delta T)$、当前门极信号$G$及桥臂电流$i_{hb}(t)$，利用IGBT/二极管状态转移图初步判断各开关的暂定状态。

3. 步骤3：同步开关预判与状态修正。应用同步导通与关断逻辑公式，检测对侧二极管是否因电感续流需强制导通（状态置1）或因承受反压需强制关断（状态置0），直接输出当前时步稳定开关状态向量$X(t)$，跳过传统迭代过程。

4. 步骤4：导纳矩阵更新与内节点收缩。根据稳定状态向量更新子网络导纳矩阵，利用高斯消元法（内节点收缩）计算仅含外部端口的等效导纳矩阵$Y_{B1}$和等效历史电流源$i_{hB1}$。

5. 步骤5：全局求解与状态恢复。将等效模型接入全局网络求解外节点电压$u_B$，利用内部节点电压恢复公式计算$u_I$，更新所有历史电流源向量，进入下一仿真时步。


### 关键参数

- **开关通态电阻**: 0.01 Ω

- **开关断态电阻**: 10^8 Ω

- **开关状态定义**: 0(T_OFF/D_OFF), 1(T_OFF/D_ON), 2(T_ON/D_OFF)

- **仿真步长范围**: 百纳秒至微秒级

## 关键公式

### 内节点收缩等效导纳矩阵

$$$Y_{B1} = Y_B - Y_{BI}Y_I^{-1}Y_{BI}^T$$$

*在同步开关状态预判完成后，用于将复杂VSC内部节点消去，形成仅含外端口的诺顿等效模型*

### 同步导通预判逻辑

$$$\begin{cases} x_2(t)=1 \\ x_1(t)=1 \end{cases} \text{ 当 } x_1(t-\Delta T)=2, x_1(t)=0, i_{hb}(t)>0 \text{ 或 } x_2(t-\Delta T)=2, x_2(t)=0, i_{hb}(t)<0$$$

*用于判断因电感续流导致对侧二极管强制导通的情况，直接修正开关状态向量*

### 同步关断预判逻辑

$$$\begin{cases} x_2(t)=0 \\ x_1(t)=0 \end{cases} \text{ 当 } x_1(t-\Delta T)=0, x_1(t)=2, i_{hb}(t)>0 \text{ 或 } x_2(t-\Delta T)=0, x_2(t)=2, i_{hb}(t)<0$$$

*用于判断因承受反压导致对侧二极管强制关断的情况，直接修正开关状态向量*
