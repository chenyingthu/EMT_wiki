## 方法细节

### 方法概述

本文提出一种基于同步开关预判与内节点收缩的半桥型VSC快速电磁暂态建模方法。首先，以半桥子电路为基本单元，定义IGBT/二极管组的三种开关状态。通过分析桥臂电流方向与门极信号，建立同步开关预判逻辑，在当前仿真步长内直接推导出上下管因续流或反压导致的稳定开关状态组合，彻底消除传统EMT仿真中因同步开关引发的全局矩阵反复迭代与因子分解。其次，结合内节点收缩技术，将包含大量内部节点的复杂VSC拓扑等效为仅含外部端口的诺顿等效电路，大幅缩减系统导纳矩阵规模。该方法支持正常运行、短路、闭锁及状态切换等全工况仿真，在保证与全详细模型精度一致的前提下，显著提升大规模电力电子网络的计算效率。

### 数学公式


**公式1**: $$$G(t)v(t) = i(t) + I_h(t)$$$

*网络节点电压方程，用于描述离散化后电力电子网络的电气关系*


**公式2**: $$$Y_{B1} = Y_B - Y_{BI}Y_I^{-1}Y_{BI}^T$$$

*内节点收缩后的等效导纳矩阵计算公式，用于消除内部节点*


**公式3**: $$$i_{hB1} = i_{hB} - Y_{BI}Y_I^{-1}i_{hI}$$$

*内节点收缩后的等效历史电流源计算公式*


**公式4**: $$$u_I = Y_I^{-1}(i_{hI} - Y_{BI}^T u_B)$$$

*内部节点电压恢复公式，用于求解外节点电压后反推内部状态*


### 算法步骤

1. 步骤1：网络元件离散化。将IGBT与二极管等效为两态可变电阻（导通0.01Ω，关断10^8Ω），利用Dommel算法将电感、电容离散为电阻与历史电流源并联的诺顿等效形式。

2. 步骤2：初步开关状态判断。根据上一时步稳定状态$x(t-\Delta T)$、当前门极信号$G$及桥臂电流$i_{hb}(t)$，利用IGBT/二极管状态转移图初步判断各开关的暂定状态。

3. 步骤3：同步开关预判与状态修正。应用同步导通与关断逻辑公式，检测对侧二极管是否因电感续流需强制导通（状态置1）或因承受反压需强制关断（状态置0），直接输出当前时步稳定开关状态向量$X(t)$，跳过传统迭代过程。

4. 步骤4：导纳矩阵更新与内节点收缩。根据稳定状态向量更新子网络导纳矩阵，利用高斯消元法（内节点收缩）计算仅含外部端口的等效导纳矩阵$Y_{B1}$和等效历史电流源$i_{hB1}$。

5. 步骤5：全局求解与状态恢复。将等效模型接入全局网络求解外节点电压$u_B$，利用内部节点电压恢复公式计算$u_I$，更新所有历史电流源向量，进入下一仿真时步。


### 关键参数

- **开关通态电阻**: 0.01 Ω

- **开关断态电阻**: 10^8 Ω

- **开关状态定义**: 0(T_OFF/D_OFF), 1(T_OFF/D_ON), 2(T_ON/D_OFF)

- **仿真步长范围**: 百纳秒至微秒级

## 关键公式

### 内节点收缩等效导纳矩阵

$$$Y_{B1} = Y_B - Y_{BI}Y_I^{-1}Y_{BI}^T$$$

*在同步开关状态预判完成后，用于将复杂VSC内部节点消去，形成仅含外端口的诺顿等效模型*

### 同步导通预判逻辑

$$$\begin{cases} x_2(t)=1 \\ x_1(t)=1 \end{cases} \text{ 当 } x_1(t-\Delta T)=2, x_1(t)=0, i_{hb}(t)>0 \text{ 或 } x_2(t-\Delta T)=2, x_2(t)=0, i_{hb}(t)<0$$$

*用于判断因电感续流导致对侧二极管强制导通的情况，直接修正开关状态向量*

### 同步关断预判逻辑

$$$\begin{cases} x_2(t)=0 \\ x_1(t)=0 \end{cases} \text{ 当 } x_1(t-\Delta T)=0, x_1(t)=2, i_{hb}(t)>0 \text{ 或 } x_2(t-\Delta T)=0, x_2(t)=2, i_{hb}(t)<0$$$

*用于判断因承受反压导致对侧二极管强制关断的情况，直接修正开关状态向量*



## 验证详情

- **验证方式**: 对比仿真分析（快速模型 vs 全详细模型）
- **测试系统**: 80模块级联型固态变压器（SST）系统
- **仿真工具**: 基于节点电压法与Dommel算法的自定义电磁暂态求解程序
- **验证结果**: 验证了快速模型在正常、短路、闭锁等工况下的波形精度与全详细模型一致，且计算效率提升20倍，有效解决了大规模半桥型VSC仿真耗时高的问题。
