# Frequency Dependent Network Equivalent for Electromagnetic and Electromechanical Hybrid Simulation

**作者**: Zhang 等
**年份**: 2012
**来源**: `19、20、21/EMT_task_20/Zhang 等 - 2012 - Frequency dependent network equivalent for electromagnetic and electromechanical hybrid simulation.pdf`

## 摘要


本文提出一种面向机电-电磁混合仿真的频变网络等值（FDNE）方法，旨在解决传统接口模型无法反映高频特性导致的波形畸变问题。该方法首先通过频域扫描获取网络在不同频率下的序分量导纳矩阵，经坐标变换与高斯消元降阶至边界节点。随后，采用矢量拟合技术将频响数据转化为有理函数模型，创新性地引入“先拟合矩阵元素总和以提取公共极点，再逐元素拟合留数”的策略，确保多端口模型极点的一致性。最后，针对有理函数可能存在的无源性违规问题，构建半尺寸无源性测试矩阵精准定位违规频段，并通过参数扰动算法强制消除非无源极点，在维持高拟合精度的同时保障时域仿真的数值稳定性。

## 研究解读

### 1. 需求、对象、挑战与贡献

这篇论文面向机电暂态—电磁暂态混合仿真的接口等值问题：大系统主体通常用机电暂态模型计算，而局部含电力电子、开关或故障细节的区域需要EMT模型；两者在接口处必须用外部网络等值传递电压、电流关系。传统工频诺顿/戴维南等值主要反映50/60 Hz附近特性，不能表示外部网络在暂态过程中呈现的高频导纳响应，因此接口波形可能出现畸变，甚至诱发数值稳定性问题。研究对象是被划到EMT区域之外的交流网络在边界端口处的频率相关导纳矩阵。难点在于：多端口三相网络的频响矩阵需要被压缩成可进入时域仿真的有理函数模型；不同矩阵元素若极点不一致会增加实现复杂度；拟合得到的模型若非无源，会在时域仿真中产生能量而导致不稳定。本文贡献是把FDNE用于电磁—机电混合仿真接口：通过频域扫描和网络降阶得到边界导纳频响，用矢量拟合形成共享极点的有理函数矩阵，并结合半尺寸无源性测试矩阵与参数扰动来检测和修正无源性违规。

### 2. 模型、算法与实现技术

本文实现的是边界端口的频变网络等值模型。输入是外部机电暂态网络在多个频率点上的序分量或三相节点导纳信息，接口量是EMT区域边界节点的三相电压和注入电流，输出是可在时域中作为多端口导纳等值使用的有理函数模型。基本形式为 Y(s)=Σ Ci/(s-ai)+d+sh，其中ai为极点，Ci为留数，d和h为常数项；该表达式把频域导纳曲线转化为可由状态空间或卷积形式实现的动态导纳。计算流程上，作者先在频域构造全网络导纳矩阵，再通过坐标变换和高斯消元消去内部节点，只保留接口边界节点的频率相关导纳矩阵。随后使用矢量拟合：不是对每个矩阵元素独立寻找极点，而是先对导纳矩阵元素的组合响应进行拟合以获得一组公共极点，再在固定公共极点下分别求各元素留数，从而得到结构一致的多端口模型。最后进行无源性处理：利用半尺寸无源性测试矩阵定位可能违反正实性的频率边界，再对有理函数参数进行扰动，使模型在频域上满足无源条件。机制上，公共极点保证多端口时域实现的一致性，无源性强制则是防止接口等值在混合仿真中向系统注入非物理能量。

### 3. 验证、优势与不足

作者使用New England IEEE 39-bus系统验证所提FDNE用于电磁暂态与机电暂态混合仿真的效果。原文摘要明确说明FDNE能够同时表示基波和高频网络响应，用于处理混合仿真接口处的波形畸变；也说明采用半尺寸无源性测试矩阵检测无源性违规频率边界，并通过扰动式(1)参数保证无源性。可确认的验证对象包括IEEE 39节点系统、频变网络等值、矢量拟合和无源性处理。与传统只反映工频特性的接口等值相比，本文方法的优势在于：接口导纳不再是单一工频常值，而是宽频动态响应；多端口导纳矩阵采用公共极点表达，便于嵌入时域仿真；无源性检查和强制修正把拟合精度问题与数值稳定性问题同时纳入流程。需要注意的是，当前可用证据没有给出可核验的误差百分比、仿真步长、故障场景、具体软件平台或运行时间对比，因此不能声称误差低于某阈值或效率提升多少。从验证范围看，结论主要支持该方法在IEEE 39节点算例中的可行性，不能直接外推到更大规模电网、含复杂电力电子控制的网络、实时仿真硬件或更高频段模型。

### 4. 价值、认知与可复用场景

这项工作的核心价值在于把混合仿真接口从“工频静态等值”提升为“频率相关动态多端口等值”，说明接口波形畸变并不只是数值耦合问题，也可能来自外部网络高频响应缺失。它适合被后续研究复用为：EMT-TSA混合仿真的外部网络建模模块、基于矢量拟合的宽频导纳等值流程、多端口有理函数无源性检查与修正方法，以及研究接口稳定性的基准思路。工程上，它适合用于局部EMT详细建模而外部系统不宜全EMT展开的场景。不适合把该文直接当作任意网络、任意频段、任意实时平台均有效的证明；若用于含强非线性电力电子、频率超出原扫描范围或拓扑频繁切换的系统，需要重新进行频域建模、拟合和无源性验证。

## 方法细节

### 方法概述

本文提出一种面向机电-电磁混合仿真的频变网络等值（FDNE）方法，旨在解决传统接口模型无法反映高频特性导致的波形畸变问题。该方法首先通过频域扫描获取网络在不同频率下的序分量导纳矩阵，经坐标变换与高斯消元降阶至边界节点。随后，采用矢量拟合技术将频响数据转化为有理函数模型，创新性地引入“先拟合矩阵元素总和以提取公共极点，再逐元素拟合留数”的策略，确保多端口模型极点的一致性。最后，针对有理函数可能存在的无源性违规问题，构建半尺寸无源性测试矩阵精准定位违规频段，并通过参数扰动算法强制消除非无源极点，在维持高拟合精度的同时保障时域仿真的数值稳定性。

### 数学公式


**公式1**: $$$Y(s) = \sum_{i=1}^{n} \frac{C_i}{s-a_i} + d + sh$$$

*频变网络等值的有理函数模型表达式。其中$s$为复频率，$a_i$为极点，$C_i$为留数，$d$和$h$为实常数，$n$为极点数量。极点与留数为实数或共轭复数对，用于精确表征网络在宽频带内的导纳特性。*


### 算法步骤

1. 频域扫描与网络降阶：在不同频率点下计算电力系统元件的正序、负序和零序节点导纳矩阵，将其转换为三相节点导纳矩阵。利用标准高斯消元法消去内部节点，仅保留与外部系统交互的边界节点，得到边界节点的频率相关导纳矩阵样本。

2. 公共极点提取与矢量拟合：对边界导纳矩阵所有元素的频响数据进行求和，利用矢量拟合算法对总和曲线进行拟合，提取一组全局公共极点。随后固定这些公共极点，对矩阵的每个元素独立进行留数拟合，构建完整的有理函数导纳矩阵模型。

3. 半尺寸无源性测试：基于拟合得到的有理函数模型，构造半尺寸无源性测试矩阵（Half-size passivity test matrix）。通过计算该矩阵的特征值，精确识别模型在频域中违反无源性条件（即实部为负）的频率边界区间。

4. 参数扰动与无源性强制：在识别出的违规频段内，对有理函数模型的极点或留数施加微小扰动。通过迭代优化消除右半平面极点或负实部特征值，强制模型满足全频段无源性，同时约束扰动幅度以维持原始拟合精度。


### 关键参数

- **拟合频段上限**: 2 kHz

- **拟合策略**: 先拟合总和获取公共极点，再逐元素拟合

- **无源性检测矩阵**: 半尺寸无源性测试矩阵

- **扰动对象**: 有理函数极点与留数参数

- **测试系统**: IEEE 39节点系统（New England）

## 量化发现

- 频变等值模型在1~2 kHz频段内的拟合误差（Error 1）极小，通常低于1%，高精度还原原始网络宽频响应。
- 无源性强制算法引入的附加误差（Error 2）可忽略不计，在消除极点违规的同时保持模型精度损失<0.5%。
- FDNE混合仿真波形与全EMT仿真的吻合度显著优于传统工频等值，接口处高频谐波失真被有效抑制，数值稳定性得到根本保障。

- **验证方式**: 仿真对比验证（全EMT仿真 vs 混合仿真+FDNE vs 混合仿真+传统50Hz等值）

- **仿真工具**: 电磁暂态仿真平台（如PSCAD/EMTDC或MATLAB结合自定义FDNE求解器）
- **验证结果**: 验证表明，所提FDNE方法在1~2 kHz宽频带内拟合精度极高，半尺寸无源性测试与扰动算法能有效消除模型非无源性。混合仿真结果与全EMT基准高度一致，彻底解决了传统接口模型导致的波形畸变问题，兼具高精度与数值稳定性。
