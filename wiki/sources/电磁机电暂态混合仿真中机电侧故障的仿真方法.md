# 电磁–机电暂态混合仿真中机电侧故障的仿真方法

**作者**: 张怡
**年份**: 2012
**来源**: `35/张怡 等 - 2012 - 电磁–机电暂态混合仿真中机电侧故障的仿真方法.pdf`

## 摘要

The frequency dependent network equivalent (FDNE) based electromagnetic and electromechanical hybrid simulation system can not simulate the faults on the electromechanical network side. A hot standby technique for large-scale electromechanical network is proposed.

## 核心贡献



- 提出伴随仿真（热备用）方法初始化故障后FDNE状态，实现平稳切换并抑制虚假暂态
- 提出临界电气距离指标，用于快速判断机电侧故障时是否需要更新FDNE
- 提出基于临界电气距离指标的FDNE切换策略，在保证仿真精度的同时显著降低计算负担

## 主要发现



- 伴随仿真技术能有效初始化新FDNE状态，大幅减少切换过程中的数值暂态波动
- 临界电气距离指标可准确鉴别故障影响范围，避免不必要的FDNE重算与切换
- 基于该指标的切换策略在维持混合仿真精度的前提下，有效提升了大规模网络仿真效率
