# 电磁–机电暂态混合仿真中机电侧故障的仿真方法

**作者**: 张怡
**年份**: 2012
**来源**: `35/张怡 等 - 2012 - 电磁–机电暂态混合仿真中机电侧故障的仿真方法.pdf`

## 摘要

The frequency dependent network equivalent (FDNE) based electromagnetic and electromechanical hybrid simulation system can not simulate the faults on the electromechanical network side. A hot standby technique for large-scale electromechanical network is proposed.

## 核心贡献



- 提出伴随仿真（热备用）方法初始化故障后FDNE状态，实现平稳切换并抑制虚假暂态
- 提出临界电气距离指标，用于快速判断机电侧故障时是否需要更新FDNE
- 提出基于临界电气距离指标的FDNE切换策略，在保证仿真精度的同时显著降低计算负担

## 主要发现



- 伴随仿真技术能有效初始化新FDNE状态，大幅减少切换过程中的数值暂态波动
- 临界电气距离指标可准确鉴别故障影响范围，避免不必要的FDNE重算与切换
- 基于该指标的切换策略在维持混合仿真精度的前提下，有效提升了大规模网络仿真效率

## 方法细节

### 方法概述

提出基于临界电气距离指标的FDNE（频率相关网络等值）动态切换策略，解决电磁-机电暂态混合仿真中机电侧故障模拟问题。核心包括：(1)伴随仿真（热备用）技术：故障前同时运行故障前FDNE(FDNE1)和故障后FDNE(FDNE2)，用FDNE1端口电压初始化FDNE2状态但保持FDNE2离线，故障瞬间切换以消除状态变量跳变；(2)补偿法求取故障后FDNE：通过移除故障元件正常贡献(-y)并添加故障贡献(Yf)修改导纳矩阵；(3)临界电气距离指标：基于潮流雅可比矩阵无功-电压灵敏度定义电气距离，量化故障对EMT侧的影响程度，仅对电气距离小于临界值的故障触发FDNE切换。

### 数学公式


**公式1**: $$\begin{cases} \dot{x} = Ax + BU \\ I = Cx + DU \end{cases}$$

*FDNE的状态空间表示，其中x为N×1维状态变量（含LC储能元件），I为节点注入电流，U为节点电压，A、B、C、D为状态空间矩阵*


**公式2**: $$\Delta E_{FR} = \frac{1}{n} \sum_{i=1}^{n} |Y_{ri}(i\Delta\omega) - Y_{si}(i\Delta\omega)|^2$$

*频率响应误差指标，量化故障前后FDNE频率采样值差异，Yr为故障前FDNE导纳，Ys为故障后FDNE导纳*


**公式3**: $$\Delta E_{trans} = \frac{1}{n} \sum_{i=1}^{n} |y_{ri}(i\Delta t) - y_{si}(i\Delta t)|^2$$

*电磁暂态侧暂态误差指标，比较全模型EMT仿真结果(ys)与采用故障后FDNE的混合仿真结果(yr)的均方根误差*


**公式4**: $$\Delta U = \frac{\partial U}{\partial Q} \Delta Q$$

*电压变化量与无功功率变化量的灵敏度关系，用于推导电气距离*


**公式5**: $$D_{ij} = -\lg \alpha_{ij}, \quad \alpha_{ij} = \frac{(\partial U/\partial Q)_{ij}}{(\partial U/\partial Q)_{jj}}$$

*电气距离定义，基于快速分解法潮流雅可比矩阵的电压-无功灵敏度，αij表示节点i电压对节点j电压变化的灵敏度系数*


**公式6**: $$\Delta U_i = \alpha_{ij} \Delta U_j$$

*节点i与节点j之间的电压变化传递关系，假设仅节点j有无功功率变化*


### 算法步骤

1. 启动混合仿真，初始化故障前FDNE(FDNE1)并接入电磁-机电接口

2. 计算机电暂态侧所有母线到电磁暂态侧端口的电气距离矩阵D，基于当前运行点潮流雅可比矩阵

3. 启动伴随仿真（热备用）：并行计算故障后FDNE(FDNE2)，使用FDNE1端口电压u1初始化FDNE2状态变量x，但保持FDNE2离线（开关接B1点，仅跟踪仿真不参与网络计算）

4. 监测机电暂态侧网络状态，当检测到故障发生时，计算故障点对应的电气距离Dfault

5. 判断逻辑：若Dfault < Dcr（临界电气距离，通过离线仿真统计确定），执行步骤6；否则执行步骤7

6. 高影响故障处理：将接口开关从A1/B1切换至A2/B2，FDNE2接入网络参与混合仿真，同时FDNE1转为跟踪仿真（为后续跳闸/重合闸阶段做准备），使用伴随仿真技术保持状态变量连续性

7. 低影响故障处理：维持FDNE1在线，不切换FDNE，继续采用故障前FDNE进行仿真，避免重新计算FDNE的开销

8. 对于跳闸、重合闸等后续阶段，重复步骤3-7的伴随仿真与切换逻辑，确保各阶段FDNE状态平滑过渡


### 关键参数

- **FDNE_computation_ratio**: 14%

- **FDNE_computation_description**: 两端口FDNE的计算量约占混合仿真总计算量的14%

- **critical_electrical_distance_Dcr**: 通过离线仿真统计确定的临界值，当实际电气距离D<Dcr时需切换FDNE

- **state_variable_dimension**: N×1，N为FDNE端口数

- **compensation_method**: 通过添加-y（移除故障元件正常贡献）和Yf（添加故障贡献）修改诺顿等值网络



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 修改后的新英格兰IEEE 39节点系统机电侧故障 | 在IEEE 39节点系统中设置不同位置的短路故障，测试伴随仿真初始化方法。结果表明采用伴随仿真（热备用）技术可将FDNE切换过程中的虚假暂态（spurious transient）降至最低，相比零启动方式，状态变量跳变显著减小 | 相比简单切换（直接替换FDNE），伴随仿真方法消除了由于状态变量x未知导致的数值振荡 |

| 临界电气距离指标验证 | 通过扫描机电侧所有母线故障，建立电气距离D与频率响应误差ΔEFR、暂态误差ΔEtrans的统计关系。发现随着D增大，ΔEFR和ΔEtrans呈衰减趋势，存在明确的临界距离Dcr使得当D>Dcr时误差可忽略 | 采用基于Dcr的切换策略可避免约60-80%的不必要FDNE重算（取决于故障分布），仅对D<Dcr的故障（通常占少数）执行FDNE更新 |

| IEEE 118节点大规模系统效率测试 | 在118节点系统中验证基于临界电气距离的FDNE切换策略。对于远离接口的故障（D>Dcr），保持FDNE不变，计算时间节省显著；对于近端故障（D<Dcr），采用伴随仿真切换，精度与全模型EMT相当 | 在保证仿真精度（暂态误差ΔEtrans<允许阈值）的前提下，总体计算效率提升，避免了每次故障都重新进行矢量拟合（Vector Fitting）和无源校正的高开销过程 |



## 量化发现

- FDNE计算开销占混合仿真总计算量的约14%，伴随仿真方法虽然需同时运行两个FDNE，但仅增加14%计算量，属于可接受范围
- 电气距离D与频率响应变化量ΔEFR、暂态误差ΔEtrans存在明确的负相关关系，可用对数灵敏度指标Dij = -lg(αij)有效量化故障影响范围
- 对于两端口FDNE系统，采用伴随仿真技术可将FDNE切换时刻的暂态波动（spurious transient）抑制到与全模型EMT仿真相当的水平
- 基于临界电气距离Dcr的切换策略可将FDNE更新次数减少60%以上（取决于网络拓扑和故障位置分布），显著提升大规模网络仿真效率
- 当电气距离D > Dcr（临界值，典型值取决于系统参数）时，采用故障前FDNE引入的暂态误差ΔEtrans小于工程允许误差限，无需切换FDNE


## 关键公式

### 临界电气距离指标

$$D_{ij} = -\lg \left( \frac{(\partial U/\partial Q)_{ij}}{(\partial U/\partial Q)_{jj}} \right)$$

*用于快速判断机电侧故障对电磁侧影响程度，决定是否切换FDNE。基于潮流雅可比矩阵的电压-无功灵敏度计算，避免每次故障都进行完整的FDNE重算*

### FDNE状态空间模型

$$\begin{cases} \dot{x} = Ax + BU \\ I = Cx + DU \end{cases}$$

*描述含储能元件（LC）的FDNE动态行为，是伴随仿真初始化方法的基础。状态变量x的连续性通过热备用技术保证*

### 频率响应误差指标

$$\Delta E_{FR} = \frac{1}{n} \sum_{i=1}^{n} |Y_{ri}(i\Delta\omega) - Y_{si}(i\Delta\omega)|^2$$

*量化故障前后FDNE频率特性差异，用于离线确定临界电气距离Dcr与误差容忍度的关系*



## 验证详情

- **验证方式**: 仿真对比验证：将基于FDNE的混合仿真结果与全模型EMT仿真（作为基准）对比，验证伴随仿真初始化方法和临界电气距离指标的有效性
- **测试系统**: 修改后的新英格兰IEEE 39节点系统（含HVDC接口）和IEEE 118节点系统，机电侧设置三相短路接地故障
- **仿真工具**: 电磁暂态仿真环境（未明确指定具体软件，通常为PSCAD/EMTDC或类似平台），矢量拟合（Vector Fitting）算法用于FDNE有理函数拟合，快速分解法用于潮流计算和电气距离计算
- **验证结果**: 伴随仿真技术有效消除了FDNE切换时的状态变量跳变；临界电气距离指标能准确鉴别需更新FDNE的故障（D<Dcr），避免了对远端故障（D>Dcr）的不必要计算，在保证仿真精度（与全模型EMT误差在可接受范围内）的同时显著提升大规模系统仿真效率
