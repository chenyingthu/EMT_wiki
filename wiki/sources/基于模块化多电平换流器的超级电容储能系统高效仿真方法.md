## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求来自MMC-SCES电磁暂态研究：这种换流器把超级电容通过非隔离双向Buck-Boost环节嵌入半桥子模块，既要模拟交直流换流，又要模拟子模块电容、超级电容、电感及开关状态的快速动态。随着电压等级和电平数增加，6个桥臂中级联大量子模块，开关器件和动态元件可达数百乃至上千，DSM逐器件建模会导致节点数、导纳矩阵维数和时步求解次数过大；AVM虽快，却平均化了子模块，不能得到不同子模块内部电压、电流差异，也难以支持桥臂内部状态分析。本文研究对象是采用超级电容经Buck-Boost接入的MMC-SCES，而不是常规MMC或蓄电池储能MMC。创新点在于把快速等效从“桥臂外特性”推进到“复杂储能子模块内部变量可回代”：先对子模块内动态元件作Dommel离散和开关二值电阻化，再对子模块及整条桥臂作戴维南等效以降维求解，最后用桥臂电流反算子模块内部节点电压，从而在保留电容电压、超级电容电压等变量的同时避免DSM的大规模网络求解。

### 2. 模型、算法与实现技术

作者提出的是基于桥臂戴维南等效电路的MMC-SCES嵌套快速同时求解模型。模型的外部接口量是每个桥臂对系统网络呈现的等效电阻和等效电压源，网络求解输出桥臂电流；内部状态量包括子模块电容历史电压源、超级电容历史电压源、子模块电感历史电压源以及由开关状态决定的T1—T4等效电阻。实现上，电容、电感先经梯形积分离散为定时步电阻与历史电压源：电容等效电阻随步长和电容值确定，电感等效电阻随电感和步长确定，历史源携带上一时刻状态。IGBT不再作为详细半导体器件，而用导通/关断二值电阻表示。随后先把储能Buck-Boost支路等效为戴维南电阻和电压源，再与半桥电容及T1、T2组合得到完整子模块戴维南参数；同一桥臂内N个级联子模块的等效电阻、电压源按串联关系求和，形成桥臂级等效。全网只求解降维后的桥臂网络。得到桥臂电流后，再利用预先符号求逆或预存的子模块3×3节点导纳矩阵逆阵，将桥臂电流和历史源组成激励向量，回代求出子模块节点电压，并更新电容、超级电容和电感的历史量进入下一时步。

### 3. 验证、优势与不足

作者在PSCAD/EMTDC中搭建双端MMC-SCES输电系统验证方法，测试系统为每站21电平、换流站额定容量400 MW、储能容量120 MW、交流侧110 kV、直流侧±200 kV；更细的电感、电容等参数原文指向附录B表B1。验证基线包括详细开关模型DSM和平均值模型AVM，比较维度包括稳态外部波形、子模块内部动态变量拟合、计算耗时和绝对误差标幺化积分λNIAE。原文报告的主要量化结论是：相对DSM，所提方法最高仿真速度提升约518倍；仿真λNIAE控制在1%以内；相对AVM，所提方法能拟合子模块内部动态变量，而AVM只能较好描述桥臂或换流器外特性。优势不只是“快”，而是通过桥臂等效减少网络求解规模，同时通过内部节点回代保留DSM中才有的子模块电容电压、超级电容电压等状态。从验证范围看，论文主要证明了双端21电平系统中与DSM/AVM的对比有效性；518倍和1%以内不能直接外推到更高电平数、不同储能接入拓扑、故障穿越、器件损耗、极端开关暂态或不同控制策略。由于IGBT被二值电阻替代，若研究目标是开关过程、损耗、过电压细节或器件级故障，该模型不等价于DSM。

### 4. 价值、认知与可复用场景

这项工作的核心认知是：对含储能支路的MMC，不必在“DSM保内部变量但很慢”和“AVM快但丢内部变量”之间二选一；只要子模块拓扑可被线性离散并能形成戴维南等效，就可把大规模网络求解压缩到桥臂层，再用桥臂电流恢复子模块内部状态。它适合复用于MMC-SCES控制策略批量仿真、双端储能型直流输电系统电磁暂态研究、需要观察超级电容电压和子模块电容电压均衡的场景，也可为其他非隔离储能接入型MMC的快速模型提供建模思路。但不应把其结果外推为器件级开关精度、损耗评估能力，或未经重新推导就套用于隔离型储能子模块、不同功率变换器拓扑和未验证的故障场景。

### 证据边界

- 来自原文：研究对象是超级电容经非隔离双向Buck-Boost电路接入子模块的MMC-SCES，测试平台为PSCAD/EMTDC，系统为双端21电平、400 MW、120 MW储能、110 kV交流侧、±200 kV直流侧。

- 原文未在给定摘录中报告可核验的仿真步长、硬件配置、完整耗时表和附录B具体电感电容数值，因此不同硬件或步长下的518倍加速不可直接复现判断。

## 方法细节

### 方法概述

本文提出了一种基于桥臂戴维南等效电路的MMC-SCES嵌套快速同时求解方法。该方法首先利用Dommel算法（梯形积分法）将子模块内的动态元件（电容、电感）等效为定时步电阻与历史电压源串联支路，将IGBT等开关器件等效为二值电阻（导通/关断状态）。随后对含储能单元的子模块进行戴维南等效，再将桥臂内级联的N个子模块聚合为单一戴维南等效电路，从而大幅降低网络节点数和导纳矩阵维数。在求解过程中，采用符号运算预先对子模块节点导纳矩阵求逆，将复杂的矩阵求逆运算转化为简单的乘法运算，显著加速计算过程。最后利用求得的桥臂电流作为激励，反向求解子模块内部各节点电压，获取电容电压、超级电容电压等全部动态变量。

### 数学公式

**公式1**: $$$u_C(t) = u_C(t-\Delta t) + \frac{\Delta t}{2C}\left[i_C(t) + i_C(t-\Delta t)\right]$$$

*电容电压的梯形积分离散化公式，用于将电容等效为电阻与电压源串联*

**公式2**: $$$U_{Chis} = u_C(t-\Delta t) + \frac{\Delta t}{2C}i_C(t-\Delta t)$$$

*电容等效电路中的附加历史电压源，代表上一时刻的状态记忆*

**公式3**: $$$R_C = \frac{\Delta t}{2C}$$$

*电容的定时步等效电阻，与仿真步长成正比*

**公式4**: $$$U_{Lhis} = u_L(t-\Delta t) + \frac{2L}{\Delta t}i_L(t-\Delta t)$$$

*电感等效电路中的附加历史电压源*

**公式5**: $$$R_L = \frac{2L}{\Delta t}$$$

*电感的定时步等效电阻，与仿真步长成反比*

**公式6**: $$$R_{DCeq} = R_{T3} + \frac{(R_L + R_{SC})R_{T4}}{R_L + R_{SC} + R_{T4}}$$$

*储能单元（含超级电容、电感、开关管T3/T4）的戴维南等效电阻*

**公式7**: $$$U_{DCeq} = \frac{R_{T4}(U_{SChis} - U_{Lhis})}{R_{SC} + R_L + R_{T4}}$$$

*储能单元的戴维南等效电压源*

**公式8**: $$$R_{SMeq} = \frac{R_{T2}[R_{DCeq}R_C + R_{T1}(R_{DCeq} + R_C)]}{R_{DCeq}R_C + (R_{DCeq} + R_C)(R_{T1} + R_{T2})}$$$

*完整子模块（含半桥和储能单元）的戴维南等效电阻*

**公式9**: $$$U_{SMeq} = \frac{R_{T2}(R_{DCeq}U_{Chis} + R_CU_{DCeq})}{R_{DCeq}R_C + (R_{DCeq} + R_C)(R_{T1} + R_{T2})}$$$

*完整子模块的戴维南等效电压源*

**公式10**: $$$R_{jr}^{eq} = \sum_{i=1}^{N}R_{SMi}^{jr}, \quad U_{jr}^{eq} = \sum_{i=1}^{N}U_{SMi}^{jr}$$$

*桥臂级戴维南等效参数，将N个级联子模块聚合为单一路径*

**公式11**: $$$\mathbf{U}_n = \mathbf{Y}_{SM}^{-1}\mathbf{I}_S$$$

*子模块节点电压求解方程，其中Y_SM为3×3导纳矩阵，I_S为包含桥臂电流和历史电压源的激励向量*

**公式12**: $$$u_{SC}(t) = U_{SChis} + (U_{n3} - U_{SChis} + U_{Lhis})\frac{R_{SC}}{R_L + R_{SC}}$$$

*根据节点电压计算超级电容实时电压的解析式*

### 算法步骤

1. 初始化：设定仿真参数，计算各动态元件的定时步等效电阻R_C、R_L、R_SC，构建子模块节点导纳矩阵Y_SM并预求逆

2. 时步推进：在当前仿真时刻t，根据开关状态（T1-T4的通断）确定各IGBT的等效电阻值（R_on=0.01Ω或R_off=0.5MΩ）

3. 子模块等效：根据当前开关状态和上一时刻历史电压源，按式(9)(10)计算各子模块的戴维南等效参数R_SMeq和U_SMeq

4. 桥臂等效：按式(11)将桥臂内N个子模块的等效参数求和，得到桥臂级戴维南等效电路R_jr^eq和U_jr^eq

5. 网络求解：求解降维后的MMC-SCES全网方程，获得桥臂电流I_ch作为激励源

6. 内部回代：利用预存的Y_SM^{-1}和激励向量I_S=[I_ch, U_Chis/R_C, (U_SChis-U_Lhis)/(R_SC+R_L)]^T，按式(13)计算子模块内部节点电压U_n1, U_n2, U_n3

7. 状态更新：按式(14)计算电容电压u_C(t)和超级电容电压u_SC(t)，更新历史电压源U_Chis、U_Lhis、U_SChis，准备下一时步计算

### 关键参数

- **R_on**: 0.01 Ω（IGBT导通电阻）

- **R_off**: 0.5 MΩ（IGBT关断电阻）

- **N**: 21（每桥臂子模块数/电平数）

- **P_rated**: 400 MW（换流站额定容量）

- **E_rated**: 120 MW（储能系统额定容量）

- **U_dc**: ±200 kV（直流母线额定电压）

- **U_ac**: 110 kV（交流系统额定电压）

- **Δt**: 电磁暂态仿真步长（具体数值未明确给出，通常为20-50μs级）

### 仿真测试

|---------|---------|----------|

| 稳态运行工况对比 | 在双端MMC-SCES系统稳态运行下，高效仿真模型与详细开关模型（DSM）的A相阀侧电压U_VA波形高度吻合，接近正弦波；直流端口电压U_dc方面，高效仿真模型和平均值模型（AVM）的高频噪声过滤效果优于DSM，因为DSM中大量子模块带来较多高频噪声 | 高效仿真模型与DSM的波形相似度极高，λNIAE指标非常小；AVM因忽略开关控制动态导致精度损失 |

## 关键公式

### 子模块节点电压快速求解公式

$$$\mathbf{U}_n = \mathbf{Y}_{SM}^{-1}\mathbf{I}_S$$$

*在获得桥臂电流后，利用预求逆的导纳矩阵快速计算子模块内部3个节点的电压，避免每时步重复求逆，是加速仿真的核心*

### 桥臂戴维南等效聚合公式

$$$R_{jr}^{eq} = \sum_{i=1}^{N}R_{SMi}^{jr}, \quad U_{jr}^{eq} = \sum_{i=1}^{N}U_{SMi}^{jr}$$$

*将桥臂内N个级联子模块聚合为单一等效电路，将网络节点数从O(N)降至O(1)，大幅降低导纳矩阵维数*

### Dommel算法等效电阻

$$$R_C = \frac{\Delta t}{2C}, \quad R_L = \frac{2L}{\Delta t}$$$

*基于梯形积分法的电磁暂态等效，将动态元件转化为电阻与历史电压源串联，是电磁暂态仿真的基础*

## 适用边界

- 适用于 MMC-SCES 电磁暂态仿真中需要兼顾详细开关模型精度和平均值模型效率的双端系统研究。
- 方法依赖桥臂戴维南等效和子模块内部节点电压回代；如果研究目标是单个 IGBT 开关瞬态、损耗或故障电弧等器件级现象，仍需 DSM 或更详细模型。
- 518 倍提速和 λNIAE 小于 1% 来自论文双端 21 电平、400MW、120MW 储能系统算例；不同电平数、控制策略、故障类型和硬件条件下需重新验证。
- 预求逆和嵌套快速求解要求子模块拓扑和开关等效关系保持在模型假设内，拓扑扩展到隔离型储能接入或其他储能器件时不能直接套用全部公式。

## 验证详情

- **验证方式**: 对比分析（与详细开关模型DSM和平均值模型AVM进行多维度对比）
- **测试系统**: 双端MMC-SCES输电系统，每站21电平，额定容量400MW，储能容量120MW，交流侧110kV，直流侧±200kV，系统详细参数见附录B表B1
- **仿真工具**: PSCAD/EMTDC仿真平台
- **验证结果**: 所提高效仿真方法在保持与DSM相近精度的同时（绝对误差标幺化积分<1%），仿真速度提升约518倍，且能保留所有动态变量（包括子模块电容电压、超级电容电压等），显著优于AVM仅模拟外特性的能力
