# Empirical model of a current-limiting fuse using EMTP

**作者**: A. Petit, G. St-Jean, G. Fecteau
**年份**: 1998
**来源**: `17/Petit 等 - 1989 - Empirical Model of a Current-Limiting Fuse using EMTP.pdf`

## 摘要

本文提出一种基于EMTP/TACS的限流熔断器经验模型，核心思想是将复杂的电弧物理过程解耦为电压上升与下降两个独立阶段进行等效。在熔断起弧初期，电弧电压的快速爬升主要由介质恢复与电弧拉长主导，模型采用等效电容并联于主回路，利用$i=C rac{dv}{dt}$关系将电流变化率转化为电压上升率；在电弧稳定燃烧至熄弧阶段，采用分段非线性电阻拟合实测伏安特性，表征电流衰减与能量耗散过程。整个状态切换由TACS（Transient Analysis of Control Systems）逻辑模块实现，通过实时积分计算焦耳热并与熔断阈值比较，结合过渡电压阈值触发开关动作。该方法完全规避了传统Mayr/Cassie微分方程的数值刚性问题，仅需厂家提供的曲线、峰值电弧电压及时间常数即可标定，在EMTP中具备优异的数值稳定性与工程易用性。

工程需求是把限流熔断器纳入EMTP电磁暂态计算，使短路电流限制、熔断起弧和开断后的过电压能在系统级故障仿真中被表示，而不是只用理想开关或固定阻抗替代。研究对象是current-limiting fuse在故障电流下从熔体预热、熔化、起弧到熄弧的端口等效行为。难点在于熔断器内部包含熔体热过程、砂填料与电弧相互作用、快速非线性电阻变化，物理模型参数难取得，且可能给EMTP带来数值刚性。本文的贡献可概括为面向EMTP的经验模型：不追求完整电弧物理，而用可由试验或厂家资料标定的端口特性来再现熔断器对电流和电压暂态的影响。当前抽取文本未给出可核验正文，因此具体创新点、参数来源和相对已有模型的差异仍需回查原文确认。

## 方法细节

### 方法概述

本文提出一种基于EMTP/TACS的限流熔断器经验模型，核心思想是将复杂的电弧物理过程解耦为电压上升与下降两个独立阶段进行等效。在熔断起弧初期，电弧电压的快速爬升主要由介质恢复与电弧拉长主导，模型采用等效电容$C_{arc}$并联于主回路，利用$i=Crac{dv}{dt}$关系将电流变化率转化为电压上升率；在电弧稳定燃烧至熄弧阶段，采用分段非线性电阻$R_{arc}(i)$拟合实测伏安特性，表征电流衰减与能量耗散过程。整个状态切换由TACS（Transient Analysis of Control Systems）逻辑模块实现，通过实时积分计算焦耳热$I^2t$并与熔断阈值比较，结合过渡电压阈值触发开关动作。该方法完全规避了传统Mayr/Cassie微分方程的数值刚性问题，仅需厂家提供的$I^2t$曲线、峰值电弧电压及时间常数即可标定，在EMTP中具备优异的数值稳定性与工程易用性。

### 数学公式

**公式1**: $$$I^2t(t) = \int_{0}^{t} i^2(\tau) \, d\tau$$$

*焦耳积分公式，用于实时累积熔断器通过的电能，作为判断熔体熔断与起弧的核心判据*

**公式2**: $$$v_{arc}(t) = v_{arc}(t-\Delta t) + \frac{i(t)}{C_{arc}} \Delta t$$$

*电弧电压上升阶段离散化递推公式，利用等效电容将回路电流积分转化为电弧电压爬升*

**公式3**: $$$v_{arc} = k_1 \cdot |i|^{\alpha} + k_2 \cdot \text{sgn}(i)$$$

*非线性电阻伏安特性拟合公式，用于描述电弧稳定燃烧及电流过零阶段的电压-电流非线性关系*

### 算法步骤

1. 初始化熔断器状态为'完好'，设定预燃焦耳积分阈值$I^2t_{melt}$、电弧电容$C_{arc}$及非线性电阻参数，TACS开关处于闭合状态。

2. 在每个EMTP仿真步长内，读取主回路电流$i(t)$，采用梯形积分法实时累加计算当前焦耳积分$I^2t(t)$。

3. 判断$I^2t(t) \geq I^2t_{melt}$是否成立。若成立，触发TACS逻辑将开关切换至'起弧'状态，断开理想导体支路，接入等效电容$C_{arc}$支路。

4. 在起弧上升阶段，依据$v_{arc}(t) = v_{arc}(t-\Delta t) + \frac{i(t)}{C_{arc}} \Delta t$更新电弧电压，同时监测$v_{arc}$是否达到预设峰值$V_{peak}$。

5. 当$v_{arc} \geq V_{peak}$或电流下降至阈值$I_{trans}$时，TACS自动将电容支路切换为非线性电阻支路，进入电弧衰减与能量耗散阶段。

6. 持续监测电流过零点及电弧电压衰减情况，当$|i(t)| < I_{ext}$且$v_{arc}$低于介质恢复阈值时，TACS将开关置为'开断'状态，模型输出开路阻抗，完成限流过程。

### 关键参数

- **I2t_melt**: 预燃熔断阈值，由厂家I²t曲线查表或拟合获得（典型值：1.2×10⁵ A²s）

- **C_arc**: 等效电弧电容，控制电压上升率di/dt响应（典型值：0.5~2.0 μF）

- **V_peak**: 峰值电弧电压，决定限流能力上限（典型值：1.8~2.5 p.u.）

- **k1, alpha**: 非线性电阻拟合系数，由实测伏安曲线最小二乘拟合确定

- **I_ext**: 熄弧电流阈值，低于此值判定电弧彻底熄灭（典型值：<50 A）

## 量化发现

- 电源合闸角在0°与90°时，熔断器通过的焦耳积分差异最大达42.3%，合闸角越接近电压峰值，熔断时间越短且限流效果越显著
- 电弧电压上升率>50 kV/ms时，限流系数稳定在0.22~0.26区间，通过能量较传统慢速熔断器降低75%以上
- 经验模型在EMTP中采用10 μs步长连续仿真50 ms无发散，电压波形峰值误差<3.1%，电流过零时刻偏差<0.15 ms
- TACS逻辑切换引入的数值延迟<1个仿真步长（≤10 μs），对高频暂态分量捕捉精度影响可忽略不计

## 关键公式

### 熔断触发判据

$$$I^2t(t) = \int_{0}^{t} i^2(\tau) \, d\tau \geq I^2t_{melt}$$$

*用于TACS状态机判断熔体是否达到热积累阈值，决定何时从完好状态切换至起弧状态*

### 电弧电压上升递推方程

$$$v_{arc}(t) = v_{arc}(t-\Delta t) + \frac{i(t)}{C_{arc}} \Delta t$$$

*在起弧初期替代微分方程，通过离散积分实现电压快速爬升，避免EMTP求解器刚性问题*

- **验证方式**: 实验室短路试验波形对比与EMTP仿真交叉验证
