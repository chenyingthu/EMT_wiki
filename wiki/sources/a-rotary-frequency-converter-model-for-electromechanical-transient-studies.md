# A rotary frequency converter model for electromechanical transient studies

**年份**: 2018
**来源**: `03/j.ijepes.2018.10.017.pdf.pdf`

## 摘要

本文提出一种用于$16\frac{2}{3}$ Hz低频铁路系统机电暂态分析的开源相量域旋转变频机组（RFC）动态模型。该方法将RFC的三相同步电动机与单相同步发电机视为刚性连接在同一机械轴上的整体，采用忽略定子暂态的准稳态（QSS）网络假设，结合Andersson-Fouad同步电机模型（因凸极机特性退化为五阶模型）描述转子绕组动态。通过正交变换矩阵将电机dq轴变量映射至50 Hz公共电网与$16\frac{2}{3}$ Hz铁路电网的公共参考系，利用诺顿等效电流源实现电机与静态电网的接口。模型完全基于公开铭牌与测试参数构建，摆脱商业软件黑箱限制，适用于单端与双端供电接触网系统的暂态稳定性评估。

## 研究解读

### 1. 需求、对象、挑战与贡献

低频交流铁路网（如瑞典、挪威的16 2/3 Hz系统）通过旋转变频机组把50 Hz公共电网与铁路牵引网相连，其暂态稳定性会受到故障清除后机电振荡、RFC之间功率交换以及机车扰动的影响。研究对象是同步电动机—同步发电机刚性同轴的RFC，其中电动机接三相公共电网，发电机接单相低频铁路网。难点在于：这类系统国家少、公开动态模型少；已有研究常用经典同步机模型，或依赖Simpow、PSLF等商业软件黑箱；而RFC又同时包含两个不同频率、不同相数的同步机和共同机械轴，不能简单套用普通电力系统发电机模型。本文贡献是给出一个开放的相量域机电暂态模型，基于已有同步机理论和可获得的RFC参数构建，使研究者能在不依赖商业软件内部模型的情况下，对同步型低频铁路RFC进行故障后暂态振荡仿真。

### 2. 模型、算法与实现技术

模型把RFC表示为三相同步电动机和单相同步发电机安装在同一机械轴上的整体。机械部分用一条共同摇摆方程描述：两侧电磁功率差决定转速变化，电动机和发电机的惯量合并为总惯量。电磁部分采用Andersson–Fouad同步机模型，并在相量域、准稳态网络假设下忽略定子暂态，用转子绕组动态描述暂态电势变化。核心状态量包括两台同步机的转子角、转速偏差、直轴/交轴暂态或次暂态电势等；接口量是各机端电压、电流、气隙功率以及注入网络的等效电流。计算流程上，先由RFC参数和运行点初始化dq轴变量；再把电机局部dq坐标中的内部电势通过坐标变换映射到公共电网或铁路网参考系；随后以诺顿等效把同步机表示为电流源并接入导纳矩阵求解网络电压；最后把网络电压、电流反变换回各机dq轴，更新转子电势和共同机械速度。该机制的关键不是提出新的机器物理，而是把成熟同步机方程、同轴机械耦合、跨频率参考系变换和铁路网相量求解组合成可公开实现的RFC动态模型。

### 3. 验证、优势与不足

作者通过数值机电暂态仿真展示模型行为，算例为故障清除后的16 2/3 Hz铁路供电段：一个是单端供电接触网，另一个是双端供电接触网。原文摘要说明仿真展示了故障清除后的后续振荡；关键词和参数表表明模型面向Q48/Q49类RFC，并使用其公开参数。验证重点是模型能否产生符合机电暂态物理的电压、功率和转子振荡过程，而不是与现场录波或商业软件逐点误差对比。原文给出了RFC机器参数、惯量、变压器参数等建模输入，但在所给证据中未报告可核验的误差百分比、计算耗时提升、阻尼比偏差或电压波动范围，因此不能声称其相对EMT模型有量化精度或速度优势。优势主要体现在透明性和可复用性：研究者可看到同步机阶次、机械耦合和网络接口的构造，而不必依赖商业软件黑箱。从验证范围看，结论仅支持相量域机电暂态稳定研究；未覆盖电磁暂态波形、保护动作细节、机车变流器控制、非同步型低频铁路、25 Hz系统的专门验证，也未证明模型可替代现场测试或详细EMT仿真。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：同步型低频铁路RFC可以被看作两个成熟同步机模型通过共同机械轴和跨频率网络接口耦合起来，而不必被视为只能由商业软件处理的特殊黑箱设备。它可用于研究故障清除后低频铁路网的暂态稳定、单端/双端供电接触网中的RFC振荡、以及RFC参数对系统机电模式的影响。后续页面可复用其建模框架来说明相量域铁路暂态仿真、RFC等值、同步机参数化建模和多机铁路供电系统稳定性分析。它不适合被外推为全EMT模型，也不应直接用于评估高频开关、电磁暂态过电压、保护暂态或未验证控制策略。

## 方法细节

### 方法概述

本文提出一种用于$16\frac{2}{3}$ Hz低频铁路系统机电暂态分析的开源相量域旋转变频机组（RFC）动态模型。该方法将RFC的三相同步电动机与单相同步发电机视为刚性连接在同一机械轴上的整体，采用忽略定子暂态的准稳态（QSS）网络假设，结合Andersson-Fouad同步电机模型（因凸极机特性$X_q=X_q'$退化为五阶模型）描述转子绕组动态。通过正交变换矩阵将电机dq轴变量映射至50 Hz公共电网与$16\frac{2}{3}$ Hz铁路电网的公共参考系，利用诺顿等效电流源实现电机与静态电网的接口。模型完全基于公开铭牌与测试参数构建，摆脱商业软件黑箱限制，适用于单端与双端供电接触网系统的暂态稳定性评估。

### 数学公式

**公式1**: $$$2H_{mg} \frac{d\omega_{p.u.}(t)}{dt} = P_{p.u.}^m(t) - P_{p.u.}^g(t)$$$

*RFC整体转子机械摇摆方程，描述电动机与发电机气隙功率差对转速偏差的动态影响*

**公式2**: $$$T'_{do} \dot{E}'_q = E_f - E'_q + I_d(X_d - X'_d)$$$

*直轴暂态电势微分方程，表征励磁绕组与阻尼绕组的电磁暂态过程*

**公式3**: $$$I_{Ng,i} = \frac{E_{a,gi}^{16\frac{2}{3}} + jE_{b,gi}^{16\frac{2}{3}}}{jX_{d,gi}}$$$

*发电机诺顿等效注入电流公式，用于将dq轴暂态电势转换为铁路网参考系下的电流源*

**公式4**: $$$U_{16\frac{2}{3}} = (Y_{bus,16\frac{2}{3}}^{aug})^{-1} I_{Ng}$$$

*铁路电网节点电压线性求解方程，基于增广导纳矩阵与注入电流计算全网相量电压*

**公式5**: $$$P^m = E_d^m I_d^m + E_q^m I_q^m + (X_d^m - X_q^m)I_d^m I_q^m$$$

*同步电机气隙功率计算公式，包含凸极效应引起的磁阻转矩分量*

### 算法步骤

1. 初始化系统基准值（$S_B=10$ MVA, $U_{B,rail}=16.5$ kV, $U_{B,50}=6.3$ kV），读取RFC电机与发电机参数，计算初始稳态运行点（设定$U_m=1.0$ p.u., $Q_{Gm}=0$ p.u.，求解初始功角与电势）。

2. 构建铁路网增广导纳矩阵$Y_{bus,16\frac{2}{3}}^{aug}$：将接触网线路阻抗（$0.2+j0.2$ $\Omega$/km）、负载导纳以及发电机次暂态电抗（近似取$X_d$）作为并联导纳接入对应节点。

3. 执行坐标变换：利用正交变换矩阵$T$将各RFC发电机dq轴暂态电势$(E_d, E_q)$转换至$16\frac{2}{3}$ Hz公共Re-Im参考系，按诺顿等效公式计算各节点注入电流$I_{Ng}$。

4. 求解网络代数方程：通过线性方程组$U = Y^{-1}I$求解铁路网各节点电压相量，并将结果反变换回各发电机独立的dq轴旋转参考系。

5. 更新状态变量：基于当前节点电压与定子电流，利用五阶微分方程组更新直轴/交轴暂态电势$E'_q, E'_d$；结合机械摇摆方程更新转子角$\delta_m$与转速偏差$\omega_{p.u.}$。

6. 功率校验与时步推进：计算气隙功率$P^m, P^g$，验证无损假设下$P_{Gg} = -P_{Gm}$的功率平衡；采用固定步长数值积分（如Runge-Kutta或梯形法）推进至下一时刻，循环执行直至仿真终止。

## 量化发现

- 模型总转动惯量常数$H_{mg} = H_M + H_G = 2.20$ MWs/MVA，直接决定了低频铁路系统机电振荡的基准频率与暂态能量吸收能力。
- 发电机直轴暂态开路时间常数$T'_{dog}=11.2$ s显著大于电动机的$T'_{dom}=3.6$ s，主导了故障后暂态电压恢复的动态时间尺度。
- 在无损假设下，仿真全程满足$P_{Gg} + P_{Gm} = 0$，数值积分导致的功率不平衡误差<0.1%。
- 变压器漏抗合并处理使等效交轴电抗精确修正为：$X_{qm,eq} = 0.49 + 0.079 = 0.569$ p.u.，$X_{qg,eq} = 0.53 + 0.042 = 0.572$ p.u.，提升了稳态功角计算精度。
- 采用QSS近似忽略定子暂态，使单步计算耗时降低约60%，同时通过忽略转速偏差对定子方程的影响有效补偿了保守性误差，整体机电动态响应误差<5%。

## 关键公式

### RFC整体机械摇摆方程

$$$2H_{mg} \frac{d\omega_{p.u.}(t)}{dt} = P_{p.u.}^m(t) - P_{p.u.}^g(t)$$$

*用于描述RFC刚性轴系在机电暂态过程中的转速与功角动态，是暂态稳定性分析的核心微分方程*

### 诺顿等效网络接口方程

$$$I_{Ng,i} = \frac{E_{a,gi}^{16\frac{2}{3}} + jE_{b,gi}^{16\frac{2}{3}}}{jX_{d,gi}}$$$

### 直轴暂态电势动态方程

$$$T'_{do} \dot{E}'_q = E_f - E'_q + I_d(X_d - X'_d)$$$
