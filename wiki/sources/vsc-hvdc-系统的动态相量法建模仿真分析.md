## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自电磁暂态—机电暂态混合仿真：VSC-HVDC 含有PWM开关动作、交流基频动态和直流电容能量动态，若在混合仿真中始终采用详细开关级时域模型，仿真步长会受开关过程限制，难以满足实时或大系统联算需求。研究对象是由两个对称电压源换流器经直流电容和直流线路连接、交流侧通过变压器接入系统的 VSC-HVDC。难点在于：详细模型能描述开关函数与三相电流、电容电压的耦合，但状态波形含高频纹波；传统平均模型又可能丢失随时间变化的基波包络。本文的贡献是在已有开关函数模型基础上，引入基于时变傅里叶级数的动态相量法，把交流侧电流、直流侧电压和开关函数表示为滑动窗傅里叶系数，保留对系统主动态有贡献的直流分量和基波分量，得到可作为混合仿真接口的降阶动态相量模型，而不是直接求解全开关时域方程。

### 2. 模型、算法与实现技术

本文模型从三相abc开关函数方程出发。以VSC1的a相为例，交流电感方程包含电阻压降、由直流电压与开关函数相乘形成的桥臂电压项、三相开关函数求和项以及交流系统电压。随后对状态变量采用动态相量定义：在一个滑动周期窗口内计算第k阶傅里叶系数。其机制是把原始瞬时波形拆成随时间慢变的包络系数与旋转因子。相量微分特性把时域导数转化为相量导数并引入-jkω项，因此基波电流的实部、虚部会出现旋转耦合；相量乘积特性把“直流电压×开关函数”“电流×开关函数”等乘积转化为傅里叶系数卷积，从而可在相量域处理换流器调制作用。PWM开关函数用面积等效得到基波占空比形式，例如d1a=0.5+(m1/2)cos(ωt-δ1)，其中调制比m和触发角δ进入模型，成为控制交流侧电压幅值和相位的接口量。最终保留交流电流的k=±1分量和直流电压的k=0分量，并利用实信号共轭对称性，把复相量拆为实部、虚部，形成以两侧a相基波电流实部/虚部及直流总电压为核心状态的低阶实数状态空间方程。

### 3. 验证、优势与不足

作者的验证方式是把动态相量模型的仿真结果与详细时域模型结果进行对比。原文说明测试对象为两端VSC-HVDC系统，两个VSC通过直流电容器及直流线路耦合，并经变压器并联接入两个交流系统；文中摘要和引言强调动态相量仿真结果相当于详细时域模型结果的包络线，可减少仿真时间，并可作为电磁暂态和机电暂态模型接口。当前给出的抽取文本只明确出现额定相电压600 V这一类测试信息；R、L、C、调制比、触发角、仿真步长、误差指标和加速比等关键数值未在证据片段中完整给出。因此，优势应被理解为机制上的和定性验证上的：该模型滤除了开关频率附近的纹波，保留基波包络和直流电压慢动态，使混合仿真可避免逐开关求解。边界也很明确：它不是开关级EMT模型的替代品，不能用于研究器件级开关瞬态、PWM谐波细节、过电压尖峰或高频谐振；若系统处于严重不平衡、故障、控制限幅、强谐波或宽频振荡场景，仅保留k=0和k=±1的截断是否足够，原文片段没有给出验证。原文未报告可核验的数值误差、加速倍数或实时步长结果。

### 4. 价值、认知与可复用场景

这项工作的重要认知在于：对VSC-HVDC这类电力电子主导设备，混合仿真不一定只能在“详细开关模型”和“完全平均模型”之间二选一；动态相量可以把波形的主要傅里叶包络作为状态，使模型既保留基波相位、幅值和直流电压动态，又避开高频开关细节。它适合被后续关于VSC-HVDC等值建模、EMT-机电暂态接口、快速暂态仿真、控制策略低阶验证的页面复用。使用时应把它视为基于特定谐波截断和开关函数平均的模型化方法，而不应外推为适用于任意拓扑、任意故障或任意频段的通用精确模型。

## 方法细节

### 方法概述

本文提出基于时变傅里叶级数的动态相量法对VSC-HVDC系统进行建模。该方法首先建立系统的详细时域开关函数模型，然后通过傅里叶分解将状态变量（交流侧电流、直流侧电压）和开关函数表示为时变傅里叶级数形式。通过保留交流量的基波分量（k=±1）和直流量的直流分量（k=0），忽略高次谐波项（|k|>1），将原时域微分方程转换为低阶的动态相量域方程。利用相量微分特性和乘积特性处理微分和乘积项，最终将复数方程分解为实部和虚部，得到适用于混合仿真的降阶实数状态空间模型。

### 数学公式


**公式1**: $$$X(\tau) = \sum_{k=-\infty}^{\infty} X_k(t) e^{jk\omega_s \tau}$$$

*时变傅里叶级数展开式，将时域周期函数X(τ)表示为一系列时变复系数Xk(t)与旋转因子e^(jkωsτ)的乘积之和，其中ωs=2π/T为基波角频率*


**公式2**: $$$X_k(t) = \frac{1}{T} \int_{t-T}^{t} x(\tau) e^{-jk\omega_s \tau} d\tau = \langle x \rangle_k(t)$$$

*动态相量（k阶傅里叶系数）的定义，通过在滑动时间窗[t-T,t]内的平均运算得到，代表原信号在第k次谐波处的时变包络*


**公式3**: $$$\frac{d \langle x \rangle_k}{dt} = \left\langle \frac{dx}{dt} \right\rangle_k - jk\omega_s \langle x \rangle_k$$$

*相量微分特性，描述时域信号微分的k阶傅里叶系数与相量微分之间的关系，包含一个附加的复频移项-jkωs⟨x⟩k*


**公式4**: $$$\langle xq \rangle_k = \sum_{i=-\infty}^{\infty} \langle x \rangle_{k-i} \langle q \rangle_i$$$

*相量乘积特性（卷积特性），两个时域信号乘积的k阶傅里叶系数等于各自傅里叶系数的离散卷积，用于处理开关函数与电流的乘积项*


**公式5**: $$$d_{1a} = 0.5 + \frac{m_1}{2} \cos(\omega t - \delta_1)$$$

*a相开关函数的基波交流分量（占空比），其中m1为调制比，δ1为触发角，0.5为直流偏置分量*


**公式6**: $$$L_1 \frac{di_{1a}}{dt} = -R_1 i_{1a} - v_{1dc} S_{1a} + \frac{v_{1dc}}{3} \sum_{j=a,b,c} S_{1j} + e_{1a}$$$

*VSC1的a相时域动态方程，包含电感压降、电阻压降、开关函数调制的直流电压项和电网电压e1a*


**公式7**: $$$\frac{d I_{1a1}^r}{dt} = -\frac{R_1}{L_1} I_{1a1}^r + \omega I_{1a1}^i - \frac{m_1 V_{1dc0}}{4L_1} \cos \delta_1 + \frac{E_{1a1}^r}{L_1}$$$

*动态相量模型中VSC1侧a相电流实部方程，包含电阻衰减项、旋转耦合项(ωI^i)、控制输入项和电网电压实部输入*


**公式8**: $$$\frac{d I_{1a1}^i}{dt} = -\frac{R_1}{L_1} I_{1a1}^i - \omega I_{1a1}^r + \frac{m_1 V_{1dc0}}{4L_1} \sin \delta_1 + \frac{E_{1a1}^i}{L_1}$$$

*动态相量模型中VSC1侧a相电流虚部方程，与实部方程通过旋转项-ωI^r耦合，形成标准的旋转向量微分方程形式*


**公式9**: $$$\frac{d(V_{1dc0}+V_{2dc0})}{dt} = \frac{3m_1}{2C_{dc}}(I_{1a1}^r \cos\delta_1 - I_{1a1}^i \sin\delta_1) - \frac{3m_2}{2C_{dc}}(I_{2a1}^r \cos\delta_2 - I_{2a1}^i \sin\delta_2)$$$

*直流侧总电压的动态相量方程（简化形式），描述两侧换流器交流电流与直流电压的功率平衡关系，系数3表示三相贡献*


### 算法步骤

1. 建立VSC-HVDC系统的详细时域开关函数模型：包括交流侧电感L、电阻R，直流侧电容Cdc，以及用开关函数S描述的VSC桥臂，建立三相abc坐标系下的微分方程（式9）

2. 选择状态变量并进行谐波分解：选取交流侧电流i1a,i1b,i1c和直流电压v1dc,v2dc作为状态变量，将其表示为时变傅里叶级数，保留交流量的基波分量（k=±1）和直流量的零次分量（k=0），忽略高次谐波（|k|≥2）

3. 开关函数傅里叶分解：将PWM控制下的开关函数S1a,S1b,S1c分解为直流分量0.5和基波交流分量（式10-12），得到等效占空比d1a,d1b,d1c

4. 应用相量微分特性转换方程：利用式(4)的相量微分特性，将时域微分方程Li'+Ri+...=e转换为相量域方程，处理d⟨i⟩k/dt与⟨di/dt⟩k的关系，引入频移项-jkωs⟨x⟩k

5. 应用相量乘积特性处理非线性项：利用式(6)的卷积特性处理电流与开关函数的乘积项⟨i·d⟩k，将时域乘积转换为相量域的求和运算（如式20所示）

6. 利用实数信号共轭对称性降阶：利用实数信号的傅里叶系数共轭对称性⟨i⟩-k=⟨i⟩k*，将k=-1的方程与k=1的方程合并，减少独立变量数量

7. 分离实部与虚部：将复数相量⟨i⟩1=I^r+jI^i分解为实部和虚部，得到实数状态空间方程（式23），消除复数运算

8. 建立最终状态空间模型：将各方程整理为矩阵形式Ẋ=AX+BU（式24），其中X=[I1a1^r, I1a1^i, I2a1^r, I2a1^i, V1dc0+V2dc0]^T为状态向量，U为输入电压向量

9. 数值求解：采用适当数值积分方法（如 ode45）求解降阶后的状态空间方程，得到系统动态响应的包络线


### 关键参数

- **m1, m2**: 两侧VSC的调制比（0≤m≤1），决定交流侧电压幅值

- **δ1, δ2**: 两侧VSC的触发角（控制角），决定有功功率和无功功率的分配

- **ωs**: 开关角频率，对应PWM载波频率

- **ω**: 系统基波角频率，对应电网频率（如314 rad/s对应50Hz）

- **L1, R1**: VSC1侧变压器和换相电抗的等效电感和电阻

- **Cdc**: 直流侧电容值，影响直流电压动态响应

- **Vdc0**: 直流电压动态相量（直流分量），代表直流侧平均电压

- **I1a1^r, I1a1^i**: VSC1侧a相基波电流的实部和虚部（动态相量）

- 谐波忽略准则：建模过程中忽略傅里叶级数中|k|>1的高次项，仅保留k=0（直流）和k=±1（基波）分量


## 关键公式

### 动态相量通用状态方程

$$$\frac{d\langle x\rangle_k}{dt} = -jk\omega_s\langle x\rangle_k + \langle f(x,u)\rangle_k$$$

*将一般时域状态方程dx/dt=f(x,u)转换为动态相量域的核心公式，适用于所有状态变量的谐波分量建模*

### 相量乘积（卷积）公式

$$$\langle xq\rangle_k = \sum_{i}\langle x\rangle_{k-i}\langle q\rangle_i$$$

*处理电力电子装置中常见的状态变量与开关函数乘积项（如i·S）的关键公式，将时域乘积转换为相量域的离散卷积*

### VSC-HVDC动态相量状态空间模型

$$$\dot{X} = AX + BU$$$

*最终得到的降阶实数状态空间标准形式，其中X=[I_{1a1}^r, I_{1a1}^i, I_{2a1}^r, I_{2a1}^i, V_{1dc0}+V_{2dc0}]^T，用于混合仿真接口*
