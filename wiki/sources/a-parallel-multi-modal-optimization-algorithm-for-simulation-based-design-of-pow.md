# A Parallel Multi-Modal Optimization Algorithm for Simulation-Based Design of Power Systems

**年份**: 2015
**来源**: `03/TPWRD.2015.2410172.pdf.pdf`

## 摘要

提出一种结合电磁暂态(EMT)仿真的并行多模态黑盒优化算法，专为电力系统复杂控制器参数设计。该方法首先在设计空间生成初始采样网格，通过并行EMT仿真评估目标函数值。随后利用邻域比较法自动识别潜在局部极值区域，并在各区域内构建正方形加密网格。采用三次样条插值技术拟合局部代理模型，通过解析求导快速定位极小值点，避免重复调用高耗时仿真器。算法支持多区域完全独立并行计算，大幅压缩设计周期。收敛后生成的显式代理函数可直接用于后优化灵敏度分析，兼顾全局多模态寻优能力与工程实用性。

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求是：电力系统控制器和装置参数越来越依赖EMT仿真评价，例如VSC-HVDC控制参数设计中，目标函数往往只能由PSCAD/EMTDC一类暂态仿真给出，无法写成关于设计变量的显式解析式。研究对象是这类“仿真驱动黑盒目标函数”的多模态优化，即不仅找一个最优点，还要尽量识别多个局部最优点，供工程师在动态性能、参数敏感性和可实施性之间选择。难点在于：一次EMT仿真代价高；非凸目标面可能有多个局部极值；传统单点局部搜索可能错过可用解；全局或多模态搜索若串行执行会拉长设计周期。本文贡献不是提出新的电磁模型，而是把多模态搜索流程设计成天然可并行的仿真调度算法：先用网格采样定位疑似局部最优区域，再在各区域内用样条代理模型估计目标函数并减少重复仿真调用；不同区域、不同采样点可独立提交到并行计算环境。其创新点在于面向电力系统EMT黑盒优化，把多局部最优识别、并行仿真执行和后续灵敏度分析可复用的目标面样本/代理模型放在同一设计平台中。

### 2. 模型、算法与实现技术

本文提出的是并行多模态优化算法及其与EMT仿真器的耦合实现。输入是设计变量范围、初始网格密度、收敛容差和由仿真输出定义的目标函数；输出是若干局部最优解、相应目标函数值，以及局部目标面的样本点和代理函数。流程分为五个阶段：初始化、局部化、目标函数估计、密度递增和收敛检查。初始化阶段在设计空间布置规则采样点，并行调用EMT仿真获得目标函数值。局部化阶段通过比较某网格点与相邻点的函数值，把可能的局部极小区域分离出来。目标函数估计阶段在每个疑似区域构造局部规则网格，并用三次样条插值形成显式代理目标面；代理模型的作用是让算法可通过解析或快速数值方式估计极值位置，而不是在每一次搜索迭代中都调用昂贵EMT仿真。收敛检查用代理预测极值处的目标函数值与该点重新EMT仿真得到的实际目标函数值之差进行判断，抽取页给出的误差形式为 error(i)=OF_ac(i)-OF_st(i)。若未收敛，则执行密度递增：在原网格间插入新点，新增点数随维数按 N_NA=(2n_ex-1)^d-(n_ex)^d 增长。该公式说明算法在高维变量下采样成本会迅速增加，也是其工程适用性的关键约束。

### 3. 验证、优势与不足

作者用两类验证说明算法有效性：一类是数学多峰基准函数，用于检查多局部极值定位能力和并行执行相对串行执行的计算时间；另一类是VSC-HVDC输电系统优化，用于展示算法可嵌入实际电力系统EMT仿真设计。原文摘要明确说进行了与顺序执行版本的计算时间比较，并用VSC-HVDC最优设计展示能力；抽取页还给出Rastrigin二维多峰测试、60处理器并行环境、PSCAD/EMTDC以及VSC-HVDC弱电网算例等细节。指标包括目标函数评估次数、串行/并行计算时间、局部最优识别情况、代理预测值与实际仿真值误差，以及工程目标如直流电压纹波、直流功率暂态和稳态误差。优势主要体现在：采样点和局部区域之间相互独立，适合并行调度；代理模型减少极值附近的重复仿真；多模态结果可让工程师不只依赖单一全局最优，而能选择对参数扰动更稳健的局部解。边界也很明确：验证集中在二维基准函数和一个VSC-HVDC设计例上，不能直接证明算法在高维控制参数、大规模交流网络、不同故障类型或实时仿真平台上同样有效；网格加密在维数升高时存在组合爆炸；若初始网格过粗，间距很近的局部极值可能被漏检。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：在EMT仿真驱动设计中，优化瓶颈不只在搜索算法本身，还在仿真调用如何组织。把黑盒目标函数采样、局部极值分区、代理模型和并行任务调度一起设计，可以把“寻找一个解”扩展为“构建多个可比较设计候选及其局部目标面”。它适合被后续涉及电力电子控制器整定、HVDC/FACTS装置参数优化、暂态响应指标权衡、仿真样本复用和后优化灵敏度分析的页面引用。它不适合被外推为通用全局最优保证算法，也不应在未验证情况下推广到高维、大扰动场景或非规则可行域问题；其效果依赖初始网格、变量维数、仿真成本、并行资源和目标函数平滑性。

## 方法细节

### 方法概述

提出一种结合电磁暂态(EMT)仿真的并行多模态黑盒优化算法，专为电力系统复杂控制器参数设计。该方法首先在设计空间生成初始采样网格，通过并行EMT仿真评估目标函数值。随后利用邻域比较法自动识别潜在局部极值区域，并在各区域内构建正方形加密网格。采用三次样条插值技术拟合局部代理模型，通过解析求导快速定位极小值点，避免重复调用高耗时仿真器。算法支持多区域完全独立并行计算，大幅压缩设计周期。收敛后生成的显式代理函数可直接用于后优化灵敏度分析，兼顾全局多模态寻优能力与工程实用性。

### 数学公式

**公式1**: $$$error(i) = OF_{ac}(i) - OF_{st}(i)$$$

*收敛误差计算公式，用于比较代理模型预测极值与实际EMT仿真值的差异，判断是否满足收敛阈值*

**公式2**: $$$N_{NA} = (n_{ne})^d - (n_{ex})^d = (2n_{ex} - 1)^d - (n_{ex})^d$$$

*网格加密新增点数计算公式，其中d为决策变量维数，n_ex和n_ne分别为加密前后单轴网格点数*

**公式3**: $$$\min f(x_1, x_2) = (x_1^2 + x_1 + x_2^2 + 2.1x_2) + \sum_{i=1}^{2} 10(1 - \cos(2\pi x_i))$$$

*Rastrigin基准测试函数，用于验证算法在不同局部极值数量下的性能与并行加速效果*

**公式4**: $$$f_1(x) = C_{VT} \int_{Transient} (V_{dcref}(t) - V_{dc1}(t))^2 dt + C_{VS} \int_{SteadyState} (V_{dcref}(t) - V_{dc1}(t))^2 dt$$$

*VSC-HVDC直流电压跟踪误差目标函数（含暂态与稳态加权积分平方误差）*

### 算法步骤

1. 初始化与初始网格生成：在设计变量约束范围内均匀散布n个初始试验点，通过并行EMT仿真计算各点目标函数值，构建覆盖全局的初始评估网格。

2. 局部极值区域定位：遍历网格中每个点，将其目标函数值与拓扑直接相邻点进行比较。若某点函数值严格小于所有邻居，则标记为潜在局部极值中心，并划定其所属的独立局部搜索区域。

3. 代理模型构建与极值估计：在每个局部区域内生成规则正方形网格，并行执行EMT仿真获取网格点函数值。采用三次样条插值法拟合该区域的显式代理模型，通过令代理模型梯度为零解析求解其理论极小值点坐标及函数值。

4. 收敛性检验：将代理模型预测的极小值点坐标输入EMT仿真器获取实际目标函数值。计算预测值与实际值的绝对误差，若误差小于预设容差（0.001），则判定该区域收敛；否则进入加密步骤。

5. 网格密度递增：对未收敛区域，在原有正方形网格的每条连线中点插入新采样点，按指数规律增加网格密度。重新并行评估新点，更新三次样条代理模型，并重复极值估计与收敛检验，直至所有局部区域均满足收敛条件。

### 关键参数

- **收敛阈值**: 0.001

- **并行计算节点数**: 60个处理器

- **代理模型类型**: 三次样条插值(Cubic-Spline)

- **目标函数类型**: 黑盒型(依赖EMT仿真输出)

- **网格加密策略**: 连线中点插入法

## 量化发现

- 在500个局部极值的复杂测试中，并行架构实现10.8倍计算加速(23625s降至2768s)
- 收敛判定误差阈值设定为0.001，代理模型预测值与实际EMT仿真值偏差控制在此范围内
- 网格加密后新增点数遵循指数增长规律，对于2维问题初始4点网格加密一次新增(2*4-1)^2 - 4^2 = 33个点
- 算法在极值密集区域(间距极小)存在约1%的漏检率(500极值漏检5个)，但整体多模态寻优成功率>99%
- 三次样条代理模型将后续灵敏度分析的计算成本从耗时EMT仿真降至毫秒级解析求导

## 关键公式

### 收敛误差判据

$$$error(i) = OF_{ac}(i) - OF_{st}(i)$$$

*用于判断代理模型拟合的局部极值是否足够精确，决定是否终止迭代或继续加密网格*

### 网格加密点数公式

$$$N_{NA} = (2n_{ex} - 1)^d - (n_{ex})^d$$$

*在未收敛的局部区域内动态增加采样点密度，提升三次样条插值精度*

### 加权积分平方误差目标函数

$$$f_k(x) = C_{kT} \int_{Transient} e_k^2(t) dt + C_{kS} \int_{SteadyState} e_k^2(t) dt$$$

*VSC-HVDC系统多目标优化核心，量化直流电压、有功功率及交流电压的暂态/稳态跟踪性能*

## 验证详情

- **验证方式**: 数值基准测试对比(串行vs并行)与电力系统EMT仿真验证
