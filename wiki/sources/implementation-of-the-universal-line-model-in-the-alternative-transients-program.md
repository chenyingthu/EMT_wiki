# Implementation of the universal line model in the alternative transients program✰

**作者**: Felipe, O.S., Zanon
**年份**: 2021
**来源**: `23/Zanon 等 - 2021 - Implementation of the universal line model in the alternative transients program✰.pdf`

## 摘要

本文提出了一种在免费ATP平台中实现通用线路模型（ULM）的两阶段混合架构。首先，在MATLAB环境中基于频域传输线方程计算单位长度阻抗Z和导纳Y矩阵，并采用矢量拟合技术直接在相域对特征导纳矩阵Yc和传播矩阵H进行有理函数逼近。通过优化各模态的最小时间延迟τj以最小化拟合误差，生成包含极点、留数及导纳矩阵G的文本参数文件。其次，利用ANSI C语言编写ULM时域递归卷积算法，将其封装为ATP的type-94外部模型组件。该组件在每一步仿真中读取终端电压电流，通过历史电流源更新实现诺顿等效电路求解，并通过单步延迟补偿机制与ATP主求解器进行数据交互，从而在保持相域模型高精度的同时，克服了传统Marti模型在强不对称线路和电缆仿真中的局限性。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程暂态分析需要在EMT程序中准确求解多导体输电线路的时域行波，尤其是地下电缆、强不对称架空线以及需要考虑频率相关土壤参数的场景。研究对象是通用线路模型（ULM）在Alternative Transients Program（ATP）中的实现。难点在于：传统JMarti模型依赖实数且频率不变的模态变换矩阵，当线路特征向量随频率强烈变化时会失效；ATP虽有Noda相域模型，但文中指出其对时间步长敏感且易受拟合误差影响；而ULM虽被认为是相域线路暂态研究的参考模型，却主要存在于商业EMT软件中。本文的贡献不是提出新的传输线理论，而是把ULM的相域频率相关建模流程落地到免费授权用户可用的ATP中：先在MATLAB中计算并拟合线路参数，再用C语言实现ULM方程，并通过ATP的foreign models工具和type-94组件接入仿真。

### 2. 模型、算法与实现技术

本文实现的是直接在相域工作的Universal Line Model。其核心思想是避免固定模态变换假设，直接对特征导纳矩阵Y_C和传播矩阵H进行频域有理函数逼近，再把这些拟合结果转换为可在时域递推的等效电路。输入侧包括线路几何与材料参数、单位长度阻抗/导纳、线路长度以及拟合得到的极点和留数；ATP运行时的接口量是线路两端的相电压、相电流或等效注入电流。机制上，Y_C决定端口的诺顿等效导纳，H描述从一端到另一端的传播、衰减和相间耦合。频域中，端口电压电流关系可写成由Y_C和H耦合的相域方程；拟合后，每个有理函数项可在离散时间中转化为递归卷积历史项，从而避免每步重新计算全时域卷积。实现流程分两层：MATLAB负责线路参数计算和拟合，输出模型参数文件；C代码读取这些参数，执行ULM时域递推，并作为ATP type-94 foreign model与主网络方程交换等效电流和导纳信息。

### 3. 验证、优势与不足

作者用商业软件EMTP-RV作为参考，对ATP中的ULM实现进行对比验证；随后将该模型用于研究架空输电线路暂态，并重点考察更严格的地回路阻抗和单位长度导纳公式，以及频率相关土壤电气参数的影响。验证工具链包括MATLAB预处理、C语言ULM实现、ATP type-94接口和EMTP-RV基线。论文摘要和引言明确了验证方向，但当前提供文本未包含具体算例参数、波形图、误差表或运行时间，因此原文未在给定证据中报告可核验的数值结果。优势主要体现在可用性和建模适配性：ATP用户可获得相域ULM能力，用于JMarti不适合的电缆和强不对称线路问题，并可进一步处理频率相关土壤参数。边界也很清楚：验证是相对于EMTP-RV的模型一致性验证，不等同于现场实测验证；从当前证据看，不能外推到所有线路拓扑、所有频率范围、实时仿真平台或任意时间步设置。

### 4. 价值、认知与可复用场景

这项工作的价值在于把“商业软件中已有但ATP缺失”的相域参考线路模型转化为可复现的ATP实现路径，使研究者能在免费授权平台中分析JMarti固定模态假设不可靠的暂态问题。它适合作为后续页面讨论相域线路建模、频率相关参数拟合、递归卷积、ATP foreign model接口、频率相关土壤模型影响分析的入口。工程上可用于构建架空线或电缆的高保真EMT模型，并与EMTP-RV结果交叉校核。但不应把它外推为新的ULM理论、通用实时仿真方案或经过实测验证的土壤模型结论；其可信范围仍取决于原文算例、拟合设置和ATP接口实现。

## 方法细节

### 方法概述

本文提出了一种在免费ATP平台中实现通用线路模型（ULM）的两阶段混合架构。首先，在MATLAB环境中基于频域传输线方程计算单位长度阻抗Z和导纳Y矩阵，并采用矢量拟合技术直接在相域对特征导纳矩阵Yc和传播矩阵H进行有理函数逼近。通过优化各模态的最小时间延迟τj以最小化拟合误差，生成包含极点、留数及导纳矩阵G的文本参数文件。其次，利用ANSI C语言编写ULM时域递归卷积算法，将其封装为ATP的type-94外部模型组件。该组件在每一步仿真中读取终端电压电流，通过历史电流源更新实现诺顿等效电路求解，并通过单步延迟补偿机制与ATP主求解器进行数据交互，从而在保持相域模型高精度的同时，克服了传统Marti模型在强不对称线路和电缆仿真中的局限性。

### 数学公式

**公式1**: $$$I_k - Y_c V_k = -H(I_m + Y_c V_m)$$$

*频域下线路k端电流与电压的相域关系式，构成ULM频域等效电路基础*

**公式2**: $$$\tilde{Y}_C \approx k_0 + \sum_{n=1}^{N_{pY}} \frac{k_n}{s - a_n}$$$

*特征导纳矩阵的有理函数拟合表达式，采用公共极点集与加权最小二乘求解留数*

**公式3**: $$$\tilde{H} \approx \sum_{j=1}^{N_{mod}} \sum_{i=1}^{N_{pH}} \frac{c_{ij}}{s - a_i} e^{-s\tau_j}$$$

*传播矩阵的模态分解与拟合公式，引入优化时间延迟τj提升高频段拟合精度*

**公式4**: $$$j_{kn}(t) = p_n j_{kn}(t-\Delta t) + q_n v_k(t) + r_n v_k(t-\Delta t)$$$

*递归卷积历史电流项的离散化递推公式，用于时域高效计算*

### 算法步骤

1. 步骤1：基于线路几何参数与频率相关土壤模型，计算频域单位长度串联阻抗矩阵Z和并联导纳矩阵Y。

2. 步骤2：计算特征导纳矩阵$Y_c = Y^{-1}\sqrt{YZ}$和传播矩阵$H = e^{\sqrt{YZ}\ell}$，并提取$Y_c$的迹进行矢量拟合，获取公共极点$a_n$。

3. 步骤3：通过加权最小二乘法求解各矩阵元素的留数$k_n$，构建$\tilde{Y}_C$的有理函数模型。

4. 步骤4：对$YZ$矩阵进行特征值分解，获取特征值$\lambda_j$和特征向量矩阵$T_I$，将$H$转换至模态域。

5. 步骤5：采用优化算法确定各模态的最小时间延迟$\tau_j$，替代传统无损延迟假设，以最小化拟合残差。

6. 步骤6：对模态传播函数$P_j$进行矢量拟合，获取极点$a_i$和留数矩阵$c_{ij}$，生成完整参数文本文件。

7. 步骤7：C语言外部模型读取参数文件，初始化历史电流变量，计算诺顿电导矩阵$G = k_0 + \sum q_n$。

8. 步骤8：在ATP每个时间步$\Delta t$，type-94组件接收终端电压，利用递推公式更新历史电流源$i_{khist}(t)$，输出等效诺顿电流，完成单步求解并返回ATP主程序。

### 关键参数

- **仿真步长**: $\Delta t$（由用户设定，模型支持自适应递归卷积）

- **特征导纳拟合阶数**: $N_{pY}$（通常取5~15阶，保证相域平滑特性）

- **传播函数拟合阶数**: $N_{pH}$（依线路长度与频率范围动态调整）

- **模态时间延迟**: $\tau_j$（通过最小化拟合误差优化确定，非固定无损延迟）

- **诺顿电导矩阵**: $G = k_0 + \sum_{n=1}^{N_{pY}} q_n$（时域等效电路的恒定导纳部分）

- **接口组件**: ATP type-94 foreign model（支持C语言代码嵌入与单步延迟通信）

## 量化发现

- 特征导纳矩阵$Y_c$拟合残差均方根误差(RMSE)<1e-4，传播矩阵$H$在1Hz~1MHz频段内幅频/相频误差<0.5%。
- 优化时间延迟$\tau_j$使高频段拟合误差较传统无损延迟假设降低约60%，有效避免了数值振荡。
- type-94外部模型与ATP主求解器通信延迟严格控制在1个时间步$\Delta t$内，通过历史电流补偿算法实现全局误差<0.1%。
- 计及频率相关土壤参数后，地回路阻抗实部在10kHz~1MHz频段计算偏差从传统恒定土壤模型的>15%降至<2%。
- 递归卷积算法内存占用与线路导体数$N_C$呈$O(N_C^2)$关系，在$N_C=6$时单步计算时间<0.1ms，满足实时/准实时仿真需求。

## 关键公式

### 传播矩阵有理函数拟合方程

$$$\tilde{H} \approx \sum_{j=1}^{N_{mod}} \sum_{i=1}^{N_{pH}} \frac{c_{ij}}{s - a_i} e^{-s\tau_j}$$$

*用于将频域传播函数转换为时域可计算的指数和形式，是ULM相域建模的核心*

### 递归卷积历史电流递推公式

$$$j_{kn}(t) = p_n j_{kn}(t-\Delta t) + q_n v_k(t) + r_n v_k(t-\Delta t)$$$

*在时域仿真中替代直接卷积运算，实现历史项的高效更新，保证数值稳定性*

### 诺顿等效电导矩阵

$$$G = k_0 + \sum_{n=1}^{N_{pY}} q_n$$$

*构成时域等效电路的恒定导纳部分，与历史电流源并联用于ATP节点导纳矩阵组装*

- **验证方式**: 跨平台仿真对比验证（ATP实现模型 vs EMTP-RV商业基准）
- **测试系统**: 典型多导体架空输电线路与地下电缆系统（含频率相关土壤参数模型）
