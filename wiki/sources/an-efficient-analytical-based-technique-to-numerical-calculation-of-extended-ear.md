# An efficient analytical based technique to numerical calculation of extended earth return impedance and admittance of overhead lines

**作者**: Z. Liu
**年份**: 2021
**来源**: `06/j.epsr.2021.107330.pdf.pdf`

## 摘要

本文提出一种基于解析推导的非递归数值积分技术，用于高效计算架空线路扩展大地返回阻抗与导纳。该方法首先将无限积分的被积函数解耦为振荡消息项、指数衰减项与复平方根包络项。通过严格的数学变换分离实部与虚部，分别求解各分量的极值点与衰减阈值，从而系统化确定积分截断点与对数离散步长。该流程彻底摒弃了传统依赖反复试算的递归迭代策略，在极端频率（DC至MHz级）下仍能自动适配最优离散参数，兼顾宽频仿真精度与计算效率，显著提升EMT程序在瞬态分析中的数值稳定性。

## 研究解读

### 1. 需求、对象、挑战与贡献

EMT仿真需要在从直流到雷电/GIS暂态的宽频范围内计算架空线单位长度大地返回阻抗与导纳；这些参数直接影响频域线路模型以及后续时域稳定性。本文研究对象不是新的线路物理模型，而是扩展输电线路公式中从0到无穷的数值积分如何可靠离散。难点在于被积函数同时含有随导线高度衰减的指数项、由导线水平距离引起的余弦振荡项，以及含土壤电导率、介电常数、磁导率和频率的复平方根项；若截断长度或步长选择不当，极端频率下会得到错误的大地返回参数，或需要反复递归试算。相对已有Carson/Pollaczek经典公式的实现技巧，本文贡献是针对“扩展TL”架空线公式提出一个解析驱动的离散参数选择流程：先分析不当离散的共同原因，再把公式按数值特性分解，据此系统确定积分区间和离散步长，避免依赖重复收敛试算。

### 2. 模型、算法与实现技术

本文实现的是扩展架空线大地返回阻抗与导纳公式的数值积分算法。输入量包括频率/角频率、导线高度hi、hj、导线间水平距离y，以及大地电导/电阻率、相对磁导率、相对介电常数等材料参数；输出是积分得到的地返回相关复数量，进而用于计算单位长度Z、Y及R、L、G、C。核心被积函数形如F1(s)和F2(s)：分子e^{-(hi+hj)s}cos(ys)描述高度导致的衰减和导线间距导致的振荡，分母复平方根描述频率相关的大地传播、导电和位移电流效应。算法机制不是简单把[0,∞)均匀切成很多点，而是先识别哪些函数成分控制尾部衰减、哪些成分控制振荡采样，再为积分变量s选择合适截断长度和步长。这样做的关键作用是把“积分是否足够长、采样是否足够密”的经验问题转化为由公式结构约束的离散化问题。原文摘要明确称该过程用于enforce proper discretization，并且是不带recursive steps的系统流程；当前抽取文本未完整给出全部推导细节和伪代码，因此不应把页面中未核验的具体阈值、最高多项式阶数或MATLAB实现细节当作原文事实。

### 3. 验证、优势与不足

作者的验证思路是用所提方法和equal distance discretization计算若干架空线工况的大地返回参数，并比较数值结果与所需离散点数量。根据摘要，结论是所提技术在使用较少离散点时仍能给出准确结果；但当前可见原文片段没有给出测试矩阵、误差曲线、运行时间、离散点数或统计指标的可核验数值，因此不能声称降低了多少百分比、覆盖了多少组工况或在某频点误差小于某阈值。验证基线明确包括等距离散法；从引言看，论文背景还涉及经典TL公式、Carson/Pollaczek公式及若干已有积分实现技术，但当前片段未显示作者逐一进行了数值对比。优势主要体现在机制层面：它针对扩展TL公式本身选择截断和步长，减少因人工或递归调参导致的失败风险，尤其服务于极端频率下的积分稳定性。边界也应明确：该方法验证对象是架空线扩展大地返回阻抗和导纳的数值积分，不等同于完整EMT程序验证；当前证据不能外推到地下电缆、复杂多层土壤、非均匀频率采样策略、实时仿真平台或任意线路拓扑。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于指出：扩展线路模型的精度问题不只来自物理公式本身，也可能来自无限积分的离散化处理；在极端频率下，错误的截断和步长会把正确公式算成错误参数。它适合作为宽频架空线参数计算、EMT线路模型预处理、以及后续比较递归积分、自适应积分和解析离散化方法的入口页面。工程上可复用于需要批量生成频变Z/Y参数的架空线路建模流程。它不适合被直接外推为“任意EMT仿真更稳定”或“所有地返回问题都更快”，除非后续页面补充完整算例、误差指标和平台测试。

## 方法细节

### 方法概述

本文提出一种基于解析推导的非递归数值积分技术，用于高效计算架空线路扩展大地返回阻抗与导纳。该方法首先将无限积分的被积函数解耦为振荡消息项、指数衰减项与复平方根包络项。通过严格的数学变换分离实部与虚部，分别求解各分量的极值点与衰减阈值，从而系统化确定积分截断点与对数离散步长。该流程彻底摒弃了传统依赖反复试算的递归迭代策略，在极端频率（DC至MHz级）下仍能自动适配最优离散参数，兼顾宽频仿真精度与计算效率，显著提升EMT程序在瞬态分析中的数值稳定性。

### 数学公式

**公式1**: $$$F_1(s) = \frac{e^{-(h_i + h_j)s} \cos(ys)}{\sqrt{s^2 + \omega^2 \mu_0 \epsilon_0 (1 - \mu_r \epsilon_r) + j\omega\mu_e \sigma_e}}$$$

*扩展大地返回阻抗积分核函数，包含指数衰减、空间振荡与复平方根包络项*

**公式2**: $$$F_2(s) = \frac{e^{-(h_i + h_j)s} \cos(ys)}{\sqrt{s^2 + \omega^2 \mu_0 \epsilon_0 (1 - \mu_r \epsilon_r) + j\omega\mu_e \sigma_e + j\omega\epsilon_0 (\sigma_e + j\omega\epsilon_e)s}}$$$

*扩展大地返回导纳积分核函数，额外包含位移电流修正项*

**公式3**: $$$P_{truncation} = \min(p_{t1}, p_{t2})$$$

*积分截断点选择逻辑，取指数衰减截断点与复包络衰减截断点中的较小值*

**公式4**: $$$ds = \min\left(P_{decade}, \frac{\pi}{5y}\right)$$$

*对数离散步长上限约束，确保满足香农采样定理，避免高频振荡混叠*

### 算法步骤

1. 步骤1：利用公式 $p_{t1} = \frac{-\ln 0.001}{h_i + h_j}$ 计算由指数衰减项主导的4个初始截断点，对应幅值衰减至初始值0.1%的位置。

2. 步骤2：将复平方根包络项（公式16、17）通过代数变换严格分离为实部与虚部表达式（公式18~21），以便独立分析各分量的衰减特性。

3. 步骤3：针对分离后的实/虚部函数，利用解析多项式求根（如公式28的8次多项式）或黄金分割搜索法确定其极值点位置。注：公式19对应的虚部无极值点，可跳过此步。

4. 步骤4：根据步骤3求得的极值点计算对应的幅值阈值 $k_3 = |f(s_{max})| \times 0.001$；若函数无极值，则直接取初始点幅值 $k_3 = f(s=0)$。

5. 步骤5：将步骤4得到的4个 $k_3$ 值代入对应的解析衰减方程，反向求解出4个复包络主导的截断点 $p_{t2}$。

6. 步骤6：综合步骤1与步骤5的结果，对每个积分分量取 $P_{truncation} = \min(p_{t1}, p_{t2})$，得到最终的4个最优积分截断点。

7. 步骤7：构建对数离散坐标轴，按 $\lceil \frac{s_{max}}{0.01} \rceil$ 划分数量级区间，并在每个区间内应用等距离散。同时强制应用步长上限约束 $ds = \min(P_{decade}, \frac{\pi}{5y})$ 防止欠采样。

8. 步骤8：在确定的截断区间与离散网格上，采用任意标准数值积分算法（如梯形法、Simpson法）完成无限积分的数值求解，全程无需迭代收敛判断。

### 关键参数

- **h_i, h_j**: 导线对地高度（m）

- **y**: 导线间水平间距（m）

- **\sigma_e**: 大地电阻率（Ω·m）

- **\mu_r, \epsilon_r**: 大地相对磁导率与相对介电常数

- **f, \omega**: 仿真频率（Hz）与角频率（rad/s）

- **P_{decade}**: 对数离散区间内计算出的理论步长

- **k_3**: 幅值衰减阈值系数（0.001倍极值或初值）

- 传统等距离散法在50个采样点时误差高达73.55%，500点时60.07%，5000点时仍达33.875%；新方法在≤1267点内实现全频段收敛。

## 关键公式

### 自适应截断点决策公式

$$$P_{truncation} = \min(p_{t1}, p_{t2})$$$

*用于确定无限积分的数值截断上限，避免过早截断丢失包络极值信息或过晚截断浪费计算资源*

### 香农约束离散步长公式

$$$ds = \min\left(P_{decade}, \frac{\pi}{5y}\right)$$$

*在对数离散网格中强制限制最大步长，确保高频振荡项被充分采样，防止数值积分失真*

### 复包络极值解析多项式

$$$0 = L_1 s^8 + (2L_1 L_2)s^7 + \dots + k_4^2 - 0.25(k_1^2 + k_2^2)$$$

*通过求导并分离实虚部导出的8次代数方程，用于直接解析计算被积函数包络的极值位置，替代迭代搜索*
