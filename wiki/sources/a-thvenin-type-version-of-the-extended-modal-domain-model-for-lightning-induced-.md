# A Thévenin-Type Version of the Extended Modal-Domain Model for Lightning-Induced Voltage Calculation

**作者**: Osis E. S. Leal, Alberto De Conti
**年份**: 2023
**来源**: `04/Leal和Conti - 2023 - A Thévenin-Type Version of the Extended Modal-Domain Model for Lightning-Induced Voltage Calculation.pdf`

## 摘要

本文提出一种用于扩展模态域（EMD）模型的新电流源计算流程，用来评估架空线路雷击感应电压。原始EMD流程需要拟合线路特征导纳$Y_c$，而主流EMT程序中Marti线路模型通常直接提供的是特征阻抗$Z_c$的拟合参数。该文把外部电磁场效应的端部电流源改写为只依赖$Z_c$拟合结果的形式，使用户可以复用ATP、EMTP、PSCAD等程序内置的线路拟合工具，并通过不同负载、变压器、避雷器和多点接地算例验证。

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求是：在ATP、EMTP、PSCAD等EMT程序中，对架空配电线路进行雷击感应电压计算时，希望直接调用已有频率相关线路模型和设备库，而不是为外部电磁场激励另写一套线路参数拟合代码。研究对象是扩展模态域模型（EMD）中用于等效入射雷电电磁场作用的线路端部电流源j0(t)、jL(t)。难点在于，原始EMD电流源计算依赖特征导纳Yc及传播函数的有理函数拟合；但工程上常用的Marti频率相关线路模型，其内置拟合工具通常输出的是特征阻抗Zc=Yc^{-1}的极点、留数和常数项，而不是Yc的拟合参数。若继续使用原流程，用户需要在EMT程序外计算并拟合Yc，增加实现复杂度，也容易造成线路模型和感应源模型参数不一致。本文的贡献不是提出新的雷电通道模型或新的线路物理模型，而是把EMD外部激励电流源的计算改写为仅依赖Zc拟合结果的Thévenin/Norton等效流程，使Marti模型内置拟合参数可直接用于雷击感应电压仿真。

### 2. 模型、算法与实现技术

本文提出的是EMD模型的Thévenin型实现流程。核心接口量包括：EMT软件对Marti线路模型得到的模态特征阻抗Zc拟合参数，即高频常数项、极点和留数；由入射雷电电磁场计算得到的线路两端等效激励电压；以及最终注入线路两端节点的相域电流源j0(t)、jL(t)。机制上，作者先把模态特征阻抗写成冲激项加指数项的时域卷积核，再用递归卷积把频率相关阻抗作用转化为每个时间步更新的历史项。无外部激励时，端口电压降由等效电阻项和内部历史项组成；引入外部电磁场后，端口方程中还必须包含由感应源引起的额外历史项。为避免直接使用Yc，论文通过Thévenin到Norton的变换，用Zc对应的等效电阻矩阵及其逆矩阵计算当前步模态电流源，并在公式中扣除上一时间步累计的外部源历史项。这样，感应电压源并不是简单除以一个常数阻抗，而是在递归卷积框架内与频率相关Zc一致地耦合。最后，模态电流源通过实常数模态变换矩阵转换回相域，作为独立电流源接入Marti线路两端节点，与EMT程序中的负载、变压器、避雷器和接地元件共同求解。

### 3. 验证、优势与不足

作者用对比仿真验证该流程。验证对象包括一条3 km三相中压架空配电线路，在匹配负载和不平衡负载下计算雷击感应电压；以及一条1.2 km单相农村配电线路，包含多点接地中性线、10 kVA宽频变压器模型和氧化锌避雷器。仿真平台为ATP，涉及EMD-ATP自定义实现、ATP内置Marti模型拟合工具以及ULM-ATP/扩展相域EPD类参考模型。基线包括原始依赖Yc拟合的EMD方法和扩展相域方法，比较指标主要是端口或设备处的感应电压波形、峰值及反射过程的一致性。原文摘要和算例说明表明，新流程在这些工况下与基线波形吻合，且可处理不同负载、变压器、避雷器和多点接地。其优势主要体现在实现接口：用户可复用Marti线路模型已有的Zc拟合结果，避免独立计算和拟合Yc，并减少EMD感应源与线路模型参数来源不一致的问题。边界也很清楚：原文未报告系统性的逐点误差表或计算耗时对比，不能据此宣称任意工况下精度可忽略或速度更快；验证集中于架空配电线路，未覆盖地下电缆、强不均匀大地、大规模互联网络或更广泛雷电通道/土壤模型组合。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于说明：雷击感应电压计算中的“外部电磁场等效源”可以与EMT程序中已有的频率相关Marti线路模型在参数层面一致耦合，不必强制以Yc拟合为入口。它解决的是工程实现中很具体的接口障碍：内置工具给Zc，而原始EMD要Yc。后续研究或工程页面若涉及配电线路雷击感应过电压、Marti线路模型二次开发、ATP/EMTP/PSCAD中的外部场等效注入、多导体模域递归卷积实现，都可以复用其Thévenin型电流源计算思想。但它不应被外推为新的雷电场模型、避雷器模型或变压器宽频模型，也不能替代对Zc拟合质量、模态变换选择、地面参数和终端设备模型有效性的检查。

### 证据边界

- 来自原文：论文明确指出原始EMD电流源计算依赖特征导纳Yc拟合，而Marti模型内置拟合工具通常提供特征阻抗Zc的极点和留数；本文贡献是仅用Zc计算j0(t)、jL(t)。
- 来自原文：有效性通过架空配电线路算例展示，包含不同负载、变压器、避雷器和多点接地；工具背景为ATP及Marti/ULM类EMT线路模型。
- 来自已有页面抽取：算例中出现3 km三相中压线路和1.2 km单相农村线路，且报告了避雷器使一次侧峰值约由50 kV降至30 kV、二次侧约由2 kV降至1.45 kV；这些数值应理解为特定算例结果。
- 原文未报告可核验的全局误差统计、逐点误差表或计算耗时对比，因此不能宣称该方法在所有条件下均比原方法更精确或更快。
- 从验证范围看：研究未系统覆盖地下电缆、复杂网络拓扑、强空间非均匀大地、不同雷电回击模型敏感性或极端非线性终端设备。
- 据方法推断：结果可靠性会继承Zc有理函数拟合、模态变换矩阵和入射电磁场计算的误差；内置拟合工具可用并不等于任意频段和任意线路几何下自动可信。

## 方法细节

### 方法概述

本文提出一种基于特征阻抗（$Z_c$）拟合的扩展模态域（EMD）模型改进方法，用于计算架空线路雷击感应电压。传统EMD模型依赖特征导纳（$Y_c$）的有理函数拟合，而主流EMT软件（如ATP、PSCAD）内置的Marti线路模型仅直接提供$Z_c$的极点与留数。新方法通过戴维南-诺顿等效变换，将外部电磁场激励转化为线路两端的电流源，并修正递归卷积算法中的历史项，使其能够直接利用EMT软件内置拟合工具输出的$Z_c$参数。该方法避免了用户自行编写外部代码计算和拟合$Y_c$的繁琐流程，实现了感应电流源计算与Marti线路模型的无缝集成，大幅简化了多导体线路雷击过电压的建模与仿真流程。

### 数学公式

**公式1**: $$$z_{c,m}^{i,i}(t) = k_0^i \delta(t) + \sum_{n=1}^{N_p^i} k_n^i e^{a_n^i t} u(t)$$$

*特征阻抗模态分量的时域有理函数逼近表达式，用于将频域拟合参数转换为时域卷积核*

**公式2**: $$$e_{0,m}(t) = r_c i_{0,m}(t) + e_{h0,m}(t-\Delta t)$$$

*无外部激励时，模态电压降的递归卷积计算公式*

**公式3**: $$$e_{0,m}(t) = r_c (i_{0,m}(t) + \bar{j}_{0,m}(t)) + e_{h0,m}(t-\Delta t) + e_{u0,m}(t-\Delta t)$$$

*引入外部感应电流源后的修正电压降方程，包含线路内部历史项与外部源历史项*

**公式4**: $$$\bar{j}_{0,m}(t) = g_c [\bar{u}_{0,m}(t) - e_{u0,m}(t-\Delta t)]$$$

*修正后的模态感应电流源计算公式，用于在诺顿等效电路中准确补偿外部电磁场激励的历史效应*

**公式5**: $$$j_0(t) = t_I \bar{j}_{0,m}(t)$$$

*通过实常数模态变换矩阵$t_I$将模域电流源转换回相域，以便注入EMT软件*

### 算法步骤

1. 利用EMT软件内置拟合工具（如ATP的Bode渐近法），在0.1 Hz至10 MHz频段内对线路特征阻抗矩阵$Z_c$进行矢量拟合，提取各模态的极点$a_n^i$、留数$k_n^i$及高频渐近常数$k_0^i$。

2. 根据拟合参数计算模态等效电阻矩阵$r_c$及其逆矩阵$g_c = r_c^{-1}$，并计算递归系数$p_n^i = (2+a_n^i \Delta t)/(2-a_n^i \Delta t)$与$q_n^i = k_n^i \Delta t/(2-a_n^i \Delta t)$。

3. 基于Barbosa-Paulino公式计算入射电磁场在频域的效应，经模态变换矩阵$t_I$转换至模域，得到时域激励电压$\bar{u}_{0,m}(t)$与$\bar{u}_{L,m}(t)$。

4. 初始化历史项矩阵$e_{h0,m}$与$e_{u0,m}$为零向量/矩阵。

5. 在每个仿真时间步$\Delta t$，利用递归公式更新线路内部电流历史项$e_{h0,m}$与外部源历史项$e_{u0,m}$，其中外部源历史项需叠加当前步与上一步的感应电流源值。

6. 代入修正公式$\bar{j}_{0,m}(t) = g_c [\bar{u}_{0,m}(t) - e_{u0,m}(t-\Delta t)]$计算当前时刻的模态感应电流源。

7. 通过相模变换$j_0(t) = t_I \bar{j}_{0,m}(t)$将电流源转换回相域，并作为外部独立电流源注入EMT软件中的Marti线路模型两端节点。

8. 重复步骤5-7直至仿真结束，同步求解线路终端非线性负载（如变压器、避雷器）的暂态响应，完成全系统电磁暂态计算。

### 关键参数

- **拟合频段**: 0.1 Hz ~ 10 MHz (20点/十倍频)

- **大地电导率**: 0.002 S/m

- **导体直流电阻**: 0.822 Ω/km

- **导体半径**: 8.5 mm

- **模态变换参考频率$f_0$**: 1 MHz

- **变压器额定容量**: 10 kVA (7967/120-240 V)

- **避雷器参数**: 12 kV级无间隙氧化锌避雷器，10 kA (8/20 μs)残压39.6 kV

- **中性线接地**: 每300 m接地，接地电阻50 Ω

## 量化发现

- 新方法计算结果与原始EMD模型及EPD参考模型在论文图示精度内接近；页面没有抽取到逐点误差表，不应把它解读为任意工况下误差可忽略。
- 避雷器安装使变压器一次侧雷击感应过电压峰值降低40%（从50 kV降至30 kV）。
- 避雷器动作使传递至变压器二次侧的过电压峰值降低约30%（从2 kV降至1.45 kV）。
- 拟合频段覆盖0.1 Hz至10 MHz，页面记录的20点/十倍频采样来自文中算例设置；其他线路和雷电波形仍需检查拟合残差。
- 模态变换矩阵参考频率在1 kHz至500 kHz范围内测试，结果无差异，证实1 MHz参考频率的鲁棒性。

## 适用边界

- 适用于已采用Marti频率相关线路模型、且EMT工具能给出$Z_c$极点和留数的雷击感应电压仿真。
- 方法解决的是外部电磁场端部电流源与线路模型参数拟合的接口问题，不替代雷电通道模型、地面模型、终端设备宽频模型或避雷器非线性模型本身。
- 论文验证集中在架空配电线路；地下电缆、极端不均匀大地、强非线性终端或大规模互联网络需重新验证。
- 若$Z_c$拟合质量不足，递归卷积电流源会继承该误差；内置拟合工具可用不等于结果自动可信。

## 关键公式

### 修正的模态感应电流源计算公式

$$$\bar{j}_{0,m}(t) = g_c [\bar{u}_{0,m}(t) - e_{u0,m}(t-\Delta t)]$$$

*用于在诺顿等效电路中准确补偿外部电磁场激励的历史效应，解决传统方法中直接叠加电流源导致递归卷积失真的问题*

### 含外源激励的戴维南型电压降递归方程

$$$e_{0,m}(t) = r_c (i_{0,m}(t) + \bar{j}_{0,m}(t)) + e_{h0,m}(t-\Delta t) + e_{u0,m}(t-\Delta t)$$$

*推导电流源修正项的核心方程，将外部场激励与线路内部状态解耦，实现与Marti线路模型的直接兼容*

## 验证详情

- **验证方式**: 对比仿真验证（与原始EMD模型、扩展相域EPD模型进行波形对比）
- **测试系统**: 3-km三相中压配电线路（匹配/不平衡负载）；1.2-km单相农村配电线路（含多点接地中性线、10kVA宽频变压器模型及氧化锌避雷器）
- **仿真工具**: ATP (EMD-ATP自定义代码、内置Marti模型拟合工具、ULM-ATP)
- **验证结果**: 在包含复杂终端负载、多点接地及非线性避雷器动作的工况下，新模型计算的感应电压波形与EPD及原始EMD模型结果高度接近。其主要价值是把EMD电流源计算改写到$Z_c$拟合参数上，减少用户外部编写和维护$Y_c$拟合代码的工作量；工程精度仍取决于线路、雷电场和终端元件模型质量。
