# Fast Voltage-Balancing Control and Fast Numerical Simulation Model for the Modular Multilevel Converter

**作者**: Feng Yu, Weixing Lin, Xitian Wang, Da Xie
**年份**: 2014
**来源**: `19、20、21/EMT_task_19/tpwrd.2014.2351397.pdf-1.pdf`

## 摘要

提出一种基于平均比较的快速电压均衡控制策略与无需修改底层求解器源码的快速数值仿真模型。控制策略摒弃传统全量排序算法，在每个控制周期计算桥臂内所有子模块(SM)电容电压的算术平均值，将各SM电压与平均值直接比较以确定投切状态，大幅降低控制器计算负担并优先保留历史开关状态。引入两个调节变量控制等效开关频率。仿真模型在PSCAD/EMTDC中构建，采用梯形积分法离散更新电容电压，利用线性化开关模型（阈值电压+导通电阻）等效IGBT/二极管伏安特性，将桥臂输出等效为受控电压源，并串联开关与晶闸管以模拟直流故障闭锁工况。进一步开发混合仿真模型，将特定关注SM替换为详细开关模型，其余SM保持快速数值模型，兼顾全局计算效率与局部开关动态细节。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自MMC-HVDC的EMT仿真与实时/准实时控制：一个桥臂常串联数百个半桥子模块，Trans Bay Cable示例为每臂216个SM，201电平MMC的5 s暂态详细开关仿真可耗时约一周。研究对象是MMC桥臂内SM电容电压均衡控制，以及在PSCAD/EMTDC中可运行的快速数值模型。难点有两类：一是详细开关模型包含大量IGBT/二极管开关事件，导致网络矩阵频繁变化、仿真极慢；二是传统电容电压排序均衡每个控制周期要对大量SM排序，控制器计算负担高。本文贡献不是简单平均值模型，而是在保留每个SM电容动态的前提下，用平均比较替代全量排序，并用受控电压源等效桥臂输出；同时不修改PSCAD/EMTDC底层求解器源码，另给出可把少数关注SM替换为详细开关模型的混合仿真方案。

### 2. 模型、算法与实现技术

控制算法在每个控制周期读取某桥臂全部SM电容电压，计算算术平均值，再将每个SM电压与平均值比较，并结合桥臂电流方向和上层最近电平控制给出的投入数量决定SM保持、投入或旁路。其机制是：充电时优先投入低于平均值的电容，放电时优先投入高于平均值的电容，从而把电压偏差拉回平均附近；同时尽量保留上一周期开关状态，并通过两个调节变量设置滞环/判据以调节等效开关频率。快速仿真模型以SM电容电压为核心状态量，以桥臂电流、SM开关状态和器件导通参数为输入，输出每个SM电压及桥臂总电压。电容电压用梯形积分更新，体现电容由桥臂电流充放电的动态；IGBT/二极管导通用阈值电压加导通电阻的线性伏安关系近似；每个SM输出电压由开关状态选通的电容电压和导通压降组成，桥臂电压为所有SM输出求和并通过受控电压源接入PSCAD/EMTDC网络。为模拟直流故障闭锁，模型还加入串联开关和晶闸管通路；混合模型则只把指定SM替换为详细开关电路，其余仍用数值模型。

### 3. 验证、优势与不足

作者在PSCAD/EMTDC中做纯数字仿真验证，基线是官方/传统详细开关模型，比较对象包括所提快速数值模型和混合模型。测试覆盖正常运行与直流侧故障闭锁等工况，系统为典型MMC-HVDC配置，论文背景给出每臂数百SM、如201电平或216个SM的规模。验证指标主要是波形一致性、SM电容电压均衡效果、故障暂态响应和仿真耗时；原文称所提模型与详细开关模型结果几乎相同，仿真速度约为详细开关模型的5000倍，并指出201电平详细模型5 s暂态通常约一周，而快速模型可到分钟级。优势在于同时解决控制排序负担和EMT仿真规模问题，并且用户无需修改PSCAD/EMTDC求解器源码。局限也很明确：论文未给出可复核的逐点误差表或系统性误差统计，页面证据不能支持自行声称某个百分比精度；快速模型保留的是SM电容和桥臂等效电压动态，不等同于完整器件开关损耗、热模型、EMI、驱动延迟或所有二极管反向恢复细节。混合模型只能观察被详细化的SM，不能代表全桥臂所有器件应力分布。

### 4. 价值、认知与可复用场景

这项工作的关键认知是：MMC仿真加速不必完全牺牲SM层级动态，也不必依赖不可访问的求解器源码接口；只要把大量开关事件转化为电容状态更新和桥臂受控电压源，同时用平均比较取代排序，就能在系统暂态研究中保留最重要的电容均衡与桥臂电压行为。它适合被后续MMC-HVDC控制策略评估、直流故障闭锁暂态研究、大规模电平数参数扫描、以及“全局快速模型+局部详细SM”的混合建模页面复用。不应外推为器件级设计模型，也不应据此直接评估开关损耗、热应力、EMI或所有SM的微观故障传播细节。

### 证据边界

- 原文明确给出：平均比较电压均衡控制、两个调节变量用于调整等效开关频率、快速模型约比详细开关模型快5000倍。
- 原文明确给出：201电平MMC详细开关模型仿真5 s动态通常约需一周；Trans Bay Cable示例为每臂216个SM。
- 原文明确给出：模型在PSCAD/EMTDC中实现，目标之一是不修改PSCAD/EMTDC底层源码；混合模型用于嵌入特定关注SM的详细开关动态。
- 关于充电时投入低电压SM、放电时投入高电压SM的描述是依据MMC电容均衡机理和平均比较方法的机制性解释；具体判据符号和两个变量名称在给定抽取文本中不可见。

## 方法细节

### 方法概述

提出一种基于平均比较的快速电压均衡控制策略与无需修改底层求解器源码的快速数值仿真模型。控制策略摒弃传统全量排序算法，在每个控制周期计算桥臂内所有子模块(SM)电容电压的算术平均值，将各SM电压与平均值直接比较以确定投切状态，大幅降低控制器计算负担并优先保留历史开关状态。引入两个调节变量控制等效开关频率。仿真模型在PSCAD/EMTDC中构建，采用梯形积分法离散更新电容电压，利用线性化开关模型（阈值电压+导通电阻）等效IGBT/二极管伏安特性，将桥臂输出等效为受控电压源，并串联开关与晶闸管以模拟直流故障闭锁工况。进一步开发混合仿真模型，将特定关注SM替换为详细开关模型，其余SM保持快速数值模型，兼顾全局计算效率与局部开关动态细节。

### 数学公式

**公式1**: $$$v_{sw} = V_{th} + R_{on} i_{sw}$$$

*功率电子开关（IGBT或二极管）线性化模型，将非线性导通特性近似为阈值电压与导通电阻的串联，用于快速数值计算。*

**公式2**: $$$v_c(t) = v_c(t-\Delta t) + \frac{\Delta t}{2C}[i_c(t) + i_c(t-\Delta t)]$$$

*基于梯形积分法的子模块直流电容电压离散更新公式，保证数值稳定性与二阶精度。*

**公式3**: $$$v_{sm,i} = S_i v_{c,i} + v_{sw,i}$$$

*第i个子模块输出电压计算式，由开关状态决定的电容电压分量与导通压降分量叠加而成。*

**公式4**: $$$v_{arm} = \sum_{i=1}^{N} v_{sm,i}$$$

*桥臂总输出电压为所有串联子模块输出电压之和，用于构建受控电压源接口。*

### 算法步骤

1. 在每个控制周期开始时，实时采集当前桥臂内全部N个子模块的电容电压值$v_{c,i}$。

2. 计算该桥臂所有子模块电容电压的算术平均值：$v_{avg} = \frac{1}{N}\sum_{i=1}^{N} v_{c,i}$。

3. 根据上层调制指令（如最近电平调制NLM）确定当前桥臂需投入的子模块目标数量$N_{on}$。

4. 将每个子模块的电容电压$v_{c,i}$与$v_{avg}$进行逐一对比，结合当前桥臂电流方向（充电或放电）判断电压偏差极性。

5. 引入两个预设调节变量作为比较阈值/滞环带，仅当$v_{c,i}$偏离$v_{avg}$超出阈值时才触发状态翻转，优先保持上一周期的开关状态以抑制等效开关频率。

6. 根据比较结果与$N_{on}$约束，直接生成各子模块的开关信号$S_i$，无需执行$O(N\log N)$复杂度的全量排序操作，输出控制脉冲。

### 关键参数

- **仿真步长**: $\Delta t$（由EMT求解器设定）

- **子模块电容**: $C$（决定电压波动与积分时间常数）

- **导通电阻**: $R_{on}$（IGBT/二极管线性化等效参数）

- **阈值电压**: $V_{th}$（IGBT/二极管线性化等效参数）

- **关断电阻**: $R_{off}$（通常设为$10^6 \Omega$量级，模拟开路状态）

- **单桥臂SM数量**: $N$（典型HVDC应用为数百个，如216个）

- **频率调节变量**: 文中未明确符号，用于设定平均比较的滞环宽度，控制等效开关频率

## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 正常稳态运行工况 | 各子模块电容电压在额定值附近波动，桥臂电流与输出电压波形平滑。 | 波形与PSCAD/EMTDC内置详细开关模型接近，页面未抽取完整误差表。 |

| 直流侧故障闭锁工况 | 故障触发后所有IGBT闭锁，并联晶闸管持续导通，MMC等效为三相二极管整流桥，故障电流上升率与续流路径动态被准确复现。 | 故障暂态过程（电流峰值、恢复时间）与详细模型一致，验证了闭锁逻辑与晶闸管等效电路的有效性。 |

| 混合模型局部动态观测 | 将特定编号子模块替换为详细开关模型后，可观察该SM内部IGBT开关瞬态、二极管反向恢复及电容电压阶跃变化，其余SM保持快速数值计算。 | 局部开关动态由详细模型承担，整体耗时仍远低于全详细模型；具体加速比取决于替换的子模块数量。 |

## 量化发现

- 所提快速数值模型仿真速度较传统详细开关模型提升约5000倍。
- 传统201电平MMC详细模型仿真5秒电磁暂态过程通常需耗时约1周，所提模型将其缩短至分钟级。
- 电压均衡控制算法计算复杂度由传统排序法的$O(N\log N)$降至$O(N)$，控制器计算负担显著降低。
- 仿真波形与详细开关模型对比在文中图示中接近；页面没有逐点误差表，不应自行给出<0.1%等未抽取指标。
- 混合模型在保留全局快速计算的同时，可对指定子模块使用详细开关模型观察微观动态。

## 适用边界

- 适用于数百子模块 MMC-HVDC 的 EMT 级控制和系统暂态研究，尤其是全详细开关模型过慢时。
- 快速模型主要保留子模块电容动态和桥臂等效电压，不等同于所有器件级开关损耗、热行为、EMI或驱动细节。
- 平均比较均衡控制依赖桥臂电容电压测量和合适的阈值/频率调节变量；不同调制策略和控制周期需要重新整定。
- 混合模型只能详细观察被替换的子模块，不能自动代表全部子模块的器件级应力分布。

## 关键公式

### 梯形积分电容电压更新方程

$$$v_c(t) = v_c(t-\Delta t) + \frac{\Delta t}{2C}[i_c(t) + i_c(t-\Delta t)]$$$

*用于EMT仿真中每个时间步长离散求解子模块直流电容的动态电压变化，保证数值稳定性。*

### 子模块输出电压合成方程

$$$v_{sm,i} = S_i v_{c,i} + v_{sw,i}$$$

*在快速数值模型中，将电容电压与线性化开关压降叠加，构建桥臂受控电压源的输入量。*

### 功率器件线性化伏安方程

$$$v_{sw} = V_{th} + R_{on} i_{sw}$$$

*替代PSCAD/EMTDC中复杂的非线性开关特性，用于加速桥臂等效电路的代数求解。*

## 验证详情

- **验证方式**: 纯数字仿真对比验证（所提快速模型/混合模型 vs. PSCAD/EMTDC官方详细开关模型）
- **测试系统**: 模块化多电平换流器(MMC)系统，典型高压直流输电(HVDC)配置，单桥臂含数百个子模块（如201电平/216 SMs）
- **仿真工具**: PSCAD/EMTDC（利用其内置数值计算组件与受控源接口，无需修改底层求解器C/Fortran源码）
- **验证结果**: 在正常稳态运行与直流故障闭锁两种典型工况下，所提模型的电气量波形、电容电压均衡特性及故障暂态响应与详细开关模型接近。论文报告约5000倍仿真加速，混合模型展示了在全局快速仿真中嵌入局部详细开关动态的可行性。
