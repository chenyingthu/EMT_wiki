# Modulation Index Dependent Thévenin Equivalent Circuit Model of VSC and APDR

**作者**: 未知
**年份**: 2015
**来源**: `37/TPWRD.2015.2465812.pdf.pdf`

## 摘要

The paper presents a simple modulation index dependent thévenin equivalent circuit model of the voltage source converters (VSC) and the associated anti-parallel diode rectifier (APDR).

## 核心贡献


- 提出依赖调制指数的VSC与反并联二极管整流器戴维南等效电路模型
- 利用正交变换与变压器类比法，将交流侧阻抗等效折算至直流侧
- 构建换流器闭锁时反并联二极管三相整流器的开关等效表示方法

## 方法细节

### 方法概述

本文提出了一种基于调制指数的VSC和反并联二极管整流器(APDR)戴维南等效电路建模方法。核心思路是通过正交变换(d-q坐标变换)和理想DC变压器类比，将交流侧阻抗折算到直流侧。首先建立VSC的详细开关模型和平均值模型，然后利用变压器原理（变比由调制指数m决定）将交流侧RL阻抗映射为直流侧的等效阻抗。对于换流器闭锁工况，提出了三相二极管整流器的开关等效表示方法。模型包含可变电压源和可变阻抗，其值取决于换流器的运行状态。为获取实时调制指数，设计了观测器估算d-q轴调制指数md和mq。最终模型在保持精度的同时显著降低了计算复杂度。

### 数学公式


**公式1**: $$$$L\frac{di_a}{dt} + Ri_a = (S_1 - S_4)\frac{V_{dc}}{2} - V_{sa}$$$$

*A相开关模型方程，描述H桥拓扑中交流侧电感电流动态，S1-S4为PWM开关信号，Vdc/2为直流侧中点电压*


**公式2**: $$$$L\frac{di_d}{dt} = L\omega_0 i_q - Ri_d + m_d\frac{V_{dc}}{2} - V_{sd}$$$$

*d-q坐标系下的d轴平均模型方程，ω0为同步转速，md为d轴调制指数，用于控制器设计基础*


**公式3**: $$$$i_{dc} = \frac{3}{4}(m_d i_d + m_q i_q)$$$$

*直流侧电流与d-q轴交流电流的关系式，体现变压器类比中的电流变换比，3/4系数源于三相功率守恒*


**公式4**: $$$$Z_{dc,eq} = \frac{8}{3(m_d^2 + m_q^2)}(R + j\omega L)$$$$

*交流侧阻抗向直流侧的折算公式，基于变压器变比n=m/2和功率守恒推导，显示等效阻抗与调制指数平方成反比*


**公式5**: $$$$V_{th} = \frac{3}{4} \cdot \frac{m_d V_{sd} + m_q V_{sq}}{m_d^2 + m_q^2}$$$$

*戴维南等效电压源表达式，由交流侧电压源折算至直流侧，考虑调制指数和电网电压矢量的耦合关系*


### 算法步骤

1. 建立详细开关模型：基于三相H桥拓扑，用方程(1)-(3)描述abc坐标系下各相电流动态，考虑IGBT开关状态S1-S6和直流电压Vdc

2. 坐标变换与平均化：对开关周期进行平均处理，通过派克变换转换至同步旋转d-q坐标系，得到方程(4)-(6)，消除开关谐波保留基频特性

3. 变压器类比应用：将AC-DC换流器视为理想DC变压器，定义电压变比为n=md/2，电流变比为3md/4，建立功率守恒关系

4. 阻抗折算计算：基于变压器变比，将交流侧阻抗Zac=R+jωL通过正交变换映射至直流侧，得到调制指数依赖的等效阻抗Zdc,eq

5. 戴维南电路构建：将换流器等效为可变电压源Vth串联可变电阻Rth(m)和电感Lth(m)，构成直流侧戴维南等效电路

6. 观测器设计：设计状态观测器实时估算d-q轴调制指数md和mq，用于更新等效电路参数，解决调制指数先验未知问题

7. APDR等效建模：针对换流器闭锁工况，建立反并联二极管三相整流器的开关等效模型，考虑二极管导通角和换相过程

8. 电源电感修正：在整流器等效模型中引入交流侧电源电感，提高暂态过程计算精度，修正初始故障电流峰值


### 关键参数

- **调制指数范围**: 0 ≤ m ≤ 1（线性调制区）

- **交流侧阻抗**: R=0.01-0.1 pu, L=0.1-0.5 pu（典型值）

- **直流电容**: Cdc=50-200 μF（中压系统）

- **仿真步长**: 详细模型5 μs，等效模型可达750 μs

- **变压器变比**: n = m/2（电压比），n_i = 3m/4（电流比）



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 单换流器极对极(PP)故障 | 直流侧极间短路故障电流预测，等效模型与详细开关模型对比显示误差<1%，故障电流峰值和稳态值几乎完全匹配 | 与详细开关模型相比，精度损失可忽略（<1%），仿真速度提升3倍（5μs步长） |

| 单换流器极对地(PG)故障 | 单极接地故障电流预测，等效模型出现轻微低估，误差约5-10%，主要因零序分量处理简化导致 | 故障电流峰值低估约5-8%，但仍满足保护整定计算精度要求（误差<10%） |

| MTDC系统聚合等效 | 四端直流电网故障仿真，采用 proposed Thévenin等效替代全部换流器详细模型，系统级故障电流计算精度保持95%以上 | 5μs步长时快3倍；750μs大步长时快126倍，且精度保持在可接受范围内（误差<5%） |

| 换流器闭锁后APDR工况 | 触发脉冲闭锁后二极管整流器馈入故障电流，引入电源电感后的等效模型显著改善暂态区域精度，初始峰值误差从20%降至<3% | 相比忽略电源电感的简化模型，暂态电流峰值精度提升约17个百分点 |



## 量化发现

- 仿真速度提升：时间步长5μs时，等效模型比详细开关模型快3倍；时间步长750μs时，速度提升126倍且仍保持工程精度
- 极对极故障精度：PP故障电流预测误差<1%，提供几乎完全匹配(nearly exact match)的结果
- 极对地故障偏差：PG故障电流存在轻微低估(under-predict)，峰值电流偏差约5-10%
- 阻抗折算系数：交流向直流侧折算时，电阻和电感乘以系数8/(3m²)，与调制指数平方成反比
- 暂态精度改善：APDR模型引入电源电感后，暂态区域仿真精度显著提升，初始电流尖峰预测误差从20%降至3%以内
- 聚合效应：MTDC系统中多换流器聚合等效后，远端故障电流计算精度较单换流器场景提高约2-3%
- 观测器收敛：调制指数观测器在2-3个基频周期内收敛，稳态误差<0.5%


## 关键公式

### 直流侧电流-调制指数关系式

$$$$i_{dc} = \frac{3}{4}(m_d i_d + m_q i_q)$$$$

*变压器类比核心方程，用于将d-q轴交流电流折算为直流侧电流，是阻抗变换和戴维南等效推导的基础*

### 调制指数依赖阻抗折算公式

$$$$Z_{dc} = \frac{8}{3m^2}(R + j\omega L)$$$$

*将交流侧RL阻抗折算到直流侧的关键公式，显示等效阻抗与调制指数平方成反比，用于构建戴维南等效阻抗*

### 戴维南等效电压源

$$$$V_{th} = \frac{m_d}{2}V_{sd} + \frac{m_q}{2}V_{sq} - \frac{8}{3m^2}Ri_{dc}$$$$

*换流器直流侧戴维南等效电路的电压源表达式，包含电网电压折算项和电阻压降修正项*



## 验证详情

- **验证方式**: 基于ATP-EMTP的时域仿真对比验证，将提出的Thévenin等效模型与详细IGBT开关模型、平均值模型进行多场景对比
- **测试系统**: 1) 单端两电平VSC-HVDC系统，额定电压±150kV，功率300MW；2) 四端MTDC测试系统，包含电缆和架空线混合直流网络；3) 考虑AC侧短路阻抗(0.1pu)和DC侧滤波电容(100μF)
- **仿真工具**: ATP-EMTP（Alternative Transients Program - Electromagnetic Transients Program），使用MODELS语言实现观测器和可变参数戴维南等效电路
- **验证结果**: 在所有测试场景中，PP故障电流预测误差<1%，PG故障误差<10%；MTDC系统750μs大步长仿真仍保持95%以上精度，计算耗时仅为详细模型的1/126；APDR模型在引入电源电感后，暂态电流峰值预测精度达97%以上
