## 方法细节

### 方法概述

本文提出了一种基于嵌套快速同时求解方法（Nested Fast Simultaneous Solution Method）的MMC-SCES高效电磁暂态仿真模型。该方法首先利用Dommel算法（梯形积分法）将子模块中的动态元件（电容、电感）等效为历史电压源与等效电阻串联的支路，将IGBT及续流二极管等效为二值电阻（导通/关断状态）。随后对含储能回路的复杂子模块进行戴维南等效，得到子模块级等效电阻和等效电压源。进一步将桥臂中所有级联子模块合并为单一的戴维南等效电路，大幅降低网络节点数和导纳矩阵维数。在求解过程中，先求解外部网络获得桥臂电流，再以桥臂电流为激励源，通过预先求逆的节点导纳矩阵快速求解子模块内部电气量，实现嵌套求解。该方法在保留所有动态变量（子模块电容电压、超级电容电压、电感电流等）的同时显著降低计算复杂度。

### 数学公式


**公式1**: $$$u_C(t) = u_C(t-\Delta t) + \frac{\Delta t}{2C}(i_C(t) + i_C(t-\Delta t))$$$

*电容元件的梯形积分离散化方程，用于将电容等效为历史电压源与电阻串联支路*


**公式2**: $$$U_{Chis} = u_C(t-\Delta t) + \frac{\Delta t}{2C}i_C(t-\Delta t)$$$

*电容等效电路中的附加历史电压源，包含上一时刻的电容电压和电流信息*


**公式3**: $$$R_C = \frac{\Delta t}{2C}$$$

*电容的定时步等效电阻，仅与仿真步长和电容值相关*


**公式4**: $$$U_{Lhis} = u_L(t-\Delta t) + \frac{2L}{\Delta t}i_L(t-\Delta t)$$$

*电感等效电路中的附加历史电压源，基于梯形积分法推导*


**公式5**: $$$R_L = \frac{2L}{\Delta t}$$$

*电感的定时步等效电阻，与电感值和仿真步长成反比*


**公式6**: $$$R_{DCeq} = R_{T3} + \frac{(R_L+R_{SC})R_{T4}}{R_L+R_{SC}+R_{T4}}$$$

*储能单元（双向Buck-Boost变换器及超级电容）的等效电阻，$R_{T3}$、$R_{T4}$为开关管等效电阻*


**公式7**: $$$U_{DCeq} = R_{T4}\frac{U_{SChis}-U_{Lhis}}{R_{SC}+R_L+R_{T4}}$$$

*储能单元的等效电压源，由超级电容历史电压源和电感历史电压源共同决定*


**公式8**: $$$R_{SMeq} = \frac{R_{T2}(R_{DCeq}R_C+R_{DCeq}R_{T1}+R_{T1}R_C)}{R_{DCeq}R_C+(R_{DCeq}+R_C)(R_{T1}+R_{T2})}$$$

*含储能子模块的戴维南等效电阻，$R_{T1}$、$R_{T2}$为半桥开关管等效电阻*


**公式9**: $$$U_{SMeq} = \frac{R_{T2}(R_{DCeq}U_{Chis}+R_CU_{DCeq})}{R_{DCeq}R_C+(R_{DCeq}+R_C)(R_{T1}+R_{T2})}$$$

*含储能子模块的戴维南等效电压源，综合反映了电容和储能单元的历史状态*


**公式10**: $$$R_{eq}^{jr} = \sum_{i=1}^N R_{SMi}^{jr}$$$

*桥臂总等效电阻，为桥臂内所有子模块等效电阻之和，$j$表示相别，$r$表示上下桥臂*


**公式11**: $$$U_{eq}^{jr} = \sum_{i=1}^N U_{SMi}^{jr}$$$

*桥臂总等效电压源，为桥臂内所有子模块等效电压源之和*


**公式12**: $$$Y_{SM}U_n = I_S$$$

*子模块节点电压方程，$Y_{SM}$为子模块节点导纳矩阵，$U_n$为节点电压向量，$I_S$为注入电流向量*


**公式13**: $$$U_n = Y_{SM}^{-1}I_S$$$

*子模块节点电压求解公式，通过预先对$Y_{SM}$进行符号运算求逆，实现快速计算*


**公式14**: $$$u_{SC}(t) = U_{SChis} + (U_{n3} - U_{SChis} + U_{Lhis})\frac{R_{SC}}{R_L+R_{SC}}$$$

*超级电容瞬时电压计算式，基于节点3电压和历史电压源计算当前时刻超级电容电压*


### 算法步骤

1. 利用Dommel算法（梯形积分法）将子模块中的电容和电感动态元件分别等效为串联的历史电压源与等效电阻支路，其中电容等效电阻$R_C=\Delta t/(2C)$，历史电压源$U_{Chis}$包含上一时刻状态；电感等效电阻$R_L=2L/\Delta t$，历史电压源$U_{Lhis}$包含上一时刻电流信息

2. 将IGBT及续流二极管等效为二值电阻：导通状态下$R_{on}=0.01\,\Omega$，关断状态下$R_{off}=0.5\,\text{M}\Omega$，使子模块等效为仅含无源电阻和直流电源的网络

3. 对含储能单元的子模块进行戴维南等效，计算储能单元等效电阻$R_{DCeq}$和等效电压$U_{DCeq}$，进而计算整个子模块的等效电阻$R_{SMeq}$和等效电压$U_{SMeq}$

4. 对桥臂中所有级联的$N$个子模块进行戴维南等效合并，计算桥臂总等效电阻$R_{eq}^{jr}=\sum R_{SMi}^{jr}$和总等效电压$U_{eq}^{jr}=\sum U_{SMi}^{jr}$，将原有$N$个节点简化为单个等效支路

5. 在EMTDC仿真平台中求解含等效桥臂的MMC-SCES外部网络，获得各时步下的桥臂电流$I_{ch}$，该电流作为激励源用于后续子模块内部计算

6. 建立子模块内部节点电压方程$Y_{SM}U_n=I_S$，其中注入电流向量$I_S$包含桥臂电流$I_{ch}$、电容历史电压源$U_{Chis}$、超级电容和电感历史电压源信息，通过预先符号求逆的$Y_{SM}^{-1}$快速求解节点电压$U_n$

7. 根据求得的节点电压$U_{n2}$（电容节点）和$U_{n3}$（超级电容节点），更新电容电压$u_C(t)=U_{n2}$和超级电容电压$u_{SC}(t)$，计算当前时步各动态元件的电气量，为下一时步的历史电压源计算提供状态变量

8. 重复步骤5-7，实现嵌套快速同时求解，在每个仿真时步先求解外部网络再求解内部子模块状态，保留所有动态变量的同时大幅降低导纳矩阵维数


### 关键参数

- **simulation_time_step**: $\Delta t$（未明确具体数值，但用于计算等效电阻）

- **IGBT_on_resistance**: $R_{on} = 0.01\,\Omega$

- **IGBT_off_resistance**: $R_{off} = 0.5\,\text{M}\Omega$

- **bridge_arm_inductance**: $L_0$

- **DC_bus_voltage**: $\pm 200$ kV



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 稳态运行工况对比 | 在双端MMC-SCES系统稳态运行下，高效仿真模型（EFM）与详细开关模型（DSM）的A相阀侧电压$U_{VA}$波形吻合度极高，均接近正弦波；直流端口电压$U_{dc}$方面，EFM和AVM相比DSM具有更好的高频噪声过滤效果。EFM能够准确拟合子模块内部动态变量（电容电压、超级电容电压），而AVM因忽略开关控制动态导致精度损失。归一化积分绝对误差（$\lambda_{NIAE}$）计算表明EFM与DSM的曲线相似程度非常高。 | 相比DSM，仿真速度提升约518倍，绝对误差标幺化积分控制在1%以内；相比AVM，内部电气量拟合精度显著提高，能够反映各子模块实时状态差异 |

| 动态响应验证 | 在控制系统作用下（交流侧电流矢量控制、储能系统恒功率控制），验证了直流端口电压纹波特性。EFM能够准确模拟直流侧暂态工况，而传统VSM（电压源等效模型）无法模拟直流侧暂态。 | EFM在保留暂态细节的同时，计算效率远高于DSM，适用于超级电容荷电状态均衡策略、子模块电容电压平衡策略等内部控制策略验证 |



## 量化发现

- 相比基于详细开关模型（DSM）的仿真方法，所提高效仿真方法的仿真速度最高可提升约518倍
- 相比详细开关模型，所提方法的仿真绝对误差标幺化积分控制在1%以内
- 稳态工况下，高效仿真模型与详细开关模型各电气量的归一化积分绝对误差（$\lambda_{NIAE}$）非常小，表明曲线相似程度极高
- 平均值模型（AVM）虽仿真速度快，但无法获知不同子模块间的差异，无法处理桥臂内部故障工况，而所提方法可保留换流器所有动态变量
- 详细等效模型（DEM）使用二值电阻替代IGBT，而本方法进一步通过戴维南等效将多子模块桥臂简化为单一支路，节点数减少比例为$N:1$（$N$为每桥臂子模块数，本仿真中$N=21$）


## 关键公式

### 子模块节点电压方程

$$$Y_{SM}U_n = I_S$$$

*在获得桥臂电流后，用于求解子模块内部三个关键节点（半桥中点、电容正极、超级电容连接点）的电压，进而更新所有动态元件状态*

### 桥臂戴维南等效聚合公式

$$$R_{eq}^{jr} = \sum_{i=1}^N R_{SMi}^{jr}, \quad U_{eq}^{jr} = \sum_{i=1}^N U_{SMi}^{jr}$$$

*将桥臂中N个级联子模块等效为单个戴维南电源，大幅降低电磁暂态仿真中的网络节点数和导纳矩阵维数，是实现计算加速的核心*

### 含储能子模块戴维南等效电阻

$$$R_{SMeq} = \frac{R_{T2}(R_{DCeq}R_C+R_{DCeq}R_{T1}+R_{T1}R_C)}{R_{DCeq}R_C+(R_{DCeq}+R_C)(R_{T1}+R_{T2})}$$$

*针对含双向Buck-Boost变换器和超级电容的复杂子模块，综合考虑半桥开关状态、储能单元状态和电容状态的等效电阻计算式*



## 验证详情

- **验证方式**: 对比分析（与详细开关模型DSM、平均值模型AVM对比）
- **测试系统**: 双端MMC-SCES输电系统，额定容量400 MW，储能容量120 MW，交流侧额定电压110 kV，直流母线额定电压±200 kV，每桥臂21个子模块（21电平）
- **仿真工具**: PSCAD/EMTDC仿真平台
- **验证结果**: 所提高效仿真模型在保持与DSM相近精度的前提下（绝对误差标幺化积分<1%，波形相似度极高），仿真速度提升约518倍；相比AVM，能够准确拟合子模块内部动态变量（超级电容电压、子模块电容电压），适用于内部故障分析和储能控制策略验证，而AVM无法反映各子模块实时状态差异
