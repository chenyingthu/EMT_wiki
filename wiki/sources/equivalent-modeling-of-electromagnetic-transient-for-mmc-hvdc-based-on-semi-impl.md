# Equivalent modeling of electromagnetic transient for MMC-HVDC based on semi-implicit delay model

**作者**: Zhao 等
**年份**: 2026
**来源**: `18/Zhao 等 - 2021 - Equivalent modeling of electromagnetic transient simulation for hybrid braking resistance converter;.pdf`

## 摘要

为了解决孤岛新能源柔性直流送出系统中混合式制动电阻变换器原始模型的巨大网络节点数严重制约电磁暂态仿真速度的问题，提出了一种基于戴维南等效及嵌套迭代的混合式制动电阻变换器电磁暂态等效建模方法，将其拆解为 4 类运行状态各异的子模块网络，结合子模块中主／辅开关器件的触发指令组合，得到各类子模块网络的戴维南等效电路，并将依此合成的混合式制动电阻变换器支路等效模型代入柔性直流送出系统网络求解，进而反推内部子模块电气特性，依此循环，迭代求解。并利用电磁暂态仿真软件 PSCAD／EMTDC 开展等效模型与原始模型的一致性仿真测试及仿真速度对比分析，结果表明所提方法能高度模拟混合式制动电阻变换器的外部运行特性及内部子模块变化特性，并可大幅缩短仿真时长。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自孤岛新能源经柔性直流送出系统：受端交流侧故障时，新能源侧功率不能被安控装置快速切除，盈余功率会推高直流电压，因此需要直流侧制动电阻变换器参与耗能。研究对象不是常规MMC本体，而是布置在受端换流站直流侧、由电力电子变换器塔与集中制动电阻串联构成的混合式制动电阻变换器，其中变换器塔含数百个拓扑相同但触发状态独立的子模块。难点在于每个子模块含T1、T2、二极管、电容、均压/限压电阻，且主/辅开关组合形成4类运行状态，不能像单一等效子模块那样整体替代；若逐个搭建详细拓扑，会引入大量网络节点并拖慢EMT求解。本文贡献是把混合式制动电阻变换器按开关指令分解为不同状态子模块，分别推导戴维南等效，再合成为支路等效模型嵌入系统网络，并在求得支路电流后反推各子模块内部电容状态。

### 2. 模型、算法与实现技术

方法由“器件离散—子模块等效—支路聚合—内部状态回代”组成。IGBT及反并联二极管用理想开关模型表示：导通为小电阻，文中给出电磁暂态仿真通常取0.01 Ω；关断简化为开路。电容用梯形积分离散为历史电压源串联等效电阻，核心状态量是子模块电容电压vC和电容电流iC，接口量是子模块端口电压/电流及支路电流，控制输入是T1、T2触发指令f1、f2。根据f1、f2组合，子模块分为4种状态：T1导通时子模块退出支路，T2决定电容是否经限压电阻放电；T1关断时子模块投入支路，T2决定电容与限压电阻的分流/放电关系。每类状态先被化为对应戴维南端口电阻和历史源；随后把投入子模块的等效源、电阻与退出子模块的小电阻及集中制动电阻串联聚合，得到整个制动支路的戴维南等效。系统网络只看到一个等效支路，求解得到当前支路电流后，再按各子模块状态计算电容电流、更新历史源和电容电压，进入下一步长循环。

### 3. 验证、优势与不足

作者使用PSCAD/EMTDC进行等效模型与原始详细模型的一致性仿真，并做仿真速度对比。测试对象为含混合式制动电阻变换器的孤岛新能源柔性直流送出系统，装置布置在受端换流站直流侧正负极之间；故障场景包括受端换流变压器网侧三相接地故障，页面整理中还提到满功率和半功率工况。基线是逐子模块搭建的原始模型，观察指标包括支路端间电压、投入子模块个数、支路电流、支路功率以及内部子模块电容电压变化。验证结论是等效模型能复现外部支路运行特性，并能通过回代得到内部电容动态；计算优势来自把数百个子模块的网络节点压缩为支路戴维南接口，避免系统矩阵直接包含全部子模块拓扑。需要注意，当前给出的原文片段未报告可核验的误差数值、加速比、时间步长敏感性或不同子模块规模下的性能曲线；“缩短仿真时长”的方向由摘要支持，但不能据此给出具体倍数。验证范围也主要限于所述拓扑、控制逻辑和交流侧三相故障，尚不能证明其适用于其他制动拓扑、非理想开关暂态、实时仿真硬件或所有故障类型。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：混合式制动电阻变换器虽含大量独立控制子模块，但对外可通过按开关状态分类的戴维南支路参与系统网络求解，而内部电容状态可在网络求解后局部回代更新，因此不必牺牲每个子模块电容动态来换取计算速度。它适合被后续EMT等值建模、柔直送出系统故障穿越分析、直流侧耗能装置控制验证、以及“外部网络等效+内部状态恢复”类模型页面复用。不宜外推为通用MMC等效模型，也不宜在缺少重新推导的情况下用于拓扑不同、开关非理想过程占主导、控制策略改变或需要精确器件损耗/电磁干扰细节的研究。

## 方法细节

### 方法概述

提出一种基于戴维南等效与嵌套迭代的混合式制动电阻变换器电磁暂态等效建模方法。首先利用理想开关模型与梯形积分法，将子模块内的IGBT、二极管及电容离散化为受触发指令控制的可变电阻与历史电压源串联的戴维南等效电路。根据主/辅开关触发指令的4种组合，将子模块划分为不同运行状态并独立推导其端口等效参数。随后，基于嵌套并行求解思想，将数百个子模块按状态分类聚合，合成整个变换器支路的戴维南等效模型（等效历史电压源与等效电阻串联）。将该支路等效模型直接代入柔性直流系统网络进行求解，获得支路电流后，反向迭代计算各子模块内部电容电流与电压，更新历史电压源，循环推进至下一仿真步长。该方法有效规避了庞大节点网络的直接求解，在保证内外特性精度的前提下大幅提升仿真速度。

### 数学公式

**公式1**: $$$v_C(t) = r_{CEQ} i_C(t) + v_{CEQ}(t-\Delta T)$$$

*电容梯形积分离散化后的戴维南等效电压方程*

**公式2**: $$$r_{CEQ} = \frac{\Delta T}{2C}$$$

*电容等效电阻，与仿真步长和电容值相关*

**公式3**: $$$v_{CEQ}(t-\Delta T) = \frac{\Delta T}{2C} i_C(t-\Delta T) + v_C(t-\Delta T)$$$

*电容等效历史电压源，由上一时刻电流和电压计算*

**公式4**: $$$r_{SEQ} = (r_{loss} + r_e) \parallel r_J$ 或 $r_{SEQ} = r_J$$$

*子模块简化等效电阻，分别对应运行状态①和②*

**公式5**: $$$r_O = r_{CEQ} \parallel r_{SEQ} + r_e$$$

*投入状态子模块的戴维南等效电阻*

**公式6**: $$$v_{OC}(t-\Delta T) = \frac{r_{SEQ}}{r_{CEQ} + r_{SEQ}} v_{CEQ}(t-\Delta T)$$$

*投入状态子模块的戴维南等效历史电压源*

**公式7**: $$$r_{armEQ} = n r_e + \sum_{y=1}^{N-n} r_{O,y} + r_m$$$

*混合式制动电阻变换器支路等效电阻*

**公式8**: $$$v_{armEQ}(t-\Delta T) = \sum_{y=1}^{N-n} v_{OC,y}(t-\Delta T)$$$

*混合式制动电阻变换器支路等效历史电压源*

### 算法步骤

1. 初始化数据：设初始时刻 $i_C(0)=0$，$v_C(0)=U_{dcN}/N$（$U_{dcN}$为额定支路电压），触发T1的子模块数 $n=0$，时间变量 $t=\Delta T$。

2. 计算各子模块电容等效电阻 $r_{CEQ}$ 与历史电压源 $v_{CEQ}(t-\Delta T)$，建立所有子模块电容的戴维南等效电路。

3. 根据系统外环控制器输出的应投入子模块个数及当前各子模块电容电压 $v_C(t-\Delta T)$，通过触发控制器计算各子模块IGBT触发指令 $f_1$ 与 $f_2$。

4. 基于各子模块辅助开关T2的导通状态，利用对应公式计算其简化等效电阻 $r_{SEQ}$。

5. 基于主开关T1的导通状态对子模块分类：若T1导通，更新退出状态子模块计数 $n=n+1$；若T1关断，利用公式计算该子模块等效电阻 $r_O$ 与历史电压源 $v_{OC}(t-\Delta T)$，完成子模块戴维南等效。

6. 汇总所有子模块等效电路，利用公式计算变换器支路等效电阻 $r_{armEQ}$ 与等效历史电压源 $v_{armEQ}(t-\Delta T)$，构建支路戴维南等效模型。

7. 将支路等效模型替代原始拓扑，代入柔性直流系统网络，调用PSCAD/EMTDC求解器计算当前步长支路电流 $i_{arm}(t)$。

8. 根据T1状态分类，分别利用对应公式计算各子模块电容电流 $i_C(t)$，进而利用离散化方程更新电容电压 $v_C(t)$。

9. 更新时间 $t=t+\Delta T$，跳转至步骤2进入下一仿真步长循环迭代。

### 关键参数

- **IGBT导通等效电阻**: 0.01 Ω

- **IGBT关断等效状态**: 开路（极大电阻）

- **二极管导通等效电阻**: 0.01 Ω

- **子模块数量N(验证简化)**: 10

- **电容电压允许波动范围**: ±10%

- **故障触发时刻**: 1.0 s

- **故障持续时间**: 0.6 s

- **集中制动电阻**: $r_m$

- 开关器件导通等效电阻固定为0.01 Ω，关断等效为开路，将非线性开关过程线性化
- 验证模型子模块数量N=10，电容电压波动范围控制在±10%以内，满足均压控制要求
- 故障测试窗口为1.0 s至1.6 s，等效模型与原始模型关键电气量波形高度一致，动态偏差在工程允许范围内

## 关键公式

### 电容梯形积分戴维南等效方程

$$$v_C(t) = \frac{\Delta T}{2C} i_C(t) + \left( \frac{\Delta T}{2C} i_C(t-\Delta T) + v_C(t-\Delta T) \right)$$$

*用于将连续电容离散化为历史电压源与等效电阻串联形式，是EMT仿真数值积分的核心*

### 支路戴维南等效合成方程

$$$v_{arm}(t) = \left( n r_e + \sum_{y=1}^{N-n} r_{O,y} + r_m \right) i_{arm}(t) + \sum_{y=1}^{N-n} v_{OC,y}(t-\Delta T)$$$

*将分类后的子模块网络聚合为单一等效支路，直接代入系统网络求解，大幅降低节点规模*

### 投入状态子模块端口电压方程

$$$v_{SM\_2}(t) = \left( \frac{r_{CEQ} r_{SEQ}}{r_{CEQ} + r_{SEQ}} + r_e \right) i_{arm}(t) + \frac{r_{SEQ}}{r_{CEQ} + r_{SEQ}} v_{CEQ}(t-\Delta T)$$$

*用于在已知支路电流后，反推处于投入状态子模块的端口电压及内部电容电气特性*
