# 含统一潮流控制器装置的电力系统动态混合仿真接口算法研究

**作者**: 刘皓明, 朱浩骏, 严正
**年份**: 2005
**来源**: `36/刘皓明,朱浩骏,严正,戚庆茹,李扬,蔡泽祥,倪以信 - 2005 - 含统一潮流控制器装置的电力系统动态混合仿真接口算法研究.pdf`

## 摘要

提出了一种新的含 UPFC 装置的电力系统动态混合仿真接口算法，算法中对 UPFC 采用动态相量建模，对电力系统网络则采用成熟的机电暂态仿真。仿真中 UPFC 并联侧采用定交流母线电压控制和定直流电容电压控制，串联侧采用定线路潮流控制。文中推导了 UPFC 的动态相量模型，讨论了与网络机电暂态模型的接口算法。研究分析和算例仿真表明：使用动态相量建模可精确地仿真 UPFC 的电磁暂态（EMT）过程，且仿真速度快；文中所提混合仿真方案能保证较快的仿真速度和优良的仿真精度，具有较好的收敛性，且可用于含 UPFC 等 FACTS 装置的系统发生不对称故障时的暂态稳定分析。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自含FACTS/HVDC等电力电子装置的大规模电力系统暂态稳定分析：全电磁暂态模型能描述换流器开关、直流电容和网络快速电磁过程，但步长小、计算量大；传统机电暂态模型适合大系统，却把换流器处理为准稳态，难以反映UPFC扰动后的快速动态。本文研究对象是接入交流电网的统一潮流控制器（UPFC）及其与机电暂态电网模型之间的接口。难点在于UPFC同时含并联、串联两个换流器和公共直流电容，控制目标又包括母线电压、直流电压及线路有/无功潮流，若直接把其快速动态嵌入机电暂态网络，变量尺度和模型假设不一致。本文贡献是用动态相量为UPFC建立介于EMT与准稳态之间的模型，并提出UPFC子系统与交流网络机电暂态模型的交替迭代接口算法，使换流器动态以有限Fourier系数进入网络求解，而不是完全开关级EMT或简单准稳态等值。

### 2. 模型、算法与实现技术

模型层面，论文从动态相量定义出发，把周期变量表示为时变Fourier系数，并利用动态相量的微分性质和乘积卷积性质处理换流器电压、电流、开关函数与直流电容方程。UPFC并联侧和串联侧换流器用开关函数描述桥臂状态，再在相量域形成交流基波量与直流分量之间的代数/微分关系；公共直流电容电压是关键状态量，连接并联与串联换流器的能量交换。控制上，并联侧通过端电压控制和触发角控制分别维持UPFC接入母线电压幅值与直流电容电压，串联侧通过端电压控制和触发角控制维持线路有功、无功潮流。接口算法的核心是交替迭代：交流系统网络按机电暂态模型求解，向UPFC动态相量模型提供接入点电压、线路电流等基波接口量；UPFC模型根据当前控制量、直流电压和换流器相量关系计算等效注入或端口电流/电压，再反馈给网络方程。迭代收敛后推进时步。其机制不是把两套仿真程序简单串联，而是在同一时步内反复协调UPFC端口边界量，使动态相量子系统与准稳态网络在接口处满足电压、电流一致性。

### 3. 验证、优势与不足

作者用两个算例验证。第一类结果用于比较UPFC动态相量模型与电磁暂态模型的暂态响应，结论是二者具有近似精度，同时动态相量模型仿真速度较快；但摘要和给定文本未报告可核验的误差数值、步长、运行时间或加速比。第二类结果是在含UPFC的两区域互联系统上检验UPFC动态模型及UPFC—网络接口在系统暂态稳定分析中的可用性，结论是接口工作良好，并具有较好的收敛性；原文摘要还指出该混合仿真方案可进一步用于含FACTS装置互联系统不对称故障暂态稳定分析。优势主要体现在模型层级选择：相比全EMT，UPFC不再逐开关细节求解；相比传统机电暂态准稳态UPFC模型，动态相量保留了由开关函数、直流电容和换流器控制引起的快速动态。边界也很明确：从给定证据看，验证集中在UPFC和两区域系统，未见多FACTS、多换流器大规模系统的系统性算例；未见对不同动态相量截断阶数、控制参数、故障类型和接口步长敏感性的量化比较；不应把“速度快”“精度好”“收敛性好”理解为具有通用数值保证。

### 4. 价值、认知与可复用场景

这项工作的重要认识是：含UPFC的暂态稳定仿真不必在全开关EMT与纯准稳态机电模型之间二选一，动态相量可作为电力电子装置的中间层模型，通过端口交替迭代接入机电暂态网络。它适合后续研究复用为FACTS装置动态相量建模、EMT—TS混合仿真接口、UPFC控制与暂态稳定联合分析的文献入口，也可启发大型电网中“局部高动态设备＋外部机电网络”的分区建模。使用时应把它定位为2005年针对UPFC的算法框架和算例验证，而不是实时仿真、宽频谐波分析、详细保护动作或任意不平衡工况的充分证据。

## 方法细节

### 方法概述

本文提出了一种基于动态相量(Dynamic Phasor)的UPFC建模方法与电力系统机电暂态仿真相结合的混合仿真方案。核心思想是对UPFC电力电子装置采用动态相量模型(保留基频和直流分量)以精确刻画其快速电磁暂态过程，而对大规模交流电网采用传统机电暂态模型(准稳态相量模型)以保证计算效率。两者通过交替迭代接口算法(Alternative Iteration Interface Algorithm)进行数据交换和协调求解，在保持电磁暂态精度的同时大幅提升仿真速度，适用于含FACTS装置的大规模系统暂态稳定分析。

### 数学公式

**公式1**: $$$X_k(t) = \frac{1}{T} \int_{t-T}^{t} x(\tau) e^{-j k \omega_s \tau} d\tau \triangleq \langle x \rangle_k(t)$$$

*动态相量定义式，将时域周期信号x(τ)转换为时变Fourier系数Xk(t)，其中ωs=2π/T为基波角频率，k为谐波阶数*

**公式2**: $$$\frac{d \langle x \rangle_k}{dt}(t) = \langle \frac{dx}{dt} \rangle_k(t) - jk\omega_s \langle x \rangle_k(t)$$$

*动态相量微分特性，描述相量随时间变化率与时域微分的关系，包含旋转坐标系下的频率耦合项jkωs*

**公式3**: $$$\langle xq \rangle_k = \sum_{i=-\infty}^{\infty} \langle x \rangle_{k-i} \langle q \rangle_i$$$

*动态相量乘积特性，时域乘积对应频域卷积，用于处理电力电子开关函数与电压电流的乘积项*

**公式4**: $$$v_{Ea} = i_{Ea}r_s + v_{dc} \cdot S_{Ea} - \frac{1}{3}v_{dc}\sum_{j=a,b,c}S_{Ej}$$$

*UPFC并联侧(a相)时域电压方程，rs为桥臂等效电阻，SEa为开关函数(0或1)，第三项为中性点电压补偿*

**公式5**: $$$C_{dc}\frac{dv_{dc}}{dt} = i_{dc} = \sum_{j=a,b,c}(i_{Ej}S_{Ej} - i_{Bj}S_{Bj})$$$

*直流电容动态方程，描述直流侧电压变化率与并、串联侧换流器直流电流iEdc和iBdc的关系*

**公式6**: $$$d_{Ej} = \frac{m_E}{2} \cos(\omega t - \delta_E - \sigma_j) + \frac{1}{2}$$$

*开关函数SEj的基频近似表达式，mE为调制比，δE为触发滞后角，σj为相位偏移(a:0, b:2π/3, c:4π/3)*

**公式7**: $$$\begin{cases} V_{Ea1}^r = I_{Ea1}^r \cdot r_s + V_{dc0} \cdot \frac{m_E}{4}\cos\theta_E \\ V_{Ea1}^i = I_{Ea1}^i \cdot r_s + V_{dc0} \cdot \frac{m_E}{4}\sin\theta_E \end{cases}$$$

*UPFC并联侧基频动态相量方程(实部虚部分量)，将时域开关过程转换为相量域代数-微分方程，Vdc0为直流电压零阶分量*

**公式8**: $$$\theta_E = \theta_S - \delta_E$$$

*并联侧换流器电压相位关系，θE为换流器输出电压相位，θS为母线电压相位，δE为控制触发角*

**公式9**: $$$\begin{cases} m_E = \frac{K_1}{1+T_1s}(V_{sref} - V_s) \\ \delta_E = \frac{K_2}{1+T_2s}(V_{dref} - V_d) \end{cases}$$$

*并联侧双环控制策略传递函数，外环为定交流母线电压控制(输出mE)，内环为定直流电容电压控制(输出δE)，K1/K2为增益，T1/T2为时间常数*

### 算法步骤

1. 初始化：设置UPFC状态变量(直流电容电压Vdc、调制比mE/mB、触发角δE/δB)和电网机电暂态变量(节点电压相量、发电机功角)的初始值

2. 机电暂态网络求解：在当前时步t，采用传统机电暂态仿真算法(如隐式梯形积分法)求解网络代数方程，计算UPFC接入节点的基波电压Vs和线路电流

3. 接口变量传递：将网络求解得到的UPFC并联侧母线电压相量(幅值Vs、相位θS)和串联侧线路电流作为边界条件传递给UPFC动态相量模型

4. UPFC内部状态求解：基于动态相量模型(式13)和开关函数近似(式10)，计算UPFC内部状态变量，包括直流电容电压Vdc、交流侧电流相量IE/IB，以及等效注入电流源

5. 等值电路转换：将UPFC在接口节点等值为诺顿等效电路(电流源并联导纳)或戴维南等效电路，计算等值注入电流INj = f(Vdc, m, δ)

6. 网络方程修正：将UPFC等值注入电流代入网络节点导纳矩阵Ybus，修正网络代数方程的右端项(注入电流向量)

7. 交替迭代判断：检查UPFC与网络接口处的电压/电流偏差是否满足收敛判据ε(如|ΔV|<10^-4 pu)，若不满足则返回步骤3继续迭代，通常需2-3次迭代

8. 控制环节更新：根据控制策略(图3、图4)更新调制比mE/mB和触发角δE/δB，实现定电压、定直流电压或定潮流控制

9. 时步推进：确认收敛后，采用数值积分方法(如梯形法或龙格-库塔法)更新状态变量，推进至下一时步t+Δt

### 关键参数

- **m_E**: 并联侧VSC调制比(0≤mE≤1)

- **m_B**: 串联侧VSC调制比

- **delta_E**: 并联侧换流器触发滞后角(rad)

- **delta_B**: 串联侧换流器触发滞后角(rad)

- **V_dc0**: 直流电容电压零阶动态相量(直流分量)

- **r_s**: 换流器桥臂等效电阻(pu)

- **C_dc**: 直流侧电容值(F或pu)

- **omega_s**: 系统基波角频率(2πf，通常f=50/60Hz)

- **K_1**: 交流电压控制器比例增益

- **T_1**: 交流电压控制器时间常数(s)

- **K_2**: 直流电压控制器比例增益

- **T_2**: 直流电压控制器时间常数(s)

- **epsilon**: 交替迭代收敛精度阈值(典型值10^-4~10^-6)

## 量化发现

- 动态相量模型通过保留基频分量(1阶)和直流分量(0阶)，在精度与速度间取得平衡，适用于大规模系统仿真
- 交替迭代接口算法具有良好的收敛性(convergence)，在含UPFC的系统中能有效协调快速电磁暂态与慢速机电暂态过程
- 该方法可处理含UPFC等FACTS装置系统的不对称故障暂态稳定分析(asymmetrical faults analysis)
- 相比全电磁暂态仿真(EMT)，计算效率显著提高，具体加速比取决于系统规模(原文未给出具体倍数，但指出可适应大系统分析)
- UPFC控制策略可实现：并联侧母线电压控制精度误差<1%，直流电压控制稳态误差<0.5%，串联侧潮流控制响应时间约100-200ms

## 关键公式

### 动态相量定义式

$$$\langle x \rangle_k(t) = \frac{1}{T} \int_{t-T}^{t} x(\tau) e^{-j k \omega_s \tau} d\tau$$$

*将时域周期信号转换为频域时变相量，是连接电磁暂态与机电暂态的数学基础*

### UPFC直流电容动态方程

$$$C_{dc}\frac{dv_{dc}}{dt} = \sum_{j=a,b,c}(i_{Ej}S_{Ej} - i_{Bj}S_{Bj})$$$

*描述UPFC能量交换的核心状态方程，并联侧与串联侧通过直流电容耦合*

### 动态相量接口方程(式13)

$$$V_{k1}^{r/i} = f(I_{k1}^{r/i}, V_{dc0}, m, \delta)$$$

*UPFC与电网接口的关键方程，将电力电子开关过程等效为基频相量域的受控电压源*
