# 基于状态空间法的高压直流输电系统电磁暂态简化模型的解析算法

**作者**: 程佩芬
**年份**: 2019
**来源**: `40/程佩芬 等 - 2019 - 基于状态空间法的高压直流输电系统电磁暂态简化模型的解析算法.pdf`

## 摘要

An analytic solution for simplified high voltage direct current (HVDC) system based on state space method is proposed, which is mainly used to simulate the transient response of DC system.

## 核心贡献



- 提出基于状态空间法的HVDC电磁暂态简化模型解析算法，通过引入辅助变量将非齐次微分方程转化为齐次方程，大幅降低计算复杂度
- 结合整流侧准稳态简化与逆变侧分段线性模型，在保证暂态响应精度的同时显著提升仿真速度，并有效避免传统数值积分中的插值振荡问题

## 主要发现



- 所提解析算法在模拟逆变侧交流故障暂态过程时，与PSCAD全电磁暂态模型结果高度吻合，验证了算法的准确性
- 该简化模型与解析求解策略有效克服了传统EMT仿真步长小、计算耗时长的瓶颈，同时消除了换相电流过零点插值引发的数值振荡
