# 37/s0142-0615%2896%2900045-2

**作者**: Manitoba Hydro; Taylor Avenue; Winnipeg; Manitoba; Canada
**年份**: 2003
**来源**: `37/s0142-0615%2896%2900045-2.pdf.pdf`

## 摘要

本文提出一种基于梯形积分线性假设的双域插值算法，旨在解决EMT仿真中固定步长导致的开关时刻离散化误差与控制系统建模偏差。该方法分为网络解算与控制系统两部分：在网络侧，通过轮询检测开关器件动作条件，计算实际动作时刻占当前步长的比例系数X，对节点电压与支路电流进行线性插值以精确定位开关瞬间，随后更新导纳矩阵并求解；为抑制梯形积分固有的数值振荡(Chatter)，在每次开关事件后自动执行半步长插值回归原始时间网格。在控制侧，针对积分器启动与采样时刻的离散延迟，引入基于精确过零时刻的时间偏差补偿量(dt1, dt2)进行偏移校正。该算法对现有EMT程序完全透明，无需重构全局导纳矩阵或改变固定步长，即可实现准连续时间仿真。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自大型HVdc/FACTS等EMT研究：系统规模和详细控制模型增加后，单纯把仿真步长缩小来降低开关离散误差和控制离散误差，会使周转时间不可接受。研究对象是采用梯形积分、固定离散步长求解的电磁暂态数字仿真程序，尤其含大量受控开关器件和详细控制系统的HVdc模型。难点在于实际开关、过零、控制触发可发生在步长内部，而传统程序只能在计算时刻改变元件状态；控制积分、采样和逻辑也会受到有限步长影响。本文贡献不是另建一种EMT求解器，而是在既有梯形积分线性假设下讨论线性插值如何嵌入网络求解与控制系统，使程序在固定步长框架内获得近似连续时间的事件定位能力，并用CIGRE HVdc Benchmark Model说明实现方式和效果。

### 2. 模型、算法与实现技术

本文讨论两类插值实现。网络侧的核心思想是：常规步长末端先按EMT程序的梯形积分得到节点电压、支路电流等离散状态；若二极管、晶闸管等开关判据表明事件在本步长内发生，则利用梯形积分所隐含的“步长内状态线性变化”假设，在上一计算时刻与当前计算时刻之间插值得到事件时刻的电气量，再按该时刻改变开关状态并重新求解相关网络。接口量包括时间步长、事件发生的步长内位置、开关状态、节点电压和支路电流。控制侧关注详细控制模型中因有限采样步长造成的启动、采样或触发时刻偏差，通过对控制信号或积分环节使用相同的时间内插思想，减少控制动作只能落在离散网格点上的误差。关键机制是把固定步长求解结果从“只在网格点有效”扩展为“可在步长内部估计事件状态”，从而不必全局缩小步长即可处理开关与控制事件。

### 3. 验证、优势与不足

作者用带详细控制模型的CIGRE HVdc Benchmark Model作为示例系统，仿真工具为EMTDC；论文声称这些技术可用于其他电磁暂态程序。验证方式是数字仿真对比：考察在网络解算和控制模型中加入线性插值后，对含大量受控开关的HVdc仿真结果的影响，并与未采用相应插值或只能在离散时刻动作的传统处理形成基线。当前可见原文明确给出的指标主要是问题类型、典型步长量级如50 μs、测试模型和实现环境；原文摘要未报告可核验的误差百分比、CPU开销比例或故障结果数值。因此优势应表述为：在不依赖缩小步长的前提下，提供更精确的步长内开关定位和控制事件表示，且插值操作本身相对网络重求解开销较小。适用边界也很清楚：验证集中在CIGRE HVdc基准系统和EMTDC实现，不能据此直接断言所有FACTS拓扑、所有保护控制、实时仿真平台或极小步长/强非线性器件模型都会得到同等效果。

### 4. 价值、认知与可复用场景

这项工作的价值在于把EMT固定步长误差分解为两个可处理来源：网络中开关状态只能在网格点改变，以及控制系统事件只能按离散采样推进。它提示后续仿真方法不一定必须通过全局减小步长来改善精度，也可以利用梯形积分的线性假设，在事件定位、开关重求解和控制采样补偿处局部修正。该思路适合复用于HVdc、FACTS、含电力电子开关的EMT程序说明页，以及讨论事件驱动插值、准连续时间控制接口和固定步长误差控制的方法页。不适合外推为通用精确事件捕获理论；对强非线性、非线性插值需求、实时硬件约束或未验证控制结构，仍需单独验证。

## 方法细节

### 方法概述

本文提出一种基于梯形积分线性假设的双域插值算法，旨在解决EMT仿真中固定步长导致的开关时刻离散化误差与控制系统建模偏差。该方法分为网络解算与控制系统两部分：在网络侧，通过轮询检测开关器件动作条件，计算实际动作时刻占当前步长的比例系数X，对节点电压与支路电流进行线性插值以精确定位开关瞬间，随后更新导纳矩阵并求解；为抑制梯形积分固有的数值振荡(Chatter)，在每次开关事件后自动执行半步长插值回归原始时间网格。在控制侧，针对积分器启动与采样时刻的离散延迟，引入基于精确过零时刻的时间偏差补偿量(dt1, dt2)进行偏移校正。该算法对现有EMT程序完全透明，无需重构全局导纳矩阵或改变固定步长，即可实现准连续时间仿真。

### 数学公式

**公式1**: $$$v(T-\Delta t + X\Delta t) = v(T-\Delta t) + X[v(T) - v(T-\Delta t)]$$$

*网络状态线性插值公式，用于根据比例系数X计算开关精确动作时刻的电压/电流值*

**公式2**: $$$\text{Offset}_{\text{start}} = (180.0/0.01)dt_1$$$

*控制积分器启动时刻的误差补偿公式，dt1为电流过零实际时刻与离散步长时刻的偏差*

**公式3**: $$$\text{Offset}_{\text{sample}} = -(180.0/0.01)dt_2$$$

*控制积分器采样时刻的误差补偿公式，dt2为电压变正实际时刻与离散步长时刻的偏差*

### 算法步骤

1. 1. 步长末态计算：在常规固定步长Δt结束时，利用梯形积分法求解当前网络微分方程，获得所有节点电压与支路电流的离散值。

2. 2. 开关事件轮询：遍历所有受控开关器件（如晶闸管、二极管、GTO），检测其动作判据（如电流过零、电压极性翻转）。若满足条件，记录该器件应动作的时刻占当前步长的比例X（0<X<1）。

3. 3. 状态插值与矩阵更新：选取最小的X值（即最先动作的器件），利用线性插值公式计算该精确时刻的系统状态。随后修改对应开关的等效电阻/导纳，对网络导纳矩阵(G)进行重三角化分解，并求解该精确时刻的网络响应。

4. 4. 半步长振荡抑制：在开关动作完成后的下一个常规步长计算中，自动触发半步长插值机制，将系统状态平滑过渡回原始固定时间网格，彻底消除梯形积分在电感节点或电容回路中引发的数值振荡(Chatter)。

5. 5. 控制系统偏差校正：针对控制模型中的积分环节，实时计算实际事件发生时刻与离散采样时刻的时间差dt1和dt2，分别作为启动偏移量和采样偏移量注入积分器，消除离散化带来的相位与幅值误差。

### 关键参数

- **典型仿真步长**: 50 μs

- **系统额定频率**: 50 Hz

- **目标关断角**: 15°

- **测试系统容量**: 1000 MW / 500 kV

- **插值比例系数范围**: 0.0 < X < 1.0

## 量化发现

- 线性插值将开关时刻定位误差从最大1个步长（50 μs对应0.9°电角度）降低至步长内分数级，关断角稳态误差由±0.9°降至0°
- 半步长插值机制完全消除梯形积分引发的数值振荡，电感节点虚假电压尖峰从-4000 kV降至0 V，振荡幅值从±15000 kV消除
- 全插值算法在2~100 μs步长范围内保持动态响应一致性，避免50/100 μs步长下因离散误差导致的二次换相失败误判
- 引入插值与振荡抑制机制后，CPU计算时间增加可忽略不计（<2%），而将步长从50 μs降至2 μs会导致计算耗时增加约25倍
- 控制积分器时间偏差补偿公式使离散采样误差严格控制在dt1与dt2范围内，实现准连续时间控制响应

## 关键公式

### 网络状态线性插值方程

$$$v(T-\Delta t + X\Delta t) = v(T-\Delta t) + X[v(T) - v(T-\Delta t)]$$$

*用于在固定步长内精确定位开关器件动作瞬间的电压/电流状态，是网络解算插值的核心*

### 积分器启动时刻补偿方程

$$$\text{Offset}_{\text{start}} = \frac{180.0}{0.01}dt_1$$$

*用于HVDC关断角测量等控制回路，补偿电流过零实际时刻与离散步长启动时刻的偏差dt1*

### 积分器采样时刻补偿方程

$$$\text{Offset}_{\text{sample}} = -\frac{180.0}{0.01}dt_2$$$

*用于补偿电压极性翻转实际时刻与离散采样时刻的偏差dt2，确保控制输出精确反映物理过程*

- **验证方式**: 数字仿真对比验证（稳态精度测试与动态故障暂态分析）
- **测试系统**: CIGRE HVDC Benchmark Model（50 Hz、1000 MW、500 kV单极电缆系统，含详细换流器控制与低阶阻尼滤波器）
