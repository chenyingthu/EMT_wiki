# Dual-Band Reduced-Order Model of an HVDC Link Embedded into a Power Network for EMT Studies

**作者**: Carlos Ruiz
**年份**: 2019
**来源**: `13&14/files/TEC.2019.2935892.pdf.pdf`

## 摘要

本文提出一种基于频域平衡实现（FDBR）的双频段模型降阶方法，专用于含电力电子换流器的电力系统电磁暂态仿真。传统全频段降阶难以兼顾电网低频机电振荡与换流器高频开关动态。该方法首先利用矢量拟合技术获取线性电网的宽频带全阶状态空间模型。随后，针对低频（Hz级）和高频（kHz级）独立频带，分别计算频域受限的可控性与可观性格拉姆矩阵，并通过求解李雅普诺夫方程量化各频带能量分布。将双频带格拉姆矩阵叠加后，经特征值分解求取相似变换矩阵，将原系统转换为内部平衡形式。最后，依据汉克尔奇异值阈值直接截断弱动态模态，生成低阶模型。该模型与详细开关网络耦合，在大幅降低ODE数量的同时，精准保留目标频带内的主导动态特性，实现高效高精度的EMT仿真。

## 研究解读

### 1. 需求、对象、挑战与贡献

现代含电力电子换流器的电网用于EMT仿真时，若保留线路、网络与VSC-HVDC开关模型的详细动态，状态维数和固定小步长积分成本会很高；若改用平均值模型，又会丢掉开关及高频谐波动态。本文研究对象是嵌入交流电网中的HVDC链路及其外部线性网络等值，目标是在EMT时域仿真中同时保留电网低频慢动态和换流器相关高频快动态。难点在于传统全频段降阶会把状态重要性在整个频域平均化，低频机电振荡和kHz级开关现象对格拉姆矩阵的贡献尺度不同，单一频带降阶容易只保留其中一类现象。本文的贡献是把频域平衡实现用于带限模型降阶，并把低频段与高频段的可控/可观能量组合成双频段平衡准则，使降阶模型不是追求全频段一致，而是面向EMT真正关心的两个频率窗口保留主导动态。

### 2. 模型、算法与实现技术

方法从线性电网的端口频响出发，先用矢量拟合得到连续时间有理状态空间模型(A,B,C,D)。输入/输出可理解为网络端口注入电流、端口电压或等效阻抗/导纳关系；降阶后的模型仍以状态方程与端口代数接口接入详细换流器开关网络。核心计算不是普通平衡截断的全频格拉姆矩阵，而是在给定频率上下限内构造频域受限可控性、可观性格拉姆矩阵。公式Wcf(Ω1,Ω2)=Wcf(Ω2)-Wcf(Ω1)用于把状态能量限制在目标频带；频域李雅普诺夫方程把频率积分转化为矩阵方程求解，避免逐点积分。对低频段和高频段分别求得格拉姆矩阵后相加，得到能同时反映两类动态的总可控/可观能量。随后通过WcfWof的特征分解或等价平衡变换得到汉克尔奇异值排序，在平衡坐标中截断小奇异值对应状态，形成降阶状态空间模型，并用梯形积分等EMT求解流程与开关换流器耦合。

### 3. 验证、优势与不足

据当前页面抽取信息，验证包括频域频响对比和时域EMT波形对比：测试对象为改进WSCC 9节点系统，在B7与B8之间嵌入双端VSC-HVDC链路，并包含频率相关线路和RL负载；频域建模/降阶使用MATLAB矢量拟合工具，时域仿真使用自定义梯形积分EMT求解器，步长10 μs。基线是1200阶全阶矢量拟合模型以及单低频、单高频降阶模型；指标包括阻抗频响重合程度、时域直流电压/交流电流波形一致性、保留阶数和VF拟合RMS误差。页面给出的量化结果为：VF RMS误差5.91×10^-11，低频/高频/双频段降阶阶数分别为26、72、125，双频段模型为原1200阶的10.4%。优势在于可解释地说明为什么低频模型不能复现开关频带、高频模型会丢失低频振荡，而双频段平衡能同时保留两类端口动态。从验证范围看，结论主要支持该算例、所选频带、所用VSC-HVDC控制/开关设置和固定步长求解器；尚不能直接证明对其他拓扑、故障、大扰动非线性控制或实时仿真平台同样成立。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：面向EMT的网络等值不应简单追求宽频全局最优，而应围绕物理现象选择频带，把机电慢动态和电力电子快动态分别纳入降阶准则。它适合复用于含VSC-HVDC、并网变流器、主动滤波器或新能源接口的网络等值页面，也可作为频域平衡截断、矢量拟合、端口动态等值和EMT加速仿真的方法入口。使用时应把它理解为线性网络部分的降阶技术，而不是完整换流器控制器或开关非线性本身的替代模型；也不应外推为任意频率范围、任意故障场景下都自动保持精度。

## 方法细节

### 方法概述

本文提出一种基于频域平衡实现（FDBR）的双频段模型降阶方法，专用于含电力电子换流器的电力系统电磁暂态仿真。传统全频段降阶难以兼顾电网低频机电振荡与换流器高频开关动态。该方法首先利用矢量拟合技术获取线性电网的宽频带全阶状态空间模型。随后，针对低频（Hz级）和高频（kHz级）独立频带，分别计算频域受限的可控性与可观性格拉姆矩阵，并通过求解李雅普诺夫方程量化各频带能量分布。将双频带格拉姆矩阵叠加后，经特征值分解求取相似变换矩阵，将原系统转换为内部平衡形式。最后，依据汉克尔奇异值阈值直接截断弱动态模态，生成低阶模型。该模型与详细开关网络耦合，在大幅降低ODE数量的同时，精准保留目标频带内的主导动态特性，实现高效高精度的EMT仿真。

### 数学公式

**公式1**: $$$W_{cf}(\Omega_1, \Omega_2) = W_{cf}(\Omega_2) - W_{cf}(\Omega_1)$$$

*频域受限可控性格拉姆矩阵差分公式，用于提取特定频带$[\Omega_1, \Omega_2]$内的系统可控能量分布。*

**公式2**: $$$AW_{cf} + W_{cf}A^* = -(BB^*F^* + FBB^*)$$$

*频域李雅普诺夫方程，结合积分项$F$求解频带受限的可控性格拉姆矩阵，避免全频域积分的计算负担。*

**公式3**: $$$W_{cf}W_{of} = T_b \Lambda_b T_b^{-1}$$$

*平衡变换特征值分解式，用于求解相似变换矩阵$T_b$，使变换后的系统可控与可观格拉姆矩阵相等且对角化。*

**公式4**: $$$\dot{x}_r = A_r x_r + B_r u, \quad y = C_r x_r + Du$$$

*降阶后的状态空间微分方程组，直接用于EMT时域数值积分求解。*

### 算法步骤

1. 构建全阶模型：利用矢量拟合（VF）技术对线性电网进行宽频带（1 Hz至5 kHz）频响拟合，获取高阶有理传递函数并转换为连续时间状态空间形式$(A, B, C, D)$。

2. 定义目标频带：根据物理特性划分低频带（1-100 Hz，覆盖机电振荡）和高频带（1-5 kHz或3-5 kHz，覆盖换流器开关动态）。

3. 计算积分矩阵F：针对各频带边界频率，利用矩阵A的稀疏/块对角结构高效计算频域积分项$F = \int_{-\Omega}^{\Omega} (j\omega I - A)^{-1} d\omega$。

4. 求解频域格拉姆矩阵：将F代入频域受限李雅普诺夫方程，分别求解低频与高频下的可控性$W_{cf}$和可观性$W_{of}$格拉姆矩阵。

5. 双频带矩阵融合：将低频与高频的格拉姆矩阵对应相加，得到综合反映双频段动态能量的总格拉姆矩阵$W_{cf}^{total}$与$W_{of}^{total}$。

6. 平衡坐标变换：求解乘积矩阵$W_{cf}^{total}W_{of}^{total}$的特征值分解，获得相似变换矩阵$T_b$，将原系统变换为内部平衡系统，使格拉姆矩阵均等于对角化的汉克尔奇异值矩阵$\Sigma$。

7. 状态截断降阶：设定汉克尔奇异值阈值（如$10^{-10}$），剔除小于阈值的弱动态模态，直接截断对应状态变量，得到降阶模型$(A_r, B_r, C_r, D)$。

8. 接口与EMT仿真：将降阶线性网络模型与详细开关换流器模型（线性时周期ODE）通过节点电压/电流接口耦合，采用梯形积分法进行固定步长时域仿真。

### 关键参数

- **全阶模型维数**: 1200

- **低频带范围**: 1 Hz ~ 100 Hz

- **高频带范围**: 3 kHz ~ 5 kHz (或 1 kHz ~ 5 kHz)

- **汉克尔奇异值截断阈值**: 1e-10

- **低频降阶维数**: 26

- **高频降阶维数**: 72

- **双频带降阶维数**: 125

- **EMT仿真步长**: 10 μs

- **换流器开关频率**: 1.5 kHz

- **VF拟合RMS误差**: 5.91e-11

## 量化发现

- 双频段降阶将系统状态维数从1200阶压缩至125阶，模型规模降至原系统的10.4%，大幅降低ODE求解计算复杂度。
- 矢量拟合（VF）全阶近似的RMS误差仅为$5.91 \times 10^{-11}$，确保频域输入数据的极高保真度。
- 采用$10^{-10}$汉克尔奇异值阈值进行截断，低频、高频及双频段降阶阶数分别为26、72和125，实现模态能量的精准筛选。
- 在10 μs固定步长EMT仿真中，双频段模型在1.5 kHz开关频率下保持时域波形与全阶模型高度一致，验证了局部频带拟合在兼顾低频机电与高频开关动态方面的数值有效性。

## 关键公式

### 频域受限格拉姆矩阵差分式

$$$W_{cf}(\Omega_1, \Omega_2) = W_{cf}(\Omega_2) - W_{cf}(\Omega_1)$$$

*用于从全频域能量分布中剥离出特定目标频带（如低频或高频）的系统可控/可观能量，是双频段局部降阶的核心数学基础。*

### 频域李雅普诺夫方程

$$$AW_{cf} + W_{cf}A^* = -(BB^*F^* + FBB^*)$$$

*在已知频域积分矩阵F的情况下，高效求解频带受限的格拉姆矩阵，避免直接数值积分带来的计算不稳定问题。*

### 降阶状态空间微分方程

$$$\dot{x}_r = A_r x_r + B_r u, \quad y = C_r x_r + Du$$$

*降阶完成后直接嵌入EMT求解器的核心方程，用于与换流器开关网络进行实时时域步进积分与接口数据交换。*

- **验证结果**: 双频段降阶模型（125阶）在1-100 Hz及1-5 kHz频带内的阻抗频响与1200阶全阶模型高度重合；时域仿真中，直流母线电压、交流侧电流及开关切换瞬态波形与全阶模型误差极小，验证了该方法在兼顾低频机电振荡与高频开关暂态方面的有效性与高精度，同时显著降低计算资源消耗。
