## 方法细节

### 方法概述

本文采用电流注入法与功率解耦控制法相结合的方法论，建立了反映DPFC内部3次谐波能量动态交换的等效数学模型。首先构建包含基波网络和3次谐波网络的等效电压源模型，将并联侧VSC1等效为基波电压源$\dot{V}_{sh1}$、VSC2等效为3次谐波电压源$\dot{V}_{sh3}$，串联侧等效为基波与3次谐波电压源并联组合；进而通过数学推导转换为电流注入模型，实现不增加节点导纳矩阵阶数的潮流计算能力；最后基于瞬时功率理论设计d-q坐标系下的功率解耦控制器，实现以3次谐波能量平衡为基础的串并联变流器协调控制。

### 数学公式


**公式1**: $$$P = \frac{T}{2} V_i I_i \cos\varphi_i$$$

*同频率电压电流产生的有功功率计算公式，用于分析3次谐波功率交换特性（式3）*


**公式2**: $$$P_s - P_{sh} + P_{se} = P_m$$$

*含DPFC系统的有功功率平衡方程，描述线路首末端功率与DPFC交换功率的关系（式4）*


**公式3**: $$$P_{sh} = P_{sh1} + P_{sh,dc}$$$

*并联侧功率分配方程，基波功率与直流电容功率之和（式5）*


**公式4**: $$$P_{sh1} = P_{se3} + P_{se,dc}$$$

*并联侧基波功率与串联侧3次谐波功率及直流电容功率的平衡关系（式6）*


**公式5**: $$$P_{se3} = P_{se1}$$$

*串联侧3次谐波接收功率与基波注入功率的等效关系，体现能量转换守恒（式7）*


**公式6**: $$$\dot{I}_{se1} = \frac{\dot{V}_s - \dot{V}_m}{jX_{se}} + \frac{\dot{V}_{se1}}{jX_{se}}$$$

*串联侧注入基波电流计算，包含线路压差和串联注入电压贡献（式9）*


**公式7**: $$$\begin{cases} \dot{I}_{inj,s1} = -(\dot{I}_{sh1} + \dot{I}_{se1}') \\ \dot{I}_{inj,m1} = \dot{I}_{se1}' \end{cases}$$$

*基波电流注入模型，描述DPFC向线路首末端节点注入的等效基波电流（式13）*


**公式8**: $$$\dot{I}_3 = \frac{\dot{V}_{sh3}}{jX_{sh3}} + \frac{\dot{V}_{se3}}{jX_{se3}}$$$

*3次谐波电流计算，由并联侧和串联侧3次谐波电压共同决定（式14）*


**公式9**: $$$\dot{I}_3 = \frac{\alpha(P_{ref}-P_m) + j\beta(Q_{ref}-Q_m)}{\sqrt{3}X_{sh3}}$$$

*基于潮流目标值的3次谐波电流指令计算，实现有功无功解耦控制（式17，其中$\alpha=K_p+K_i/s$，$\beta=K_p'+K_i'/s$）*


**公式10**: $$$I_{sh,d} = \frac{\text{Re}(\dot{V}_{se1}\dot{I}_L^*)}{V_s} = \frac{1}{V_s}(V_{se1,x}\frac{P_m\cos\theta_m + Q_m\sin\theta_m}{V_m} + V_{se1,y}\frac{Q_m\cos\theta_m - P_m\sin\theta_m}{V_m})$$$

*并联侧d轴电流指令，用于控制有功功率和直流电压（式18）*


**公式11**: $$$I_{sh,q} = (K_p + \frac{K_i}{s})(V_{sref} - V_s)$$$

*并联侧q轴电流指令，用于控制无功功率和母线电压（式19）*


### 算法步骤

1. 建立DPFC等效电压源模型：将并联侧三相变流器VSC1等效为基波电压源$\dot{V}_{sh1}$，单相变流器VSC2等效为3次谐波电压源$\dot{V}_{sh3}$；串联侧单相变流器组等效为基波电压源$\dot{V}_{se1}$与3次谐波电压源$\dot{V}_{se3}$的并联组合

2. 推导内部功率交换方程：基于傅里叶分析建立基波与3次谐波间的功率平衡关系，确保$P_{se3}=P_{se1}$，即串联侧吸收的3次谐波功率完全转换为注入线路的基波功率

3. 构建电流注入模型：通过数学变换将电压源模型转换为向线路首末端节点s和m注入的等效电流源，包括基波分量$\dot{I}_{inj,s1}$、$\dot{I}_{inj,m1}$和3次谐波分量$\dot{I}_{inj,s3}$、$\dot{I}_{inj,m3}$

4. 设计并联侧功率解耦控制器：基于瞬时功率理论进行d-q坐标变换，d轴控制有功功率（维持直流电容电压），q轴控制无功功率（维持母线电压$V_s$为目标值$V_{sref}$）

5. 计算3次谐波电流指令：根据线路潮流目标值$P_{ref}$、$Q_{ref}$与实际值$P_m$、$Q_m$的偏差，通过PI调节器生成3次谐波电流幅值指令，确保串并联侧能量平衡

6. 实现串并联协调控制：并联侧VSC2向系统注入3次谐波电流，经线路传输后被串联侧吸收，维持串联侧直流电容电压恒定，同时串联侧根据潮流控制需求注入基波电压

7. 模型验证与仿真分析：在ADPSS平台分别建立详细电磁暂态模型（微秒级时间常数）和用户自定义等效模型，对比验证3次谐波能量交换动态特性，并在WSCC9系统及500kV实际电网进行稳态功率调控测试


### 关键参数

- **$X_{se}$**: 串联侧等效阻抗（基波）

- **$X_{sh}$**: 并联侧等效阻抗（基波）

- **$X_{se3}$**: 串联侧3次谐波等效阻抗

- **$X_{sh3}$**: 并联侧3次谐波等效阻抗

- **$K_p, K_i$**: 有功功率（d轴）PI控制器比例、积分系数

- **$K_p', K_i'$**: 无功功率（q轴）PI控制器比例、积分系数

- **$V_{sref}$**: 并联侧接入母线电压目标值

- **$P_{ref}, Q_{ref}$**: 线路潮流有功、无功功率目标值

- **$\theta_m$**: 线路末端电压相角

- **$\theta_{se}$**: 串联侧等效电压源相角



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 等效模型与详细电磁暂态模型对比验证 | 在ADPSS电磁暂态仿真平台上，对比了基于详细开关器件的DPFC模型（时间常数微秒级）与所提等效数学模型的3次谐波能量交换动态特性，验证了等效模型对内部能量交换过程的准确复现能力 | 等效模型在保持仿真精度的同时，计算效率显著高于详细开关模型，且可直接应用于大规模电网潮流计算，而详细模型受算法限制仅适用于小规模系统 |

| WSCC9系统稳态功率调控 | 在标准WSCC9节点测试系统中进行稳态潮流控制实验，验证了DPFC等效模型在机电暂态仿真环境下的功率调节能力 | 实现了与理论分析一致的潮流调节效果，证明了模型在标准测试系统中的有效性 |

| 500kV实际电网应用验证 | 在某实际500kV输电网络中建立用户自定义模型，进行稳态功率调控实验，验证模型在高压大电网中的实用性和有效性 | 成功实现了对实际电网潮流的稳态调控，证明了模型从理论到工程应用的适用性 |



## 量化发现

- 详细开关器件电磁暂态模型的时间常数在微秒级，仿真精度高但受限于计算资源，仅能实现小规模系统仿真
- 所建等效模型准确复现了3次谐波在DPFC串并联变流器间的动态能量交换过程，3次谐波功率$P_{se3}$与基波注入功率$P_{se1}$满足$P_{se3}=P_{se1}$的功率平衡关系
- 基于电流注入法的等效模型不增加节点导纳矩阵的阶数，控制简单灵活，适用于大规模电网（如500kV实际电网）的潮流计算
- 并联侧d-q解耦控制实现了有功功率（直流电压）与无功功率（母线电压）的独立调节，$I_{sh,q}>0$时发出无功提升电压，$I_{sh,q}<0$时吸收无功降低电压
- 3次谐波电流指令与线路潮流偏差$(P_{ref}-P_m)$和$(Q_{ref}-Q_m)$成正比，通过PI调节器$\alpha=K_p+K_i/s$和$\beta=K_p'+K_i'/s$实现无静差控制


## 关键公式

### 3次谐波能量平衡方程

$$$P_{se3} = P_{se1}$$$

*描述DPFC串联侧吸收的3次谐波功率与注入线路的基波功率之间的等效转换关系，是协调串并联变流器能量交换的核心约束条件*

### 基波电流注入方程

$$$\dot{I}_{inj,s1} = -(\frac{\dot{V}_{sh1}}{jX_{sh}} + \frac{\dot{V}_{se1}}{jX_{se}})$$$

*将DPFC等效为向线路首端节点注入的电流源，用于潮流计算中不修改节点导纳矩阵的电压源等效方法*

### 3次谐波电流指令方程

$$$\dot{I}_3 = \frac{\alpha(P_{ref}-P_m) + j\beta(Q_{ref}-Q_m)}{\sqrt{3}X_{sh3}}$$$

*基于潮流目标偏差生成3次谐波电流参考值，实现并联侧向串联侧的能量传输控制，确保串联侧直流电容电压恒定*



## 验证详情

- **验证方式**: 对比分析与多场景仿真验证：1）等效模型与详细电磁暂态模型的仿真结果对比；2）标准测试系统（WSCC9）稳态实验；3）实际500kV电网应用验证
- **测试系统**: 1）详细模型对比测试系统；2）WSCC9标准节点系统；3）某500kV实际输电网络
- **仿真工具**: ADPSS（电力系统全数字仿真装置），利用其电磁暂态仿真功能建立详细开关模型，利用用户自定义模型（UDM）功能建立等效数学模型进行机电暂态仿真
- **验证结果**: 所建等效模型能够准确复现详细开关模型在电磁暂态过程中的3次谐波能量交换特性，验证了模型的正确性；在WSCC9系统和500kV实际电网的稳态功率调控实验证明了模型适用于大规模电网潮流计算与稳态功率调控，兼顾了仿真精度与计算效率
