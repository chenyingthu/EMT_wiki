# Development and Validation of a New Detailed EMT-type Component-based Load Model

**作者**: M., Torabi, Milani
**年份**: 2021
**来源**: `13&14/files/j.epsr.2021.107290.pdf.pdf`

## 摘要

本文提出一种基于电磁暂态（EMT）计算的详细元件级综合负荷建模方法。该方法采用“测量-元件”混合策略，首先在EMTP中构建涵盖开关电源、LED驱动、单相/三相感应电机及变频驱动器等现代终端用电设备的精确电路模型，并统一采用标幺值参数化。通过开发分层子电路架构，在顶层掩码中开放母线电压、总负荷需求、各元件百分比占比及变频电机比例等用户自定义接口。仿真时，依据预设比例将各元件模型按比例聚合至配电母线，并结合现场录波数据动态修正电机脱扣与重合闸逻辑及机械负载转矩特性，从而实现对商业与居民负荷在电压暂降等扰动下动态响应的高保真复现。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程上，暂态稳定、故障穿越和电压暂降分析常需要负荷在扰动期间及扰动后的真实动态响应；但传统ZIP、指数负荷或简化综合负荷模型通常把大量终端设备等效成静态阻抗、导纳或恒功率，难以再现电机转速/转矩动态、电力电子整流前端、照明驱动和变频负荷在电压变化下的非线性行为。本文研究对象是商业与居民配电负荷的聚合模型，重点不是单个设备，而是由多类终端元件组合成的母线级负荷。难点在于终端设备类型多、控制和保护逻辑不同，且扰动中电压不平衡、谐波和暂态过程会影响电流与功率恢复。相对已有PSSE CMLD等模型，本文贡献是提出面向EMT计算的详细元件级负荷模型，将三相/单相感应电机、电力电子负荷、变频驱动和高效照明等作为电路级组件聚合，并用Hydro-Quebec电压暂降现场数据校验和修正模型行为。

### 2. 模型、算法与实现技术

本文实现的是一种EMTP中的component-based综合负荷模型。基本思路是先为各类终端负荷建立可在电磁暂态步长下求解的子电路，再按用户给定的负荷组成比例聚合到配电母线。模型的输入接口包括母线电压、总负荷水平、各类终端设备占比以及与电机/驱动相关的配置；输出则是三相电流、有功功率、无功功率及扰动期间的暂态波形。核心状态量随元件而异：感应电机包含电磁与机械动态，电力电子负荷包含整流/直流侧电容电压，照明和开关电源通过等效电阻或驱动电路描述其电压相关耗能特性。文中给出的SMPS/CFL等效电阻公式本质上把直流侧电压映射为瞬时吸收功率或等效耗电支路，使电压暂降时负荷不再被固定为恒功率。实现流程是：建立底层元件模型，统一参数标幺化或容量缩放；在EMTP中封装为层级子电路；按负荷组成自动调整各子模型容量；用现场录得的电压暂降波形驱动模型；比较仿真与实测的电流、功率动态，并据此调整负荷组成和电机连接/重连等经验特性。

### 3. 验证、优势与不足

作者用Hydro-Quebec在实际电网电压暂降事件中记录的现场数据验证模型，仿真工具为EMTP。原文摘要和引言明确说明验证对象覆盖商业和居民负荷，验证方式是把实测电压扰动施加到所建综合负荷模型，比较仿真得到的负荷动态响应与实测记录。可用指标主要是扰动期间及恢复后的三相电流、有功功率、无功功率波形是否能跟随现场记录，以及模型是否能再现电机连接/重连、电力电子负荷非恒功率响应等现象。相对静态ZIP、指数模型和把电力电子简单等效为恒功率的CMLD类做法，优势在于它在EMT层面保留了终端设备差异和电路动态，因此更适合研究电压变化、不平衡扰动和恢复过程中的短时暂态行为。需要注意的是，所给证据没有完整呈现所有算例图表和误差统计；若不查阅原文表图，不能声称某一误差百分比或精度提升倍数。验证边界也主要限于作者掌握的Hydro-Quebec电压暂降录波、商业/居民场景和EMTP离线仿真，不能直接外推到所有馈线结构、所有负荷构成、频率稳定研究或实时仿真步长。

### 4. 价值、认知与可复用场景

这项工作的主要价值在于把“负荷是静态功率需求”的视角推进到“负荷是由可辨识终端设备动态组成的聚合对象”。它提醒后续研究：电压暂降后的电流冲击、功率恢复和负荷损失，可能来自电机保护、电力电子直流侧和照明驱动等具体机制，而不是单一ZIP参数能解释的现象。该模型适合作为EMT级配电负荷建模、现场录波复现实验、综合负荷模型参数辨识、CMLD改进和电压暂降响应分析的入口。它不适合被直接当作通用负荷库或稳定仿真标准模型使用；若用于新的地区、行业负荷、保护设定或控制型电力电子设备，应重新获取负荷构成和现场扰动数据进行校验。

### 证据边界

- 原文明确给出的事实包括：模型为详细EMT-type component-based load model，仿真工具为EMTP，验证数据来自Hydro-Quebec记录的电压暂降事件，目标场景为商业和居民负荷。
- 原文引言明确批评了传统静态模型以及PSSE CMLD中将电力电子近似为恒功率、用代数性能模型近似单相电机的局限；本文相对贡献应限定在这些问题背景下理解。

- SMPS、CFL等效电阻公式属于元件建模机制证据；但不同设备参数、控制器细节和容量缩放规则需要查阅原文完整模型章节后才能严格复现。
- 电机脱扣、重连比例、延时等若来自页面抽取而非所示原文片段，应作为待核对参数使用；没有原文图表支撑时不宜作为通用工程整定值引用。
- 验证范围从已给证据看主要是电压暂降事件下的离线EMTP仿真，尚不能证明该模型适用于所有故障类型、长期动态、频率扰动、实时仿真或高渗透率新型电力电子负荷场景。

## 方法细节

### 方法概述

本文提出一种基于电磁暂态（EMT）计算的详细元件级综合负荷建模方法。该方法采用“测量-元件”混合策略，首先在EMTP中构建涵盖开关电源、LED驱动、单相/三相感应电机及变频驱动器等现代终端用电设备的精确电路模型，并统一采用标幺值参数化。通过开发分层子电路架构，在顶层掩码中开放母线电压、总负荷需求、各元件百分比占比及变频电机比例等用户自定义接口。仿真时，依据预设比例将各元件模型按比例聚合至配电母线，并结合现场录波数据动态修正电机脱扣与重合闸逻辑及机械负载转矩特性，从而实现对商业与居民负荷在电压暂降等扰动下动态响应的高保真复现。

### 数学公式

**公式1**: $$R_{eq} = \frac{v_{dc}^2}{P_{rated}}$$

*开关电源(SMPS)等效电阻计算公式，用于表征直流侧电容电压与额定功率的关系*

**公式2**: $$R_{eq,charging} = 11v_{dc} + 0.021v_{dc}^2 + 1300$$

*紧凑型荧光灯(CFL)充电状态下的等效电阻多项式拟合公式*

**公式3**: $$R_{eq,discharging} = 23.7v_{dc} + 274$$

*紧凑型荧光灯(CFL)放电状态下的等效电阻线性拟合公式*

**公式4**: $$V_{thr} = 2.7\text{ V}, \quad R_d = 0.3\ \Omega$$

*LED线性模型参数，包含阈值电压与动态串联电阻*

### 算法步骤

1. 1. 在EMTP中搭建各终端负荷元件的底层电路模型，包括静态电阻、SMPS、CFL、LED驱动器、三相/单相感应电机及PWM-VSI变频驱动器，并将所有电气参数统一转换为标幺值体系。

2. 2. 构建EMTP分层子电路（Subcircuit）架构，配置顶层用户掩码（Mask），定义输入变量：配电母线电压等级、总负荷有功需求、各元件功率占比、变频驱动电机控制比例。

3. 3. 通过内置脚本读取掩码参数，按百分比权重自动缩放各子电路的容量参数，实现多类型负荷的数学聚合，生成完整的综合负荷节点模型。

4. 4. 导入Hydro-Quebec现场录制的电压暂降波形数据，通过理想受控电压源施加至聚合负荷模型的输入端，复现实际电网扰动。

5. 5. 在时域仿真中嵌入电机动态控制逻辑：当母线电压降至额定值45%~55%且持续1~3个周波时，触发接触器脱扣逻辑切除电机负荷；当电压恢复至约65%额定值后，延时20 ms触发70%电机重合闸。

6. 6. 运行电磁暂态仿真，提取三相电流、有功功率与无功功率的动态时域波形，并与现场实测数据进行谐波频谱、暂态峰值及恢复轨迹的逐点对比。

7. 7. 根据仿真与实测的偏差反馈，迭代调整负荷组成比例、单相电机机械转矩特性（恒转矩/二次方转矩）及脱扣/重合闸时序参数，直至动态响应高度吻合。

### 关键参数

- **C_dc**: 0.036 pu（直流侧电容电抗标幺值，适用于SMPS与LED驱动）

- **LED_threshold_voltage**: 2.7 V

- **LED_dynamic_resistance**: 0.3 Ω

- **motor_dropout_voltage_range**: 45%~55% 额定电压

- **motor_dropout_duration**: 1~3 个工频周波

- **motor_reconnection_delay**: 电压恢复至~65%后 20 ms

- **single_phase_motor_torque_type**: 二次方转矩（Quadratic torque，经实测验证最优）

- **dryer_composition**: 30% 单相感应电机 + 70% 静态电阻

### 仿真测试

| 商业负荷案例（蒙特利尔购物中心） | 预故障负荷1.52 MW，暂降持续0.296 s，三相电压分别跌落67.9%、64.0%、62.8%。仿真准确复现了电机脱扣导致的电流骤降及电压恢复时的冲击电流峰值，三相电流波形与实测谐波含量高度一致。 | 相比传统静态ZIP模型，该模型成功捕捉了电机重合闸产生的暂态冲击电流，有功/无功功率动态轨迹误差显著降低，解决了静态模型无法反映负荷恢复过程的问题。 |

| 居民负荷案例2（含工业电机脱扣场景） | 预故障负荷7 MW，故障持续0.7 s，三相电压跌落52.8%、53.5%、54.7%。A相故障延迟12个周波发生。仿真中1.5 MW工业电机在暂降开始后3个周波脱扣，故障后负荷降至5.5 MW，电流与功率跌落幅度与实测一致。 | 模型准确量化了1.5 MW（约21.4%）工业负荷的脱扣贡献，动态功率曲线与实测记录的匹配度远超PSSE CMLD模型中简化恒功率电力电子假设的仿真结果。 |

## 量化发现

- 电机接触器脱扣阈值精确位于额定电压的45%~55%区间，且需持续1~3个工频周波。
- 电压恢复至约65%额定值后，70%的电机负荷在20 ms内完成重合闸，引发显著的暂态冲击电流。
- 商业案例中，1.52 MW负荷在0.296 s暂降期间三相电压最大跌落达67.9%，仿真电流谐波频谱与实测数据吻合。
- 居民案例2中，7 MW预故障负荷因电压暂降损失1.5 MW工业电机负荷，占比约21.4%，故障持续0.7 s。
- 单相电机采用二次方转矩机械负载模型时，其动态响应与现场录波数据的匹配度显著优于恒转矩模型，验证了转矩特性对暂态仿真的关键影响。

## 关键公式

### SMPS等效电阻模型

$$R_{eq} = \frac{v_{dc}^2}{P_{rated}}$$

*用于在EMTP中表征电视机、电脑等开关电源类负荷的直流侧等效阻抗，实现电压波动下的非线性功率响应计算。*

### CFL充电态等效电阻

$$R_{eq,charging} = 11v_{dc} + 0.021v_{dc}^2 + 1300$$

*用于紧凑型荧光灯在直流电容充电阶段的非线性电阻拟合，精确刻画电压暂降期间照明负荷的瞬态电流特性。*

- **测试系统**: 加拿大蒙特利尔某12.47 kV商业馈线（购物中心）及230/25 kV变电站供电的居民配电区域（含约600户住宅与小型工业）
- **仿真工具**: EMTP（时域电磁暂态仿真软件）
- **验证结果**: 模型在电压暂降及恢复全过程中高精度复现了三相电流、有功与无功功率的动态波形，准确捕捉了电机脱扣、重合闸冲击电流及负荷恢复特性。相比传统静态ZIP模型及PSSE CMLD模型，该EMTP元件级模型消除了将电力电子简化为恒功率负荷的误差，动态响应误差显著降低，验证了其在不对称故障与大电压波动场景下的工程适用性。
