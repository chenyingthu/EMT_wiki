# 39/pes.2011.6039582

**作者**: 
**年份**: 2011
**来源**: `39/pes.2011.6039582.pdf.pdf`

## 摘要

—In the next few years many installations of long EHV AC cables and HVDC links are planned on the French grid. Prior to these huge system changes various EMT studies have to be conducted to prevent problems. Some of these studies require large scale network models. To limit errors when building large networks and to gain time, the French TSO RTE has decided to develop an interface between system tools and EMTP. This XML based interface enables SCADA data to be automatically imported into an EMTP graphical user interface. This paper describes the environment developed in EMTP-RV, imported data from SCADA to EMTP and studies performed with the full models. Simulation speed, data availability and model reduction issues are addressed in this paper.
