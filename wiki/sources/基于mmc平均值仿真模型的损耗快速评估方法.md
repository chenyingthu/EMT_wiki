# Fast Loss Evaluation Method Based on MMC Average Simulation Model; 基于MMC平均值仿真模型的损耗快速评估方法

**作者**: Sun, Hao
**年份**: 2024
**来源**: `19、20、21/EMT_task_19/Sun和Hao - 2024 - Fast Loss Evaluation Method Based on MMC Average Simulation Model; [基于 MMC 平均值仿真模型的损耗快速评估方法].pdf`

## 摘要

提出一种基于MMC平均值模型的损耗快速在线评估方法。首先基于确定的调制策略和电容电压平衡策略，构建开关频率曲面以反映任意稳态工况下的开关频率；其次利用平均值模型计算开关频率上限和开关损耗功率上限；然后结合开关频率曲面数据，通过比例关系近似快速计算实际开关损耗；最后提出基于衰减函数的开关损耗注入方法，利用桥臂串联受控电压源吸收的能量表征损耗，避免产生注入电压尖峰并改善注入效果。验证信息：模块化多电平换流器(MMC)系统，包含6个桥臂，每个桥臂由N个半桥子模块(HBSM)和桥臂电抗器组成，采用三相结构；在PSCAD中搭建计及损耗注入的MMC平均值仿真模型，验证了所提方法在损耗计算精度和计算效率方面的优势。通过与详细模型对比，验证了开关频率曲面的准确性和损耗注入方法的有效性，实现了损耗的快速在线评估和对仿真结果的实时表征

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求是：在MMC-HVDC等工程中，换流器损耗直接影响散热设计、器件选型以及运行经济性/可靠性评估，但详细EMT模型要跟踪数百个子模块的开关状态，计算慢；解析法虽快，却难准确处理调制和电容电压平衡策略引起的开关损耗。研究对象是采用半桥子模块的三相MMC，重点计算IGBT和二极管的导通损耗、开关损耗，并把损耗影响注入平均值仿真模型。难点在于：开关损耗不仅由桥臂电流和电容电压决定，还由每个控制周期子模块投入/切除顺序决定，而这些状态在平均值模型中被等效掉了。本文的创新是把“开关频率曲面”作为平均值模型与真实投切行为之间的桥梁：先在确定调制和电容电压平衡策略下离线/预处理获得全稳态工况的开关频率，再在线由平均值模型计算开关频率上限和开关损耗上限，用频率比例近似恢复实际开关损耗；同时用桥臂串联受控电压源吸收能量表征损耗，并通过衰减函数注入开关损耗以避免电压尖峰。

### 2. 模型、算法与实现技术

方法由三部分组成。第一部分是损耗器件模型：IGBT和二极管导通损耗用线性通态压降模型计算，即通态压降由等效电阻项和固定压降项组成；开通、关断、反向恢复能量则由器件手册曲线拟合成关于电流的二次函数，并按子模块电容电压与参考电压的比例修正。第二部分是开关损耗快速计算算法。作者先基于给定调制策略和电容电压平衡策略构建半平均值模型，用它收集不同稳态工况下的实际开关频率，二维插值得到开关频率曲面。在线运行时，平均值模型提供桥臂电流、子模块投入数n(t)、电容电压等接口量；根据相邻控制周期投入数变化计算理论最大投切次数，得到开关频率上限f_max，并结合电流方向、开关事件类型和器件能量曲线求得开关损耗功率上限P_sw_max。随后由当前工况查表得到f_real，用P_sw≈(f_real/f_max)P_sw_max估计实际开关损耗。第三部分是损耗注入：导通损耗和开关损耗被转化为桥臂串联受控电压源吸收的能量。由于直接按P_loss/i_arm给电压可能在电流过零或开关损耗集中注入时产生尖峰，作者提出衰减函数，使开关损耗在后续时间内平滑释放，从而让平均值模型既保留快速性，又能反映损耗对电压、电流和能量平衡的影响。

### 3. 验证、优势与不足

作者在PSCAD/EMTDC中建立计及损耗注入的MMC平均值仿真模型，并与能反映子模块开关动态的详细模型进行对比验证。测试对象为三相MMC，包含6个桥臂，每个桥臂由N个半桥子模块和桥臂电抗器串联组成。验证内容主要包括：一是用半平均值模型生成的开关频率曲面是否能反映不同稳态工况下的实际开关频率；二是基于频率比例的开关损耗计算结果与详细模型后处理结果是否一致；三是加入桥臂串联受控电压源和衰减函数后，损耗注入是否能避免传统直接注入方式可能产生的电压尖峰，并改善仿真中的损耗表征。优势体现在机制层面：平均值模型不再显式模拟大量IGBT/二极管开断过程，在线计算只需桥臂平均量、投入数变化和查表插值；开关频率曲面把调制策略和电容均压策略的影响预先编码进数据表，因而比完全解析估算更贴近实际投切行为；损耗注入使损耗不只是离线统计量，而能进入EMT模型能量平衡。限制是：当前证据摘要未给出可核验的误差、耗时、步长或工况网格数量等数值；方法依赖预先确定的调制和电容电压平衡策略，策略改变后频率曲面需重建；验证集中在稳态工况和PSCAD仿真模型，不能直接外推到故障暂态、闭锁过程、不同子模块拓扑或硬件实时平台。

### 4. 价值、认知与可复用场景

这项工作的关键认知是：MMC平均值模型虽然丢失了单个子模块开关状态，但可通过“开关频率曲面+开关损耗上限”的方式补回开关损耗统计信息，从而在不运行详细开关模型的情况下进行在线损耗评估。它适合用于MMC平均值EMT仿真、换流站损耗在线估算、含损耗能量平衡分析、散热或器件选型前期筛算，以及后续研究中需要把调制/均压策略影响压缩为查表数据的建模页面。它不适合被直接当作任意MMC拓扑、任意控制策略或故障暂态下的通用损耗模型；若换用全桥子模块、新型均压策略、不同PWM/NLM实现或考虑结温耦合，应重新校准器件损耗曲线和开关频率曲面。

## 方法细节

### 方法概述

提出一种基于MMC平均值模型的损耗快速在线评估方法。首先基于确定的调制策略和电容电压平衡策略，构建开关频率曲面以反映任意稳态工况下的开关频率；其次利用平均值模型计算开关频率上限和开关损耗功率上限；然后结合开关频率曲面数据，通过比例关系近似快速计算实际开关损耗；最后提出基于衰减函数的开关损耗注入方法，利用桥臂串联受控电压源吸收的能量表征损耗，避免产生注入电压尖峰并改善注入效果。

### 数学公式

**公式1**: $$$P_{T_{cond}} = U_{CE}i_{CE} = (R_{T0}i_{CE} + U_{CE0})i_{CE}$$$

*IGBT通态损耗功率计算，采用线性近似模型，其中$R_{T0}$为通态电阻，$U_{CE0}$为通态电压偏置*

**公式2**: $$$P_{D_{cond}} = U_{D}i_{F} = (R_{D0}i_{F} + U_{D0})i_{F}$$$

*二极管通态损耗功率计算，采用线性近似模型*

**公式3**: $$$E_{on}(i_{CE}) = (a_1i_{CE}^2 + b_1i_{CE} + c_1)\frac{u_c}{u_{ref}}$$$

*IGBT开通损耗能量计算，基于数据手册曲线拟合的二次函数模型*

**公式4**: $$$E_{off}(i_{CE}) = (a_2i_{CE}^2 + b_2i_{CE} + c_2)\frac{u_c}{u_{ref}}$$$

*IGBT关断损耗能量计算*

**公式5**: $$$E_{rec}(i_{F}) = (a_3i_{F}^2 + b_3i_{F} + c_3)\frac{u_c}{u_{ref}}$$$

*二极管反向恢复损耗能量计算*

**公式6**: $$$P_{sw} \approx \frac{f_{real}}{f_{max}}P_{sw_{max}}$$$

*实际开关损耗快速计算核心公式，基于开关频率比例关系，$f_{real}$为实际开关频率，$f_{max}$为开关频率上限，$P_{sw_{max}}$为开关损耗功率上限*

**公式7**: $$$f_{max} = \frac{\sum(n(t)_{(1-0)} + n(t)_{(0-1)})}{2N(t_2-t_1)}$$$

*开关频率上限计算，基于相邻控制周期内子模块最大投切变化次数，$n(t)_{(1-0)}$和$n(t)_{(0-1)}$分别为投入到切除和切除到投入的子模块数*

### 算法步骤

1. 基于确定的调制策略（如最近电平调制NLM或载波移相调制CPS-PWM）和电容电压平衡策略（如排序算法），构建半平均值模型用于收集开关频率数据

2. 通过半平均值模型在全工况范围内网格化运行，获得各网格点处的开关频率数据，采用二维插值方法生成连续的开关频率曲面$f_{surf}(P,Q)$或$f_{surf}(M,\cos\varphi)$

3. 利用MMC平均值模型实时计算当前工况下的桥臂电流波形和子模块投入数目$n(t)$

4. 基于当前控制周期和上一控制周期的子模块投入数目$n(t)$和$n(t-\Delta T)$，根据最大投切原则计算开关频率上限$f_{max}$：若$n(t) \geq N-n(t-\Delta T)$，则$n(1-0)=N-n(t)$，$n(0-1)=N-n(t-\Delta T)$；否则$n(1-0)=n(t-\Delta T)$，$n(0-1)=n(t)$

5. 根据电流方向和开关次数计算开关损耗能量上限$E(t)_{pa_{sw_{max}}}$，并求平均得到开关损耗功率上限$P_{sw_{max}}$

6. 根据当前运行工况（有功功率、无功功率或调制比、功率因数角）查询开关频率曲面获得实际开关频率$f_{real}$

7. 利用比例关系$P_{sw} = (f_{real}/f_{max})P_{sw_{max}}$快速计算实际开关损耗

8. 根据桥臂电流方向和子模块投入数目计算导通损耗$P_{cond}$

9. 将总损耗$P_{loss} = P_{cond} + P_{sw}$通过受控电压源以衰减函数形式注入桥臂：$u_{inj}(t) = P_{loss}(t)/i_{arm}(t) \cdot g(t)$，其中$g(t)$为设计的衰减函数，避免电压尖峰

### 关键参数

- **调制策略**: 最近电平调制(NLM)或载波移相调制(CPS-PWM)

- **电压平衡策略**: 基于排序的电容电压平衡策略

- **桥臂子模块数**: N

- **控制周期**: $\Delta T$

- **等效电容**: $C_{eq}$

- **桥臂电抗器**: $L_a$

- **IGBT通态电阻**: $R_{T0}$

- **IGBT通态电压偏置**: $U_{CE0}$

- **二极管通态电阻**: $R_{D0}$

- **参考截止电压**: $u_{ref}$

- **开关损耗拟合参数**: $a_i, b_i, c_i$ (i=1,2,3)

## 量化发现

- 开关损耗计算精度：通过开关频率曲面结合比例关系$P_{sw}/P_{sw_{max}} = f_{real}/f_{max}$，实现开关损耗的高精度近似计算
- 计算效率提升：基于平均值模型的损耗评估方法避免了对数百个子模块开关动态的详细仿真，计算速度显著优于详细电磁暂态模型
- 存储需求：开关频率曲面通过二维插值生成，仅需存储网格点处的开关频率数据，数据量远小于详细仿真时域波形数据
- 注入稳定性：基于衰减函数的损耗注入方法有效抑制电压尖峰，改善注入效果，确保仿真稳定性
- 适用范围：方法适用于任意稳态工况，不受单位功率因数限制，具有通用性

## 关键公式

### 开关损耗快速计算比例公式

$$$P_{sw} \approx \frac{f_{real}}{f_{max}}P_{sw_{max}}$$$

*在获得开关频率上限$P_{sw_{max}}$和实际开关频率$f_{real}$后，用于快速计算实际开关损耗，避免详细统计开关次数*

### 开关频率上限计算

$$$f_{max} = \frac{\sum(n(t)_{(1-0)} + n(t)_{(0-1)})}{2N(t_2-t_1)}$$$

*基于平均值模型输出，假设每个控制周期子模块投切状态尽可能改变（即上一周期投入的优先切除，切除的优先投入），计算理论最大开关频率*

### 开关损耗能量拟合公式

$$$E_{on/off/rec}(i) = (ai^2 + bi + c)\frac{u_c}{u_{ref}}$$$

*基于器件数据手册，通过二次曲线拟合获得开关损耗能量与电流、电压的关系，用于计算开关损耗上限*

### 桥臂平均通态损耗

$$$P_{pa_{cond}} = \frac{1}{t_2-t_1}\int_{t_1}^{t_2}\{P_{T_{cond}}[\rho_1n_{pa} + \rho_2(N-n_{pa})] + P_{D_{cond}}[\rho_2n_{pa} + \rho_1(N-n_{pa})]\}d\tau$$$

*根据子模块投入数目$n_{pa}$和电流方向系数$\rho_1, \rho_2$计算桥臂在一个周期内的平均通态损耗*
