## 方法细节

### 方法概述

本文提出了一种基于拉普拉斯域长卷积和并行处理的电磁暂态仿真新方法。该方法首先在拉普拉斯域建立系统的节点导纳矩阵，利用Kron降阶法将网络矩阵缩减至仅包含开关节点和观测节点的低维形式。通过数值拉普拉斯逆变换(INLT)将降阶后的阻抗矩阵转换到时域，采用辅助电流源模拟开关操作（闭合/断开）引起的暂态事件。核心计算负担来自长卷积运算，通过多相正交镜像滤波器组(QMF)结构实现并行计算，从而支持超实时仿真。该方法避免了传统有理函数逼近导致的无源性破坏问题，适用于统计性绝缘协调研究等需要大量重复仿真的场景。

### 数学公式


**公式1**: $$$I = Y_{bus} V$$$

*节点导纳方程，描述网络在拉普拉斯域的电压电流关系*


**公式2**: $$$Y_{red} = Y_{11} - Y_{12}Y_{22}^{-1}Y_{21}$$$

*Kron降阶公式，将高维节点导纳矩阵缩减为仅含关注节点的低维矩阵*


**公式3**: $$$Z_{red} = [Y_{red}]^{-1}$$$

*降阶阻抗矩阵，用于计算暂态响应*


**公式4**: $$$Z_{SS} = -Z_{red}Y_{12}Y_{22}^{-1}$$$

*稳态阻抗矩阵，用于计算初始稳态工作点*


**公式5**: $$$V_1 = V_{SS} + V_{TR}$$$

*总电压响应分解为稳态分量和暂态扰动分量*


**公式6**: $$$V_{SS} = Z_{SS}I_2$$$

*稳态电压响应计算，其中I2为外部注入电流*


**公式7**: $$$V_{TR} = Z_{red}I_1$$$

*暂态电压响应计算，由开关操作引起的电流注入产生*


**公式8**: $$$i_{close}(t) = -v_{k,l}(t)u(t-t_{close})/R_{sw}$$$

*开关闭合操作的诺顿等效电流源模型*


**公式9**: $$$i_{open}(t) = -i_{k,l}(t)u(t-t_{open})$$$

*开关断开操作的辅助电流源模型*


**公式10**: $$$V_1^{(p)} = Z_{red}^{(p)}I_1^{(p)} + V_1^{(p-1)}$$$

*第p次开关操作后的电压响应递推公式*


**公式11**: $$$v_1^{(p)} = z^{(p)} * i_{SW}^{(p)} + v_1^{(p-1)}$$$

*时域卷积形式的暂态响应计算，*表示卷积运算*


**公式12**: $$$N_t = \omega_{max} \times T_{max} / 2\pi$$$

*数值拉普拉斯变换所需的采样点数计算公式*


### 算法步骤

1. 网络矩阵构建与降阶：建立完整网络的节点导纳矩阵$Y_{bus}$，通过Kron降阶法将其分块为$Y_{11}$（含开关和观测节点）、$Y_{12}$、$Y_{21}$、$Y_{22}$，计算降阶矩阵$Y_{red} = Y_{11} - Y_{12}Y_{22}^{-1}Y_{21}$，并求逆得到$Z_{red}$

2. 稳态初始化：根据外部注入电流$I_2$和稳态阻抗矩阵$Z_{SS} = -Z_{red}Y_{12}Y_{22}^{-1}$，计算初始稳态电压$V_{SS} = Z_{SS}I_2$，此时开关状态对应的$I_1$为零向量

3. 开关事件建模：对于开关闭合操作，计算辅助电压源$v_{close}(t) = -v_{k,l}(t)u(t-t_{close})$并转换为诺顿等效电流源$i_{close}(t) = v_{close}(t)/R_{sw}$，其中$R_{sw}$取极小值（对应$G_{sw}=10^6$ S）；对于开关断开操作，直接注入抵消电流$i_{open}(t) = -i_{k,l}(t)u(t-t_{open})$

4. 拉普拉斯域响应计算：根据开关操作修改$Y_{red}$（最多修改4个元素）并重新求逆得到新的$Z_{red}^{(p)}$，提取与开关操作相关的列组合成$Z^{(p)}$表征网络对该操作的响应

5. 时域转换与预处理：利用逆数值拉普拉斯变换(INLT)将$Z^{(p)}$转换到时域得到$z^{(p)}$，设定最大频率$\omega_{max}$和仿真时长$T_{max}$，计算所需采样点数$N_t$

6. 并行卷积计算：采用多相QMF滤波器组结构，将长卷积$z^{(p)} * i_{SW}^{(p)}$分解为多个并行子滤波器运算，各子滤波器处理不同频段或时间窗的信号分量

7. 序列事件处理：对于多开关操作序列($p=1,2,...,P$)，通过递推公式$v_1^{(p)} = z^{(p)} * i_{SW}^{(p)} + v_1^{(p-1)}$逐步计算，利用叠加原理累积各事件的影响

8. 结果合成与输出：将各并行通道的计算结果合并，加上稳态分量$v_{SS}$，得到完整的暂态电压响应波形


### 关键参数

- **开关等效电导**: $G_{sw} = 10^6$ S（理想开关近似）

- **额定角频率**: $\omega_0 = 100\pi$ 或 $120\pi$ rad/s（对应50Hz或60Hz系统）

- **阻尼常数**: $c$（拉普拉斯域复变量$s = c + j\omega$的实部）

- **最大仿真频率**: $\omega_{max}$（决定暂态现象的最高频率分量）

- **最大仿真时间**: $T_{max}$（单次仿真的时间窗长度）

- **采样点数**: $N_t = \omega_{max} \times T_{max} / 2\pi$

- **降阶后节点数**: $N_1$（原网络$N$个节点经Kron降阶后保留的节点数，$N_1 \ll N$）

## 量化发现

- 计算复杂度降低：通过Kron降阶将原网络节点数$N$缩减至$N_1$（仅含开关和观测节点），矩阵求逆计算量从$O(N^3)$降至$O(N_1^3)$，其中$N_1 \ll N$（典型配电网中$N$可达数千而$N_1$仅数十）
- 仿真加速比：利用多相QMF滤波器组并行计算长卷积，实现百倍于实时的超实时仿真速度（>100x real-time），适用于需要数百次重复仿真的统计性暂态分析
- 开关建模精度：采用$G_{sw} = 10^6$ S模拟理想开关，引入的数值误差可控，与PSCAD/EMTDC对比结果高度一致
- 数值稳定性：避免了传统有理函数逼近（vector fitting）导致的无源性破坏（passivity violation）问题，保证了宽频带内的数值稳定性
- 算法局限性：当前实现不适用于长时程仿真（时间窗>$1$秒）和详细电力电子器件（如IGBT详细模型）的仿真
- 内存需求：长卷积计算需要存储历史数据，内存占用与$N_t = \omega_{max}T_{max}/2\pi$成正比，限制了可仿真的时间窗长度


## 关键公式

### Kron网络降阶公式

$$$Y_{red} = Y_{11} - Y_{12}Y_{22}^{-1}Y_{21}$$$

*将大规模网络导纳矩阵缩减为仅含关注节点（开关和观测点）的低维矩阵，是降低计算维度的核心步骤*

### 时域卷积递推公式

$$$v_1^{(p)} = z^{(p)} * i_{SW}^{(p)} + v_1^{(p-1)}$$$

*计算第p次开关操作后的暂态响应，通过卷积运算将网络 impulse response 与开关电流源波形结合，支持多事件序列的逐步计算*

### 稳态阻抗计算

$$$Z_{SS} = -Z_{red}Y_{12}Y_{22}^{-1}$$$

*用于计算初始稳态工作点，将外部电源$I_2$映射到关注节点的稳态电压，支持谐波稳态初始化（多频率点计算）*



## 验证详情

- **验证方式**: 对比验证（与商业仿真软件PSCAD/EMTDC及传统拉普拉斯变换方法对比）
- **测试系统**: 17节点配电网络测试系统，包含多段线路、负载和开关设备，用于验证开关操作引起的电磁暂态现象
- **仿真工具**: PSCAD/EMTDC（作为参考基准）、MATLAB（算法原型实现）、FPGA开发板（并行架构硬件验证）
- **验证结果**: 所提方法在17节点系统上的计算结果与PSCAD/EMTDC及传统数值拉普拉斯变换具有高度一致性，验证了算法的准确性；通过多相QMF并行结构实现了超实时仿真（数百倍实时速度），证明了该方法在统计性暂态分析（如绝缘协调研究）中的应用潜力；FPGA实现验证了并行计算架构的硬件可行性，但指出当前方法限于短时暂态（<1s）和宏观开关建模，不适用于详细电力电子仿真
