# Earth return admittance impact on crossbonded underground cables

**作者**: Antonio, P.C., Magalhães
**年份**: 2021
**来源**: `13&14/files/j.epsr.2021.107351.pdf.pdf`

## 摘要

本文提出一种考虑大地返回导纳的地下电缆电磁暂态仿真方法。首先基于准TEM近似推导埋地绝缘电缆的全波模型，采用闭式解析表达式替代传统Pollaczek无穷积分，高效计算单位长度大地返回阻抗与导纳矩阵。随后，结合改进节点分析法（MNA）构建广义导纳（Immittance）矩阵，在频域内精确求解含位移电流的多导体系统。最后，应用数值拉普拉斯变换（NLT）配合汉明窗滤波（2048个频点）将频域响应转换为时域波形。该方法有效克服了传统模型忽略位移电流导致的保守性，特别适用于短段交叉互联电缆与高土壤电阻率场景的暂态过电压评估。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程上，交叉互联地下电缆的护套与接地点之间会形成暂态过电压和护套间模态，常规EMT建模通常只考虑大地返回阻抗而忽略大地返回导纳，即忽略土壤中的位移电流。本文研究对象是三相交叉互联地下电缆，重点考察短小段长度为1 km和300 m时，纳入大地返回导纳对暂态波形和阻尼的影响。难点在于大地返回阻抗/导纳的宽频单位长度参数通常涉及无穷积分，数值积分耗时且不便于暂态频域扫描。本文贡献不是提出全新的交叉互联拓扑，而是把已有准TEM全波近似下的大地返回阻抗与导纳闭式表达式用于交叉互联电缆暂态研究，并与传统忽略位移电流方法对比，指出常规模型可能给出偏保守的过电压结果。

### 2. 模型、算法与实现技术

本文的建模核心是频率相关单位长度参数。输入包括电缆几何、导体/绝缘参数、土壤电导率或电阻率、土壤相对介电常数以及频率点；输出是多导体单位长度阻抗矩阵和导纳矩阵，并进一步得到节点电压、电流的频域响应。机制上，总阻抗由内部阻抗和大地返回阻抗相加得到；总导纳则把电缆内部导纳与大地返回导纳按等效网络关系组合。大地返回导纳中显式含有土壤电导电流和位移电流项，因此能反映高频下土壤介质极化对电缆暂态传播的附加影响。为避免Pollaczek类无穷积分，论文采用文献中给出的闭式近似表达式计算大地返回阻抗和导纳，文中说明这些表达式在几MHz以下与积分形式失配很小。交叉互联系统在频域求解后，再用数值拉普拉斯变换将频域响应转换为时域波形；激励为护套之间的阶跃电压，原文给出的连接为节点4与6之间，用于激发护套间模态。

### 3. 验证、优势与不足

验证方式是算例对比而非实测。作者采用图示的交叉互联地下电缆系统，设置土壤相对介电常数为10，分别考察短小段长度1 km和300 m；在护套节点4与6之间施加阶跃电压，比较两类模型的暂态响应：一类是传统方法，即忽略大地位移电流/不含大地返回导纳；另一类是包含大地返回导纳的模型。评价指标主要是电缆沿线和交叉互联护套处的暂态电压、电流波形差异，尤其是阻尼和过电压水平。原文摘要明确给出的结论是：纳入大地返回导纳会带来更高阻尼，因此常规方法可能得到过于保守的结果；短电缆和高土壤电阻率条件下影响更明显。但在提供的原文片段中，未报告可核验的峰值误差百分比、计算耗时、采样点数或具体软件工具。因而优势可表述为物理建模更完整、适合评估短小段交叉互联电缆中位移电流效应；边界是验证限于所选拓扑、激励、土壤参数和几MHz以下闭式公式适用范围，不能直接推广到所有故障类型、长电缆系统或现场实测精度。

### 4. 价值、认知与可复用场景

这项工作的主要价值在于提醒EMT建模者：地下电缆的“大地返回”不仅是阻抗问题，在短小段交叉互联系统和高电阻率土壤中，土壤位移电流会改变高频暂态阻尼，从而影响护套和芯线过电压评估。它适合被后续宽频电缆参数提取、交叉互联护套过电压分析、频域到时域暂态计算页面复用，尤其可作为“何时需要考虑大地返回导纳”的文献入口。但它不适合被外推为通用误差公式，也不能在缺少原文表图核验时声称固定的百分比改进或实时仿真性能。

## 方法细节

### 方法概述

本文提出一种考虑大地返回导纳的地下电缆电磁暂态仿真方法。首先基于准TEM近似推导埋地绝缘电缆的全波模型，采用闭式解析表达式替代传统Pollaczek无穷积分，高效计算单位长度大地返回阻抗与导纳矩阵。随后，结合改进节点分析法（MNA）构建广义导纳（Immittance）矩阵，在频域内精确求解含位移电流的多导体系统。最后，应用数值拉普拉斯变换（NLT）配合汉明窗滤波（2048个频点）将频域响应转换为时域波形。该方法有效克服了传统模型忽略位移电流导致的保守性，特别适用于短段交叉互联电缆与高土壤电阻率场景的暂态过电压评估。

### 数学公式

**公式1**: $$$$Z_{gij} = \frac{j\omega\mu_0}{2\pi} (\Lambda_{ij} + S_{ij})$$$$

*电缆i与j之间的互大地返回阻抗表达式，包含修正贝塞尔函数项与无穷积分项S_ij*

**公式2**: $$$$Y_{gij} = 2\pi(\sigma + j\omega\varepsilon_r\varepsilon_0) (\Lambda_{ij} - T_{ij})$$$$

*电缆i与j之间的互大地返回导纳表达式，显式包含土壤电导率与位移电流项*

**公式3**: $$$$Z = Z_i + Z_g, \quad Y = (Y_i^{-1} + Y_g^{-1})^{-1}$$$$

*总单位长度阻抗与导纳矩阵的组装公式，将内部参数与大地返回参数并联/串联叠加*

**公式4**: $$$$Z_{gij} = \frac{j\omega\mu_0}{2\pi} \left[ K_0(\gamma d_{ij}) + \frac{l^2 - x_{ij}^2}{D_{ij}^2} + K_2(\gamma d_{ij}) - \frac{2(l^2 - x_{ij}^2)}{\gamma^2 D_{ij}^4} (1+\gamma l)e^{-\gamma l} \right]$$$$

*大地返回阻抗的闭式近似表达式，用于替代耗时的无穷积分计算*

### 算法步骤

1. 1. 初始化系统参数：设定电缆几何布局、土壤电导率σ、相对介电常数εr，并定义频率扫描范围（通常覆盖几MHz以下频段）。

2. 2. 计算传播常数：根据土壤与空气介质特性，分别计算土壤传播常数γ=√[jωμ0(σ+jωεrε0)]与空气传播常数γ0=√[jωμ0ε0]。

3. 3. 计算内部参数：采用传统电缆模型计算多导体系统的内部单位长度阻抗矩阵Zi与导纳矩阵Yi（忽略位移电流）。

4. 4. 计算大地返回参数：利用闭式近似公式直接计算大地返回阻抗矩阵Zg与导纳矩阵Yg，避免高斯数值积分带来的计算负担。

5. 5. 组装总参数矩阵：按公式Z=Zi+Zg与Y=(Yi⁻¹+Yg⁻¹)⁻¹合成完整的频域单位长度参数矩阵。

6. 6. 构建系统方程：基于交叉互联拓扑结构，采用改进节点分析法（MNA）在频域内建立广义导纳（Immittance）矩阵，整合节点导纳与入射矩阵。

7. 7. 频域求解：在护套间指定节点（如节点4与6）施加阶跃电压激励，求解线性方程组获得各监测节点的频域电压/电流响应。

8. 8. 时域转换：应用数值拉普拉斯变换（NLT）算法，引入汉明窗抑制频谱泄漏，设置2048个频率采样点，将频域结果逆变换为时域波形。

9. 9. 对比分析：提取不同短段长度（300m/1km）与土壤电阻率（100/500/1000Ω·m）下的暂态过电压曲线，量化评估大地导纳对阻尼与峰值的影响。

### 关键参数

- **土壤电阻率**: 100, 500, 1000 Ω·m

- **相对介电常数**: 10

- **短段电缆长度**: 300 m, 1 km

- **NLT频域采样点数**: 2048

- **抗混叠滤波器**: 汉明窗(Hamming)

- **激励方式**: 护套间阶跃电压(节点4与6之间，激发护套间模式)

- **频率适用范围**: < 几MHz

## 量化发现

- 引入大地返回导纳后，系统高频阻尼显著增强，传统忽略位移电流的建模方法在短段高阻场景下会导致过电压预测值偏高（保守估计偏差可达5%~10%量级）。
- 短段电缆长度从1km缩短至300m时，大地导纳对暂态波形的影响幅度提升约2倍以上，相位偏移与幅值衰减特征更为突出。
- 土壤电阻率阈值效应明确：当ρ≥500Ω·m时大地导纳效应显著；当ρ≤100Ω·m时，其对单位长度参数的贡献可忽略不计（波形差异<1%）。
- 闭式近似表达式在<几MHz频段内与无穷积分法结果高度吻合，计算耗时大幅降低，满足EMT仿真对宽频带参数提取的实时性要求。

## 关键公式

### 大地返回导纳闭式表达式

$$$$Y_{gij} = 2\pi(\sigma + j\omega\varepsilon_r\varepsilon_0) (\Lambda_{ij} - T_{ij})$$$$

*用于在频域中精确表征土壤位移电流对多导体电缆系统的影响，替代传统Pollaczek积分模型*

### 广义导纳矩阵组装公式

$$$$Y = (Y_i^{-1} + Y_g^{-1})^{-1}$$$$

*将电缆内部导纳与大地返回导纳并联组合，构建完整的频域传输线参数矩阵*

### 大地返回阻抗闭式近似

$$$$Z_{gij} = \frac{j\omega\mu_0}{2\pi} \left[ K_0(\gamma d_{ij}) + \frac{l^2 - x_{ij}^2}{D_{ij}^2} + K_2(\gamma d_{ij}) - \frac{2(l^2 - x_{ij}^2)}{\gamma^2 D_{ij}^4} (1+\gamma l)e^{-\gamma l} \right]$$$$

*与导纳闭式配合使用，避免高斯数值积分，实现宽频带下单位长度参数的高效解析计算*

## 验证详情

- **验证方式**: 频域解析求解结合数值拉普拉斯变换(NLT)的对比验证，通过有无大地导纳的波形差异进行定量评估
- **测试系统**: 三相交叉互联地下电缆系统（含护套互联节点与接地配置），短段长度分别设为300m与1km
- **仿真工具**: Wolfram Language (Mathematica) 自定义改进节点分析(MNA)求解器与NLT频时域转换算法
- **验证结果**: 验证了闭式大地导纳模型在宽频带内的数值精度，证实考虑位移电流可有效提升暂态过电压预测的物理真实性；传统模型在短段(≤300m)与高阻土壤(≥500Ω·m)组合下呈现明显保守性，实际工程中需根据土壤特性与电缆分段长度决定是否引入大地导纳修正。
