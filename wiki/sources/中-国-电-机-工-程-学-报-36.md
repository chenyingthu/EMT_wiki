## 方法细节

### 方法概述

本文提出了一种基于随机微分方程(SDE)的电力电子化电力系统随机电磁暂态仿真算法。该方法在保留传统EMTP算法框架的基础上，通过建立考虑参数随机迁移的动态伴随电路模型，实现了对系统内部参数随机迁移（乘性激励）和外部随机扰动的统一仿真。核心创新在于将元件参数（R、L、C）的随机迁移过程建模为SDE，采用隐式Milstein格式（梯形或后向）进行离散化，构建相应的动态伴随电路模型，使其与EMTP的节点导纳法求解框架直接兼容。该方法特别针对电力电子器件频繁开关导致的电磁暂态时间尺度特性，处理了系统刚性与随机性相互作用的问题。

### 数学公式


**公式1**: $$$d\eta(t) = \alpha(\eta, t) dt + \beta(\eta, t) dW(t), \quad \eta(t_0) = \eta_0$$$

*随机微分方程的一般矢量形式，描述系统状态的偏移过程与扩散过程，其中$W(t)$为$n_w$维独立标准Wiener过程*


**公式2**: $$$\eta_n = \eta_{n-1} + [(1-\theta)\alpha(\eta_{n-1}) + \theta\alpha(\tilde{\eta}_n)]\Delta t + D(\eta_{n-1})$$$

*隐式Milstein格式离散方程，当$\theta=0.5$时为梯形Milstein格式，$\theta=1$时为后向Milstein格式，$D(\cdot)$为扩散算子*


**公式3**: $$$D(\eta_{n-1}) = \beta(\eta_{n-1})\Delta W_{n-1} + \frac{1}{2}\beta(\eta_{n-1})\beta'(\eta_{n-1})(\Delta W_{n-1}^2 - \Delta t)$$$

*Milstein格式的扩散算子，包含Wiener过程的一阶和二阶项，确保1阶强收敛精度*


**公式4**: $$$i_{L,n} = \frac{\Delta t}{L_n + \alpha(\tilde{L}_n)\Delta t}(v_{i,n} - v_{j,n}) + K_{A,n}i_{L,n-1} - \frac{K_{A,n}i_{L,n-1}}{L_{n-1}}D(L_{n-1})$$$

*电感元件的后向Milstein格式动态伴随电路模型，$K_{A,n} = L_n/(L_n + \alpha(\tilde{L}_n)\Delta t)$，体现了电感参数随机迁移对电流的乘性激励效应*


**公式5**: $$$G_{L,n} = \Delta t/[2L_n + \alpha(\tilde{L}_n)\Delta t]$$$

*梯形Milstein格式下电感动态伴随电路的等效导纳，实现了与EMTP节点导纳矩阵的直接集成*


**公式6**: $$$i_{R,n} = \frac{1}{R_n}(v_{i,n} - v_{j,n}) = G_{R,n}(v_{i,n} - v_{j,n})$$$

*电阻元件的随机伴随电路模型，其中$R_n$通过Milstein格式离散更新：$R_n = R_{n-1} + \alpha(\tilde{R}_n)\Delta t + D(R_{n-1})$*


**公式7**: $$$dv_C(t) = \frac{i_C(t) - v_C(t)\alpha(C(t))}{C(t)}dt - \frac{v_C(t)\beta(C(t))}{C(t)}dW(t)$$$

*电容参数随机迁移下的电压微分方程，体现了电容值变化与电压的耦合关系及能量变化特性*


### 算法步骤

1. 初始化：设定仿真参数（步长$\Delta t$、总时长）、系统拓扑、元件初始参数$R_0, L_0, C_0$及SDE的偏移函数$\alpha(\cdot)$和扩散函数$\beta(\cdot)$

2. 节点导纳矩阵构建：根据当前时刻元件参数$R_n, L_n, C_n$，计算各元件动态伴随电路的等效导纳$G_{R,n}, G_{L,n}, G_{C,n}$，组装系统节点导纳矩阵$Y_n$

3. 历史电流源计算：基于前一时刻状态变量（电感电流$i_{L,n-1}$、电容电压$v_{C,n-1}$）和参数$L_{n-1}, C_{n-1}$，计算各元件的历史电流源$I_{hist,n-1}$和随机电流源$I_{rand,n-1}$（含$D(\cdot)$项）

4. 网络方程求解：求解线性方程组$Y_n V_n = I_{hist,n-1} + I_{rand,n-1}$，得到当前时刻节点电压$V_n$

5. 支路量更新：根据节点电压计算各支路电流$i_{R,n}, i_{L,n}, i_{C,n}$及电容电压$v_{C,n}$

6. 参数随机迁移更新：采用隐式Milstein格式（梯形或后向）更新元件参数：$L_n = L_{n-1} + [(1-\theta)\alpha(L_{n-1}) + \theta\alpha(\tilde{L}_n)]\Delta t + D(L_{n-1})$，其中预估点$\tilde{L}_n$由Euler-Maruyama格式计算

7. 时间推进：$n \leftarrow n+1$，判断仿真是否结束，否则返回步骤2


### 关键参数

- **$\Delta t$**: 仿真计算步长，需满足数值稳定性条件

- **$\theta$**: 隐式Milstein格式参数，$\theta=0.5$为梯形格式，$\theta=1$为后向格式

- **$\alpha(\cdot)$**: SDE偏移过程函数，描述参数确定性漂移

- **$\beta(\cdot)$**: SDE扩散过程函数，描述参数随机波动强度

- **$\Delta W_{n-1}$**: Wiener过程增量，满足$\Delta W_{n-1} \sim \mathcal{N}(0, \Delta t)$

- **$K_{A,n}, K_{B,n}$**: 电感动态伴随电路的系数，$K_{A,n} = L_n/(L_n + \alpha(\tilde{L}_n)\Delta t)$，$K_{B,n} = 2L_n/(2L_n + \alpha(\tilde{L}_n)\Delta t)$



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 简单RLC串联电路 | 在RLC电路中引入电感和电容参数的随机迁移，采用后向Milstein格式进行仿真。结果显示参数随机迁移导致的电流电压轨迹呈现随机波动特性，与确定性EMTP仿真结果相比，轨迹包络线能够覆盖确定性结果。蒙特卡罗多轨迹统计结果显示均值与确定性结果高度一致 | 与EMTP确定性计算结果相比，均值误差极小；与蒙特卡罗多轨迹仿真结果相比，单条轨迹能准确反映随机动态过程，无需大量样本统计 |

| 三相两电平VSC电路 | 在电压源换流器(VSC)系统中考虑直流侧电容和交流侧电感的参数随机迁移。仿真结果显示所提算法能够准确捕捉电力电子开关动作与参数随机迁移的相互作用，输出电压电流波形在保持EMTP精度的同时体现了参数不确定性带来的波动范围 | 与蒙特卡罗方法相比，计算效率显著提升（避免多轨迹重复计算），同时保持了与EMTP框架的直接兼容性，可在PSCAD/EMTDC、RTDS等平台实现 |



## 量化发现

- Milstein格式（包括隐式梯形和后向格式）具有1阶强收敛精度，显著优于Euler-Maruyama格式的1/2阶收敛性
- 隐式Milstein格式中的梯形和后向形式在数值稳定性方面表现优于显式Heun格式，特别是在处理乘性噪声时稳定性不受步长限制
- 所提动态伴随电路模型实现了与EMTP算法框架的直接兼容，无需修改节点导纳法的基本结构
- 对于电感元件，参数随机迁移引入的乘性激励通过项$\frac{K_{A,n}i_{L,n-1}}{L_{n-1}}D(L_{n-1})$体现，其幅值与电流大小成正比
- 仿真算法能够处理微秒级电磁暂态时间尺度与参数随机迁移慢动态过程的强刚性耦合问题


## 关键公式

### 电感元件动态伴随电路通用形式

$$$i_{L,n} = G_{L,n}(v_{i,n} - v_{j,n}) + I_{L,hist,n-1} + I_{L,rand,n-1}$$$

*将电感支路表示为诺顿等效电路，其中$G_{L,n}$为等效导纳，$I_{L,hist,n-1}$为确定性历史电流源，$I_{L,rand,n-1}$为随机历史电流源，实现与EMTP节点法的兼容*

### 隐式Milstein离散格式

$$$\eta_n = \eta_{n-1} + [(1-\theta)\alpha(\eta_{n-1}) + \theta\alpha(\tilde{\eta}_n)]\Delta t + \beta(\eta_{n-1})\Delta W_{n-1} + \frac{1}{2}\beta(\eta_{n-1})\beta'(\eta_{n-1})(\Delta W_{n-1}^2 - \Delta t)$$$

*用于离散化元件参数（R、L、C）的随机微分方程，是算法实现1阶强收敛精度的核心，其中$\theta=0.5$时为梯形格式，兼顾精度与稳定性*



## 验证详情

- **验证方式**: 对比验证：与EMTP确定性仿真结果对比验证均值准确性；与蒙特卡罗多轨迹仿真结果对比验证随机动态轨迹的正确性
- **测试系统**: 1) 简单RLC串联电路，用于验证基本元件模型；2) 三相两电平VSC（电压源换流器）电路，用于验证电力电子化系统的适用性
- **仿真工具**: 基于EMTP算法框架实现，兼容PSCAD/EMTDC、RTDS等电磁暂态仿真平台；采用SUNDIALS工具集进行算法对比测试
- **验证结果**: 所提动态伴随电路模型和随机电磁暂态仿真算法能够准确高效地反映参数随机迁移扰动下的系统运行特性。与EMTP确定性计算相比，均值轨迹高度一致；与蒙特卡罗多轨迹结果相比，单条轨迹统计特性吻合，验证了算法对EMTP框架的直接兼容性和正确性。算法实现了1阶强收敛精度，数值稳定性优于显式格式
