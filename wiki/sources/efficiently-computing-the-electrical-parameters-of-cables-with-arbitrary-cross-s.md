## 方法细节

### 方法概述

提出基于二维矩量法(MoM)的准静电场离散框架，用于高效提取任意截面地下电缆的并联导纳矩阵$[Y]$。该方法将电缆横截面的电容计算严格解耦为内部区域$[C^i]$与外部区域$[C^e]$两个独立静电场问题。内部问题通过自由空间格林函数将拉普拉斯方程转化为边界积分方程，采用一维直线单元离散任意形状导体边界，利用分段常数基函数展开面电荷密度，并通过点匹配法（Delta测试函数）在单元形心处构建线性代数方程组(SLAE)。外部区域利用同轴圆柱几何的解析解直接计算。结合绝缘介质的复介电常数合成复电容矩阵，进而导出频域导纳矩阵。串联阻抗矩阵$[Z]$则沿用前期优化的SVS-EFIE（频率>100Hz）或V-EFIE（频率≤100Hz）MoM方法。通过解析处理自作用奇异积分、采用高斯-勒让德数值积分计算互作用项，并利用导纳矩阵MoM求解的频率无关特性，实现全频域参数在常规计算机上的快速提取。

### 数学公式


**公式1**: $$$\nabla_t^2 \phi(\mathbf{r}) = 0, \quad \mathbf{E}_t = -\nabla_t \phi$$$

*准静电场拉普拉斯方程，描述导体外部横截面电势分布，是提取导纳矩阵的基础控制方程。*


**公式2**: $$$\sum_{\alpha'=1}^n \int_{\Gamma_{\alpha'}} G(\mathbf{r}, \mathbf{r}') q_{\alpha'}(\mathbf{r}') dl' = V_\alpha$$$

*基于格林定理导出的边界积分方程，将电势与导体表面电荷密度关联，用于MoM离散化。*


**公式3**: $$$[Z^{\alpha,\alpha'}]_{m,m'} = \int_0^1 G(\mathbf{r}_m^\alpha(1/2), \mathbf{r}_{m'}^{\alpha'}(t)) dt$$$

*MoM阻抗矩阵元素定义，表示源单元对测试单元形心的电势贡献，需处理奇异积分。*


**公式4**: $$$[\hat{C}] = (\varepsilon_i [C^i] + \varepsilon_e [C^e]) + \frac{1}{j\omega\varepsilon_0}(\sigma_i [C^i] + \sigma_e [C^e])$$$

*复电容矩阵合成公式，结合内外区域自由空间电容与介质复介电常数，直接决定导纳矩阵。*


**公式5**: $$$[Y] = j\omega[\hat{C}] = [G] + j\omega[C]$$$

*并联导纳矩阵最终表达式，由复电容矩阵乘以角频率得到，包含电导与电容分量。*


### 算法步骤

1. 1. 几何建模与边界离散：根据电缆实际截面（任意形状内导体+圆形外护套），将每个导体边界划分为$M_\alpha$个一维直线段单元，记录顶点坐标与单元长度。

2. 2. 建立积分方程：引入自由空间二维格林函数$G(\mathbf{r}, \mathbf{r}') = -\frac{1}{2\pi}\ln|\mathbf{r}-\mathbf{r}'|$，将拉普拉斯方程转化为关于表面电荷密度$q(\mathbf{r})$的Fredholm第二类积分方程。

3. 3. MoM基函数展开：在每个直线单元上采用分段常数基函数$P_m^\alpha(\mathbf{r})$展开未知电荷密度，将连续积分方程转化为离散求和形式。

4. 4. 点匹配测试：在每个单元形心处放置Delta测试函数$T_m^\alpha(\mathbf{r}) = \delta(\mathbf{r} - \mathbf{r}_m^\alpha(1/2))$，将积分方程转化为$N \times N$线性代数方程组(SLAE) $[Z][q] = [V]$。

5. 5. 矩阵元素计算：对于自作用项（源与测试在同一单元），解析求解$-\ln|\mathbf{r}-\mathbf{r}'|$的奇异积分；对于互作用项，采用$K=6$点高斯-勒让德数值积分计算。

6. 6. 求解电荷分布：依次设置第$\alpha$个导体电位为1V，其余导体为0V，求解SLAE得到各单元电荷密度系数$q_m^\alpha$。

7. 7. 计算内部电容矩阵：对每个导体表面的电荷密度沿边界积分$[C^i]_{\alpha,\alpha'} = \sum_{m=1}^{M_\alpha} L_m^\alpha q_m^\alpha$，得到自由空间内部电容矩阵。

8. 8. 计算外部电容矩阵：利用同轴圆柱解析公式$C^e = \frac{2\pi\varepsilon_0}{\ln(r_g) - \ln(r_{se})}$构建外部电容矩阵$[C^e]$，仅对角线元素非零。

9. 9. 合成导纳矩阵：代入介质复介电常数$\hat{\varepsilon} = \varepsilon + \frac{\sigma}{j\omega\varepsilon_0}$，按公式合成复电容矩阵$[\hat{C}]$，最终计算$[Y] = j\omega[\hat{C}]$。

10. 10. 阻抗矩阵联合计算：对同一几何模型，采用优化的SVS-EFIE/V-EFIE MoM求解准静磁问题得到$[Z]$，组合$[Z]$与$[Y]$输出全频域传输线参数。


### 关键参数

- **导体材料**: 铜，电阻率 $\rho = 1.68 \times 10^{-8} \ \Omega\cdot\text{m}$

- **绝缘材料**: 交联聚乙烯(XLPE)，相对介电常数 $\varepsilon_r = 2.3$，损耗角正切 $\tan\delta = 0$

- **数值积分阶数**: 高斯-勒让德积分点数 $K = 6$

- **电缆长度**: 20 km

- **仿真时间步长**: $10 \ \mu\text{s}$

- **总仿真时长**: 0.02 s

- **开路测试接地电阻**: $1 \ \text{k}\Omega$

- **短路测试接地电阻**: $1 \ \text{m}\Omega$

- **频率切换阈值**: 100 Hz（高于此值用SVS-EFIE，低于等于此值用V-EFIE）

## 量化发现

- 传统圆形截面近似假设在计算非圆形电缆$[Y]$和$[Z]$矩阵时会导致超过5%的系统性误差，本方法通过精确几何离散彻底消除该误差。
- 导纳矩阵的MoM求解过程与频率完全无关，单次离散矩阵可重复用于全频域计算，使$[Y]$提取速度比$[Z]$提取快10倍以上。
- 时域EMTP仿真验证表明，采用本方法提取的参数可使暂态波形误差控制在10%~30%的波形容限内，幅值误差<5%，满足电力系统高精度暂态分析标准。
- 采用6点高斯-勒让德积分结合奇异项解析处理，在保证矩阵元素计算精度的同时，将数值积分计算量降低至可接受水平，支持常规计算机快速求解。
- 外部电容解析解$C^e = \frac{2\pi\varepsilon_0}{\ln(r_g) - \ln(r_{se})}$的引入，使多层同轴结构的计算复杂度几乎不增加，算法具备极强的工程扩展性。


## 关键公式

### 准静电边界积分方程

$$$\sum_{\alpha'=1}^n \int_{\Gamma_{\alpha'}} G(\mathbf{r}, \mathbf{r}') q_{\alpha'}(\mathbf{r}') dl' = V_\alpha$$$

*用于将拉普拉斯方程转化为可离散求解的积分形式，是MoM提取内部电容矩阵的核心控制方程。*

### 复电容矩阵合成公式

$$$[\hat{C}] = (\varepsilon_i [C^i] + \varepsilon_e [C^e]) + \frac{1}{j\omega\varepsilon_0}(\sigma_i [C^i] + \sigma_e [C^e])$$$

*在获得自由空间内外电容矩阵后，结合介质复介电常数计算实际工况下的复电容，直接决定导纳矩阵的频响特性。*

### 并联导纳矩阵表达式

$$$[Y] = j\omega[\hat{C}] = [G] + j\omega[C]$$$

*EMTP传输线模型必需的输入参数之一，由复电容矩阵直接导出，包含电导$[G]$与电容$[C]$分量。*

### MoM阻抗矩阵元素计算公式

$$$[Z^{\alpha,\alpha'}]_{m,m'} = -\frac{L_m^\alpha}{2\pi}\ln|\mathbf{r}_m^\alpha(1/2) - \mathbf{r}_{m'}^{\alpha'}(t_q)| + \sum_{k=1}^K w_k G(\mathbf{r}_m^\alpha(1/2), \mathbf{r}_{m'}^{\alpha'}(t_k))$$$

*用于构建SLAE，包含奇异项解析解与高斯数值积分项，确保自作用与互作用电势贡献的高精度计算。*



## 验证详情

- **验证方式**: 频域数值对比（与商业FEM软件及经典闭式近似公式对比）与时域电磁暂态仿真验证（开路/短路阶跃响应测试）
- **测试系统**: 20km长多导体地下电缆系统，包含管道型、扇形及实际非对称4导体截面，导体为铜，绝缘为XLPE
- **仿真工具**: 自定义二维MoM求解代码、商业有限元软件(FEM)、PSCAD/EMTDC电磁暂态仿真平台
- **验证结果**: 频域结果与FEM高度吻合，显著优于传统近似公式；时域PSCAD仿真中，首端电流与末端感应电压波形无畸变，幅值与相位误差均控制在工程容限内（仿真误差<0.1%，幅值误差<5%），验证了所提参数提取方法在EMTP应用中的高精度与高可靠性。
