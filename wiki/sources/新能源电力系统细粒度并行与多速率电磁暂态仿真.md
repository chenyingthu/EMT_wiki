# 新能源电力系统细粒度并行与多速率电磁暂态仿真

**作者**: 王啟国1, 徐晋1, 汪可友1 等
**年份**: 2024
**来源**: `19、20、21/EMT_task_19/Wang 等 - 2024 - Fine-grained Parallel and Multi-rate Electromagnetic Transient Simulation for Power System with Rene.pdf`

## 摘要

随着可再生能源的快速发展，电力系统设备类型越来越多，系统振荡特征越来越复杂，对电磁暂态仿真的精度和效率提出了更高要求。基于大规模集成电路设计中所使用的延迟插入法（LIM），提出了新能源电力系统的细粒度建模方法，并结合图形处理器（GPU）的资源优势，实现了算法的并行求解。所提方法将传统交流电网与电力电子设备进行解耦，并基于混合数值稳定性判据和局部截断误差的方法确定了各子系统的步长。然后，通过插值实现了新能源电力系统的多速率仿真。最后，基于 GPU 硬件平台，以含新能源接入的改进 39 节点系统为例验证了所提方法的精度，并以不同规模的新能源接入、不同仿真步长的组合验证了所提方法在仿真效率方面的优势。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自高比例新能源接入后的电磁暂态仿真：一方面，变换器开关、高频控制和场站内部相互作用要求小步长、细模型；另一方面，大电网、同步机、变压器、线路和负荷又使系统规模迅速增大。研究对象是含传统交流电网与直驱风机/变换器等电力电子设备的新能源电力系统EMT仿真。难点在于传统NAM/LU类方法受节点导纳矩阵求解和稀疏性影响，分网并行又要处理子系统规模均衡、接口解耦和多步长交互。本文的贡献不是简单做GPU加速，而是把集成电路中的延迟插入法LIM引入电力系统细粒度建模，将三相节点、三相支路作为并行单元，并通过受控源扩展使同步机、变压器、变换器等非标准LIM设备接入统一框架；同时用新能源出口滤波电容解耦常规动态系统与快速动态系统，再结合稳定性判据、局部截断误差和插值实现多速率求解。

### 2. 模型、算法与实现技术

方法的核心是把电力网络改写为LIM可计算的节点拓扑和支路拓扑。LIM要求节点含电容、支路含电感，采用蛙跳法后，节点电压在半步时刻更新，支路电流在整步时刻更新：节点电压只依赖本节点历史电压和相邻支路电流，支路电流只依赖本支路历史电流及两端节点电压，因此天然适合在GPU上按节点/支路分配线程。为覆盖电力系统设备，作者将LIM中的独立电压源、独立电流源扩展为受控源：输电线路、变压器、同步发电机、负荷、电机和变换器可通过等效受控电流源或受控电压源嵌入上层LIM框架。接口量是新能源出口滤波电容电压，输入为网侧变换器交流侧电流和电网侧支路电流，式(3)用电容电流积分更新接口电压，再反馈给两侧子系统。快速动态系统用小步长h求解电力电子部分，常规动态系统用mh步长求解交流电网；当两侧步长不同步时，通过插值获得接口变量估计值，从而在不让整个系统都采用最小步长的情况下保持耦合计算。

### 3. 验证、优势与不足

作者在GPU硬件平台上，以含新能源接入的改进39节点系统验证方法。测试对象包括传统交流电网中的同步发电机、变压器、输电线路、负荷，以及新能源场站中的直驱风机、机侧/网侧变换器和控制系统。验证思路包括两类：一是与传统EMT仿真结果对比，用于检查LIM细粒度模型和接口解耦后的波形精度；二是在不同新能源接入规模、不同多速率步长组合下测试计算效率，用于说明GPU细粒度并行和多速率策略的效率优势。基线主要是传统EMT/NAM类求解思路或传统仿真结果，指标从摘要和抽取文本看包括精度一致性和仿真效率。优势在机制上体现为：节点和支路更新互不形成全局矩阵求解依赖，更匹配GPU；常规电网不必随电力电子开关全程采用最小步长；接口电容提供了物理上可解释的解耦变量。但当前页面未给出可核验的加速比、误差、GPU型号、步长组合、插值阶次和稳定性阈值；因此不能据此声称具体速度提升或误差水平。从验证范围看，结论主要限于改进39节点及其新能源接入场景，尚不能外推到任意拓扑、故障类型、控制策略、实时仿真硬件或极端高频振荡场景。

### 4. 价值、认知与可复用场景

这项工作的关键认知是：大规模新能源EMT仿真的瓶颈不一定只能通过更快的矩阵分解解决，也可以通过重构计算图，把网络拆成可并行的物理节点/支路状态更新，并用实际滤波电容作为多速率接口。它适合用于后续研究中比较LIM与NAM、MATE、节点分裂、长线解耦等并行EMT方法，也可作为GPU化详细新能源场站仿真、多速率接口设计、受控源统一建模的文献入口。工程上，它适合启发含大量变换器设备的离线EMT加速仿真。它不适合被直接外推为所有电力系统实时仿真方案，也不能在缺少原文参数和数值结果时作为定量性能证据。

## 方法细节

### 方法概述

本文提出基于延迟插入法(LIM)的新能源电力系统细粒度建模与多速率并行仿真方法。该方法将系统解耦为常规动态系统（传统交流电网，含输电网络、变压器、同步发电机、负荷）和快速动态系统（电力电子设备，含直驱风机、变换器及控制系统），以新能源出口滤波电容作为解耦接口。通过将LIM基本拓扑中的独立源拓展为受控源，实现了非LIM形式设备的接入。采用蛙跳法(leapfrog)进行离散化，利用GPU实现节点-支路级的细粒度并行求解。基于混合数值稳定性判据和局部截断误差确定子系统步长（常规系统步长mh，快速系统步长h），通过插值实现多速率数据交互。

### 数学公式

**公式1**: $$$$V_{a,n+1/2} = \frac{1}{hG_a + C_a} \left[ C_a V_{a,n-1/2} - h \sum_{o=1}^{M_a} (I_{ao,n} - I_{a,n}) \right]$$$$

*LIM节点电压更新方程，采用蛙跳法离散化，节点电压更新仅依赖本地历史电压和相邻支路电流，支持并行计算*

**公式2**: $$$$I_{ab,n+1} = \left(1 - \frac{h}{L_{ab}}R_{ab}\right) I_{ab,n} + \frac{h}{L_{ab}}(V_{a,n+1/2} - V_{b,n+1/2} + V_{ab,n+1/2})$$$$

*LIM支路电流更新方程，支路电流更新仅依赖本地历史电流和两端节点电压，与节点电压计算解耦*

**公式3**: $$$$V_{Cf,n+1/2} = V_{Cf,n-1/2} + h C_{if}^{-1} (I_{gsc,n} + I_{b,n})$$$$

*接口电容电压更新方程，作为常规动态系统与快速动态系统的解耦接口，$C_{if}$为接口电容对角矩阵，$I_{gsc}$为变换器侧电流，$I_b$为电网侧电流*

**公式4**: $$$$I_{GT,n+1} = A_1 I_{GT,n} + A_2 V_{GT,n+1/2}$$$$

*同步发电机及其出口变压器(GT)的受控源模型，等效为受控电流源接入LIM框架，$A_1$、$A_2$为系数矩阵，$I_{GT}$包含三相电流和转子电流*

**公式5**: $$$$I_{gsc,n+1} = I_{gsc,n} + hB_1 A_3 (B_2 V_{Cf,n+1/2} - V_{gsc,n+1/2})$$$$

*网侧变换器交流侧电流更新方程，$B_1$、$B_2$为转换矩阵，$A_3$为系数矩阵，$V_{gsc}$为等效受控电压源向量*

**公式6**: $$$$\begin{cases} V_{gsc,n+1/2} = B_3 S B_4 V_{dc,n-1/2} + B_5 S B_6 V_{dc,n+1/2} \\ I_{dcgsc,n+1} = S I_{gsc,n+1} \end{cases}$$$$

*变换器受控源更新方程，$S$为开关控制信号矩阵，$B_3$-$B_6$为转换矩阵，实现直流侧与交流侧的解耦计算*

### 算法步骤

1. 初始化系统参数：设置快速动态系统步长$h$，常规动态系统步长$mh$（$m$为整数倍率），初始化节点电压$V_{a,1/2}$、支路电流$I_{ab,0}$、接口电容电压$V_{Cf,1/2}$等状态变量

2. 时间循环开始：设置仿真总步数$n_{max}$，当前时间步$n=1$

3. 常规动态系统判断：检查是否满足$n = w \cdot m$（$w$为正整数），若满足则执行步骤4，否则跳转至步骤5

4. 常规动态系统求解：基于LIM框架求解输电网络（π型等效）、变压器（漏抗+理想变压器）、同步发电机（受控电流源模型$I_{GT}$）、负荷（恒阻抗RL模型），采用步长$mh$更新节点电压和支路电流

5. 快速动态系统求解：并行求解各新能源单元，包括直驱永磁同步电机（GT支路形式）、机侧/网侧变换器（受控源模型），采用步长$h$更新状态变量，变换器根据开关信号$S$更新受控源值

6. 接口变量更新：根据式(3)更新接口电容电压$V_{Cf}$，获取网侧变换器电流$I_{gsc}$和电网侧电流$I_b$，计算新的接口电压

7. 数据交互与插值：将更新的接口电压$V_{Cf}$反馈至两侧子系统，常规系统通过插值获取快速系统的估计值，实现多速率数据交换

8. 时间推进：$n = n + 1$，若$n > n_{max}$则结束仿真，否则返回步骤3

### 关键参数

- **h**: 快速动态系统（电力电子设备）仿真步长，需满足开关器件高频动作过程的精度要求

- **m**: 多速率步长比，常规动态系统步长与快速动态系统步长之比（$m \geq 1$的整数）

- **C_if**: 接口滤波电容对角矩阵，作为系统解耦的关键元件参数

- **M_a**: 与节点$a$相邻的支路数量，决定节点电压计算时的电流求和项数

- **S**: 开关控制信号矩阵$S = [S_a, S_b, S_c]^T$，表征三相桥臂开关状态（上桥臂导通为1，下桥臂导通为0）

## 量化发现

- 细粒度建模将系统解耦为三相节点/支路级并行单元，使得仿真效率不随系统规模增长而显著变化，解决了传统分网并行方法中子系统规模均匀分配的难题
- 基于混合数值稳定性判据（结合LIM稳定性条件和局部截断误差）确定子系统步长，确保多速率仿真的数值稳定性
- 接口电容参数和仿真步长的选取满足解耦有效性条件，保证常规动态系统（步长$mh$）与快速动态系统（步长$h$）的交互精度
- GPU并行实现使得节点电压更新和支路电流更新完全并行化，每个节点/支路对应独立的GPU线程，充分发挥细粒度并行硬件优势

## 关键公式

### LIM节点电压更新方程

$$$$V_{a,n+1/2} = \frac{1}{hG_a + C_a} \left[ C_a V_{a,n-1/2} - h \sum_{o=1}^{M_a} (I_{ao,n} - I_{a,n}) \right]$$$$

*在每个仿真步长内并行更新所有节点电压，是细粒度并行的核心，适用于常规动态系统的输电网络节点*

### 接口电容解耦方程

$$$$V_{Cf,n+1/2} = V_{Cf,n-1/2} + h C_{if}^{-1} (I_{gsc,n} + I_{b,n})$$$$

*作为常规动态系统与快速动态系统的解耦界面，通过滤波电容电压实现两侧独立求解和多速率仿真*
