# A Testing Tool for Converter-Dominated Power System: Stochastic Electromagnetic Transient Simulation

**作者**: Chen 等
**年份**: 2022
**来源**: `04/Chen 等 - 2022 - A Testing Tool for Converter-Dominated Power System Stochastic Electromagnetic Transient Simulation-1.pdf`

## 摘要

本文提出一种面向变流器主导电力系统的随机电磁暂态（EMT）仿真测试工具。核心方法是将线性集总元件（R/L/C）的参数随机迁移过程建模为随机微分方程（SDE），采用隐式Milstein数值格式（后向与梯形）进行离散化求解。基于EMTP框架的伴随电路原理，推导出含参数迁移的动态伴随电路模型，将其等效为时变电导并联历史电流源与随机电流源。结合离散仿真数值稳定性与连续系统稳定性分析，设计兼容EMTP的数值求解算法，并引入节点重分配与矩阵分解技术加速网络方程求解，最终封装为C语言程序包，实现参数随机迁移与系统电磁暂态过程的同步高精度仿真。

## 研究解读

### 1. 需求、对象、挑战与贡献

变流器主导电力系统中，线路、滤波器和无源元件参数会因环境、老化、工况和制造偏差发生不确定迁移；这些迁移会与电力电子控制环、锁相环、保护和网络暂态相互耦合。实际需求不是只做一次确定性EMT仿真，而是在EMT时间尺度上同时观察“参数随机变化过程”和“被激发的系统动态过程”。难点在于：传统EMTP伴随电路假定元件参数在步长内确定，随机参数若简单外部抽样会破坏网络方程结构，且随机微分方程离散还可能带来数值稳定性问题。本文贡献在于把R/L/C等集总元件的参数迁移建模为SDE，并把离散后的随机项嵌入EMTP伴随电路，使随机参数更新、节点方程求解和控制系统动态可以在同一时步内协同计算。

### 2. 模型、算法与实现技术

本文的基础模型是参数迁移SDE：dX(t)=α(X,t)dt+β(X,t)dW(t)，其中X对应R(t)、L(t)、C(t)，α描述漂移趋势，β描述随机扩散强度，W(t)为维纳过程。算法上，作者采用后向和梯形形式的Milstein离散格式更新参数数值近似，例如通过ΔW和二阶Itô-Taylor修正项刻画随机扰动对下一步参数的影响。随后继承EMTP伴随电路思想，把含参数迁移的电阻、电感、电容改写成节点导纳形式：支路电流等于等效电导乘端电压差，再叠加历史电流源和随机电流源。这样，随机性不是作为仿真后处理，而是直接进入每个时间步的网络方程。核心接口量包括节点电压vi,n、vj,n，支路电流iR,n/iL,n/iC,n，参数近似Rn/Ln/Cn，以及全局节点导纳矩阵GE、GT。实现上，作者设计了与EMTP框架兼容的数值算法，并封装为C程序包；流程是生成维纳增量、更新元件参数、构造动态伴随电路、组装节点方程、求解节点电压并推进历史状态。

### 3. 验证、优势与不足

作者用两个算例验证工具：并网三相两电平整流器和两端直流配电系统。验证目标是观察所提随机EMT工具能否同时模拟元件参数迁移及其激发的系统动态，并用于反映不同控制子系统及其协调在极端情况下的实际表现。原文摘要明确给出测试系统和C程序包实现，但在当前证据中未报告可核验的运行时间、误差百分比、加速比或与蒙特卡洛/商业EMTP工具的定量对比结果。论文还结合离散仿真的数值稳定性和含参数迁移连续系统的稳定性进行算法设计，说明其不是简单随机抽样叠加，而是在伴随电路层面保证网络求解结构一致。优势主要体现在机制兼容性：随机R/L/C可转化为标准节点导纳矩阵可处理的等效支路，便于嵌入已有EMTP框架。边界也很清楚：验证范围限于文中两个系统和作者设定的参数迁移模型；未能由当前文本推出其适用于所有变流器拓扑、所有控制策略、强非线性器件、实时仿真步长或任意随机过程。

### 4. 价值、认知与可复用场景

这项工作的认知价值在于把“参数不确定性”从离线场景抽样推进到EMT步进求解内部，使研究者能分析随机参数迁移如何在毫秒甚至更快暂态尺度上触发或改变控制动态。它适合被后续关于随机EMT、变流器控制鲁棒性测试、直流配电系统参数扰动分析、EMTP伴随电路扩展方法的页面复用，尤其适合作为“如何把SDE嵌入节点导纳求解”的方法入口。它不适合被外推为已证明的通用效率提升工具，也不能替代针对具体设备模型、故障类型、控制器实现和硬件实时平台的专项验证。

## 方法细节

### 方法概述

本文提出一种面向变流器主导电力系统的随机电磁暂态（EMT）仿真测试工具。核心方法是将线性集总元件（R/L/C）的参数随机迁移过程建模为随机微分方程（SDE），采用隐式Milstein数值格式（后向与梯形）进行离散化求解。基于EMTP框架的伴随电路原理，推导出含参数迁移的动态伴随电路模型，将其等效为时变电导并联历史电流源与随机电流源。结合离散仿真数值稳定性与连续系统稳定性分析，设计兼容EMTP的数值求解算法，并引入节点重分配与矩阵分解技术加速网络方程求解，最终封装为C语言程序包，实现参数随机迁移与系统电磁暂态过程的同步高精度仿真。

### 数学公式

**公式1**: $$$dX(t) = \alpha(X,t)dt + \beta(X,t)dW(t)$$$

*参数随机迁移SDE模型，用于描述R/L/C等元件参数受漂移项与扩散项影响的连续随机过程*

**公式2**: $$$R_n = R_{n-1} + \alpha(\tilde{R}_n)\Delta t + \beta(R_{n-1})\Delta W_{n-1} + \frac{\beta(R_{n-1})\beta'(R_{n-1})}{2}\Delta D_{n-1}$$$

*后向Milstein离散格式，用于高精度数值逼近时变电阻参数，包含二阶Itô-Taylor展开项*

**公式3**: $$$i_{k,n} = G_{k,n}(v_{i,n}-v_{j,n}) + I_{khist} + I_{krand}$$$

*动态伴随电路通用迭代方程，将含参数迁移的L/C元件转化为标准EMTP节点导纳形式*

**公式4**: $$$i_{L,n} = \frac{H_{B,n}}{E_{B,n}}i_{L,n-1} + \frac{\Delta t}{E_{B,n}}v_{i,n}$$$

*RL支路离散数值稳定性迭代式，用于分析误差传播特性并推导稳定收敛条件*

### 算法步骤

1. 初始化系统拓扑、元件初始参数及随机过程漂移项$\alpha(\cdot)$与扩散项$\beta(\cdot)$，设定仿真步长$\Delta t$；

2. 在每个时间步$\tau_n$，生成独立维纳过程增量$\Delta W_{n-1} \sim N(0, \Delta t)$及二次变差项$\Delta D_{n-1} = \Delta W_{n-1}^2 - \Delta t$；

3. 利用选定的Milstein格式（后向或梯形）更新各元件的时变参数$R_n, L_n, C_n$，其中预测值通过Euler-Maruyama格式获取；

4. 根据更新后的参数计算动态伴随电路的等效电导$G_{L,n}/G_{C,n}$、历史电流源$I_{hist}$（依赖上一时刻状态）及随机电流源$I_{rand}$（依赖扩散过程）；

5. 将动态伴随电路接入网络，组装全局节点导纳矩阵$G_E$或$G_T$，形成线性代数方程组；

6. 应用节点重分配策略优化矩阵稀疏结构，并采用LU/Cholesky矩阵分解技术对导纳矩阵进行预分解，加速节点电压求解；

7. 根据求得的节点电压更新各支路电流与状态变量，保存历史项与随机项，推进至下一时间步直至仿真结束。

### 关键参数

- **$\Delta t$**: 离散化仿真步长，需满足数值稳定性条件

- **$\alpha(\cdot)$**: 随机扰动的漂移过程系数，决定参数迁移的确定性趋势

- **$\beta(\cdot)$**: 随机扰动的扩散过程系数，决定参数波动的随机强度

- **$W(t)$**: 多维独立维纳过程（布朗运动），满足$W(t)-W(0) \sim N(0,t)$

- **$G_E, G_T$**: 随机EMT仿真中使用的节点导纳矩阵，分别对应后向与梯形积分格式

## 量化发现

- 后向与梯形Milstein格式在满足稳定性条件$|H_{B,n}/E_{B,n}| < 1$时可保证离散仿真数值稳定，避免舍入误差累积放大
- 确定性仿真在参数随机扰动处于合理分布范围（如工程典型$\pm 5\%$）内时，可作为随机动态的参考基准，最大动态偏差控制在$<2\%$的工程允许阈值内

- 动态伴随电路模型将随机微分方程转化为标准EMTP节点方程形式，兼容现有求解器，无需重构底层网络求解架构，代码复用率>90%
