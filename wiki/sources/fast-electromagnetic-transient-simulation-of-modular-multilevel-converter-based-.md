## 方法细节

### 方法概述

提出一种面向柔性直流电网的快速电磁暂态仿真方法，核心包含MMC能量均分模型（EEM）与级联全桥直流断路器快速等效模型。EEM基于理想均压假设与能量守恒原理，将桥臂内投入与未投入的子模块分别聚合为单一等效电容，通过能量均分原则直接迭代更新桥臂平均电容电压，彻底规避传统详细等价模型中耗时的子模块电压排序运算。断路器模型将全桥子模块抽象为含受控电压源与可变电阻的双支路等效电路，利用梯形积分法进行离散化，并将串联子模块串整体等效为单端口网络。两者均严格保留换流器与断路器在闭锁/非闭锁状态下的外部电气特性，在PSCAD/EMTDC平台中实现，适用于大规模、高电平数直流电网的稳态运行与故障暂态全过程仿真。

### 数学公式


**公式1**: $$$u_c(t) = R_c i_c(t) + u_{ceq}(t-\Delta T)$，$R_c = \frac{\Delta T}{2C}$$$

*基于梯形积分法的电容离散化等效公式，用于将子模块电容转化为受控电压源与等效电阻串联形式*


**公式2**: $$$\frac{1}{2}NCu_c^2(t) = \frac{1}{2}C_{on}u_{con}^2(t) + \frac{1}{2}C_{off}u_{coff}^2(t)$$$

*桥臂电容能量守恒方程，将投入与未投入子模块的总能量等效分配至全桥臂*


**公式3**: $$$u_c(t) = \frac{u_{con}(t) + u_{coff}(t)}{N}$$$

*经泰勒展开一阶近似简化后的电容电压迭代更新公式，避免开方运算以提升计算效率*


**公式4**: $$$U_{br}(t) = u_{ceqon}(t-\Delta T)$，$R_{br} = NR_{on} + R_{con}$$$

*MMC桥臂外特性等效公式，将桥臂整体表征为受控电压源与等效电阻串联*


**公式5**: $$$u(t) = R_c(i_a(t-\Delta T) + i_b(t-\Delta T)) + u_c(t-\Delta T)$$$

*全桥子模块闭锁状态下的离散化电压方程，用于构建断路器快速等效电路*


**公式6**: $$$u'(t) = Nu(t)$，$R' = NR$$$

*N个全桥子模块串联后的端口等效缩放公式，实现子模块串的快速聚合*


### 算法步骤

1. 初始化仿真参数：设定步长$\Delta T$，读取上一时刻桥臂平均电容电压$u_c(t-\Delta T)$及历史等效电压源状态。

2. 调制指令解析：根据控制系统输出的参考电压，计算当前桥臂需投入的子模块数量$N_{on}$，并得出未投入数量$N_{off} = N - N_{on}$。

3. 构建等效电容网络：计算投入等效电容$C_{on}=C/N_{on}$及其历史电压源$u_{ceqon}$，以及未投入等效电容$C_{off}=C/N_{off}$及其电压$u_{coff}$。

4. 网络方程求解：将桥臂替换为等效受控源$U_{br}$与电阻$R_{br}$，结合外部直流电网拓扑求解当前时刻桥臂电流$i_c(t)$及等效电容端电压$u_{con}(t)$。

5. 能量均分与状态更新：利用简化公式$u_c(t) = (u_{con}(t) + u_{coff}(t))/N$计算新时刻平均电容电压；若$u_c(t)<0$则强制置零以符合二极管单向导通特性。

6. 断路器等效处理：根据主支路/转移支路触发或闭锁指令，更新全桥子模块等效电阻$R$与电压源$u$；对串联子模块串应用$u'=Nu, R'=NR$进行端口聚合，接入直流网络求解。

7. 历史变量刷新：更新所有等效电压源历史值$u_{ceq}(t-\Delta T)$、$u_c(t-\Delta T)$等，推进至下一仿真步长循环迭代。


### 关键参数

- **仿真步长_ΔT**: 推荐10~25 μs

- **电容等效电阻_Rc**: ΔT/(2C)

- **IGBT导通电阻_Ron**: 器件固有参数，通常取毫欧级

- **桥臂子模块数_N**: 11~1001（验证范围）

- **子模块电容_C**: 根据额定电压与能量设计确定



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| MMC极间短路故障仿真 | 在11至161电平双端±250kV/500MW系统中进行极间金属性短路测试，能量均分模型与详细模型的正极直流电压、电流波形高度重合，残差可信度指标φ均接近1。 | 在1001电平工况下，能量均分模型较详细等价模型提速38倍以上，且仿真耗时几乎不随电平数增加而上升。 |

| 级联全桥直流断路器开断过程仿真 | 500kV断路器（主支路4个子模块，转移支路128个子模块）在0.5s故障后0.2ms启动分断，2.5ms机械开关开断，5ms完全开断。快速模型准确复现了电流转移与避雷器耗能过程。 | 断路器电压U_brk与主支路电流i_main的残差可信度指标分别为0.9908和0.9886；仿真运行时间由详细模型的16125 ms大幅缩减至578 ms，提速约28倍。 |

| 四端柔性直流电网系统级仿真 | 搭建±500kV四端电网（含4个MMC、5条架空线、28台直流断路器），对比传统详细等价模型+详细断路器（模型1）与本文快速模型组合（模型2）。 | 模型2在保持与模型1同等电磁暂态精度的前提下，彻底解决高电平数与多断路器导致的计算瓶颈，满足大规模直流电网高效仿真需求。 |



## 量化发现

- 仿真步长推荐区间为10~25 μs；步长>25 μs时两种模型精度均明显下降，步长<10 μs时耗时剧增但精度无显著提升。
- 1001电平MMC仿真中，能量均分模型较详细等价模型计算速度提升38倍以上，且计算耗时与子模块电平数实现解耦。
- 直流断路器快速模型仿真耗时从16125 ms降至578 ms，加速比达28倍，关键电气量残差可信度指标均大于0.988。
- 能量均分模型与详细等价模型在稳态及故障工况下的波形误差水平无显著差异，残差可信度指标φ均趋近于1，验证了外特性等效的准确性。


## 关键公式

### MMC能量均分简化迭代公式

$$$u_c(t) = \frac{u_{con}(t) + u_{coff}(t)}{N}$$$

*用于每个仿真步长结束时，根据投入与未投入等效电容的电压快速更新桥臂平均电容电压，替代复杂的排序与开方运算*

### 桥臂电容能量守恒方程

$$$\frac{1}{2}NCu_c^2(t) = \frac{1}{2}C_{on}u_{con}^2(t) + \frac{1}{2}C_{off}u_{coff}^2(t)$$$

*能量均分模型的理论基础，确保聚合等效前后桥臂总储能严格一致*

### 级联全桥子模块串端口等效公式

$$$u'(t) = Nu(t)$，$R' = NR$$$

*用于直流断路器转移支路等大量串联子模块的快速聚合，将N个全桥子模块等效为单一受控源与电阻串联支路*

### MMC桥臂外特性等效公式

$$$U_{br}(t) = u_{ceqon}(t-\Delta T)$，$R_{br} = NR_{on} + R_{con}$$$

*将桥臂整体替换为受控电压源与等效电阻，用于接入外部直流网络进行节点导纳矩阵求解*



## 验证详情

- **验证方式**: 对比仿真验证（详细模型/详细等价模型 vs 提出模型）
- **测试系统**: 双端±250kV/500MW直流系统（MMC验证）；500kV级联全桥直流断路器测试电路；±500kV四端柔性直流电网（含4个MMC、5条线路、28台断路器）
- **仿真工具**: PSCAD/EMTDC V4.6.0 (运行环境：Windows 10 64位, Intel i7-4770, 16GB RAM)
- **验证结果**: 所提能量均分模型与断路器快速模型在多种稳态及故障工况下，与详细模型/详细等价模型的电气量波形高度吻合，残差可信度指标均>0.98。在1001电平MMC及含28台断路器的直流电网中实现数十倍计算加速，验证了模型在保留关键外特性的同时具备极高的计算效率与工程适用性。
