# 混合型MMC全状态高效电磁暂态仿真方法研究

**作者**: 连攀杰 等
**年份**: 2022
**来源**: `33/连攀杰 等 - 2021 - 混合型MMC全状态高效电磁暂态仿真方法研究.pdf`

## 摘要

由半桥子模块和全桥子模块组成的混合型模块化多电平换流器(modular multilevel converter，MMC)兼顾直流故障穿越能力和经济性，具有广阔的工程运用前景。针对混合型MMC 电磁暂态模型存在等效复杂、内部节点多、计算效率低的问题，文章分析混合型 MMC 解闭锁模式的工作状态，提出一种“混合型 MMC 全状态高效电磁暂态仿真方法”：根据混合型 MMC 中半桥子模块和全桥子模块的闭锁特性，提出一种混合型 MMC 的闭锁等效方法，提高混合型 MMC 闭锁模式的仿真效率；根据混合型 MMC 的调制特性，改进灵活堆排序的电容电压排序算法，提高混合型 MMC 解锁模式的仿真效率。最后，结合 Matlab 和 PSCAD 典型算例进行测试，验证所提高效混合型 MMC 全状态电磁暂态仿真方法的精确性和快速性。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自混合型MMC柔性直流系统的全电磁暂态仿真：在直流故障、保护闭锁、故障清除后解锁等过程中，模型既要反映子模块电容、二极管通断和桥臂电感的快速暂态，又要能嵌入大规模交直流电网仿真。研究对象是由半桥子模块Nh和全桥子模块Nf串联构成的混合型MMC桥臂及其解锁、闭锁全状态等效。难点在于闭锁后IGBT全部关断，半桥和全桥子模块由二极管自然换流决定状态，存在正向充电、反向充电/旁路、截止等离散切换；若直接使用PSCAD二极管模型，单桥臂需引入6个二极管等效并增加内部节点，还会带来开关插值计算。另一方面，解锁模式又需要按电容电压排序实现均压，子模块数多时排序成为计算负担。本文的贡献不是简单换用平均模型，而是在戴维南等效框架下区分闭锁与解锁机制：闭锁时分别针对半桥、全桥子模块构造等效与状态判别，避免依赖外部二极管器件模型；解锁时利用混合型MMC调制特性改进灵活堆排序，减少无效排序计算，从而形成面向解闭锁切换的全状态EMT实现路径。

### 2. 模型、算法与实现技术

本文以桥臂戴维南等效作为统一接口，向外部网络输出桥臂等效电压u_arm,eq和等效电阻R_arm,eq，其中u_arm,eq由半桥子模块等效电压、全桥子模块等效电压和桥臂电感历史电压组成，R_arm,eq由半桥、全桥子模块等效电阻组成。闭锁半桥部分利用同一桥臂同类型子模块电流相同的特性，把多个半桥子模块的闭锁二极管行为压缩为虚拟二极管及串联电容等效：虚拟二极管通断电阻取Nh倍单管通断电阻，半桥电容等效电压为各电容历史电压求和，等效电阻为Nh倍离散电容电阻。计算中先按上一时步通断状态组装网络，求解后用端电压或电流符号检查是否发生状态越界；若截止到导通或导通到截止发生在步长内部，则用线性插值定位过零时刻，并结合变步长后退欧拉重新计算。闭锁全桥部分则根据全桥二极管组合的正向充电、反向充电和截止状态，直接给出相应戴维南电阻、电压表达式，并用充电电压函数及端口电压符号判断预估状态是否正确，必要时同样插值修正切换时刻。解锁模式下，核心输入是调制环节给出的各桥臂需投入子模块数及子模块电容电压；算法输出投退选择和更新后的桥臂等效量。改进灵活堆排序的作用是只围绕调制需要和电流方向选择待投入/切除子模块，避免每步全量排序。

### 3. 验证、优势与不足

作者采用Matlab和PSCAD/EMTDC典型算例验证方法的正确性与快速性。原文摘要和引言明确说明对比背景包括详细模型、戴维南等效模型以及借助PSCAD自带二极管模型模拟闭锁的做法；从给出的文本看，关键基线是传统闭锁等效中单桥臂使用6个二极管模型、由此增加内部节点，并且部分已有方法在解锁时仍进行二极管插值、闭锁时仍进行电容电压排序。验证关注的指标包括波形精度、解闭锁状态切换时刻处理、内部节点数量和计算耗时。优势主要体现在三个机制层面：第一，闭锁半桥使用虚拟二极管，闭锁时才参与状态判别，解锁时退出，减少无关二极管插值；第二，闭锁全桥不逐个展开二极管支路，而是用状态相关的戴维南等效和充电判据描述正/反向充电与截止；第三，解锁均压控制中用改进堆排序降低电容电压选择开销。可核验的结构性数字是传统方法单桥臂需6个二极管等效，而本文意在消除这些内部节点；但在当前抽取文本中未给出具体仿真耗时、加速倍数、误差百分比、测试系统参数或波形误差表，因此不能把“快速”“准确”扩展为可量化结论。从验证范围看，结论受所用Matlab/PSCAD算例、步长、控制策略、子模块数、故障类型和对比模型约束，尚不能直接外推到所有混合MMC拓扑、实时仿真硬件或任意保护控制策略。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于把混合型MMC的“全状态”仿真拆成两个本质不同的问题：闭锁时应准确处理二极管自然换流与步内切换，解锁时应减少电容电压排序和投退选择的计算冗余。它适合被后续大电网EMT建模、柔直故障穿越仿真、MMC保护闭锁/重启策略验证、戴维南等效模型加速和排序算法优化页面复用，尤其可作为“闭锁等效如何避免内部节点膨胀”的方法入口。使用时应保留其前提：同桥臂同类型子模块在闭锁下具有相同工作状态，外部网络通过桥臂戴维南端口耦合。不适合把本文直接当作控制性能、硬件实时性或非半桥/全桥混合拓扑的证据，也不应在缺少原文实验表图时声称具体加速倍数。

## 主要发现

## 方法细节

### 方法概述

本文提出一种混合型MMC全状态高效电磁暂态仿真方法，基于戴维南等效理论分别针对闭锁和解锁模式进行优化。在闭锁模式下，针对半桥子模块提出虚拟二极管模型，仅在闭锁时投入以处理插值问题，解锁时退出以避免不必要的计算；针对全桥子模块，基于其闭锁特性设计电压充电函数直接模拟工作状态，通过插值算法精确捕捉状态切换时刻。在解锁模式下，采用改进的灵活堆排序算法优化电容电压排序过程。该方法通过消除冗余内部节点和避免无效计算，显著提升了混合型MMC在全状态下的仿真效率。

### 数学公式

**公式1**: $$$u_{\text{arm,eq}}(t) = u_{\text{all\_sm,eq}}^{\text{HB}}(t) + u_{\text{all\_sm,eq}}^{\text{FB}}(t) + u_L(t)$$$

*桥臂戴维南等效电压，等于半桥子模块等效电压、全桥子模块等效电压与桥臂电感等效电压之和*

**公式2**: $$$R_{\text{arm,eq}}(t) = R_{\text{all\_sm,eq}}^{\text{HB}}(t) + R_{\text{all\_sm,eq}}^{\text{FB}}(t)$$$

*桥臂戴维南等效电阻，等于半桥子模块等效电阻与全桥子模块等效电阻之和*

**公式3**: $$$R_{\text{ON}}^{\text{DIO}} = N_h R_{\text{ON}}$$$

*半桥子模块虚拟二极管模型通态等效电阻，为单个子模块二极管导通电阻的Nh倍*

**公式4**: $$$R_{\text{OFF}}^{\text{DIO}} = N_h R_{\text{OFF}}$$$

*半桥子模块虚拟二极管模型断态等效电阻，为单个子模块二极管关断电阻的Nh倍*

**公式5**: $$$R_{\text{all\_C}}^{\text{HB}}(t) = N_h R_C$$$

*桥臂所有半桥子模块电容串联后的戴维南等效电阻*

**公式6**: $$$u_{\text{all\_C}}^{\text{HB}}(t) = \sum_{i=1}^{N_h} u_{C_i}(t)$$$

*桥臂所有半桥子模块电容串联后的戴维南等效电压，为各子模块电容电压之和*

**公式7**: $$$R_{\text{ON}}^{\text{FB}} = \frac{(R_{\text{ON}} + R_{\text{OFF}})R_C + 2R_{\text{ON}}R_{\text{OFF}}}{2R_C + (R_{\text{ON}} + R_{\text{OFF}})}$$$

*全桥子模块闭锁模式正投入状态下的戴维南等效电阻*

**公式8**: $$$R_{\text{OFF}}^{\text{FB}} = R_{\text{OFF}}$$$

*全桥子模块闭锁模式截止状态下的戴维南等效电阻*

**公式9**: $$$u_i^{\text{FB}}(t) = \frac{R_{\text{OFF}}}{R_{\text{ON}} + R_C + R_{\text{OFF}}} u_{C_{\text{eq}}}^i(t)$$$

*全桥子模块闭锁模式正投入状态下的戴维南等效电压*

**公式10**: $$$u_{\text{charge}}(t) = u_{\text{FBsm}}(t) - \left(\sum_{i=1}^{N_f} u_{C_{\text{eq}}}^i(t)\right)^2$$$

*全桥子模块闭锁模式充电电压函数，用于判断工作状态及插值计算*

### 算法步骤

1. 半桥子模块闭锁仿真流程：

2. 步骤1：初始化虚拟二极管模型参数，包括设置通态电阻$R_{\text{ON}}^{\text{DIO}} = N_h R_{\text{ON}}$和断态电阻$R_{\text{OFF}}^{\text{DIO}} = N_h R_{\text{OFF}}$，以及阻尼支路参数

3. 步骤2：基于当前仿真步长和数值积分算法（如后退欧拉法），求解虚拟二极管模型的戴维南等效支路参数$R_{\text{eq}}^{\text{DIO}}$和$V_{\text{eq}}^{\text{DIO}}$

4. 步骤3：将桥臂内所有半桥子模块电容串联，计算其戴维南等效电阻$R_{\text{all\_C}}^{\text{HB}}(t) = N_h R_C$和等效电压$u_{\text{all\_C}}^{\text{HB}}(t) = \sum_{i=1}^{N_h} u_{C_i}(t)$

5. 步骤4：构建混合型MMC的节点导纳矩阵和右端项，将虚拟二极管支路和电容等效支路纳入系统方程

6. 步骤5：将混合型MMC模型与全网联立求解，获得各节点电压，进而计算虚拟二极管模型端口的电压和电流

7. 步骤6：判断虚拟二极管模型本步计算的正确性，处理四种情形：

8. - 情形1：若上一步为截止状态且本步两端电压小于0，则计算正确，保持截止状态

9. - 情形2：若上一步为截止状态且本步两端电压大于0，则计算错误，需根据历史电压和本步电压进行线性插值，寻找电压过零时刻，在该时刻将可变电阻修改为$R_{\text{ON}}^{\text{DIO}}$，并基于多步变步长后退欧拉法重新执行步骤2

10. - 情形3：若上一步为导通状态且本步电流大于0，则计算正确，保持导通状态

11. - 情形4：若上一步为导通状态且本步电流小于0，则计算错误，需根据历史电流和本步电流进行线性插值，寻找电流过零时刻，在该时刻将可变电阻修改为$R_{\text{OFF}}^{\text{DIO}}$，并基于多步变步长后退欧拉法重新执行步骤2

12. 步骤7：当所有虚拟二极管模型均计算正确后，完成本步计算，存储当前虚拟二极管状态、电压和电流值作为下一步的历史值

13. 步骤8：进入下一仿真时步，重复步骤2-7

14. 全桥子模块闭锁仿真流程：

15. 步骤1：根据公式$u_{\text{charge}}(t) = u_{\text{FBsm}}(t) - (\sum_{i=1}^{N_f} u_{C_{\text{eq}}}^i(t))^2$计算充电电压值

16. 步骤2：基于上一步的历史状态预判本步全桥子模块的工作状态（正投入、负投入或截止），并据此确定桥臂全桥子模块的戴维南等效支路参数（等效电阻和等效电压）

17. 步骤3：确定混合型MMC的导纳矩阵和右端项，将全桥等效支路纳入系统方程

18. 步骤4：将混合型MMC归入全网进行求解，获得各节点电压，进而计算桥臂全桥子模块端口电压$u_{\text{FBsm}}(t)$和电流

19. 步骤5：根据$u_{\text{charge}}(t)$和$u_{\text{FBsm}}(t)$的符号判断预判是否正确：

20. - 若$u_{\text{charge}}(t) < 0$且$u_{\text{FBsm}}(t) < 0$，则预判正确，子模块处于负投入状态

21. - 若$u_{\text{charge}}(t) > 0$且$u_{\text{FBsm}}(t) > 0$，则预判错误，需采用插值算法寻找$u_{\text{charge}}(t)$的过零点$t^+$，在$t^+$时刻将工作状态修正为正投入，并基于多步变步长后退欧拉法重新进行计算

22. - 类似地处理其他状态转换情况（如正投入转截止、负投入转截止等）

23. 步骤6：完成本步计算并存储状态，进入下一步

### 关键参数

- **$N_h$**: 单个桥臂中半桥子模块（Half-bridge Sub-module）的数量

- **$N_f$**: 单个桥臂中全桥子模块（Full-bridge Sub-module）的数量

- **$N$**: 单个桥臂中子模块总数，$N = N_h + N_f$

- **$R_{\text{ON}}$**: 二极管的导通电阻（on-state resistance）

- **$R_{\text{OFF}}$**: 二极管的关断电阻（off-state resistance），通常取值很大（如1MΩ）

- **$R_C$**: 子模块电容经过数值积分算法离散后的等效电阻

- **$u_{C_i}(t)$**: 第$i$个子模块电容的等效电压源值

- **$u_L(t)$**: 桥臂电感经过数值积分离散后的等效电压源

- **$R_L(t)$**: 桥臂电感经过数值积分离散后的等效电阻

- **$u_{\text{FBsm}}(t)$**: 桥臂所有全桥子模块的外界电压（端口电压）

## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 混合型MMC闭锁模式仿真 | 测试了半桥和全桥子模块在闭锁状态下的等效精度。传统方法需为单桥臂配置6个二极管模型以模拟闭锁状态，而所提方法通过虚拟二极管模型（半桥）和直接状态模拟（全桥），消除了这6个内部节点，同时通过插值算法精确捕捉了二极管开关时刻，避免了定步长仿真产生的畸变点。 | 相比传统PSCAD二极管模型，单桥臂内部节点数量减少6个 |

| 混合型MMC解锁模式仿真 | 测试了改进灵活堆排序算法在电容电压均衡控制中的效率。所提算法根据混合型MMC的调制特性优化了排序过程，在保持与详细模型相同精度的前提下，减少了排序计算量。 | 避免了解锁模式下二极管模型不必要的插值计算，以及闭锁模式下不必要的电容电压排序计算 |

| 全状态联合仿真 | 结合Matlab和PSCAD平台对混合型MMC在解闭锁转换过程中的仿真性能进行了验证，包括直流故障穿越等典型场景。 | 实现了全状态下的高效仿真，兼顾精度与速度 |

## 量化发现

- 单桥臂内部节点减少数量：6个（相比基于PSCAD自带二极管模型的传统等效方法）
- 半桥子模块虚拟二极管等效通态电阻：$R_{\text{ON}}^{\text{DIO}} = N_h R_{\text{ON}}$，断态电阻：$R_{\text{OFF}}^{\text{DIO}} = N_h R_{\text{OFF}}$
- 全桥子模块闭锁等效通态电阻：$R_{\text{ON}}^{\text{FB}} = \frac{(R_{\text{ON}} + R_{\text{OFF}})R_C + 2R_{\text{ON}}R_{\text{OFF}}}{2R_C + (R_{\text{ON}} + R_{\text{OFF}})}$，断态电阻：$R_{\text{OFF}}^{\text{FB}} = R_{\text{OFF}}$
- 解锁模式下计算效率提升：通过避免二极管模型不必要的插值计算和优化电容电压排序算法实现
- 闭锁模式下计算效率提升：通过避免电容电压排序计算和减少内部节点实现
- 仿真精度：通过多步变步长后退欧拉法进行插值，精确捕捉开关时刻，消除定步长仿真产生的畸变点

## 关键公式

### 全桥子模块充电电压函数

$$$u_{\text{charge}}(t) = u_{\text{FBsm}}(t) - \left(\sum_{i=1}^{N_f} u_{C_{\text{eq}}}^i(t)\right)^2$$$

*用于全桥子模块闭锁模式下工作状态的判断和插值计算，通过判断该值的正负确定子模块处于正投入、负投入还是截止状态*

### 全桥子模块闭锁等效通态电阻

$$$R_{\text{ON}}^{\text{FB}} = \frac{(R_{\text{ON}} + R_{\text{OFF}})R_C + 2R_{\text{ON}}R_{\text{OFF}}}{2R_C + (R_{\text{ON}} + R_{\text{OFF}})}$$$

*全桥子模块在闭锁模式正投入或负投入状态下的戴维南等效电阻计算*

### 桥臂戴维南等效电压

$$$u_{\text{arm,eq}}(t) = u_{\text{all\_sm,eq}}^{\text{HB}}(t) + u_{\text{all\_sm,eq}}^{\text{FB}}(t) + u_L(t)$$$

*将混合型MMC桥臂中的半桥子模块群、全桥子模块群和桥臂电感分别等效为戴维南支路后的总等效电压*
