# An improved high frequency white-box lossy transformer model for the calculation of power systems electromagnetic transients

**作者**: N. Hooda
**年份**: 2017
**来源**: `07&08/An improved high frequency white-box lossy transformer model for the calculation of power systems electromagnetic transients.pdf`

## 摘要

本文针对单水源树状重力流供水管网的管径优化设计问题，提出了一种基于线性规划（LP）的全局最优求解框架。传统设计工具多采用启发式算法或整数线性规划（ILP），前者无法保证全局最优，后者在允许单管段组合多种管径时会因引入大量布尔变量和线性化约束导致计算复杂度呈指数级增长。本文突破性地摒弃了显式建模“每段管道最多使用两种相邻管径”的离散选择逻辑，转而将每条管道视为由所有可用商用管径分段组成的连续体，引入连续变量表示各管径的铺设长度。基于管径成本函数的凸性数学特性，该LP模型在满足节点最小压力与流量守恒约束时，求解器会自动收敛至仅含最多两种相邻非零管径长度的最优解。该方法成功将NP-hard的ILP问题转化为多项式时间可解的LP问题，在保证全局最优性的同时实现了计算效率的跨越式提升。

## 研究解读

### 1. 需求、对象、挑战与贡献

原文研究的实际需求并非电磁暂态变压器建模，而是发展中国家农村单水源、重力供水、树状管网的低成本设计。研究对象是在已知拓扑、节点需水量、节点最低压力要求、管段长度和商用离散管径集合时，为每条管段选择一种或多种管径，使总管材成本最低且各需求点压力达标。难点在于：管径只能从离散商业规格中选取，管径越大压力损失越小但成本越高；一般情况下同一条 link 可由不同管径的多个 pipe segments 组成，若直接显式描述“每条边最多两种相邻管径”等组合结构，会引入复杂离散选择。已有 BRANCH、WATERGEMS 等工具面向一般多管径情形多采用启发式或遗传算法，不能保证最优；已有 ILP 只处理“每条 link 仅一种管径”的特殊情形。本文贡献是提出一个面向一般多管段情形的线性规划 LP 表述，不强制预先枚举每条边的两管径组合，却能在求解结果中自然得到每条边至多两种管径的结构，同时保持全局最优性，并实现为 JalTantra 设计系统。

### 2. 模型、算法与实现技术

论文提出的核心模型把每条网络 link 与每个可用管径的组合都定义为一个连续变量，变量表示该管径在该 link 上铺设的长度，而不是用整数变量选择某个离散管径。输入包括单水源树状网络拓扑、各节点高程和需求、最低服务压力、各 link 长度、可用商业管径及其单位成本和水力参数；输出是每条 link 上各管径对应的铺设长度及总成本。目标函数为所有 link、所有候选管径的“单位长度成本×铺设长度”之和最小。每条 link 需要满足长度守恒约束，即各候选管径长度之和等于该 link 总长度。由于网络是树状且单水源，link 流量可由下游需求累加确定，不需要环网水力平衡迭代。压力约束沿水源到需求节点的唯一路径累计水头损失，要求剩余水头不低于用户给定最低压力。水头损失通过 Hazen-Williams 形式按给定流量、管径和粗糙系数计算为单位长度损失系数，再与对应长度变量相乘，因此进入 LP 时保持线性。论文还讨论了另一种试图显式编码“每条 link 至多两种相邻管径”的模型，但结果表明这种把最优结构硬写进约束的方式反而运行更差；更有效的是不限制每条 link 的管径数量，让 LP 自然产生稀疏的最优组合。该模型被集成到 JalTantra，系统还提供 GIS 功能以便录入网络细节。

### 3. 验证、优势与不足

作者通过在六个管网算例上比较不同模型和工具来验证方法，原文说明这些网络包括真实和测试网络，但在所给抽取文本中尚未列出完整表格数值。实现环境是 JalTantra；对比对象包括 BRANCH 这类单水源树状管网启发式工具、WATERGEMS 等不保证全局最优的商业/遗传算法思路，以及文献中只允许每条 link 一种管径的 ILP 特殊模型。验证指标主要是能否满足压力约束、是否可求得一般多管段问题的最优解、运行时间表现，以及所得方案是否呈现“每条 link 至多两种相邻管径”的结构。优势在于：该 LP 表述覆盖了一般的多 pipe segments per link 问题，而不是退化到每条边单管径；同时避免了显式组合枚举，因而比作者尝试的相邻双管径约束模型更容易求解。它还把已有理论中关于最优结构的性质转化为一种可由 LP 解自然体现的结果，而不是额外整数约束。从验证范围看，结论主要适用于单水源、无环、重力供水、管径选择已离散给定且流量由需求确定的农村供水网络。原文抽取片段未给出可核验的具体运行时间、成本下降百分比或网络规模表格，因此不能据此声称某个数值倍数的速度提升。该方法也未在环状供水网、多水源、泵站调度、动态用水、可靠性冗余或电力系统 EMT 场景中验证。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于说明：对于树状重力供水管网的离散管径设计，未必需要把所有离散组合结构显式建成整数规划；若把每种管径在每条 link 上的铺设长度作为连续变量，并用路径压力约束表达水力可行性，LP 解可自然恢复已知的两相邻管径最优结构。它能解决农村供水网络中“既允许一条管段混合多种管径、又希望获得全局最优成本”的设计问题，适合被后续关于管网优化、JalTantra 系统、树状网络资源分配、离散选择连续化建模的页面复用。它不适合被外推为 EMT 仿真、变压器高频白盒建模或一般电力暂态算法的证据，也不应外推到环网、多源、含泵调度或未给定确定流量的水力系统。

## 方法细节

### 方法概述

本文针对单水源树状重力流供水管网的管径优化设计问题，提出了一种基于线性规划（LP）的全局最优求解框架。传统设计工具多采用启发式算法或整数线性规划（ILP），前者无法保证全局最优，后者在允许单管段组合多种管径时会因引入大量布尔变量和线性化约束导致计算复杂度呈指数级增长。本文突破性地摒弃了显式建模“每段管道最多使用两种相邻管径”的离散选择逻辑，转而将每条管道视为由所有可用商用管径分段组成的连续体，引入连续变量表示各管径的铺设长度。基于管径成本函数的凸性数学特性，该LP模型在满足节点最小压力与流量守恒约束时，求解器会自动收敛至仅含最多两种相邻非零管径长度的最优解。该方法成功将NP-hard的ILP问题转化为多项式时间可解的LP问题，在保证全局最优性的同时实现了计算效率的跨越式提升。

### 数学公式

**公式1**: $$$O(\cdot) = \sum_{i=1}^{N_L} \sum_{j=1}^{N_P} C_{ij}(D_{ij}) l_{ij}$$$

*目标函数：最小化全网管道总建设成本，其中$C_{ij}$为单位长度成本，$l_{ij}$为管段长度连续变量。*

**公式2**: $$$\sum_{j=1}^{N_P} l_{ij} = L_i$$$

*管道长度约束：确保每条链路中各管径分段长度之和等于该链路总设计长度$L_i$。*

**公式3**: $$$P_n \le H_R - E_n - \sum_{i \in S_n} \sum_{j=1}^{N_P} HL'_{ij} l_{ij}$$$

*节点压力约束：保证任意节点$n$的剩余水头不低于最小压力要求$P_n$，考虑了从水源到该节点路径上的累计水头损失。*

**公式4**: $$$HL'_{ij} = \frac{10.68 \cdot Q_i^{1.852}}{C_{HW,j} \cdot D_j^{4.87}}$$$

*Hazen-Williams单位长度水头损失公式：用于计算特定管径和流量下的摩擦水头损失系数，其中$Q_i$为链路流量，$C_{HW,j}$为粗糙系数，$D_j$为管径。*

### 算法步骤

1. 网络拓扑与参数初始化：输入单水源树状管网结构（节点高程、需水量、最小压力阈值、链路长度）及商用管材库（离散管径集合、对应单位成本、Hazen-Williams粗糙系数）。

2. 确定性流量分配：基于树状网络无环特性，从末端需水节点向水源逆向递推，直接计算每条链路的稳态设计流量$Q_i$，无需迭代水力平衡。

3. LP变量定义：为每条链路$i$的每种可用管径$j$创建连续决策变量$l_{ij}$，表示该管径在该链路中的实际铺设长度。

4. 构建目标函数：建立全网总成本最小化模型$O = \sum \sum C_{ij} l_{ij}$，将离散管径选择转化为连续长度优化。

5. 施加物理与工程约束：添加链路总长度守恒约束$\sum_j l_{ij} = L_i$；基于Hazen-Williams公式计算单位水损$HL'_{ij}$，构建节点最小压力约束不等式组。

6. 调用LP求解器：将构建的线性规划模型输入GLPK求解器，利用单纯形法或内点法进行多项式时间迭代求解。

7. 解后处理与结构提取：解析求解器输出的$l_{ij}$值，自动过滤长度为零的管径组合。依据凸性理论，每条链路仅保留长度大于零的至多两种相邻管径及其对应长度。

8. 输出优化方案：生成各链路的管径分段配置清单及对应造价，完成自动化设计。

### 关键参数

- **N_L**: 网络链路总数

- **N_P**: 可用商用管径种类数（实验取10）

- **L_i**: 第i条链路的总长度

- **C_{ij}**: 第j种管径的单位长度造价

- **P_n**: 节点n要求的最小服务压力

- **H_R**: 参考水源节点的供水水头

- **E_n**: 节点n的地面高程

- **Q_i**: 链路i的设计流量（由节点需水量确定）

- **C_{HW,j}**: 第j种管材的Hazen-Williams粗糙系数

## 关键公式

### 节点压力约束核心方程

$$$P_n \le H_R - E_n - \sum_{i \in S_n} \sum_{j=1}^{N_P} \left( \frac{10.68 \cdot Q_i^{1.852}}{C_{HW,j} \cdot D_j^{4.87}} \right) l_{ij}$$$

*用于在LP优化过程中确保管网末端及中间节点的水压满足最低服务标准，是连接水力计算与成本优化的关键桥梁。*

### 线性化管网造价目标函数

$$$O(\cdot) = \sum_{i=1}^{N_L} \sum_{j=1}^{N_P} C_{ij}(D_{ij}) l_{ij}$$$

*将离散管径选择转化为连续长度变量乘积，消除ILP中的非线性项与布尔变量，使问题可在多项式时间内求解。*
