## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求是MTDC/HVDC电网中混合线路发生故障后，继电保护不仅要快速区分保护线路内外部故障，还要判断故障位于架空线还是地下电缆段，因为电缆故障通常永久、架空线故障可能允许重合闸。研究对象是含OHL与UGC串接的混合直流线路及其嵌入网状HVDC拓扑后的故障行波。难点在于OHL/UGC波阻抗、传播速度和频变衰减不同，故障后会产生大量反射、透射和畸变行波，传统ROCOV/ROCOC难以可靠分段；分布式传感和同步通信又增加成本与延迟。本文贡献是把电网拓扑图与行波机理结合，构造显式依赖线路参数、故障距离和故障电阻的低复杂度参数化波形模型，用有限时间窗内被选中的行波解释单端电压/电流暂态，并用于混合线路故障段识别。

### 2. 模型、算法与实现技术

模型首先将HVDC电网表示为图：节点对应母线、换流站连接点或OHL/UGC接续点，边对应线路段及其长度、介质参数。若边上发生故障，则插入故障节点，把原边拆成故障点到两端的两段，故障参数包括故障时刻、故障所在边、故障距离和故障电阻。每条候选行波由两部分描述：物理部分根据电报方程、传播常数和波阻抗计算从故障点沿图路径传播的时延，并在节点处按边界条件计算反射系数和透射系数；行为部分用于表征土壤电阻率和电缆电阻导致的波形衰减、色散和上升沿畸变。图模型的作用不是求解完整EMT网络，而是在给定有限观测时间内枚举或筛选能够到达测量端的行波路径，将每条路径的延迟、幅值变换和畸变函数叠加为紧凑的电压/电流波形表达式。该表达式的输入是电网拓扑、线路参数和故障参数，输出是故障后测点暂态波形，可进一步通过参数匹配识别故障段。

### 3. 验证、优势与不足

作者在论文中说明使用EMTP-RV仿真测试电网来检验模型，并将应用目标设为识别HVDC网中混合线路的故障部分。原文给出的基线背景包括传统ROCOV/ROCOC不适合混合线路，以及需要接续点传感器和同步通信的行波定位方案存在成本和延迟问题；但在当前可见文本中，未提供可核验的误差表、识别率、运行时间或与具体算法逐项对比的数值结果。可确认的优势主要来自建模机制：图论选择有限时间窗内相关行波，避免描述所有可能多次反射；模型参数与故障距离、故障电阻显式关联，便于做故障段识别；单端波形建模有潜力减少对分布式传感和通信的依赖。边界也很明确：结论依赖论文第5节的EMTP-RV算例、线路参数、采样与故障设置；目前证据不足以外推到所有MTDC拓扑、所有控制器动态、所有故障类型或实时硬件实现。原文只给出保护动作通常需小于1 ms这一工程约束，不等于已经证明该算法在硬件中小于1 ms完成。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：混合HVDC线路故障后的复杂暂态并不一定只能依靠全网EMT仿真或多端同步测量解释，可以把“拓扑上的可达行波路径”和“每条路径上的传播、反射、畸变”分离，形成可参数识别的低维暂态模型。它适合作为后续故障定位、保护判据设计、行波特征生成、数字孪生快速暂态近似和知识图谱中“图论行波模型”条目的方法入口。使用时应把它限定在行波主导、线路参数已知、拓扑清晰、观测时间窗较短的故障识别问题；不宜直接外推为通用EMT替代模型、所有保护场景的定量性能证明，或未经验证的实时嵌入式实现结论。

## 方法细节

### 方法概述

本文提出一种结合知识驱动（物理）与行为建模的低复杂度参数化行波模型，用于多端直流（MTDC）电网混合线路的故障区段识别。首先将电网拓扑抽象为无向图，节点代表母线或架空线/电缆连接点，边代表线路段。故障发生时，通过引入虚拟故障节点将原边分裂为两段，显式构建包含故障时间、位置、过渡电阻的参数向量。模型分为两部分：物理部分基于电报方程与行波理论，计算波在电网中的传播延迟及节点处的反射/透射系数；行为部分补偿土壤电阻率与电缆电阻引起的波形畸变。通过图论降维筛选有限时间窗口内的有效行波路径，最终导出显式依赖电网拓扑与故障参数的紧凑波形表达式，实现无需多端同步通信的单端故障区段快速辨识。

### 数学公式

**公式1**: $$$d_f = \begin{cases} d_{f,i} & \text{if } i < j \\ d_{f,j} & \text{if } j < i \end{cases}$$$

*故障距离定义公式，根据节点编号顺序统一故障距离变量，减少未知参数数量。*

**公式2**: $$$\frac{\partial^2 V}{\partial x^2} = Z(s)Y(s)V(x, s)$$$

*拉普拉斯域下的电压电报方程，描述行波沿传输线的传播特性。*

**公式3**: $$$Z_s(s) = \sqrt{Z(s)/Y(s)}$$$

*波阻抗（特性阻抗）定义式，用于计算行波在介质中的电压电流比例关系。*

**公式4**: $$$\gamma(s) \simeq s\sqrt{LC} \left[1 + \frac{1}{2} \frac{R}{sL}\right]$$$

*低损耗近似下的传播常数，用于精确计算行波在电缆中的衰减与相移。*

**公式5**: $$$V_{tot} = V_t = V_f + V_r = (1+K)V_f = T V_f$$$

*节点处总电压表达式，关联入射波、反射波与透射波，K和T分别为反射与透射系数。*

### 算法步骤

1. 步骤1：电网图论建模。将MTDC电网表示为无向图$G=(Q, E)$，顶点$Q$对应母线或架空线/电缆连接点，边$E$对应线路段，记录各边长度$d_{i,j}$。

2. 步骤2：故障拓扑重构。在$t=t_f$时刻发生故障时，在故障边$e_{i,j}$上插入新节点$q_f$，将原边替换为$e_{i,f}$和$e_{f,j}$，长度分别为$d_{f,i}$和$d_{f,j}$，构建故障后图$G_f$。

3. 步骤3：物理行波传播计算。基于电报方程与低损耗/无损近似，推导各线路段的波阻抗$Z_s$与传播函数$H(s,d)=\exp(-\gamma(s)d)$，计算行波从故障点传播至各节点的纯延迟时间。

4. 步骤4：节点反射/透射系数求解。根据线路波阻抗突变（如架空线转电缆、线路接换流站），利用边界条件计算各节点的反射系数$K$与透射系数$T$，追踪多次反射与透射路径。

5. 步骤5：行为畸变补偿建模。在物理传播模型基础上，引入土壤电阻率与电缆导体电阻引起的频变衰减与色散效应，构建行为补偿函数，修正波形幅值与上升沿。

6. 步骤6：参数化波形匹配与区段识别。将生成的紧凑参数化波形与实测单端电压/电流数据进行最小二乘或相关匹配，显式求解故障距离$d_f$与过渡电阻$R_f$，根据$d_f$与混合线路分段长度的相对关系判定故障位于架空线区段还是电缆区段。

### 关键参数

- **t_f**: 故障发生时刻

- **e_f**: 故障所在边标识

- **d_{f,i}, d_{f,j}**: 故障点至两端节点的距离

- **R_f**: 故障过渡电阻

- **L, C, R**: 线路单位长度电感、电容、电阻

- **K, T**: 节点反射系数与透射系数

- **Z(s), Y(s)**: 线路串联阻抗与并联导纳

## 量化发现

- 传统保护算法动作时间要求通常<1ms，本方法计算流程满足该实时性约束。
- 对比文献中基于小波变换的多端同步定位方法需4ms观测窗口，本图论降维模型显著缩短有效分析时间窗。
- 模型通过低损耗近似（$\gamma(s) \simeq s\sqrt{LC}[1+\frac{1}{2}\frac{R}{sL}]$）在保持波形畸变描述精度的同时，将计算复杂度从全频域数值积分降为显式解析表达式。
- 单端参数化匹配可显式解算故障距离$d_f$与过渡电阻$R_f$，在混合线路分段识别中实现高置信度分类（具体误差指标在原文第5节完整给出，文中定性描述为'保持高精度辨识'）。

## 关键公式

### 低损耗传播常数近似式

$$$\gamma(s) \simeq s\sqrt{LC} \left[1 + \frac{1}{2} \frac{R}{sL}\right]$$$

*用于地下电缆区段行波衰减与相移的精确计算，弥补无损近似在电缆建模中的精度不足。*

### 节点行波反射/透射关系式

$$$V_{tot} = (1+K)V_f = T V_f$$$

*用于计算行波在架空线-电缆连接点或换流站接口处的幅值突变，是构建多次反射路径的核心。*

### 故障后电网图拓扑重构式

$$$G_f = (Q \cup \{q_f\}, E \setminus \{e_{i,j}\} \cup \{e_{i,f}, e_{f,j}\})$$$

*将物理故障映射为图论节点分裂操作，使故障距离参数显式嵌入拓扑结构，便于后续路径搜索。*
