# Electromechanical-electromagnetic Hybrid Simulation Technology With Large Number of Electromagnetic

**作者**: Chen 等
**年份**: 2022
**来源**: `17/Chen 等 - 2020 - Electromechanical-electromagnetic Hybrid Simulation Technology With Large Number of Electromagnetic.pdf`

## 摘要

伴随特高压直流的飞速发展和其输电容量阶跃式提升，我国电网呈现出“强直弱交”的特点。送受端、交直流、多直流之间耦合日趋紧密，电网运行特性日趋复杂。在大规模交直流电网的仿真分析中，直流输电系统需要采用电磁暂态仿真以获得准确动态特性，其余交流电网采用机电暂态仿真以提高仿真效率。然而，含多条直流电磁暂态模型的混合仿真在数据建立、方式调整及平稳启动方面存在困难，制约了大规模电网分析的工作效率和仿真精度。提供了通过直流标准化建模和数据映射拼接实现混合仿真数据自动建立的解决方案，提高了建模效率；提供了直流输电电磁模型运行方式自动调整、混合仿真接口电压箝位平稳启动的方法，使混合仿真能在 0.6s 的仿真过程内达到稳态。所提方法提高了大规模交直流电网机电–电磁混合仿真分析的工作效率和仿真精度。

## 研究解读

### 1. 需求、对象、挑战与贡献

这篇论文面向的是国调方式计算、批量故障扫描等大规模交直流电网分析需求：交流主网希望用机电暂态保持规模和速度，而多回特高压/高压直流又必须用电磁暂态模型才能描述换相失败、控制响应、不对称故障等快速过程。研究对象不是单个HVDC模型精细化，而是“含大量电磁直流模型”的机电–电磁混合仿真建模、初始化和启动流程。难点在于：每个运行方式下需把潮流数据、直流工程模型、接口位置、换流变抽头、滤波器投切、控制指令和接口初值全部对齐；若仍靠人工拼接和调参，批量校核成本高且启动时容易因机电侧与电磁侧接口电压/潮流不一致产生暂态冲击。本文贡献是把这一工程流程自动化：用直流标准化模型库和数据映射拼接生成混合仿真数据；用直流内部潮流计算自动给出抽头、滤波器和控制参考；再用接口电压箝位及自动切除判据实现从潮流点平稳启动。

### 2. 模型、算法与实现技术

方法由三层实现组成。第一层是标准化电磁直流模型库：针对实际直流工程预先建立可复用EMT模型，并维护其与方式计算或在线潮流数据中直流元件的映射关系；输入是分网方案和潮流/方式数据，输出是被选中的直流EMT模型、交流机电网络以及自动定义的机电–电磁接口。第二层是电磁直流运行工况自动初始化：以潮流给出的直流功率、电压、电流、交流母线电压和无功交换为边界，计算换流器理想空载直流电压、换流变变比、换相角、功率因数角和所需滤波器无功容量，从而确定换流变抽头档位、滤波器投切组数以及整流侧触发角/逆变侧关断角等控制参考。相关公式的作用不是建立新的HVDC物理模型，而是把机电潮流解转换成电磁模型可直接启动的设备状态和控制指令。第三层是接口平稳启动：在接口换流母线并联幅值、相角和潮流方向与目标潮流一致的临时理想电压源，启动后监测箝位电源输出功率，当功率低于额定直流功率5%等条件或达到预设时间时切除，使电磁侧与机电侧从受控支撑过渡到自由耦合运行。

### 3. 验证、优势与不足

作者以实际大规模交直流混联电网为应用背景，页面记录的测试系统包含复奉、锦苏、宾金等10余回特高压/高压直流，并以某满功率3000MW的±500kV直流工程进行多工况初始化和启动性能测试；仿真平台记录为ADPSS。验证指标包括混合仿真建模耗时、直流功率/电压/电流初值偏差、触发角和关断角偏差、交直流无功交换偏差、箝位电源切除时间以及切除时接口冲击。页面给出的结果显示：单方式建模由约15小时降至0.5小时，15个方式可节省超过200小时；在300MW至3000MW工况下直流功率和电压误差小于1%，触发角/关断角偏差小于1°，无功交换偏差小于半组滤波器容量；箝位启动约0.6s达到稳态。优势主要体现在把“建模—工况调整—平稳启动—故障集扫描”形成可批量执行的工程流程，而不仅是单次仿真技巧。从验证范围看，论文主要证明了常规潮流运行点下的多直流混合仿真初始化和启动效率；对不同EMT步长、其他仿真平台、MMC-HVDC或新能源控制深度耦合场景、严重故障后的动态准确性与纯EMT全网基准之间的误差，页面未提供可核验的系统性结果。

### 4. 价值、认知与可复用场景

这项工作的核心认知是：大规模机电–电磁混合仿真的瓶颈不只在求解器速度，而在运行方式数据到电磁模型状态之间缺少可自动闭环的“翻译层”。通过标准化模型库、潮流到EMT初值的映射、以及接口箝位退出判据，论文把多回直流的混合仿真从人工工程配置问题转化为可批量生成和校核的问题。它适合被后续有关交直流大电网在线安全分析、多馈入直流换相失败扫描、混合仿真数据自动生成、HVDC模型初始化和接口启动策略的页面复用。不宜外推为所有电磁暂态问题的精度保证，也不能替代对具体直流控制保护模型、故障类型和仿真平台的再验证。

## 方法细节

### 方法概述

本文提出一种面向含大量电磁直流模型的机电-电磁混合仿真自动化技术框架。首先构建标准化电磁直流模型库，通过数据映射与拼接技术自动提取目标直流工程模型，实现混合仿真数据的快速建立。其次，基于潮流边界条件进行直流内部潮流计算，自动求解换流变抽头位置、交流滤波器投切组数及控制系统指令值，完成电磁模型工况的自动初始化与稳态微调。最后，在机电-电磁接口处引入幅值相角与潮流一致的理想箝位电压源，通过实时监测箝位电源输出功率，当其绝对值低于直流额定功率5%或达到设定时间时自动切除，实现混合仿真在0.6s内无暂态冲击平稳启动。该方法彻底替代了传统人工建模、手动调参和设定切除时间的繁琐流程，适用于大规模交直流电网的批量方式校核与在线分析。

### 数学公式

**公式1**: $$$V_0 = (U_d + \frac{3B X_C}{\pi} I_d) / \cos \alpha$$$

*计入换相过程的直流电压算式，用于计算理想空载直流电压，是求解换流变变比的基础*

**公式2**: $$$n = \frac{3\sqrt{2} B U_s}{\pi V_0}$$$

*换流变压器变比计算公式，根据交流母线电压、运行极数和理想空载电压确定抽头基准*

**公式3**: $$$\mu = \arccos(\cos \alpha - \frac{2n X_T I_d}{U_s}) - \alpha$$$

*换相角计算公式，用于评估换流器换相过程对无功消耗的影响*

**公式4**: $$$\tan \varphi = \frac{\frac{\pi \mu}{180} - \sin \mu \cos(2\alpha + \mu)}{\sin \mu \sin(2\alpha + \mu)}$$$

*直流功率因数角正切值计算，用于推导换流站无功消耗*

**公式5**: $$$Q_f = P_{DC} \tan \varphi - Q_s$$$

*交流滤波器需提供的无功补偿容量，由换流站总无功消耗与交流系统注入无功之差得出*

**公式6**: $$$(|P_s| \le 0.05 P_{Dref}) \lor (|P_s| \le P_{min}) \lor (T_{now} \ge T_{init})$$$

*接口箝位电源自动切除判据，满足任一条件即断开开关，实现无扰动退出*

### 算法步骤

1. 步骤1：构建标准化电磁直流模型库，建立模型库元件与方式计算/在线潮流数据中直流元件的映射关系表。

2. 步骤2：根据混合仿真分网预案确定电磁侧仿真范围，程序自动从模型库调取对应直流工程模型，完成机电侧大电网数据与电磁侧局部电网数据的自动拼接。

3. 步骤3：程序自动定义机电-电磁混合仿真接口，并在接口换流母线处并联由理想电压源与受控开关组成的临时箝位电源。

4. 步骤4：基于潮流边界数据执行直流内部潮流计算，依次求解换流变变比、换相角、功率因数及滤波器无功需求，自动计算换流变抽头位置（按式6/7四舍五入取整）和滤波器投切组数（按式8/9校验容量偏差）。

5. 步骤5：将计算得到的直流功率/电流指令、触发角/关断角参考值写入简化控制保护模型，启动混合仿真。在箝位电源支撑下，系统进入稳态校验阶段。

6. 步骤6：实时监测箝位电源输出功率$P_s$，若$|P_s| \le 0.05 P_{Dref}$或达到预设时间$T_{init}$，则自动断开受控开关切除箝位电源，保存稳态运行点并触发预设故障集进行批量扫描。

### 关键参数

- **整流侧触发角参考值**: 15°

- **逆变侧关断角参考值**: 17°

- **六脉冲换流器个数B**: ±500kV系统为2，±1000kV系统为4

- **换流变抽头调节分度**: 1.25%

- **箝位电源切除功率阈值**: 直流额定功率的5%

- **稳态建立目标时间**: 0.6 s

## 关键公式

### 直流换相电压方程

$$$V_0 = (U_d + \frac{3B X_C}{\pi} I_d) / \cos \alpha$$$

*用于根据潮流给定的直流电压、电流及触发角反推理想空载电压，是后续计算换流变变比和抽头位置的核心依据*

### 滤波器无功补偿计算方程

$$$Q_f = P_{DC} \tan \varphi - Q_s$$$

*在已知换流站有功功率、功率因数角及交流系统注入无功时，精确计算需投入的交流滤波器总容量，指导滤波器组数初始化*

### 箝位电源自动切除判据

$$$(|P_s| \le 0.05 P_{Dref}) \lor (|P_s| \le P_{min}) \lor (T_{now} \ge T_{init})$$$
