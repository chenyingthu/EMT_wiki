# On fixed-point iterations for the solution of control equations in power systems transients

**作者**: C.F. Mugombozi
**年份**: 2014
**来源**: `29/j.epsr.2014.03.035.pdf.pdf`

## 摘要

本文提出一种基于压缩映射定理的形式化分析方法，用于确定电磁暂态(EMT)仿真中控制系统方程的不动点(FP)迭代收敛性。方法通过图论技术（Coates图）在控制方程图上引入适当割集(cutset)以识别反馈回路路径，将控制系统分解为前向通路应用序列G和反馈函数φ。通过分析雅可比矩阵的谱范数(spectral norm)来验证压缩条件，建立FP方法的安全应用准则。当谱范数小于1时，采用简单的顺序块评估替代牛顿法，避免耗时的线性化过程和矩阵重构，在保证精度的同时提升计算效率。

## 研究解读

### 1. 需求、对象、挑战与贡献

EMT仿真中的控制系统常由大量控制块、非线性块和反馈互连构成；一旦存在代数环，简单按块顺序执行会失效，必须在每个时间步求解一组非线性控制方程。已有Newton法较系统、鲁棒，但需要线性化、构造/更新矩阵，工程实现和计算开销较高；直接采用fixed-point迭代虽只需顺序评估控制块，却难以预先判断是否会收敛，尤其当反馈路径含非线性块时，简单“断环”可能改变模型行为或导致失败。本文研究对象是电力系统暂态仿真中的控制方程反馈环，而不是电磁网络本身。创新点在于：把是否可安全使用FP迭代的问题形式化为反馈路径迭代函数的压缩性判定；借助Coates图和割集识别反馈变量，将控制系统拆成前向块序列和反馈映射，再用雅可比矩阵范数判定收敛条件，从而在执行FP前给出可操作的安全准则。

### 2. 模型、算法与实现技术

本文的核心实现思想是先不直接对整个控制框图做盲目的顺序求值，而是在图结构上找出形成回路的反馈变量。通过Coates图表示控制方程依赖关系，选取割集切断所有回路，得到一组被隔离的反馈变量e；其余控制块按无环前向路径组成应用序列G，给出输出y=G(e)。反馈路径再被写成e=φ(u,y)，其中u是外部输入，y是前向路径计算结果，φ是需要迭代的反馈映射。于是每个时间步的求解可写成e^{k+1}=φ(u,G(e^k))。压缩映射定理提供理论依据：若φ在所关心邻域内满足压缩条件，常用判据是相关雅可比矩阵的谱范数小于1，则FP迭代从合适初值出发会收敛到唯一不动点。与Newton法相比，该流程不构造非线性方程组的Newton修正矩阵，而是反复执行控制块的前向序列和反馈更新；残差或相邻迭代差用于判断是否达到预设容差。其机制价值在于把“能否用简单FP替代Newton”从经验选择变成基于反馈环导数性质的判定问题。

### 3. 验证、优势与不足

作者在EMTP-RV环境中说明算法，并将结果与Newton方法联系起来讨论；原文摘要明确给出的测试对象包括风电发电相关的实际控制系统和一个用户自定义电气机械模型案例，正文引言还说明该工作面向电力系统暂态仿真中的实际控制系统。验证重点不是提出新的物理设备模型，而是检查：当反馈迭代函数满足压缩性判据时，FP是否能在给定容差下得到与求解非线性控制方程一致的结果，并减少Newton法所需的线性化和矩阵形成过程。优势主要体现在实现路径更简单：每次迭代只顺序评估控制块，适合已有控制框图仿真器集成；同时，压缩性分析避免了无条件使用FP导致发散的风险。需要注意的是，提供的原文摘录未报告可核验的数值结果、运行时间表或具体误差曲线，因此不能据此声称固定百分比加速。其适用边界也很明确：若反馈路径的雅可比范数不满足压缩条件、步长或非线性导致局部导数放大，FP可能收敛慢或失败；这种情况下仍需Newton法、模型重组或改变仿真设置。

### 4. 价值、认知与可复用场景

这项工作提升的核心认知是：控制系统代数环并不必然都要用Newton法求解，也不能简单靠人工断环处理；关键在于反馈映射本身是否是压缩映射。它为EMT软件中的控制块求解器提供了一个“先分析反馈环、再选择求解器”的入口，可被后续关于控制框图自动排序、代数环检测、混合Newton/FP求解策略、风电控制器暂态仿真和用户自定义模型求解的页面复用。它不适合被外推为“FP总体优于Newton”，也不能脱离具体反馈路径、步长、非线性块和容差设置来判断收敛性或计算效率。

## 方法细节

### 方法概述

本文提出一种基于压缩映射定理的形式化分析方法，用于确定电磁暂态(EMT)仿真中控制系统方程的不动点(FP)迭代收敛性。方法通过图论技术（Coates图）在控制方程图上引入适当割集(cutset)以识别反馈回路路径，将控制系统分解为前向通路应用序列G和反馈函数φ。通过分析雅可比矩阵的谱范数(spectral norm)来验证压缩条件，建立FP方法的安全应用准则。当谱范数小于1时，采用简单的顺序块评估替代牛顿法，避免耗时的线性化过程和矩阵重构，在保证精度的同时提升计算效率。

### 数学公式

**公式1**: $$$e = \phi(e)$$$

*不动点迭代的基本形式，其中φ为向量函数，e为未知数向量*

**公式2**: $$$\|\phi(e) - \phi(\hat{e})\| < \alpha\|e - \hat{e}\|, \quad 0 \leq \alpha < 1$$$

*压缩映射条件，α为压缩常数，保证迭代收敛的充分条件*

**公式3**: $$$\|\phi'(\hat{e})\|_{spectral} < 1$$$

*基于雅可比矩阵谱范数的收敛判据，谱范数定义为矩阵A^TA最大特征值的平方根*

**公式4**: $$$e = \phi(u, y)$$$

*反馈变量方程，u为独立输入向量，y为前向通路输出*

**公式5**: $$$y = G(e)$$$

*前向通路应用序列，表示控制框图的顺序评估路径*

**公式6**: $$$\Psi = e - \phi(u, y) = 0$$$

*牛顿法的目标函数，用于与FP方法对比*

**公式7**: $$$e^{k+1} = \phi(G(e^k))$$$

*不动点迭代公式，k为迭代计数器*

**公式8**: $$$\Psi^{k+1} = e^{k+1} - \phi(G(e^k))$$$

*FP迭代的残差计算，用于收敛判断*

**公式9**: $$$\frac{\|\Psi^k - \Psi^{k+1}\|}{\|\Psi^k\|} \leq \varepsilon_{tol}$$$

*相对容差收敛判据，ε_tol为预定义的相对容差限值*

### 算法步骤

1. 在控制方程的Coates图上引入适当割集，消除所有循环路径，确保邻接矩阵全零特征值条件满足

2. 识别割集对应的反馈变量集合β̂，这些变量代表被切断的反馈回路路径

3. 构建前向通路应用序列G(e)，表示从反馈变量到输出的控制框图顺序评估路径

4. 构建反馈函数φ(u,y)，表示从输入和输出到反馈变量的映射关系

5. 计算反馈函数φ在不动点处的雅可比矩阵φ'(ê)，并计算其谱范数（即A^TA最大特征值的平方根）

6. 验证压缩条件：若谱范数<1，则判定FP迭代收敛，可进行步骤7；否则需采用牛顿法或减小步长

7. 初始化迭代计数器k=0，设置初始向量e^0（通常采用前一时间步的解）

8. 执行前向评估：计算y = G(e^k)，按控制框图顺序依次评估各模块

9. 执行反馈更新：计算e^{k+1} = φ(u, y)

10. 计算目标函数残差：Ψ^{k+1} = e^{k+1} - φ(G(e^k))

11. 检查收敛判据：计算相对误差‖Ψ^k - Ψ^{k+1}‖/‖Ψ^k‖，若小于ε_tol则收敛，进入步骤12；否则k=k+1，返回步骤8

12. 接受当前解e^{k+1}作为该时间步的解，进入下一时间步

### 关键参数

- **time_step**: Δt = 1 μs（PLL案例中小步长保证收敛）

- **relative_tolerance**: ε_tol ∈ {1e-3, 1e-4, 1e-5}

- **absolute_tolerance**: ε_abs ∈ {1e-8, 1e-9, 1e-10}

- **contraction_threshold**: 谱范数<1（压缩映射条件）

- **input_disturbance_time**: t = 125 ms（正弦输入幅值跳变时刻）

## 量化发现

- 时间步长Δt=1μs时，PLL系统的压缩条件得到满足，FP迭代收敛
- 容差组合1e-3/1e-8下，FP方法计算时间为牛顿法的96%，即慢4%
- 容差组合1e-4/1e-9下，FP方法与牛顿法计算时间比为1.00，速度相同
- 容差组合1e-5/1e-10下，FP方法计算时间为牛顿法的98%，即快2%
- 输入在t=125ms发生幅值跳变时，步长增大会导致FP方法收敛性能显著下降
- 非线性块（PROD、SIN、COS）的数值导数与梯形规则离散化效应是导致FP方法在大步长下性能劣化的主要原因

## 关键公式

### 压缩映射谱范数判据

$$$\|\phi'(\hat{e})\|_{spectral} < 1$$$

*用于预先判定给定控制系统是否适合采用不动点迭代，需计算反馈函数雅可比矩阵的谱范数（最大奇异值），若小于1则保证收敛*

### 不动点迭代公式

$$$e^{k+1} = \phi(G(e^k))$$$

*控制系统求解的核心迭代式，通过顺序评估前向通路G和反馈函数φ实现，无需构建和求解线性方程组*

### 相对收敛判据

$$$\frac{\|\Psi^k - \Psi^{k+1}\|}{\|\Psi^k\|} \leq \varepsilon_{tol}$$$

*实际仿真中采用的收敛判断标准，结合相对容差(ε_tol)和绝对容差控制求解精度，通常设为1e-4至1e-5量级*

## 验证详情

- **验证方式**: 对比验证（与牛顿法对比）和参数敏感性分析
- **测试系统**: 1) 含两个反馈回路(f3,f4)的锁相环(PLL)控制系统；2) 风力发电实际控制系统；3) 用户定义的电气机械模型
- **仿真工具**: EMTP-RV仿真引擎（基于梯形积分规则的电磁暂态仿真软件）
