# Rational approximation of frequency domain responses by vector fitting - Power Delivery, IEEE Transactions on

**作者**: Bjørn Gustavsen (M)
**年份**: 2004
**来源**: `32/VF_paper.pdf`

## 摘要

矢量拟合（Vector Fitting）是一种通过极点迁移实现有理函数逼近的两阶段迭代算法。该方法通过引入辅助缩放函数σ(s)，将原始非线性优化问题（同时优化极点和留数）转化为两个线性最小二乘问题。第一阶段通过拟合σ(s)和σ(s)f(s)来重新定位极点（极点识别），利用σ(s)的零点作为改进后的极点；第二阶段利用这些新极点重新拟合原始函数f(s)以确定留数（留数识别）。该方法的关键创新在于允许使用复数起始极点，从而有效处理包含多个谐振峰的非光滑频率响应，避免了传统方法中的病态条件问题。

## 研究解读

### 1. 需求、对象、挑战与贡献

电力系统电磁暂态仿真需要把元件和网络的频率相关效应准确带入时域模型；这些效应来自导体涡流、介质弛豫等，会使阻抗、导纳或RLC矩阵随频率变化。工程上常得到的是离散频域响应，若直接做全数值卷积，长时步仿真代价高；若用低阶有理函数替代，则可把卷积改写为递推形式。研究对象因此是测量或计算得到的频域响应，尤其是网络等值、变压器响应这类带多个谐振峰、需要复极点描述的响应。难点在于传统多项式比值拟合虽可线性化，但矩阵列含不同幂次的s，宽频段和高阶拟合时尺度差异导致病态；早期矢量拟合用实起始极点适合平滑函数，但对多谐振峰响应会失败。本文的贡献是在矢量拟合框架中允许复数起始极点，并讨论其基本性质和实现细节，使极点迁移法可用于含大量谐振峰的频域响应拟合。

### 2. 模型、算法与实现技术

本文采用部分分式形式表示目标有理函数：f(s)=Σ c_n/(s-a_n)+d+s h，其中极点和留数可为实数或共轭复数对，d和h为实系数。核心输入是离散频率点上的响应f(s)，输出是可用于时域递推卷积的极点a_n、留数c_n以及常数/比例项。算法不是同时非线性优化极点和留数，而是分两阶段把问题转成已知极点下的线性最小二乘。第一阶段给定一组起始极点，引入未知缩放函数σ(s)，并令σ(s)与σ(s)f(s)使用同一组起始极点进行有理逼近；这样可以通过线性方程求出缩放函数和缩放后函数的系数。机制上，σ(s)的零点被解释为改进后的极点，实现“pole relocation”。第二阶段固定这些新极点，重新对原始未缩放的f(s)做部分分式最小二乘，求得留数、d和h。本文相对早期实现的关键实现变化是：对有谐振峰的响应不再只用实起始极点，而采用复数起始极点，从一开始就给拟合基函数提供可表达振荡模态的位置，从而改善多峰响应下的极点识别。

### 3. 验证、优势与不足

原文说明用三类例子展示方法：人工构造的频率响应、实测变压器响应以及网络等值响应；这些对象覆盖了本文关注的多谐振峰电力系统频域建模场景。基线主要是两类已有思路：一是多项式比值拟合，其线性化后因s的高次幂造成尺度和条件数问题；二是早期矢量拟合，它使用实起始极点，对输电线建模中的平滑函数有效，但作者后来发现其在多谐振峰响应中会失败。指标层面，原文摘要和引言强调能得到有理函数近似、适合网络等值和变压器响应，并说明计算机代码由第一作者公开提供；但在给定抽取文本中未报告可核验的误差数值、阶数、运行时间或条件数。优势主要体现在方法机制上：复起始极点使极点迁移能覆盖谐振型响应，而共享极点的σ(s)与σ(s)f(s)构造保持了线性最小二乘求解形式。边界也应明确：验证集中在频域响应拟合本身，不等于已经验证任意EMT时域仿真场景下的稳定性、无源性、实时性或故障暂态精度；从抽取文本看，也没有给出与现代无源拟合、约束优化或多端口大规模算例的系统量化比较。

### 4. 价值、认知与可复用场景

这项工作的核心认知价值是把“高阶宽频有理拟合难”重新表述为“如何通过缩放函数迁移极点”的问题，并指出起始极点的类型决定了矢量拟合能否处理谐振响应。它适合被后续变压器宽频模型、网络等值、频率相关元件建模、EMT递推卷积实现等页面复用，尤其可作为理解VF、极点-留数模型和频域到时域接口的入口。不宜把本文直接外推为已解决所有工程约束：例如无源性保持、稳定极点处理策略、噪声鲁棒性、端口数扩展、实时仿真步长影响等，需要依赖后续论文或具体实现验证。

## 方法细节

### 方法概述

矢量拟合（Vector Fitting）是一种通过极点迁移实现有理函数逼近的两阶段迭代算法。该方法通过引入辅助缩放函数σ(s)，将原始非线性优化问题（同时优化极点和留数）转化为两个线性最小二乘问题。第一阶段通过拟合σ(s)和σ(s)f(s)来重新定位极点（极点识别），利用σ(s)的零点作为改进后的极点；第二阶段利用这些新极点重新拟合原始函数f(s)以确定留数（留数识别）。该方法的关键创新在于允许使用复数起始极点，从而有效处理包含多个谐振峰的非光滑频率响应，避免了传统方法中的病态条件问题。

### 数学公式

**公式1**: $$$$f(s) = \sum_{n=1}^{N} \frac{c_n}{s - a_n} + d + s h$$$$

*待拟合的有理函数形式，其中$c_n$和$a_n$为留数和极点（实数或共轭复数对），$d$和$h$为实数常数项和比例项*

**公式2**: $$$$(\sigma f)_{fit}(s) = \sigma_{fit}(s)f(s)$$$$

*增广系统的关键关系式，通过引入未知缩放函数σ(s)建立线性关系，其中$(\sigma f)_{fit}$和$\sigma_{fit}$共享相同的起始极点*

**公式3**: $$$$\sigma_{fit}(s) = \frac{\prod_{n=1}^{N}(s - z_n)}{\prod_{n=1}^{N}(s - \bar{a}_n)}, \quad (\sigma f)_{fit}(s) = \frac{\prod_{n=1}^{N+1}(s - \bar{z}_n)}{\prod_{n=1}^{N}(s - \bar{a}_n)}$$$$

*σ(s)和σ(s)f(s)的部分分式展开形式，其中$\bar{a}_n$为起始极点，$z_n$和$\bar{z}_n$为零点*

**公式4**: $$$$f(s) \approx \frac{(\sigma f)_{fit}(s)}{\sigma_{fit}(s)} = \frac{\prod_{n=1}^{N+1}(s - \bar{z}_n)}{\prod_{n=1}^{N}(s - z_n)}$$$$

*通过相除获得改进的有理逼近，其中σ(s)的零点$z_n$成为f(s)的新极点，实现极点迁移*

**公式5**: $$$$\bar{a}_n = -\alpha + j\beta, \quad \alpha = \frac{\beta}{100}$$$$

*复数起始极点的推荐配置策略，实部为虚部的1/100，确保足够小的实部以避免病态问题*

### 算法步骤

1. 阶段1-极点识别：指定一组起始极点$\bar{a}_n$（对于谐振响应使用复数共轭对，对于光滑响应使用实数），构建增广系统$\sigma(s)f(s)$和$\sigma(s)$，两者使用相同极点

2. 建立线性方程组：在采样频率点$s_k$处建立最小二乘问题$Ax=b$，其中未知向量$x$包含$c_n$、$\tilde{c}_n$、$d$、$h$、$\tilde{d}$、$\tilde{h}$（带波浪号项对应σ(s)的留数）

3. 求解最小二乘问题：通过正规方程或QR分解求解超定系统，获得$\sigma_{fit}(s)$和$(\sigma f)_{fit}(s)$的部分分式展开系数

4. 计算新极点：求解$\sigma_{fit}(s)=0$的根（即$\sigma_{fit}(s)$的零点），这些零点构成改进后的极点集$a_n^{new}$。若极点实部为正（不稳定），则取实部负号强制稳定

5. 阶段2-留数识别：使用阶段1获得的新极点$a_n^{new}$作为固定极点，重新建立关于原始函数$f(s)$的线性最小二乘问题$Ax=b$，此时仅求解留数$c_n$、常数项$d$和比例项$h$

6. 迭代优化：将阶段2获得的极点作为新的起始极点，重复阶段1-2直至收敛（通常1-3次迭代即可达到机器精度）

### 关键参数

- **起始极点数量**: 通常等于预期系统阶数N，对于18阶系统使用20个极点（10对复数共轭）

- **复数极点分布**: 虚部在频率范围内线性分布，实部按虚部的1/100计算：$\alpha = \beta/100$

- **实数极点分布**: 对于光滑函数，沿频率轴线性或对数分布

- **收敛判据**: 相邻迭代间极点变化小于阈值或残差RMS误差小于$10^{-10}$量级

## 量化发现

- 使用复数起始极点且实部为虚部1/100（即$\alpha = \beta/100$）时，可完全消除宽频带拟合中的病态条件问题
- 对于18阶测试函数，20阶矢量拟合的RMS误差为3.8×10⁻¹²，40阶拟合的RMS误差为1.6×10⁻¹²，达到双精度机器精度量级
- 极点估计精度：相对误差约10⁻⁷；留数估计精度：相对误差约10⁻⁷
- 算法收敛性：通常1-3次迭代即可收敛，新极点作为起始极点使用时残差不再显著下降
- 计算效率：向量拟合处理多端口系统时，所有元素共享相同极点，使时域卷积计算速度提高约2倍
- 不稳定极点处理：通过简单地将正实部取负（$a_n^{stable} = -\text{Re}(a_n) + j\text{Im}(a_n)$），可确保最终模型稳定

## 关键公式

### 部分分式展开形式的有理函数

$$$$f(s) = \sum_{n=1}^{N} \frac{c_n}{s - a_n} + d + s h$$$$

*作为待逼近的目标函数形式，用于网络等值、变压器模型和传输线频变参数的宽频建模*

### 增广系统方程（标准化形式）

$$$$\sigma_{fit}(s) = 1 + \sum_{n=1}^{N} \frac{\tilde{c}_n}{s - \bar{a}_n}, \quad (\sigma f)_{fit}(s) = \sum_{n=1}^{N} \frac{c_n}{s - \bar{a}_n} + d + s h$$$$

*阶段1中使用，强制$\sigma(s)$在高频趋近1以消除解的歧义性，通过最小二乘同时拟合$\sigma(s)$和$\sigma(s)f(s)$*

### 极点迁移机制

$$$$f(s) \approx \frac{(\sigma f)_{fit}(s)}{\sigma_{fit}(s)} \Rightarrow \text{zeros of } \sigma_{fit}(s) \text{ become poles of } f(s)$$$$

*矢量拟合的核心数学原理，通过求解$\sigma_{fit}(s)$的零点获得改进的极点位置，实现从起始极点到最优极点的迁移*

## 验证详情

- **验证方式**: 数值验证与对比分析
- **测试系统**: 人工构造的18阶严格真有理函数，包含密集谐振峰（45000Hz附近存在两个不同阻尼的谐振模式），以及实测变压器频率响应数据
