# Z.function convolution in EHV

**作者**: W D Humpage; K P Wong T T Nguyen
**年份**: 2002
**来源**: `40/0142-0615%2883%2990035-2.pdf.pdf`

## 摘要

本文提出了一种基于z平面函数卷积的电磁暂态分析方法，作为传统频域-时域变换卷积的替代方案。该方法将z平面作为频域和时域之间的中间变换步骤，几乎完全在z平面内建立完整的分析公式，仅将最终所需的方程组通过离散函数卷积从z平面变换到时域。核心创新在于通过引入中间变量，将复杂的z域有理函数乘积转换为简洁的递归求解序列，实现了传输线前向脉冲响应和浪涌阻抗函数的高效数值计算。方法特别处理了波阻抗函数的分离，将其分解为瞬时常数部分和动态部分，确保数值稳定性。

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求来自EHV输电网络电磁暂态计算：线路合闸、残留电荷释放、短路故障等过程由频率相关传输线参数、行波传播延迟、端部边界条件共同决定。传统做法常从频域响应反变换到时域，再进行时间卷积；这类卷积需要保存和累加较长历史项，且频率相关线路函数若近似不当会影响数值稳定性。本文研究对象是EHV传输线的前向脉冲响应函数和浪涌阻抗函数，以及它们嵌入电源、开关端、远端或故障点边界条件后的递归暂态求解。难点在于：传播延迟、衰减、频率相关波阻抗和多相模态变换必须同时进入离散时间计算；若直接处理z域有理函数乘积或时域卷积，公式复杂且不利于逐步递推。本文的贡献不是简单把卷积换一种写法，而是把z平面作为频域到时域之间的中间层：先在z平面建立完整边界方程，再只把最终需要求解的z函数卷积转化为离散递归序列。通过引入中间变量，前向脉冲响应和浪涌阻抗的有理函数乘积被拆成分母递推和分子加权输出，并在综合这些函数时设置稳定性检查，使线路函数作为稳定系统函数进入网络求解。

### 2. 模型、算法与实现技术

算法以模态域单模线路函数为基础，再回到多相向量形式。前向脉冲响应写为F_Q(z)=z^{-m}/(1+Σb_k z^{-k})，其中z^{-m}表示波传播时间相对采样间隔Δt的整数延迟；浪涌阻抗写为Z_0(z)=Z_c(1+Σc_k z^{-k})/(1+Σd_k z^{-k})，描述频率相关波阻抗。计算机制的关键是把这类z域乘积变成状态化递推。例如对F_Q(z)与输入g_1(z)的卷积，不直接展开全历史卷积，而定义中间变量W_Q(n)，先用当前延迟输入g_1(n-m)和若干历史W_Q(n-k)递推分母动态，再用分子系数对W_Q的当前/历史值加权得到输出f_1(n)。对Z_0(z)则先分离为瞬时项Z_c和动态项Z_d(z)，使当前步电压电流关系中保留一个可直接求解的常数阻抗/导纳部分，其余频率相关效应只通过历史递推项进入；这避免当前未知量被动态卷积隐式缠绕。接口量包括开关端、远端或故障点的电压电流v(z)、i(z)，源阻抗z_s(z)，以及模态—相域变换矩阵C_1、C_2。整体流程是在z平面写线路特性方程、源模型和边界条件，形成求解所需方程；每个时间步先更新各线路函数的中间变量和历史卷积项，再求端口电压电流，最后更新历史量。

### 3. 验证、优势与不足

作者用代表性暂态算例与标准time-convolution analysis进行对比验证。原文明确说明应用场景包括：等效电源模型下的输电线路合闸，以及短路故障操作中的电磁暂态；页面抽取还给出一条160 km、400 kV EHV线路合闸算例，并提到线路带残留电荷、线路参数见附录。验证基线是早期频域到时域转换形成的标准时间卷积方法，而不是商业EMTP软件或现场实测；原文未报告可核验的计算时间、误差百分比、峰值偏差等数值结果，因此只能说作者报告了代表性波形的接近对比，不能量化其加速或精度提升。优势主要体现在机制层面：z域有理函数被转化为短递归序列，减少直接长卷积的计算负担；浪涌阻抗拆成瞬时常数项和历史动态项，使网络方程在当前步更容易求解；线路前向脉冲响应和浪涌阻抗在z平面综合时经过稳定性检查，有助于保证嵌入网络后的数值稳定性。从验证范围看，论文没有证明该方法适用于任意复杂拓扑、强非线性元件、电力电子控制器、非整数传播延迟处理、极端宽频模型或实时仿真硬件；也没有给出不同步长、不同拟合阶数M_1/M_2下的系统误差和稳定裕度统计。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：频率相关传输线暂态并不一定要先形成显式时域核再做时间卷积，也可以先在z平面完成线路函数和边界条件的组合，然后把最终z函数卷积降阶为递归滤波形式。它为后续数字滤波线路模型、递归卷积、频率相关EMT线路等值提供了一个清晰入口：哪些部分应作为当前步瞬时导纳进入网络方程，哪些部分应作为历史状态递推保存。它适合被用于研究EHV线路合闸、故障行波、频率相关传输线接口模型，以及比较z域递归卷积与传统时域卷积的页面复用。不适合被外推为对所有EMT场景都更快或更准；若缺少原文具体系数综合方法、步长选择和算例波形，不能把该页当作定量性能证据。

## 方法细节

### 方法概述

本文提出了一种基于z平面函数卷积的电磁暂态分析方法，作为传统频域-时域变换卷积的替代方案。该方法将z平面作为频域和时域之间的中间变换步骤，几乎完全在z平面内建立完整的分析公式，仅将最终所需的方程组通过离散函数卷积从z平面变换到时域。核心创新在于通过引入中间变量，将复杂的z域有理函数乘积转换为简洁的递归求解序列，实现了传输线前向脉冲响应和浪涌阻抗函数的高效数值计算。方法特别处理了波阻抗函数的分离，将其分解为瞬时常数部分和动态部分，确保数值稳定性。

### 数学公式

**公式1**: $$$$F_Q(z) = \frac{z^{-m}}{1 + \sum_{k=1}^{M_1} b_k z^{-k}}$$$$

*传输线前向脉冲响应函数，其中z^{-m}表示波传播延迟，m为波传播时间与采样间隔的比值，分母为M1阶多项式表示衰减特性*

**公式2**: $$$$Z_0(z) = Z_c \frac{1 + \sum_{k=1}^{M_2} c_k z^{-k}}{1 + \sum_{k=1}^{M_1} d_k z^{-k}}$$$$

*传输线浪涌阻抗函数（波阻抗），Zc为特性阻抗，分子分母分别为M2阶和M1阶多项式，用于拟合频率依赖特性*

**公式3**: $$$$Z_0(z) = Z_c + Z_d(z) = Z_c + \frac{Z_c N_0(z)}{1 + D_0(z)}$$$$

*浪涌阻抗函数的分离形式，将阻抗分解为瞬时部分Zc（当前步）和动态部分Zd(z)（历史步），确保递归计算时数值稳定性*

**公式4**: $$$$W_Q(n) = g_1(n-m) - \sum_{k=1}^{m_1} D_Q(k) W_Q(n-k)$$$$

*前向脉冲响应卷积的中间变量递归方程，将输入函数g1延迟m个采样点后，与历史中间变量值加权求和*

**公式5**: $$$$f_1(n) = F\left[W_Q(n) + \sum_{k=1}^{m_1} N_Q(k)W_Q(n-k)\right]$$$$

*前向脉冲响应输出的最终递归计算式，F为传播衰减系数，N_Q(k)为z平面分子多项式系数，*表示离散序列卷积*

**公式6**: $$$$W_0(n) = g_0(n) - \sum_{k=1}^{M_2} D_0(k) W_0(n-k)$$$$

*浪涌阻抗函数卷积的中间变量递归方程，用于计算波阻抗的卷积积分，避免直接计算完整的卷积和*

**公式7**: $$$$f_0(n) = Z_c g_0(n) + Z_c \sum_{k=1}^{M_2} N_0(k) W_0(n-k)$$$$

*浪涌阻抗卷积输出，包含瞬时项Zc*g0(n)和动态历史项的加权和，其中N0(k)=c_k-d_k*

**公式8**: $$$$i_s(z) = C_2^{-1} Y_c \{V_s^p(z) - F_Q^p(z) v_r^p(z) - [Z_{0a}^p(z) + Z_s^p(z)] i_s^p(z)\}$$$$

*合闸分析中的开关端电流求解方程，结合模态变换矩阵C1、C2和相域变量，Yc为组合导纳矩阵*

### 算法步骤

1. 将传输线频率依赖参数（衰减常数、相位常数、特性阻抗）通过z变换转换为z域有理函数形式，得到F_Q(z)和Z_0(z)的分子分母多项式系数

2. 对浪涌阻抗函数Z_0(z)进行代数分离，分解为常数项Z_c（表示瞬时响应）和剩余有理函数项Z_d(z)（表示频率依赖的历史响应），确保当前步计算仅依赖历史值

3. 针对前向脉冲响应F_Q(z)，引入中间变量W_Q(z)构造辅助方程，将原始的复杂有理函数乘积转换为W_Q(n) = g_1(n-m) - ΣD_Q(k)W_Q(n-k)的递归形式，其中m为传播延迟对应的采样点数

4. 计算前向脉冲响应输出f_1(n) = F[W_Q(n) + ΣN_Q(k)W_Q(n-k)]，其中F为线路传播衰减因子，N_Q(k)为分子多项式系数，实现仅通过历史W_Q值和当前输入计算输出

5. 对浪涌阻抗函数执行类似递归转换，建立W_0(n) = g_0(n) - ΣD_0(k)W_0(n-k)的递归关系，并计算输出f_0(n) = Z_c g_0(n) + Z_c ΣN_0(k)W_0(n-k)

6. 构建完整的网络方程组：结合传输线特性方程v_s(z) - Z_0(z)i_s(z) = F_Q(z)v_r(z)和电源模型方程V_s(z) = V_f(z) - Z_s(z)i_s(z)，在z平面建立边界条件方程

7. 执行模态-相域变换：通过实模态变换矩阵C1、C2（在选定频率下计算）将解耦的模态变量转换为相域变量，v^p = C1 v，i^p = C2 i

8. 递归时域求解：从n=0开始，在每个时间步长Δt内，先计算中间变量W_Q(n)和W_0(n)，再计算线路端电压电流，最后更新历史数据用于下一步计算，直到仿真结束

### 关键参数

- **m**: 波传播时间与采样间隔的比值（transit time / Δt），决定传输线延迟的采样点数

- **Δt**: 时域采样间隔，单位秒

- **M1**: 前向脉冲响应分母多项式阶数，决定衰减特性的建模精度

- **M2**: 浪涌阻抗函数分子和分母多项式阶数，决定频率依赖特性的拟合精度

- **Zc**: 线路特性阻抗（ surge impedance ），标称值

- **F**: 传播常数相关的衰减系数，F = e^{-αl}，α为衰减常数，l为线路长度

- **NQ(k), DQ(k)**: z平面分子和分母多项式的第k项系数，在时域卷积中作为权重系数

- **C1, C2**: 实模态变换矩阵，用于相域和模态变量间的转换，C1为电压变换矩阵，C2为电流变换矩阵

## 关键公式

### 前向脉冲响应函数

$$$$F_Q(z) = \frac{z^{-m}}{1 + \sum_{k=1}^{M_1} b_k z^{-k}}$$$$

*描述传输线信号正向传播特性的z域传递函数，用于计算行波从一端传播到另一端并衰减的过程，是构建递归卷积算法的基础*

### 递归卷积核心方程

$$$$W_Q(n) = g_1(n-m) - \sum_{k=1}^{m_1} b_k W_Q(n-k)$$$$

*将复杂的z域有理函数卷积转换为时域递归计算的关键方程，通过引入中间变量W_Q，将无限脉冲响应(IIR)滤波器形式的传输线模型转换为可实时计算的差分方程*

### 合闸暂态求解方程

$$$$i_s(z) = C_2^{-1} Y_c \{V_s^p(z) - F_Q^p(z) v_r^p(z) - [Z_{0a}^p(z) + Z_s^p(z)] i_s^p(z)\}$$$$
