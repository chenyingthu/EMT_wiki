## 方法细节

### 方法概述

本文提出一种基于切向插值与Loewner矩阵（LM）铅笔的非迭代FDNE建模方法。该方法直接从多端口网络的频域导纳响应数据出发，通过引入左右切向方向向量对矩阵数据进行降维采样，构建Loewner矩阵与移位Loewner矩阵。利用奇异值分解（SVD）分析矩阵铅笔的奇异值衰减特性，自动识别系统最优阶数并提取投影子空间，进而直接计算广义描述符状态空间矩阵（E, A, B, C）。针对有理逼近中常见的不规则部分（D与Y∞矩阵），设计了专门的提取与分离算法，确保最终状态空间模型在时域EMT仿真中的数值稳定性。整个过程无需初始极点猜测，彻底规避了传统矢量拟合（VF）的迭代收敛问题，并在多数情况下天然保持无源性。

### 数学公式


**公式1**: $$$E\dot{x}(t) = Ax(t) + Bu(t), \quad y(t) = Cx(t) + Du(t) + Y_\infty \dot{u}(t)$$$

*广义描述符状态空间模型，用于表示FDNE的时域动态，其中E可能奇异，Y∞为高频不规则项*


**公式2**: $$$H(s) = C(sE - A)^{-1}B + D + sY_\infty$$$

*系统传递函数，包含正则部分与不规则部分，是频域拟合的目标函数*


**公式3**: $$$\mathbb{L}_{ij} = \frac{v_j r_i - l_j w_i}{\mu_j - \lambda_i}$$$

*Loewner矩阵元素构造公式，基于左右插值点与切向响应数据的差商形式*


**公式4**: $$$H(\lambda_i)r_i = w_i, \quad l_j H(\mu_j) = v_j$$$

*切向插值约束条件，定义多端口网络在特定频率与方向上的输入输出映射关系*


### 算法步骤

1. 1. 频域数据采集与预处理：在目标频率范围[s_min, s_max]内采样多端口网络的导纳矩阵Y(s)，获取复频域响应数据，并剔除明显噪声点。

2. 2. 数据划分与切向方向定义：将采样频率点集S划分为右插值点集{λ_1,...,λ_k}与左插值点集{μ_1,...,μ_h}。针对多端口特性，定义右切向方向矩阵R∈C^{m×k}与左切向方向矩阵L∈C^{h×p}，以降低矩阵维度并保留端口耦合信息。

3. 3. 构造Loewner矩阵与移位矩阵：利用差商公式计算Loewner矩阵L与移位Loewner矩阵L_σ。其中L_ij=(v_j r_i - l_j w_i)/(μ_j - λ_i)，L_σ对应sH(s)的插值数据，两者共同构成系统矩阵铅笔。

4. 4. SVD降阶与阶数辨识：对Loewner矩阵进行奇异值分解L = UΣV^H。根据奇异值谱的显著衰减拐点（或设定能量阈值如99.9%）自动确定系统最优阶数n，并截取前n个奇异向量构建左右投影矩阵。

5. 5. 状态空间矩阵投影计算：利用投影矩阵将L与L_σ映射为广义状态空间矩阵：E = -V_n^H L U_n, A = -V_n^H L_σ U_n, B = V_n^H R, C = L U_n。

6. 6. 不规则部分提取与稳定性处理：通过高频渐近行为分离D与Y∞矩阵，将其从状态空间模型中剥离，确保剩余正则部分(sE-A)可逆且极点严格位于左半平面，获得稳定模型。

7. 7. 无源性验证与EMT集成：计算全频段Re(Y)矩阵特征值验证无源性，并将提取的(E,A,B,C,D,Y∞)转换为EMT仿真器兼容的节点导纳或受控源形式，完成等值模型部署。


### 关键参数

- **插值点划分策略**: 左右数据集比例直接影响拟合精度，推荐采用交叉或均匀划分（如50%/50%）

- **切向方向选择**: 通常采用单位向量或基于端口物理特性的正交基，用于多端口降维

- **SVD截断阈值**: 基于奇异值能量占比（通常>99.5%）或数值噪声底限自动确定阶数

- **频率采样范围**: 覆盖EMT仿真关注的全频段（如1Hz~10kHz），需包含谐振峰与高频衰减区



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 典型多端口输电网络等值 | 在4个不同规模电力系统测试案例中，LM方法成功拟合导纳矩阵，全频段相对拟合误差均控制在0.8%以内，自动识别的模型阶数与物理网络主导模态一致。 | 与矢量拟合（VF）相比，拟合精度相当（误差差值<0.2%），但无需迭代，计算时间缩短约40%，且完全避免初始极点选择不当导致的发散问题。 |

| 含电力电子接口的复杂网络 | 针对高频开关器件引起的宽频响应，LM方法在10Hz-5kHz范围内保持高精度逼近，阶跃响应无振荡发散，实部导纳矩阵特征值全频段大于0。 | VF方法需额外执行无源性强制算法（耗时增加25%），而LM方法天然保持无源性，模型提取后直接满足EMT稳定性要求。 |



## 量化发现

- 全频段导纳矩阵拟合相对误差<0.8%，最大幅值偏差<1.2%，相位偏差<0.5°
- 模型阶数自动识别准确率100%，SVD奇异值谱拐点清晰，无需人工试凑
- 非迭代特性使单次建模计算时间较传统VF减少30%~50%，且无收敛失败记录
- 数据划分策略对精度影响显著：左右插值点比例偏离50:50超过20%时，拟合误差上升约3~5倍
- 无源性验证通过：全频段Re(Y)矩阵最小特征值>0，阶跃响应稳态误差<0.1%，无负阻尼振荡


## 关键公式

### Loewner矩阵差商构造式

$$$\mathbb{L}_{ij} = \frac{v_j r_i - l_j w_i}{\mu_j - \lambda_i}$$$

*用于从切向插值数据直接构建系统矩阵，是连接频域采样与状态空间实现的核心桥梁*

### 广义描述符传递函数

$$$H(s) = C(sE - A)^{-1}B + D + sY_\infty$$$

*FDNE频域等值的目标模型形式，包含正则动态与高频不规则项，用于EMT时域离散化*

### SVD投影状态矩阵提取式

$$$E = -V_n^H \mathbb{L} U_n, \quad A = -V_n^H \mathbb{L}_\sigma U_n$$$

*通过截断SVD子空间将Loewner矩阵映射为降阶状态空间参数，实现非迭代模型降阶*



## 验证详情

- **验证方式**: 频域响应误差分析、时域阶跃响应仿真、全频段无源性特征值验证、与矢量拟合（VF）基准对比
- **测试系统**: 4个典型电力系统测试网络（涵盖多端口输电线路、含分布式电源的配电网及VSC-HVDC接口等值场景）
- **仿真工具**: MATLAB（算法实现与SVD计算）、PSCAD/EMTDC或EMTP-RV（时域EMT仿真验证）、自定义导纳矩阵采样脚本
- **验证结果**: LM方法在精度、稳定性与无源性方面与工业标准VF方法相当，但凭借非迭代特性彻底消除收敛风险，自动阶数辨识功能显著提升建模效率。数据划分策略的优化是保证高精度的关键，提取不规则部分后的模型可直接用于大规模EMT暂态仿真，验证了其在现代电力系统宽频等值中的工程实用性。
