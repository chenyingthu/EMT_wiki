## 研究解读

### 1. 需求、对象、挑战与贡献

电磁暂态仿真需要用微秒甚至纳秒级步长跟踪开关动作、线路行波、电感电流或电容电压突变等快速过程，传统EMTP类程序多按时间步串行推进，时间步数很大时实时性受限。本文研究对象不是某一具体装置控制策略，而是电力系统电磁暂态微分动力系统的数值积分并行化，尤其是基于临界阻尼调整法（CDA）的时域求解过程。难点在于：隐式梯形法虽A稳定但缺乏阻尼，突变时会出现数值振荡；CDA通过在突变处切换隐式欧拉法抑制振荡，但仍是递推式时间串行；已有parareal方法用于高振荡系统可能不收敛，paraexp又主要适合线性系统。本文的贡献是把整个时间窗口内的CDA离散方程写成全局代数系统，通过对由步长和积分参数构成的下三角矩阵H进行实特征值分解，将时间耦合系统变换为多个相互独立的子问题，从而在保持CDA阻尼机制的同时实现时间维度并行；并进一步给出非线性系统下基于牛顿迭代和同一对角化思想的时间解耦求解框架。

### 2. 模型、算法与实现技术

算法从一般微分系统x'=f(x)+g(t)出发，在时间窗口内选取N个网格点，用统一积分格式x_n=x_{n-1}+h_n[(1-α_n)ẋ_{n-1}+α_nẋ_n]表示CDA：正常时α=1/2对应隐式梯形法，突变时α=1对应隐式欧拉法。将所有时间点状态堆叠为X=[x(t_1)^T…x(t_N)^T]^T，把差分关系写成(A⊗I_m)X=(B⊗I_m)F(X)+(B⊗I_m)G(t)，再令H=B^{-1}A，得到(H⊗I_m)X-F(X)=G(t)。这里核心状态量是各时间点的m维状态x_i；输入包括初值x_0、外部激励g(t_i)、时间步长h_i和积分参数α_i；输出是整个时间窗口上的状态轨迹X。关键机制是H由时间离散格式决定，若各α_i h_i使特征值λ_i=1/(α_i h_i)互异，则H可实对角化H=PDP^{-1}。线性情形下，经(P^{-1}⊗I_m)变换后，原Nm维耦合问题被拆成N个m维问题(λ_kI_m-U)y_k=ĝ_k，可并行求解。非线性情形下，作者给出牛顿迭代框架：计算残差，利用P^{-1}变换残差，在平均状态附近构造雅可比J，分别求解[λ_kI_m-J]Δy_k=ṙ_k，再经P反变换更新全时间窗口状态。为保证H可对角化，文中采用完全变步长策略，如渐近平均法使h_k在T/N附近小幅偏离且互异。

### 3. 验证、优势与不足

作者用数值算例验证方法，测试对象为高压长输电线路空载合闸电磁暂态过程，线路由电报方程描述，并通过空间离散形成常微分方程系统；页面抽取信息显示空间离散段数M=50、状态维数为2M+1=101。验证基线主要是传统串行CDA计算思路，目的在于说明所提方法能把同一时间窗口内的CDA离散方程解耦并并行求解，同时保留CDA对突变数值振荡的抑制能力。原文摘要称算例具有良好时间并行性并获得有效加速比，但当前证据片段未给出可核验的运行时间、处理器数量、并行效率、误差范数或加速比数值，因此不能把“显著提高”转写为定量结论。优势主要体现在算法结构上：把Nm阶全局问题转化为N个m阶子问题，时间点之间不再递推依赖；相较parareal，作者强调其对高振荡电磁暂态问题更适配；相较paraexp，框架覆盖线性和非线性微分系统。边界也很明确：验证算例主要是线路空载合闸和101维线性系统，未见对大规模复杂电网、强非线性电力电子开关网络、控制器模型、故障组合、商业EMTP/PSCAD结果或硬件实时平台的系统对比；完全变步长和特征值互异是算法成立条件之一，也可能影响与固定步长EMT程序的接口兼容性。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：电磁暂态时间并行不一定只能依赖预测-校正式parareal，也可以从隐式积分离散矩阵本身入手，通过选择步长和CDA参数使时间耦合矩阵可对角化，从代数层面实现严格解耦。它适合被后续研究复用为EMT时间窗口并行求解器、CDA并行化方案、状态空间线路模型加速计算方法，或作为比较parareal、paraexp及多速率EMT算法的基准思路。工程上可用于探索实时仿真、高压线路暂态、需要大量小步长计算的离线批量仿真加速。不适合直接外推为已在复杂电网、商业仿真平台或硬件实时系统中成熟可用；也不能在没有误差和加速比数据时声称其相对所有并行EMT方法更优。

## 方法细节

### 方法概述

本文提出了一种基于临界阻尼调整(CDA)算法与矩阵实特征值分解(对角化)相结合的电磁暂态时间并行计算方法。该方法将时间离散后的微分方程组表示为大规模代数方程组$(H \otimes I_m)X - F(X) = G(t)$，其中$H$为下三角矩阵。通过对$H$进行特征值分解$H = PDP^{-1}$，将原问题解耦为N个独立的子问题，实现时间维度上的并行计算。对于非线性系统，采用牛顿迭代法，并在每次迭代中利用相同的对角化技术解耦雅可比矩阵，形成严格的时间解耦并行牛顿算法。

### 数学公式

**公式1**: $$x_n = x_{n-1} + h_n[(1-\alpha_n)\dot{x}_{n-1} + \alpha_n\dot{x}_n]$$

*通用数值积分公式，当$\alpha_n=1/2$时为隐式梯形法，$\alpha_n=1$时为隐式欧拉法（CDA策略）*

**公式2**: $$(A \otimes I_m)X = (B \otimes I_m)F(X) + (B \otimes I_m)G(t)$$

*时间离散后的全局代数方程组，其中$A$和$B$分别为差分矩阵和权重矩阵*

**公式3**: $$H = B^{-1}A = PDP^{-1}$$

*矩阵$H$的实特征值分解，$D$为特征值对角矩阵，$P$为特征向量矩阵，实现对角化*

**公式4**: $$(\lambda_k I_m - U)y_k = \tilde{g}_k(t), \quad k \in (1,N)$$

*解耦后的线性子问题，各时间网格点可并行求解，$\lambda_k$为第$k$个特征值*

**公式5**: $$[\lambda_k I_m - J(x^{\eta})]\Delta y_k^{\eta} = \tilde{r}_k(X^{\eta})$$

*非线性系统的牛顿迭代修正方程，雅可比矩阵$J$在各时间点上独立求解*

**公式6**: $$h_k = \frac{(1+\varepsilon)^k}{\sum_{j=1}^N (1+\varepsilon)^j}T, \quad \varepsilon=\pm 0.01$$

*渐近平均法步长选择公式，确保各时间步长互异以满足对角化条件*

### 算法步骤

1. 时间网格配置：选定N个时间网格点，确定时间窗口长度T，采用渐近平均法或Radau网格法选择互异的时间步长$h_k$，满足$\sum_{k=1}^N h_k = T$且各$h_k$互不相同

2. 积分方法选择：依据CDA策略，在检测到突变时刻采用隐式欧拉法($\alpha_k=1$)，正常工况采用隐式梯形法($\alpha_k=1/2$)，确定各点参数$\alpha_k$

3. 初始化：置迭代次数$\eta=0$；在各时间网格点$k$赋予状态变量初值$x_k^{\eta}$；依据$x^{\eta} = \frac{1}{N}\sum_{k=1}^N x_k^{\eta}$计算平均向量作为牛顿迭代基准

4. 并行残差计算：依据方程(7)并行计算$F(X^{\eta})$，依据方程(8)并行计算$G(t)$，得到残差$R(X^{\eta}) = G(t) + F(X^{\eta}) - (H \otimes I_m)X^{\eta}$

5. 残差变换：利用特征向量矩阵计算变换后的残差$\tilde{r}_k(X^{\eta})$，即$\tilde{R}(X) = (P^{-1} \otimes I_m)R(X)$

6. 雅可比计算：依据$J(x^{\eta}) = \frac{\partial f(x)}{\partial x}|_{x=x^{\eta}}$计算平均点处的雅可比矩阵

7. 并行线性求解：并行求解N个独立的m维线性系统$[\lambda_k I_m - J(x^{\eta})]\Delta y_k^{\eta} = \tilde{r}_k(X^{\eta})$，得到$\Delta Y^{\eta}$

8. 状态更新：通过反变换$\Delta X^{\eta} = (P \otimes I_m)\Delta Y^{\eta}$计算修正量，更新$X^{\eta+1} = \Delta X^{\eta} + e_N \otimes x^{\eta}$

9. 收敛判断：检查牛顿迭代收敛性，若未收敛则令$\eta = \eta+1$，重新计算平均向量并返回步骤4；若收敛则进入下一时间窗口

### 关键参数

- **N**: 时间网格点数量，决定并行度

- **alpha_k**: 积分方法参数，取值为0.5（梯形法）或1（欧拉法）

- **epsilon**: 步长调整系数，取值为±0.01，确保步长互异

- **lambda_k**: 矩阵H的第k个特征值，$\lambda_k = 1/(\alpha_k h_k)$

- **m**: 系统状态变量维度（测试案例中m=101）

- **M**: 空间离散段数（测试案例中M=50）

## 量化发现

- 系统维度：测试案例采用101维状态空间（2×50+1，其中50为空间离散段数）
- 空间离散：输电线路均分为M=50段进行空间离散
- 时间步长：必须采用完全变步长策略，所有步长$h_k$必须互异以满足矩阵对角化条件
- 步长选择参数：渐近平均法中$\varepsilon$取±0.01，确保$h_k$在T/N附近微小偏离
- 特征值性质：矩阵H的特征值为实数且互异，$\lambda_k = 1/(\alpha_k h_k)$，保证可对角化
- 适用时间尺度：电磁暂态计算时间步长为μs级，特快速暂态过电压(VFTO)计算为ns级
- 算法复杂度：将大规模$(Nm)×(Nm)$矩阵求解解耦为N个独立的$m×m$矩阵并行求解

## 关键公式

### 解耦前的全局雅可比矩阵

$$H \otimes I_m - I_N \otimes J(x^{\eta})$$

*牛顿迭代中需要求解的大型块矩阵，通过特征值分解可解耦*

### 对角化后的解耦矩阵

$$D \otimes I_m - I_N \otimes J(x^{\eta})$$

*经$P^{-1} \otimes I_m$相似变换后，成为块对角矩阵，各块$\lambda_k I_m - J(x^{\eta})$可独立并行求逆*

### 特征向量矩阵元素

$$p_{ij} = \frac{\prod_{k=j+1}^{i}[(1-\alpha_k)h_k + \alpha_j h_j]}{\prod_{k=j+1}^{i}(\alpha_j h_j - \alpha_k h_k)}$$

*下三角矩阵H的特征向量矩阵P的解析表达式，用于实际计算中的矩阵向量乘法*
