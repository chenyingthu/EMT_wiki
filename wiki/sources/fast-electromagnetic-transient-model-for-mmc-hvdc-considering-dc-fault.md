## 方法细节

### 方法概述

本文基于麦克斯韦方程组的完整场解，推导了适用于多相地下电缆的大地返回阻抗与导纳广义公式。该方法突破了传统EMTP软件中仅采用Pollaczek阻抗公式及忽略大地返回导纳（或依赖准TEM近似）的局限。首先，通过引入模态电流变换矩阵将多导体系统解耦，利用完整场解积分表达式精确描述大地中轴向电场分布。随后，结合电缆表面边界条件与电报方程，构建包含绝缘层参数与大地电磁特性的广义阻抗/导纳矩阵。通过求解特征值问题获取各模态传播常数与变换矩阵，最终实现从低频至高频的统一建模。该方法不仅修正了传统模型在低频段的计算偏差，还通过引入大地返回导纳项显著提升了浪涌仿真中的阻尼特性与稳态收敛平滑度。

### 数学公式


**公式1**: $$$\mathbf{P} \mathbf{B}_v = -k_v^2 \mathbf{B}_v$$$

*模态传播特征值方程，用于求解多相电缆系统的波传播常数$k_v$与模态解耦矩阵$\mathbf{B}_v$*


**公式2**: $$$\mathbf{P} = \mathbf{P}_2^{-1} \mathbf{P}_1$$$

*系统广义传播矩阵构建式，整合绝缘层阻抗/导纳与大地返回参数*


**公式3**: $$$\mathbf{P}_1 = \mathbf{Z}_{ins} + \frac{j\omega\mu_0}{4\pi} [\mathbf{\Delta}_1(v) + 2\mathbf{\Delta}_2(v)]$$$

*广义阻抗矩阵分量，包含绝缘层阻抗与大地自场/互场积分项*


**公式4**: $$$\mathbf{P}_2 = \mathbf{Y}_{ins}^{-1} + \frac{1}{4\pi(\sigma_1 + j\omega\epsilon_1)} [\mathbf{\Delta}_1(v) + 2\mathbf{\Delta}_3(v)]$$$

*广义导纳矩阵分量，显式引入大地电导率与介电常数，突破准TEM假设*


**公式5**: $$$Z_{sn}(v) = \frac{k_v^2 - k_{ins}^2}{j\omega 2\pi\epsilon_{ins}} \ln\left(\frac{r_{on}}{r_{in}}\right)$$$

*第n根电缆在第v模态下的表面阻抗表达式*


### 算法步骤

1. 建立多相电缆几何与材料模型，定义导体半径、绝缘层介电常数/磁导率、大地电导率/介电常数及埋深等基础参数。

2. 基于完整场解推导大地中轴向电场积分表达式，分离自场与互场分量，构建包含$\Delta_1, \Delta_2, \Delta_3$积分核的广义场函数$\Lambda$。

3. 应用电缆表面边界条件，将轴向电场与导体表面阻抗关联，建立多导体系统的耦合电压-电流关系式。

4. 引入模态变换矩阵$\mathbf{B}$，将相域变量转换至模域，推导系统传播矩阵$\mathbf{P} = \mathbf{P}_2^{-1}\mathbf{P}_1$，其中$\mathbf{P}_1$和$\mathbf{P}_2$分别表征广义阻抗与导纳特性。

5. 求解特征值问题$\mathbf{P}\mathbf{B}_v = -k_v^2\mathbf{B}_v$，获取各模态传播常数$k_v$及对应的模态变换矩阵列向量。

6. 基于求得的$k_v$与$\mathbf{B}$，反算频域下的广义阻抗与导纳矩阵，并集成至EMT仿真程序中进行时域浪涌计算与波传播特性评估。


### 关键参数

- **$\sigma_1, \epsilon_1$**: 大地电导率与介电常数

- **$r_{in}, r_{on}$**: 电缆导体内半径与绝缘层外半径

- **$\mu_{ins}, \epsilon_{ins}$**: 绝缘层磁导率与介电常数

- **$k_v$**: 第v模态传播常数（复数，含衰减与相位信息）

- **$\mathbf{Z}_{ins}, \mathbf{Y}_{ins}$**: 绝缘层阻抗与导纳对角矩阵

- **$\Delta_1, \Delta_2, \Delta_3$**: 完整场解积分核函数，表征大地电磁场分布

## 量化发现

- 在数十kHz（tens of kHz）低频段，大地返回导纳对电缆波传播特性的影响显著，传统忽略该参数的模型将引入不可忽略的计算误差。
- 新公式在110 kV交叉互联电缆现场测试中实现高度吻合，验证了其在实际工程尺度下的精度与适用性。
- 频率适用范围覆盖从工频至数MHz（Sommerfeld-Goubau波过渡区以下），突破了准TEM近似的频段限制。
- 浪涌仿真中，新模型相比传统EMTP模型展现出更强的物理阻尼特性，有效抑制了数值振荡，提升了暂态计算的收敛平滑度。


## 关键公式

### 模态传播特征值方程

$$$\mathbf{P} \mathbf{B}_v = -k_v^2 \mathbf{B}_v$$$

*用于求解多相电缆系统的波传播常数与模态解耦矩阵，是连接场解与电路参数的核心桥梁*

### 广义传播矩阵构建式

$$$\mathbf{P} = \mathbf{P}_2^{-1} \mathbf{P}_1$$$

*在完整场解框架下整合绝缘层与大地返回参数，用于替代传统EMTP中的准TEM近似矩阵*

### 大地轴向电场完整场解表达式

$$$E_{1x}(x,y,z) = S_1 \sum_{m=1}^N \sum_{v=1}^N \Lambda(y,z,d_m,h_m,v) B_{mv} i_v(x)$$$

*用于精确计算地下电缆周围大地的电磁场分布，是推导广义阻抗/导纳公式的物理基础*



## 验证详情

- **验证方式**: 现场实测数据对比 + MoM-So高精度数值方法交叉验证 + 传统EMTP模型对比分析
- **测试系统**: 110 kV交叉互联地下电缆系统（含多相导体、金属护套与复杂接地配置）
- **仿真工具**: 自定义数值计算程序（基于完整场解推导）、EMTP/EMT型仿真软件（内置Pollaczek模型作为基线）、MoM-So算法验证平台
- **验证结果**: 新推导公式计算结果与110 kV电缆现场浪涌测试数据高度吻合；在波传播特性评估中，与MoM-So方法结果一致，验证了理论推导的正确性；相比传统EMTP模型，新公式在暂态仿真中提供更强的物理阻尼与更平滑的稳态收敛，显著提升了地下电缆电磁暂态仿真的精度与可靠性。
