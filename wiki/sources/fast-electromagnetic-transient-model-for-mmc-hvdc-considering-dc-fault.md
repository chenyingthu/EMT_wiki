# Fast Electromagnetic Transient Model for MMC-HVDC Considering DC Fault

**作者**: ilhan
**年份**: 2018
**来源**: `19、20、21/EMT_task_20/TPWRD.2018.2796089.pdf.pdf`

## 摘要

本文基于麦克斯韦方程组的完整场解，推导了适用于多相地下电缆的大地返回阻抗与导纳广义公式。该方法突破了传统EMTP软件中仅采用Pollaczek阻抗公式及忽略大地返回导纳（或依赖准TEM近似）的局限。首先，通过引入模态电流变换矩阵将多导体系统解耦，利用完整场解积分表达式精确描述大地中轴向电场分布。随后，结合电缆表面边界条件与电报方程，构建包含绝缘层参数与大地电磁特性的广义阻抗/导纳矩阵。通过求解特征值问题获取各模态传播常数与变换矩阵，最终实现从低频至高频的统一建模。该方法不仅修正了传统模型在低频段的计算偏差，还通过引入大地返回导纳项显著提升了浪涌仿真中的阻尼特性与稳态收敛平滑度。

## 研究解读

### 1. 需求、对象、挑战与贡献

地下电缆浪涌和EMT仿真需要准确的频率相关线路参数，尤其是大地返回阻抗与大地返回导纳；这些参数直接决定波传播常数、特性阻抗、衰减和暂态过电压波形。本文研究对象不是MMC-HVDC故障模型，而是多相地下电缆的大地返回阻抗/导纳统一计算公式及其在浪涌分析中的影响。难点在于地下电缆电磁场与有损大地耦合，电场、磁场、导体表面边界和线路电压定义相互关联；传统EMTP中主要采用Pollaczek大地返回阻抗并只保留绝缘介质导纳，常忽略大地返回导纳，而准TEM推导又依赖特定电压积分参考和频段假设。本文的贡献是基于Maxwell方程完整场解和电报方程，为多相地下电缆推导广义的大地返回阻抗与导纳公式，并讨论不同线路电压定义下相关公式的关系；同时说明准TEM公式可由完整场公式退化得到，从而把既有Pollaczek、Papadopoulos等模型放入同一比较框架，用于评估大地返回导纳对波传播和浪涌仿真的实际影响。

### 2. 模型、算法与实现技术

本文提出的是频域线路参数计算模型，而非时域控制算法。输入包括电缆几何布置、导体/绝缘层参数、埋设位置、多相电缆数量以及大地电导率、介电常数等电磁参数；输出是随频率变化的阻抗矩阵、导纳矩阵，以及由其进一步得到的传播特性。模型机制是先从完整电磁场解出发描述地下电缆和大地中的场分布，再结合电缆表面边界条件，把轴向电场与导体电流、表面电压联系起来；随后利用电报方程把这些场量转化为多导体线路的Z、Y矩阵。文中还讨论了三种线路电压计算方式，因为地下电缆的电压积分路径会影响大地返回阻抗/导纳表达式，尤其是导纳项的定义。计算流程上，可先在每个频点求得完整场公式给出的Z和Y，再由ZY或YZ矩阵求传播常数、模态传播特性和特性阻抗，最后把这些频率响应用于EMTP/EMT型浪涌仿真。关键作用在于：阻抗项描述大地返回电流引起的纵向电压降，导纳项描述电缆与大地之间由有损介质和位移电流共同形成的横向电流通道；两者同时进入电报方程后，才能反映大地参数对衰减和相位的影响。

### 3. 验证、优势与不足

作者采用三类证据验证公式有效性：第一，将新公式计算的电缆阻抗和导纳与已有公式进行比较，包括Pollaczek阻抗公式以及文中提到的Papadopoulos等相关公式；第二，用带导纳表示的MoM-So方法作为数值交叉验证，检查新公式在频率响应上的一致性；第三，在110 kV交叉互联地下电缆现场测试基础上进行浪涌仿真对比，观察新公式与实测浪涌响应之间的吻合情况。测试对象明确为交叉互联电缆，仿真关注波传播特性和浪涌暂态波形；基线主要是EMTP或EMT型工具中采用的既有公式，尤其是Pollaczek大地返回阻抗加绝缘导纳的建模方式。优势体现在两个方面：一是能够显式纳入大地返回导纳，使大地介电常数、电导率对传播特性的影响进入模型；二是浪涌仿真中相对于既有公式表现出更多阻尼和更平滑的稳态收敛。需要注意，原文摘要和引言未报告可核验的误差百分比、计算速度提升或通用精度数字；因此不能声称误差小于某阈值或振荡降低多少。从验证范围看，结论主要支撑地下电缆、所研究频段和110 kV交叉互联电缆场景，不应直接外推到架空线、复杂分层土壤、所有电缆结构、所有故障类型或实时仿真步长。

### 4. 价值、认知与可复用场景

这项工作的核心认知价值在于提醒EMT建模者：地下电缆浪涌分析中，大地返回导纳并非总能忽略，甚至在数十kHz范围内也可能影响传播特性；只用Pollaczek阻抗和绝缘导纳可能低估大地参数对衰减、相位和暂态收敛过程的作用。它适合作为地下电缆宽频参数建模、EMTP线路模型改进、浪涌过电压研究、交叉互联电缆现场测试解释，以及比较准TEM与完整场解方法的入口页面。后续页面若讨论频率相关Z/Y矩阵、模态传播、地下电缆雷击或操作浪涌，可复用其建模框架。但它不适合被外推为MMC-HVDC直流故障快速模型，也不应作为任意电力电子系统EMT加速算法的证据。

### 证据边界

- 原文明确给出的论文题名和内容是地下电缆大地返回阻抗/导纳广义公式；当前元数据中的“Fast Electromagnetic Transient Model for MMC-HVDC Considering DC Fault”与抽取文本不一致，应以PDF正文证据为准。
- 原文摘要和引言明确说明公式基于Maxwell方程完整场解、电报方程，并与现有EMTP公式、MoM-So方法和110 kV交叉互联电缆现场测试比较；这些属于直接证据。
- 原文提到浪涌仿真结果相对既有公式具有更多阻尼并更平滑收敛，但所给文本未提供可核验的误差百分比、幅值降低比例或收敛时间数字，不能编造成量化结论。

## 方法细节

### 方法概述

本文基于麦克斯韦方程组的完整场解，推导了适用于多相地下电缆的大地返回阻抗与导纳广义公式。该方法突破了传统EMTP软件中仅采用Pollaczek阻抗公式及忽略大地返回导纳（或依赖准TEM近似）的局限。首先，通过引入模态电流变换矩阵将多导体系统解耦，利用完整场解积分表达式精确描述大地中轴向电场分布。随后，结合电缆表面边界条件与电报方程，构建包含绝缘层参数与大地电磁特性的广义阻抗/导纳矩阵。通过求解特征值问题获取各模态传播常数与变换矩阵，最终实现从低频至高频的统一建模。该方法不仅修正了传统模型在低频段的计算偏差，还通过引入大地返回导纳项显著提升了浪涌仿真中的阻尼特性与稳态收敛平滑度。

### 数学公式

**公式1**: $$$\mathbf{P} \mathbf{B}_v = -k_v^2 \mathbf{B}_v$$$

*模态传播特征值方程，用于求解多相电缆系统的波传播常数$k_v$与模态解耦矩阵$\mathbf{B}_v$*

**公式2**: $$$\mathbf{P} = \mathbf{P}_2^{-1} \mathbf{P}_1$$$

*系统广义传播矩阵构建式，整合绝缘层阻抗/导纳与大地返回参数*

**公式3**: $$$\mathbf{P}_1 = \mathbf{Z}_{ins} + \frac{j\omega\mu_0}{4\pi} [\mathbf{\Delta}_1(v) + 2\mathbf{\Delta}_2(v)]$$$

*广义阻抗矩阵分量，包含绝缘层阻抗与大地自场/互场积分项*

**公式4**: $$$\mathbf{P}_2 = \mathbf{Y}_{ins}^{-1} + \frac{1}{4\pi(\sigma_1 + j\omega\epsilon_1)} [\mathbf{\Delta}_1(v) + 2\mathbf{\Delta}_3(v)]$$$

*广义导纳矩阵分量，显式引入大地电导率与介电常数，突破准TEM假设*

**公式5**: $$$Z_{sn}(v) = \frac{k_v^2 - k_{ins}^2}{j\omega 2\pi\epsilon_{ins}} \ln\left(\frac{r_{on}}{r_{in}}\right)$$$

*第n根电缆在第v模态下的表面阻抗表达式*

### 算法步骤

1. 建立多相电缆几何与材料模型，定义导体半径、绝缘层介电常数/磁导率、大地电导率/介电常数及埋深等基础参数。

2. 基于完整场解推导大地中轴向电场积分表达式，分离自场与互场分量，构建包含$\Delta_1, \Delta_2, \Delta_3$积分核的广义场函数$\Lambda$。

3. 应用电缆表面边界条件，将轴向电场与导体表面阻抗关联，建立多导体系统的耦合电压-电流关系式。

4. 引入模态变换矩阵$\mathbf{B}$，将相域变量转换至模域，推导系统传播矩阵$\mathbf{P} = \mathbf{P}_2^{-1}\mathbf{P}_1$，其中$\mathbf{P}_1$和$\mathbf{P}_2$分别表征广义阻抗与导纳特性。

5. 求解特征值问题$\mathbf{P}\mathbf{B}_v = -k_v^2\mathbf{B}_v$，获取各模态传播常数$k_v$及对应的模态变换矩阵列向量。

6. 基于求得的$k_v$与$\mathbf{B}$，反算频域下的广义阻抗与导纳矩阵，并集成至EMT仿真程序中进行时域浪涌计算与波传播特性评估。

### 关键参数

- **$\sigma_1, \epsilon_1$**: 大地电导率与介电常数

- **$r_{in}, r_{on}$**: 电缆导体内半径与绝缘层外半径

- **$\mu_{ins}, \epsilon_{ins}$**: 绝缘层磁导率与介电常数

- **$k_v$**: 第v模态传播常数（复数，含衰减与相位信息）

- **$\mathbf{Z}_{ins}, \mathbf{Y}_{ins}$**: 绝缘层阻抗与导纳对角矩阵

- **$\Delta_1, \Delta_2, \Delta_3$**: 完整场解积分核函数，表征大地电磁场分布

## 量化发现

- 在数十kHz（tens of kHz）低频段，大地返回导纳对电缆波传播特性的影响显著，传统忽略该参数的模型将引入不可忽略的计算误差。
- 新公式在110 kV交叉互联电缆现场测试中实现高度吻合，验证了其在实际工程尺度下的精度与适用性。
- 频率适用范围覆盖从工频至数MHz（Sommerfeld-Goubau波过渡区以下），突破了准TEM近似的频段限制。
- 浪涌仿真中，新模型相比传统EMTP模型展现出更强的物理阻尼特性，有效抑制了数值振荡，提升了暂态计算的收敛平滑度。

## 关键公式

### 模态传播特征值方程

$$$\mathbf{P} \mathbf{B}_v = -k_v^2 \mathbf{B}_v$$$

*用于求解多相电缆系统的波传播常数与模态解耦矩阵，是连接场解与电路参数的核心桥梁*

### 广义传播矩阵构建式

$$$\mathbf{P} = \mathbf{P}_2^{-1} \mathbf{P}_1$$$

*在完整场解框架下整合绝缘层与大地返回参数，用于替代传统EMTP中的准TEM近似矩阵*

### 大地轴向电场完整场解表达式

$$$E_{1x}(x,y,z) = S_1 \sum_{m=1}^N \sum_{v=1}^N \Lambda(y,z,d_m,h_m,v) B_{mv} i_v(x)$$$

*用于精确计算地下电缆周围大地的电磁场分布，是推导广义阻抗/导纳公式的物理基础*
