# 直驱式风电机组机电暂态建模及仿真

**作者**: 高峰 等
**年份**: 2022
**来源**: `16/高峰 等 - 2011 - 直驱式风电机组机电暂态建模及仿真.pdf`

## 摘要

基于矢量解耦控制策略，建立了适用于电力系统暂态稳定分析的直驱式风力发电机组机电暂态模型。为模型做了 2 点合理假设：有功功率和无功功率在暂态过程中完全解耦；测量的电气量准确，和实际的电气量完全相等。基于这些条件，从实际的物理装置出发，对直驱式风电机组的电磁暂态模型进行简化。考虑了变频器直流电压的动态过程和低电压穿越的运行特性。仿真结果表明，该模型和电磁暂态模型动态响应曲线一致，结果准确、可靠，可以用于分析大规模直驱式风电场接入对电力系统安全稳定性的影响。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自大规模直驱式永磁同步风电机组并网后的电力系统暂态稳定分析：若直接采用含PMSG、电机侧/网侧PWM变流器、滤波环节和控制器的电磁暂态模型，模型细、步长小、收敛和计算负担较重；而当时电力系统分析软件中常用的“暂态电抗后可控电流源”等外特性模型又难以反映直驱机组内部直流母线、变流器控制和低电压穿越过程。本文研究对象是采用背靠背双PWM全功率变流器、矢量解耦控制的直驱式PMSG风电系统。难点在于：既要保留影响故障暂态的内部能量交换与控制响应，又要把适合EMT的小时间尺度开关/电磁过程化简到机电暂态稳定仿真可用的形式。本文贡献不是简单等值为电流源，而是在两个明确假设下——暂态中有功/无功完全解耦、测量电气量等于实际量——从物理装置和控制结构出发化简详细模型，同时保留PWM变流器直流电压动态和低电压穿越运行特性，用于大规模直驱风电场接入对电网安全稳定影响的分析。

### 2. 模型、算法与实现技术

本文建立的是直驱PMSG风电系统机电暂态模型，结构上仍对应风力机、永磁同步发电机、背靠背双PWM变流器、直流电容、卸荷电路、滤波电感和电网接口。输入/扰动可包括风速、桨距角、电网电压及故障电压跌落；核心状态和接口量包括发电机电气转速、机械/电磁转矩、PMSG d-q轴电流、电网侧d-q轴电流、直流母线电压、注入电网的有功和无功功率等。风力机部分用机械转矩公式和Cp-λ-β关系给出气动输入；PMSG部分由d-q轴电压方程、电磁转矩方程和转子运动方程描述机电能量转换；网侧变流器由滤波电感d-q轴方程及功率方程连接电网；直流侧用 C u_dc du_dc/dt = P_s - P_g 表示发电机侧输入功率与网侧输出功率不平衡导致的电容充放电。模型工作的关键机制是利用矢量解耦控制：在理想测量和有功/无功解耦假设下，发电机及变流器数学模型中的d-q轴耦合项与控制器解耦补偿项相互抵消，从而弱化高频电磁耦合和PWM开关细节，但不删除直流母线能量平衡。低电压穿越部分通过考虑卸荷电路/直流电压限制，使故障期间因网侧送出能力下降引起的直流过压过程仍能在机电暂态仿真中体现。

### 3. 验证、优势与不足

作者采用与电磁暂态模型对比的方式验证机电暂态模型：在相同扰动或故障条件下，比较所建模型与详细EMT模型的动态响应曲线。原文摘要明确指出两类模型动态响应曲线基本一致，因而认为模型合理、准确，可用于分析大规模直驱式PMSG风电场并网对电网安全稳定性的影响。可比变量从模型结构看应包括发电机转速、电磁转矩、直流母线电压、输出有功/无功功率以及低电压穿越相关量；但当前提供文本没有给出完整算例系统拓扑、仿真平台名称、步长、风机容量、控制参数，也没有列出曲线误差或加速比。优势主要体现在建模层面：相比只基于外特性的可控电流源模型，它保留了直驱机组内部变流器直流电压动态和LVRT特征；相比详细EMT模型，它去除了PWM开关及部分d-q耦合细节，更适合电力系统暂态稳定程序中的大规模接入研究。从验证范围看，本文结论依赖矢量解耦控制、理想测量和有功/无功完全解耦假设；对于非理想测量延迟、控制限幅细节、PLL动态、开关谐波、保护动作、非对称故障及高频电磁振荡等，当前证据不足以证明该机电暂态模型仍准确。原文未报告可核验的数值结果，因此不能量化其误差、收敛改进或计算提速。

### 4. 价值、认知与可复用场景

这项工作的价值在于给出一种介于详细EMT模型和过度简化外特性模型之间的直驱风电机组建模思路：把影响系统暂态稳定的慢变量和能量通道保留下来，尤其是PMSG机电转换、背靠背变流器功率传递、直流母线能量平衡和LVRT过程；同时借助矢量解耦控制假设压缩d-q轴电磁细节。它适合被后续大规模风电场并网稳定分析、风电机组机电暂态模型库、直驱机组等值建模、低电压穿越策略影响评估等页面复用。不适合直接外推为开关级EMT、谐波分析、保护暂态、控制器非理想实现或未经验证的新型变流器拓扑模型。

## 方法细节

### 方法概述

基于矢量解耦控制策略，建立适用于电力系统暂态稳定分析的直驱式永磁同步风电机组(PMSG)机电暂态模型。核心思路是通过两个关键假设（有功/无功完全解耦、测量值与实际值完全一致）对详细电磁暂态模型进行简化，抵消发电机及变流器模型中的d-q轴耦合项与控制解耦项，同时保留PWM变流器直流电压动态过程和低电压穿越(LVRT)特性，实现计算效率与精度的平衡。

### 数学公式

**公式1**: $$$T_m = \frac{1}{2}\rho\pi R^3 \frac{C_p(\lambda, \beta)}{\lambda^2} v_w^2$$$

*风力机机械转矩方程，其中$\rho$为空气密度，$R$为风轮半径，$v_w$为风速，$\lambda$为叶尖速比，$C_p$为风能利用系数*

**公式2**: $$$C_p(\lambda, \beta) = 0.22\left(\frac{116}{\lambda_i} - 0.4\beta - 5\right)e^{-12.5/\lambda_i}$$$

*风能利用系数计算式，$\beta$为桨距角，$\lambda_i$为中间变量*

**公式3**: $$$\lambda_i = \frac{1}{\frac{1}{\lambda + 0.08\beta} - \frac{0.035}{\beta^3 + 1}}$$$

*叶尖速比中间计算变量*

**公式4**: $$$u_{ds} = R_s i_{ds} + L_d \frac{di_{ds}}{dt} - L_q \omega_e i_{qs}$$$

*PMSG定子d轴电压方程（电动机惯例），$L_d, L_q$为直交轴电感，$R_s$为定子电阻，$\omega_e$为电气转速*

**公式5**: $$$u_{qs} = R_s i_{qs} + L_q \frac{di_{qs}}{dt} + L_d \omega_e i_{ds} + \omega_e \psi_f$$$

*PMSG定子q轴电压方程，$\psi_f$为永磁磁链*

**公式6**: $$$T_e = 1.5p[(L_d - L_q)i_{ds}i_{qs} + \psi_f i_{qs}]$$$

*电磁转矩方程，$p$为极对数*

**公式7**: $$$P_s = -T_e \omega = -1.5(u_{ds}i_{ds} + u_{qs}i_{qs})$$$

*发电机输出有功功率（采用电动机惯例故为负值或需根据参考方向调整）*

**公式8**: $$$J \frac{d\omega_e}{p dt} = T_m - T_e$$$

*转子机械运动方程，$J$为转动惯量*

**公式9**: $$$u_{dg} = e_{dg} - R i_{dg} - L \frac{di_{dg}}{dt} + \omega_s L i_{qg}$$$

*电网侧变流器d轴电压方程，$e_{dg}$为电网电压d轴分量，$R, L$为滤波电阻和电感，$\omega_s$为同步电角速度*

**公式10**: $$$u_{qg} = e_{qg} - R i_{qg} - L \frac{di_{qg}}{dt} - \omega_s L i_{dg}$$$

*电网侧变流器q轴电压方程*

**公式11**: $$$P_g = -1.5(u_{dg}i_{dg} + u_{qg}i_{qg})$$$

*变流器注入电网的有功功率*

**公式12**: $$$Q_g = -1.5(-u_{dg}i_{qg} + u_{qg}i_{dg})$$$

*变流器注入电网的无功功率*

**公式13**: $$$C u_{dc} \frac{du_{dc}}{dt} = P_s - P_g$$$

*直流侧电容电压动态方程，$C$为直流电容，$u_{dc}$为直流电压，体现输入输出功率不平衡引起的电压变化*

### 算法步骤

1. 建立完整的电磁暂态详细模型：包括风力机空气动力学模型、PMSG d-q轴模型（含交叉耦合项）、背靠背双PWM变流器模型（含高频开关细节）及矢量控制系统

2. 提出并应用关键假设1：假设暂态过程中有功功率和无功功率完全解耦，忽略d-q轴间的动态耦合作用

3. 提出并应用关键假设2：假设测量环节理想化，即测量的电气量（电流、电压、转速）与实际值完全相等且无延迟

4. 模型简化-发电机侧：基于假设，发电机控制策略中的解耦补偿项（$-L_q\omega_e i_{qs}$和$+L_d\omega_e i_{ds}$）与数学模型中的固有耦合项精确抵消，消除d-q轴交叉耦合，实现转矩与无功的独立控制

5. 模型简化-电网侧：同理，电网侧变流器控制中的解耦项（$+\omega_s L i_{qg}$和$-\omega_s L i_{dg}$）与模型耦合项抵消，实现有功（直流电压）和无功功率的独立控制

6. 采用平均值模型（Average-Value Model）代替详细开关模型，忽略PWM高频开关过程，保留基波特性

7. 建立直流电压动态模型：保留式(13)描述的直流电容充放电过程，确保能准确模拟功率不平衡时的暂态过程

8. 集成低电压穿越(LVRT)模型：在直流侧增加卸荷电路（Crowbar）模型，通过滞环控制逻辑（当$u_{dc} > u_{dcmax}$时投入卸荷电阻，当$u_{dc} < u_{dcmin}$时切除）模拟故障期间的功率耗散和直流过压抑制

9. 建立状态空间表达：将简化后的方程组整理为适用于机电暂态仿真的状态空间形式，状态变量通常包括$i_{ds}, i_{qs}, i_{dg}, i_{qg}, \omega_e, u_{dc}$等

10. 模型验证：通过与详细电磁暂态模型(EMT)在相同扰动下的动态响应对比，验证简化模型的准确性和计算效率

### 关键参数

- **发电机侧参数**: $L_d, L_q$（直交轴电感，单位H），$R_s$（定子电阻，单位Ω），$\psi_f$（永磁磁链，单位Wb），$p$（极对数），$J$（转动惯量，单位kg·m²）

- **变流器及电网侧参数**: $R, L$（电网侧滤波电阻Ω和电感H），$C$（直流侧电容，单位F），$\omega_s$（电网同步电角速度，rad/s）

- **控制参数**: PI控制器比例积分系数（原文未给出具体数值），$i_{ds\_ref}=0$（无功电流参考值，单位A），$u_{dc\_ref}$（直流电压参考值，单位V）

- **LVRT参数**: $u_{dcmax}$（直流电压上限阈值，单位V），$u_{dcmin}$（直流电压下限阈值，单位V），$R_{crowbar}$（卸荷电阻，单位Ω）

- **风力机参数**: $\rho$（空气密度，kg/m³），$R$（叶片半径，m），$\beta$（桨距角，deg）

## 量化发现

- 模型简化基于两个核心假设：假设1（有功无功完全解耦）和假设2（测量值100%准确等于实际值），在此条件下数学模型中的耦合项与控制解耦项精确抵消（抵消误差理论上为0）
- 直流电压动态由一阶微分方程$C u_{dc} \frac{du_{dc}}{dt} = P_s - P_g$描述，时间常数取决于电容$C$和等效功率差
- 发电机侧采用$id=0$控制策略时，电磁转矩方程简化为$T_e = 1.5p\psi_f i_{qs}$，实现转矩与无功电流的线性化控制
- 电网侧变流器采用电网电压定向控制（$u_{dg}=0$）时，有功功率$P_g$与$d$轴电流$i_{dg}$成正比，无功功率$Q_g$与$q$轴电流$i_{qg}$成正比，实现有功无功的解耦控制
- 原文未提供具体的数值误差百分比（如误差<0.5%）或计算加速比（如提速10倍）的量化数据，仅定性描述为'基本一致'、'计算效率高'、'数值收敛性好'

## 关键公式

### 直流侧电压动态方程（能量守恒形式）

$$$C u_{dc} \frac{du_{dc}}{dt} = P_s - P_g$$$

*描述背靠背变流器直流电容的充放电过程，是机电暂态模型与简化外特性电流源模型的关键区别，用于模拟功率不平衡导致的直流电压波动，是分析低电压穿越特性的核心方程*

### PMSG d-q轴电压方程

$$$u_{ds} = R_s i_{ds} + L_d \frac{di_{ds}}{dt} - L_q \omega_e i_{qs}$ 和 $u_{qs} = R_s i_{qs} + L_q \frac{di_{qs}}{dt} + L_d \omega_e i_{ds} + \omega_e \psi_f$$$

*在假设测量准确的前提下，控制器的解耦项$-L_q\omega_e i_{qs}$和$+L_d\omega_e i_{ds}$分别与方程中的耦合项$+L_q\omega_e i_{qs}$和$-L_d\omega_e i_{ds}$（移项后）抵消，实现模型简化*

### 简化电磁转矩方程

$$$T_e = 1.5p\psi_f i_{qs}$（当$i_{ds}=0$时）$$

*在表贴式或凸极率很低的永磁发电机中采用$id=0$控制策略时的简化形式，用于最大功率追踪(MPPT)控制，以最小电流实现最大转矩输出*
