# A multi-functional series compensator to squirrel cage induction generator

**年份**: 2015
**来源**: `02/Jiang 等 - 2015 - A multi-functional series compensator to squirrel cage induction generator.pdf`

## 摘要

为改善鼠笼型异步风电机（squirrel cage induction generator, SCIG）的低电压穿越能力，提出了一种适用于SCIG的多功能串联补偿器（multi-functional series compensator, MFSC）。MFSC可连接于风机定子与并网点之间，能够实现风电场低电压穿越、故障后风机并网点电压快速恢复、系统短路故障限流等多种功能。通过分析电网故障下的SCIG风机工作特性，给出了MFSC功能实现的数学模型；重点介绍了MFSC的不同功能控制策略、多模式切换流程及新增晶闸管控制短路支路优化控制策略。采用PSCAD/EMTDC软件建立了仿真系统，仿真结果验证了MFSC多种功能应用的有效性。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自早期并网鼠笼型异步风电机组的低电压穿越改造。SCIG定子直接接入电网，故障时并网点电压跌落会使电磁转矩按电压平方下降，机械转矩与电磁转矩失衡导致转子加速；故障清除后又可能大量吸收无功，拖慢机端电压恢复并诱发脱网。研究对象是串联在SCIG定子与并网点之间的多功能串联补偿器MFSC。难点在于单一方案各有短板：SVC/STATCOM受低电压和短路容量影响，无功支撑有限；DVR能维持机端电压但正常损耗和故障大电流威胁功率器件；串联制动电阻或电抗器投切存在动作时序、冲击和发热问题。本文贡献不是单纯增加补偿容量，而是把DVR电压补偿、串联限流阻抗和低损耗旁路集成到一个装置，通过晶闸管旁路开关K、串联变压器、VSC/LC滤波器和晶闸管短路支路S的多模式切换，使正常、小扰动和大故障分别采用不同工作机理。其创新点集中在MFSC拓扑、不同模式下的数学模型、模式切换逻辑，以及用于避免切换冲击的晶闸管控制策略。

### 2. 模型、算法与实现技术

论文先用SCIG稳态等值关系解释故障机理：有功、无功和电磁转矩由机端电压U、定转子电阻电抗及转差率s决定，电压跌落使电磁转矩下降；转子运动方程J dωr/dt=Tm-Te说明若机械输入不能快速降低，转速会继续上升。因此MFSC的控制目标是同时抬升发电机端电压、限制故障电流并在故障后恢复PCC电压。装置接口量包括线路电流IL、PCC电压、发电机端电压、补偿电压、晶闸管支路电流和IGBT触发状态。正常时K导通、S关断、IGBT闭锁，装置旁路以降低损耗；小扰动时K和S断开，VSC经串联变压器注入补偿电压，按全补偿策略计算所需电压幅值和相位，使负载侧或机端电压保持目标值；大故障时检测到电流越限并经延时确认后，闭锁故障相IGBT、触发S导通，使LC滤波器和S支路阻抗经变压器折算为串联限流阻抗。文中给出Zlim与变压器变比、滤波电感、晶闸管支路阻抗及触发延迟角α的关系，用于解释可通过α调节等效限流阻抗。切换实现上采用基于电流阈值、延时和RS触发器的逻辑；故障清除后等待S支路电流过零再恢复DVR模式；从旁路到限流切换时利用强制换流关断K，以避免逆变器输出端被旁路晶闸管短接。

### 3. 验证、优势与不足

作者采用PSCAD/EMTDC建立含SCIG风电场的并网仿真系统，风电场经升压变压器接入电网，并在线路上设置电压暂降、对称三相短路和不对称两相接地短路等工况。对比基线主要是未投入MFSC的SCIG系统，观察量包括机端/PCC电压、有功和无功功率、电磁转矩与机械转矩差值、转速趋势、线路故障电流以及模式切换过程。页面抽取结果显示，DVR模式最大补偿电压设为额定电压的15%；在限流测试中，投入MFSC后故障冲击电流峰值由约1.02降至约0.31，降幅约69.6%；在15%电压暂降下，未补偿系统转矩波动持续到约0.61 s，而MFSC使转矩较快恢复平衡；不对称故障下，未补偿系统有功输出可降至0，补偿后可维持输出。优势体现为同一串联设备覆盖低损耗待机、小扰动电压恢复和大故障限流三类功能，并通过分相控制和晶闸管过零/强制换流降低模式切换风险。边界也很明确：验证是数字仿真而非硬件实验或实时仿真；原文页面未展示与STATCOM、SVC、DVR、SBR等方案在相同容量和成本下的系统对比；结论依赖所设电网强度、故障持续时间、阈值、变压器变比、滤波器和晶闸管参数，不能直接外推到所有风场和保护配置。

### 4. 价值、认知与可复用场景

这项工作提供的核心认知是：SCIG的LVRT问题不能只看无功支撑，还要同时处理故障电流、机端电压、转矩失衡和故障后电压恢复；串联装置若具备可切换阻抗和可控电压源两种形态，就能在不同故障严重度下采用不同能量与保护机制。它适合被后续研究复用于SCIG风电场LVRT改造方案设计、串联型故障限流器/DVR混合拓扑建模、EMT仿真中的模式切换逻辑、晶闸管过零关断和强制换流控制页面。不适合直接外推为通用风机控制结论，例如不能据此断言其对双馈或全功率变流风机同样有效，也不能在缺少热设计、损耗、保护配合和硬件开关应力验证时用于工程定型。

## 方法细节

### 方法概述

提出一种串联于鼠笼异步发电机（SCIG）定子与并网点之间的多功能串联补偿器（MFSC）拓扑。该装置由反并联晶闸管旁路开关（K）、串联变压器、电压源型变流器、LC输出滤波器及反并联晶闸管短路支路（S）构成。通过多模式灵活切换实现三大核心功能：正常运行时K导通旁路以降低器件损耗；电网小扰动时K与S断开，变流器工作于动态电压恢复器（DVR）模式进行全电压补偿；电网大故障时立即闭锁IGBT并触发S导通，利用LC滤波器与S支路阻抗经变压器耦合形成串联限流阻抗，有效抬升机端电压并限制短路电流。控制策略核心采用基于电流阈值与RS触发器的多模式切换逻辑，结合强制换流法快速关断旁路晶闸管K，以及过零检测法确保短路支路S安全关断，彻底避免模式切换时逆变器输出端短路冲击，保障SCIG机组低电压穿越（LVRT）与故障后电压快速恢复。

### 数学公式

**公式1**: $$$P = \frac{U^2 (R_r/s)}{(R_s + R_r/s)^2 + (X_s + X_r)^2}$$$

*SCIG输出有功功率表达式，用于分析故障期间有功输出下降与转子加速机理*

**公式2**: $$$T_e = \frac{p U^2 (R_r/s)}{\omega_s \left[ (R_s + R_r/s)^2 + (X_s + X_r)^2 \right]}$$$

*电磁转矩方程，揭示电压跌落导致电磁转矩减小、机械转矩不平衡引发转速飞升的过程*

**公式3**: $$$Z_{lim} = k^2(\omega L_f + Z_{SCR})$$$

*晶闸管完全导通时串联补偿阻抗最大值，k为变压器变比，用于故障限流与电压支撑*

**公式4**: $$$Z_{lim} = k^2\left(\omega L_f + Z_{SCR} \frac{\pi}{\pi - 2\alpha + \sin 2\alpha}\right)$$$

*通过调节晶闸管触发延迟角α改变等效串联阻抗大小的控制方程*

**公式5**: $$$U_{gen} = U_{PCC} + U_{lim}$$$

*故障期间机端电压抬升原理，U_lim为串联阻抗压降，用于维持发电机端口电压*

**公式6**: $$$U_0 = \sqrt{U_{PCC(2)}^2 + U_{PCC(1)}^2 - 2U_{PCC(2)}U_{PCC(1)}\cos(\varphi - \psi)}$$$

*DVR模式下全补偿策略所需的补偿电压幅值计算式*

### 算法步骤

1. 1. 待机模式监测：系统正常运行时，线路电流处于正常范围，RS触发器输出Q保持低电平。旁路晶闸管K导通，短路支路S关断，变流器IGBT脉冲封锁，MFSC处于低损耗待机状态。

2. 2. 小扰动DVR切换：检测到并网点电压发生暂降（小扰动），控制系统立即断开K与S。激活IGBT驱动信号，采用全补偿控制策略实时计算补偿电压相量，维持PCC电压恒定，补偿电压幅值上限设定为15%。

3. 3. 大故障限流切换：实时监测线路电流$I_L$，当$|I_L| > I_{set}$且经$\Delta t_1$延时仍超限时，置位S1为高电平。将$I_L$与2倍基波正余弦相乘经低通滤波提取直流分量，经$\Delta t_2$延时后与S1相与生成RS触发器S端高电平。触发器Q翻转为高电平，立即封锁故障相IGBT脉冲，并触发短路支路S导通，LC滤波器与S支路阻抗串入线路实现限流与电压支撑。

4. 4. 故障清除与恢复：故障消失后$|I_L| < I_{ret}$且经$\Delta t_2$延时确认，RS触发器R端置高，Q翻转为低电平。系统持续监测S支路电流$i_{SCR}$，等待其过零点确认晶闸管完全关断后，重新触发IGBT，切换至DVR模式加速并网点电压恢复。

5. 5. 强制换流控制：在待机模式向限流模式切换瞬间，控制逆变器输出与K支路电流方向相反的电压，强制旁路晶闸管K迅速关断，克服半控器件自然关断延迟，确保模式切换的可靠性。

### 关键参数

- **I_set**: 故障电流启动设定阈值

- **I_ret**: 故障电流返回设定阈值

- **Δt1**: 故障电流超限延时判定时间

- **Δt2**: 模式切换与状态确认延时时间

- **U_max**: DVR模式最大补偿电压幅度（15%额定电压）

- **k**: 串联变压器变比

- **α**: 晶闸管触发延迟角（0~π）

## 量化发现

- 故障冲击电流峰值由1.02降低至0.31，降幅约69.6%
- 15%电压暂降下，无MFSC系统转矩波动恢复时间长达0.11s（至0.61s），有MFSC系统实现快速平衡
- 不对称故障下，未补偿系统有功输出最低降至0，补偿后维持稳定输出
- DVR模式最大补偿电压幅度设定为额定电压的15%
- 晶闸管过零关断与强制换流策略实现模式切换零短路冲击，保护逆变器模块

## 关键公式

### SCIG电磁转矩方程

$$$T_e = \frac{p U^2 (R_r/s)}{\omega_s \left[ (R_s + R_r/s)^2 + (X_s + X_r)^2 \right]}$$$

*用于分析电网故障导致电压跌落时，电磁转矩减小引发转子加速与机械冲击的机理*

### 晶闸管可控串联阻抗方程

$$$Z_{lim} = k^2\left(\omega L_f + Z_{SCR} \frac{\pi}{\pi - 2\alpha + \sin 2\alpha}\right)$$$

*通过调节触发角α动态改变等效补偿阻抗，实现故障期间的灵活限流与电压支撑*

### 机端电压抬升方程

$$$U_{gen} = U_{PCC} + U_{lim}$$$

*描述串联阻抗压降如何补偿并网点电压跌落，维持发电机端口电压稳定*

## 验证详情

- **验证方式**: 数字仿真验证
- **测试系统**: 含SCIG风电场的并网测试系统，经升压变压器接入电网，在输电线路L2设置对称三相短路、不对称两相接地短路及电压暂降故障点
- **仿真工具**: PSCAD/EMTDC
- **验证结果**: 仿真全面验证了MFSC在正常运行、小扰动及大故障工况下的多功能切换能力。装置成功实现故障电流限制（峰值降至0.31）、机端电压支撑、DVR快速电压恢复及多模式平滑切换，有效抑制SCIG转子加速与转矩冲击，显著提升低电压穿越能力与暂态稳定性。
