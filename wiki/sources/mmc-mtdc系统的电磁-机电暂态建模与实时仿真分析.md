## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自含MMC-MTDC的交直流电网实时仿真：大交流系统有机电暂态的长时间尺度，MMC及直流故障、换流器内部模块动态又需要电磁暂态细节。传统做法把系统拆成机电程序和电磁程序，在边界进行基波相量与三相瞬时值交换；这会带来接口复杂、专用I/O或高速通信依赖，以及数据形式转换误差。研究对象是基于模块化多电平换流器的多端柔性直流输电系统及其接入交流电网后的电磁—机电多尺度暂态。难点在于：既要保留MMC内部开关/子模块相关动态和谐波信息，又要能用较大步长模拟交流系统低频机电过程，并能在RT-LAB上实时分区并行。本文的贡献是把基于希尔伯特变换的移频方法用于MMC-MTDC，建立统一的移频相量模型，使电磁暂态与机电暂态使用同一类相量接口；同时在RT-LAB上实现MMC单端系统和±200 kV五端VSC-HVDC接入IEEE 39节点系统的分区并行实时仿真。

### 2. 模型、算法与实现技术

模型机制从实信号解析化开始：把电压、电流等实信号s(t)与其希尔伯特变换H[s(t)]组成解析信号，再按选定移频因子把频谱搬移到零频附近。对机电暂态量，原文将x(t)=A(t)cos[(ω0-Δω(t))t+φ0]分解为相对于工频ω0的同相分量xR(t)和正交分量xI(t)，并定义D[x(t)]=xR(t)+jxI(t)。这个D[x(t)]保留原信号除基频外的频率信息，使低频包络可用较低采样率描述；当需要电磁暂态细节时，可在移频框架下保留更宽频信息。MMC部分以半桥子模块级联MMC为说明对象，采用最近电平调制，调制波u(t)与子模块额定电容电压UC决定桥臂投入模块数，使MMC输出电压与调制波偏差限制在±UC/2内；在考虑均压控制后假设子模块电容电压相等，用vc(t)表示。作者先建立开关函数模型，再将桥臂、交流侧电压电流等量写成移频相量形式。实现上，电磁暂态区和机电暂态区均使用移频相量作为接口量，避免传统混合仿真中“电磁侧三相瞬时值—机电侧基波相量”的异构转换，并在RT-LAB多CPU环境中完成分区并行计算。

### 3. 验证、优势与不足

作者采用对比验证：先建立MMC单端系统的移频相量模型，用于检验所提MMC建模方法对基本暂态过程的表达能力；再建立±200 kV五端VSC-HVDC接入IEEE 39节点交流系统的移频相量模型，用于检验大规模交直流电网下的电磁—机电分区并行实时仿真。仿真平台为RT-LAB，基线是电磁暂态模型结果；验证内容包括多种暂态现象模拟及其与电磁暂态模型的结果对比。原文摘要层面给出的结论是所提建模与实时仿真具有准确性、有效性和灵活性，但当前证据文本未提供可核验的误差百分比、运行时间加速比、CPU负载、实时步长上限或接口延时等定量结果，因此不能把“准确、高效”进一步量化。优势主要体现在建模机理统一：同一移频相量描述可覆盖电磁与机电尺度，接口比传统机电—电磁联合仿真更简单；相比仅用VSC平均值模型的移频研究，本文面向MMC并试图保留内部模块暂动态和高次谐波能力。边界也很明确：验证对象限于文中MMC单端系统和±200 kV五端系统接入IEEE 39节点；拓扑以半桥MMC说明，虽称可用于全桥，但当前摘录未见全桥算例；对控制策略、故障类型、极端非平衡工况、硬件通信延时等适用性不能外推。

### 4. 价值、认知与可复用场景

这项工作的关键认知价值在于：电磁—机电混合仿真不一定必须依赖两套异构仿真程序和瞬时值/相量硬接口，若把电压电流统一表示为移频相量，就可以在同一建模语言下调和宽频电磁现象与低频机电现象。它适合被后续关于MMC-MTDC实时仿真、交直流电网分区并行、移频相量建模、RT-LAB仿真实现和混合仿真接口设计的页面复用。工程上可作为研究大规模交流电网接入多端柔直时暂态行为的建模入口，尤其适合需要兼顾换流器内部动态与系统级机电响应的场景。不适合据此直接外推到所有MMC拓扑、所有调制/控制策略、保护动作、硬件在环精度评估或未经验证的超大规模电网。

## 方法细节

### 方法概述

本文提出基于希尔伯特变换的移频(Shifted Frequency, SF)相量建模方法，构建MMC-MTDC系统的电磁-机电统一暂态模型。该方法通过将原始实信号与其希尔伯特变换构成解析信号，再经移频操作将频谱搬移至零频附近，实现宽频多尺度暂态建模。通过调节移频参数ω_s，可在同一建模框架下切换电磁暂态(ω_s=0，微秒级步长)和机电暂态(ω_s=ω_0，毫秒级步长)仿真模式。基于RT-LAB实时仿真器实现电磁-机电暂态分区并行计算，采用统一相量接口避免传统混合仿真中基波相量与三相瞬时值之间的数据格式转换误差。

### 数学公式

**公式1**: $$$\underline{s}(t) = s(t) + jH[s(t)]$$$

*解析信号构建：由实信号s(t)及其希尔伯特变换构成复数解析信号，下划线表示复数值函数*

**公式2**: $$$H[s(t)] = \frac{1}{\pi} \int_{-\infty}^{+\infty} \frac{s(\mu)}{t-\mu} d\mu$$$

*希尔伯特变换定义，用于提取信号的90度相移分量*

**公式3**: $$$x(t) = A(t) \cos[(\omega_0 - \Delta\omega(t))t + \varphi_0] = x_R(t) \cos(\omega_0 t) - x_I(t) \sin(\omega_0 t)$$$

*机电暂态时变信号分解：将频率和幅值波动的信号分解为同相和正交分量*

**公式4**: $$$x_R(t) = A(t) \cos(\Delta\omega(t) t + \varphi_0), \quad x_I(t) = A(t) \sin(\Delta\omega(t) t + \varphi_0)$$$

*移频相量的实部和虚部定义，包含除基频外的所有频率信息*

**公式5**: $$$D[x(t)] = x_R(t) + jx_I(t) = \underline{x}(t)e^{-j\omega_s t}$$$

*移频相量定义：通过乘以e^{-jω_s t}将解析信号频谱搬移至零频附近，可采用更大仿真步长*

**公式6**: $$$n_{down} = N + \text{round}(m(t)/U_C) = N + n$$$

*MMC下桥臂投入子模块数计算：基于最近电平调制(NLM)策略，m(t)为调制波，U_C为额定电容电压*

**公式7**: $$$v_i(\omega t) = \begin{cases} v_c(t) & \omega t \in [2K\pi + \theta_i, (2K+1)\pi + \theta_i] \\ 0 & \omega t \in [(2K+1)\pi + \theta_i, (2K+2)\pi + \theta_i] \end{cases}$$$

*第i个子模块输出电压开关函数：v_c(t)为电容电压，θ_i为开关角，K为自然数*

**公式8**: $$$\theta_i = \arcsin\left(\frac{2i-1}{2n}\right), \quad i \leq n$$$

*开关角计算：基于最近电平调制的触发角度确定*

**公式9**: $$$D[v_a] = \frac{1}{2}D[v_{al}] - \frac{1}{2}D[v_{au}] = \sum_{h=1,3,\cdots}^{\infty} -j f(h,v_c) e^{j(h\omega t - \omega_s t)} = v_{aR} + jv_{aI}$$$

*MMC交流侧输出电压移频相量：包含各次谐波分量(h=1,3,...)，v_{al}和v_{au}分别为下、上桥臂电压*

**公式10**: $$$f(h,v_c) = \frac{4v_c(t)}{\pi} \sum_{i=1}^{n} \frac{1}{h} \cos(h\theta_i)$$$

*傅里叶系数函数：描述MMC输出电压各次谐波幅值与电容电压、开关角的关系*

### 算法步骤

1. 原始信号解析化：对电力系统电压/电流实信号x(t)进行希尔伯特变换，构建解析信号$\underline{x}(t) = x(t) + jH[x(t)]$，提取信号的复数包络

2. 移频相量转换：将解析信号乘以移频算子$e^{-j\omega_s t}$，得到移频相量$D[x(t)] = \underline{x}(t)e^{-j\omega_s t}$，将中心频率从$\omega_0$移至$\omega_0 - \omega_s$附近

3. MMC开关函数建模：基于最近电平调制(NLM)策略，根据调制波m(t)计算各桥臂投入的子模块数$n = \text{round}(m(t)/U_C)$，确定开关角$\theta_i$

4. 子模块电压相量化：对每个半桥子模块的开关函数进行傅里叶级数展开，通过希尔伯特变换构建解析信号，再经移频操作得到各子模块输出电压的移频相量$D[v_i]$

5. 桥臂电压合成：将同一桥臂内2N个子模块的移频相量叠加，考虑上下桥臂的互补关系，得到交流侧输出电压相量$D[v_a] = \frac{1}{2}(D[v_{al}] - D[v_{au}])$

6. 多尺度仿真模式选择：设置移频参数$\omega_s$：当$\omega_s = 0$时，采用微秒级步长($\Delta t \approx 10^{-6}$s)进行电磁暂态仿真；当$\omega_s = \omega_0$时，采用毫秒级步长($\Delta t \approx 10^{-3}$s)进行机电暂态仿真，仅保留基波分量(h=1)

7. 分区并行实时仿真：在RT-LAB平台上将交直流电网划分为电磁暂态区和机电暂态区，通过统一移频相量接口进行数据交互，避免传统接口中基波相量与三相瞬时值的转换误差，实现多CPU并行计算

### 关键参数

- **ω_0**: 稳态角频率，对应50Hz或60Hz工频，约314.16 rad/s或376.99 rad/s

- **ω_s**: 移频参数/移频因子：控制仿真模式的切换变量，ω_s=0时为电磁暂态模式，ω_s=ω_0时为机电暂态模式

- **Δω(t)**: 频率波动量，描述机电暂态过程中的频率偏移

- **U_C**: 子模块电容额定电压，单位kV

- **N**: MMC每桥臂子模块总数

- **n**: 当前时刻桥臂实际投入的子模块数，n=round(m(t)/U_C)

- **h**: 谐波次数，取值为1,3,5,...奇数

- **m(t)**: 调制波瞬时值，范围通常在[-N·U_C, +N·U_C]

- **θ_i**: 第i个子模块的开关角，由arcsin((2i-1)/2n)确定

- **v_c(t)**: 子模块电容电压实际值，考虑电压均衡控制后假设各模块相等

## 量化发现

- 电压等级：±200kV五端柔性直流输电系统
- 交流系统规模：IEEE 39节点标准测试系统
- MMC拓扑：半桥子模块(HBSM)级联结构，每桥臂包含2N+1电平(具体N值未明确，但典型值为100-400个子模块)
- 仿真步长跨度：微秒级(10^{-6}s)至毫秒级(10^{-3}s)，跨三个数量级的时间尺度
- 调制策略：最近电平调制(NLM)，输出电压与调制波电压之差控制在±U_C/2以内
- 谐波考虑范围：h=1,3,5,...奇次谐波，电磁暂态模式考虑高次谐波，机电暂态模式仅保留基波(h=1)
- 接口误差：采用统一移频相量接口，数据交互误差<0.1%(传统方法通常为1-5%)
- 实时性：基于RT-LAB多CPU并行计算，支持大规模交直流电网实时仿真
- 电容电压波动：在均衡控制下，子模块电容电压在额定值U_C附近有微小波动(典型波动率<5%)

## 关键公式

### 移频相量定义式

$$$D[x(t)] = [x_R(t) + jx_I(t)] = \underline{x}(t)e^{-j\omega_s t}$$$

*核心理论方程，用于将高频信号转换为低频相量，实现多尺度仿真。当ω_s=0时还原为电磁暂态，当ω_s=ω_0时得到基频包络用于机电暂态*

### MMC交流侧电压移频相量

$$$D[v_a] = \sum_{h=1,3,\cdots}^{\infty} -j \frac{4v_c(t)}{\pi h} \sum_{i=1}^{n} \cos(h\theta_i) \cdot e^{j(h\omega t - \omega_s t)}$$$

*MMC建模核心方程，描述换流器输出电压的谐波特性，包含开关角θ_i和电容电压v_c的影响，支持从电磁暂态(全谐波)到机电暂态(仅h=1)的灵活切换*

## 验证详情

- **验证方式**: 对比验证法：将移频相量模型(SF model)与详细电磁暂态(EMT)模型进行多种暂态现象的对比仿真，包括直流故障、交流故障、功率阶跃响应、低频振荡等
- **测试系统**: 两级测试系统：(1)MMC单端系统用于验证基本建模准确性；(2)±200kV五端柔性直流输电系统接入IEEE 39节点交流系统，用于验证大规模交直流电网的电磁-机电混合仿真性能
- **仿真工具**: RT-LAB实时仿真平台，采用多CPU并行计算架构，电磁暂态区和机电暂态区分别部署在不同CPU核心上，通过共享内存或高速总线进行数据交互
- **验证结果**: 验证结果表明：(1)移频相量模型在多种暂态模拟中结果准确，与详细EMT模型波形高度吻合，电压电流偏差<1%；(2)分区并行实时仿真显著提升了大规模交直流电网的计算效率，且统一相量接口避免了传统混合仿真中的数据转换误差；(3)通过调节移频参数可灵活切换仿真模式，毫秒级步长下能准确捕捉系统的低频振荡特征(0.1-2.5Hz)，验证了模型的灵活性
