# A State Variables Elimination-Based EMTP-Type Constant Admittance Equivalent Modeling Method for Power Electronic Converters

**作者**: Xu 等
**年份**: 2025
**来源**: `03/Xu 等 - 2025 - A State Variables Elimination-Based EMTP-Type Constant Admittance Equivalent Modeling Method for Pow.pdf`

## 摘要

本文提出一种基于状态变量消去的EMTP型恒定导纳等效建模方法。该方法采用“网络-节点电压-历史电流源”三层架构，通过矩阵分裂技术将系统状态矩阵分解为恒定部分（）与开关时变部分（）。结合梯形法与前向欧拉法进行混合离散化，引入端口输出方程消去内部状态变量，直接推导出低阶等效节点电压方程。其核心在于等效导纳矩阵严格恒定，历史电流源可通过端口电压反演状态变量直接更新，彻底摒弃传统节点消去法（NEM）中变导纳矩阵求逆与内部支路反求的繁琐流程，显著降低计算复杂度，并具备对模块化/单元化电力电子设备的强通用性。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自高比例电力电子并网后的EMT仿真压力：若用详细开关模型描述大量VSC、光伏单元或PET，节点规模和开关事件会使离线仿真耗时很长；原文引言举例称PSCAD/EMTDC以10 μs步长仿真100 MW光伏电站1 s可能超过3 h。研究对象不是控制器设计本身，而是电力电子变换器面向EMTP网络求解器的等效接口模型。难点在于既要保留内部电感、电容、开关状态等动态特征，又要避免传统节点消去法在开关变化时反复形成/求逆变导纳矩阵，以及求解外部节点后再反求内部支路量的串行负担。本文贡献是把变换器写成“网络—节点电压—历史电流源”三层结构，通过状态变量消去直接生成低阶端口节点电压方程，使外部网络看到的是恒定导纳加历史电流源，同时内部状态可由端口电压反演更新。

### 2. 模型、算法与实现技术

方法从以内部状态x为核心的状态空间方程出发：Mẋ=(Aα+Aβ)x+Bun，其中un是外部端口节点电压，端口注入电流由输出方程is=Px给出。Aα收集拓扑和元件中的恒定部分，Aβ承载开关通断引起的时变部分。离散化时，论文采用矩阵分裂并结合梯形/前向欧拉思想，把会影响网络导纳的部分限制在Aα相关项中；随后把t+Δt时刻内部状态从离散状态方程和端口输出方程中消去，得到is(t+Δt)=Yn un(t+Δt)+ih(t)。这里Yn是等效端口导纳，按页面给出的形式与M、Aα、B、P和步长有关，而不直接随开关状态切换；ih则汇集上一时刻状态、端口电压和开关时变项的历史贡献。外部EMTP求解器只需把该变换器当作恒定导纳并联历史电流源接入全网节点方程。网络电压求得后，再用x(t+Δt)=K un(t+Δt)+φ(t)反演内部状态，为下一步更新历史电流源和开关逻辑服务。因此，该算法的关键不是简单降阶，而是把“内部状态更新”和“外部节点求解”解耦为固定导纳接口加状态反演流程。

### 3. 验证、优势与不足

原文摘要说明作者用离线EMT仿真和实验对比验证准确性，并声称在多种场景下具有较高准确性和效率；当前页面还列出两级VSC、光伏发电机组PVPGU和模块化PET等测试对象，以及PSCAD/EMTDC作为离线详细仿真基线、MATLAB/自定义程序作为等效算法实现。但在用户提供的原文片段中，尚未展示具体波形、误差指标、实验平台参数或运行时间表，因此除引言中100 MW光伏电站仿真耗时例子外，不能把操作数195/488、稳定步长112 μs等数值当作已由片段核验的最终结论；这些需要回到论文表图复核。可确认的优势机制是：恒定导纳接口减少EMTP全网矩阵因开关状态变化而重构的负担；状态变量消去避免传统NEM先消去内部节点、再由外部解反求内部量的复杂流程；保留状态方程使其比纯平均模型更有机会反映内部动态。从验证范围看，结论主要受所选拓扑、控制策略、开关模型、步长和故障/闭锁场景约束；尚不能外推到所有变换器、所有实时仿真平台或任意大步长稳定性。

### 4. 价值、认知与可复用场景

这项工作的认知价值在于把电力电子装置的EMT等效问题从“怎样把复杂内部网络做节点消去”转化为“怎样构造对外恒定导纳、对内可反演状态的端口模型”。它适合用于大规模光伏场站、模块化电力电子设备、含大量相似变换器的EMTP仿真页面，尤其可作为恒定导纳接口、历史电流源更新、状态变量消去和NEM替代方法的知识入口。后续若研究快速并网仿真、设备聚合等值或仿真器接口封装，可复用其三层架构和端口方程思想。但不应把它直接等同于控制等值、频域小信号模型或通用稳定性判据，也不应在缺少原文表图支持时声称适用于未验证拓扑、极端故障或任意仿真步长。

## 方法细节

### 方法概述

本文提出一种基于状态变量消去的EMTP型恒定导纳等效建模方法。该方法采用“网络-节点电压-历史电流源”三层架构，通过矩阵分裂技术将系统状态矩阵分解为恒定部分（$A_\alpha$）与开关时变部分（$A_\beta$）。结合梯形法与前向欧拉法进行混合离散化，引入端口输出方程消去内部状态变量，直接推导出低阶等效节点电压方程。其核心在于等效导纳矩阵严格恒定，历史电流源可通过端口电压反演状态变量直接更新，彻底摒弃传统节点消去法（NEM）中变导纳矩阵求逆与内部支路反求的繁琐流程，显著降低计算复杂度，并具备对模块化/单元化电力电子设备的强通用性。

### 数学公式

**公式1**: $$$M \dot{x}(t) = (A_\alpha + A_\beta)x(t) + B u_n(t)$$$

*经矩阵分裂后的系统状态空间方程，$M$为电感电容系数矩阵，$A_\alpha$为恒定矩阵，$A_\beta$表征开关通断时变特性，$u_n$为外部端口节点电压输入。*

**公式2**: $$$i_s(t+\Delta t) = Y_n u_n(t+\Delta t) + i_h(t)$$$

*推导出的低阶端口等效节点电压方程，$Y_n$为恒定导纳矩阵，$i_h(t)$为历史电流源，实现变换器与外部电网的直接接口。*

**公式3**: $$$x(t+\Delta t) = K u_n(t+\Delta t) + \phi(t)$$$

*状态变量直接反演公式，利用已求解的外部节点电压更新内部状态变量，避免传统方法中内部支路电压电流的反求。*

**公式4**: $$$Y_n = \frac{\Delta t}{2} P Q B, \quad Q = \left(M - \frac{\Delta t}{2} A_\alpha\right)^{-1}$$$

*恒定导纳矩阵解析式，证明$Y_n$仅与拓扑参数、积分步长和恒定矩阵相关，与开关状态无关，确保导纳恒定。*

### 算法步骤

1. 步骤1：建立电力电子变换器的原始状态空间方程，将外部端口节点电压作为输入，端口注入电流作为输出。利用矩阵分裂技术将系统矩阵分解为恒定部分$A_\alpha$和开关时变部分$A_\beta$，形成$M \dot{x}(t) = (A_\alpha + A_\beta)x(t) + B u_n(t)$。

2. 步骤2：采用梯形法与前向欧拉法相结合的混合积分策略对状态方程进行离散化，保留端口电气变量，消去$t+\Delta t$时刻的内部状态变量矩阵$x(t+\Delta t)$。

3. 步骤3：结合离散化结果与端口输出方程$i_s = P x$，推导出低阶等效节点电压方程$i_s = Y_n u_n + i_h$。计算并预存恒定导纳矩阵$Y_n$及中间矩阵$R$，历史电流源$i_h(t)$由上一时刻状态$x(t)$和端口电压$u_n(t)$线性组合构成。

4. 步骤4：将等效电路接入外部网络，利用EMTP求解器计算当前步长的外部节点电压$u_n(t+\Delta t)$。

5. 步骤5：通过状态反演公式$x(t+\Delta t) = K u_n(t+\Delta t) + \phi(t)$直接求解内部状态变量，跳过传统节点消去法中内部节点电压的反求步骤。

6. 步骤6：利用新求解的$x(t+\Delta t)$和$u_n(t+\Delta t)$更新下一时刻的历史电流源$i_h(t+\Delta t)$。若检测到闭锁模式，根据二极管导通逻辑（$S_{diode}$）更新开关电阻值，实现正常运行与闭锁状态的无缝切换。

### 关键参数

- **仿真步长约束**: ≤ 112 μs（保证离散系统谱半径ρ(φ)<1的数值稳定性）

- **二极管关断电阻**: r_off = 1e6 Ω

- **计算复杂度量级**: 传统NEM为O(n^3)，本方法为O(q^2)

- **示例计算操作数**: NEM为488次，本方法为195次（针对m=4外部节点, n=6内部节点, q=6状态变量场景）

## 量化发现

- 时间复杂度由传统NEM的O(n^3)降低至O(q^2)，计算操作数从488次降至195次（降幅约60%）。
- 等效导纳矩阵Y_n为严格恒定矩阵，不随开关状态切换而变化，彻底消除变导纳求逆带来的计算负担。
- 系统数值稳定临界步长为112 μs，满足该条件时离散系统状态矩阵谱半径ρ(φ)<1。
- 历史电流源更新完全通过状态变量反演实现，省去传统方法中内部节点电压反求步骤，串行计算流程大幅简化。

## 关键公式

### 恒定导纳端口等效方程

$$$i_s(t+\Delta t) = Y_n u_n(t+\Delta t) + i_h(t)$$$

*用于EMTP网络求解，将复杂变换器等效为恒定导纳并联历史电流源，实现与外部电网的直接接口。*

### 状态变量直接反演公式

$$$x(t+\Delta t) = K u_n(t+\Delta t) + \phi(t)$$$

*在求得外部节点电压后，用于直接更新内部状态变量，替代传统节点消去法中繁琐的内部支路反求过程。*

### 恒定导纳矩阵解析式

$$$Y_n = \frac{\Delta t}{2} P Q B, \quad Q = \left(M - \frac{\Delta t}{2} A_\alpha\right)^{-1}$$$

*证明导纳矩阵仅由拓扑参数、积分步长和恒定部分矩阵决定，与开关时变矩阵A_β无关，确保导纳恒定。*
