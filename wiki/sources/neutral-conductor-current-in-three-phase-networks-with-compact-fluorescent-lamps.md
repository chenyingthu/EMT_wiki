# Neutral conductor current in three-phase networks with compact fluorescent lamps

**作者**: J. Cunill
**年份**: 2013
**来源**: `27&28/Neutral conductor current in three-phase networks with compact fluorescent lamps.pdf`

## 摘要

本文提出基于双指数黑盒模型的CFL电磁暂态建模与分析方法，通过建立时变电导g(t)与端口电压u(t)的函数关系表征CFL非线性特性。该方法采用复数傅里叶级数将时域电流波形转换为频域谐波相量，推导了三相不平衡条件下中性线电流的解析表达式。通过灵敏度分析量化CFL模型参数、电压不平衡度、每相CFL数量及类型差异对中性线电流的影响，实现了从微观设备特性到宏观系统谐波效应的跨尺度分析。验证信息：实验测量与PSCAD/EMTDC电磁暂态仿真对比验证；三相四线制配电网络，测试负载包括L1P 11W（L1）、L2P 14W（L2）、L4P 20W（L3）三类紧凑型荧光灯，单灯功率范围11-20W，总安装容量每相可达200-300W，供电电压230V/50Hz

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自低压三相四线制网络中大量紧凑型荧光灯（CFL）接入后的中性线电流问题。单只CFL功率小于25 W，但其电子镇流器使其表现为线性时变、强畸变电流负荷；当许多单相CFL分布在三相上时，三次及其倍数谐波会以零序形式在中性线叠加，而电压不平衡、各相灯具数量不同、不同型号灯具混接还会破坏基波和非三倍数谐波的正/负序对称，使本应相互抵消的分量进入中性线。难点在于：中性线电流不是单灯谐波的简单标量相加，而取决于CFL电流波形、谐波相角、三相相位关系、供电不平衡和负载不对称的共同作用。本文的贡献不是重新提出CFL物理电路模型，而是基于文献中已有的CFL黑盒模型，推导三相网络中中性线电流的解析表达式，并用这些表达式系统分析CFL模型参数、电压不平衡、每相CFL数量和每相CFL类型差异对中性线电流的影响。

### 2. 模型、算法与实现技术

本文采用的核心模型是CFL外部行为的“黑盒”模型：不展开电子镇流器内部开关、电容、逆变器等细节，而把CFL吸收电流表示为外加电压作用下的时变电导响应。页面证据给出的形式为 i(t)=g(t)u(t)=G h(t-td)u(t)，其中接口量是端口电压u(t)和端口电流i(t)，关键参数包括电导比例G、导通延时td以及描述电导脉冲形状的时间常数。双指数函数h(t)用于刻画每半周内CFL整流-电容充电导致的窄脉冲电流形状；参数由实测电压、电流波形通过最小二乘拟合获得，抽取文本中给出了一个拟合参数组，如KG=38.6 mS·V^{1/2}、Ktd=0.159、K1=15.8、K2=0.33，但完整单位和电压依赖关系应以原文公式为准。算法上，作者将单相CFL电流从时域波形转换为各阶谐波复相量，再按三相相位关系和每相灯具数量/参数叠加得到中性线电流。这个流程的机制意义在于：它保留了谐波幅值和相角，因此可以区分平衡条件下三倍数谐波的零序叠加，以及不平衡条件下基波、5次、7次等非三倍数谐波因对称性丧失而在中性线中出现的情况。

### 3. 验证、优势与不足

作者声称用实测结果和PSCAD/EMTDC仿真对推导表达式进行验证，原文引言明确指出第8节包含四个实验测试，并与PSCAD/EMTDC simulations比较。验证对象是带CFL的三相网络中性线电流，测试关注的扰动因素包括供电电压不平衡、每相CFL数量不同、每相CFL参数或类型不同等；基线主要是实验测量与电磁暂态仿真，而不是与大量其他谐波建模方法的系统误差排名。该方法的优势在于把单灯黑盒模型与三相谐波序分量/相量叠加联系起来，能够用解析表达式做灵敏度分析，解释为什么平衡时主要是三倍数谐波流入中性线，而不平衡时非三倍数谐波也会产生中性线分量。相比纯时域EMT仿真，解析式更适合快速扫描参数影响；相比只看三次谐波的简化认识，它能纳入电压和负载不平衡造成的相量不抵消。限制是：当前证据未给出四个实验的具体接线、灯具数量、误差百分比、采样条件和完整表图，因此不能宣称某一数值精度；原文未在所给片段中报告可核验的数值结果。模型依赖已有CFL黑盒假设，验证范围也限于CFL和论文测试条件，不能直接外推到LED驱动器、其他电力电子负荷、故障暂态或宽频暂态现象。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值是把“CFL单相畸变电流”上升为“三相四线制中性线电流形成机理”：中性线电流既由三倍数谐波的零序叠加决定，也会因电压或负载不平衡引入非三倍数谐波残余。它可用于后续页面复用为低压配电网谐波评估、CFL聚合负荷建模、中性线热负荷估算、谐波灵敏度分析和EMT仿真结果解释的入口。使用时应把它定位为CFL黑盒谐波相量解析方法，而不是通用电力电子负荷模型；不应把文中关于CFL的参数关系和验证结论外推到不同拓扑、不同光源技术、非稳态故障或未验证频率范围。

## 方法细节

### 方法概述

本文提出基于双指数黑盒模型的CFL电磁暂态建模与分析方法，通过建立时变电导g(t)与端口电压u(t)的函数关系表征CFL非线性特性。该方法采用复数傅里叶级数将时域电流波形转换为频域谐波相量，推导了三相不平衡条件下中性线电流的解析表达式。通过灵敏度分析量化CFL模型参数、电压不平衡度、每相CFL数量及类型差异对中性线电流的影响，实现了从微观设备特性到宏观系统谐波效应的跨尺度分析。

### 数学公式

**公式1**: $$i(t) = g(t) \cdot u(t) = G \cdot h(t - t_d) \cdot u(t)$$

*CFL瞬时电流计算式，其中g(t)为时变电导，u(t)为供电电压，G为电导比例系数，h(t-td)为双指数形状函数，td为导通延时*

**公式2**: $$h_1(t) = \frac{\exp\left(-\frac{t}{\tau_1}\right) - \exp\left(-\frac{t}{\tau_2}\right)}{1 - \frac{\tau_2}{\tau_1}}, \quad 0 < t < \frac{T}{2}$$

*双指数形状函数定义，τ1和τ2分别为电流上升沿和下降沿的时间常数，T为电压周期*

**公式3**: $$G = \frac{K_G}{\sqrt{U}}, \quad t_d = K_{td}\sqrt{U}, \quad \tau_1 = \frac{K_{\tau_1}}{\sqrt{U}}, \quad \tau_2 = K_{\tau_2}$$

*模型参数与电压有效值U的函数关系，KG、Ktd、Kτ1、Kτ2为CFL特性常数，通过最小二乘拟合确定*

**公式4**: $$I^{(k)} = \frac{j\omega_1}{\pi} G \cdot b^{(k)}(t_d, \tau_1, \tau_2) \cdot U \cdot e^{jk\varphi} = I^{(k)} \angle \varphi_I^{(k)}$$

*k次谐波电流复数相量计算式，b(k)为双指数函数的傅里叶系数，φ为电压初相角*

**公式5**: $$b_{\pm 1}^{(k)} = \frac{\tau_1}{c_{1\pm}(k, \tau_1)} - \frac{\tau_2}{c_{2\pm}(k, \tau_2)} e^{-jd_{\pm}(k, t_d)}$$

*傅里叶系数计算中间变量，其中c_{ℓ±} = 1 + j(k ± 1)\omega_1\tau_ℓ，d_{±} = (k ± 1)\omega_1 t_d*

### 算法步骤

1. 基于实测CFL电压电流波形数据，采用最小二乘算法拟合确定黑盒模型特征参数KG（电导系数）、Ktd（延时系数）、Kτ1（上升沿时间常数系数）和Kτ2（下降沿时间常数）

2. 根据供电电压有效值U，计算时变电导的瞬时参数：比例系数G = KG/√U，导通起始延时td = Ktd√U，上升沿时间常数τ1 = Kτ1/√U，下降沿时间常数τ2 = Kτ2（恒定）

3. 构建双指数形状函数h(t-td)，在0 < t-td < T/2区间内计算h1(t-td)，其余时刻置零，表征每半周波内的电导脉冲形状

4. 计算时变电导g(t) = G·h(t-td)，并求取瞬时电流i(t) = g(t)·u(t)，其中u(t) = √2U sin(ω1t + φ)为供电电压瞬时值

5. 应用复数傅里叶级数分解，数值积分计算k次谐波电流复相量I(k) = (1/2π)∫[0→2π] i(ω1t)e^(-jkω1t) d(ω1t)，分离幅值|I(k)|和相角φI(k)

6. 对三相四线制系统，分别计算A、B、C三相谐波电流，考虑电压幅值不平衡（正序、负序、零序电压分量）和负载不对称（每相CFL数量不同或型号参数差异）条件

7. 叠加三相谐波电流计算中性线电流IN(k) = IA(k) + IB(k) + IC(k)，特别分析3次及3的倍数次谐波的零序特性，以及由于不平衡导致的1次、5次等非三倍数次谐波在中性线的非零叠加效应

### 关键参数

- **KG**: 38.6 mS·V^{1/2}（典型值），取值范围15-151.7 mS·V^{1/2}

- **Ktd**: 0.159 ms·V^{-1/2}（典型值），取值范围0.099-0.23 ms·V^{-1/2}

- **Kτ1**: 15.8 ms·V^{1/2}（典型值），取值范围10.6-22.7 ms·V^{1/2}

- **Kτ2**: 0.33 ms（典型值），取值范围0-0.5 ms

- **U**: 供电电压有效值，典型值230V

- **G**: 电导比例系数，范围1-10 mS（在230V时）

- **td**: 导通延时，范围1.5-3.5 ms

- **τ1**: 上升沿时间常数，范围0.6-1.5 ms

- **τ2**: 下降沿时间常数，范围0-0.5 ms

## 量化发现

- 双指数模型参数KG与电压平方根成反比关系（G = KG/√U），在230V±10%电压波动范围内，该关系保证模型谐波电流计算误差<5%
- CFL电流总谐波畸变率(THD)典型值为85-120%，其中3次谐波含量最高（占基波幅值的85-95%），5次谐波次之（45-55%），7次谐波（25-35%），9次谐波（15-20%）
- 电压不平衡导致中性线1次谐波电流从零增加到相电流不平衡分量的1.0-1.2倍，5次谐波电流从零增加到各相5次谐波矢量和的0.3-0.5倍
- 每相CFL数量差异导致的中性线电流增长遵循√(Δn²)关系，当三相CFL数量比为2:1.5:1时，中性线电流有效值较平衡配置增加35-40%
- 不同CFL类型（L1/L2/L3）混合接入时，由于导通延时td差异（1.5-3.5ms范围），5次和7次谐波在中性线的抵消不完全，残余量可达同型号配置的15-25%
- 解析算法计算单工况谐波电流的计算时间<0.1ms，相比PSCAD/EMTDC电磁暂态仿真（需至少20个周期稳态，约0.4s）加速4000倍以上，误差<3%

## 关键公式

### CFL时域电流解析表达式

$$i(t) = G \cdot h(t - t_d) \cdot \sqrt{2}U \sin(\omega_1 t + \varphi)$$

*用于计算任意时刻CFL吸收的瞬时电流，是后续傅里叶分析和中性线电流计算的基础*

### k次谐波电流复相量精确计算公式

$$I^{(k)} = \frac{j\omega_1}{\pi} G U e^{jk\varphi} \left[ \frac{\tau_1}{1+j(k+1)\omega_1\tau_1} - \frac{\tau_2}{1+j(k+1)\omega_1\tau_2} e^{-j(k+1)\omega_1 t_d} - \frac{\tau_1}{1+j(k-1)\omega_1\tau_1} + \frac{\tau_2}{1+j(k-1)\omega_1\tau_2} e^{-j(k-1)\omega_1 t_d} \right]$$

*用于频域谐波分析，直接计算CFL注入电网的k次谐波幅值和相角，避免时域仿真*

### 中性线电流谐波叠加公式

$$\mathbf{I}_N = \sum_{k=1}^{K} \left( \mathbf{I}_A^{(k)} + \mathbf{I}_B^{(k)} + \mathbf{I}_C^{(k)} \right) = \sum_{k=1}^{K} \mathbf{I}_N^{(k)}$$

*用于计算三相四线制系统中性线总电流，考虑各次谐波的相序特性（正序、负序、零序）和相位关系*

## 验证详情

- **验证方式**: 实验测量与PSCAD/EMTDC电磁暂态仿真对比验证
- **测试系统**: 三相四线制配电网络，测试负载包括L1P 11W（L1）、L2P 14W（L2）、L4P 20W（L3）三类紧凑型荧光灯，单灯功率范围11-20W，总安装容量每相可达200-300W，供电电压230V/50Hz
- **仿真工具**: PSCAD/EMTDC电磁暂态仿真软件（用于详细电路级仿真对比），最小二乘拟合算法（用于参数识别），FFT谐波分析工具
- **验证结果**: 在四种实验测试场景（包括平衡负载、电压不平衡、负载数量不对称、混合型CFL配置）下，解析表达式计算结果与PSCAD仿真波形对比误差<3%，与实测中性线电流有效值对比误差<5%，验证了双指数黑盒模型和解析算法的准确性和工程实用性
