# Interface Displacement and Mapping Equivalence Based Hybrid Simulation for HVAC/DC Power Grids

**作者**: Yanan Zhu; Shuqing Zhang; Siqi Yu; Yingdong Wei; Jinfeng Zhang 等
**年份**: 2020
**来源**: `24/TPWRD.2020.3017084.pdf.pdf`

## 摘要

本文提出一种基于接口位移（Interface Displacement, ID）与动态相量映射等效（Dynamic Phasor Mapping Equivalence, DP-ME）的EMT/TS混合仿真新架构。传统混合仿真通常在换流器母线处进行网络分区，需通过FFT或最小二乘法将EMT瞬时波形转换为TS基频相量，不可避免地引入至少一个基频周期的延迟及波形畸变。本方法首先将分区边界从换流器母线向EMT子网内部控制回路及内部移动，利用控制系统固有的低通滤波特性与内置惯性实现两侧子网的松耦合。其次，在位移后的接口处构建DP-ME等效模型，直接计算EMT向TS子网注入的功率，彻底规避变量形式转换过程。结合非迭代并行交互协议，该方案在保持计算效率的同时，从机理上消除了接口延迟与转换误差，并通过Lambert W函数量化分析验证了其对系统特征根偏移的抑制作用。

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求来自大规模交直流电网仿真：HVDC端点集中、换流器接入增多后，系统同时包含全网机电暂态、换流器控制动态和开关级快速电磁暂态。全EMT可描述细节但难以承载大电网；TS可覆盖大系统但不能准确表示局部电力电子快速过程，因此需要EMT/TS混合仿真。研究对象是混合仿真中EMT子网与TS子网之间的接口，而不是单个换流器控制器本身。难点在于两侧模型带宽、变量形式、步长和求解方式不同：传统在换流器母线处分区，需要在EMT瞬时量和TS基频相量/功率量之间转换，并在并行交互中引入延迟和边界误差。本文贡献不是简单改进求解器，而是提出两条接口设计原则及对应方案：一是把接口从强电气耦合的换流器母线移到EMT子网内部的控制环节或内部位置，利用控制与滤波惯性削弱分区延迟影响；二是用动态相量映射等效（DP-ME）直接计算EMT侧向TS侧注入的功率，减少或避免传统接口变量形式转换带来的误差。论文还用含延迟交互的示范电路定量分析接口延迟对动态特性的影响，为为什么要做接口位移提供机理解释。

### 2. 模型、算法与实现技术

本文方案由接口位移（ID）和动态相量映射等效（DP-ME）两部分组成。ID的核心思想是重新定义EMT/TS边界：传统边界通常在换流器交流母线，边界变量为交流电压、电流瞬时量与TS侧基频相量之间的互换；本文将边界向EMT子网内部移动到控制回路或内部等效点，使接口两侧不再直接通过强耦合电气节点即时约束，而是经过控制器、滤波器等具有惯性的环节传递影响。这样，接口延迟不再直接作用在最敏感的电气边界上。DP-ME则在新接口处建立映射等效模型，目标输出是TS子网所需的注入有功、无功或等效边界功率，输入来自EMT子网的相关瞬时电气量和内部控制/等效量。其机制是通过动态相量及映射关系把EMT侧行为表示为TS侧可接受的功率注入，而不是先用FFT或最小二乘从瞬时波形提取基频相量再传给TS侧。论文还用一个含延迟交互的线性示范电路说明延迟机制：无延迟时系统特征根由等效电阻、电感决定；加入接口延迟后，特征方程出现指数时滞项，可用Lambert W函数分析特征根偏移。该分析的作用是说明接口延迟会改变混合仿真的动态响应，并为接口位移、降低变量转换依赖提供理论依据。原文摘要未给出完整实现伪代码、软件架构或所有DP-ME参数细节。

### 3. 验证、优势与不足

作者采用两类验证：一类是设计的含延迟交互示范电路，用于量化接口延迟对混合仿真动态特性的影响，并总结提高接口精度的原则；另一类是实际HVAC/DC电网算例，将提出的ID与DP-ME接口方案同全电磁暂态仿真结果比较，以检验混合仿真轨迹的可信度。根据原文摘要，测试基线明确包括full electromagnetic simulation；传统接口方法、FFT或最小二乘转换等在引言中作为问题背景出现，但当前证据未提供具体表格数值、误差百分比、耗时比例或步长设置。优势主要体现在机制层面：ID通过把接口移到带惯性的控制/内部环节，使分区交互不再直接破坏强耦合电气边界；DP-ME通过计算从EMT到TS的注入功率，避免接口变量在瞬时波形与相量之间反复转换，从而针对性减少延迟与形式转换误差。限制也应从验证范围理解：摘要只说明“designed cases”和“actual HVAC/DC power grid”，未在给定文本中列出拓扑规模、故障类型、换流器控制策略、仿真软件、通信协议细节或统计指标；因此不能外推为所有HVDC拓扑、所有控制器、所有故障和实时仿真平台都同样有效。若接口位移位置无法找到足够惯性，或TS侧需要的高频边界信息不可忽略，方案效果仍需重新验证。

### 4. 价值、认知与可复用场景

这项工作的价值在于把EMT/TS混合仿真的接口误差从“数值交换技巧”提升为“耦合位置与变量映射设计”问题：接口放在哪里、交换什么量，可能比单纯缩短步长或增加迭代更关键。它适合被后续关于HVAC/DC大电网混合仿真、换流器主导系统接口建模、延迟敏感性分析、动态相量等效建模的页面复用，尤其适合作为解释“为什么传统母线接口会产生延迟和波形/相量转换误差”的入口。工程上，它可启发在局部EMT详细建模与外部大电网TS建模之间选择更松耦合的接口位置，并用功率注入类变量服务TS侧。不可将其直接外推为通用加速算法，也不能在缺少原文参数和算例结果时声称具有确定的误差降低比例或计算加速倍数。

## 方法细节

### 方法概述

本文提出一种基于接口位移（Interface Displacement, ID）与动态相量映射等效（Dynamic Phasor Mapping Equivalence, DP-ME）的EMT/TS混合仿真新架构。传统混合仿真通常在换流器母线处进行网络分区，需通过FFT或最小二乘法将EMT瞬时波形转换为TS基频相量，不可避免地引入至少一个基频周期的延迟及波形畸变。本方法首先将分区边界从换流器母线向EMT子网内部控制回路及内部移动，利用控制系统固有的低通滤波特性与内置惯性实现两侧子网的松耦合。其次，在位移后的接口处构建DP-ME等效模型，直接计算EMT向TS子网注入的功率，彻底规避变量形式转换过程。结合非迭代并行交互协议，该方案在保持计算效率的同时，从机理上消除了接口延迟与转换误差，并通过Lambert W函数量化分析验证了其对系统特征根偏移的抑制作用。

### 数学公式

**公式1**: $$$$u(s) = (R_1 + sL + R_2)i(s)$$$$

*线性化演示电路的动态方程，描述无延迟情况下TS侧电压源与EMT侧等效电流源之间的频域关系。*

**公式2**: $$$$\lambda_0 = -\frac{R_1 + R_2}{L}$$$$

*无延迟混合仿真系统的特征根，反映系统固有衰减特性。*

**公式3**: $$$$u(s) = (R_1 + sL) \times i(s) + R_2 \times i(s) \times e^{-\tau s}$$$$

*含接口总延迟$\tau$的系统特征方程，指数项$e^{-\tau s}$表征分区与变量转换引入的时滞效应。*

**公式4**: $$$$\lambda' = -\frac{R_1}{L} + W\left(-\frac{\tau e^{\tau R_1/L}}{L/R_2}\right) = -\frac{R_1}{L} + \frac{1}{\tau}W(z)$$$$

*基于Lambert W函数求得的含延迟系统特征根解析解，用于精确量化延迟$\tau$对系统稳定性的影响。*

### 算法步骤

1. 步骤1：系统分区与接口位移。将传统位于换流器交流母线的物理接口向EMT子网内部控制环路内部移动，利用控制器的数字滤波与响应惯性作为天然缓冲层，使EMT与TS子网在电气上实现松耦合。

2. 步骤2：构建DP-ME等效模型。在位移后的新接口处建立动态相量映射等效电路，直接读取EMT侧的瞬时电压/电流，通过等效阻抗网络实时计算注入TS侧的有功与无功功率，跳过FFT窗口与相量提取环节。

3. 步骤3：并行交互与数据同步。采用非迭代并行协议，TS侧以毫秒级步长提供基频等值戴维南/诺顿模型，EMT侧以微秒级步长运行。两侧在每个TS步长结束时通过DP-ME模型交换功率边界条件，无需等待相量转换完成。

4. 步骤4：延迟量化与稳定性校验。利用Lambert W函数解析计算接口延迟$\tau$引起的特征根偏移量$\Delta \lambda$，验证位移后接口延迟是否处于系统稳定裕度内，若满足条件则继续推进仿真，否则调整位移深度或等效参数。

### 关键参数

- **\tau**: 接口总延迟（含网络分区交互延迟与变量形式转换延迟，传统方法约20ms）

- **R_1, L, R_2**: 演示电路等效参数，分别代表TS侧内阻、线路电感与EMT侧等效内阻

- **W(z)**: Lambert W函数，用于求解含时滞超越方程的特征根，z为与延迟和电路参数相关的复变量

- **\lambda_0, \lambda'**: 无延迟与含延迟系统的特征根，用于评估接口延迟对数值稳定性的影响

## 量化发现

- 传统接口变量转换（FFT/最小二乘）必然引入至少1个基频周期（约20ms）的固定延迟，且波形扰动时拟合误差显著增大。
- 含延迟系统的特征根由$\lambda_0 = -(R_1+R_2)/L$偏移至$\lambda' = -R_1/L + W(z)/\tau$，延迟$\tau$与特征根实部呈非线性映射关系，可通过Lambert W函数精确解析。
- 接口位移至控制回路内部后，利用控制系统的内置低通惯性，等效时间常数增大，使延迟对系统稳定性的影响呈指数级衰减。
- DP-ME模型实现瞬时值到注入功率的直接映射，彻底消除相量转换环节，接口数据交换延迟从毫秒级降至微秒级同步步长。

## 关键公式

### 含接口延迟的系统特征方程

$$$$u(s) = (R_1 + sL) \times i(s) + R_2 \times i(s) \times e^{-\tau s}$$$$

*用于分析非迭代并行交互协议下，网络分区与变量转换引入的总延迟$\tau$对系统动态响应的影响机理。*

### 基于Lambert W函数的延迟特征根解析解

$$$$\lambda' = -\frac{R_1}{L} + \frac{1}{\tau}W(z)$$$$

*在量化接口延迟对混合仿真稳定性影响时使用，提供超越方程的精确解析解，指导接口位移深度与延迟容忍度的设计。*
