## 方法细节

### 方法概述

提出一种将动态谐波域（DHD）与有理函数模型无缝接口的通用框架，用于频变网络等值的电磁暂态分析。首先利用矢量拟合（VF）法对网络端口导纳矩阵（Y矩阵）进行极点-留数有理逼近，将其转换为连续时间状态空间模型。随后采用梯形积分规则对状态方程进行离散化，消除当前时刻输入对状态变量的直接代数依赖。接着将离散化方程映射至DHD，通过构造Toeplitz矩阵与谐波微分矩阵将时变周期系统转化为线性时不变系统，实现频域谐波暂态与稳态的直接求解。针对DHD建模导致的状态维数爆炸问题，引入平衡实现理论计算可控性与可观性格拉姆矩阵，提取汉克尔奇异值进行模型降阶，在保留主导动态特性的前提下显著降低计算负担，最终通过频域代数/差分方程直接输出各次谐波的时变幅值与相位。

### 数学公式


**公式1**: $$$x(\tau) = \sum_{h=-\infty}^{\infty} X_h(t) e^{jh\omega\tau}$$$

*动态谐波域（DHD）展开式，将周期或准周期信号表示为时变复傅里叶系数与基频谐波的乘积和。*


**公式2**: $$$\dot{X}(t) = (A_h - D_h)X(t) + B_h U$$$

*DHD状态空间方程，$A_h$和$B_h$为系统矩阵的Toeplitz形式，$D_h$为谐波微分矩阵，将时变系统转化为线性时不变形式。*


**公式3**: $$$f(s) = \sum_{m=1}^{N} \frac{r_m}{s-a_m} + d + se$$$

*矢量拟合（VF）极点-留数有理逼近模型，用于精确拟合频变Y矩阵元素。*


**公式4**: $$$x_k = \alpha x_{k-1} + B v_{k-1}$$$

*梯形积分离散化后的状态更新方程，消除了$k$时刻输入$v_k$的直接馈通，保证数值稳定性。*


**公式5**: $$$X_k^d = \alpha_d X_{k-1}^d + B_d V_{k-1}^d, \quad I_k^d = \tilde{C}_d X_k^d + G_d V_k^d$$$

*DHD域离散化系统方程，所有矩阵均为Toeplitz结构，维度随谐波数$h$和极点阶数$N$呈线性增长。*


**公式6**: $$$\xi_i = \sqrt{\lambda_i(PQ)}$$$

*汉克尔奇异值定义，$P$和$Q$分别为可控性与可观性格拉姆矩阵，用于量化各状态对系统输入输出能量的贡献度。*


### 算法步骤

1. 步骤1：频域扫描与Y矩阵构建。在目标频带（如1Hz~100kHz）内以固定步长（如10Hz）计算网络端口导纳矩阵$Y(s)$的精确频率响应。

2. 步骤2：矢量拟合（VF）有理逼近。对$Y(s)$的每个元素应用带松弛的VF算法，迭代优化极点位置，直至收敛，获得$N$阶极点-留数模型$f(s) = C(sI-A)^{-1}B + d + se$。

3. 步骤3：连续状态空间转换。将极点-留数模型直接映射为连续时间状态空间方程$\dot{x} = Ax + Bv$，$i = Cx + Dv$。

4. 步骤4：梯形积分离散化。采用梯形规则对状态方程进行离散，推导得到$x_k = \alpha x_{k-1} + B v_{k-1}$与$i_k = \tilde{C} x_k + G v_k$，消除代数环并保证数值稳定性。

5. 步骤5：DHD域映射。将离散化方程中的标量/向量扩展为谐波向量，利用Toeplitz矩阵构造$\alpha_d, B_d, \tilde{C}_d, G_d$，并引入微分矩阵$D_h$，形成DHD差分方程组。

6. 步骤6：平衡实现降阶。求解Lyapunov方程获取格拉姆矩阵$P$和$Q$，计算汉克尔奇异值$\xi_i$。设定主导性阈值（如$10^{-3}$）或误差界（如$<0.05$），截断低能量状态，得到降阶模型。

7. 步骤7：频域谐波求解。稳态下直接求解代数方程$X = -(A_h - D_h)^{-1}B_h U$；暂态下按时间步长迭代差分方程，直接输出各次谐波的时变复数幅值，无需后处理FFT。


### 关键参数

- **VF拟合阶数**: 测试1: 10阶; 测试2: 50阶; 测试3: 45阶

- **谐波阶数(h)**: 5阶（覆盖基波至5次谐波）

- **仿真步长**: 测试1: 100μs; 测试2: 50μs

- **频率扫描范围**: 1Hz ~ 100kHz（步长10Hz）

- **降阶阈值**: 汉克尔奇异值主导条件 $10^{-3}$，或加性误差界 $<0.05$

- **矩阵维度公式**: $\alpha_d, D_d, B_d$ 维度为 $(n_1 N)(2h+1) \times (n_1 N)(2h+1)$，$\tilde{C}_d, G_d$ 维度为 $(n_2)(2h+1) \times (n_1 N)(2h+1)$



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 双端口线性电路（已知Y矩阵） | 端口1注入含基波、3次、5次谐波的电压源，端口2在0.5ms时接地切换。DHD方法直接输出各次谐波时变响应，最大拟合偏差低于$10^{-14}$。 | DHD仿真耗时0.135s，传统EMTP耗时0.378s，速度提升约2.8倍；转换为实极点模型后进一步降至0.107s（额外提速21%）。 |

| 三相配电系统（6×6 Y矩阵） | 含架空线与电缆分布参数模型，端口A/B注入不平衡基波及3/5次谐波。稳态求解耗时0.039s；暂态全阶模型仿真耗时112.13s。 | 传统EMTP耗时36.56s（DHD全阶较慢），但经平衡实现降阶至96阶后，DHD耗时降至47.023s（较全阶提速58%），实极点转换可进一步降至71.82s。 |

| 5节点配电系统（含变压器与负荷，15×15 Y矩阵） | 验证降阶方法在更复杂含非线性/频变元件系统中的适用性，采用45阶VF拟合，频带1Hz~100kHz。 | 通过汉克尔奇异值截断实现状态压缩，在保持暂态波形高度一致的前提下，显著缓解DHD矩阵维度随谐波数平方增长的计算瓶颈。 |



## 量化发现

- Y矩阵VF拟合精度极高：$Y_{11}$、$Y_{12}$、$Y_{22}$幅值最大偏差分别为$3.297 \times 10^{-14}$、$1.777 \times 10^{-15}$、$5.808 \times 10^{-14}$。
- 双端口电路DHD稳态/暂态求解速度比EMTP快2.8倍（0.135s vs 0.378s），实极点转换可再提速21%。
- 三相系统全阶DHD暂态仿真耗时112.13s，经平衡实现降阶至96阶后降至47.023s，计算负担降低约58%，且加性误差严格控制在$<0.05$。
- 频带上限对暂态精度影响显著：50kHz上限导致高频分量丢失，暂态波形严重失真；100kHz上限下DHD与EMTP时域波形完全吻合。
- 汉克尔奇异值跨度达5个数量级（最大值0.631，最小值$1.45 \times 10^{-6}$），证明系统存在大量低能量冗余状态，降阶空间巨大。


## 关键公式

### DHD状态空间方程

$$$\dot{X}(t) = (A_h - D_h)X(t) + B_h U$$$

*用于将时变周期系统转化为线性时不变系统，是频域直接求解谐波暂态与稳态的核心数学基础。*

### 梯形积分离散化状态更新

$$$x_k = \alpha x_{k-1} + B v_{k-1}$$$

*消除$k$时刻输入对状态的直接依赖，保证数值积分稳定性，为后续DHD映射提供离散化基础。*

### 汉克尔奇异值

$$$\xi_i = \sqrt{\lambda_i(PQ)}$$$

*用于平衡实现降阶，量化各状态对系统输入输出能量的贡献，指导低能量状态的截断以压缩模型阶数。*



## 验证详情

- **验证方式**: 时域波形对比分析（DHD直接输出 vs EMTP传统时步积分 vs WFFT后处理）
- **测试系统**: 双端口线性电路、三相配电网络（含分布参数线路/电缆）、5节点配电系统（含YNd11变压器与14MVA负荷）
- **仿真工具**: EMTP Works v2.0.2（基线对比）、MATLAB/自定义DHD求解器（本文方法）、Unipower Unilyzer 900（实测数据参考）
- **验证结果**: DHD方法在稳态与暂态下均能精确复现频变网络谐波响应，时域波形与EMTP高度吻合（偏差可忽略）。WFFT在开关切换等暂态期间因窗函数截断产生虚假振荡，而DHD直接输出无此缺陷。频带上限需覆盖至100kHz以保证高频暂态精度。平衡实现降阶在误差界$<0.05$内有效压缩状态维数，验证了框架在复杂频变网络中的高效性与数值稳定性。
