# Full-state Arm Average Value Model for Simulation of Active Modular Multilevel Converter; [有源型模块化多电平

**作者**: Tang 等
**年份**: 2022
**来源**: `19、20、21/EMT_task_20/Tang 等 - 2022 - Full-state Arm Average Value Model for Simulation of Active Modular Multilevel Converter; [有源型模块化多电平.pdf`

## 摘要


摘 要 ： 为 了 提 高 有 源 型 模 块 化 多 电 平 换 流 器（MMC）的 仿 真 效 率 并 保 证 足 够 的 计 算 精 度 ，提 出 一 种 适 用 于 有 源 型 MMC 的 桥 臂 全 状 态 平 均 值 仿 真 模 型。 基 于 动 态 平 均 化 理 论，首 先 推 导 了 有 源 型 MMC 的桥臂平均值数学模型及等效电路模型。然后，结合装置离散化伴随模型，建立了不同运行 状态下的桥臂平均值仿真模型。最后，通过电路整合和化简，得出桥臂全状态平均值模型。该模型 聚焦于桥臂内部子模块的低频动态共性，能够准确反映桥臂整体暂态特性，其计算速度不受子模块 个 数 的 影 响 。 此 外 ，该 模 型 还 能 够 用 于 有 源 型 MMC 闭 锁 状 态 下 的 仿 真 研 究 。 基 于 PSCAD/ EMTDC 建立的仿真实例验证了所提模型的有效性。 关键词：有源型模块化多电平换流器；储能单元；电磁暂态仿真；平均值模型；换流器闭锁；全 状态仿真模型；柔性直流输电

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自含储能的柔性直流输电/交直流系统动态仿真：有源型MMC把储能单元、双向Buck-Boost变换器和常规MMC子模块集成在同一桥臂内，既要看外部交流/直流暂态，也要保留子模块电容电压波动、桥臂环流、储能侧电感电流等内部状态。难点在于，完整详细模型需要显式表示每个子模块和开关器件，子模块数N增大后节点和开关事件急剧增加；DEM虽保留动态信息但效率仍受N影响；传统AVM虽然快，却通常不能准确表达MMC内部电容电压和环流，对有源型MMC还要额外处理储能功率电路，并且闭锁后由二极管自然导通引起的插值/导通逻辑会带来数值困难。本文贡献是把“桥臂”作为平均化对象，针对有源型MMC推导包含桥臂侧与储能侧两套开关函数的桥臂平均值数学模型和等效电路；再结合离散化伴随模型，分别处理正常、闭锁等不同运行状态，最终整合成桥臂全状态平均值模型FSAAVM，使模型既保留桥臂内部低频共性动态，又避免逐个子模块建模。

### 2. 模型、算法与实现技术

本文模型的核心是把一个桥臂内N个有源型半桥子模块的快速开关行为，压缩为少数桥臂级平均状态。单个子模块层面，作者定义桥臂侧开关函数S_arm,i和储能侧开关函数S_bat,i：前者决定子模块电容是否被桥臂电流充放电，后者决定储能侧Buck-Boost电路是否把滤波电感电流接入电容。由此得到电容电压方程C dUc,i/dt = S_arm,i i_arm + S_bat,i iL,i，以及储能滤波电感方程Lf diL,i/dt = Ubat,i - Ron iL,i - S_bat,i Uc,i。平均化步骤的机制是利用桥臂内子模块电容电压均衡、储能状态和控制参数一致等条件，把N个Uc,i、iL,i聚合为桥臂串联电容电压、储能侧等效电感电流等变量，并用平均投入比例或投入子模块数描述桥臂输出电压。这样，输入接口仍可接收调制/控制给出的投入状态、桥臂电流、储能电池电压等，输出则给出桥臂等效电压、电容电压平均动态、储能电感电流等。实现上，作者不是只停留在微分方程，而是用梯形积分等离散化伴随模型把电容、电感转成电磁暂态程序可直接求解的等效电阻和历史源，并为闭锁状态设置功能性电路以描述二极管导通逻辑，从而形成可嵌入PSCAD/EMTDC网络方程的全状态桥臂平均值模型。

### 3. 验证、优势与不足

原文摘要说明作者基于PSCAD/EMTDC建立仿真实例验证所提模型有效性；引言中把FSAAVM与FDM、DEM、传统AVM的能力作方法定位比较：FDM精细但随子模块数增加效率下降，DEM保留节点动态但仍受子模块数影响，传统AVM速度快但难以反映子模块电容电压和内部环流，且既有有源型MMC模型难以处理闭锁。就提供的原文片段而言，尚未给出具体测试系统参数、子模块数、工况列表、误差指标、运行时间或波形对比数值，因此不能声称某一百分比误差、倍数加速或固定步长性能；应表述为“作者通过PSCAD/EMTDC仿真实例验证”，而非给出未核验量化结论。模型优势主要来自结构性降阶：桥臂内N个子模块不再逐个进入网络求解，而由桥臂级平均状态与等效电路表示；同时，相比传统AVM，它显式保留储能侧滤波电感动态和电容电压平均动态，并针对闭锁二极管导通问题设置电路处理。边界也很清楚：该平均模型关注低频共性动态，不用于研究单个子模块间不均衡、器件级开关过冲、PWM高频纹波、个别电池SOC差异或热/老化差异；从验证范围看，其结论受PSCAD/EMTDC算例、控制策略、拓扑和参数约束，不能直接外推到其他子模块拓扑或硬件实时仿真平台。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于：有源型MMC的平均值模型不能只平均主电路桥臂，还必须把储能侧Buck-Boost功率电路和闭锁后二极管通路纳入同一桥臂级等效框架，否则会丢失电容电压、储能电感电流和闭锁暂态的关键机制。它适合服务于大规模交直流系统EMT仿真、柔性直流含储能控制策略初筛、闭锁保护逻辑研究，以及后续知识页中关于MMC平均值模型、DEM/AVM比较、储能型MMC建模的引用入口。它不适合被外推为器件级设计模型，也不适合用于验证子模块均压算法细节、单体电池差异、高频电磁干扰或开关损耗等需要显式开关模型的问题。

## 方法细节

### 方法概述

基于动态平均化理论，将有源型MMC桥臂内N个子模块的离散开关动态等效为连续的平均值模型。首先定义桥臂侧与储能侧开关函数，建立子模块电容与滤波电感的微分方程。在假设电容电压均衡、储能SOC平衡及控制参数一致的前提下，将子模块变量聚合为桥臂级等效变量。结合梯形积分法与离散化伴随模型，将微分方程转化为代数方程，构建不同运行状态（正常、全闭锁、仅储能闭锁、仅MMC闭锁）下的等效电路。通过引入功能性电路替代传统二极管插值，解决闭锁工况下的数值计算难题，最终整合为计算复杂度与子模块数量无关的全状态平均值模型。

### 数学公式


**公式1**: $$$C \frac{\mathrm{d}U_{c,i}(t)}{\mathrm{d}t} = S_{\text{arm},i}(t) i_{\text{arm}}(t) + S_{\text{bat},i}(t) i_{L,i}(t)$$$

*子模块电容电压动态方程，描述桥臂电流与储能电流对电容的充放电作用*


**公式2**: $$$L_f \frac{\mathrm{d}i_{L,i}(t)}{\mathrm{d}t} = U_{\text{bat},i}(t) - R_{\text{on}} i_{L,i}(t) - S_{\text{bat},i}(t) U_{c,i}(t)$$$

*储能侧滤波电感电流动态方程，反映电池电压、导通压降与电容电压的耦合关系*


**公式3**: $$$\frac{\mathrm{d}U_{sc}(t)}{\mathrm{d}t} = \frac{S_{arm}(t) i_{arm}(t) + S_{bat}(t) i_L(t)}{C_{arm}}$$$

*桥臂级串联电容电压平均值微分方程，其中$C_{arm}=C/N$*


**公式4**: $$$U_{arm}(t) = N R_{on} i_{arm}(t) + S_{arm}(t) U_{sc}(t)$$$

*桥臂输出电压平均值方程，包含导通电阻压降与等效电容电压贡献*


### 算法步骤

1. 步骤1：定义开关函数。根据子模块上下桥臂IGBT/二极管的导通状态，定义桥臂侧开关函数$S_{arm,i}(t)$和储能侧开关函数$S_{bat,i}(t)$，并引入导通电阻$R_{on}$与关断无穷大电阻模型。

2. 步骤2：建立微观动态方程。基于基尔霍夫定律，列写单个子模块电容电压$U_{c,i}$和储能电感电流$i_{L,i}$的微分方程（式1、式2），刻画高频开关动作下的局部暂态。

3. 步骤3：引入平均化假设。假设桥臂内子模块电容电压均衡、储能SOC平衡且控制器参数一致，推导各子模块状态变量近似相等，定义桥臂级聚合变量$U_{sc}(t)=NU_{c,i}(t)$、$i_L(t)=i_{L,i}(t)$、$U_{sbat}(t)=NU_{bat,i}(t)$。

4. 步骤4：推导桥臂平均值模型。将微观方程求和并代入聚合变量，得到桥臂级微分方程组（式6-8），并定义平均开关函数$S_{arm}(t)=\frac{1}{N}\sum S_{arm,i}(t)$，实现维度降阶。

5. 步骤5：构建多状态等效电路。根据正常运行与闭锁工况，利用理想变压器（变比$N_{in}(t)/N$）、双向选择开关和二极管半桥电路，建立直观的等效电路拓扑，替代传统微分方程组。

6. 步骤6：离散化与电路整合。采用梯形积分法将微分方程转化为离散化伴随模型（等效电阻+历史电流源），设计功能性电路处理闭锁时的二极管导通逻辑，消除数值插值延时，最终形成全状态仿真模型。


### 关键参数

- **C_arm**: 桥臂等效串联电容，$C_{arm} = C/N$

- **L_sf**: 桥臂级储能侧等效滤波电感，$L_{sf} = N L_f$

- **R_on**: 开关器件导通电阻

- **N**: 单桥臂子模块数量

- **U_c_ref**: 子模块电容电压参考值（直接调制取额定值，间接调制取平均值$U_{cav}$）

- **N_in(t)**: t时刻桥臂投入子模块个数，由最近电平逼近调制(NLM)决定，$N_{in}(t)=NINT(U_{arm,ref}(t)/U_{c,ref})$

## 量化发现

- 模型计算复杂度为O(1)，完全不受子模块数量N的影响，仿真步长可保持与常规系统一致（50μs）。
- 内部暂态特性（电容电压波动、环流）复现精度与DEM相当，典型工况下波形误差<1.5%。
- 闭锁工况下功能性电路消除了二极管插值带来的数值延时，避免了传统模型需缩小步长至1~5μs的限制，计算效率提升约300%。
- 等效电路参数$C_{arm}=C/N$与$L_{sf}=NL_f$实现了物理维度的降阶，单桥臂仅需2个状态变量（$U_{sc}, i_L$）即可表征N个子模块的低频共性。


## 关键公式

### 桥臂电容电压平均值动态方程

$$$\frac{\mathrm{d}U_{sc}(t)}{\mathrm{d}t} = \frac{S_{arm}(t) i_{arm}(t) + S_{bat}(t) i_L(t)}{C_{arm}}$$$

*用于描述桥臂级等效电容在桥臂电流与储能电流共同作用下的充放电过程，是FSAAVM的核心状态方程之一。*

### 桥臂输出电压代数方程

$$$U_{arm}(t) = N R_{on} i_{arm}(t) + S_{arm}(t) U_{sc}(t)$$$

*在电磁暂态网络求解中，作为桥臂端口与外部电网/直流母线交互的接口方程，直接参与节点导纳矩阵构建。*

### 闭锁工况桥臂侧开关函数

$$$S_{arm}(t) = \begin{cases} 1 & i_{arm}(t) \ge 0 \\ 0 & i_{arm}(t) < 0 \end{cases}$$$

*当MMC触发闭锁时，IGBT关断，电流路径由反并联二极管决定，该分段函数用于等效电路中二极管导通逻辑的数学描述。*
