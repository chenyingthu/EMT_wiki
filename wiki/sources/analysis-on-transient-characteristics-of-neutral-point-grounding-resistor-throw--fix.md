## 主要发现


- 特征频段内消弧线圈电感不影响出线暂态电流分布，模型中可忽略
- 低阻故障暂态电流含高频振荡与低频直流分量，解析解与仿真高度吻合
- 高阻故障暂态过程主要由接地变压器零模电感与线路对地电容串联谐振主导



## 方法细节

### 方法概述

本文针对灵活接地系统并联小电阻投入时的暂态过程，首先基于Karenbauer模量变换与均匀传输线理论，构建包含线路分布参数的零模网络分析模型。通过推导故障线路入端阻抗的相频特性，界定“特征频段”，并在此频段内提出忽略消弧线圈电感影响的等值电路简化方法。针对低阻与高阻接地故障，分别建立三阶与二阶微分方程，解析求解暂态电流与电压的表达式。进一步推导谐振频率、衰减时间常数及最大幅值的解析式，量化分析系统参数对暂态特征的影响规律。最后，结合各线路零模电流分布特征阐明保护理论依据，并通过数字仿真验证模型准确性。

### 数学公式


**公式1**: $$$Z_{\text{in}} = Z_{cx} \frac{R_f + jZ_{cx} \tan(\omega \sqrt{L_{n0}C_{n0}} l)}{Z_{cx} + jR_f \tan(\omega \sqrt{L_{n0}C_{n0}} l)}$$$

*故障线路入端阻抗表达式，用于分析线路阻抗随频率变化的相频特性，界定特征频段及简化模型适用范围。*


**公式2**: $$$L_Z L_0 C_\Sigma \frac{d^3 i_f}{dt^3} + C_\Sigma (R_n L_0 + R L_Z) \frac{d^2 i_f}{dt^2} + (R_n R C_\Sigma + L_Z + L_0) \frac{di_f}{dt} + (R_n + R) i_f = U_m \sin(\omega_0 t + \varphi)$$$

*低阻接地故障暂态电流三阶微分方程，描述小电阻投入后故障点电流的高频振荡与低频直流分量演化过程。*


**公式3**: $$$\omega_f = \frac{\sqrt{3}}{2} \frac{\sqrt[3]{Y_1 - Y_2}}{\sqrt[3]{3 L_Z L_0 C_\Sigma}}$$$

*高频振荡角频率解析式，直接关联系统参数与故障条件，用于预测暂态主频并指导滤波器设计。*


**公式4**: $$$L_Z C_\Sigma \frac{d^2 u_{C\Sigma}}{dt^2} + R_n C_\Sigma \frac{du_{C\Sigma}}{dt} + u_{C\Sigma} = u_r$$$

*高阻故障高频段暂态电压二阶微分方程，揭示接地变压器零模电感与系统对地电容串联谐振主导的暂态机理。*


### 算法步骤

1. 基于Karenbauer模量变换提取零模分量，构建含分布参数线路、接地变压器、消弧线圈及并联小电阻的零模网络拓扑模型，设定小电阻投入时刻为暂态分析起点。

2. 推导健全线路与故障线路的入端阻抗相频特性，计算各线路首次谐振频率，定义0至最小首次谐振频率区间为“特征频段”，作为后续模型简化的频率边界。

3. 在特征频段内分析阻抗性质：健全线路呈容性，低阻故障线路呈感性，高阻故障线路呈容性；据此判定消弧线圈感抗远大于并联阻抗，将其从等值电路中忽略，建立低阻/高阻故障的简化RLC电路。

4. 针对简化电路列写微分方程（低阻为三阶，高阻低频为二阶、高频为二阶），利用一元三次方程求根公式或二阶系统通解公式求解特征根，分离高频振荡模态与低频直流模态。

5. 结合小电阻投入前的稳态初始条件（电容电压、电感电流），确定微分方程特解系数，得到暂态电流/电压的完整时域解析表达式。

6. 提取解析式中的高频振荡频率、衰减时间常数及幅值参数，进行参数扫描与偏导分析，量化揭示其随故障电阻、线路长度、系统电容电流、中性点接地电阻及接地变压器电感的变化规律。

7. 基于暂态零序电压解析式，推导健全线路与故障线路的零序电流分布公式，对比极性、幅值关系，形成适用于灵活接地系统的暂态保护选线与检测理论依据。


### 关键参数

- **系统对地电容电流**: ≤200 A（典型仿真取58 A）

- **消弧线圈脱谐度**: -10% ~ -5%

- **接地变压器零模电感**: ≤30 mH（典型取20 mH）

- **中性点接地电阻(Rn/3)**: 4~16 Ω（典型取16 Ω）

- **线路波阻抗Zcx**: √(Ln0/Cn0)

- **临界故障电阻**: 等于线路波阻抗Zcx



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 低阻接地故障暂态频率验证 | 在10kV系统、5Ω故障电阻、1km电缆线路条件下，简化模型计算谐振频率范围为93~2323Hz，与分布参数仿真结果高度一致。 | 相对误差控制在±7%以内，显著优于传统集中参数模型在高频段的偏差，满足现有采样装置频带要求。 |

| 低阻故障衰减时间常数验证 | 高频暂态分量衰减时间常数δ1在65.2~14038范围内变化，对应暂态持续时间为0.07~15ms。 | 简化模型计算值与分布参数仿真值相对误差基本在±10%以内，精度满足工程保护动作时间窗需求。 |

| 高阻接地故障暂态过程验证 | 高阻故障下暂态主要由接地变压器电感与对地电容串联谐振主导，衰减时间常数分布在6.25~50ms。 | 因简化模型忽略故障点接地支路，衰减时间常数相对误差最高达34%，但谐振频率误差仍<7%，不影响故障检测有效性。 |



## 量化发现

- 暂态谐振频率变化范围为93 Hz至2323 Hz，覆盖现有配电网采样装置有效频带。
- 低阻故障高频暂态电流幅值分布在5.3 A至434.4 A之间，且在零序电压峰值投入时达到最大，过零时最小。
- 低阻故障衰减时间常数对应暂态持续时间为0.07~15 ms，高阻故障为6.25~50 ms。
- 当出线数超过3条时，低阻故障下故障线路暂态零序电流幅值必然大于健全线路，且两者极性严格相反。
- 高阻故障下，健全线路暂态零序电流仅为自身对地电容电流，故障线路电流为电容电流与故障点电阻电流的线性叠加，无明显幅值极性关系。
- 特征频段内消弧线圈感抗远大于并联阻抗（X_inL ≪ ωL_p），其电感对暂态电流分布影响可忽略，模型简化误差可控。


## 关键公式

### 故障线路入端阻抗公式

$$$Z_{\text{in}} = Z_{cx} \frac{R_f + jZ_{cx} \tan(\omega \sqrt{L_{n0}C_{n0}} l)}{Z_{cx} + jR_f \tan(\omega \sqrt{L_{n0}C_{n0}} l)}$$$

*用于分析线路阻抗随频率变化的相频特性，界定特征频段及简化模型适用范围。*

### 低阻故障暂态电流微分方程

$$$L_Z L_0 C_\Sigma \frac{d^3 i_f}{dt^3} + C_\Sigma (R_n L_0 + R L_Z) \frac{d^2 i_f}{dt^2} + (R_n R C_\Sigma + L_Z + L_0) \frac{di_f}{dt} + (R_n + R) i_f = U_m \sin(\omega_0 t + \varphi)$$$

*描述并联小电阻投入后低阻接地故障的三阶暂态过程，用于求解高频振荡与直流分量。*

### 高频振荡角频率解析式

$$$\omega_f = \frac{\sqrt{3}}{2} \frac{\sqrt[3]{Y_1 - Y_2}}{\sqrt[3]{3 L_Z L_0 C_\Sigma}}$$$

*直接关联系统参数与故障条件，用于预测暂态主频，指导滤波器设计与保护频带选择。*

### 高阻故障线路零序电流表达式

$$$i_{0nz}(t) = \frac{u_{C\Sigma z}(t)}{R_f} + C_{n0} \frac{du_{C\Sigma z}(t)}{dt}$$$

*揭示高阻故障下故障线路电流由电阻分量与电容导数分量叠加而成，区别于低阻故障的纯容性反极性特征。*



## 验证详情

- **验证方式**: 数字仿真验证与现场试验数据对比
- **测试系统**: 10 kV灵活接地配电网仿真模型（含8条出线，混合架空线与电缆线路，采用分布参数模型）
- **仿真工具**: MATLAB/Simulink
- **验证结果**: 简化模型计算的暂态谐振频率与分布参数仿真结果相对误差均在±7%以内；低阻故障衰减时间常数误差<±10%，高阻故障因忽略接地支路误差最高达34%，但整体暂态波形特征与理论解析高度吻合，验证了模型在特征频段内的有效性与工程适用性。
