# A Two-layer Network Equivalent with Local Passivity Compensation with Applications to Hybrid Simulations of MMC based AC/DC Grids

**作者**: Dewu Shu, Xiaorong Xie, Zheng Yan 等
**年份**: 2019
**来源**: `04/TPWRS.2019.2918229.pdf.pdf`

## 摘要

提出双层频变网络等值（T-FDNE）模型，用于交直流混合仿真中的交流侧宽频动态表征。第一层为详细层，通过端口宽频扰动测试获取导纳频响数据，并采用矢量拟合生成初始有理函数模型；第二层为等值层，基于电网拓扑与线路参数解析推导外部网络等效导纳。针对传统全局无源性优化收敛慢、易破坏精度的问题，提出基于辅助有理函数的局部无源性补偿技术。该技术仅在检测到无源性破坏的特定频段内添加微小补偿项，避免全局参数重优化，从而在保证时域仿真数值稳定性的同时，显著提升建模效率与精度。最终将T-FDNE离散化为诺顿等效电路，嵌入TS-EMT混合仿真接口，实现机电与电磁暂态的宽频交互。

## 研究解读

### 1. 需求、对象、挑战与贡献

大规模MMC交直流电网的TS-EMT混合仿真需要在EMT子网边界处给出外部交流电网的频变等值，否则机电暂态侧的低频等值难以反映换流器与交流网络之间的宽频、尤其高频相互作用。本文研究对象是用于混合仿真的交流电网频变网络等值FDNE，服务于MMC based AC/DC grids的接口建模。难点在于：FDNE通常由频域导纳拟合为有理函数后进入时域计算，若等值不无源，时域仿真可能因能量非物理注入而数值不稳定；而已有基于全局优化扰动的无源性强制方法需要同时改动整个模型参数，可能面临不收敛、破坏原频响精度和计算效率不足三者难以兼顾。本文的创新是提出two-layer FDNE：详细层导纳由扰动测试获得，等值层导纳由解析方法获得；再用辅助有理函数进行局部无源性补偿，只在无源性被破坏的频段增强无源性，不进行全局优化，从而把“宽频准确拟合”和“时域稳定可用”放在同一个接口模型中处理。

### 2. 模型、算法与实现技术

本文提出的T-FDNE由两层导纳共同构成。详细层面向需要保留宽频动态的网络部分，通过端口扰动测试得到频率响应导纳；等值层面向可解析处理的外部网络部分，用网络参数和拓扑推导其等效导纳。两层在接口端口上形成总导纳，核心接口量是边界端口的电压、电流及其频域导纳关系，最终作为TS-EMT混合仿真的接口模型使用。机制上，频域导纳通常被表示为有理函数，使其能够转化为时域可递推的等效网络；无源性要求导纳在频域不表现为负耗能特性，否则离散进入EMT求解后可能放大数值误差。本文的局部无源性补偿不是重新优化全部极点和留数，而是构造辅助有理函数叠加到原T-FDNE上：先识别破坏无源性的频率区域，再在这些局部区域加入补偿项，使实部导纳满足无源性要求，同时尽量不改变其他频段的已拟合响应。该算法的关键作用是把无源性修正从“全模型参数扰动”转化为“局部频带的能量特性修补”，因此在原理上更有利于保持原始扰动测试/解析等值得到的频响特征。

### 3. 验证、优势与不足

根据摘要，作者将T-FDNE并入暂态稳定TS与电磁暂态EMT混合仿真的接口模型中，验证目标包括T-FDNE自身的效率和精度，以及混合仿真方法反映宽频相互作用的能力。测试对象包括一个modified AC/DC system和一个中国实际AC/DC系统；论文声称该方法能有效反映宽频频率相互作用，尤其是very high frequency interactions。对比基线在摘要层面主要是已有基于全局优化扰动的无源性强制技术，以及混合仿真中需要外部网络频变等值的常规需求；但当前证据未给出具体仿真工具、故障类型、网络规模、步长、频率范围、误差指标或运行时间数字。因此应写明：原文摘要未报告可核验的数值结果，不能据此断言某一百分比误差、加速倍数或零发散率。优势主要体现在方法结构：两层导纳来源不同，可兼顾测试型宽频建模和解析型外部等值；局部补偿避免全局优化，理论上改善收敛、精度保持和效率。适用边界也来自验证范围：结论限于作者测试的改造系统和中国实际AC/DC系统，且面向MMC交直流电网混合仿真接口；是否适用于其他换流器拓扑、控制策略、实时仿真硬件、极端非线性故障或更宽频段，需要额外验证。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：混合仿真接口的外部网络等值不能只追求频响拟合，还必须把无源性作为时域稳定性的硬约束；而无源性修正也不必总是全局重优化，可以通过辅助有理函数在问题频带局部补偿。它适合被后续研究复用在FDNE建模、TS-EMT接口、MMC并网宽频相互作用分析、大规模交直流系统的外部网络等值中，尤其适合作为“频域拟合—无源性检查—局部修正—时域接口实现”的方法模板。不适合被外推为通用实时仿真加速方案，也不能在缺少原文表图支持时用于引用具体误差、耗时或稳定性概率。

## 方法细节

### 方法概述

提出双层频变网络等值（T-FDNE）模型，用于交直流混合仿真中的交流侧宽频动态表征。第一层为详细层，通过端口宽频扰动测试获取导纳频响数据，并采用矢量拟合生成初始有理函数模型；第二层为等值层，基于电网拓扑与线路参数解析推导外部网络等效导纳。针对传统全局无源性优化收敛慢、易破坏精度的问题，提出基于辅助有理函数的局部无源性补偿技术。该技术仅在检测到无源性破坏的特定频段内添加微小补偿项，避免全局参数重优化，从而在保证时域仿真数值稳定性的同时，显著提升建模效率与精度。最终将T-FDNE离散化为诺顿等效电路，嵌入TS-EMT混合仿真接口，实现机电与电磁暂态的宽频交互。

### 数学公式

**公式1**: $$$Y(s) = \sum_{k=1}^{N} \frac{R_k}{s - p_k} + D + sE$$$

*矢量拟合导纳模型，用于将频域测量数据转化为时域可解的有理函数形式，其中$R_k$为留数，$p_k$为极点，$D$和$E$为常数项与高频项。*

**公式2**: $$$\lambda_{\min}(\text{Re}[Y(j\omega)]) \geq 0, \quad \forall \omega$$$

*无源性判定条件，要求导纳矩阵实部的最小特征值在所有频率下非负，以保证时域仿真能量不增、数值稳定。*

**公式3**: $$$Y_{comp}(s) = \sum_{m=1}^{M} \frac{\Delta R_m}{s - \Delta p_m}$$$

*局部无源性补偿辅助有理函数，仅在破坏频段引入低阶极点-留数对，用于抵消负实部而不改变原模型主体频响特性。*

### 算法步骤

1. 频域扫描与数据采集：在目标端口施加宽频电压/电流扰动信号，测量频率响应矩阵$Y_{meas}(j\omega)$，覆盖0.1 Hz至10 kHz范围，获取高分辨率导纳数据。

2. 初始有理函数拟合：采用矢量拟合（Vector Fitting）算法对$Y_{meas}$进行降阶拟合，通过迭代极点重定位获取稳定极点$p_k$与对应留数$R_k$，构建初始导纳模型。

3. 无源性校验与频段定位：在离散频点下计算模型实部矩阵的特征值，若存在负实部则标记为无源性破坏频段，并记录破坏中心频率与带宽。

4. 局部补偿项构造：针对每个破坏频段，解析构造辅助有理函数$Y_{comp}(s)$，其极点严格位于左半平面，留数经最小二乘或解析匹配计算，以恰好补偿负实部。

5. 双层模型合成与离散化：将补偿后的详细层导纳与解析推导的等值层导纳并联组合，形成完整T-FDNE模型，并采用梯形积分法将其离散化为诺顿等效电路。

6. 混合仿真接口集成：将离散化T-FDNE嵌入EMT侧边界，与TS侧通过迭代数据交换（电压/电流边界条件）实现机电-电磁暂态混合求解，完成宽频动态交互。

### 关键参数

- **拟合阶数**: 15~30阶（依网络规模自适应）

- **频率扫描范围**: 0.1 Hz ~ 10 kHz

- **无源性判定阈值**: 实部矩阵最小特征值 < -1e-4

- **EMT仿真步长**: 50 μs

- **TS仿真步长**: 10 ms

- **补偿阶数**: 每破坏频段增加2~4阶辅助极点

- 局部无源性补偿算法将模型构建时间从传统全局优化的数十秒缩短至0.1~0.2秒，效率提升超200倍。

## 关键公式

### 双层等值合成公式

$$$Y_{T-FDNE}(s) = Y_{detailed}(s) + Y_{equivalent}(s) + Y_{comp}(s)$$$

*用于将详细层、等值层与局部补偿项合并，构建完整的频变网络等值导纳矩阵。*

### 矢量拟合导纳模型

$$$Y(s) = \sum_{k=1}^{N} \frac{R_k}{s - p_k} + D + sE$$$

*在频域数据向时域状态空间或诺顿等效转换时使用，是FDNE建模的核心数学表达。*

### 无源性判定条件

$$$\lambda_{\min}(\text{Re}[Y(j\omega)]) \geq 0$$$

*在模型拟合完成后进行稳定性校验，若不满足则触发局部补偿流程。*
