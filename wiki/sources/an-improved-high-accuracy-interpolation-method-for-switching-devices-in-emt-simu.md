## 方法细节

### 方法概述

提出一种结合半步长后向欧拉（BE）法与插值/外推策略的改进算法，用于固定步长EMT仿真中的开关器件暂态模拟。传统梯形积分法在离散时间步内处理开关动作时，易因电压不连续引发数值振荡（chatter）、电压尖峰及虚假开关损耗。该方法在检测到开关动作时刻 $t_z$ 后，首先利用常规插值获取 $t_z^-$ 时刻状态；随后采用半步长BE法计算 $t_z + \Delta t/2$ 时刻的网络解，利用BE法历史电流仅依赖连续状态变量（如电感电流）的特性，避免梯形法因电压突变导致的误差；接着用梯形法计算 $t_z + 3\Delta t/2$ 时刻解；最后通过插值或外推将状态同步回原始时间网格 $t$。该方法无需额外矩阵重三角化，计算步数与工业界瞬时插值法一致，同时彻底消除“提前一个时间步”误差与数值振荡。

### 数学公式


**公式1**: $$$V(t_z + \frac{\Delta t}{2}) = Y_{new}^{-1}[I_{source}(t_z + \frac{\Delta t}{2}) + I_{history}^{BE}(t_z^-)]$$$

*半步长后向欧拉法求解开关后 $t_z + \Delta t/2$ 时刻节点电压，利用BE历史电流源避免电压不连续误差*


**公式2**: $$$V(t_z + \frac{3\Delta t}{2}) = Y_{new}^{-1}[I_{source}(t_z + \frac{3\Delta t}{2}) + I_{history}(t_z + \frac{\Delta t}{2})]$$$

*使用梯形积分法求解 $t_z + 3\Delta t/2$ 时刻节点电压，恢复梯形法的高精度与稳定性*


**公式3**: $$$I_{history}^{BE}(t-\Delta t) = i_{km}(t-\Delta t)$$$

*BE法电感历史电流项，仅包含上一时刻电感电流，不含电压项，从根本上消除数值振荡*


**公式4**: $$$I_{history}(t-\Delta t) = i_{km}(t-\Delta t) + \frac{1}{R_L}(v_k(t-\Delta t)-v_m(t-\Delta t))$$$

*梯形法电感历史电流项，包含电压项，开关瞬间电压突变会导致历史电流计算误差及振荡*


### 算法步骤

1. 步骤1：在原始时间步 $t$ 处，使用开关前的导纳矩阵 $Y_{old}$ 和已知历史状态求解当前网络解 $V(t)$。

2. 步骤2：通过线性插值，将 $t-\Delta t$ 和 $t$ 时刻的节点电压、支路电流及历史电流项插值至实际开关动作时刻 $t_z$（即 $t_z^-$ 状态）。

3. 步骤3：切换至开关后的导纳矩阵 $Y_{new}$，采用半步长后向欧拉（BE）法计算 $t_z + \Delta t/2$ 时刻的网络解。此时历史电流源 $I_{history}^{BE}$ 仅由 $t_z^-$ 时刻的连续状态变量（如电感电流）更新，避免梯形法因电压不连续引入的误差。

4. 步骤4：保持 $Y_{new}$ 不变，采用梯形积分法计算 $t_z + 3\Delta t/2$ 时刻的网络解，利用梯形法的高精度特性推进仿真。

5. 步骤5：根据 $t_z + \Delta t/2$ 和 $t_z + 3\Delta t/2$ 时刻的计算结果，通过线性插值或外推，将所有节点电压、支路电流及历史电流项同步回原始时间网格点 $t$，完成单步仿真循环。


### 关键参数

- **仿真步长**: 50 μs（主测试），0.1 μs（高精度参考基准）

- **开关动作时刻**: t_on = 0.40015 s, t_off = 0.59985 s

- **测试硬件环境**: AMD Ryzen 7 2700X CPU @ 3.7 GHz, 32 GB RAM

- **梯形法等效电阻**: R_L = 2L / Δt

- **BE法等效电阻**: R_L = L / Δt

| 晶闸管/电感电路数值振荡（Chatter）抑制测试 | 在自然换流关断时刻，无插值、普通插值及瞬时插值均出现明显的电感电压数值振荡；本文方法利用BE法固有数值阻尼特性，在 $t_z + \Delta t/2$ 步自然消除振荡，无需额外抑制步骤。 | 彻底消除chatter现象，波形平滑度与0.1 μs基准一致，省去传统方法所需的额外半步插值处理。 |

| 计算耗时对比测试 | 在10~50 μs步长范围内进行1秒仿真，本文方法与瞬时插值法运行时间曲线几乎重合，仅比普通插值法略高。 | 计算耗时增加可忽略不计（<1%），保持与工业标准瞬时插值法相同的矩阵求解次数，无额外计算惩罚。 |



## 量化发现

- 虚假开关损耗从普通插值法的212.6 μJ大幅降低至约2 μJ，消除率超过99%。
- 彻底消除传统瞬时插值法固有的“提前一个时间步（one time-step advance）”数值误差，电感电流/电压波形与0.1 μs参考解的相位偏差趋近于0。
- 半步长BE法等效电阻 $R_L = L/\Delta t$ 与全步长梯形法 $R_L = 2L/\Delta t$ 在导纳矩阵结构上兼容，无需重新三角化矩阵，单步计算耗时增加<1%。
- 在固定50 μs步长下，开关时刻捕捉精度达到微秒级，电感电压/电流波形与0.1 μs高精度基准的相对误差<0.5%。


## 关键公式

### 半步长后向欧拉求解方程

$$$V(t_z + \frac{\Delta t}{2}) = Y_{new}^{-1}[I_{source}(t_z + \frac{\Delta t}{2}) + I_{history}^{BE}(t_z^-)]$$$

*开关动作后首个半步长计算，利用BE法历史电流仅含连续状态变量的特性，避免电压突变误差并抑制振荡*

### 半步长梯形积分求解方程

$$$V(t_z + \frac{3\Delta t}{2}) = Y_{new}^{-1}[I_{source}(t_z + \frac{3\Delta t}{2}) + I_{history}(t_z + \frac{\Delta t}{2})]$$$

*第二个半步长计算，切换回梯形法以保证连续系统仿真的数值精度与稳定性*

### BE法电感历史电流源

$$$I_{history}^{BE}(t-\Delta t) = i_{km}(t-\Delta t)$$$

*后向欧拉法伴随模型核心项，不含历史电压，从根本上切断数值振荡的传播路径*



## 验证详情

- **验证方式**: 纯数字仿真对比验证（与高精度小步长基准及多种传统插值法进行波形、损耗、耗时对比）
- **测试系统**: 简单IGBT/二极管/电感电路、晶闸管/电感电路、单相两电平逆变器（4开关）、双有源桥DAB电路（16开关）
- **仿真工具**: 自研/定制EMT仿真程序（基于节点分析法与伴随电路模型，文中提及与PSCAD/EMTDC、NETOMAC等工业程序原理对比）
- **验证结果**: 在固定50 μs步长下，成功消除开关动作引起的数值振荡与虚假损耗，精准捕捉开关时刻，多开关复杂拓扑下波形与0.1 μs基准高度一致，且未增加显著计算负担，验证了算法在工业级EMT仿真中的高精度与高效性。
