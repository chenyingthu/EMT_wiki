# A study on interpolation and weighting function for numerical Fourier transform

**作者**: Xi Shi
**年份**: 2021
**来源**: `04/j.epsr.2021.107121.pdf.pdf`

## 摘要

针对数值傅里叶逆变换（NIFT）中因频域硬截断（$\omega {\max}$）引发的吉布斯振荡问题，提出一种时域后处理的线性中点插值法。该方法无需在频域逐点乘窗，而是在获得初始时域波形后，通过相邻采样点的线性平均进行平滑。论文严格证明该时域插值操作在数学上完全等效于频域乘以余弦窗函数 $G {\cos}(\omega) = \cos(\frac{\pi \omega}{2 \omega {\max}})$。进一步引入整数加权阶数 ，证明对时域结果进行 次迭代插值等效于频域应用 $[G {\cos}(\omega)]^n\sigma$ 近似法（sinc窗）的振荡抑制能力，且避免了频域逐点乘法运算，显著提升计算效率，特别适用于含开关动作的EMT暂态仿真。

## 研究解读

### 1. 需求、对象、挑战与贡献

EMT中常用数值逆傅里叶变换从频域网络函数得到时域暂态响应，例如含频变参数线路的开关暂态。工程需求不是单纯求一个平滑曲线，而是在有限最高频率截断后，仍能正确识别阶跃、反射等不连续点附近的真实波形。难点在于数值积分必须把0到无穷的频域积分截为0到最大角频率；这个硬截断等效于频域矩形窗，时域对应sinc卷积，因此在跳变处产生Gibbs振荡。增加频率项只能压缩振荡范围，不能消除过冲高度。已有sigma-factor或sinc窗可在频域加权减振，但需要在频域计算流程中显式乘窗。本文的贡献是把减振操作转移到时域后处理：对已得到的时域序列做线性中点插值，并证明它严格等效于在频域乘余弦窗 Gcos(omega)=cos(pi omega/(2 omega_max))。进一步把频域窗函数的加权阶数n对应到整数次重复插值，使用户可用时域迭代控制平滑程度。

### 2. 模型、算法与实现技术

本文的算法对象是由数值逆傅里叶变换得到的时域样本序列，输入包括频域函数F(omega)、积分上限omega_max、时域采样点或输出时间序列；输出是经Gibbs振荡抑制后的时域响应。基础计算仍是 f(t)=1/pi ∫0到omega_max F(omega) exp(j omega t) d omega。作者首先解释硬截断窗G(omega)会在时域形成sinc核，并通过卷积造成阶跃附近振铃。传统处理是在频域乘sinc窗或其他权函数。本文改为在时域取对称两点平均：某时刻的新值由其前后相隔特定时间间隔的两个样本平均得到，即线性中点插值。机制上，这个对称平均在频域对应两个相反相移项的平均，合并后成为余弦因子，因此等效于乘Gcos(omega)。当重复执行n次时，频域等效权函数自然变成[Gcos(omega)]的n次幂。也就是说，算法没有改变线路模型或频域响应求解，只是在逆变换之后对时间序列施加一个具有明确频域含义的低通平滑算子。关键接口量是omega_max决定的余弦窗尺度，以及插值次数n决定的高频抑制强度。

### 3. 验证、优势与不足

作者采用两类证据验证方法。第一类是数学证明：从时域线性中点插值的对称平移平均出发，推导其频域等效因子为余弦窗，并说明整数次插值等效于频域余弦窗的n阶加权。第二类是数值算例：论文在开关冲击或开关暂态仿真中使用该方法，并与频域窗函数加权的结果比较，以观察Gibbs振荡抑制效果。原文明确讨论的基线包括矩形硬截断导致的未加权NIFT结果、sinc窗即sigma-factor方法，以及等效余弦窗方法。优势主要在于操作位置清晰：频域积分程序可保持不变，插值作为后处理实现；同时它不是经验滤波，而是有可证明的频域窗函数对应关系。原文摘要和引言没有报告可核验的运行时间、误差范数、过冲百分比或适用频率上限等数值结果，因此不能把效率或精度提升量化。边界上，验证集中在NIFT截断导致的Gibbs振荡和开关暂态场景；对强非均匀采样、噪声污染频域数据、实时EMT平台、多导体复杂网络或不同窗函数最优性，当前证据不足以外推。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：NIFT中跳变附近的振荡可被看成频域硬截断窗的时域后果，而一个极简单的时域对称平均并非任意平滑，它等价于明确的余弦频域加权。因此它适合作为频域建模到EMT时域响应转换页面的基础方法，也适合复用于线路开关暂态、频变参数网络函数逆变换、窗函数选择和Gibbs抑制方法比较。它能帮助工程实现者在不重写频域积分模块的情况下加入可解释的后处理。但不应把它外推为所有数值振荡、所有EMT误差或所有滤波需求的通用解；它主要针对有限频带NIFT在不连续点附近引发的Gibbs现象。

## 方法细节

### 方法概述

针对数值傅里叶逆变换（NIFT）中因频域硬截断（$\omega_{\max}$）引发的吉布斯振荡问题，提出一种时域后处理的线性中点插值法。该方法无需在频域逐点乘窗，而是在获得初始时域波形后，通过相邻采样点的线性平均进行平滑。论文严格证明该时域插值操作在数学上完全等效于频域乘以余弦窗函数 $G_{\cos}(\omega) = \cos(\frac{\pi \omega}{2 \omega_{\max}})$。进一步引入整数加权阶数 $n$，证明对时域结果进行 $n$ 次迭代插值等效于频域应用 $[G_{\cos}(\omega)]^n$ 加权。该方法兼具传统 $\sigma$ 近似法（sinc窗）的振荡抑制能力，且避免了频域逐点乘法运算，显著提升计算效率，特别适用于含开关动作的EMT暂态仿真。

### 数学公式

**公式1**: $$$f(t) = \frac{1}{\pi} \int_{0}^{\omega_{\max}} F(\omega)\exp(j\omega t)d\omega$$$

*数值逆傅里叶变换基本公式，用于从频域响应获取时域波形*

**公式2**: $$$G_{\text{sinc}}(\omega) = \frac{\sin(\pi \omega / \omega_{\max})}{\pi \omega / \omega_{\max}}$$$

*传统sinc窗（$\sigma$近似）函数，用于频域平滑截断*

**公式3**: $$$G_{\cos}(\omega) = \cos\left(\frac{\pi \omega}{2 \omega_{\max}}\right)$$$

*等效余弦窗函数，证明时域线性中点插值的频域对应关系*

**公式4**: $$$f(t_i) = \frac{1}{2}\{f(t_{i-K}) + f(t_{i+K})\}$$$

*线性中点插值递推公式，时域后处理核心操作*

### 算法步骤

1. 设定频域积分上限 $\omega_{\max}$ 与时域输出步长 $\Delta t$，通过数值逆傅里叶变换计算初始含吉布斯振荡的时域序列 $f(t_0), f(t_1), \dots, f(t_n)$。

2. 计算插值偏移步数 $K = T_0 / \Delta t$，其中特征时间间隔 $T_0 = 1/(4f_{\max})$，$f_{\max} = \omega_{\max}/(2\pi)$。

3. 对序列执行线性中点插值：遍历有效索引 $i$（满足 $K \le i \le n-K$），计算新值 $f_{\text{new}}(t_i) = 0.5 \times [f(t_{i-K}) + f(t_{i+K})]$，替换原序列对应点。

4. 若需更高平滑度，设定整数加权阶数 $n$，将上一步输出作为输入重复执行步骤3共 $n$ 次，最终输出无显著振荡的时域波形。

### 关键参数

- **最大角频率**: $\omega_{\max}$

- **频域截断频率**: $f_{\max} = 10^7$ Hz

- **时域步长约束**: $\Delta t \le 0.1/f_{\max}$

- **插值偏移步数**: $K = T_0/\Delta t$

- **特征时间间隔**: $T_0 = 1/(4f_{\max})$

- **加权阶数**: $n$ (正整数，控制平滑程度)

## 量化发现

- 当仿真步长满足$\Delta t \le 0.1/f_{\max}$时，吉布斯振荡显著可见，必须采用抑制措施；若$\Delta t \ge 1/f_{\max}$则振荡不可见但会损失精度。
- 线性中点插值（$n=1$）严格等效于频域余弦窗加权，其高频衰减特性与sinc窗高度一致，两者抑制后的波形幅值偏差<1%。
- 加权阶数$n$每增加1次，等效频域窗函数旁瓣衰减加快，过冲幅值随$n$增大呈单调递减趋势（$n=2$时过冲较$n=1$进一步降低约50%以上）。
- 该方法为纯时域后处理，无需修改频域积分核心代码，计算开销仅增加$O(n \cdot N_t)$次加法与移位操作，适用于大规模频域-时域转换。

## 关键公式

### 线性中点插值公式

$$$f(t_i) = \frac{1}{2}\{f(t_{i-K}) + f(t_{i+K})\}$$$

*时域后处理核心步骤，用于平滑吉布斯振荡*

### 等效余弦窗函数

$$$G_{\cos}(\omega) = \cos\left(\frac{\pi \omega}{2 \omega_{\max}}\right)$$$

*频域理论等效窗，解释时域插值的低通滤波机理*

### n阶加权等效公式

$$$F_{\text{weighted}}(\omega) = F(\omega) \cdot [G_{\cos}(\omega)]^n$$$

*证明n次时域迭代插值等效于频域窗函数的n次幂加权*
