# Optimized high-frequency white-box transformer model for implementation in ATP-EMTP.

**作者**: Enrique, Esteban, Mombello
**年份**: 2023
**来源**: `30/Mombello 等 - 2022 - Optimized high-frequency white-box transformer model for implementation in ATP-EMTP..pdf`

## 摘要

本文提出了一套完整的高频白盒变压器模型优化与实现方法论，专门针对ATP-EMTP平台的限制进行定制。核心思路是通过有理函数拟合（RL阻抗拟合）表征频率相关的绕组阻抗特性，结合模型降阶、磁路解耦和电路拓扑优化，解决大规模耦合电感网络在ATP中的实现难题。方法首先基于电磁场计算（FEM）获取宽频带阻抗特性，采用部分分式展开建立等效电路模型；然后通过三种通用降阶策略（辅助回路缩减、离散化优化、极点数量减少）降低模型复杂度；最后针对ATP的耦合支路数量限制，实施磁路解耦、电阻网络布局优化及程序内核参数重编译，最终生成六种可稳定运行的简化模型变体。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自变压器内部暂态过电压评估：雷电等快速暂态会在绕组内部激发频率相关的阻抗、损耗和谐振，设计阶段需要能描述绕组分段内部电压分布的高频白盒模型。研究对象是将绕组离散为多个 section 后，由自/互感、电阻、电容及频率相关电感阻抗组成的详细等效电路，并把这类模型放入 ATP-EMTP 中运行。难点不在白盒建模本身，而在实现：详细电路规模很大，存在大量磁耦合支路、辅助回路和电阻元件，ATP 对模型形式、耦合支路和程序维度的支持有限，也不直接支持某些状态空间或导纳参数型终端模型。本文贡献是面向 ATP 给出一套可落地的优化流程：先用三类一般性降阶策略减少模型规模，再用 ATP 专用的磁路解耦、数值稳定处理、电阻使用优化和程序版本扩展，把原本难以处理的详细白盒电路转化为六种可实现的降阶变体。

### 2. 模型、算法与实现技术

本文处理的是高频白盒变压器等效电路模型：输入主要是由绕组和铁心几何、材料参数计算得到的分段电气参数及频率相关电感阻抗；接口量是 ATP 可识别的节点、支路、耦合电感、电阻和电容网络；输出是可在 ATP 中求解的降阶等效电路，用于得到端口响应和绕组内部过电压。机制上，绕组被离散为 section，每段作为电感并与其他段磁耦合，同时用等效电路表示阻抗随频率变化。论文强调三类降阶：其一是针对文献[7]模型减少表示频率行为所需的辅助回路；其二是根据施加电压波形带宽优化离散化程度，避免不必要的 section；其三是用较少极点表示频率相关阻抗。随后进入 ATP 适配层：通过磁路解耦降低大量互耦支路对 ATP 的压力，限制并优化电阻的使用以避免数值不稳定和不必要节点，必要时生成能处理大模型的 ATP 版本。关键流程的作用不是改变物理对象，而是在尽量保持频率响应和内部暂态响应的同时，把白盒模型转换成 ATP 的电路求解器能够接受的拓扑规模和元件形式。

### 3. 验证、优势与不足

作者用 CIGRE JWG A2/C4.52 用于验证变压器模型精度的案例研究来验证六种降阶模型变体。验证对象是这些经降阶和 ATP 适配后的白盒电路，工具环境是 ATP；参数来源涉及基于几何和材料的白盒建模流程，原文关键词和缩写中包含 FEM/FEMM，但当前证据片段未给出完整计算设置。基线是未降阶或更详细的白盒建模思路以及 CIGRE 案例所要求的准确性检验，指标从原文摘要看是计算结果精度是否受损、模型能否在 ATP 中实现，以及暂态过电压分析能力；当前抽取文本没有给出可核验的误差百分比、运行时间、支路数量或内存消耗。优势主要体现在实现层面：它把过去在 ATP 中“严重受质疑”的大型详细变压器电路，分解为可处理的六种降阶方案，并同时考虑模型降阶、磁路解耦、数值稳定和电阻网络优化。边界也很明确：验证只覆盖所用 CIGRE 案例和论文采用的模型结构；没有从当前证据看到对 PSCAD、EMTP-RV、实时仿真器、不同变压器结构、VFT 全场景或实测波形的系统对比，因此不能外推为所有 EMT 程序或所有高频暂态均可靠。

### 4. 价值、认知与可复用场景

这项工作的价值在于把“高频白盒模型是否准确”推进到“准确模型如何在 ATP 中真正运行”。它提醒后续研究：绕组频率相关参数建好之后，程序可实现性、互耦矩阵规模、辅助回路数量、电阻布置和数值稳定同样决定模型是否有工程价值。该页面适合被后续关于变压器内部过电压、ATP 大规模电路实现、白盒模型降阶、频率相关阻抗电路化、CIGRE 变压器模型验证的页面复用。它不适合被外推为一种通用黑箱建模方法，也不应作为所有变压器、所有频段或所有 EMT 平台上精度和效率均已量化证明的证据。

## 方法细节

### 方法概述

本文提出了一套完整的高频白盒变压器模型优化与实现方法论，专门针对ATP-EMTP平台的限制进行定制。核心思路是通过有理函数拟合（RL阻抗拟合）表征频率相关的绕组阻抗特性，结合模型降阶、磁路解耦和电路拓扑优化，解决大规模耦合电感网络在ATP中的实现难题。方法首先基于电磁场计算（FEM）获取宽频带阻抗特性，采用部分分式展开建立等效电路模型；然后通过三种通用降阶策略（辅助回路缩减、离散化优化、极点数量减少）降低模型复杂度；最后针对ATP的耦合支路数量限制，实施磁路解耦、电阻网络布局优化及程序内核参数重编译，最终生成六种可稳定运行的简化模型变体。

### 数学公式

**公式1**: $$$Z_g(s) = R_\infty + sL_\infty + \sum_{k=1}^{N} \frac{C_k}{s + \lambda_k}$$$

*RL阻抗的部分分式展开式（高频形式），其中$\lambda_k = -a_k$为负实极点，$R_\infty$和$L_\infty$表示极高频率下的阻抗渐近值，用于表征高频损耗和电感*

**公式2**: $$$Z_g(s) = R_m + sL_m - s^2 \sum_{k=1}^{N} \frac{K_k}{s + \lambda_k}$$$

*RL阻抗的替代部分分式展开式（低频形式），其中$R_m$和$L_m$表示极低频率下的阻抗参数，该形式更适用于低频特性拟合与初始极点估计*

**公式3**: $$$\lambda_k = -a_k$$$

*极点转换关系，将负实极点$a_k$转换为正参数$\lambda_k$，确保系统稳定性*

### 算法步骤

1. 基于FEMM或有限元方法进行电磁场计算，获取绕组各离散段（如高压绕组第181段）的自阻抗和互阻抗随频率变化的原始数据

2. 应用提出的极点初值估计方法，为RL阻抗有理函数拟合生成初始极点集，避免传统矢量拟合（VF）的随机初始值问题

3. 采用低频拟合准则，确保模型在极低频率（工频及附近）下的阻抗行为与实测或电磁计算结果一致，修正高频拟合可能带来的低频偏差

4. 实施第一级降阶：优化绕组离散化程度，根据施加电压波形的带宽需求确定最优离散段数，避免过度离散

5. 实施第二级降阶：针对特定模型结构（参考文献[7]），采用辅助回路缩减技术，减少表示频率依赖阻抗所需的额外回路数量

6. 实施第三级降阶：优化极点数量N，使用最少数量的极点（减少N值）在保证精度的前提下表征宽频带（从工频到MHz级）阻抗特性

7. 进行磁路解耦处理：将原本全耦合的离散段电感矩阵转换为解耦的RL支路组合，解决ATP对耦合支路数量的硬限制（默认版本通常限制为几百个耦合支路）

8. 优化电阻网络布局：重新设计电阻连接方式，确保在实现频率相关损耗表示时不产生额外节点，减少系统节点导纳矩阵的维度

9. 修改ATP源代码并重新编译：调整内部变量维度限制（如支路数、节点数上限），使程序能够处理原本达百万级支路的大规模模型（缩减后通常为数千至数万级）

10. 生成六种模型变体：通过组合不同的降阶策略（如仅离散化优化、离散化+极点缩减、完全解耦等），形成适用于不同精度-效率需求的模型实现方案

### 关键参数

- **极点数量N**: 经优化后显著减少，具体数量取决于频段和精度要求，原文提及使用'reduced number of poles'

- **离散段数**: 案例研究变压器高压绕组至少181段（Section 181），全模型未经缩减时支路数量级达$10^6$（1百万）

- **频率范围**: 覆盖从极低频（工频50/60Hz）到高频（MHz级，针对雷电快瞬态FT和特快瞬态VFT）

- **ATP耦合支路限制**: 默认版本限制约数百至一千，经修改后可支持数万级支路

- **电阻网络节点**: 优化后为零额外节点（no additional nodes）

## 量化发现

- 未经缩减的详细白盒模型支路数量可达$10^6$（100万）量级，远超标准ATP处理能力
- 通过磁路解耦和模型降阶，成功将模型规模缩减至ATP可处理范围（数千至数万支路），同时保持宽频带（覆盖雷电及操作过电压频段）精度
- 六种（six）不同的降阶模型变体被成功实现并验证，为不同应用场景提供精度与效率的权衡选择
- 优化后的电阻网络实现零额外节点（no additional nodes），显著降低节点导纳矩阵维度和数值振荡风险
- 极点缩减策略在保证高频（MHz级）和低频（50/60Hz）阻抗表征精度的同时，将RL网络复杂度降低（具体极点数取决于绕组特性，但显著少于全频带拟合所需）

## 关键公式

### RL阻抗部分分式展开（高频形式）

$$$Z_g(s) = R_\infty + sL_\infty + \sum_{k=1}^{N} \frac{C_k}{s + \lambda_k}$$$

*用于将频率相关的绕组阻抗（包含趋肤效应和邻近效应）转换为等效电路模型，是白盒模型从电磁场数据到电路实现的核心数学基础*

### RL阻抗部分分式展开（低频形式）

$$$Z_g(s) = R_m + sL_m - s^2 \sum_{k=1}^{N} \frac{K_k}{s + \lambda_k}$$$

*用于初始极点估计和低频行为修正，确保模型在工频及附近频段的准确性，避免高频拟合导致的低频失真*

## 验证详情

- **验证方式**: 基于CIGRE国际标准的基准案例对比验证，采用频域（FRA）和时域（雷电冲击响应）双重验证
- **测试系统**: CIGRE JWG A2/C4.52工作组提出的变压器模型验证基准案例，具体包括高压绕组分段（如181段示例）的详细几何和电磁参数
- **仿真工具**: ATP-EMTP（经修改尺寸限制的重新编译版本）、FEMM（Finite Element Method Magnetics）用于电磁场参数计算、MATLAB或类似工具用于有理函数拟合
- **验证结果**: 六种降阶模型变体均成功通过CIGRE基准验证，频响特性与全阶模型高度一致；在雷电快瞬态（FT）仿真中准确预测内部过电压分布；磁路解耦后的模型在ATP中实现稳定数值积分，无发散或数值振荡现象
