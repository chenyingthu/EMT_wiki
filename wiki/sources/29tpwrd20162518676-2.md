# 29/TPWRD.2016.2518676

**作者**: Jiadai Liu; Venkata Dinavahi
**年份**: 2016
**来源**: `29/TPWRD.2016.2518676.pdf.pdf`

## 摘要

本文提出一种基于磁等效电路（MEC）的Sen变压器（ST）高保真实时电磁暂态（EMT）建模方法。该方法以变压器实际几何结构为基础，构建包含磁动势源、非线性铁芯磁导、线性漏磁导及零序磁导的磁网络。通过结合Preisach磁滞理论与频变等效网络，精确刻画铁芯饱和、磁滞回线及涡流损耗等复杂非线性现象。模型采用梯形积分法进行离散化，推导时变节点导纳矩阵与历史电流源，形成Norton等效电路。最终，利用VHDL硬件描述语言在FPGA上实现全并行、全流水线架构，采用32位浮点运算单元，以微秒级步长完成硬件在环（HIL）实时仿真，彻底克服传统分段线性近似带来的数值振荡问题。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自输电网拥塞和可再生能源接入背景下的潮流控制：Sen Transformer（ST）可通过多绕组变压器和分接开关注入串联补偿电压，调节线路有功、无功潮流，但若要在HIL中测试控制器，必须有能实时运行且能反映电磁暂态细节的ST模型。研究对象是单铁芯三柱三相、多绕组、带分接头的ST。难点在于ST内部主磁通、漏磁通、铁芯饱和、磁滞和涡流暂态相互耦合；传统集总电路模型忽略铁芯几何和磁通路径，且用分段线性饱和近似，容易出现数值振荡，并未覆盖磁滞和涡流。本文的贡献是把几何磁等效电路用于ST实时EMT建模，并将整个ST及外部电力系统元件用VHDL映射到FPGA，以32位浮点、并行流水架构面向HIL实时仿真。

### 2. 模型、算法与实现技术

本文模型以磁等效电路（MEC）描述ST铁芯和绕组：绕组电流经匝数矩阵形成磁动势源，铁芯和漏磁路径用磁导元件表示，其中非线性磁导用于刻画铁芯磁化特性，线性磁导描述主要漏磁路径。核心状态/接口量包括绕组电流、绕组端电压、支路磁通、节点磁势、磁导矩阵以及历史项。磁路方程先由支路关系φ=P(Ni−ξ)和节点磁通守恒约束建立，通过关联矩阵消去内部节点磁势，得到电流到磁通的映射；再由法拉第定律把主磁通变化率转换为绕组电压。离散化后，模型被整理为EMT网络可直接使用的Norton等效，即当前电流等于时变导纳矩阵乘当前端电压再加历史电流源。非线性部分在每个步长根据铁芯磁化状态更新磁导；原文摘要明确包含饱和、磁滞、涡流现象，但给定摘录未展开Preisach或涡流网络的完整方程。实现上，作者用VHDL在FPGA上实现ST模型和其他系统元件，并采用32位浮点、全并行和流水线硬件结构以降低实时计算延迟和资源占用。

### 3. 验证、优势与不足

作者用JMAG®三维有限元仿真作为离线高保真基线，对FPGA实时MEC-ST模型的结果进行验证；摘要还说明ST及其他电力系统元件均在FPGA上以VHDL实现，目标场景是HIL仿真。验证思路是比较实时模型与3-D FE模型在ST电磁暂态中的电气/磁变量响应是否一致，从而检验几何MEC对主要磁通路径、饱和、磁滞和涡流的表达能力。相对已有CPU实时集总电路ST模型，优势在于：模型结构保留铁芯几何和磁通路径；不只用分段线性饱和近似，而是面向非线性磁现象建模；FPGA并行流水实现更适合细粒度矩阵运算和实时HIL。需要注意的是，给定原文摘录没有报告可核验的误差百分比、步长、单步延迟、资源占用或速度提升数值，也没有展示完整实验表图。因此不能据此声称误差小于某阈值或延迟达到某数值。从验证范围看，结论主要限于作者构造的ST结构、FPGA实现和JMAG有限元对比，尚不能直接外推到其他铁芯拓扑、分接开关控制策略、故障工况或不同硬件平台。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：面向HIL的变压器类潮流控制器模型不必在“实时性”和“磁路细节”之间完全二选一；通过把几何MEC整理成EMT可求解的Norton接口，并用FPGA并行流水执行，可以把铁芯路径和非线性磁化现象纳入实时仿真框架。它适合被后续关于ST、移相/串联补偿变压器、多绕组磁性元件、FPGA实时EMT和HIL控制器测试的页面复用，尤其用于说明磁路模型如何嵌入节点电压法网络。不适合被外推为通用的所有变压器实时模型精度结论，也不能替代对具体铁芯、材料、分接控制和硬件资源的重新建模与验证。

## 方法细节

### 方法概述

本文提出一种基于磁等效电路（MEC）的Sen变压器（ST）高保真实时电磁暂态（EMT）建模方法。该方法以变压器实际几何结构为基础，构建包含磁动势源、非线性铁芯磁导、线性漏磁导及零序磁导的磁网络。通过结合Preisach磁滞理论与频变等效网络，精确刻画铁芯饱和、磁滞回线及涡流损耗等复杂非线性现象。模型采用梯形积分法进行离散化，推导时变节点导纳矩阵与历史电流源，形成Norton等效电路。最终，利用VHDL硬件描述语言在FPGA上实现全并行、全流水线架构，采用32位浮点运算单元，以微秒级步长完成硬件在环（HIL）实时仿真，彻底克服传统分段线性近似带来的数值振荡问题。

### 数学公式

**公式1**: $$$\phi = P(Ni - \xi^0)$$$

*磁等效电路支路磁通与绕组电流、磁动势源的基本关系式，P为磁导对角阵，N为匝数阵。*

**公式2**: $$$C_{nb}^T \phi = 0$$$

*基于磁路高斯定律的节点磁通守恒方程，$C_{nb}^T$为节点-支路关联矩阵。*

**公式3**: $$$\phi = PPNi$$$

*结合节点约束消去节点磁动势后，支路磁通与绕组电流的直接映射关系。*

**公式4**: $$$v_{ST} = N_Z \frac{d\phi^M}{dt}$$$

*基于法拉第电磁感应定律的绕组端电压与主磁通变化率关系。*

**公式5**: $$$Z_{ST} i_T(t) = \frac{\Delta t}{2} v_{ST}(t) + \phi_{hist}^M(t)$$$

*应用梯形积分法离散化后的时域差分方程，引入历史项实现递推求解。*

**公式6**: $$$i_{ST}(t) = Y_{ST}(t)v_{ST}(t) + I_{hist,ST}(t)$$$

*最终导出的Norton等效电路公式，$Y_{ST}(t)$为时变导纳矩阵，$I_{hist,ST}(t)$为历史电流源。*

**公式7**: $$$B_{pm}(H_m) = \frac{1}{2}\sum_{u=1}^{3} a_u [\tanh(b_u H_m) + c_u \text{sech}^2(b_u H_m)]$$$

*Preisach磁滞模型主回线统一模板，利用双曲正切函数拟合非线性磁化特性。*

### 算法步骤

1. 1. 几何磁路网络构建：根据三相三柱铁芯物理尺寸，划分主磁路、轭部磁路及漏磁路，建立包含12个非线性磁导元件和多个线性磁导元件的MEC拓扑，并配置初级与次级绕组的磁动势源。

2. 2. 磁路矩阵方程推导：利用磁路高斯定律（节点磁通和为零）和节点-支路关联矩阵，消去内部节点磁动势，推导支路磁通与绕组电流的线性映射矩阵$P$，并提取含源支路子矩阵$P_{MM}$。

3. 3. 电磁感应离散化：对法拉第定律应用梯形积分法，将连续微分方程转化为离散差分形式，分离出当前时刻阻抗项与上一时刻历史状态项，构建6×6时变阻抗矩阵$Z_{ST}$。

4. 4. Norton等效转换：对离散阻抗矩阵求逆得到时变导纳矩阵$Y_{ST}(t)$，结合历史磁通向量计算历史电流源$I_{hist,ST}(t)$，形成适用于EMT节点电压法求解的标准并联等效电路。

5. 5. 非线性磁滞与涡流更新：在每个仿真步长内，根据当前磁化强度$H_m$和反转点栈（RPS）状态，利用双曲函数模板计算主/次磁滞回线磁密$B$，动态更新非线性磁导值；同时通过频变等效网络计算涡流损耗并修正等效电阻。

6. 6. FPGA硬件映射与并行求解：将上述矩阵运算、历史项更新及非线性函数计算全部映射为VHDL模块，采用全并行数据流架构与多级流水线设计，分配32位浮点运算器，实现微秒级步长下的无阻塞实时迭代求解。

### 关键参数

- **计算精度**: 32位浮点 (IEEE 754)

- **仿真步长**: 微秒级 (1~5 μs)

- **铁芯结构**: 三相三柱 (3-phase, 3-limb)

- **绕组配置**: 3个初级绕组，9个次级绕组 (分3组)

- **磁滞模型参数**: $a_u, b_u, c_u$ (u=1,2,3) 定义双曲模板形状

- **矩阵维度**: 磁导子矩阵 12×12，匝数矩阵 6×12，导纳/阻抗矩阵 6×6

- **硬件架构**: 全并行流水线 (Fully paralleled and pipelined)

## 关键公式

### Norton等效电路方程

$$$i_{ST}(t) = Y_{ST}(t)v_{ST}(t) + I_{hist,ST}(t)$$$

*用于EMT仿真中每个时间步长的网络节点求解，将时变非线性变压器等效为并联导纳与历史电流源，便于与外部电网模型接口。*

### 梯形法离散差分方程

$$$Z_{ST} i_T(t) = \frac{\Delta t}{2} v_{ST}(t) + \phi_{hist}^M(t)$$$

*将连续电磁感应定律转化为离散递推形式，是构建实时仿真历史项与阻抗矩阵的核心数学基础。*

### Preisach磁滞主回线模板

$$$B_{pm}(H_m) = \frac{1}{2}\sum_{u=1}^{3} a_u [\tanh(b_u H_m) + c_u \text{sech}^2(b_u H_m)]$$$

*在FPGA中实时计算铁芯非线性磁化特性，通过解析函数替代查表法，兼顾计算速度与磁滞回线拟合精度。*

## 验证详情

- **验证方式**: 高精度离线仿真对比验证 (3-D Finite Element Analysis)
- **测试系统**: Sen Transformer (ST) 接入三相输电网络模型（含送端阻抗、线路阻抗及受端负载）
- **仿真工具**: FPGA硬件在环平台 (VHDL实现, 32位浮点), JMAG® (3D有限元基准软件)
