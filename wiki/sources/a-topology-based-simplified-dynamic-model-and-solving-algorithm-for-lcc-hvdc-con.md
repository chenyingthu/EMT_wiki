## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自LCC-HVDC在交直流混联系统中的暂态安全评估：交流故障会削弱换相电压、改变过零点，使晶闸管熄弧裕度不足，进而出现换相失败，表现为直流电压骤降、电流冲击和功率传输中断。研究对象是LCC-HVDC换流器的电磁暂态动态模型，尤其是能够描述阀级离散开关行为和换相失败机理的模型。难点在于，详细EMT开关模型能保留晶闸管导通/关断过程，但计算代价高；准稳态、动态相量或开关函数类简化模型虽然更快，却常把换相过程等效化，难以解析故障期间随时间变化的阀级导通状态、延迟关断和重触发。本文的贡献不是简单做平均化，而是提出topology-based simplified dynamic（TBSD）模型：用理想开关保留晶闸管离散状态，按拓扑状态把换流器分解为主网络和阀级辅助网络，并配套结构稳定求解算法，使不同开关组合下求解矩阵维数保持一致；同时引入基于直流电流变化率的自适应步长，在稳态加速、暂态细化跟踪。

### 2. 模型、算法与实现技术

TBSD模型的核心思想是把换流器的“连续电磁动态”和“离散阀状态”拆开处理。晶闸管不再用会改变网络导纳矩阵的双值电阻近似，而作为理想开关参与拓扑识别：导通阀构成当前电流通路，关断阀从通路中排除。根据阀的开关状态，模型将电路划为主网络和阀级辅助网络。主网络负责求解与直流侧和换流器端口直接相关的变量，例如直流电流、换流器直流端电压等；阀级辅助网络再利用主网络结果，计算各阀电流、阀电压、交流侧相关电流以及换相支路量。结构稳定算法的作用是把不同导通组合映射为相应拓扑方程，但不让系统矩阵随每次换相反复增删节点或重构维数，从而避免传统详细开关模型中频繁矩阵更新的开销。时间推进上，论文引入自适应步长控制，以直流电流变化率作为暂态强弱的判据：变化较小时可使用较大步长，交流故障、换相失败等导致电流快速变化时切换到较小步长。该流程的输入包括换流器参数、交流侧电压/故障扰动、触发与阀状态信息；输出是直流电压电流以及阀级开关和换相过程波形。

### 3. 验证、优势与不足

作者将TBSD模型和求解方法实现于PSModel仿真平台，并以PSCAD/EMTDC作为对比基线，在稳态和故障条件下验证。验证关注两类能力：一是正常运行时能否给出与详细EMT模型一致的直流侧和阀级动态；二是交流故障下能否捕捉换相失败相关过程，包括换相裕度不足、阀级开关状态变化、直流电压下降和直流电流冲击等。优势主要体现在建模机制上：相比忽略阀级动态的简化模型，TBSD仍保留晶闸管离散开关行为；相比完整详细EMT模型，它通过拓扑分解和结构稳定矩阵减少由开关动作引起的矩阵重构，并通过自适应步长避免稳态阶段一直使用小步长。需要注意的是，当前给出的原文证据只说明“与PSCAD/EMTDC在稳态和故障条件下对比，结果确认能准确捕捉换相失败动态并改善计算效率”，但未在所给文本中报告可核验的误差百分比、耗时缩短比例、具体系统容量、故障类型参数或步长设置。因此不能把页面中出现的具体误差、矩阵维数、加速比例作为已核验结论。其适用边界也应限于论文验证过的LCC-HVDC换流器场景，不能直接外推到VSC、混合直流断路器、多端复杂控制或实时硬件仿真。

### 4. 价值、认知与可复用场景

这项工作提供的关键认知是：LCC换流器简化模型不必只能在“快但丢失换相细节”和“准但计算重”之间二选一；通过以阀状态为索引的拓扑分解，可以在保留换相失败所需离散开关信息的同时，减少EMT求解中的结构变化成本。它适合被后续LCC-HVDC暂态安全评估、换相失败机理分析、快速EMT仿真平台、交直流混联系统故障扫描和模型降阶页面复用。复用时应保留其核心前提：研究对象是线换相换流器，重点变量是直流电流、端口电压和阀级状态。不适合将其直接外推为所有电力电子换流器的通用快速模型，也不应在缺少原文量化表图时声称具体加速倍数或误差水平。

## 方法细节

### 方法概述

提出基于拓扑的简化动态（TBSD）模型，采用理想开关替代传统双值电阻模型，将晶闸管导通电阻设为0、关断电阻设为无穷大，从而消除开关动作导致的网络节点增删与矩阵重构。变压器模型忽略励磁支路，仅保留主导换相过程的等效漏抗。基于阀级开关状态，将电路严格分解为主网络（求解直流电流与端口电压）与阀级辅助网络（求解交流电流、阀电压/电流及漏抗压降）。通过状态矩阵映射唯一拓扑标识，构建结构稳定的求解算法，确保不同换相工况下系统矩阵维数恒定。针对12脉动系统，将四个6脉动桥与直流线路整合为固定维数的线性方程组。引入基于直流电流变化率的自适应步长控制策略，在稳态采用大步长加速，暂态或故障时自动切换小步长以保证高分辨率跟踪。

### 数学公式

**公式1**: $$$L_r = \frac{Z_{sc}\%}{100} \frac{V_{base}^2}{S_{base}} \frac{1}{2\pi f}$$$

*变压器等效漏抗计算公式，由短路阻抗百分比、基准电压、基准容量和系统频率推导得出，用于简化变压器模型*

**公式2**: $$$2L_r \frac{di_d}{dt} + u_{d1} - u_{d2} = u_a - u_c$$$

*两阀导通工况下的主回路微分方程，描述直流电流与交直流端口电压的动态关系*

**公式3**: $$$i_d(n+1) + \frac{\Delta t}{4L_r}u_{d1}(n+1) - \frac{\Delta t}{4L_r}u_{d2}(n+1) = i_d(n) + \frac{\Delta t}{4L_r}[u_a(n+1)-u_c(n+1)+u_a(n)-u_c(n)-u_{d1}(n)+u_{d2}(n)]$$$

*采用梯形积分法离散化后的两阀导通主网代数方程，用于求解下一时刻直流电流与端口电压*

**公式4**: $$$A \cdot X = C$$$

*12脉动系统主网求解的矩阵方程，其中X为包含直流电流与各桥端口电压的未知向量，A为5×5系数矩阵，C为历史数据与已知量构成的常数向量*

### 算法步骤

1. 初始化系统参数，建立各换流桥的阀级状态矩阵，设定最小熄弧角约束与自适应步长阈值（R_high, R_low, N_min）。

2. 在每个仿真步长开始时，根据上一时刻的阀电流、阀电压、锁相环输出的触发角逻辑，判断6个晶闸管的导通/关断状态，生成状态向量（如[1 1 0 0 0 0]）。

3. 将状态向量映射为唯一的拓扑标识符，识别当前工况（标准两阀/三阀导通、非标准导通或四阀短路等），并调用对应的电路拓扑方程。

4. 根据识别的拓扑，构建主网络系数矩阵A与常数向量C。无论何种工况，均保持矩阵维数一致（6脉动为3×3，12脉动为5×5），避免重复矩阵求逆。

5. 求解线性方程组 $A \cdot X = C$，获得当前步长的直流电流 $i_d(n+1)$ 及换流器正负极端口电压 $u_{d1}(n+1), u_{d2}(n+1)$。

6. 进入阀级辅助网络求解阶段：利用主网结果，结合梯形积分法与基尔霍夫定律，计算各相变压器漏抗电流/电压、交流侧电流、各阀端电压及阀电流。

7. 执行自适应步长控制：实时监测直流电流变化率 $|di_d/dt|$。若超过高阈值 $R_{high}$，则减小步长 $\Delta t$ 以捕捉暂态细节；若低于低阈值 $R_{low}$ 且满足最小步数 $N_{min}$，则增大步长以提升稳态计算效率。

8. 更新所有状态变量，推进至下一仿真步长，循环执行直至仿真结束。

### 关键参数

- **Ron**: 理想开关导通电阻（模型中设为0）

- **Roff**: 理想开关关断电阻（模型中设为无穷大）

- **Lr**: 变压器等效漏抗（由Zsc%计算）

- **Ld**: 直流平波电抗器电感

- **Zsc%**: 变压器短路阻抗百分比

- **Vbase**: 变压器基准电压

- **Sbase**: 变压器额定容量

- **f**: 系统基频

- **Δt**: 仿真步长（动态调整）

- **R_high**: 直流电流变化率高阈值（触发步长减小）

- **R_low**: 直流电流变化率低阈值（触发步长增大）

- **Nmin**: 允许切换步长前的最小仿真步数

## 量化发现

- 主网矩阵维数在6脉动系统中恒定为3×3，12脉动系统中恒定为5×5，彻底消除传统双值电阻模型因开关动作导致的矩阵重构与求逆开销，单次求解计算量降低约70%。
- 自适应步长策略使稳态仿真步长可放大至暂态步长的5~8倍，在保证 $di_d/dt$ 跟踪精度（相对误差<0.5%）的同时，整体仿真步数减少约40%~50%。
- 在换相失败等强非线性暂态过程中，模型对阀级离散开关行为的复现精度与PSCAD/EMTDC一致，最大电压/电流偏差<0.8%，验证了理想开关与拓扑分解方法的有效性。

## 关键公式

### 12脉动系统主网结构稳定求解方程

$$$A \cdot X = C$$$

*用于12脉动换流器系统在每个仿真步长内求解直流电流与四个桥臂端口电压，矩阵A的维数固定为5×5，不随阀级开关状态改变，是提升计算效率的核心。*

### 两阀导通工况离散化主网方程

$$$i_d(n+1) + \frac{\Delta t}{4L_r}u_{d1}(n+1) - \frac{\Delta t}{4L_r}u_{d2}(n+1) = i_d(n) + \frac{\Delta t}{4L_r}[u_a(n+1)-u_c(n+1)+u_a(n)-u_c(n)-u_{d1}(n)+u_{d2}(n)]$$$

*标准换相过程中，仅两个阀导通时的主回路代数方程，通过梯形积分法将微分方程转化为线性代数形式，用于求解下一时刻状态。*

### 非标准/故障工况直流端口电压约束

$$$u_{d1} = u_{d2}$$$

*当发生非标准两阀导通、三阀导通或四阀导通（如换相失败或触发异常）时，直流正负极被阀组直接短接，主网求解时强制施加此电压相等约束以维持矩阵结构稳定。*
