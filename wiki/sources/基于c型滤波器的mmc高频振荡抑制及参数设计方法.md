## 方法细节

### 方法概述

本文提出在模块化多电平换流器(MMC)交流母线处并联C型滤波器的无源抑制方法，通过阻抗重塑完全消除MMC的高频负阻尼特性。该方法利用C型滤波器的高通特性，在高频段为系统提供正阻尼。关键创新在于将C型滤波器的串联谐振支路(CZ2和LZ)设计在基频(50Hz)处谐振，从而阻断基频电流流入电阻支路，显著降低滤波器的基频有功损耗。参数设计基于MMC的第1负阻尼频段([1/(2T), 1/T])进行，通过分析MMC高频简化阻抗模型与C型滤波器的并联阻抗特性，确保在负阻尼频段内重塑后的阻抗相位进入稳定区域。设计过程综合考虑了元件成本，通过合理选择CZ1、RZ和LZ，使高频段电流主要流经RZ支路，提供足够的正阻尼以抵消MMC的负阻尼。

### 数学公式


**公式1**: $$$G_d = e^{-sT} \approx \frac{s^{10} + (-1)^9 M_9 s^9 + \cdots + (-1)^0 M_0 s^0}{s^{10} + M_9 s^9 + \cdots + M_0 s^0}$$$

*链路延时环节的10阶Pade近似，用于精确建模控制系统延时对高频阻抗的影响*


**公式2**: $$$Z'_{\text{MMC}} = \frac{s L_{\text{eq}} + G_{\text{PI}} G_d}{1 - G_d}$$$

*MMC高频简化阻抗模型完整表达式，其中Leq为等值电感，GPI为电流内环PI控制器传递函数*


**公式3**: $$$Z_{\text{MMC}} = \frac{s L_{\text{eq}}}{1 - G_d}$$$

*忽略PI控制项后的MMC高频段简化阻抗，适用于纯无源阻尼分析场景*


**公式4**: $$$Z_Z = \frac{1}{sC_{Z1}} + \left(sL_Z + \frac{1}{sC_{Z2}}\right) // R_Z$$$

*C型滤波器的阻抗表达式，包含干路电容CZ1、串联谐振支路(CZ2和LZ)及阻尼电阻RZ*


**公式5**: $$$Z_{\text{MMCRE}} = Z_{\text{MMC}} // Z_Z = \frac{sL_{\text{eq}}}{1 - G_d} // \left[\frac{1}{sC_{Z1}} + \left(sL_Z + \frac{1}{sC_{Z2}}\right) // R_Z\right]$$$

*并联C型滤波器后MMC的等效重塑阻抗，用于稳定性分析和参数设计*


**公式6**: $$$\angle Z_{\text{MMC}} = \frac{\pi}{2} - \angle(1 - \cos(2\pi Tf) + j\sin(2\pi Tf))$$$

*MMC阻抗相位计算公式，用于确定负阻尼频段范围*


**公式7**: $$$Z_3 = \frac{R_1(R_2^2 + K_2^2) + R_2(R_1^2 + K_1^2)}{(R_1 + R_2)^2 + (K_1 + K_2)^2} + j\frac{K_2(R_1^2 + K_1^2) + K_1(R_2^2 + K_2^2)}{(R_1 + R_2)^2 + (K_1 + K_2)^2}$$$

*MMC与C型滤波器并联后的阻抗实部(R)和虚部(K)计算，用于分析相位关系*


**公式8**: $$$\begin{cases} (K_2^2 - R_2^2)(R_1 - R_2) - 2R_2K_1K_2 < 0 \\ K_2(R_1^2 + K_1^2) + K_1(R_2^2 + K_2^2) < 0 \end{cases}$$$

*并联阻抗相位θ3小于C型滤波器相位θ2的数学条件，确保重塑后的相位进入稳定区域*


### 算法步骤

1. 确定MMC第1负阻尼频段范围：根据链路延时T计算频段[1/(2T), 1/T]，例如当T=600μs时，频段为[833Hz, 1667Hz]

2. 选择干路电容CZ1：根据系统无功补偿限制和低频段阻抗要求确定，CZ1增大会降低中高频阻抗但增加无功补偿，需在无功容量和阻尼效果间权衡

3. 设计串联谐振支路参数：根据基频f0=50Hz计算CZ2和LZ的谐振条件，满足$2\pi f_0 = 1/\sqrt{L_Z C_{Z2}}$，使串联支路在基频处呈现极低阻抗，阻断基频电流流入RZ

4. 选择阻尼电阻RZ：根据第1负阻尼频段的阻抗幅值和相位要求，通过阻抗并联公式计算所需RZ范围，确保在该频段内重塑阻抗相位大于-90°(正阻尼)

5. 验证高频段特性：检查频率>1/T时重塑阻抗是否接近纯阻性(相位接近0°)，确保第2及后续负阻尼频段被有效抑制

6. 成本优化：在满足阻尼要求前提下，根据元件市场价格优化CZ1、LZ、RZ的组合，实现成本最优的参数配置


### 关键参数

- **链路延时T**: 约600μs（典型值），决定第1负阻尼频段位置

- **第1负阻尼频段**: [833Hz, 1667Hz]（当T=600μs时）

- **等值电感Leq**: $L_T + 0.5L$，典型值取决于变压器漏感和桥臂电感

- **基频f0**: 50Hz，CZ2与LZ的谐振频率

- **干路电容CZ1**: 数十至数百μF，根据系统无功容量限制选取

- **阻尼电阻RZ**: 数十至数百Ω，决定高频段正阻尼强度

- **串联电感LZ**: 数十mH，与CZ2构成基频谐振回路

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 高频振荡抑制验证（案例1） | 在未加C型滤波器时，系统在1010Hz处存在阻抗交点且相位差大于180°，发生约20次谐波的高频振荡；并联C型滤波器后，该频段负阻尼完全被消除，振荡消失 | 与传统RLC串联阻抗重塑器相比，基频有功损耗大幅降低（理论上接近零，因谐振支路在基频处阻断电流） |

| 鲁西异步联网工程复现 | 成功抑制1271Hz左右的高频振荡，验证了所提方法对实际工程振荡频率（约25次谐波）的有效性 | 与有源抑制策略（如锁相环添加复合滤波或电压前馈加低通滤波）相比，能完全消除负阻尼而非仅避开交点，电网阻抗变化时稳定性更 robust |

| 渝鄂联网工程复现 | 同时抑制1810Hz和700Hz两个频段的振荡风险，验证了方法对多频段负阻尼的抑制能力 | 传统A型滤波器或二阶高通滤波器未给出具体参数设计方法，而本文方法基于第1负阻尼频段定量设计，参数选取更精确 |

| 稳态特性验证 | 并联C型滤波器后，基频处MMC阻抗与滤波器阻抗并联，由于CZ2-LZ谐振支路在基频处阻抗极低，基频电流不流经RZ，系统稳态运行点不变，无功功率交换与未加滤波器时基本一致 | 与无谐振支路的单纯RC阻尼相比，基频损耗减少90%以上（理论上可减少至谐振回路等效电阻水平） |

| 暂态特性验证 | 在系统发生三相短路故障或功率阶跃变化时，C型滤波器不影响MMC的控制响应速度和故障穿越能力，暂态过程中未出现新的振荡模式 | 与在桥臂电抗并联无源支路的方法相比，不影响桥臂电流动态，对MMC内部环流控制无影响 |



## 量化发现

- MMC第1负阻尼频段严格位于[1/(2T), 1/T]区间内，当链路延时T=600μs时，该频段为833Hz至1667Hz，覆盖实际工程中观测到的1271Hz振荡
- C型滤波器在基频(50Hz)处，由于CZ2和LZ串联谐振，该支路阻抗理论上接近0（实际为谐振回路等效电阻），流经RZ的基频电流衰减超过40dB，基频有功损耗<0.1%额定功率
- 在f=1/T（约1667Hz）处，MMC阻抗相位由π跳变为0，相位变化率为$-2\pi T$ rad/Hz
- 当频率>2000Hz时，重塑后的阻抗相位从MMC的-90°(负阻尼)提升至接近0°(正阻尼)，相位改善超过90°
- 在1010Hz（第1负阻尼频段内），并联C型滤波器后系统阻抗相位从-135°提升至-45°以上，相位裕度增加>90°，满足奈奎斯特稳定判据
- CZ1取值每增加100μF，中频段(1-2kHz)阻抗降低约30%，但无功补偿量增加约5%额定容量
- RZ取值在50-200Ω范围内时，高频段(>1kHz)阻尼效果与RZ值近似成正比，但>200Ω后电感特性增强，阻尼效果边际递减


## 关键公式

### 重塑阻抗模型

$$$Z_{\text{MMCRE}} = \frac{sL_{\text{eq}}}{1 - e^{-sT}} // \left[\frac{1}{sC_{Z1}} + \left(sL_Z + \frac{1}{sC_{Z2}}\right) // R_Z\right]$$$

*用于分析并联C型滤波器后MMC的等效阻抗特性，是稳定性分析和参数设计的核心方程*

### 基频谐振条件

$$$f_{\text{res}} = \frac{1}{2\pi\sqrt{L_Z C_{Z2}}} = 50\text{Hz}$$$

*设计C型滤波器串联支路参数时必须满足的谐振频率方程，确保基频处低损耗*

### MMC相位特性

$$$\angle Z_{\text{MMC}} = \frac{\pi}{2} - \arctan\left(\frac{\sin(2\pi Tf)}{1 - \cos(2\pi Tf)}\right)$$$

*用于计算第1负阻尼频段[1/(2T), 1/T]内MMC的负阻尼角度范围，指导滤波器参数设计目标*



## 验证详情

- **验证方式**: 电磁暂态仿真验证与理论阻抗扫频对比
- **测试系统**: 基于MMC的柔性直流输电系统，交流侧采用π级联模型表征分布参数特性，包含变压器漏感LT、桥臂电感L和等效电阻R，具体参数见附录A表A3
- **仿真工具**: MATLAB/Simulink，使用Power System Blockset搭建详细电磁暂态模型，包含完整控制环节（电流内环PI控制、链路延时、电压前馈等）
- **验证结果**: 仿真结果表明：1) 未加滤波器时在1010Hz出现高频振荡，与理论预测的负阻尼频段交点一致；2) 加入C型滤波器后，1010Hz及更高频段振荡完全消除，系统稳定运行；3) 基频有功损耗较传统RLC方案降低超过90%；4) 三相短路故障下，故障电流和恢复特性与无滤波器时基本一致，证明不影响暂态特性；5) 阻抗扫频结果与高频简化模型在500-4000Hz频段内误差<5%，验证了简化模型的准确性
