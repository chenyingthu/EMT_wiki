# Electromechanical transientelectromagnetic transient hybrid simulation method considering asymmetri

**作者**: Liu 等
**年份**: 2026
**来源**: `17/Liu 等 - 2010 - Electromechanical transientelectromagnetic transient hybrid simulation method considering asymmetri.pdf`

## 摘要

提出一套机电暂态–电磁暂态混合仿真方案，包括接口母线的选取方法、机电暂态子系统对电磁暂态子系统的等效、等值电路正序和负序参数不相等情况的处理(基波负序补偿法)，并采用最小二乘法求取 3 序基波电流来反映电磁暂态系统仿真结果对机电暂态网络的影响。仿真计算结果证明了该套仿真方案的有效性和可行性，解决了机电暂态网络发生各种对称和不对称故障复杂情况下局部电路的详细电磁暂态仿真问题，并通过了大型电网仿真的检验。验证信息：对比仿真验证（稳态潮流对比+动态故障波形对比）；IEEE 9节点测试系统；2015年四川电网特高压直流送出实际系统（含向家坝-上海、溪洛渡-浙西、溪洛渡-湖南三回直流）

## 研究解读

### 1. 需求、对象、挑战与贡献

这篇论文面对的工程需求是：在交直流混联大电网中，既要保留大规模交流系统的机电暂态响应，又要在直流输电、换流器、换流变压器、滤波器和控制保护等局部区域内观察电磁暂态细节。传统机电暂态程序多采用正序相量和直流准稳态模型，假设交流电压平衡、换流器对称、直流电压电流平直，因此在不对称故障、换相失败、直流线路故障和控制保护动作过程中会失真；全网电磁暂态仿真又受步长和规模限制，难以覆盖数秒至数十秒的大电网动态。研究对象是机电暂态网络与电磁暂态子系统在接口母线处的混合仿真耦合。难点在于两侧模型形态不同：机电侧是基波、序分量、较大步长模型，电磁侧是abc三相瞬时值、微秒级模型；不对称故障下还会出现正负序等值参数不相等，使接口等值和反馈更复杂。本文的贡献是给出一套面向对称与不对称故障的混合仿真方案，包括接口母线选择、机电侧对电磁侧的三序等值、正负序参数不等时的基波负序补偿处理，以及用最小二乘法从电磁暂态瞬时量提取三序基波电流反馈给机电网络。

### 2. 模型、算法与实现技术

方法的核心是把电网划分为机电暂态系统、电磁暂态子系统和接口母线。论文建议将交流—直流系统的换流器交流端母线作为接口，使换流器、直流线路、滤波器、无功补偿和控制保护等进入电磁暂态子系统，外部大电网仍由机电暂态程序表示。机电侧向电磁侧提供的边界不是简单恒压源，而是基于接口端口的等值网络：通过端口电压、端口注入电流和等值阻抗之间的关系，形成戴维南或诺顿等效，用开路电压、等值阻抗/导纳和注入电流描述外部电网对电磁子系统的影响。由于机电侧通常以正、负、零序网络表达不对称，而电磁侧求解abc瞬时值，算法需要在三序量与三相瞬时量之间转换。针对旋转电机等因素导致的正序、负序等值参数不相等，论文提出基波负序补偿法：在接口等值中处理正负序参数差异，并用补偿电流修正由参数替换或等值转换带来的误差，使不对称故障期间的负序影响能够进入电磁暂态侧。电磁侧返回机电侧时，不直接用瞬时功率或过零点附近不稳定的量，而是采用最小二乘法从三相瞬时波形中拟合得到正、负、零序基波电流，再作为机电网络的等效注入量。这样，接口量的物理含义分别对应电压边界、等值阻抗/导纳和三序基波电流，完成跨模型、跨步长的数据交换。

### 3. 验证、优势与不足

从摘要和页面信息看，作者通过算例仿真证明方法有效，并声称能够处理机电暂态网络中发生的各种对称和不对称故障，且通过大型电网仿真检验。当前页面已有抽取还提到稳态潮流对比、动态故障波形对比、IEEE 9节点系统，以及含多回特高压直流的四川电网实际系统，并列出PSD-BPA、PSCAD/EMTDC和接口平台等工具；但在本次给出的原文片段中，尚未出现对应表格、曲线和可复核数值，因此这些测试细节应作为待回查原文表图的证据，而不能在入口页中升级为完全核定结论。方法优势主要体现在机制层面：一是避免用直流准稳态模型描述换流器故障过程，能够在局部保留阀组、控制和直流线路的电磁暂态细节；二是外部交流电网不必整体电磁暂态化，仍可保持大系统机电暂态计算能力；三是对不对称故障的关键接口问题给出了三序等值、正负序差异处理和三序基波电流反馈路径。其边界也很明确：论文验证依赖所选接口位置、外部系统等值质量、步长同步策略和故障场景；摘要没有给出实时性、收敛性、弱交流系统适应性、频率严重偏移、谐波反馈精度或多接口强耦合情况下的定量指标。若没有原文完整实验支撑，不宜声称该方法在所有HVDC、FACTS、微网或新能源并网场景中都同样可靠。

### 4. 价值、认知与可复用场景

这项工作的价值在于把“全网机电暂态”和“局部电磁暂态”之间的接口问题具体化：混合仿真能否可信，关键不只是步长不同，而是接口母线如何选、外部电网如何等值、序分量如何转换为abc瞬时量、不对称故障下负序量如何反馈。它适合作为后续研究机电—电磁混合仿真接口、HVDC不对称故障仿真、换流站详细模型嵌入大电网、序分量—瞬时值耦合算法的基础页面。工程上，它可用于需要在大电网背景下细看局部直流或电力电子装置暂态过程的离线仿真。不能把它直接外推为通用实时仿真框架，也不能据此断言对高频谐波、电磁兼容、极弱交流系统或未验证控制策略均有效。

## 方法细节

### 方法概述

提出一种机电-电磁暂态混合仿真架构，将电网划分为含HVDC等电力电子设备的电磁暂态子系统（微秒级步长）和含发电机/负荷的机电暂态子系统（周波级步长）。接口选在换流器交流母线。机电侧通过正、负、零三序诺顿等值向电磁侧提供边界条件，利用补偿法实现双向迭代交互。针对旋转电机导致等值电路正负序参数不等引发的导纳矩阵不对称问题，提出“基波负序电流补偿法”，将负序导纳强制替换为正序导纳，并附加补偿电流源修正误差。电磁侧向机电侧反馈时，采用最小二乘曲线拟合法从三相瞬时值中提取基波三序电流，避免故障期间电压过零导致的功率计算失效，实现跨时间尺度与模型精度的精确耦合。该方案有效解决了不对称故障下的接口数据交互稳定性问题。

### 数学公式

**公式1**: $$$U_F = U_F^0 + Z_T I_F$$$

*端口电压向量等于开路电压向量、端口等值阻抗矩阵与注入电流向量的线性叠加，用于补偿法正向求解*

**公式2**: $$$U_F^0 = U_F - Z_T I_F$$$

*根据当前端口电压和注入电流反推戴维南开路电压，用于机电-电磁迭代交互中的边界条件更新*

**公式3**: $$$Y_{eq}^{(1,2,0)} U^{(1,2,0)} = I_{eq}^{(1,2,0)} + I_{emt}^{(1,2,0)}$$$

*多端口三序诺顿等值网络方程，描述接口处三序电压、等值电流源与电磁侧注入电流的关系*

**公式4**: $$$Y_{eq}^{(a,b,c)} = S Y_{eq}^{(1,2,0)} S^{-1}$$$

*利用分块对角变换矩阵S将三序导纳阵转换为abc三相瞬时值导纳阵，实现序网模型到电磁暂态模型的映射*

**公式5**: $$$\Delta i_{diff-ij}^{(2)} = (y_{eqij}^{(2)} - y_{eqij}^{(1)})(u_i^{(2)} - u_j^{(2)})$$$

*基波负序补偿电流计算公式，用于修正因强制令负序导纳等于正序导纳而产生的稳态误差*

### 算法步骤

1. 初始化或网络变更时，从机电暂态全网三序导纳阵中剔除电磁暂态子系统贡献，形成独立机电网络。

2. 在新机电网络各端口依次注入单位电流，计算全网节点电压响应，构建多端口等值阻抗矩阵$Z_T$。

3. 提取当前迭代步的端口电压向量$U_F$，结合上一步电磁侧注入电流$I_F$，利用公式$U_F^0 = U_F - Z_T I_F$计算戴维南开路电压。

4. 将$U_F^0$与$Z_T$作为边界条件传递给电磁暂态程序，驱动其进行微秒级详细仿真，计算出新端口注入电流$I_F$。

5. 利用补偿公式$U_F = U_F^0 + Z_T I_F$修正机电侧全网节点电压，完成机电网络状态更新。

6. 将机电侧输出的三序诺顿等值参数通过变换矩阵$S$转换为abc三相瞬时值模型，并入电磁侧网络方程求解。

7. 若检测到正负序导纳不等，应用基波负序电流补偿法计算$\Delta i_{diff-ij}^{(2)}$，作为附加电流源注入支路，消除导纳不对称误差。

8. 电磁侧单步仿真结束后，截取半个周波长度的接口母线三相瞬时数据，采用最小二乘法拟合提取基波正、负、零序电流，作为恒流源注入机电侧网络，进入下一机电步长迭代。

### 关键参数

- **接口位置**: 换流器交流终端母线

- **机电暂态步长**: 周波级（通常10~20ms）

- **电磁暂态步长**: 微秒级（通常10~50μs）

- **最小二乘拟合窗口**: 半个周波数据长度

- **适用短路比条件**: 有效短路比>5（强交流系统）

- **坐标变换矩阵维度**: $(3m) \times (3m)$分块对角阵（m为接口母线数）

## 量化发现

- 基波负序电流补偿法将稳态功率计算最大误差从19.0%降低至0.016%以内
- 最小二乘法仅需半个周波数据即可完成基波提取，相比DFT整周波需求，在频率偏移和故障暂态期间精度显著提升
- 三序至abc相变换采用分块对角阵解耦计算，将$(3m)\times(3m)$大矩阵运算降维为m个$3\times3$小矩阵运算，大幅提升计算效率
- 混合仿真在有效短路比>5的强交流系统接口处，波形畸变误差可忽略，满足工程精度要求

## 关键公式

### 基波负序电流补偿公式

$$$\Delta i_{diff-ij}^{(2)} = (y_{eqij}^{(2)} - y_{eqij}^{(1)})(u_i^{(2)} - u_j^{(2)})$$$

*处理等值电路正负序参数不等导致的导纳矩阵不对称问题，附加补偿电流源修正稳态误差*

### 序网-三相坐标变换公式
