# Frequency-Domain Simulation of Electromagnetic Transients Using Variable

**作者**: Abner Ramirez; Reza Iravani; Fellow
**年份**: 2015
**来源**: `19、20、21/EMT_task_20/TPWRD.2015.2449754.pdf.pdf`

## 摘要

该方法将电磁暂态分析时间窗划分为多个长度可相等或不等的子时间窗，每个子窗独立设置采样步长。系统被虚拟划分为“研究区域”和“外部子系统”，外部子系统通过有理函数拟合降阶为频域网络等值（FDNE）。利用频域代数关系直接求解各子窗的节点电压与初始条件，彻底避免了传统时域方法中的卷积运算与导纳矩阵重复三角分解。通过灵活匹配开关事件时刻划分子窗，实现快慢动态过程的高效同步仿真。该方法将集中参数电路的初始条件概念推广至频变网络，使大规模电力系统在保持高精度的同时大幅降低计算负担，特别适用于含离散开关动作与多时间尺度动态的复杂暂态分析。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自大规模电力网络的EMT仿真：开关、故障等离散事件会激发很快的行波/高频暂态，而负荷或系统响应又包含较慢动态。传统时域EMTP类方法为保证快暂态精度往往全程使用小步长，并在拓扑变化时处理卷积、导纳矩阵更新或重新三角分解，计算代价随网络规模和频变元件增加而上升。本文研究对象是可被划分为“研究区”和“外部子系统”的频变网络，外部子系统用有理函数形式的频域网络等值（FDNE）表示。相对作者此前只面向单条输电线的分段时间-频域方法，本文的贡献是把该思想扩展到电力网络等值：把总仿真时间窗划成长度可不同的子窗，每个子窗独立选采样步长，并在子窗边界用FDNE代数计算初始条件电流源，从而使开关事件可落在子窗边界，快慢动态可用不同采样密度处理。

### 2. 模型、算法与实现技术

算法的核心不是逐点时域积分，而是在每个子时间窗内进行频域求解。系统先虚拟分成研究区和外部子系统；从研究区看外部子系统，其频率响应由有理函数拟合成FDNE。第一子窗在零初始条件下求解完整网络，得到节点电压，尤其是研究区与外部子系统之间的接口节点电压。随后利用这些接口电压和FDNE的频域状态关系，在频域代数地求出下一子窗所需的初始条件，不数值求解FDNE对应的ODE；这些初始条件以频率相关电流源形式接到接口节点。若研究区发生开关等拓扑变化，则在进入对应子窗时更新研究区模型。每个后续子窗通过方程YV=Icalc+Iknown求解：Y是包含研究系统参数的导纳矩阵，V是接口相关节点电压，Icalc是由上一子窗初始条件换算来的电流源，Iknown是研究区已知电流源。该流程把“初始条件”从集中参数电路推广到频变网络等值，并允许子窗长度和采样步长按事件和动态速度重设。

### 3. 验证、优势与不足

作者用文献[4]的双开关输电线路算例验证。开关S1在t=0闭合，S2在t=0.05 s闭合，闭合电导为100 mho，断开状态电导为0；系统按图1分为外部子系统和研究区。所提方法在Matlab中实现，并与PSCAD/EMTDC时域仿真比较节点2电压波形。为达到相近精度，PSCAD/EMTDC采用15 μs步长，共7334个样本；所提方法按表I划分子窗，其中一个子窗用1024个样本以捕捉初始暂态。图3显示两者在初始合闸暂态和0.05 s二次开关暂态处吻合良好。CPU时间方面，Matlab版FD方法为0.49 s，PSCAD/EMTDC为0.64 s；该PSCAD时间不含传输线拟合过程8.1 s。作者还将Matlab代码生成MEX文件，运行时间为0.20 s，约为Matlab脚本的2.4倍速度。优势主要体现在：可按开关时刻布置子窗、可在同一仿真内用不同采样步长处理快慢动态、FDNE初始条件通过代数关系计算。边界是验证只覆盖一个基准网络和两次预设开关；若开关时刻落在子窗内部，作者只说明需迭代或重新分配子窗，未给出该情形的数值验证。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：频域EMT仿真不必只适用于固定时间窗或单个频变元件；只要能在子窗边界传递等效初始条件，就可以把频变外部网络压缩为FDNE，并把研究区内的离散拓扑变化局部化处理。它适合被后续研究复用于大系统等值、含开关事件的离线EMT加速仿真、多速率暂态分析，以及研究区/外部网络分区建模。它不应被直接外推为实时仿真方案、任意非线性控制系统求解器，或所有频率响应复杂网络都高效；作者明确指出，当驱动点导纳矩阵含大量频率谐振时，FDNE拟合阶数会增加。

## 方法细节

### 方法概述

该方法将电磁暂态分析时间窗划分为多个长度可相等或不等的子时间窗，每个子窗独立设置采样步长。系统被虚拟划分为“研究区域”和“外部子系统”，外部子系统通过有理函数拟合降阶为频域网络等值（FDNE）。利用频域代数关系直接求解各子窗的节点电压与初始条件，彻底避免了传统时域方法中的卷积运算与导纳矩阵重复三角分解。通过灵活匹配开关事件时刻划分子窗，实现快慢动态过程的高效同步仿真。该方法将集中参数电路的初始条件概念推广至频变网络，使大规模电力系统在保持高精度的同时大幅降低计算负担，特别适用于含离散开关动作与多时间尺度动态的复杂暂态分析。

### 数学公式

**公式1**: $$$$ \mathbf{Y} \mathbf{V} = \mathbf{I}_{\text{calc}} + \mathbf{I}_{\text{known}} $$$$

*频域节点电压求解核心方程。其中$\mathbf{Y}$为包含研究系统参数的节点导纳矩阵，$\mathbf{V}$为接口节点电压向量，$\mathbf{I}_{\text{calc}}$为上一子窗传递的频变等效电流源，$\mathbf{I}_{\text{known}}$为研究区内已知的独立电流源。该方程用于在每个子时间窗内直接代数求解网络状态，替代时域步进积分。*

### 算法步骤

1. 步骤1（零初始条件频域求解）：在首个子时间窗内，假设$t=0$时刻系统初始条件为零，在频域内求解整个网络（含外部子系统与研究区），获取所有节点电压，特别是研究区与外部子系统之间的接口节点电压向量。

2. 步骤2（频域初始条件代数计算）：基于步骤1得到的接口节点电压，利用外部子系统的FDNE模型，在频域内通过代数运算直接求解状态变量（避免对ODE进行数值积分），生成下一子窗所需的频变电流源作为初始条件。

3. 步骤3（拓扑更新与当前子窗求解）：将步骤2计算的频变电流源注入接口节点。若研究区内发生开关动作等拓扑变化，在此步骤更新网络模型。随后利用核心方程$\mathbf{Y} \mathbf{V} = \mathbf{I}_{\text{calc}} + \mathbf{I}_{\text{known}}$在频域求解当前子窗的节点电压。

4. 步骤4（迭代推进）：重复步骤2和3，依次推进至后续所有子时间窗，直至完成整个分析时间窗的暂态仿真。若开关时刻落在子窗内部，可通过迭代或重新分配子窗边界处理。

### 关键参数

- **子时间窗划分策略**: 根据开关事件或动态特性变化点灵活设定起止时刻，无需等长

- **采样步长设置**: 各子窗独立选择（如快动态阶段采用15μs，慢动态阶段采用更大步长）

- **FDNE拟合阶数**: 随驱动点导纳矩阵中频率谐振点数量增加而提高，影响模型复杂度

- **开关导通电导**: 100 mho（断开状态设为0）

- **初始暂态采样点数**: 首个子窗采用1024个采样点以高精度捕捉初始冲击

## 量化发现

- 在保持与PSCAD/EMTDC同等精度的前提下，MATLAB版仿真耗时为0.49 s，较PSCAD的0.64 s减少约23.4%。
- 若计入PSCAD的传输线有理函数拟合时间（8.1 s），所提频域方法整体计算效率提升超过一个数量级。
- MEX编译版执行时间仅0.20 s，较原生MATLAB代码提速2.4倍，验证了算法在高效编程环境下的巨大潜力。
- 首个子窗仅需1024个采样点即可高精度捕捉初始暂态过程，避免了全局采用极小步长导致的计算冗余。
