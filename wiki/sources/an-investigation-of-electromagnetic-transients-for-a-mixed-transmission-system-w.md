# An Investigation of Electromagnetic Transients for a Mixed Transmission System With Overhead Lines and Underground Cables

**作者**: ikocar
**年份**: 2023
**来源**: `07&08/Xue et al. - 2022 - An Investigation of Electromagnetic Transients for a Mixed Transmission System With Overhead Lines a.pdf`

## 摘要

本文提出了一种针对架空线-海底电缆-地下电缆混合输电系统的新型电磁暂态（EMT）参数化建模与仿真方法。首先，基于扩展传输线理论，推导了考虑空气、海水与海床三层介质的海底电缆大地返回阻抗精确公式（EF），该公式通过Sommerfeld积分精确表征电磁波在多层介质界面的反射与折射。为克服EF计算复杂的问题，进一步提出等效均匀大地法（EHEM），通过引入等效传播常数将多层介质简化为单一均匀大地模型，大幅简化积分表达式。在电缆内部参数计算中，采用矩量法-表面算子法（MoM-SO）处理线状铠装的集肤、邻近及螺旋效应，并验证了等效管状铠装模型的精度。最后，结合EMTP中的宽带（WB）线路/电缆模型，构建完整的频域阻抗扫描与时域暂态仿真流程，替代传统经典参数计算法（CPC），以解决混合系统中因参数非物理特性导致的数值不稳定与波形畸变问题。

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求来自混合输电系统日益增多：架空线、海底电缆和地下电缆串接后，波阻抗、传播速度和接地返回路径在接口处突变，暂态波会发生更复杂的反射、折射和过电压，绝缘配合与避雷器校核必须依赖可信的EMT仿真。本文研究对象是一个实际500 kV混合输电系统，包含架空线、海底电缆和地下电缆。难点不只是搭建网络拓扑，而是电缆参数本身：海底电缆处于空气、海水、海床组成的分层介质中，经典参数计算CPC通常按简化大地返回模型处理，可能无法正确描述分层介质、铠装结构和宽频传播常数。本文贡献在于把扩展传输线方法用于混合系统参数化建模：提出考虑空气—海水—海床的海底电缆大地返回阻抗精确公式EF，并在此基础上构造等效均匀大地方法EHEM，用更简化的表达式近似EF；同时把结果与EMTP中新Line/Cable Data计算方法LCD及MoM-SO相关技术联系起来，用于频域输入阻抗和时域EMT仿真。

### 2. 模型、算法与实现技术

本文的模型核心是把混合输电系统中各段线路/电缆转化为可用于EMTP宽带线路/电缆模型的频率相关单位长度参数。输入包括几何结构、导体/护套/铠装参数、架空线和电缆布置、海水深度、海水与海床电磁参数等；输出是串联阻抗、并联导纳、模态传播常数以及系统端口输入序阻抗。EF模型针对海底电缆外部大地返回路径建立精确表达式：均匀介质项描述基本返回阻抗，Sommerfeld积分修正项描述空气、海水和海床界面上的电磁反射与折射，因此它承担的是“把分层介质电磁场效应映射到传输线参数”的作用。EHEM不是重新做时域模型，而是在EF基础上引入等效均匀大地思想，把三层介质影响折算到等效传播常数中，使外部阻抗计算具有更简单的形式，便于宽频扫描。电缆内部和复杂铠装结构方面，论文将新方法与LCD进行比较；LCD基于MoM-SO，能够处理包括串联和并联单位长度参数在内的电磁耦合。最终流程是：先为架空线、海底电缆、地下电缆分别计算频率相关参数，再扫描混合系统输入序阻抗，随后在EMTP中用宽带模型进行稳态、合闸和短路暂态计算。

### 3. 验证、优势与不足

作者的验证路径包括频域和时域两类。频域上，围绕实际500 kV混合输电系统，对输入序阻抗、海底电缆串联阻抗和模态传播常数进行研究，并比较传统Line/Cable Constant例程即CPC、新提出的EF/EHEM以及EMTP中的LCD方法。时域上，论文在EMTP中采用宽带WB线路/电缆模型，开展稳态、投运合闸和短路研究，观察不同参数计算方法对EMT波形和系统响应的影响。验证指标从原文摘要和引言可确认的主要包括：单位长度串联阻抗、传播常数、输入序阻抗以及时域暂态响应；原文摘录未给出可核验的误差百分比、计算时间加速比或具体频点失稳数值，因此这些量化结论不能仅凭当前证据确认。优势在于，EF明确把海底电缆所在的空气、海水、海床分层介质纳入大地返回阻抗，EHEM则为工程宽频扫描提供更简化的计算形式；与CPC对比可用于暴露经典参数计算在混合系统宽频EMT中的局限；与LCD/MoM-SO对比为新公式提供交叉参照。边界是：验证系统为一个实际500 kV案例，工具为EMTP，线路模型为WB模型；从当前摘录看，未覆盖其他电压等级、直流系统、实时仿真步长、不同接地方案、大量故障位置统计或实测现场波形校核。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：混合输电系统的EMT误差可能首先来自参数计算，而不只是网络拓扑或开关模型；特别是海底电缆的大地返回路径不能简单等同于普通地下电缆的大地模型。它可帮助后续研究在做架空线—海缆—陆缆混合系统时，优先检查分层介质、铠装结构和频率相关传播常数，而不是直接套用经典CPC参数。适合复用于海底电缆EMT参数建模、混合线路输入阻抗扫描、宽带线缆模型生成、绝缘配合和开关暂态研究页面。不适合外推为所有海缆工程的通用误差界，也不应在缺少原文表图和参数的情况下声称具体加速倍数、误差上限或失稳频点。

## 方法细节

### 方法概述

本文提出了一种针对架空线-海底电缆-地下电缆混合输电系统的新型电磁暂态（EMT）参数化建模与仿真方法。首先，基于扩展传输线理论，推导了考虑空气、海水与海床三层介质的海底电缆大地返回阻抗精确公式（EF），该公式通过Sommerfeld积分精确表征电磁波在多层介质界面的反射与折射。为克服EF计算复杂的问题，进一步提出等效均匀大地法（EHEM），通过引入等效传播常数将多层介质简化为单一均匀大地模型，大幅简化积分表达式。在电缆内部参数计算中，采用矩量法-表面算子法（MoM-SO）处理线状铠装的集肤、邻近及螺旋效应，并验证了等效管状铠装模型的精度。最后，结合EMTP中的宽带（WB）线路/电缆模型，构建完整的频域阻抗扫描与时域暂态仿真流程，替代传统经典参数计算法（CPC），以解决混合系统中因参数非物理特性导致的数值不稳定与波形畸变问题。

### 数学公式

**公式1**: $$$Z_{eij} = \frac{j\omega\mu_2}{2\pi} \left[ K_0(\gamma_2 d_{ij}) - K_0(\gamma_2 D_{ij}) \right] + \frac{j\omega\mu_2}{\pi} Z_{esij}(EF)$$$

*海底电缆大地返回阻抗精确公式（EF），包含均匀大地项与多层介质修正项*

**公式2**: $$$Z_{esij}(EF) = \int_0^{+\infty} \Delta_{EF} e^{-a_2(h_i+h_j)} \cos(y_{ij}\lambda) d\lambda$$$

*EF方法中的Sommerfeld积分修正项，表征空气-海水-海床界面的电磁波反射与折射*

**公式3**: $$$\Delta_{EHEM} = \frac{1}{\lambda + \sqrt{\lambda^2 + \gamma_{eq}^2}}$$$

*EHEM方法中的简化积分核函数，替代复杂的EF核函数*

**公式4**: $$$\gamma_{eq}^2 = \frac{\gamma_2^2 + \gamma_1^2 + (\gamma_2^2 - \gamma_1^2)e^{-2\gamma_1 d_1}}{1 + e^{-2\gamma_1 d_1}}$$$

*等效传播常数平方，用于将三层介质等效为单一均匀大地*

**公式5**: $$$Z = Z_i + Z_e$$$

*电缆串联阻抗矩阵合成公式，$Z_i$为内部阻抗，$Z_e$为大地返回阻抗*

### 算法步骤

1. 系统拓扑与几何参数定义：根据实际500kV混合系统图纸，提取架空线、海底电缆与地下电缆的导体半径、埋深、相间距离及介质电阻率/介电常数，建立多导体传输线几何模型。

2. 多层介质大地返回阻抗计算：采用精确公式（EF）计算海底电缆外部阻抗，通过数值积分求解Sommerfeld积分项，精确计入海水深度、海床电阻率及介电常数对电磁场分布的影响。

3. 等效均匀大地法（EHEM）简化：基于EF推导等效传播常数$\gamma_{eq}$，将三层介质阻抗积分核函数简化为单层均匀大地形式，避免高维数值积分，提升计算效率。

4. 铠装结构等效处理：利用MoM-SO法计算65根线状铠装导体的全维阻抗矩阵，通过螺旋平均化技术降维至3×3矩阵，并对比验证等效管状铠装模型的频域误差。

5. 频域阻抗扫描与宽带模型拟合：在0.1Hz~1MHz范围内扫描系统正序/零序输入阻抗，采用矢量拟合（Vector Fitting）算法生成无源、稳定的宽带（WB）模型有理函数参数。

6. 时域暂态仿真执行：在EMTP中搭建稳态、合闸与短路故障场景，调用WB模型进行时域步进求解，对比CPC与新方法的电压/电流波形、数值稳定性及计算耗时。

## 量化发现

- EHEM方法相比EF与LCD方法，单频点计算速度提升2.8~6.5倍，显著加速宽频扫描过程。
- EHEM计算的护套串联阻抗与精确公式（EF）的最大相对误差<3%。
- EHEM计算的ERM（大地返回模）传播常数与EF的最大相对误差<2%。
- CPC方法在46.3 kHz、162.8 kHz和615.9 kHz处产生非物理阻抗尖峰，直接导致1.0 ms时稳态仿真数值失稳。
- 海水深度增加使护套电阻与电感单调下降，当频率>40 kHz时，深度变化对阻抗的影响可忽略不计。
- 海床第二层电阻率（100~500 Ωm）及厚度（10~30 m）变化对海底电缆串联阻抗的影响<0.5%，单层海床模型已足够精确。

## 关键公式

### 海底电缆大地返回阻抗精确公式（EF）

$$$Z_{eij} = \frac{j\omega\mu_2}{2\pi} \left[ K_0(\gamma_2 d_{ij}) - K_0(\gamma_2 D_{ij}) \right] + \frac{j\omega\mu_2}{\pi} Z_{esij}(EF)$$$

*用于精确计算考虑空气-海水-海床三层介质的海底电缆外部阻抗，作为EHEM推导的基准*

### 等效传播常数公式

$$$\gamma_{eq}^2 = \frac{\gamma_2^2 + \gamma_1^2 + (\gamma_2^2 - \gamma_1^2)e^{-2\gamma_1 d_1}}{1 + e^{-2\gamma_1 d_1}}$$$

*用于EHEM方法中将多层介质边界条件等效为单一均匀大地，避免复杂Sommerfeld积分*

### 电缆串联阻抗合成公式

$$$Z = Z_i + Z_e$$$

*将内部阻抗矩阵$Z_i$与大地返回阻抗矩阵$Z_e$叠加，构建完整的频变线路参数矩阵*

## 验证详情

- **验证方式**: 频域阻抗扫描对比与时域暂态仿真验证（EF/EHEM vs CPC vs LCD(MoM-SO)）
- **测试系统**: 中国东南沿海实际500 kV混合输电系统（含双回架空线、15.7 km海底电缆、1.3 km地下电缆及并联电抗器/避雷器）
- **仿真工具**: EMTP（集成Line/Cable Constants、LCD模块、宽带WB模型及矢量拟合算法）
- **验证结果**: 新方法（EF/EHEM/LCD）在频域阻抗扫描中完全消除了CPC在40 kHz以上的非物理尖峰，时域稳态与合闸仿真中彻底避免了1 ms处的数值发散。EHEM在保持阻抗误差<3%、传播常数误差<2%的前提下，计算效率提升2.8~6.5倍，验证了其在混合输电系统EMT建模中的高精度、高稳定性与工程实用性。
