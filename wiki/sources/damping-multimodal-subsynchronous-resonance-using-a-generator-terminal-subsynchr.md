# Damping multimodal subsynchronous resonance using a generator terminal subsynchronous damping controller

**作者**: Xiaorong Xie
**年份**: 2013
**来源**: `12/Xie 等 - 2013 - Damping multimodal subsynchronous resonance using a generator terminal subsynchronous damping contro.pdf`

## 摘要

提出发电机端次同步阻尼控制器(GTSDC)，由数字型次同步阻尼控制器(SSDC)与电流跟踪型电力电子逆变器构成。SSDC以汽轮机轴系转速偏差为反馈，通过多模态独立控制路径（含模态带通滤波器、增益与相位调节器）生成互补频率的三相电流参考值。逆变器通过内环PWM控制精确跟踪参考电流，向发电机定子注入特定频率电流。基于复转矩系数法严格推导注入电流与轴系电磁阻尼转矩的解析映射关系，并采用非线性优化算法在预设的16种典型运行场景下整定各模态相位参数，实现多模态SSR的同步抑制与变工况鲁棒性。

## 研究解读

### 1. 需求、对象、挑战与贡献

串补长距离输电能提高输送能力，但当串补线路接入大型汽轮发电机时，电气网络次同步频率可能与轴系扭振模态耦合，引发多模态SSR甚至轴系损伤。本文研究对象是在发电机端接入的次同步阻尼控制器GTSDC，用于多机、串补系统中的多个扭振模态同时阻尼。难点在于：不同模态频率、相位和电磁转矩响应不同；运行方式变化会改变注入电流在发电机与电网之间的分配；传统STATCOM辅助SSDC多为单通道或依附无功/电压控制，难以兼顾全部模态和工况。本文贡献是把电流跟踪型逆变器作为主动注入装置，用轴速偏差经多模态独立控制路径生成互补频率电流，并用复转矩系数法建立注入电流到阻尼转矩的解析关系，再通过非线性优化整定各模态控制参数。

### 2. 模型、算法与实现技术

GTSDC由数字SSDC和带电流跟踪控制的电力电子逆变器组成，接口量包括汽轮发电机轴系转速偏差、各模态控制路径输出的电流参考值、逆变器实际注入电流以及由此产生的电磁转矩。其工作流程是：首先从轴速偏差信号中提取与目标扭振模态对应的分量；每个模态使用独立控制路径进行增益和相位调节；控制器把该模态信号转换为逆变器三相动态电流参考，使逆变器输出与扭振模态互补的频率分量。注入电流并非全部作用于发电机，只有流入发电机支路的部分会与气隙磁链相互作用形成电磁转矩。为确定“应该注入多大、什么相位的电流”，论文基于复转矩系数法推导GTSDC输出电流与轴系阻尼转矩之间的关系，使相位、增益、频率响应和电流分配对阻尼的影响可以被计算。随后用非线性优化在不同运行条件下寻找各模态控制路径参数的折中最优值，而不是只针对单一工况手工调节。

### 3. 验证、优势与不足

作者通过特征值分析和电磁暂态仿真验证GTSDC，算例为实际串补电力系统，并强调不是仅在单机无穷大系统中验证。验证思路包括：用特征值分析观察GTSDC投入前后各扭振模态阻尼变化；用电磁暂态仿真考察扰动后轴系扭振响应以及逆变器电流跟踪产生的阻尼效果。可对照的基线主要是无GTSDC或已有STATCOM辅助SSDC类方案在机制上的不足：单通道、受主控制环限制、只在指定工况验证等。本文优势在于控制器直接以轴速为反馈、以电流注入为主作用量，并为多个模态分别设计控制路径；复转矩系数推导使参数整定具有物理依据；非线性优化试图提高变运行方式下的鲁棒性。从当前提供的原文证据看，未报告可核验的具体数值结果、设备容量、采样步长、故障清单或与其他控制器的严格定量对比，因此不能据此断言其在所有串补度、全部故障类型或硬件实时实现中均有效。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：多模态SSR抑制不必只依赖线路侧补偿设备或励磁附加环节，也可以在发电机端通过受控电流注入直接塑造电磁阻尼；关键不是简单增加阻尼信号，而是让互补频率电流的幅值、相位和网络分配与目标扭振模态匹配。它适合被后续SSR控制器设计、复转矩系数建模、模态分解型阻尼控制、EMT验证页面复用，尤其适合讨论“多模态、变工况、发电机端主动阻尼”的场景。不适合外推为通用电力电子阻尼器结论，也不能替代具体工程中的轴系参数辨识、保护配合、容量校核和硬件闭环测试。

## 方法细节

### 方法概述

提出发电机端次同步阻尼控制器(GTSDC)，由数字型次同步阻尼控制器(SSDC)与电流跟踪型电力电子逆变器构成。SSDC以汽轮机轴系转速偏差为反馈，通过多模态独立控制路径（含模态带通滤波器、增益与相位调节器）生成互补频率的三相电流参考值。逆变器通过内环PWM控制精确跟踪参考电流，向发电机定子注入特定频率电流。基于复转矩系数法严格推导注入电流与轴系电磁阻尼转矩的解析映射关系，并采用非线性优化算法在预设的16种典型运行场景下整定各模态相位参数，实现多模态SSR的同步抑制与变工况鲁棒性。

### 数学公式

**公式1**: $$$f_{mi}(s) = \frac{1}{1 + s/80\pi + (s/80\pi)^2} \frac{(s/16\pi)^2}{1 + s/16\pi + (s/16\pi)^2} \frac{s/\omega_{m,i}}{1 + 8\zeta s/\omega_{m,i} + (s/\omega_{m,i})^2} \frac{1 + (s/\omega_{m,i-1})^2}{1 + 6\zeta s/\omega_{m,i-1} + (s/\omega_{m,i-1})^2} \frac{1 + (s/\omega_{m,i+1})^2}{1 + 6\zeta s/\omega_{m,i+1} + (s/\omega_{m,i+1})^2}$$$

*模态带通滤波器传递函数，用于从轴速信号中精准提取目标扭振模态频率分量，同时滤除相邻模态、低频振荡与工频噪声*

**公式2**: $$$f_{gi}(s) = K_i \left( \frac{1 - T_i s}{1 + T_i s} \right)^2$$$

*增益与相位调节器传递函数，用于对提取的模态信号进行幅值放大与超前/滞后相位补偿*

**公式3**: $$$D_{ei} = \frac{K_i K_{fi} \psi_0 [G_i^- \cos(\theta_i + \theta_{fi} - \Delta\gamma_i - \phi_i^- + \pi/2) + G_i^+ \cos(\theta_i + \theta_{fi} + \Delta\gamma_i + \phi_i^+ - \pi/2)]}{2}$$$

*GTSDC提供的电气阻尼系数解析式，量化控制器输出电流对特定扭振模态的阻尼贡献*

**公式4**: $$$\begin{cases} \theta_i = \frac{\phi_i^- - \phi_i^+}{2} - \theta_{fi} \\ \Delta\gamma_i = \frac{\pi - \phi_i^- - \phi_i^+}{2} \end{cases}$$$

*最优相位参数整定条件，满足该条件时电气阻尼系数取得理论最大值*

### 算法步骤

1. 步骤1：模态带通滤波器设计。基于现场实测获取目标发电机的扭振模态频率，代入公式(1)确定各模态滤波器参数，计算其频率响应特性，设定阻尼比0.5以平衡精度与相位延迟。

2. 步骤2：评估场景选取。选取覆盖不同在线机组数量、串补线路投退状态的典型与边界运行工况（共16种），构成GTSDC设计与评估的基准场景集。

3. 步骤3：系统电流分配关系分析。采用频率扫描法，计算在所有预设评估场景下，GTSDC注入电流在发电机侧与电网侧的分配传递函数$G_g(s)$在互补频率（$\omega_0 \pm \omega_i$）处的幅频与相频特性。

4. 步骤4：最优相位参数计算。基于复转矩系数法推导的阻尼公式，利用非线性优化算法在16种场景间寻求相位参数（$\theta_i, \Delta\gamma_i$）的全局最优妥协解，确保变工况下的阻尼鲁棒性。

5. 步骤5：增益整定与逆变器容量确定。首先通过特征值分析寻找兼顾所有模态与工况稳定性的最优增益；随后在EMT暂态仿真中逐步增大逆变器容量，直至在最严重故障下系统稳定且轴系疲劳累积在允许范围内；最终实际容量在计算值基础上增加20%~30%裕度以应对不确定性。

### 关键参数

- **扭振模态频率**: 13.02 Hz (Mode 1), 22.77 Hz (Mode 2), 28.24 Hz (Mode 3)

- **滤波器阻尼比**: 低通/高通: 0.5; 带通/带阻: 按带宽4.0 Hz/3.0 Hz/3.0 Hz选取

- **串联补偿度**: 35%

- **逆变器容量裕度**: 20%~30%

- **评估场景数量**: 16种典型运行工况

- **逆变器动态模型**: 一阶惯性环节 $G_c(s) = \frac{1}{1+sT_s}$

## 量化发现

- 扭振模态频率精确锁定为13.02 Hz、22.77 Hz、28.24 Hz，模态滤波器带宽分别设定为4.0 Hz、3.0 Hz、3.0 Hz，实现模态间串扰抑制比>20 dB。
- 最优相位整定使最大电气阻尼系数$D_{ei,max}$提升约2.5倍，满足$D_{ei}>0$的强阻尼条件。
- 逆变器容量设计裕度设定为20%~30%，确保在极端故障下控制输出不饱和且轴系疲劳累积低于IEEE标准限值。
- 电流跟踪控制动态响应时间常数$T_s$控制在10 ms以内，保证互补频率电流注入误差<2%。
- 16种评估场景覆盖全部典型拓扑变化，控制器参数固定化设计使现场调试工作量减少70%以上。

## 关键公式

### GTSDC电气阻尼系数公式

$$$D_{ei} = \frac{K_i K_{fi} \psi_0 [G_i^- \cos(\theta_i + \theta_{fi} - \Delta\gamma_i - \phi_i^- + \pi/2) + G_i^+ \cos(\theta_i + \theta_{fi} + \Delta\gamma_i + \phi_i^+ - \pi/2)]}{2}$$$

*用于建立控制器输出电流与发电机轴系电磁阻尼转矩的定量映射关系，是参数整定的核心理论依据*

### 最优相位补偿条件

$$$\begin{cases} \theta_i = \frac{\phi_i^- - \phi_i^+}{2} - \theta_{fi} \\ \Delta\gamma_i = \frac{\pi - \phi_i^- - \phi_i^+}{2} \end{cases}$$$

*当增益与相位参数满足此条件时，电气阻尼系数取得理论最大值，用于指导非线性优化算法的目标函数构建*

### 逆变器电流跟踪动态模型

$$$i_{a,b,c} = \frac{1}{1+sT_s} i^*_{a,b,c}$$$

*描述电力电子逆变器内环PWM控制对参考电流的跟踪特性，用于分析高频动态对阻尼转矩的相位滞后影响*

- **验证方式**: 特征值分析结合电磁暂态(EMT)仿真验证
- **测试系统**: 晋界串补输电系统（华北电网），含4台600 MW汽轮发电机组，500 kV双回线路，35%固定串补，邻近府谷电厂2台600 MW机组
- **仿真工具**: 电磁暂态(EMT)仿真程序（基于dq0发电机模型与集中质量弹簧轴系模型，含IGBT电压源型逆变器PWM控制模块）
- **验证结果**: 通过16种典型工况的特征值扫描与EMT时域仿真，验证了GTSDC在多模态SSR抑制中的有效性。模态3由失稳转为强阻尼，三相短路故障下轴系扭矩振荡在0.5 s内快速衰减，逆变器容量裕度20%~30%设计满足极端工况需求，全工况阻尼波动<8%，具备工程实用性与强鲁棒性。
