# 适用于电磁暂态高效仿真的变流器分段广义状态空间平均模型

**作者**: 王磊
**年份**: 2019
**来源**: `40/王磊 等 - 2019 - 适用于电磁暂态高效仿真的变流器分段广义状态空间平均模型.pdf`

## 摘要

针对新能源并网变流器电磁暂态仿真精度和效率的平衡问题，研究新能源并网变流器的常用平均化建模方法，提出适应大规模新能源并网电磁暂态高效仿真的变流器分段广义状态空间平均建模方法。该方法将分段技术应用于变流器广义状态空间模型中，将变流器动作特性一致的时间段合并研究，实现分段时间上变流器电磁暂态的变尺度建模。应用提出的变流器分段广义状态空间平均模型进行算例仿真，结果表明该模型实现了变流器暂态仿真中效率和精度的平衡，能够适应大规模新能源并网变流器的高效仿真。

## 研究解读

### 1. 需求、对象、挑战与贡献

实际需求是：在含大规模新能源并网的电力系统电磁暂态仿真中，变流器数量多、开关频率高，如果采用详细开关模型，需要微秒级小步长跟踪每次开关动作，计算负担重；若采用传统平均模型，又可能丢失电磁暂态中与开关相关的高频分量和谐波。研究对象是新能源并网变流器，尤其是PWM调制下的并网变流器电磁暂态模型。难点在于变流器既有开关周期尺度的快速非连续动作，又有控制、电网扰动等较慢动态，单一时间尺度模型难以同时兼顾。本文的贡献不是简单“平均化”，而是把分段思想引入广义状态空间平均法：将开关动作特性相近的时间段合并，在每个分段内用傅里叶系数描述状态量，从而在保留必要谐波信息的同时减少逐开关建模的代价。

### 2. 模型、算法与实现技术

本文提出分段广义状态空间平均模型，即P-GSSA。其基础是PWM开关函数Si：由调制波与三角载波比较得到各相开关状态，再由一个开关周期内Si的积分得到占空比di。传统状态空间平均模型用占空比对不同开关状态下的状态方程加权，得到平均状态变量y；广义状态空间平均法进一步把时域状态量展开为滑动傅里叶系数〈x〉k，用微分性质把dx/dt转换为傅里叶系数的动态方程，用乘积卷积性质处理状态方程中的乘积项。P-GSSA的关键机制是分段：不把全部仿真时间固定在同一平均窗口，而是识别动作特性一致或近似一致的连续时间段，在每段内建立GSSA方程。核心输入包括PWM调制信息、开关周期、待保留的傅里叶阶数以及分段判据；核心输出是各分段内状态变量的傅里叶系数及其重构的电压、电流等电气量。保留阶数越高，越接近开关细节；只保留最低阶时，则退化为传统状态空间平均意义上的模型。

### 3. 验证、优势与不足

从给出的原文摘要和引言看，作者通过算例仿真验证所提P-GSSA模型，并将其定位为适用于大规模新能源并网变流器电磁暂态高效仿真的平均化模型。原文明确讨论的基线主要是几类平均化建模方法：动态相量法、电路平均法、状态空间平均法、分段状态空间平均法和广义状态空间平均法。比较逻辑是：动态相量偏毫秒级，难以覆盖微秒级电磁暂态；电路平均难以反映内部开关高频动作；状态空间平均保留基频特性但谐波能力不足；GSSA可保留谐波但阶数高时模型复杂；P-GSSA试图在GSSA中加入分段，以避免全时段高复杂度建模。优势主要体现在建模机制上：按动作特性相近的时段组织GSSA，可实现多时间尺度表达，并在必要时保留高频分量。需要注意的是，当前提供的原文片段未给出可核验的误差百分比、加速比、步长设置、算例拓扑细节或仿真平台结果图表，因此不能把“高效”解释为已有明确量化提升。从验证范围看，结论应限于作者算例覆盖的变流器和工况。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于把“平均化是否丢失开关细节”的问题转化为“在哪些时间段、保留哪些傅里叶阶数才足够”的建模问题。它为电磁暂态仿真提供了一种介于详细开关模型和低阶平均模型之间的层级模型：对开关特性变化不剧烈的区间合并处理，对需要谐波信息的状态量用GSSA表达。该思路适合被后续用于新能源并网变流器等值建模、多时间尺度EMT仿真、平均模型与详细模型混合仿真、模型降阶和仿真加速方法页面复用。不适合直接外推到未验证的拓扑、复杂控制策略、强非周期开关、严重故障穿越或实时仿真硬件性能结论；这些都需要重新给出分段准则、保留阶数和误差验证。

## 方法细节

### 方法概述

本文提出分段广义状态空间平均法(Piecewise Generalized State Space Averaging, P-GSSA)，将分段技术与广义状态空间平均法(GSSA)有机结合。该方法通过分析PWM变流器开关函数的"近周期性"，将占空比相似、动作特性一致的连续开关周期合并为同一分段，在各分段时间内建立广义状态空间平均模型。该方法保留了状态变量的高频分量及谐波特性，同时通过分段变尺度建模显著降低计算负担，实现了电磁暂态仿真中精度与效率的平衡，适用于大规模新能源并网场景。

### 数学公式

**公式1**: $$$$S_i = \begin{cases} 1, & v_m > v_c \\ 0, & v_m \leq v_c \end{cases}, \quad i=a,b,c$$$$

*三相PWM变流器开关函数定义，基于正弦调制波vm与三角载波vc比较生成*

**公式2**: $$$$d_i = \frac{1}{T_s}\int_{t-T_s}^{t} S_i(\tau) d\tau$$$$

*周期性开关函数占空比计算，Ts为开关周期*

**公式3**: $$$$\dot{y}(t) = A_0 y(t) + b_0 + \sum_{i=1}^{m} (A_i y(t) + b_i)D_i(t)$$$$

*状态空间平均模型，对开关导通和关断时的线性状态方程进行加权平均*

**公式4**: $$$$\langle x \rangle_k(t) = \frac{1}{T}\int_{t-T}^{t} x(\tau)e^{-jk\omega_s \tau} d\tau$$$$

*k阶傅里叶系数定义，用于广义状态空间平均法将时域状态变量变换到频域*

**公式5**: $$$$\left\langle \frac{dx}{dt} \right\rangle_q = \frac{d\langle x \rangle_q}{dt} - jq\omega_s \langle x \rangle_q$$$$

*傅里叶变换微分性质，用于建立状态变量导数的频域方程*

**公式6**: $$$$\langle xy \rangle_q = \sum_{i=-\infty}^{\infty} \langle x \rangle_{q-i} \langle y \rangle_i$$$$

*傅里叶变换卷积性质（乘积规则），用于处理状态空间模型中的非线性乘积项*

### 算法步骤

1. 时间初始化与矫正：检查初始时刻T10与系统起始时刻ts0是否一致，若不一致则进行时间矫正ts0 = T10 + Δt

2. 输入信号定义：给定正弦调制波vm和三角载波vc，确定PWM调制方式

3. 仿真参数设定：给定系统总建模时长T和占空比方差限值ε1（阈值参数）

4. 开关时刻计算：基于PWM调制原理计算各相开关动作时刻tp（开通时刻）和tf（关断时刻）

5. 分段初始化：设置分段计数器n=1，初始开关周期数r=2，准备进行占空比相似性分析

6. 占空比方差计算：计算前r个开关周期占空比的方差σr，评估开关函数的"近周期性"强度

7. 近周期性判断：若方差σr ≥ ε1，则增加周期数r=r+1并返回步骤6重新计算；若σr < ε1，则初步确定第n段时长Tn=(r-1)Ts

8. 工况变化检验：检查Tn时间段内变流器运行工况是否发生改变。若发生突变，设工况改变时刻为Tnk，将原Tn分割为Tnk-Tn0作为当前段，剩余时段新开分段

9. 工况恢复检查：判断系统工况是否恢复到之前状态，以确定是否需要调整后续分段策略

10. 终止判断：累计已分配时间ΣTn，若小于总时长T，则n=n+1返回步骤5开始下一段分析；否则结束循环，确定总分段数n和各段时间Tn

### 关键参数

- **ε1**: 占空比方差限值，用于判断连续开关周期是否具有"近周期性"的阈值标准

- **r**: 连续开关周期数，表示被合并的具有相似占空比的周期数量

- **Ts**: 开关周期（载波周期），PWM变流器的基本时间单位

- **Tn**: 第n个分段的时长，根据精度要求通常为1~3个载波周期

- **k**: 傅里叶级数阶数，广义状态空间平均中保留的谐波阶次，决定模型精度和复杂度

- **ωs**: 开关角频率，ωs=2π/Ts，用于傅里叶变换中的频率基准

## 量化发现

- 分段长度通常为1~3个载波周期(Ts)，在保证模型精度的前提下最大化计算效率
- 传统电磁暂态仿真需将步长设为开关周期的1/10甚至更小，而P-GSSA方法通过分段变尺度建模可采用更大仿真步长
- 当保留傅里叶阶数k较低时，模型可有效提升仿真效率；当k=0（仅保留基频分量）时，广义状态空间平均法退化为传统状态空间平均法
- 占空比方差阈值ε1决定分段合并的严格程度，方差小于ε1的连续r个周期被合并为一段
- 模型阶数随保留的傅里叶系数阶数增加而增加，需权衡精度要求与计算复杂度

## 关键公式

### k阶傅里叶系数定义式

$$$$\langle x \rangle_k(t) = \frac{1}{T}\int_{t-T}^{t} x(\tau)e^{-jk\omega_s \tau} d\tau$$$$

*广义状态空间平均法的核心，用于将时域状态变量x(t)变换为频域系数，建立多谐波状态空间模型*

### 广义状态空间平均状态方程

$$$$\dot{x}_k(t) = \langle f(x(t),u(t)) \rangle_k - jk\omega_s x_k(t)$$$$

*基于傅里叶微分性质建立的状态变量频域微分方程，x_k(t)为k阶傅里叶系数，用于替代原时域状态方程实现降维*
