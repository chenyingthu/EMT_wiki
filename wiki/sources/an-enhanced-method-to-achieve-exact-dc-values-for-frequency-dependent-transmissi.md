# An Enhanced Method to Achieve Exact DC Values for Frequency-dependent Transmission lines

**作者**: H.M.J., De, Silva
**年份**: 2023
**来源**: `07&08/De Silva and Liu - 2023 - An Enhanced Method to Achieve Exact DC Values for Frequency-dependent Transmission lines.pdf`

## 摘要

针对传统通用线路模型(ULM)在极低频段(接近0Hz)有理函数拟合时存在的直流值偏差、时域人工过冲、大残差/极点比导致的不稳定以及模型阶数膨胀等问题，提出一种无需额外约束优化的直流精确校正方法。该方法首先通过解析计算获取特征导纳矩阵和传播函数矩阵的精确直流理论值。随后，重构有理函数的数学表达形式，将直流值作为硬性约束直接嵌入残差计算过程中，确保在时拟合函数严格等于理论直流值。为提升低频段精度并抑制时域振荡，引入指数衰减型低频加权函数对0~1Hz频段进行加权拟合。最后，利用低频段模态延迟差异可忽略的物理特性，对传播函数实施极点聚合与阶数缩减，并结合改进的递归卷积算法处理大残差/极点比问题，在保障全频段(1mHz~1MHz)拟合精度的同时显著降低时域仿真计算量。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自HVDC电缆和架空线路的EMT暂态仿真：模型既要覆盖直流稳态，又要覆盖开关、雷击、故障等直到MHz量级的宽频响应。研究对象是通用线路模型（ULM）/相域频变线路模型中两类关键频域函数：传播函数矩阵A(s)和特征导纳矩阵Yc(s)的有理函数逼近。难点在于，传统曲线拟合通常从几Hz开始，向0Hz外推时即使误差很小，也会在时域表现为错误直流值、人工过冲；若直接把拟合频带扩展到mHz，又可能带来大残差/极点比、稳定性和模型阶数问题。本文的贡献是修改有理函数表达与残差计算，使0Hz处强制等于解析直流值；再用低频加权改善近直流拟合，并对传播函数低频部分重排和降阶，以避免依赖额外约束优化。

### 2. 模型、算法与实现技术

本文没有提出新的线路物理方程，而是在ULM所需的频域函数拟合层改造实现。输入是由线路参数计算得到的Yc(s)、A(s)频率样本及其解析直流值；输出是可用于EMT Norton等值和递归卷积的有理函数极点、残差、常数项或重排后的传播函数项。核心机制是：在Yc(s)和A(s)已经写成有理函数后，不把0Hz当作普通拟合点，而是在残差或常数项求解过程中直接嵌入“s=0时等于解析dc值”的代数条件。这样时域递归卷积调用的函数本身在直流处就是正确的，而不是靠增加低频采样点近似逼近。低频加权函数用于提高mHz到Hz范围样本在最小二乘拟合中的影响，目的是减少近直流误差导致的时域过冲。传播函数部分还利用低频区传播延迟差异较小的特性进行函数重排和阶数削减，从而降低每个时间步递归卷积需要维护的历史项数量。

### 3. 验证、优势与不足

作者用地下电缆和架空线路算例验证方法；原文摘要和引言明确提到通过频域特性比较展示直流校正效果，并通过HVDC线路开路、短路条件下的时域仿真检验响应。对比基线是传统ULM频变线路曲线拟合方法，以及文献中已有的直流校正、两阶段低频拟合和低频补偿思路。指标主要包括：0Hz直流值是否精确、低频拟合是否改善、时域直流响应是否避免错误稳态和人工过冲、传播函数阶数及计算量是否下降。优势在于它把直流约束写进有理函数构造本身，不需要传播函数高频误差修正所依赖的额外约束优化，因此实现路径更直接。需要注意的是，提供的原文片段未报告可核验的数值结果，例如误差百分比、降阶比例、运行时间或具体软件平台；因此不能据此声称固定的精度提升或计算加速。验证范围也限于作者算例中的地下电缆、架空线和开短路工况。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值是指出：宽频曲线拟合“低频很准”并不等同于“直流正确”，EMT中一个接近0Hz的微小频域偏差也可能被长期暂态放大为错误稳态或人工过冲。它适合被后续频变线路模型、ULM实现、HVDC电缆暂态仿真、相域有理函数拟合和递归卷积实现页面复用，尤其用于说明如何把解析dc约束嵌入拟合函数。它不适合外推为所有线路拓扑、所有故障类型、实时仿真步长或含复杂换流器控制系统下都必然稳定有效。

## 方法细节

### 方法概述

针对传统通用线路模型(ULM)在极低频段(接近0Hz)有理函数拟合时存在的直流值偏差、时域人工过冲、大残差/极点比导致的不稳定以及模型阶数膨胀等问题，提出一种无需额外约束优化的直流精确校正方法。该方法首先通过解析计算获取特征导纳矩阵$Y_c(s)$和传播函数矩阵$A(s)$的精确直流理论值。随后，重构有理函数的数学表达形式，将直流值作为硬性约束直接嵌入残差计算过程中，确保在$s=0$时拟合函数严格等于理论直流值。为提升低频段精度并抑制时域振荡，引入指数衰减型低频加权函数对0~1Hz频段进行加权拟合。最后，利用低频段模态延迟差异可忽略的物理特性，对传播函数实施极点聚合与阶数缩减，并结合改进的递归卷积算法处理大残差/极点比问题，在保障全频段(1mHz~1MHz)拟合精度的同时显著降低时域仿真计算量。

### 数学公式

**公式1**: $$$Y_{c_{i,j}}(s=0) = Y_{c_{dc_{i,j}}} = \sum_{q=1}^{M} \frac{c_q}{-a_q} + d$$$

*特征导纳在0Hz的直流约束条件，用于建立常数项d与残差、极点的关系*

**公式2**: $$$d = \sum_{q=1}^{M} \frac{c_q}{a_q} + Y_{c_{dc_{i,j}}}$$$

*重构后的常数项计算公式，确保拟合函数在s=0时精确等于解析直流值*

**公式3**: $$$Y_{c_{i,j}}(s) = \sum_{q=1}^{M} c_q \left( \frac{1}{s-a_q} + \frac{1}{a_q} \right) + Y_{c_{dc_{i,j}}}$$$

*修正后的特征导纳有理函数表达式，直接强制直流精确*

**公式4**: $$$A_{dc_{i,j}} = \sum_{p=1}^{N_g} \sum_{i=1}^{N(p)} \frac{c_{pi}}{-a_{pi}}$$$

*传播函数在0Hz的直流约束条件*

**公式5**: $$$c_{11} = \sum_{p=1}^{N_g} \sum_{i=q}^{N(p)} \frac{-c_{pi} a_{11}}{a_{pi}} - a_{11} A_{dc_{i,j}}$$$

*传播函数首个残差反推公式，用于在不改变函数形式的前提下强制直流精确*

**公式6**: $$$W(f) = (W_{max} - 1)e^{-f\beta} + 1$$$

*低频加权函数，用于在拟合过程中动态提升低频段权重*

### 算法步骤

1. 步骤1：基于线路几何与材料参数，解析计算特征导纳矩阵$Y_c(s)$和传播函数矩阵$A(s)$在0Hz的精确直流理论值$Y_{c_{dc}}$与$A_{dc}$。

2. 步骤2：使用矢量拟合(Vector Fitting)算法在宽频带(如1mHz~1MHz)内提取$Y_c(s)$的公共极点$a_q$和$A(s)$的模态极点$a_{pi}$。

3. 步骤3：针对$Y_c(s)$，利用最小二乘法计算各极点残差$c_q$，代入公式$d = \sum_{q=1}^{M} \frac{c_q}{a_q} + Y_{c_{dc_{i,j}}}$计算常数项，重构为$Y_{c_{i,j}}(s) = \sum_{q=1}^{M} c_q \left( \frac{1}{s-a_q} + \frac{1}{a_q} \right) + Y_{c_{dc_{i,j}}}$以强制直流值。

4. 步骤4：针对$A(s)$，利用最小二乘法计算除首个残差外的其他残差$c_{pi}$，通过公式$c_{11} = \sum_{p=1}^{N_g} \sum_{i=q}^{N(p)} \frac{-c_{pi} a_{11}}{a_{pi}} - a_{11} A_{dc_{i,j}}$反推首个残差，重构传播函数表达式，确保直流精确且不引入高频偏差。

5. 步骤5：在拟合优化过程中应用加权函数$W(f) = (W_{max} - 1)e^{-f\beta} + 1$，对0~1Hz频段施加高权重，提升低频逼近精度并抑制时域人工过冲。

6. 步骤6：检查低频段相近极点，将其聚合至第一模态并移除冗余极点，实施阶数缩减策略以降低模型复杂度。

7. 步骤7：采用改进的两段插值递归卷积算法进行时域求解，有效处理残差/极点比大于1e6的极端情况，确保数值稳定性。

### 关键参数

- **最大加权因子_Wmax**: 100

- **衰减常数_β**: 1.0

- **拟合起始频率**: 0.001 Hz (1 mHz)

- **拟合终止频率**: 1.0 MHz

- **最大允许拟合误差**: 0.1%

- **接地电阻率**: 100 Ω·m

- **短路测试接地电阻**: 1 mΩ

## 关键公式

### 特征导纳直流校正公式

$$$Y_{c_{i,j}}(s) = \sum_{q=1}^{M} c_q \left( \frac{1}{s-a_q} + \frac{1}{a_q} \right) + Y_{c_{dc_{i,j}}}$$$

*用于重构$Y_c(s)$有理函数，通过调整常数项强制$s=0$时等于解析直流值，避免额外优化*

### 传播函数首残差反推公式

$$$c_{11} = \sum_{p=1}^{N_g} \sum_{i=q}^{N(p)} \frac{-c_{pi} a_{11}}{a_{pi}} - a_{11} A_{dc_{i,j}}$$$

*在已知其他残差和精确直流值$A_{dc}$的情况下，计算首个残差以保证传播函数直流精确且不改变高频渐近特性*

### 低频加权函数

$$$W(f) = (W_{max} - 1)e^{-f\beta} + 1$$$

*在矢量拟合过程中动态调整频率权重，提升0~1Hz频段拟合精度并抑制时域过冲*
