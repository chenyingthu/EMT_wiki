## 方法细节

### 方法概述

针对传统通用线路模型(ULM)在极低频段(接近0Hz)有理函数拟合时存在的直流值偏差、时域人工过冲、大残差/极点比导致的不稳定以及模型阶数膨胀等问题，提出一种无需额外约束优化的直流精确校正方法。该方法首先通过解析计算获取特征导纳矩阵$Y_c(s)$和传播函数矩阵$A(s)$的精确直流理论值。随后，重构有理函数的数学表达形式，将直流值作为硬性约束直接嵌入残差计算过程中，确保在$s=0$时拟合函数严格等于理论直流值。为提升低频段精度并抑制时域振荡，引入指数衰减型低频加权函数对0~1Hz频段进行加权拟合。最后，利用低频段模态延迟差异可忽略的物理特性，对传播函数实施极点聚合与阶数缩减，并结合改进的递归卷积算法处理大残差/极点比问题，在保障全频段(1mHz~1MHz)拟合精度的同时显著降低时域仿真计算量。

### 数学公式


**公式1**: $$$Y_{c_{i,j}}(s=0) = Y_{c_{dc_{i,j}}} = \sum_{q=1}^{M} \frac{c_q}{-a_q} + d$$$

*特征导纳在0Hz的直流约束条件，用于建立常数项d与残差、极点的关系*


**公式2**: $$$d = \sum_{q=1}^{M} \frac{c_q}{a_q} + Y_{c_{dc_{i,j}}}$$$

*重构后的常数项计算公式，确保拟合函数在s=0时精确等于解析直流值*


**公式3**: $$$Y_{c_{i,j}}(s) = \sum_{q=1}^{M} c_q \left( \frac{1}{s-a_q} + \frac{1}{a_q} \right) + Y_{c_{dc_{i,j}}}$$$

*修正后的特征导纳有理函数表达式，直接强制直流精确*


**公式4**: $$$A_{dc_{i,j}} = \sum_{p=1}^{N_g} \sum_{i=1}^{N(p)} \frac{c_{pi}}{-a_{pi}}$$$

*传播函数在0Hz的直流约束条件*


**公式5**: $$$c_{11} = \sum_{p=1}^{N_g} \sum_{i=q}^{N(p)} \frac{-c_{pi} a_{11}}{a_{pi}} - a_{11} A_{dc_{i,j}}$$$

*传播函数首个残差反推公式，用于在不改变函数形式的前提下强制直流精确*


**公式6**: $$$W(f) = (W_{max} - 1)e^{-f\beta} + 1$$$

*低频加权函数，用于在拟合过程中动态提升低频段权重*


### 算法步骤

1. 步骤1：基于线路几何与材料参数，解析计算特征导纳矩阵$Y_c(s)$和传播函数矩阵$A(s)$在0Hz的精确直流理论值$Y_{c_{dc}}$与$A_{dc}$。

2. 步骤2：使用矢量拟合(Vector Fitting)算法在宽频带(如1mHz~1MHz)内提取$Y_c(s)$的公共极点$a_q$和$A(s)$的模态极点$a_{pi}$。

3. 步骤3：针对$Y_c(s)$，利用最小二乘法计算各极点残差$c_q$，代入公式$d = \sum_{q=1}^{M} \frac{c_q}{a_q} + Y_{c_{dc_{i,j}}}$计算常数项，重构为$Y_{c_{i,j}}(s) = \sum_{q=1}^{M} c_q \left( \frac{1}{s-a_q} + \frac{1}{a_q} \right) + Y_{c_{dc_{i,j}}}$以强制直流值。

4. 步骤4：针对$A(s)$，利用最小二乘法计算除首个残差外的其他残差$c_{pi}$，通过公式$c_{11} = \sum_{p=1}^{N_g} \sum_{i=q}^{N(p)} \frac{-c_{pi} a_{11}}{a_{pi}} - a_{11} A_{dc_{i,j}}$反推首个残差，重构传播函数表达式，确保直流精确且不引入高频偏差。

5. 步骤5：在拟合优化过程中应用加权函数$W(f) = (W_{max} - 1)e^{-f\beta} + 1$，对0~1Hz频段施加高权重，提升低频逼近精度并抑制时域人工过冲。

6. 步骤6：检查低频段相近极点，将其聚合至第一模态并移除冗余极点，实施阶数缩减策略以降低模型复杂度。

7. 步骤7：采用改进的两段插值递归卷积算法进行时域求解，有效处理残差/极点比大于1e6的极端情况，确保数值稳定性。


### 关键参数

- **最大加权因子_Wmax**: 100

- **衰减常数_β**: 1.0

- **拟合起始频率**: 0.001 Hz (1 mHz)

- **拟合终止频率**: 1.0 MHz

- **最大允许拟合误差**: 0.1%

- **接地电阻率**: 100 Ω·m

- **短路测试接地电阻**: 1 mΩ



## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| 100km HVDC地下电缆短路测试 | 未校正时，直流短路电流随拟合起始频率变化显著(1Hz时为117.08kA，0.1Hz时为91.28kA，0.01Hz时为87.92kA，0.001Hz时为88.08kA)；采用新方法后，无论起始频率如何，直流电流均精确收敛至理论值88.04kA，且传播函数阶数从36降至30。 | 直流误差从最高33.0%降至0%，模型阶数降低约16.7%，计算负担显著下降 |

## 量化发现

- 直流值校正后，时域仿真稳态误差严格为0%，彻底消除传统方法因低频拟合偏差导致的直流漂移
- 低频加权拟合使0~1Hz频段幅频特性误差显著降低，有效抑制了时域响应中的人工过冲与数值振荡
- 传播函数阶数缩减策略使模型阶数降低约15%~20%（如从36阶降至30阶），直接减少递归卷积计算量
- 改进的递归卷积算法可稳定处理残差/极点比大于1e6的极端情况，确保时域仿真绝对稳定
- 全频段(1mHz~1MHz)最大拟合误差控制在0.1%以内，高频段特性与传统方法完全一致


## 关键公式

### 特征导纳直流校正公式

$$$Y_{c_{i,j}}(s) = \sum_{q=1}^{M} c_q \left( \frac{1}{s-a_q} + \frac{1}{a_q} \right) + Y_{c_{dc_{i,j}}}$$$

*用于重构$Y_c(s)$有理函数，通过调整常数项强制$s=0$时等于解析直流值，避免额外优化*

### 传播函数首残差反推公式

$$$c_{11} = \sum_{p=1}^{N_g} \sum_{i=q}^{N(p)} \frac{-c_{pi} a_{11}}{a_{pi}} - a_{11} A_{dc_{i,j}}$$$

*在已知其他残差和精确直流值$A_{dc}$的情况下，计算首个残差以保证传播函数直流精确且不改变高频渐近特性*

### 低频加权函数

$$$W(f) = (W_{max} - 1)e^{-f\beta} + 1$$$

*在矢量拟合过程中动态调整频率权重，提升0~1Hz频段拟合精度并抑制时域过冲*



## 验证详情

- **验证方式**: 频域特性对比与时域开短路暂态仿真验证
- **测试系统**: 100km HVDC地下电缆系统、20km HVDC电缆系统、100km交直流并行架空线系统
- **仿真工具**: 基于通用线路模型(ULM)框架的EMT仿真软件（兼容PSCAD/EMTDC等商业软件架构）
- **验证结果**: 频域验证表明校正后的$Y_c(s)$和$A(s)$在1mHz~1MHz全频段与理论曲线高度吻合，误差<0.1%；时域开短路测试证实直流响应精确收敛至解析值，无过冲或振荡，且模型阶数降低、计算效率提升，大残差/极点比下仿真保持稳定。
