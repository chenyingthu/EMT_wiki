## 摘要

This paper proposes a graph based matrix level partitioning methodology for parallel electromagnetic transient (EMT) simulation. By partitioning the graph associated with the admittance matrix, an efficient bordered block diagonal (BBD) matrix form is obtained, which is particularly suitable for parallel implementation. ## 核心贡献



- 提出基于图论的矩阵级划分方法，自动将导纳矩阵转化为加边块对角形式
- 设计适配BBD结构的分块LU分解算法，实现节点方程的粗粒度并行求解
- 在共享内存架构实现全自动并行求解器，无需人工干预且不依赖线路传播延时
