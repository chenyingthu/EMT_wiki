# 30/TPWRD.2017.2714639

**作者**: T420
**年份**: 2017
**来源**: `30/TPWRD.2017.2714639.pdf.pdf`

## 摘要

本文提出一种基于图论的矩阵级全自动划分方法，用于共享内存架构下的并行电磁暂态（EMT）仿真。核心思想是避开传统基于电路拓扑或行波传播延时的划分局限，直接从修正节点导纳矩阵的稀疏结构出发构建无向图。利用METIS多级k路图划分算法，将图分割为多个内部块与一个边界块，进而通过行列置换将导纳矩阵重组为加边块对角（BBD）形式。为提升分解效率，在块内引入近似最小度（AMD）排序算法以最小化LU分解过程中的填充元。针对BBD结构设计专用的分块LU分解策略，将全局矩阵分解为具有特定稀疏模式的L和U矩阵。在共享内存MIMD架构下，利用统一地址空间实现线程间高效数据共享，将内部块分解、边界块更新、前代与回代过程映射为粗粒度并行任务。同时，将网络元件按类型聚类并均匀分配至各线程，实现历史项计算与伴随电路更新的负载均衡，最终达成无需人工干预的全自动并行求解。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求来自大规模EMT仿真计算时间过长，而处理器性能提升已从单核主频转向多核并行。研究对象是基于修正节点分析形成的EMT网络导纳矩阵及其每个时间步的线性方程求解，而不是某一类特定设备模型。难点在于传统并行EMT常依赖分布参数线路的行波传播延时来天然解耦；当系统没有长线路、线路很短、只有电缆或集中参数线路时，这种解耦不存在，强行加入stub线路或一时间步延迟又可能带来误差甚至数值不稳定。已有diakoptics类方法虽不依赖行波延时，但多在电路拓扑层面分割，物理意义清晰但自动化困难，往往需要用户干预。本文的贡献是把划分对象从电路网络转移到导纳矩阵的稀疏图：对矩阵关联图进行自动图划分，将矩阵重排为bordered block diagonal（BBD）形式，再利用BBD结构组织分块LU与粗粒度并行求解，从而在共享内存多核计算机上实现不依赖传输线延时的自动并行EMT仿真。

### 2. 模型、算法与实现技术

本文核心算法以EMT每步的MNA方程Ax=b为入口，其中A是由网络伴随模型形成的导纳矩阵，x为节点电压及可能的附加未知量，b为电源和历史项构成的右端项。算法先根据A的非零结构构造无向图：矩阵变量对应图节点，非零耦合对应边；若矩阵非对称，则可用对称化的结构来表达耦合关系。随后用图划分把变量分成若干内部块和边界变量，使内部块之间没有直接耦合，只通过边界块相连。按该划分对A做行列置换后得到BBD矩阵：对角块Aii代表可独立求解的子系统，边界行列代表子系统间接口。计算机制上，先并行分解各Aii=LiiUii，再由这些分解计算内部块与边界块的耦合项，最终对边界Schur补进行分解。前代/回代也按同一结构展开：内部块可并行处理，边界变量在同步后求解，再回代得到各内部变量。实现上面向MIMD共享内存架构，线程通过统一地址空间访问矩阵和右端项，避免显式消息传递；论文还说明对网络元件计算进行并行化，用于每步历史项和伴随电路更新。

### 3. 验证、优势与不足

作者在自研共享内存并行EMT程序中验证方法，原文证据明确给出两类可扩展测试系统：一是包含许多分布参数级联段的地下电缆系统，二是多个IEEE 14节点系统互联形成的系统，且第二类只使用集中参数线路模型。验证平台为共享内存多核计算机，扩展性测试最高到64核。对比基线主要是串行或较少核心执行下的同一EMT求解流程，以及传统依赖传输线行波延时的并行分割思想；原文摘要只称仿真可被显著加速，但在当前提供文本中未报告可核验的加速比、效率、运行时间或误差数值。优势主要体现在三点：第一，不要求系统中存在传播时间大于步长的分布参数线路，因此适合短线路、电缆、微电网、船舶电网或电力电子密集系统；第二，划分发生在矩阵稀疏结构层面，较电路级diakoptics更易自动化；第三，BBD形式把大规模线性求解转化为多个内部块和一个边界问题，适合共享内存下粗粒度并行。边界也很明确：验证只覆盖文中两个可扩展算例和最高64核共享内存平台，未从当前证据看到分布式内存、GPU、实时仿真步长、强非线性开关事件负载变化、与商业EMT软件或其他自动划分器的系统性数值对比。

### 4. 价值、认知与可复用场景

这项工作的关键认知是：EMT并行化不必先寻找电气网络中的“物理切割点”或依赖传输线延时，而可以把每步线性方程的稀疏耦合结构作为并行性的来源。它能解决无长线路、短电缆或集中参数网络中难以自动分区的问题，并为后续矩阵重排、Schur补、分块LU、共享内存线程调度等页面提供方法入口。工程上适合复用于大规模离线EMT仿真器、自动网络分区模块、多核CPU线性求解内核，以及需要避免人工stub线路或人为延迟的场景。不宜外推为所有EMT模型都能获得同等加速，也不应据此推断其在分布式集群、GPU、硬实时仿真或任意非线性控制系统中同样有效。

## 方法细节

### 方法概述

本文提出一种基于图论的矩阵级全自动划分方法，用于共享内存架构下的并行电磁暂态（EMT）仿真。核心思想是避开传统基于电路拓扑或行波传播延时的划分局限，直接从修正节点导纳矩阵的稀疏结构出发构建无向图。利用METIS多级k路图划分算法，将图分割为多个内部块与一个边界块，进而通过行列置换将导纳矩阵重组为加边块对角（BBD）形式。为提升分解效率，在块内引入近似最小度（AMD）排序算法以最小化LU分解过程中的填充元。针对BBD结构设计专用的分块LU分解策略，将全局矩阵分解为具有特定稀疏模式的L和U矩阵。在共享内存MIMD架构下，利用统一地址空间实现线程间高效数据共享，将内部块分解、边界块更新、前代与回代过程映射为粗粒度并行任务。同时，将网络元件按类型聚类并均匀分配至各线程，实现历史项计算与伴随电路更新的负载均衡，最终达成无需人工干预的全自动并行求解。

### 数学公式

**公式1**: $$$Ax = b$$$

*修正节点分析（MNA）方程，描述网络节点电压/电流未知向量x与已知源向量b的线性关系*

**公式2**: $$$A = LU$$$

*全局稀疏矩阵的LU分解，将导纳矩阵分解为下三角矩阵L与上三角矩阵U*

**公式3**: $$$A_{ii} = L_{ii}U_{ii}, \quad i=1\dots m$$$

*对角块独立LU分解，各内部子矩阵可完全并行计算*

**公式4**: $$$L_{ni} = A_{ni}U_{ii}^{-1}, \quad U_{in} = L_{ii}^{-1}A_{in}$$$

*边界子矩阵计算，利用已分解的对角块求解内部块与边界块的耦合项*

**公式5**: $$$A_{nn} - \sum_{i=1}^m L_{ni}U_{in} = L_{nn}U_{nn}$$$

*边界块（Schur补）的LU分解，需同步所有内部块计算结果后串行执行*

**公式6**: $$$Ly = b, \quad Ux = y$$$

*前代与回代方程，用于求解中间变量y及最终节点状态x*

### 算法步骤

1. 图构建：根据修正导纳矩阵A的非零元素构建无向图，若A_ij或A_ji非零，则在节点i与j间添加边；若矩阵非对称，则使用A+A^T生成图以保证无向性。

2. 图划分：调用METIS多级k路划分算法，将图划分为m个内部子图与1个边界子图，确保内部块间无直接连接，仅通过边界节点耦合，且各块规模尽可能均衡。

3. 矩阵重排：根据划分结果对A进行行列置换，形成BBD结构；随后在各对角块内应用近似最小度（AMD）排序算法，优化非零元分布以减少后续LU分解的填充元。

4. 分块LU分解：并行计算各对角块A_ii的LU分解得到L_ii, U_ii；利用公式并行计算边界子矩阵L_ni, U_in；设置同步屏障，待所有内部块完成后，串行计算边界块L_nn, U_nn。

5. 并行前代回代：并行求解各块的前代方程L_ii*y_i = b_i；同步后计算边界中间变量y_n；并行执行回代U_ii*x_i = y_i - U_in*x_n，最终同步求解边界变量x_n。

6. 状态更新与负载均衡：将网络元件按类型（电容、电感、线路等）分组并均匀分配至各线程，利用已求得的节点电压并行更新历史电流源与伴随电路参数，推进至下一时间步。

## 关键公式

### 对角块独立LU分解

$$$A_{ii} = L_{ii}U_{ii}, \quad i=1\dots m$$$

*用于并行计算各内部子矩阵的分解，各块互不依赖，是粗粒度并行的核心*

### 边界块Schur补分解

$$$A_{nn} - \sum_{i=1}^m L_{ni}U_{in} = L_{nn}U_{nn}$$$

*在所有内部块分解完成后串行执行，处理块间耦合关系，需设置同步点*

### 并行回代方程

$$$U_{ii}x_i = y_i - U_{in}x_n, \quad i=1\dots m$$$

*在求得边界变量x_n后，各内部块可独立并行回代求解节点电压*
