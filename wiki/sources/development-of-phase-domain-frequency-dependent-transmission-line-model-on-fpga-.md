# Development of phase domain frequency-dependent transmission line model on FPGA for real-time digital simulator

**作者**: Jiadai Liu
**年份**: 2021
**来源**: `13&14/files/J.EPSR.2021.107305.pdf-1.pdf`

## 摘要

本文提出一种基于FPGA的频变相域（FDPD）输电线路实时仿真模型。该方法直接在相域进行计算，避免了传统模域模型中恒定变换矩阵在非对称线路中引入的误差。模型基于诺顿等效电路，将频域特性通过矢量拟合转化为时域递归卷积形式。为突破实时仿真步长限制，硬件架构采用全流水线与并行计算设计，将矩阵参数按“并行维度（列）”与“流水线维度（行）”存储，实现多路乘加运算的同步执行。针对FPGA资源与精度的矛盾，设计了48位自定义浮点数据格式，有效抑制长卷积累加过程中的舍入误差累积。模型通过Bergeron行波接口或嵌入式模式与RTDS NovaCor实时仿真器交互，支持微秒级步长的高频暂态分析。

## 研究解读

### 1. 需求、对象、挑战与贡献

实时数字仿真器中的输电线路模型必须在固定仿真步长内完成计算，同时还要保留线路耦合、传播延迟和频率相关参数，否则会低估暂态过电压、过电流和高频行波效应。本文研究对象是EMT实时仿真中的多导体频变相域输电线路模型。难点在于：相域频变模型避免了模域常数变换矩阵对非对称、未换位线路带来的误差，但它需要处理矩阵卷积、传播延迟和多极点递归状态，计算量和存储访问压力远高于常参数或模域模型。本文贡献不是简单把模型搬到FPGA，而是将FDPD线路的诺顿等效、特征导纳卷积和传播矩阵卷积组织成可流水、可并行的硬件计算结构，并采用自定义48位浮点格式改善长时间递归卷积中的舍入误差问题，使该相域频变模型能够接入实时数字仿真网络。

### 2. 模型、算法与实现技术

模型以线路两端的相电压、注入电流和历史电流为接口量，用诺顿等效把线路表示为当前端口电压经等效导纳矩阵作用后减去历史项：i_k(t)=G*v_k(t)-i_histk(t)。历史项由两部分构成：一是特征导纳Y_c对上一时刻端电压的卷积，二是传播矩阵H对对端反射电流经延迟后的卷积，即i_histk(t)=Y_c*v_k(t-Δt)-2H*i_mr(t-Δt-τ)。频域函数通过矢量拟合转为极点—留数形式后，卷积不再直接积分，而用一阶递归状态量更新，例如x_Yc(t)=α_Yc x_Yc(t-Δt)+v_k(t-Δt)，再由c_Yc x_Yc(t)得到卷积输出。硬件实现上，矩阵系数、极点、留数和状态量按适合列并行、行流水的方式存入FPGA双端口RAM；每个时间步内并行更新多个状态、执行矩阵—向量乘加、形成历史电流和诺顿注入电流。模型可通过Bergeron行波接口与RTDS NovaCor主网交换端口量，也可嵌入网络求解流程以减少接口引入的响应偏差。

### 3. 验证、优势与不足

页面记录的验证包括离线软件对比和实时硬件在环测试：测试对象覆盖三相双电源输电系统故障、多导体架空线或电缆模型，以及频率响应扫描；工具和平台包括PSCAD/EMTDC离线基准、RTDS NovaCor实时主网求解器和Xilinx FPGA板卡。对比指标主要是暂态电压电流波形一致性、频率响应、实时步长、硬件资源和数值稳定性。优势体现在三点：第一，相域计算避免了模域频变模型中常数变换矩阵对非对称线路的潜在误差；第二，全流水和并行结构将复杂FDPD卷积计算压缩到微秒级实时步长；第三，48位自定义浮点格式用于抑制32位单精度在长递归卷积中可能出现的累积舍入误差。边界也很明确：验证仍局限于文中线路规模、极点/延迟组数、FPGA平台、接口方式和频率扫描范围；对更大导体数、更复杂电网拓扑、不同矢量拟合阶数、长期HIL闭环稳定性以及保护控制器实物联调的普适结论，不能仅由本文验证推出。

### 4. 价值、认知与可复用场景

这项工作的重要认知是：相域频变线路模型的实时化瓶颈不只在公式复杂，而在递归卷积状态、矩阵乘加和存储访问能否被重新组织成硬件友好的数据流。它可用于后续研究中讨论FPGA加速EMT元件模型、实时仿真中线路模型接口、非对称线路高频暂态建模、自定义浮点精度与资源折中等问题。工程上，它适合服务于需要较小步长的输电线路暂态、行波和电力电子相互作用研究。不适合把结论直接外推为所有线路模型、所有FPGA或所有实时仿真器都能达到相同精度和步长。

## 方法细节

### 方法概述

本文提出一种基于FPGA的频变相域（FDPD）输电线路实时仿真模型。该方法直接在相域进行计算，避免了传统模域模型中恒定变换矩阵在非对称线路中引入的误差。模型基于诺顿等效电路，将频域特性通过矢量拟合转化为时域递归卷积形式。为突破实时仿真步长限制，硬件架构采用全流水线与并行计算设计，将矩阵参数按“并行维度（列）”与“流水线维度（行）”存储，实现多路乘加运算的同步执行。针对FPGA资源与精度的矛盾，设计了48位自定义浮点数据格式，有效抑制长卷积累加过程中的舍入误差累积。模型通过Bergeron行波接口或嵌入式模式与RTDS NovaCor实时仿真器交互，支持微秒级步长的高频暂态分析。

### 数学公式

**公式1**: $$$i_k(t) = G * v_k(t) - i_{histk}(t)$$$

*线路k端诺顿等效电流方程，表示当前时刻注入电流由等效导纳矩阵与电压的卷积减去历史电流项构成。*

**公式2**: $$$i_{histk}(t) = Y_c * v_k(t - \Delta t) - 2H * i_{mr}(t - \Delta t - \tau)$$$

*历史电流项计算公式，包含特征导纳卷积与传播矩阵对反射电流的延迟作用。*

**公式3**: $$$x_{Yc}(t) = \alpha_{Yc} x_{Yc}(t - \Delta t) + v_k(t - \Delta t)$$$

*状态变量递归更新方程，用于将频域卷积转化为时域一阶差分形式，降低计算复杂度。*

**公式4**: $$$Y_c * v_k(t) = c_{Yc} x_{Yc}(t)$$$

*利用更新后的状态变量直接计算特征导纳卷积，避免直接时域积分。*

### 算法步骤

1. 参数初始化与存储：将线路的等效导纳矩阵G、特征导纳Yc、传播矩阵H及其对应的极点、留数、比例项按列优先顺序存入FPGA双端口RAM中，配置并行与流水线访问维度。

2. 状态变量更新：在每个仿真步长开始时，并行读取上一时刻的电压和反射电流，利用递归公式同步更新状态变量x_Yc和x_H，结果写回RAM。

3. 卷积与历史电流计算：将更新后的状态变量与系数矩阵c_Yc、c_H进行并行乘加运算，同时计算G_H*i_mr项，三者累加得到完整的历史电流向量。

4. 诺顿等效注入电流求解：读取当前步长网络求解器提供的端电压，与等效导纳矩阵G进行矩阵向量乘法，减去历史电流项，得到当前时刻的反射电流。

5. 接口数据交换：通过光纤接口将计算得到的注入电流发送至RTDS NovaCor主网求解器，并接收下一时刻的节点电压，完成闭环实时迭代。若采用嵌入式模式，则跳过Bergeron接口延迟，直接参与全网节点导纳矩阵求解。

### 关键参数

- **仿真时钟频率**: 100 MHz

- **数据格式**: 单精度浮点(32位) / 自定义高精度浮点(48位)

- **最大支持导体数(n)**: 8或12

- **最大极点/延迟组数(p/g)**: 20极点/6延迟组(8导体模型)，20极点/4延迟组(12导体模型)

- **接口通信速率**: 2.0 Gbps (8B/10B编码)

- **硬件平台**: Xilinx Virtex-7 VC707 (32位), UltraScale+ VCU118 (48位)

## 仿真结果

### 仿真测试

| 测试场景 | 结果描述 | 对比基线 |

|---------|---------|----------|

| Case I: 三相系统单相接地故障 | 对比大时间步长模型与FPGA小步长模型，两端电流与故障母线电压波形高度重合，验证了模型在暂态故障下的准确性。 | FPGA实现步长降至2.4µs（8导体）和2.8µs（12导体），波形与基准模型误差可忽略。 |

| Case II: 送端电压阶跃响应精度对比 | PSCAD离线结果、大时间步长模型与48位FPGA模型波形完全一致；32位单精度FPGA模型因卷积累加误差导致电流无法恢复至稳态值，出现明显发散。 | 48位模型步长为3.27µs，彻底消除32位模型的数值发散问题，精度与离线PSCAD一致。 |

| Case III: 1Hz-1000Hz频率响应扫描 | 带Bergeron接口的FPGA模型在高频段与处理器基准模型存在微小偏差；移除接口采用嵌入式模式后，频率响应曲线与基准模型完全吻合。 | 嵌入式模式步长增加至5.0µs（8导体）和5.8µs（12导体），换取全频段频率响应零偏差。 |

## 量化发现

- 全流水线并行架构将FDPD线路模型实时仿真步长压缩至2.4µs~3.27µs，较传统处理器大时间步长模型（约50µs）提升约15~20倍。
- 48位自定义浮点格式在VCU118上实现时，LUT占用率为48.39%，DSP48占用率为47.08%，在资源允许范围内彻底消除了32位单精度模型在长卷积中的数值发散问题。
- 采用Bergeron接口模式时，8导体与12导体模型时间步长分别为2.4µs和2.8µs；切换为无接口嵌入式模式后，步长分别增加至5.0µs和5.8µs，换取了全频段（1Hz-1000Hz）频率响应的零偏差。
- 硬件资源利用率在VC707上接近极限（LUT>91%，DSP48>90%），证明单精度设计已无法在同等步长下扩展至更高精度或更多导体数。

## 关键公式

### 诺顿等效注入电流方程

$$$i_k(t) = G * v_k(t) - i_{histk}(t)$$$

*用于实时求解线路端口的电流注入值，是连接线路模型与外部网络求解器的核心接口。*

### 状态变量递归更新公式

$$$x_{Yc}(t) = \alpha_{Yc} x_{Yc}(t - \Delta t) + v_k(t - \Delta t)$$$

*将频域卷积转化为时域差分，是FPGA流水线计算的核心，大幅降低乘加运算延迟。*

### 等效导纳矩阵表达式

$$$G = d + r Y_c \lambda Y_c$$$

*结合极点、留数与比例项构建线路的频变特性，用于计算当前时刻的等效导纳。*

## 验证详情

- **验证方式**: 离线软件对比仿真与实时硬件在环（HIL）测试
- **测试系统**: 三相双电源输电系统（含故障母线）、多导体架空线路/电缆系统
- **仿真工具**: PSCAD/EMTDC（离线基准）、RTDS NovaCor（实时主网求解器）、Xilinx FPGA（VC707/VCU118）
- **验证结果**: 时域故障暂态波形与基准模型误差可忽略；48位浮点设计有效解决32位模型的累加误差发散问题；频率扫描验证了Bergeron接口对高频响应的轻微影响及嵌入式模式的精确性；整体实现微秒级步长，满足高频电力电子与暂态过电压实时仿真需求。
