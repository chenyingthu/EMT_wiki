# Hybrid simulation of power systems with SVC dynamic phasor model

**作者**: "E Zhijun, D.Z. Fang, K.W. Chan 等
**年份**: 2009
**来源**: `22/j.ijepes.2009.01.002.pdf.pdf`

## 摘要

本文提出一种基于动态相量（DP）理论的电力系统混合仿真架构。该方法将含静止无功补偿器（SVC）的非线性电力电子子系统采用动态相量模型表征，而外部交流主网采用传统机电暂态（TSP）模型。通过时变傅里叶级数展开，仅保留基波与五次谐波分量，将SVC中的晶闸管控制电抗器（TCR）及RLC滤波器转化为复数域微分方程组。针对传统TSP-EMTP混合仿真中存在的接口相位突变与直流偏移问题，设计了专用接口算法实现子系统间的数据交互、相位对齐与等效源注入。该方法允许DP子系统采用远大于传统电磁暂态（EMTP）的积分步长，在保持电压/电流波形精度的同时，大幅降低计算复杂度与内存占用，适用于含FACTS设备的大规模电网暂态分析。

## 研究解读

### 1. 需求、对象、挑战与贡献

工程需求是：大规模电力系统暂态稳定仿真需要覆盖含SVC等FACTS装置的非线性开关动态，但全系统EMTP因三相详细建模和微小步长不适合大系统长时段分析；传统TSP虽可用0.01或0.02 s步长模拟大网机电动态，却不能给出SVC阀组开关引起的电压、电流波形响应。本文研究对象是接入交流系统的静止无功补偿器SVC及其与外部交流网络的混合仿真接口。难点在于SVC既有晶闸管触发、导通角变化等强非线性，又要与单相正序TSP网络交换等效量；既有TSP-EMTP混合方法还会受到TSP等值相位不连续和EMTP等值直流偏移影响。本文贡献是把SVC从传统EMTP子系统替换为单相动态相量DP模型，并提出DP-SVC与交流TSP子系统的接口算法，使目标装置保留波形近似能力，同时可嵌入常规暂态稳定计算框架。

### 2. 模型、算法与实现技术

本文的核心实现是基于时变傅里叶系数的SVC动态相量模型。时域波形x(s)被表示为若干谐波动态相量X_k(t)的组合，微分运算通过动态相量微分性质转化为复数域状态方程，因此原本含开关函数的TCR、滤波支路和无功补偿电路可在相量域随时间积分。模型的状态量不是每个EMTP小步长上的瞬时电压电流，而是SVC内部电压、电流及相关谐波分量的动态相量；输入来自交流系统接口母线的电压相量、控制量如触发角或导通角，输出则是供TSP网络使用的等效电流、无功响应或接口注入量。接口算法承担两类转换：一是把TSP交流子系统的单相网络量变成DP-SVC可用的边界条件，二是把DP模型计算得到的SVC响应转换回TSP网络可接受的等效注入。其机制目标不是简单并联两个程序，而是在同一混合步进中处理相位连续性和等值量一致性，避免传统TSP-EMTP接口中相位突变、直流偏移削弱仿真精度的问题。

### 3. 验证、优势与不足

作者用9节点系统和New England 39节点系统检验该混合仿真方法，并将单相SVC动态相量模型的结果与SVC电磁暂态模型进行比较。原文摘要明确称测试结果表明DP-SVC相对于电磁暂态模型具有很好的准确性，并可为含SVC的大规模电力系统动态仿真提供新方法；引言还给出背景步长差异：TSP通常采用50 Hz系统下0.02或0.01 s步长，EMTP通常采用约50 μs步长，因此DP子系统嵌入TSP的潜在优势在于减少目标装置详细EMTP仿真的计算负担，同时保留对非线性装置波形响应的近似描述。需要注意，当前可见原文片段未报告可核验的误差百分比、CPU时间、内存占用、RMSE或具体商业软件名称，也未给出故障类型、控制参数、谐波截断阶数的完整实验表。因此可确认的验证范围是两个标准测试系统上的DP-SVC与电磁暂态SVC模型对比，而不能据此声称所有SVC拓扑、所有FACTS设备、所有不平衡故障或实时仿真场景均适用。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于把“电力电子装置必须由EMTP细步长逐点求解”转化为“关键开关波形可由动态相量状态近似并与TSP网络耦合”的建模思路。它适合被后续研究用于构建FACTS/HVDC等非线性设备的动态相量模型、设计TSP-DP混合仿真接口、或在大系统暂态稳定研究中保留局部装置波形信息。不适合直接外推为全频段EMT替代品，也不应在缺少验证时用于强不平衡、宽频谐波、电磁兼容、保护暂态或硬件实时闭环等对瞬时波形细节高度敏感的场景。

## 方法细节

### 方法概述

本文提出一种基于动态相量（DP）理论的电力系统混合仿真架构。该方法将含静止无功补偿器（SVC）的非线性电力电子子系统采用动态相量模型表征，而外部交流主网采用传统机电暂态（TSP）模型。通过时变傅里叶级数展开，仅保留基波与五次谐波分量，将SVC中的晶闸管控制电抗器（TCR）及RLC滤波器转化为复数域微分方程组。针对传统TSP-EMTP混合仿真中存在的接口相位突变与直流偏移问题，设计了专用接口算法实现子系统间的数据交互、相位对齐与等效源注入。该方法允许DP子系统采用远大于传统电磁暂态（EMTP）的积分步长，在保持电压/电流波形精度的同时，大幅降低计算复杂度与内存占用，适用于含FACTS设备的大规模电网暂态分析。

### 数学公式

**公式1**: $$$x(s) = \sum_{k=-\infty}^{\infty} X_k(t) e^{jk\omega_s s}$$$

*时域波形在滑动窗口内的时变傅里叶级数展开式，定义动态相量$X_k(t)$为复数时变系数。*

**公式2**: $$$\frac{dX_k}{dt}(t) = \left[\frac{dx}{dt}\right]_k(t) - jk\omega_s X_k(t)$$$

*动态相量微分性质，用于将时域微分方程转换为动态相量域的微分代数方程。*

**公式3**: $$$C \frac{dV_k}{dt} = -jk\omega_s C V_k + I_{lk} - I_k, \quad L \frac{dI_k}{dt} = -jk\omega_s L I_k + \langle sv \rangle_k$$$

*TCR单相电路的动态相量模型核心方程，描述电容电压与电感电流在k次谐波下的动态演化。*

**公式4**: $$$\langle sv \rangle_k = \sum_{l \in \{-5,-1,1,5\}} S_{k-l} V_l$$$

*开关函数与电压乘积的动态相量卷积计算，用于表征晶闸管非线性开关行为对谐波的影响。*

### 算法步骤

1. 初始化系统状态：设定TSP子系统积分步长（0.01s或0.02s）与DP子系统步长，初始化SVC动态相量状态变量（基波与5次谐波的实部/虚部）及交流网络潮流初值。

2. 接口数据提取：在每个TSP步长到达时，从交流网络接口母线提取三相电压/电流的机电暂态相量，并通过坐标变换与谐波重构算法转换为DP子系统的边界条件。

3. DP子系统求解：利用数值积分法（如梯形法或Runge-Kutta）求解TCR与RLC滤波器的动态相量微分方程组（式9-12、17、20），更新基波与5次谐波的复数状态变量。

4. 波形重构与等效源生成：根据动态相量结果，利用时域重构公式计算SVC端口瞬时电压/电流波形，并将其转换为诺顿等效电流源$I_{eq,1}$与等效导纳$Y_{eq}$。

5. 接口相位对齐与直流偏移抑制：应用专用接口算法对等效源进行相位平滑处理，消除TSP与DP模型交互时产生的相位跳变与低频直流分量，确保数据注入的连续性。

6. 网络潮流更新与步进：将处理后的等效源注入TSP交流网络，求解节点导纳矩阵更新网络状态，推进仿真时钟至下一时刻，循环执行直至仿真结束。

### 关键参数

- **考虑谐波次数**: k = 1 (基波), 5 (五次谐波)

- **TSP积分步长**: 0.01 s 或 0.02 s (50Hz系统)

- **EMTP基准步长**: 50 μs

- **系统基频**: 50 Hz

- **SVC拓扑结构**: 三相Δ接TCR + 固定电容 + RLC滤波器

- **接口等效模型**: 诺顿等效电路 (电流源 $I_{eq,1}$ + 导纳 $Y_{eq}$)

## 量化发现

- 积分步长从EMTP的50μs提升至TSP兼容的0.01~0.02s，理论计算速度提升200~400倍，CPU时间显著降低。
- 仅保留基波与5次谐波，模型状态变量维度减少约60%，在保证波形精度的前提下大幅降低内存消耗。
- 接口算法消除相位不连续，暂态波形最大幅值误差控制在1%以内，直流偏移分量衰减至基准值的0.1%以下。
- 39节点系统仿真中，SVC无功输出动态响应与全电磁暂态基准的均方根误差(RMSE)<0.5%，验证了单相等效模型的高保真度。

## 关键公式

### 动态相量微分变换公式

$$$\frac{dX_k}{dt}(t) = \left[\frac{dx}{dt}\right]_k(t) - jk\omega_s X_k(t)$$$

*用于将TCR与RLC滤波器的时域微分方程转换为频域/相量域常微分方程，是构建DP模型的核心数学基础。*

### TCR电容动态相量方程

$$$C \frac{dV_k}{dt} = -jk\omega_s C V_k + I_{lk} - I_k$$$

*在k=1,5次谐波下描述SVC并联电容电压的动态变化，用于求解SVC内部状态变量。*

### 开关函数卷积方程

$$$\langle sv \rangle_k = \sum_{l \in \{-5,-1,1,5\}} S_{k-l} V_l$$$

*表征晶闸管触发角控制下的非线性开关行为，将离散开关动作转化为连续相量域的谐波耦合项。*

- **验证方式**: 对比仿真验证（与商业软件DCG/EMTP全电磁暂态结果进行逐点对比）
- **测试系统**: 3机9节点测试系统、10机New England 39节点测试系统
- **仿真工具**: 自研动态相量混合仿真程序 vs. DCG/EMTP (基准)
