# Approximate Voltage-Behind-Reactance Induction Machine Model for Efficient Interface With EMTP Network Solution

**作者**: Liwei Wang, Juri Jatskevich
**年份**: 2010
**来源**: `09/Wang和Jatskevich - 2010 - Approximate voltage-behind-reactance induction machine model for efficient interface with EMTP netwo.pdf`

## 摘要

本文提出近似电压后电抗（AVBR）感应电机模型，旨在解决传统相域（PD）和精确VBR模型因电导子矩阵随转速变化而导致EMTP网络导纳矩阵需每步重分解的计算瓶颈。方法首先基于隐式梯形积分法离散电机微分方程，构建电机-网络接口方程。通过分析发现，离散化PD模型的等效电阻矩阵在对称线性假设下实为常数。针对VBR模型，其等效电阻矩阵的非对角元素与转速相关，但理论证明其与对角元素的比值呈时间步长平方阶（$O(\Delta t^2)$）。因此，AVBR模型通过忽略这些微小的转速相关非对角系数，使等效电阻矩阵退化为常数对角阵，进而获得恒定电导子矩阵。该模型实现了电机电气/机械变量与EMTP网络的直接、非迭代同步求解，彻底避免了网络矩阵的重复LU分解。

## 研究解读

### 1. 需求、对象、挑战与贡献

这篇论文面对的是EMTP类电磁暂态仿真中的一个具体接口问题：感应电机既要以足够详细的电磁动态参与网络节点方程，又不能让电机模型破坏网络导纳矩阵可预分解、可快速重复求解的数值结构。研究对象是鼠笼式感应电机在EMTP网络中的相域接口模型，尤其是传统dq模型、相域耦合电路模型（PD）和电压后电抗模型（VBR）之间的取舍。难点在于：电力网络通常用abc物理相坐标建模，而电机内部转子动态常用dq坐标更高效；若用预测量把dq电机接入网络，可保持常数电导矩阵但会引入接口误差；若用直接相域耦合，可提高同步求解一致性，但转子位置或转速相关项会增加计算负担，甚至导致网络矩阵需要随时间步更新和重新因子分解。本文的贡献不是泛泛地“提高效率”，而是证明离散化PD模型可以构造成常数机器电导子矩阵，并进一步提出AVBR模型：在VBR离散接口中忽略转速相关系数，使机器对外呈现常数电导子矩阵，从而保留VBR直接接口和较好数值结构，同时避免每步重新分解全网G矩阵。

### 2. 模型、算法与实现技术

本文的核心实现思路是把感应电机写成适合EMTP节点分析的Norton型接口：外部网络看到的是机器端abc相电压、注入电流、等效电导子矩阵和历史电流源。PD模型部分的作用是说明，即使在相域耦合电路表达下，只要经过合适离散化和矩阵整理，机器对网络贡献的电导子矩阵可以是不随时间变化的，这为避免全网G矩阵重复因子分解提供理论依据。VBR模型则把定子仍放在abc相坐标，以便直接接入物理网络；转子子系统放在dq参考系，以避免完整相域模型中转子位置相关互感带来的结构复杂性。AVBR的关键机制是在离散VBR接口方程中识别出含转子速度的系数，并将这些速度相关项近似忽略；这样机器端口关系可整理成“常数电导矩阵×端电压 + 历史源”的形式。计算流程上，初始化时把机器常数电导子矩阵并入全网节点导纳矩阵并完成稀疏因子分解；每个时间步只更新由上一时刻电流、磁链、转速等形成的历史项，随后与网络方程同步求解端电压和电流，再更新电机电气及机械状态。关键不在于另行设计控制器，而在于改变电机-网络接口的代数结构，使直接同步求解不再以时变G矩阵为代价。

### 3. 验证、优势与不足

原文摘要说明作者通过case studies验证AVBR相对已有EMTP电机模型在数值准确性和效率上的改进；引言中列出的对比对象包括传统dq接口模型、PSCAD/EMTDC式预测Norton源、ATP通用机模型、EMTP-RV中可通过Newton迭代实现同步求解的机器模型，以及相域PD模型和已有VBR模型。依据提供的原文片段，可以确定的验证目标是：检查AVBR在保持常数机器电导子矩阵的同时，是否减少预测接口误差和避免每步网络矩阵重因子分解。优势主要体现在两个机制层面：第一，相比依赖前一步电压或预测电气量的dq接口，AVBR允许机器和网络变量在同一节点方程框架内直接耦合，因而理论上减少接口滞后误差；第二，相比精确但含速度相关导纳的接口，AVBR使机器G子矩阵常数化，适合EMTP中反复求解同一拓扑网络方程。需要注意的是，当前给出的原文证据未包含具体算例表、机器容量、步长、CPU时间或误差百分比，因此不能在此入口页把“显著改进”量化为可核验数字。从验证范围看，结论主要适用于线性感应机模型、EMTP型梯形积分/节点分析框架和论文所测试的暂态场景；对饱和、深槽效应、强非线性磁路、复杂控制驱动、故障穿越、多机大规模系统或实时仿真硬件上的性能，不能仅凭该片段外推。

### 4. 价值、认知与可复用场景

这项工作的主要认知价值在于指出：EMTP电机模型的效率瓶颈不只来自微分方程阶数，也来自机器端口等效导纳是否随时间变化；只要把电机离散方程整理成常数电导加历史源，就能显著改善与网络求解器的耦合方式。AVBR适合作为后续研究中“直接接口、常数G矩阵、避免预测误差”的建模模板，可被用于比较dq、PD、VBR类感应机模型，也可为同步机、变速机组或电机群的EMT接口设计提供思路。工程上，它适合关注电机启动、机电暂态与网络电磁暂态耦合且希望减少网络矩阵重分解开销的仿真程序。它不适合被直接外推为任意电机、任意非线性磁路或任意控制系统下都准确高效的通用结论；近似项被忽略后的误差仍应随具体参数、步长和工况重新评估。

## 方法细节

### 方法概述

本文提出近似电压后电抗（AVBR）感应电机模型，旨在解决传统相域（PD）和精确VBR模型因电导子矩阵随转速变化而导致EMTP网络导纳矩阵需每步重分解的计算瓶颈。方法首先基于隐式梯形积分法离散电机微分方程，构建电机-网络接口方程。通过分析发现，离散化PD模型的等效电阻矩阵在对称线性假设下实为常数。针对VBR模型，其等效电阻矩阵的非对角元素与转速相关，但理论证明其与对角元素的比值呈时间步长平方阶（$O(\Delta t^2)$）。因此，AVBR模型通过忽略这些微小的转速相关非对角系数，使等效电阻矩阵退化为常数对角阵，进而获得恒定电导子矩阵。该模型实现了电机电气/机械变量与EMTP网络的直接、非迭代同步求解，彻底避免了网络矩阵的重复LU分解。

### 数学公式

**公式1**: $$$G_{total} \mathbf{V} = \mathbf{I}_{hist}$$$

*EMTP全局网络节点方程，将电机等效电导子矩阵与历史电流源并入后直接求解节点电压*

**公式2**: $$$R_{eq}^{PD} = R_s + \frac{2}{\Delta t} L_{ss} - \frac{2}{\Delta t} L_{sr} \left(R_r + \frac{2}{\Delta t} L_{rr}\right)^{-1} L_{rs}$$$

*离散化相域模型的等效电阻矩阵，证明在对称绕组下为常数矩阵*

**公式3**: $$$R_{eq}^{VBR} = \text{diag}(d_j) + \text{off-diag}(k_j)$$$

*精确VBR模型等效电阻矩阵结构，其中非对角项$k_j$含转速依赖系数*

**公式4**: $$$R_{eq}^{AVBR} \approx \text{diag}(d_j)$$$

*AVBR近似等效电阻矩阵，通过忽略$k_j$获得常数对角阵*

**公式5**: $$$\frac{\|\mathbf{x} - \tilde{\mathbf{x}}\|}{\|\mathbf{x}\|} \leq \frac{\|R_{eq}^{-1}\| \|R_{eq} - \tilde{R}_{eq}\|}{1 - \|R_{eq}^{-1}\| \|R_{eq} - \tilde{R}_{eq}\|}$$$

*基于矩阵范数推导的近似误差上界，用于量化忽略非对角项引入的数值误差*

### 算法步骤

1. 采用隐式梯形积分法对感应电机的定子相坐标和转子dq坐标微分方程进行离散化，得到包含历史电流源的等效电路形式。

2. 推导VBR模型的等效电阻矩阵$R_{eq}$，识别其对角元素$d_j$（常数）与非对角元素$k_j$（含转速$\omega_r$和时间步长$\Delta t$）。

3. 进行量级分析，证明比值$|k_j/d_j|$与$\Delta t^2$成正比，在典型EMTP步长下矩阵呈现强对角占优特性。

4. 实施近似处理：将$R_{eq}$中的非对角元素$k_j$置零，得到仅含常数对角项的近似矩阵$R_{eq}^{AVBR}$。

5. 对$R_{eq}^{AVBR}$求逆，获得恒定不变的电机等效电导子矩阵$G_{eq}^{AVBR}$。

6. 将$G_{eq}^{AVBR}$与历史电流源直接叠加至EMTP全局网络导纳矩阵$G_{total}$中，利用稀疏矩阵技术一次性完成LU分解。

7. 在每个仿真步长内，直接求解节点电压方程，无需迭代或重分解，并同步更新转子机械运动方程与历史项。

## 量化发现

- AVBR模型单步计算耗时为1.9 μs，较PD模型（4.3 μs）提速约126%，较精确VBR模型（2.2 μs）提速13.6%。
- 非对角系数与对角系数比值$|k_j/d_j|$呈$O(\Delta t^2)$衰减，在Δt=500 μs时比值降至$10^{-3}$量级，矩阵强对角占优。
- 最严苛工况（3 HP电机、同步转速、Δt=500 μs）下，理论近似误差上界仅为1.2%，实际仿真误差远低于此。
- 恒定电导子矩阵使EMTP网络导纳矩阵的LU分解仅需在仿真初始化时执行一次，彻底消除每步重分解的计算负担。
- 模型精度排序为：精确VBR > AVBR > PD > 传统dq预测接口模型，AVBR在保持VBR级精度的同时实现最高计算效率。
