# CIGRE HVDC Benchmark系统 (CIGRE HVDC Benchmark)

## 概述

CIGRE HVDC基准系统是高压直流输电(HVDC)研究领域中最为权威和广泛应用的国际标准测试系统。该系统由国际大电网委员会(CIGRE, International Council on Large Electric Systems)专门工作组于1990年代提出，旨在为LCC-HVDC(电网换相换流器型高压直流)系统的建模、仿真和控制研究提供一个统一、标准化的验证平台。

作为电力电子与电力系统交叉领域的核心基准，CIGRE HVDC Benchmark系统被全球数百个研究机构和工业实验室采用。其标准化的系统参数和详尽的运行数据使得不同研究团队之间可以进行有效的结果对比和方法验证，极大促进了HVDC技术的学术交流和工程应用。

## 系统特点

### 拓扑结构

**基本拓扑**:
- 配置方式: 单极HVDC系统(可扩展为双极运行)
- 接线形式: 双十二脉波换流器串联
- 接地方式: 单极金属回线或单极大地回线
- 对称性: 整流侧与逆变侧对称配置

**换流器结构**:
- 每极: 两个6脉波换流桥串联构成12脉波换流器
- 每个6脉波桥: 6个晶闸管阀臂
- 阀臂配置: 每臂由多个晶闸管串联组成
- 变压器接线: 星形/三角形组合实现12脉波运行

### 电气参数

**额定运行条件**:
| 参数 | 整流侧 | 逆变侧 |
|------|--------|--------|
| 额定功率 | 1000 MW | 1000 MW |
| 直流电压 | +500 kV | -500 kV |
| 直流电流 | 2000 A | 2000 A |
| 交流母线电压 | 345 kV | 230 kV |
| 交流系统频率 | 60 Hz | 50 Hz |

**运行范围**:
- 最小功率: 10%额定功率(100 MW)
- 最大过负荷: 110%额定功率(短时)
- 直流电压调节范围: 70%-100%额定电压
- 功率反转能力: 支持双向功率传输

### 交流系统特性

**短路比(SCR)特性**:
- 整流侧SCR: 约2.5(弱交流系统)
- 逆变侧SCR: 约2.5(弱交流系统)
- 有效短路比(ESCR): 考虑无功补偿后的有效值
- 系统强度: 属于典型弱交流系统，对控制策略敏感

**交流系统阻抗**:
- 整流侧: 短路容量约1300 MVA
- 逆变侧: 短路容量约900 MVA
- 阻抗角: 典型值75-85度

## 换流器参数

### 晶闸管阀参数

**阀臂结构**:
- 每臂晶闸管数量: 90-120个(取决于额定电压)
- 晶闸管串联: 满足500kV直流电压绝缘要求
- 均压电路: RC阻尼电路保证电压均匀分配
- 阀电抗: 限制di/dt保护晶闸管

**晶闸管额定值**:
| 参数 | 典型值 |
|------|--------|
| 额定正向重复峰值电压 | 8-10 kV |
| 额定正向平均电流 | 3-5 kA |
| 通态压降 | 1.5-2.5 V |
| 关断时间 | 200-500 μs |
| 反向恢复期电荷 | 1000-3000 μC |
| du/dt耐受 | 1000-2000 V/μs |
| di/dt耐受 | 100-300 A/μs |

**阀阻尼电路**:
- 阻尼电阻: 50-200 Ω
- 阻尼电容: 0.5-2.0 μF
- 均压电阻: 50-100 kΩ
- 连接方式: 并联在晶闸管两端

### 触发角与关断角范围

**正常运行范围**:
| 控制角 | 整流侧范围 | 逆变侧范围 | 典型值 |
|--------|------------|------------|--------|
| 触发角α | 10°-20° | - | 15° |
| 超前触发角β | - | 30°-40° | 38° |
| 关断角γ | - | 15°-20° | 18° |
| 换相重叠角μ | 15°-25° | 15°-25° | 20° |

**最小限制值**:
- 最小触发角αmin: 5°(保证正向电压触发)
- 最小关断角γmin: 15°-17°(保证晶闸管可靠关断)
- 最大超前触发角βmax: 60°(交流电压限制)

**动态响应**:
- 触发角变化速率: 1-5°/ms
- 最小触发脉冲宽度: 100-200 μs
- 阀组延迟时间: 0.5-1.0 ms

### 换相电抗

**换相电抗组成**:
- 变压器漏抗: 主要组成部分
- 系统阻抗: 交流系统等值阻抗
- 附加电抗: 平波电抗器(如有)

**等值换相电抗**:
- 整流侧: Xc = 27.2 Ω(每6脉波桥)
- 逆变侧: Xc = 27.2 Ω(每6脉波桥)
- 换相电抗标幺值: 0.15-0.20 p.u.

**换相过程**:
- 换相时间: 1-3 ms
- 换相重叠角μ: 取决于负载电流和交流电压
- 换相电抗压降: 与直流电流成正比

## 控制系统

### 整流侧控制

**控制层级结构**:
```
极控级
  └── 换流器控制
      ├── 电流控制环
      ├── 电压控制环
      └── 触发角控制
```

**定电流控制**:
- 控制目标: 维持直流电流恒定
- 电流指令: 来自功率指令或上级控制
- 控制器类型: PI控制器
- 带宽: 30-100 Hz
- 增益参数: Kp = 0.5-2.0, Ki = 10-50

**VDCOL功能**:
- 功能: 低压限流(Voltage Dependent Current Order Limiter)
- 作用: 交流电压降低时自动降低电流指令
- 特性曲线: 分段线性特性
- 启动电压: 0.7-0.8 p.u.
- 最小电流: 0.1-0.3 p.u.

**触发角控制**:
- 控制输出: 晶闸管触发角α
- 触发方式: 等间隔触发(60°间隔)
- 触发脉冲: 高频脉冲列(5-10 kHz)
- 相位同步: 锁相环(PLL)同步

### 逆变侧控制

**定关断角控制**:
- 控制目标: 维持关断角γ恒定
- 设定值: 15°-18°(典型值)
- 测量: 通过阀电压过零点检测
- 控制器: PI控制器，响应时间20-50 ms

**定电压控制**:
- 备用控制: 当关断角控制失效时启用
- 电压设定: 额定直流电压
- 适用场景: 弱交流系统或故障恢复期间

**电流裕度控制**:
- 电流裕度: 整流侧电流指令 - 逆变侧电流指令
- 典型裕度: 0.1 p.u.(约10%额定电流)
- 作用: 确保逆变侧作为从属站运行

### 附加控制功能

**换相失败预测(CFPREV)**:
- 功能: 预测并防止换相失败
- 判据: 交流电压下降速率、阀电流变化率
- 动作: 提前增大触发角或投入无功补偿
- 响应时间: 5-10 ms

**功率调制控制**:
- 功能: 阻尼低频振荡
- 输入信号: 交流频率、功率偏差
- 调制增益: 1-5%功率变化
- 频率范围: 0.1-2.0 Hz

**故障恢复控制**:
- 低压恢复: 电压恢复后的斜率控制
- 电流爬升: 从最小电流逐步恢复
- 时序控制: 与交流断路器配合

## 典型测试场景

### 交流系统故障

**单相接地故障**:
- 故障位置: 整流侧或逆变侧交流母线
- 故障类型: A相接地
- 接地阻抗: 0-10 Ω
- 故障持续时间: 100-200 ms
- 观测指标: 直流电压跌落、功率波动、恢复时间

**三相短路故障**:
- 故障位置: 交流出线或母线
- 故障阻抗: 接近零(金属性短路)
- 电压跌落: 0-30%额定电压
- 故障清除: 线路保护动作切除
- 系统响应: VDCOL动作、电流限幅

**故障影响分析**:
- 直流功率中断时间
- 无功消耗变化
- 交流电压恢复特性
- 控制系统响应

### 直流系统故障

**直流线路接地故障**:
- 故障位置: 线路中点或端部
- 故障电阻: 0-100 Ω
- 故障检测: 直流电压不平衡保护
- 清除方式: 移相重启或闭锁重启

**换流阀故障**:
- 阀短路: 单个晶闸管或整个阀臂
- 阀开路: 触发失效或晶闸管损坏
- 保护措施: 旁通对、紧急移相
- 系统切换: 降额运行或停运

### 换相失败

**换相失败机理**:
- 起因: 交流电压降低或畸变
- 过程: 关断角不足导致阀误导通
- 结果: 直流短路、电流激增
- 恢复: 依赖下一次成功换相

**测试方法**:
- 交流电压跌落: 模拟远端故障影响
- 谐波注入: 测试电压畸变容忍度
- 无功扰动: 模拟滤波器投切

**预防与恢复**:
- CFPREV控制效果
- 低压限流配合
- 快速恢复策略

### 功率阶跃响应

**阶跃测试条件**:
- 阶跃幅度: ±10%, ±20%, ±50%额定功率
- 变化速率: 受电流变化率限制(通常1-10 MW/s)
- 测试目的: 验证控制系统动态性能

**评估指标**:
- 上升时间: 功率从10%到90%所需时间
- 超调量: 功率超调百分比
- 稳定时间: 进入±2%误差带时间
- 振荡次数: 阻尼比评估

### 启动与停运过程

**启动顺序**:
1. 交流侧充电: 断路器合闸
2. 解锁顺序: 先逆变侧后整流侧
3. 电压建立: 直流电压升至额定值
4. 功率爬升: 电流逐步增至运行值

**停运顺序**:
1. 功率降低: 电流降至最小值
2. 闭锁顺序: 先整流侧后逆变侧
3. 交流侧断电: 断路器分闸
4. 残余电荷释放

**测试重点**:
- 解锁过电压抑制
- 电流无冲击建立
- 无功功率平衡
- 谐波电流限制

## 研究应用

### LCC-HVDC建模验证

**模型验证目的**:
- 验证自定义模型的正确性
- 比较不同建模方法的精度
- 确认模型参数设置合理性

**验证内容**:
- 稳态特性: 伏安特性、功率特性
- 动态响应: 阶跃响应、扰动响应
- 故障特性: 换相失败、故障恢复
- 谐波特性: 谐波电流、谐波阻抗

**对比基准**:
- PSSE标准模型
- PSCAD/EMTDC示例
- MATLAB/Simulink模型
- 厂家专用模型

### 控制器设计

**控制器优化**:
- PI参数整定: 基于系统辨识或经验公式
- 现代控制: 状态反馈、观测器设计
- 自适应控制: 增益调度、模型参考自适应
- 智能控制: 模糊控制、神经网络

**仿真验证**:
- 小信号稳定: 特征值分析
- 大信号稳定: 时域仿真
- 鲁棒性测试: 参数摄动、模型不确定性
- 抗干扰性: 测量噪声、参数漂移

### 故障分析与保护

**故障类型研究**:
- 阀故障: 短路、开路、误触发
- 变压器故障: 绕组短路、接地故障
- 直流线路故障: 接地、极间短路
- 交流系统故障: 对称与不对称故障

**保护配合研究**:
- 阀保护: 过流、过压、温度
- 换流变保护: 差动、瓦斯、过负荷
- 直流保护: 线路保护、接地保护
- 交流保护: 距离保护、零序保护

**恢复策略**:
- 自动重启: 成功率和重启时间
- 降压运行: 故障后降电压运行
- 功率回降: 从低功率逐步恢复

### 交直流混合系统研究

**系统稳定性**:
- 电压稳定性: 换流站接入点电压支撑
- 功角稳定性: 直流功率调制阻尼振荡
- 频率稳定: 直流功率紧急支援

**协调控制**:
- 交直流协调: 潮流优化、电压协调
- 多回直流协调: 并联运行、功率分配
- 新能源接入: 风电、光伏送出

### 实时仿真测试

**硬件在环测试**:
- 控制器HIL: 真实控制器+仿真换流站
- 保护HIL: 真实保护装置测试
- 功率放大器: 小信号功率接口

**仿真平台要求**:
- 步长要求: 20-50 μs(电磁暂态)
- 实时性: 严格实时约束
- 并行计算: 多核CPU或FPGA
- I/O接口: 模拟量、数字量接口

**测试案例**:
- 控制器功能验证
- 极端工况测试
- 保护逻辑验证
- 人机界面测试

## Benchmark意义

### 跨工具对比基准

**软件平台对比**:
| 软件 | 特点 | 适用分析 |
|------|------|----------|
| PSCAD/EMTDC | 电磁暂态仿真 | 详细暂态过程 |
| PSSE | 机电暂态仿真 | 系统稳定性分析 |
| MATLAB | 灵活建模 | 控制算法研究 |
| RTDS | 实时仿真 | 硬件在环测试 |
| DIgSILENT | 综合仿真 | 多场景分析 |

**标准化意义**:
- 消除模型差异: 统一系统参数
- 结果可比性: 相同测试条件下的对比
- 方法验证: 新算法的标准化验证流程
- 教学培训: 统一的教学案例

### 模型验证标准

**验证流程**:
1. 建立标准模型
2. 执行标准测试用例
3. 与参考结果对比
4. 误差分析与修正
5. 模型认证

**验收标准**:
- 稳态误差: <1%
- 暂态误差: <5%(峰值)
- 响应时间误差: <10%
- 稳定性结论一致

### 学术交流共同语言

**论文发表**:
- 方法可复现: 读者可用相同系统验证
- 结果可信: 与公认基准对比
- 创新突出: 与已有方法性能对比清晰

**国际合作**:
- 数据共享: 统一的数据格式和参数
- 联合研究: 多机构协作基础
- 标准制定: CIGRE工作组技术基础

## 变体和扩展

### 双极扩展

**双极配置**:
- 两个单极系统并联
- 共用接地极
- 额定功率: 2×1000 MW = 2000 MW
- 运行灵活性: 单极停运时另一极可继续运行

**双极运行模式**:
- 双极对称: 两极电流相等
- 双极不对称: 两极电流不等(接地极限流)
- 单极大地回线: 一极停运时大地返回
- 单极金属回线: 通过金属导线返回

### 多端直流扩展

**三端配置**:
- 整流站: 1个
- 逆变站: 2个
- 功率分配: 根据需求调节各站功率

**控制策略**:
- 主从控制: 一个站控制电压，其余控制电流
- 电压裕度控制: 多端协调的改进策略
- 直流电压下垂: 分散式控制

### VSC-HVDC改造

**改造研究**:
- 研究目的: 对比LCC与VSC技术
- 系统参数: 保持相同功率和电压等级
- 控制策略: 电压源型换流器控制
- 应用场景: 新能源并网、城市供电

**技术对比**:
| 特性 | LCC-HVDC | VSC-HVDC |
|------|----------|----------|
| 换流器类型 | 晶闸管 | IGBT/IGCT |
| 无功需求 | 需要 | 不需要 |
| 黑启动 | 不能 | 能 |
| 短路容量 | 要求高 | 要求低 |
| 模块化 | 较复杂 | 灵活 |
