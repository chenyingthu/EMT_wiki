# PMSG海上风电场测试系统 (PMSG Offshore Wind Farm Test System)

## 概述

永磁同步直驱(PMSG, Permanent Magnet Synchronous Generator)海上风电场是大规模风电研究的重要测试平台。该测试系统基于全功率变流器结构，采用永磁同步发电机与风轮直接耦合的传动方式，省去了齿轮箱环节，具有高可靠性、高效率和电网友好性等显著优势。

PMSG海上风电场测试系统在电磁暂态(EMT)仿真研究中具有重要地位。随着海上风电向大规模、深远海方向发展，单个风电场装机容量可达数百兆瓦甚至吉瓦级，对仿真效率和模型准确性提出了更高要求。该测试系统为风电场等值建模、聚合模型验证、故障穿越分析、电磁暂态加速仿真等前沿研究提供了标准化的验证平台。

**系统特点**:
- **全功率变流器结构**: 发电机通过背靠背变流器与电网完全解耦
- **无齿轮箱设计**: 直驱结构提高可靠性，降低维护成本
- **海上环境适应**: 针对海上风电汇集和远距离输电场景优化
- **灵活控制策略**: 支持多种先进控制算法和控制目标
- **规模化验证**: 适用于风电场级聚合和等值研究

**研究价值**:
- 验证[[cs-dam-model]]等先进聚合模型的有效性
- 评估海上风电汇集系统的电气特性
- 分析大规模风电场与电网的交互影响
- 测试故障穿越和低电压/高电压穿越能力
- 验证电磁暂态加速仿真算法的精度与效率

## 控制系统

### 机侧变流器控制

**转速/转矩控制策略**:

**最大功率点跟踪(MPPT)**:
```
风速v → 最优转速ω_opt → 转矩指令T* → MSC控制
         ↑_________________|
           转速反馈ω
```

**控制模式**:
| 控制模式 | 应用场景 | 控制目标 |
|----------|----------|----------|
| 转速控制 | 变速运行 | 维持最优转速 |
| 转矩控制 | 稳态发电 | 跟踪最优转矩 |
| 功率限制 | 高风速 | 限制输出功率 |
| 变桨配合 | 限功率区 | 协调桨距控制 |

**MPPT算法**:
- **最佳叶尖速比法**: 维持最优λ值
- **功率曲线查表法**: 基于实测功率曲线
- **爬山搜索法**: 在线寻优算法
- **最优转矩法**: T_opt = k·ω²关系

### 网侧变流器控制

**电压定向矢量控制(VOC)**:

```
直流电压Udc_ref → PI控制器 → 有功电流id_ref
                              ↓
电压相位θ → Park变换 → 电流id, iq → 电流环PI
                              ↓
                         PWM调制 → GSC
```

**双环控制结构**:
| 控制环 | 带宽 | 功能 |
|--------|------|------|
| 电压外环 | 10-50Hz | 维持Udc恒定 |
| 电流内环 | 500-2000Hz | 快速电流跟踪 |

**控制坐标系**:
- **d轴**: 定向电网电压矢量方向
- **q轴**: 超前d轴90°，控制无功功率
- **id**: 控制有功功率(直流电压)
- **iq**: 控制无功功率

### 故障穿越控制

**低电压穿越(LVRT)**:

**触发条件**:
- 并网点电压跌落至额定值的20%-90%
- 持续时间超过设定阈值(如100ms)

**控制响应**:
| 电压跌落 | 控制策略 | 目标 |
|----------|----------|------|
| 轻度(>70%) | 无功支撑 | 注入无功电流 |
| 中度(50-70%) | 有功限制 | 防止过电流 |
| 重度(<50%) | 直流斩波 | 消耗过剩功率 |

**无功电流注入**:
```
ΔIq = K·(1 - V/Vn)  (当V < 0.9Vn)
```
其中K为无功支撑系数，通常要求K ≥ 2

**高电压穿越(HVRT)**:
- 电压升高至110%-130%额定值
- 吸收无功功率
- 防止设备过电压损坏

### 无功电压支撑

**控制模式**:
| 控制模式 | 设定值 | 应用场景 |
|----------|--------|----------|
| 单位功率因数 | Q=0 | 正常发电 |
| 恒定无功 | Q=Q_ref | 电压调节 |
| 恒定功率因数 | cosφ=const | 功率因数要求 |
| 电压控制 | V=V_ref | 电压支撑 |

**电压下垂控制**:
```
Q = Q0 + Kv·(Vref - V)
```
实现多机并联时的无功功率分配

## 研究应用

### 风电场等值建模

**等值建模需求**:

大规模风电场包含数十至数百台风电机组，详细建模计算量巨大。等值建模通过聚合方法将多机系统简化为等效单机或少数几台机组，在保证精度的同时大幅提升仿真效率。

**等值方法**:
| 方法 | 原理 | 适用场景 |
|------|------|----------|
| 单机等值 | 功率加权平均 | 粗略估算 |
| 多机等值 | 分群聚合 | 精度要求较高 |
| 动态等值 | 保留主导模态 | 稳定性分析 |
| 频率相关等值 | 宽频特性 | 电磁暂态 |

**PMSG风电场等值特点**:
- 全变流器机组无惯量直接耦合
- 等值参数与风况分布相关
- 集电网络阻抗不可忽略
- 需考虑控制策略一致性

### CS-DAM模型验证

**CS-DAM模型简介**:

CS-DAM(Combined System Dynamic Aggregation Model)是一种先进的风电场聚合模型，通过动态聚合同质机组，在保证等值精度的同时显著降低模型复杂度。

**验证内容**:
| 验证项目 | 测试方法 | 精度要求 |
|----------|----------|----------|
| 稳态特性 | 功率-风速曲线 | 误差<3% |
| 动态响应 | 风速阶跃响应 | 误差<5% |
| 故障特性 | 低电压穿越 | 误差<10% |
| 稳定性 | 小扰动分析 | 特征值匹配 |

**验证流程**:
1. 建立详细模型(每台风机)
2. 建立CS-DAM等值模型
3. 施加相同扰动条件
4. 对比输出响应曲线
5. 量化评估等值误差

### 海上风电汇集

**汇集系统研究**:

海上风电场通过集电网络汇集各台风机的输出功率，集电系统的电气特性对整个风电场的动态行为有显著影响。

**研究内容**:
| 研究方向 | 研究内容 | 关键问题 |
|----------|----------|----------|
| 集电网络 | 海缆参数影响 | 充电功率、谐振 |
| 无功补偿 | 无功配置优化 | 电压稳定 |
| 故障特性 | 集电线路故障 | 保护配合 |
| 谐波分析 | 变流器谐波汇集 | 谐波放大 |

**集电系统影响**:
- 海缆电容产生的无功功率
- 长距离传输的电压分布
- 故障电流水平与保护整定
- 多机并联的相互影响

### 故障穿越分析

**故障穿越能力验证**:

PMSG风电场需具备故障穿越能力，在电网故障期间维持并网运行，支撑电网电压恢复。

**分析内容**:
| 分析类型 | 分析目标 | 关键参数 |
|----------|----------|----------|
| LVRT | 低电压穿越 | 电压跌落深度、持续时间 |
| HVRT | 高电压穿越 | 电压升高幅度 |
| FRT协调 | 多场站协调 | 无功分配策略 |
| 保护配合 | 继电保护 | 动作时间、选择性 |

**仿真验证**:
- 电压跌落至20%额定电压
- 故障持续150-1000ms
- 无功电流注入要求
- 有功功率恢复速度
