## 概述

鲁西背靠背异步联网工程是中国首个超高压大容量柔性直流背靠背工程，也是世界上容量最大的背靠背柔性直流工程之一。该工程位于云南省曲靖市罗平县与广西壮族自治区百色市西林县交界的南盘江畔，是南方电网公司"西电东送"战略的重要组成部分。

### 工程背景

**战略意义**:
- 实现云南电网与广西电网的异步联网
- 解决云南电网异步联网后的稳定问题
- 提高云电外送能力和系统稳定性
- 优化南方电网整体运行性能

**历史地位**:
- 中国首个超高压大容量柔性直流背靠背工程
- 世界上容量最大的背靠背柔性直流工程之一
- 标志着中国在柔性直流输电技术领域达到世界领先水平
- 为后续同类型工程提供了宝贵经验

**投运信息**:
- 开工建设: 2015年
- 正式投运: 2017年6月
- 建设单位: 中国南方电网有限责任公司
- 设计单位: 中国能源建设集团云南省电力设计院

## 建模方法

### 详细模型

**模型描述**:
详细模型对每个IGBT和二极管进行详细建模，适用于高精度离线仿真。

**建模要素**:
| 元件 | 模型类型 | 参数来源 |
|------|----------|----------|
| IGBT | 开关模型/物理模型 | 数据手册 |
| 二极管 | 理想开关/详细模型 | 数据手册 |
| 电容器 | 理想电容/ESR模型 | 实测参数 |
| 电抗器 | 理想电感/饱和模型 | 设计参数 |
| 变压器 | 理想变比/饱和模型 | 铭牌参数 |

**详细开关模型**:
```
function v_sm = detailed_sm_model(state, v_c, i_arm)
    if state == ON
        v_sm = 0;  % IGBT导通压降
    else
        v_sm = v_c; % 电容电压
    end
end
```

**仿真特点**:
- 精度: 最高，能反映开关细节
- 速度: 最慢，不适合大规模系统
- 应用: 器件级分析、损耗计算
- 工具: MATLAB/Simulink、PSIM

### 戴维南等效模型

**模型描述**:
将子模块等效为可变电压源与电阻串联，大幅提高仿真效率。

**等效电路**:
```
        R_th
    ┌───█████───┐
    │           │
   V_th         │
    │           │
    └───────────┘
        │
       i_arm
```

**等效参数计算**:
```
N_on: 投入子模块数
N_total: 总子模块数
V_c_avg: 平均电容电压

V_th = N_on × V_c_avg
R_th = (N_total - N_on) × R_bypass / N_total
```

**模型优势**:
- 计算效率: 比详细模型快100-1000倍
- 精度保持: 保留了电容电压动态
- 适用性广: 适用于大规模MMC仿真
- 实时仿真: 支持硬件在环仿真

**仿真工具**:
- PSCAD/EMTDC: 内置MMC模型
- RT-LAB: 实时仿真平台
- RTDS: 实时数字仿真器
- MATLAB/Simulink: 自定义实现

### 平均值模型

**模型描述**:
平均值模型忽略开关细节，将MMC等效为受控电压源和电流源。

**等效电路**:
```
交流侧            MMC平均值模型          直流侧
  │                    │                  │
  │    ┌───────────────┼───────┐          │
  │    │               │       │          │
  V_ac  R_eq         V_conv   L_eq       V_dc
  │    │               │       │          │
  │    └───────────────┼───────┘          │
  │                    │                  │
```

**控制方程**:
```
% 交流侧电压方程
V_conv_d = m_d × V_dc/2
V_conv_q = m_q × V_dc/2

% 功率方程
P = 3/2 × (V_conv_d × i_d + V_conv_q × i_q)
Q = 3/2 × (V_conv_q × i_d - V_conv_d × i_q)

% 直流侧方程
I_dc = P / V_dc
C_dc × dV_dc/dt = I_dc_in - I_dc_out
```

**调制比计算**:
```
m_d = 2 × V_conv_d / V_dc
m_q = 2 × V_conv_q / V_dc
m = sqrt(m_d² + m_q²)  % 调制比幅值
```

**模型特点**:
- 速度最快: 适合系统级研究
- 精度适中: 忽略高频开关细节
- 应用广泛: 潮流计算、稳定性分析
- 控制设计: 便于控制器设计

**适用场景**:
| 应用场景 | 推荐模型 | 理由 |
|----------|----------|------|
| 器件级分析 | 详细模型 | 需要开关细节 |
| 保护研究 | 戴维南模型 | 需要故障电流 |
| 系统稳定性 | 平均值模型 | 关注机电暂态 |
| 控制器设计 | 平均值模型 | 便于理论分析 |
| 实时仿真 | 戴维南模型 | 计算效率与精度平衡 |
