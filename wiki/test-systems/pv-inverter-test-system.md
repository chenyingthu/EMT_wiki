# 光伏逆变器测试系统 (PV Inverter Test System)

## 概述

光伏逆变器(PV Inverter)是光伏发电系统的核心设备，负责将光伏阵列产生的直流电能转换为符合电网要求的交流电能。作为连接可再生能源与电网的关键接口，光伏逆变器不仅需要实现高效的电能转换，还必须满足严格的并网标准和故障穿越要求。在EMT(Electromagnetic Transient)仿真研究中，光伏逆变器系统是重要的研究对象，涉及MPPT控制、并网控制、低电压穿越(LVRT)、孤岛检测等关键技术。

光伏逆变器测试系统是专门为验证光伏逆变器性能而设计的标准化测试平台，能够模拟各种工况下的光伏阵列输出特性和电网运行状态，全面评估逆变器的发电效率、电能质量、保护功能和动态响应能力。该测试系统广泛应用于逆变器产品研发、型式试验、入网检测和科学研究等领域。

## MPPT控制

最大功率点跟踪(Maximum Power Point Tracking, MPPT)是光伏逆变器的核心功能，用于在变化的环境条件下实时跟踪光伏阵列的最大功率输出点。

### 扰动观察法(P&O)

**基本原理**:
通过周期性地对光伏阵列的输出电压施加微小扰动，观察功率变化方向，从而确定下一步的扰动方向。

**算法流程**:
```
1. 测量当前电压V(k)和电流I(k)
2. 计算当前功率P(k) = V(k) × I(k)
3. 与上一时刻功率P(k-1)比较
4. 若P(k) > P(k-1):
   - 若V(k) > V(k-1): 继续正向扰动
   - 若V(k) < V(k-1): 反向扰动
5. 若P(k) < P(k-1): 反向扰动
6. 更新V(k-1)=V(k), P(k-1)=P(k)
7. 等待下一采样周期
```

**数学描述**:
```
ΔV = V(k) - V(k-1)
ΔP = P(k) - P(k-1)

if ΔP > 0:
    if ΔV > 0: Vref = V(k) + ΔVstep
    else: Vref = V(k) - ΔVstep
else:
    if ΔV > 0: Vref = V(k) - ΔVstep
    else: Vref = V(k) + ΔVstep
```

**特点分析**:
- 优点: 实现简单、成本低、无需光伏特性先验知识
- 缺点: 稳态振荡、快速变化时误判、部分遮阴失效

**改进方法**:
- 变步长P&O: 根据功率变化率调整步长
- 自适应P&O: 动态调整扰动周期
- 三阶段P&O: 区分暂态和稳态

### 增量电导法(IncCond)

**基本原理**:
基于光伏阵列P-V曲线的特性，在最大功率点处满足dP/dV = 0。

**数学推导**:
```
P = V × I
dP/dV = d(V×I)/dV = I + V×(dI/dV)

在MPP点:
dP/dV = 0 ⇒ I + V×(dI/dV) = 0
⇒ dI/dV = -I/V
```

**判断准则**:
```
if |dI/dV + I/V| < ε: 已达MPP
if dI/dV > -I/V: 在MPP左侧，增大V
if dI/dV < -I/V: 在MPP右侧，减小V
```

**实现方式**:
```
ΔI = I(k) - I(k-1)
ΔV = V(k) - V(k-1)

if ΔV ≠ 0:
    if |ΔI/ΔV + I/V| < threshold:
        # 保持在MPP
    elif ΔI/ΔV > -I/V:
        Vref = V(k) + ΔVstep  # 左移
    else:
        Vref = V(k) - ΔVstep  # 右移
else:
    if ΔI > 0: Vref = V(k) + ΔVstep
    elif ΔI < 0: Vref = V(k) - ΔVstep
```

**特点分析**:
- 优点: 理论上无稳态振荡、跟踪精度高
- 缺点: 计算复杂度高、对噪声敏感、需高精度采样

**与P&O比较**:
| 特性 | P&O | IncCond |
|------|-----|---------|
| 实现复杂度 | 简单 | 较复杂 |
| 稳态振荡 | 有 | 理论上无 |
| 跟踪速度 | 中等 | 快 |
| 硬件要求 | 低 | 较高 |
| 部分遮阴 | 易失效 | 需改进 |

## 并网控制

并网控制是光伏逆变器的核心功能，负责实现与电网的安全、稳定、高效连接。

### 电流控制

**控制目标**:
- 输出电流正弦度高
- 谐波含量低
- 快速动态响应
- 抗干扰能力强

**PI控制**:
```
          ┌─────────────┐
i* ──────→│  Kp + Ki/s  ├──────→ vref
          │             │        │
    ┌─────┴─────┐      │      ┌─┴─┐
    │   Plant   │←─────┘      │PWM│
    └───────────┘             └─┬─┘
         │                      │
         └──────────────────────┘
```

- 优点: 结构简单、实现容易
- 缺点: 对交流信号存在稳态误差

**PR控制(比例谐振)**:
```
Gpr(s) = Kp + Kr×s/(s² + ω₀²)
```

- 在基频处增益无穷大
- 消除稳态误差
- 适合单相系统

**同步旋转坐标系(dq控制)**:
```
        ┌─────────┐         ┌─────────┐
id* ───→│  PI_d   ├────→ vdref ───→│         ├──→ id
        │         │                │  Park   │
iq* ───→│  PI_q   ├────→ vqref ───→│Inverse  ├──→ iq
        └─────────┘                └─────────┘
```

- 三相交流量转换为直流量
- 简化控制器设计
- 有功/无功解耦控制

**模型预测控制(MPC)**:
- 利用系统模型预测未来状态
- 优化目标函数选择最优开关
- 动态响应快
- 计算复杂度高

### 电压控制

**应用场景**:
- 弱电网条件
- 孤岛运行模式
- 微电网主控单元

**控制结构**:
```
Vref ───→ 电压环 ───→ 电流环 ───→ PWM ───→ 逆变器
          (PI)        (PI/PR)              │
           │           │                    │
           └────← Vmeas ←───────────────────┘
```

**下垂控制**:
```
f = f₀ - kₚ×(P - P₀)
V = V₀ - kₒ×(Q - Q₀)
```

- 实现多机并联功率分配
- 无需通信
- 适合微电网应用

### 有功/无功调节

**有功功率控制**:
- 最大功率跟踪模式
- 限功率运行模式
- 频率响应模式

**无功功率控制**:
- 功率因数控制
- 电压支撑模式
- 无功补偿模式

**控制模式**:
| 模式 | 控制目标 | 应用场景 |
|------|----------|----------|
| 单位功率因数 | Q=0 | 常规发电 |
| 超前/滞后 | 设定功率因数 | 电网要求 |
| 电压调节 | 维持节点电压 | 弱电网 |
| 无功设定 | 指定无功功率 | 电网调度 |

**动态响应要求**:
- 有功响应时间: <1s (10%-90%)
- 无功响应时间: <0.5s
- 功率因数调节范围: 0.9超前-0.9滞后

### 功率因数控制

**功率因数定义**:
```
PF = P/√(P² + Q²) = cos(φ)
```

**控制策略**:
1. 检测实际功率因数
2. 与目标值比较
3. 调节无功电流分量
4. 实时跟踪补偿

**控制框图**:
```
PFref ───→ 比较器 ───→ PI ───→ iq* ───→ 电流环 ───→ 逆变器
            ↑                              │
            └────── cos(φ)计算 ←───────────┘
```

**补偿能力**:
- 感性负载补偿(超前)
- 容性负载补偿(滞后)
- 动态范围: ±0.9

## 典型测试场景

- 光照变化范围: 200-1000 W/m²
- 变化速率: 10-100 W/m²/s
- 变化模式: 斜坡、阶跃、正弦

```
      斜坡  阶跃  扰动
```

**评价指标**:
- MPPT效率: 实际能量/理论最大能量
- 跟踪速度: 到达新MPP的时间
- 跟踪精度: 平均功率偏差

**典型测试用例**:

| 阶跃变化 | 500→800 W/m² 阶跃 | 瞬态 | 稳定时间<1s |
| 周期性 | 正弦波动 ±20% | 10周期 | 无失步现象 |

- 短路电流随温度变化较小
- 最大功率点电压随温度变化

**测试条件**:
- 温度范围: -20°C ~ +60°C
- 温度变化率: 1-5°C/min

**温度特性曲线**:

```
   │            ╲ Pmax
-15├─────────────╲
   └───────────────→ 温度(°C)
    -20  0   25  60
```

**评价指标**:
- 温度系数验证
- MPP跟踪精度
- 效率温度特性

### LVRT测试

**测试目的**:
验证逆变器在电网电压跌落时的低电压穿越能力，确保故障期间和故障后稳定并网运行。

**LVRT要求**:
```
电压(p.u.)
  │
1.1├───────────────────────
   │  ┌────┐
 1.0├──┤    ├─────────────
   │  │    │
0.90├──┤    ├────────────── 必须保持并网
   │  │    │   ┌──────┐
0.20├──┘    └───┤      ├── 可短时脱网
   │             │      │
  0 ├─────────────┴──────┴──→ 时间(ms)
      0    150  625    2000
```

**测试场景**:

**三相短路故障**:
- 电压跌落深度: 0%, 20%, 50%, 85%
- 故障持续时间: 150ms-2000ms
- 恢复速度: 0.5-2.0 p.u./s

**单相接地故障**:
- 单相电压跌落
- 非故障相电压升高
- 负序分量影响

**相间短路故障**:
- 两相电压跌落
- 不对称工况
- 零序分量

**测试要求**:
| 电压跌落 | 持续时间 | 有功恢复 | 无功支撑 |
|----------|----------|----------|----------|
| 0-20% | ≤2s | 故障后2s内恢复 | 提供无功 |
| 20-90% | 按曲线 | 故障后2s内恢复 | 提供无功 |
| >90% | 连续运行 | - | 按需调节 |

**无功支撑要求**:
```
当电压跌落时:
- 注入无功电流支撑电压恢复
- 无功电流响应时间 < 30ms
- 无功电流增量 ≥ 2% In/p.u.电压跌落
```

**有功恢复要求**:
- 故障清除后2s内恢复至90%预故障值
- 恢复速率可调
- 无超调或振荡

### 孤岛检测

**测试目的**:
验证逆变器在电网断电时能够及时检测孤岛状态并停止运行，防止非计划孤岛运行带来的安全和电能质量问题。

**孤岛检测方法**:

**被动式检测**:
- 过/欠电压保护(OVP/UVP)
- 过/欠频率保护(OFP/UFP)
- 电压相位跳变检测
- 谐波检测

**主动式检测**:
- 主动频率偏移(AFD)
- 滑模频率偏移(SMS)
- 阻抗检测
- 输出功率扰动

**测试电路**:
```
    Grid ──┬─── RLC负载 ───┬── PV逆变器
           │               │
          开关            并网点
```

**测试条件**:
- RLC负载谐振频率: 50±0.1Hz
- 负载品质因数: Qf = 1.0±0.1
- 功率匹配: Pload = Ppv ±5%

**测试用例**:
| 测试场景 | 负载条件 | 检测时间要求 |
|----------|----------|--------------|
| 完美匹配 | Qf=1, f=50Hz, Pmatch | ≤2s |
| 频率偏移 | f=50.5Hz | ≤2s |
| 电压偏移 | V=1.05p.u. | ≤2s |
| 功率不匹配 | Pload=0.5Ppv | ≤2s |

**非计划孤岛运行危害**:
- 人身安全风险
- 设备损坏风险
- 电能质量恶化
- 重合闸失败

### 谐波测试

**测试目的**:
验证逆变器输出电流的谐波含量满足并网标准要求，评估电能质量。

**谐波限制标准**:

**IEEE 519限值**:
| 谐波次数 | 限值(%额定电流) |
|----------|-----------------|
| 2-10 | 4.0% |
| 11-30 | 2.0% |
| 31-100 | 1.5% |
| THD | 5.0% |

**IEC 61000-3-6限值**:
- 奇次谐波: 3-15次各有限值
- 偶次谐波: 2-10次各有限值
- 总谐波畸变率: 5%

**测试条件**:
- 额定功率运行
- 不同功率点(10%, 25%, 50%, 75%, 100%)
- 不同电网阻抗

**谐波测量**:
```
THD = √(Σ(Ih²)) / I₁ × 100%

其中:
Ih: h次谐波电流有效值
I₁: 基波电流有效值
```

**影响因素**:
- 调制策略选择
- 开关频率
- 滤波器设计
- 电网阻抗
- 死区时间

**改善措施**:
- 优化调制策略
- 提高开关频率
- 改进滤波器
- 谐波补偿

## 并网标准

### IEEE 1547

**标准范围**:
IEEE 1547系列标准规定了分布式电源(包括光伏逆变器)并网的技术要求和测试程序。

**主要版本**:
- IEEE 1547-2003: 原版标准
- IEEE 1547-2018: 重大修订版
- IEEE 1547.1-2020: 测试程序

**关键要求**:

**电压和频率范围**:
| 参数 | 正常运行范围 | 允许运行范围 |
|------|--------------|--------------|
| 电压 | 0.88-1.10 p.u. | 0.50-1.10 p.u. |
| 频率 | 59.3-60.5 Hz | 56.5-62.0 Hz |

**低电压穿越**:
- 电压跌落至0时仍需支撑2s
- 无功电流支撑要求
- 电压恢复后有功功率恢复要求

**频率响应**:
- 频率过高: 有功功率降额
- 频率过低: 持续运行或增加输出
- 频率变化率耐受

**电能质量**:
- 直流注入: <0.5%额定电流
- 闪变: 符合IEEE 1453
- 谐波: 符合IEEE 519

**测试项目**:
- 稳态性能测试
- 暂态性能测试
- 孤岛检测测试
- 抗扰度测试
- 电能质量测试

**标准范围**:
IEC 61727规定了光伏系统并网接口的技术特性，适用于容量不超过100kVA的并网光伏系统。

**技术要求**:

**功率因数**:
- 额定功率时: ≥0.9超前或滞后
- 可调范围: 0.9超前-0.9滞后

**电压范围**:

**频率范围**:
- 正常运行: 49.5-50.5Hz (50Hz系统)
- 保护动作: 47-52Hz

**响应时间**:
- 电压异常响应: <0.2s
- 频率异常响应: <0.2s

**电能质量**:
- 谐波: 符合IEC 61000-3-2/3-6
- 闪变: 符合IEC 61000-3-3
- 直流注入: <1%额定电流

## 仿真建模

### 光伏阵列模型

**数学模型**:
```
I = Iph - Is×[exp(q(V+IRs)/nNkT) - 1] - (V+IRs)/Rsh

参数说明:
Iph: 光生电流
Is: 反向饱和电流
Rs: 串联电阻
Rsh: 并联电阻
n: 理想因子
N: 串联电池数
k: 玻尔兹曼常数
T: 温度
q: 电子电荷量
```

**简化模型**:
```
I = Isc×[1 - C1×(exp(V/(C2×Voc)) - 1)]

其中:
C1 = (1 - Imp/Isc)×exp(-Vmp/(C2×Voc))
C2 = (Vmp/Voc - 1)/ln(1 - Imp/Isc)
```

### 逆变器控制模型

**平均模型**:
- 忽略开关细节
- 保留控制动态
- 适合系统级仿真

**开关模型**:
- 详细开关过程
- 高频谐波分析
- 计算量大

### EMT仿真要点

**时间步长**:
- 开关模型: 1-10μs
- 平均模型: 10-100μs
- 系统级仿真: 100μs-1ms

**关键设置**:
- 数值积分方法
- 开关模型处理
- 初始条件设置
- 收敛性控制
