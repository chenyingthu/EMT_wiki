# IEEE 14节点系统 (IEEE 14-Bus System)

## 概述

IEEE 14节点系统是电力系统研究中最基础、最广泛使用的小型标准测试系统之一。该系统最早于1962年由美国电气与电子工程师协会(IEEE)提出，作为电力系统潮流计算和基础分析的基准测试案例。系统以美国中西部地区的简化电力网络为原型，包含14个母线、5台同步发电机（含1台平衡机），总装机容量约为272.4MW，总负荷约为259MW。

作为入门级测试系统，IEEE 14节点系统在保持足够结构复杂性的同时，具有计算量小、数据完整、结构清晰的优点。它是电力系统课程教学的首选案例，也是验证新算法、测试软件功能和进行初步研究的基础平台。几乎所有商业电力系统分析软件（如PSS/E、PowerFactory、MATPOWER）都将其作为内置示例数据，使其成为电力工程领域公认的"Hello World"测试案例。

## 系统特点

### 规模与结构

**基本参数**:
- 节点数: 14个
- 发电机: 5台（含1台平衡机）
- 负荷节点: 11个PQ节点
- 支路数: 20条（线路+变压器）
- 基准容量: 100 MVA
- 基准频率: 60 Hz

**网络结构**:
- 主网架: 138kV和69kV混合输电网络
- 区域结构: 以母线4-9为核心形成环网结构
- 发电机分布: 分布在母线1、2、3、6、8
- 负荷分布: 除母线1外，其余母线均带有负荷

**拓扑特征**:
- 混合拓扑: 辐射状与环网结构相结合
- 双电压等级: 138kV和69kV通过变压器互联
- 分层供电: 发电机升压后接入主干网

### 电压等级

**发电电压**:
- 母线1: 69kV（平衡机出口电压）
- 母线2: 69kV（PV节点）
- 母线3: 69kV（PV节点）
- 母线6: 13.8kV（同步调相机）
- 母线8: 18kV（PV节点）

**输电电压**:
- 138kV: 主干网架（母线4、5、7、9之间的连接）
- 69kV: 区域配电网络
- 通过变压器实现电压等级变换

**配电电压**:
- 34.5kV/13.8kV: 部分发电机出口电压
- 通过降压变压器向负荷供电

### 发电容量

**总发电容量**:
- 额定容量: 约272.4MW
- 总负荷: 约259MW
- 网损: 约13.4MW（约4.9%）
- 基准容量: 100 MVA

## 研究应用

### 潮流计算教学

**教学价值**:
- 涵盖所有基本潮流计算概念
- 平衡节点、PV节点、PQ节点完整示例
- 牛顿-拉夫逊法和高斯-赛德尔法验证
- 收敛特性分析的理想案例

**常用算法测试**:
- 牛顿-拉夫逊法(Newton-Raphson)
- 快速解耦法(Fast Decoupled Load Flow)
- 直流潮流(DC Load Flow)
- 前推回代法(Forward-Backward Sweep)

**收敛特性**:
- 牛顿-拉夫逊法: 通常3-5次迭代收敛
- 收敛容差: 0.001 p.u.（有功和无功功率失配）
- 良好的收敛性，适合教学演示

### 短路电流计算

**对称短路分析**:
- 三相短路容量计算
- 短路电流分布
- 各母线短路容量评估

**不对称短路分析**:
- 单相接地故障
- 两相短路故障
- 两相接地故障

**保护整定**:
- 过电流保护整定
- 距离保护整定
- 差动保护整定

### 状态估计

**量测配置**:
- 节点注入量测
- 支路潮流量测
- 电压幅值量测

**估计方法**:
- 加权最小二乘法(WLS)
- 不良数据检测与辨识
- 可观性分析

**教学优势**:
- 系统规模适中，便于手工验证
- 各种量测配置可测试
- 不良数据场景可模拟

### 最优潮流

**优化目标**:
- 发电成本最小化
- 传输损耗最小化
- 电压偏差最小化

**约束条件**:
- 发电机出力限制
- 节点电压约束

**算法验证**:
- 内点法(Interior Point Method)
- 序列二次规划(SQP)
- 粒子群优化(PSO)
- 遗传算法(GA)

### 特点对比

**IEEE 14节点系统**:
- 优点: 规模小、计算快、数据完整、易于理解
- 缺点: 缺乏复杂性，不适用于高级研究
- 适用: 教学、基础算法验证、软件测试

**IEEE 30节点系统**:
- 优点: 包含优化相关数据（成本函数）
- 缺点: 发电机数量较少
- 适用: 最优潮流、经济调度研究

**IEEE 39节点系统**:
- 优点: 发电机数量多，适合暂态稳定分析
- 缺点: 数据复杂度高
- 适用: 暂态稳定、小信号稳定分析

**IEEE 118节点系统**:
- 优点: 大规模系统，接近实际电网规模
- 缺点: 计算量大，调试困难
- 适用: 大规模算法验证、并行计算测试

### 选择建议

| 研究目的 | 推荐系统 | 理由 |
|----------|----------|------|
| 教学演示 | IEEE 14 | 简单易懂，手工验证可行 |
| 算法初步验证 | IEEE 14/30 | 快速迭代，调试方便 |
| 优化研究 | IEEE 30/118 | 包含完整经济数据 |
| 稳定分析 | IEEE 39 | 发电机数量适中 |
| 大规模测试 | IEEE 118 | 接近实际系统规模 |

```matlab
% 节点数据: [bus_i type Pd Qd Gs Bs area Vm Va baseKV zone Vmax Vmin]
```

- MATPOWER安装目录: `matpower/case14.m`

**数据文件**:

**导入方式**:
- 通过File -> Open打开示例数据
