# VSC-HVDC五端测试系统

## 概述

VSC-HVDC五端测试系统是多端直流电网(MTDC - Multi-Terminal DC)研究中最具代表性的基准测试系统之一。该系统由五个电压源换流站(VSC)通过直流网络互联构成，用于验证多端直流系统的控制策略、功率分配算法、故障隔离机制以及运行特性分析。五端系统既保留了足够复杂的网络结构以反映实际MTDC电网的关键特性，又保持了适中的规模便于学术研究和大规模仿真验证。

五端VSC-HVDC系统的设计参考了中国张北柔性直流电网示范工程、欧洲地中海环网(MedLink)以及北欧North Sea Wind Power Hub等实际工程的前期研究需求。张北四端柔性直流电网工程(世界上首个输送大规模可再生能源的四端环形柔性直流电网)的成功投运，验证了多端直流技术的工程可行性，而五端测试系统在此基础上增加了一个换流站，进一步扩展了网络复杂度，为更复杂的控制策略和保护方案研究提供了平台。

该测试系统在电力系统研究中的应用价值体现在多个层面：首先，它是验证MTDC系统主从控制、下垂控制、电压裕度控制等多种控制策略的标准平台；其次，它用于研究多端直流系统的功率重分配、潮流优化和电压协调问题；第三，它是评估直流故障特性、开发故障检测与隔离方案的重要工具；第四，它为含高比例可再生能源的交直流混联系统动态特性研究提供了基准场景。

## 系统特点

### 规模与结构

**基本参数**:
- 换流站数量: 5个VSC换流站
- 额定容量: 每端100-600 MW(总计1500-2000 MW)
- 直流电压等级: ±200 kV至±500 kV(典型值±320 kV)
- 直流网络拓扑: 放射状或环状可配置
- 交流系统频率: 50 Hz或60 Hz

**网络结构**:
- 送端换流站: 2-3个，连接风电场或传统电源
- 受端换流站: 2-3个，连接负荷中心或主网
- 直流线路: 4-6条，电缆或架空线混合
- 交流侧网络: 可简化为理想电压源或连接详细交流系统模型

**拓扑特征**:
- 典型拓扑1(放射状): 中心换流站连接四个终端，类似星形结构
- 典型拓扑2(环状): 五个换流站形成闭合环网，提高供电可靠性
- 典型拓扑3(链状): 串联结构，适用于长距离输电走廊
- 混合拓扑: 放射状与环状结合，兼顾经济性与可靠性

## 控制系统

### 主从控制(Master-Slave Control)

主从控制是MTDC系统最基本的控制策略之一，其核心理念是在多端直流网络中选择一个换流站作为主站，负责维持直流电压稳定，其余换流站作为从站，按照设定的功率指令运行。

**主站控制**:
- 控制目标: 维持直流电压恒定
- 控制模式: Udc-Q控制(直流电压-无功功率)
- 电压参考值: 1.0 p.u. (额定直流电压)
- 电压控制带宽: 50-200 Hz
- 功率调节能力: 自动平衡系统功率盈余/缺额

**从站控制**:
- 控制目标: 维持有功和无功功率恒定
- 控制模式: P-Q控制(有功-无功功率)
- 功率参考值: 根据调度指令或优化结果设定
- 功率调节范围: 0.1-1.0 p.u.
- 功率变化率: 受限于换流器容量和系统约束

**主站选择原则**:
- 优先选择容量最大的换流站作为主站
- 送端或受端换流站均可作为主站
- 主站故障时需进行主站切换
- 多主站运行可实现冗余备份

### 下垂控制(Droop Control)

下垂控制模拟同步发电机的频率-功率下垂特性，使多个换流站能够共同参与直流电压调节，实现功率的自动分配，提高系统冗余性和可靠性。

**下垂特性方程**:
```
ΔUdc = -Rd × (P - P0)
```
其中:
- ΔUdc: 直流电压偏差
- Rd: 下垂系数(电压-功率斜率)
- P: 实际有功功率
- P0: 有功功率参考值

**下垂系数设计**:
| 换流站 | 下垂系数Rd(kV/MW) | 额定容量(MW) | 调节范围 |
|--------|-------------------|--------------|----------|
| VSC1 | 0.05-0.1 | 600 | ±10% |
| VSC2 | 0.08-0.15 | 400 | ±10% |
| VSC3 | 0.1-0.2 | 300 | ±10% |
| VSC4 | 0.06-0.12 | 500 | ±10% |
| VSC5 | 0.08-0.15 | 400 | ±10% |

**下垂控制优点**:
- 无需通信即可实现功率分配
- 单站故障不影响其他站运行
- 功率自动按下垂系数比例分配
- 支持即插即用功能

**下垂控制挑战**:
- 存在稳态电压偏差
- 功率分配比例固定，不够灵活
- 需要优化下垂系数以平衡响应速度和稳态精度

### 电压裕度控制(Voltage Margin Control)

电压裕度控制是日本学者针对MTDC系统提出的一种改进控制策略，通过设置电压裕度阈值，实现不同控制模式之间的平滑切换，避免多个换流站同时调节直流电压导致的控制冲突。

**控制原理**:
- 正常运行: 主站维持电压，从站定功率运行
- 功率越限: 从站切换至电压控制模式
- 主站故障: 电压裕度最大的从站自动接管电压控制

**电压裕度设置**:
- 上限裕度: Udc_max = 1.05-1.1 p.u.
- 下限裕度: Udc_min = 0.9-0.95 p.u.
- 死区设置: ±0.01-0.02 p.u.(防止频繁切换)

**模式切换逻辑**:
```
if Udc > Udc_max then
    切换至降功率模式
else if Udc < Udc_min then
    切换至增功率模式
else
    维持定功率模式
end if
```

**应用特点**:
- 无需高速通信支持
- 控制模式切换平滑
- 适应功率波动和故障场景
- 适用于放射状和环状拓扑

### 功率均分控制

功率均分控制是一种高级MTDC控制策略，旨在实现多端直流系统中功率的优化分配，考虑线路损耗、换流站损耗以及系统约束条件。

**优化目标**:
- 最小化总传输损耗
- 均衡各换流站利用率
- 满足功率平衡约束
- 遵守设备容量限制

**约束条件**:
- 功率平衡: ΣPi = 0 (忽略损耗)
- 容量约束: |Pi| ≤ Pi_max
- 电压约束: Udc_min ≤ Udc_i ≤ Udc_max
- 电流约束: |Idc_i| ≤ Idc_max

**优化方法**:
- 集中式优化: 中央控制器计算最优功率分配
- 分布式优化: 基于一致性算法的分布式协调
- 模型预测控制(MPC): 考虑未来状态进行优化

**通信需求**:
- 测量信息: 各换流站电压、电流、功率
- 控制指令: 功率参考值或电压参考值
- 通信延迟: 通常要求<100ms
- 通信可靠性: 需要冗余通信通道

## 典型测试场景

### 功率重分配测试

**测试目的**: 验证MTDC系统在功率指令变化时的动态响应特性，评估控制系统的跟踪性能和稳定性。

**测试设置**:
- 初始状态: 系统稳态运行，各换流站按设定功率运行
- 扰动类型: 阶梯式功率变化或斜坡式功率变化
- 变化幅度: 单站功率变化10%-50%
- 变化速率: 10-100 MW/s

**典型场景**:
1. **风电功率波动模拟**:
   - VSC1功率从500MW降至300MW(模拟风速下降)
   - 功率变化率: 50 MW/s
   - 持续时间: 4s

2. **负荷突增模拟**:
   - VSC4功率从-400MW变为-500MW(负荷增加)
   - 阶梯变化，瞬时完成
   - 观察主站功率响应

3. **功率反转测试**:
   - VSC2从送端(+300MW)切换为受端(-100MW)
   - 评估功率反转过程中的稳定性

**评价指标**:
- 调节时间: 功率达到新稳态的95%所需时间
- 超调量: 功率或电压的最大偏差
- 稳态误差: 实际功率与指令功率的偏差
- 振荡次数: 达到稳态前的振荡周期数

**预期结果**:
- 调节时间: 50-200ms(取决于控制带宽)
- 超调量: <5%
- 稳态误差: <1%
- 直流电压波动: <±3%

### 单端故障退出测试

**测试目的**: 评估MTDC系统在单端换流站故障退出后的生存能力，验证功率重分配和电压恢复能力。

**故障设置**:
- 故障类型: 换流站闭锁(Blocking)、交流侧故障、直流侧故障
- 故障位置: VSC1(主站)或VSC2(从站)
- 故障清除: 故障换流站退出运行，其余站继续运行
- 切除时间: 故障后20-100ms

**测试场景**:
1. **主站故障退出**:
   - VSC1(主站)在t=1s时故障闭锁
   - 系统失去主站电压控制
   - 评估电压裕度控制或下垂控制的响应
   - 观察新主站的自动切换过程

2. **从站故障退出**:
   - VSC2(送端从站)故障退出
   - 系统损失200-400MW送电能力
   - 评估功率缺额在主站和其他从站之间的分配

3. **主站切换测试**:
   - 主站VSC1计划性退出
   - 预先指定VSC4作为新主站
   - 验证主从切换的平滑性

**关键指标**:
- 直流电压最大跌落: <10-15%
- 电压恢复时间: <200-500ms
- 功率重分配时间: <100-300ms
- 其余换流站是否保持稳定运行

**保护配合**:
- 直流断路器动作时间: <10ms
- 换流站保护动作时间: <5ms
- 通信切换时间: <50ms

### 直流故障隔离测试

**测试目的**: 验证MTDC系统直流故障检测、定位和隔离能力，评估故障对系统其余部分的影响以及恢复策略。

**故障类型**:
1. **极间短路故障**:
   - 故障电阻: 0.1-10 Ω
   - 故障位置: 线路中点或端点
   - 故障电流: 可达额定电流的5-10倍

2. **单极接地故障**:
   - 接地电阻: 1-100 Ω
   - 故障极: 正极或负极
   - 故障电流: 较极间故障小，但持续时间长

3. **断线故障**:
   - 线路断开导致网络拓扑变化
   - 评估环网变放射状后的功率重分配

**测试场景**:
1. **线路故障隔离**:
   - Line 1-2在t=1s时发生极间短路
   - 直流断路器在5-10ms内动作隔离故障
   - 评估非故障线路的功率传输恢复

2. **换流站直流侧故障**:
   - VSC3直流侧出口故障
   - 换流站闭锁保护动作
   - 评估对直流网络电压的冲击

3. **永久性故障恢复**:
   - 故障隔离后尝试重合闸
   - 永久性故障时再次隔离
   - 评估故障定位准确性

**评价指标**:
- 故障检测时间: <1-5ms
- 故障定位精度: ±0.5-1km
- 隔离时间: <10ms(机械断路器)或<5ms(固态断路器)
- 故障期间电压跌落: <20%
- 非故障区域恢复时间: <100-300ms

### 风电功率波动测试

**测试目的**: 模拟大规模风电通过VSC-HVDC接入电网的场景，评估系统对风电功率波动的响应能力和控制策略的有效性。

**风电模型**:
- 风电场容量: 500-1000MW
- 风机类型: 永磁同步发电机(PMSG)或双馈感应发电机(DFIG)
- 风速模型: 平均风速叠加湍流和阵风

**波动场景**:
1. **湍流响应**:
   - 风速波动: ±10-20%的平均风速
   - 波动周期: 1-10s
   - 评估系统对高频功率波动的滤波能力

2. **阵风穿越**:
   - 阵风幅度: 5-15m/s
   - 阵风持续时间: 10-60s
   - 评估低电压穿越和高电压穿越能力

3. **风电场切出**:
   - 极端风速导致风电场全部切出
   - 功率损失: 500-1000MW瞬时为0
   - 评估主站功率平衡能力和频率支撑

**控制策略**:
- 有功功率变化率限制: 5-10%额定容量/分钟
- 利用VSC直流电容储能平抑功率波动
- 配合储能系统(如需要)进一步平滑输出

**评价指标**:
- 直流电压波动范围: <±5%
- 交流频率偏差: <±0.2Hz
- 功率变化率限制效果: 达标率>90%
- 低电压穿越成功率: >95%

## 研究应用

### MTDC控制策略研究

五端VSC-HVDC测试系统为MTDC控制策略的研究开发提供了标准化平台，主要研究方向包括:

**高级控制策略**:
- [[model-predictive-control]]: 考虑多约束的模型预测控制
- [[adaptive-droop]]: 自适应下垂系数调节
- [[distributed-control]]: 基于多智能体的分布式协调控制
- [[hierarchical-control]]: 分层控制架构(站级-系统级-网络级)

**控制性能优化**:
- 多时间尺度协调控制
- 交流侧和直流侧联合优化
- 考虑通信延迟的鲁棒控制设计
- 故障情况下的紧急控制策略

**控制稳定性分析**:
- 小信号稳定性分析
- 大扰动暂态稳定性评估
- 控制参数优化设计
- 多换流站交互作用分析

### 多尺度建模研究

五端系统支持不同时间尺度的建模需求，用于研究MTDC系统的多物理场耦合特性:

**电磁暂态建模(EMT)**:
- 开关级详细模型: 用于分析换流器内部动态
- 平均值模型(AVM): 用于系统级快速仿真
- 混合仿真: 详细模型与简化模型结合

**机电暂态建模**:
- 与详细交流系统模型接口
- 研究MTDC对交流电网稳定性的影响
- 评估直流功率调制对低频振荡的阻尼效果

**稳态/潮流分析**:
- MTDC潮流计算算法
- 交直流联合潮流
- 最优潮流(OPF)与损耗最小化

### 故障保护研究

五端系统是研究MTDC故障特性和保护方案的重要工具:

**故障分析**:
- 直流故障电流特性分析
- 故障传播机理研究
- 故障定位算法验证
- 故障对换流站的影响评估

**保护方案**:
- 直流断路器(DCCB)配置与协调
- 故障限流器(FCL)应用

## 仿真案例

### 案例1: 功率重分配动态响应

**系统配置**:
- 拓扑: 环状五端系统
- 控制策略: 主从控制(VSC1为主站)
- 初始运行点: VSC1(+500MW), VSC2(+300MW), VSC3(+200MW), VSC4(-400MW), VSC5(-600MW)

**扰动设置**:
- t=1.0s: VSC2功率指令从+300MW变为+200MW(降功率100MW)
- 功率变化率: 100 MW/s
- 总仿真时间: 3s

**仿真结果分析**:

1. **功率响应**:
   - VSC2功率从300MW平滑下降至200MW
   - 调节时间约80ms，无超调
   - VSC1作为主站，功率自动从500MW调整至400MW以平衡功率缺额
   - 其余换流站功率基本保持不变

2. **直流电压响应**:
   - 电压瞬时跌落约2%(从1.0p.u.降至0.98p.u.)
   - 电压恢复时间约150ms
   - 各站电压波动范围<±3%

3. **交流侧响应**:
   - 送端交流电压基本稳定
   - 受端交流频率偏差<0.05Hz
   - 无功功率支撑正常

**结论**: 主从控制策略下，功率重分配动态响应快速，直流电压波动在允许范围内，系统保持稳定运行。

### 案例2: 主站故障与切换

**系统配置**:
- 拓扑: 放射状五端系统(星形)
- 控制策略: 电压裕度控制
- 主站: VSC1(额定600MW)
- 备用主站: VSC4(具备电压控制能力)

**故障设置**:
- t=1.0s: VSC1发生严重故障，直流侧过流保护动作
- t=1.02s: VSC1闭锁退出运行
- 系统失去主站，直流电压开始下降

**动态过程**:

1. **故障瞬间(t=1.0s-1.02s)**:
   - 直流侧电流急剧上升至额定值的300%
   - 直流电压开始下降
   - 保护系统检测到过流

2. **主站退出(t=1.02s)**:
   - VSC1闭锁，停止功率传输
   - 系统剩余功率: VSC2(+300MW)+VSC3(+200MW)-VSC5(-400MW)=+100MW盈余
   - 直流电压持续下降

3. **自动切换(t=1.05s)**:
   - VSC4检测到直流电压低于下限裕度(Udc<0.95p.u.)
   - VSC4自动从P-Q控制切换为Udc-Q控制
   - VSC4接管主站功能，吸收盈余功率100MW

4. **恢复稳态(t=1.2s)**:
   - 直流电压恢复至0.98-1.0p.u.范围
   - VSC4功率稳定在-500MW(原-400MW+新增-100MW)
   - 系统在新的运行点建立稳态

**关键指标**:
- 直流电压最低值: 0.88p.u.(跌落12%)
- 电压恢复时间: 约200ms
- 功率重新分配完成时间: 约180ms
- 非故障换流站未发生跳闸

**结论**: 电压裕度控制策略成功实现了主站故障时的自动切换，系统在失去主站后能够快速建立新的稳态运行点，证明了MTDC系统具备较强的故障生存能力。

### 案例3: 直流线路故障隔离

**系统配置**:
- 拓扑: 环状五端系统
- 线路: Line 2-3连接VSC2与VSC3
- 保护配置: 各线路配置混合式直流断路器

**故障设置**:
- t=1.0s: Line 2-3中点发生极间短路故障
- 故障电阻: 0.5Ω(近似金属性短路)
- t=1.005s: 断路器检测并动作
- t=1.01s: 故障线路隔离

**故障过程分析**:

1. **故障发生(t=1.0s)**:
   - 故障点电压瞬间跌落至接近0
   - 故障电流由线路分布电容和换流站提供
   - 电容放电电流峰值达额定电流的8-10倍
   - 故障电流上升率di/dt极高

2. **故障检测(t=1.0s-1.005s)**:
   - VSC2和VSC3检测到电流突增和电压突降
   - 保护算法在1ms内确认故障
   - 触发直流断路器分闸命令

3. **故障隔离(t=1.005s-1.01s)**:
   - 混合式断路器: 固态开关快速开断(0.1ms)+机械开关承载额定电流
   - 故障电流在5ms内降至0
   - 限流电抗器抑制电流上升

4. **系统恢复(t=1.01s-1.5s)**:
   - 环网变为放射状拓扑(Line 2-3断开)
   - 潮流自动重新分配至剩余线路
   - VSC2和VSC3功率通过其他路径传输
   - 直流电压在300ms内恢复至正常范围

**功率重分配结果**:
- 故障前: Line 1-2(200MW), Line 2-3(150MW), Line 3-4(100MW)
- 故障后: Line 1-2(350MW,增加), Line 3-4(250MW,增加), Line 2-3(0MW,断开)
- 功率绕经VSC1和VSC4传输，线路负载率有所上升但仍处于安全范围

**评价**:
- 故障检测时间: 1ms
- 隔离时间: 5ms(固态)或10ms(混合)
- 非故障线路电压波动: <5%
- 系统保持连续运行，未发生换流站闭锁

**结论**: 直流断路器配合快速保护能够实现故障线路的快速隔离，环状拓扑的冗余性保证了系统在单一线路故障时仍能维持运行，验证了MTDC直流保护方案的有效性。

### 案例4: 风电功率波动平抑

**系统配置**:
- VSC1连接1000MW海上风电场
- 控制策略: 改进下垂控制+虚拟惯量控制
- 风电模型: 包含100台10MW永磁直驱风机

**风速场景**:
- 平均风速: 12m/s(额定风速)
- 湍流强度: 10%
- 模拟时长: 60s

**风电功率特性**:
- 瞬时功率波动范围: 800-1000MW(±10%)
- 波动频率: 0.1-1Hz
- 最大变化率: 50MW/s

**控制策略对比**:

1. **传统下垂控制**:
   - 直流电压波动: ±4.5%
   - 受端交流频率波动: ±0.15Hz
   - 功率变化率限制效果: 一般

2. **改进控制(下垂+虚拟惯量)**:
   - 利用VSC直流侧电容提供虚拟惯量支持
   - 直流电压波动: ±2.8%(降低38%)
   - 受端交流频率波动: ±0.08Hz(降低47%)
   - 功率变化率限制: 30MW/s(有效平滑)

3. **附加储能系统**:
   - 配置50MW/100MWh电池储能
   - 直流电压波动: ±1.5%
   - 功率输出平滑度显著提高

**虚拟惯量控制原理**:
```
ΔP_inertia = -J_virtual × d(Udc)/dt
```
其中J_virtual为虚拟惯量系数，通过控制直流电压变化率来模拟同步机惯量响应。

**能量分析**:
- 直流电容储能变化: ±5MJ(对应±2%电压变化)
- 虚拟惯量支撑持续时间: 2-5s
- 储能系统充放电深度: <20%/循环

**结论**: 虚拟惯量控制能够有效利用VSC直流侧电容储能平抑风电功率波动，配合储能系统可进一步提升平滑效果，为大规模可再生能源通过MTDC并网提供了可行的技术方案。
