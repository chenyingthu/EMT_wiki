# 储能变流器(PCS)测试系统

## 概述

储能变流器(Power Conversion System, PCS)是电池储能系统(Battery Energy Storage System, BESS)的核心能量转换设备，负责实现电池直流电能与电网交流电能之间的双向转换。PCS作为连接电池储能单元与电力系统的关键接口，不仅承担着功率变换的基本功能，还集成了系统控制、保护协调、能量管理等多种功能。

PCS测试系统是用于验证储能变流器性能、功能和可靠性的专用测试平台，广泛应用于PCS产品研发、型式试验、出厂检测、并网认证等场景。通过构建标准化的测试系统，可以对PCS的各项电气性能指标、控制功能、保护特性进行全面评估，确保产品满足电网接入标准和应用场景需求。

储能变流器在电力系统中具有广泛应用场景，包括一次调频、二次调频、削峰填谷、备用电源、电能质量治理、新能源消纳等领域。不同应用场景对PCS的性能要求各有侧重，测试系统需要覆盖这些差异化需求。

## 系统结构

### 总体架构

PCS测试系统的基本架构包括以下几个核心部分：

**1. 电池模拟单元**
- 电池模拟器或实际电池组
- 提供可编程的直流电压/电流源
- 模拟不同类型电池的充放电特性
- 模拟电池老化、故障等异常状态

**2. 储能变流器(PCS)**
- DC/AC双向变流器
- 功率变换主电路
- 控制与保护系统
- 人机交互界面

**3. 电网模拟单元**
- 电网模拟器或实际电网连接
- 可编程电压源
- 模拟电网电压、频率波动
- 模拟电网故障和电能质量问题

**4. 负载/能量回馈单元**
- 可编程负载
- 能量回馈装置
- 实现测试能量的循环利用

**5. 测量与监控系统**
- 高精度功率分析仪
- 示波器与数据采集系统
- 绝缘监测装置
- 温度/湿度等环境监控

### 电气拓扑

**主电路拓扑**:
```
电池组/模拟器 ←→ DC母线 ←→ DC/AC变流器 ←→ AC滤波器 ←→ 隔离变压器 ←→ 电网/负载
```

**DC侧结构**:
- 电池接口: BMS通信、预充电路、熔断保护
- DC母线: 支撑电容、母线电压检测
- DC/DC变换器(可选): 升降压匹配、多电池组管理

**AC侧结构**:
- 逆变桥臂: IGBT/SiC功率模块
- 滤波器: LCL滤波器或L滤波器
- 隔离变压器: 工频变压器或高频变压器
- 并网开关: 断路器、接触器

### 控制系统架构

**分层控制结构**:

| 层级 | 功能 | 响应时间 |
|------|------|----------|
| 系统层 | 能量管理、模式调度、通信 | 秒级-分钟级 |
| 控制层 | 功率控制、电压调节、保护 | 毫秒级 |
| 调制层 | PWM生成、脉冲分配 | 微秒级 |

**控制硬件**:
- 主控制器: DSP/FPGA/ARM处理器
- 通信接口: CAN、RS485、Ethernet
- 采样系统: 电压/电流传感器、ADC
- 驱动电路: IGBT驱动、保护电路

## 工作模式

### 并网充电模式

**模式描述**:
从电网吸收电能，转换为直流电能为电池充电。

**控制目标**:
- 恒功率充电: 维持设定充电功率
- 恒流充电: 维持设定充电电流
- 恒压充电: 电池电压达到阈值时切换

**功率流向**:
```
电网 → AC/DC整流 → DC母线 → DC/DC降压 → 电池
```

**关键参数**:
- 充电效率: 96-98%
- 充电电流: 0.1C-1C可调
- 功率因数: ≥0.99(感性)

**应用场景**:
- 削峰填谷: 低谷电价时段充电
- 新能源消纳: 吸收风电/光伏过剩出力
- 备用准备: 保持电池满荷电状态

### 并网放电模式

**模式描述**:
将电池储存的电能转换为交流电馈入电网。

**控制目标**:
- 恒功率放电: 维持设定放电功率
- 恒流放电: 维持设定放电电流
- 跟踪调度: 响应电网AGC/AVC指令

**功率流向**:
```
电池 → DC/DC升压 → DC母线 → DC/AC逆变 → 电网
```

**关键参数**:
- 放电效率: 96-98%
- 放电深度: 80-90%DOD
- 功率因数: ≥0.99(容性)

**应用场景**:
- 削峰填谷: 高峰电价时段放电
- 调频服务: 参与电网一次/二次调频
- 需求响应: 响应电网调度指令

### 离网供电模式

**模式描述**:
脱离电网独立运行，为本地负荷供电。

**控制目标**:
- 恒压恒频: 维持电压/频率稳定
- 负荷跟踪: 功率平衡控制
- 多机并联: 功率均分与环流抑制

**控制策略**:
- V/f控制: 电压频率双闭环
- 下垂控制: 多机并联功率分配
- 虚拟同步机: 增强惯量支撑

**关键参数**:
- 电压精度: ±2%
- 频率精度: ±0.5Hz
- 波形畸变率: ≤3%

**应用场景**:
- 孤岛微网: 偏远地区独立供电
- 应急备用: 电网故障时保障关键负荷
- 计划孤岛: 有计划的离网运行

### 并离网平滑切换

**切换类型**:

**计划切换**:
- 离网转并网: 检测电网电压/频率/相位，同步后合闸
- 并网转离网: 断开并网开关，切换为V/f控制
- 切换时间: 100-500ms

**非计划切换**:
- 电网故障检测: 过/欠压、过/欠频、相位突变
- 快速脱网: 20ms内断开并网开关
- 无缝切换: UPS级切换(<10ms)

**切换策略**:
1. 检测电网状态
2. 判断切换条件
3. 执行模式切换
4. 稳定新运行点

**技术要求**:
- 切换冲击: 电压波动<10%
- 连续供电: 关键负荷不间断
- 故障穿越: 低电压穿越/高电压穿越

## 控制策略

### 恒功率控制

**控制原理**:
通过闭环控制使PCS输出功率跟踪设定值，维持功率恒定。

**控制框图**:
```
P_ref → [PI调节器] → [电流环] → [PWM] → [变流器] → P_out
         ↑                              ↓
         └────[功率计算]←───────────────┘
```

**控制参数**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| Kp | 0.1-1 | 比例增益 |
| Ki | 10-100 | 积分增益 |
| 带宽 | 10-100Hz | 功率环带宽 |

**应用场景**:
- 削峰填谷: 按调度计划设定功率
- 调频服务: 响应频率偏差信号
- 新能源平滑: 平抑功率波动

### 恒流控制

**控制原理**:
维持电池充放电电流恒定，保护电池并延长寿命。

**控制策略**:
- 充电恒流: 限制最大充电电流
- 放电恒流: 限制最大放电电流
- 电流闭环: 快速电流跟踪

**关键特性**:
- 电流精度: ±1%
- 响应速度: <10ms
- 过流保护: 1.2倍额定电流触发

**应用考量**:
- 电池特性: 不同电池类型电流限制不同
- SOC影响: 高/低SOC时降低电流
- 温度补偿: 根据温度调整限值

### 调频控制

**一次调频**:
根据系统频率偏差自动调整输出功率。

**调频特性**:
```
ΔP = - (1/R) × (f - f0) / f0 × Pn
```
其中: R为调差系数(4-5%), f0为额定频率, Pn为额定功率

**调频参数**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 调差系数 | 4-5% | 频率变化4-5%对应100%功率变化 |
| 响应时间 | <3s | 达到90%目标功率 |
| 调节精度 | ±0.1Hz | 稳态频率偏差 |

**二次调频(AGC)**:
- 接收调度中心AGC指令
- 调节容量: ±10%-100%Pn
- 调节速率: ≥1%Pn/s

### 削峰填谷控制

**优化目标**:
根据电价信号和负荷预测，优化充放电策略，实现经济效益最大化。

**控制策略**:

**基于电价的优化**:
```
电价低谷 → 充电
电价高峰 → 放电
电价平段 → 待机
```

**基于负荷的优化**:
- 预测次日负荷曲线
- 优化充放电时段和功率
- 降低需量电费

**约束条件**:
- 电池SOC限制: 20%-90%
- 充放电次数限制
- 功率限制: 不超过额定功率
- 容量衰减考虑

**算法实现**:
- 动态规划
- 模型预测控制(MPC)
- 强化学习优化

## 研究应用

### SOC管理研究

**SOC估算方法**:

**安时积分法**:
```
SOC(t) = SOC(0) + (1/Cn) × ∫η×I(t)dt
```
其中: Cn为额定容量, η为库伦效率

**开路电压法**:
- 建立SOC-OCV曲线
- 静置电压测量
- 适用于校准和初始化

**模型法**:
- 等效电路模型
- 状态观测器
- 卡尔曼滤波

**SOC均衡控制**:
- 被动均衡: 电阻耗散
- 主动均衡: 能量转移
- 均衡策略优化

### 电池建模研究

**等效电路模型**:

**Rint模型**:
```
Vbatt = OCV - I×Rint
```
简单但精度有限

**Thevenin模型**:
```
Vbatt = OCV - I×R0 - V1
RC回路描述极化效应
```
兼顾精度和复杂度

**二阶RC模型**:
```
Vbatt = OCV - I×R0 - V1 - V2
两个RC回路分别描述浓差极化和电化学极化
```
精度最高

**电化学模型**:
- 基于物理过程的建模
- P2D(Pseudo Two-Dimensional)模型
- 参数辨识方法

### 黑启动研究

**黑启动能力**:

**定义**:
当电网发生大面积停电时，储能系统能够不依赖外部电源，自行启动并为电网恢复提供电源支持的能力。

**技术条件**:
- 具备离网运行能力
- 足够的能量储备
- 电压/频率建立能力
- 负荷逐步加载能力

**控制策略**:

**软启动策略**:
```
步骤1: 建立空载电压
步骤2: 接入小负荷(变压器励磁)
步骤3: 逐步增加负荷
步骤4: 稳定运行后考虑并列
```

**负荷优先级管理**:
- 一级负荷: 调度中心、通信系统
- 二级负荷: 重要用户、辅助设备
- 三级负荷: 一般用户

**应用场景**:

**微网黑启动**:
- 光储微网
- 风储微网
- 多能互补微网

**电网恢复支撑**:
- 为传统电源提供启动电源
- 逐步恢复电网供电
- 协调其他储能参与
