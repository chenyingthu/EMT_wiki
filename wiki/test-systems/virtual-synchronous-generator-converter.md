## 概述

虚拟同步机(Virtual Synchronous Generator, VSG)技术是一种先进的构网型(Grid-Forming)变流器控制策略，通过模拟同步发电机的转子运动特性，使电力电子变流器具备类似同步机的惯量支撑和一次调频能力。VSG控制使逆变器能够主动参与电网频率和电压调节，为现代电力系统提供重要的频率稳定支撑，特别是在高比例可再生能源渗透的电网中具有重要的应用价值。

**核心思想**: 通过在变流器控制算法中嵌入同步发电机的摇摆方程(Swing Equation)，使逆变器对外呈现惯性响应特性和阻尼特性。

**主要优势**:
- 提供虚拟惯量支撑，减缓频率变化速率
- 实现一次调频功能，参与频率稳态调节
- 增强电网稳定性，提高抗扰动能力
- 支持孤岛运行和并网运行两种模式
- 多台VSG可并联运行实现功率均分

**技术定位**: VSG是构网型变流器的核心技术之一，与跟网型(Grid-Following)变流器相比，VSG具有主动支撑电网的能力，被视为未来100%新能源电力系统的关键技术。

## 系统架构

### 硬件拓扑

**主电路结构**:
```
直流侧(650-1000V) → DC-AC变流器 → LC滤波器 → 隔离变压器 → 交流电网(380V)
```

**关键组件**:
- **直流电源**: 光伏阵列、储能电池或直流母线
- **全桥逆变器**: IGBT或SiC MOSFET功率器件
- **LC滤波器**: 滤除开关谐波，典型截止频率1-2kHz
- **隔离变压器**: 提供电气隔离和电压匹配
- **测量单元**: 电压、电流、频率、相位检测

### 控制系统架构

**分层控制结构**:

**功率级控制**(内环):
- 电流内环控制: 响应时间1-10ms
- 电压外环控制: 响应时间10-100ms
- PWM调制: 载波频率5-20kHz

**VSG算法层**(外环):
- 有功-频率控制: 模拟原动机和调速器
- 无功-电压控制: 模拟励磁系统
- 虚拟惯量计算: 实时惯量响应
- 阻尼控制: 提供振荡抑制

**系统级协调**(上层):
- 功率参考给定
- 模式切换控制
- 保护逻辑
- 通信接口

## VSG控制策略

### 有功-频率控制

**下垂特性**:
VSG通过有功-频率下垂特性参与电网频率调节，模拟同步发电机的一次调频功能。

**控制目标**:
- 频率偏差时自动调整有功输出
- 多台VSG并联时实现功率均分
- 提供频率支撑，防止频率崩溃

**下垂曲线方程**:
```
P = P_ref - (1/R) × (f - f_nom)
```
其中:
- P: VSG输出有功功率
- P_ref: 有功功率参考值
- R: 频率下垂系数 (Hz/p.u.)
- f: 电网实际频率
- f_nom: 额定频率(50Hz)

**下垂系数选择**:
- 较小R值(0.02): 强频率支撑，但功率波动大
- 较大R值(0.1): 弱频率支撑，功率波动小
- 典型值: 0.04 (对应4%下垂)

### 无功-电压控制

**电压下垂特性**:
VSG通过无功-电压下垂特性参与电网电压调节，模拟同步发电机的励磁调节功能。

**控制目标**:
- 电压偏差时自动调整无功输出
- 维持并网点电压稳定
- 支持无功功率分配

**下垂曲线方程**:
```
Q = Q_ref - Kv × (V - V_nom)
```
其中:
- Q: VSG输出无功功率
- Q_ref: 无功功率参考值
- Kv: 电压下垂系数
- V: 并网点实际电压
- V_nom: 额定电压

**控制模式**:
- **恒无功模式**: 固定Q_ref，不参与电压调节
- **恒电压模式**: 固定V_ref，全力支撑电压
- **下垂模式**: 平衡无功分配和电压支撑

### 虚拟惯量控制

**惯量支撑原理**:
传统同步发电机通过旋转转子储存动能，在频率变化时释放或吸收能量。VSG通过控制算法模拟这一特性。

**惯量功率计算**:
```
P_inertia = J × ω × (dω/dt) ≈ 2H × (df/dt)
```
其中:
- J: 虚拟转动惯量
- ω: 角频率
- H: 惯量常数
- df/dt: 频率变化率(ROCOF)

**惯量响应特性**:
- 频率下降时(df/dt < 0): 释放有功功率，抑制频率跌落
- 频率上升时(df/dt > 0): 吸收有功功率，抑制频率上升
- 惯量响应时间: 毫秒级

**惯量常数影响**:
| 惯量常数H | 频率支撑能力 | 系统稳定性 | 适用场景 |
|-----------|--------------|------------|----------|
| 0.5-1s | 弱 | 高 | 强电网 |
| 2-5s | 中等 | 中 | 一般微电网 |
| 5-10s | 强 | 需阻尼 | 弱电网/孤岛 |

### 阻尼控制

**阻尼作用**:
抑制系统功率振荡，提高动态稳定性。VSG的阻尼控制模拟同步发电机的阻尼绕组效应。

**阻尼功率计算**:
```
P_damping = D × (ω - ω_nom)
```
其中:
- D: 阻尼系数
- ω: 实际角频率
- ω_nom: 额定角频率

**阻尼效果**:
- 抑制功角振荡
- 减少频率超调
- 加快系统稳定

**阻尼系数设计**:
- 过小: 振荡抑制不足，系统失稳风险
- 过大: 响应迟缓，惯量支撑效果减弱
- 最优: 根据系统特征选择，典型值50-100 p.u.

## 控制方程

### 转子运动方程

**摇摆方程(Swing Equation)**:
VSG的核心是模拟同步发电机的转子运动方程，描述转子角速度与电磁转矩和机械转矩的关系。

```
J × (dω/dt) = T_m - T_e - D × (ω - ω_nom)
```
其中:
- J: 虚拟转动惯量 (kg·m²)
- ω: 转子角速度 (rad/s)
- T_m: 机械转矩 (N·m)，由有功下垂控制产生
- T_e: 电磁转矩 (N·m)，等于输出有功除以角速度
- D: 阻尼系数
- ω_nom: 额定角频率 (314.16 rad/s)

**功率形式方程**:
```
2H × S_n × (df/dt) = P_m - P_e - D_p × (f - f_nom)
```
其中:
- H: 惯量常数 (s)
- S_n: 额定容量 (VA)
- P_m: 机械功率 (W)
- P_e: 电磁功率/输出有功 (W)
- D_p: 频率下垂系数

### 电磁功率计算

**瞬时有功功率**:
```
P_e = v_d × i_d + v_q × i_q
```

**瞬时无功功率**:
```
Q_e = v_q × i_d - v_d × i_q
```
其中v_d, v_q为d-q坐标系下的电压，i_d, i_q为电流。

**功率测量滤波**:
由于开关频率谐波存在，功率测量需要低通滤波:
```
P_filt = α/(s + α) × P_e
```
其中α为滤波带宽，典型值10-100 rad/s。

### 内电势控制

**虚拟内电势生成**:
VSG控制生成虚拟内电势E，其幅值和相位决定了变流器的输出特性。

**内电势幅值**:
```
E = E_nom + Kv × (Q_ref - Q_e)
```
或由电压下垂决定:
```
E = V_nom - Dq × (Q_e - Q_ref)
```

**内电势相位角**:
```
θ = ∫(ω - ω_g)dt
```
其中ω为VSG虚拟角频率，ω_g为电网角频率。

**内电势与端电压关系**:
```
E∠δ = V∠0 + I×(R + jX)
```
其中δ为功角，R和X为虚拟阻抗。

### 阻抗环控制

**虚拟阻抗**:
为了模拟同步发电机的阻抗特性和实现功率解耦，VSG引入虚拟阻抗。

**虚拟阻抗方程**:
```
V_ref = E - I × Z_v
Z_v = R_v + jX_v
```
其中:
- Z_v: 虚拟阻抗
- R_v: 虚拟电阻 (0.01-0.05 p.u.)
- X_v: 虚拟电抗 (0.05-0.2 p.u.)

**虚拟阻抗作用**:
- 功率解耦: 减小有功-无功耦合
- 限流保护: 故障时限制输出电流
- 改善稳定性: 调整系统阻尼特性

**自适应虚拟阻抗**:
根据运行工况动态调整:
```
Z_v = Z_nom × (1 + k×|I - I_nom|)
```

## 典型测试场景

### 负载突增测试

**测试目的**:
验证VSG在负载突变时的频率支撑能力和动态响应特性。

**测试设置**:
- 初始状态: VSG带50%额定负载稳定运行
- 负载突变: 增加50%额定负载(阶跃变化)
- 测量变量: 频率、有功功率、电压

**性能评估指标**:
| 指标 | 典型要求 | 说明 |
|------|----------|------|
| 频率跌落最低点 | >49.5 Hz | 避免低频减载 |
| 频率变化率 | <2 Hz/s | 限制ROCOF |
| 恢复时间 | <5s | 达到新稳态 |
| 稳态频率偏差 | <0.5 Hz | 满足电能质量 |

**预期响应过程**:
1. t=0: 负载突增，频率开始下降
2. t=0-0.5s: 惯量支撑启动，释放有功功率
3. t=0.5-3s: 下垂控制调节，功率重新分配
4. t=3-5s: 系统达到新稳态

### 频率阶跃测试

**测试目的**:
评估VSG在电网频率阶跃变化时的响应特性和调频能力。

**测试设置**:
- 初始状态: 并网运行，50%负载
- 频率阶跃: 电网频率从50Hz突降至49.5Hz
- 保持时间: 10-30s
- 频率恢复: 阶跃回50Hz

**响应特性分析**:
- **响应时间**: 从频率变化到功率调整的延迟
- **超调量**: 功率调整过程中的最大偏差
- **调节时间**: 达到90%稳态调整量的时间
- **稳态误差**: 最终功率与目标值的偏差

**性能要求**:
- 响应时间: <200ms
- 超调量: <20%
- 调节时间: <2s
- 稳态误差: <5%

### 孤岛切换测试

**测试目的**:
验证VSG在并网/孤岛模式切换时的无缝切换能力和稳定性。

**并网转孤岛测试**:
- 初始状态: VSG并网运行，承担部分负载
- 切换操作: 断开并网开关
- 验证项目:
  - 电压连续性
  - 频率稳定性
  - 负载供电不中断

**孤岛转并网测试**:
- 初始状态: VSG孤岛带负载运行
- 同期检测: 电压、频率、相位同步
- 并网操作: 闭合并网开关
- 验证项目:
  - 无冲击电流
  - 功率平滑转移

**切换性能指标**:
| 指标 | 要求 | 说明 |
|------|------|------|
| 电压波动 | <10% | 切换过程中 |
| 频率偏差 | <1Hz | 切换瞬间 |
| 切换时间 | <100ms | 检测+执行 |
| 功率波动 | <20% | 切换后恢复时间 |

### 并联运行测试

**测试目的**:
验证多台VSG并联运行时的功率均分性能和环流抑制能力。

**测试配置**:
- 2-4台VSG并联
- 共同带载(总负载不超过总容量)
- 各VSG采用相同下垂系数

**功率均分测试**:
- 稳态功率均分: 各VSG输出有功/无功比例
- 动态功率均分: 负载变化过程中的功率分配
- 均分精度: 各单元功率偏差范围

**环流测试**:
- 测量并联VSG间的环流电流
- 评估环流抑制效果
- 验证虚拟阻抗作用

**性能要求**:
- 有功均分误差: <5%
- 无功均分误差: <10%
- 环流电流: <10%额定电流
- 功率振荡: 阻尼比>0.3

### 短路故障测试

**测试目的**:
验证VSG在电网短路故障时的低电压穿越能力和故障恢复特性。

**故障类型**:
- 三相短路故障
- 两相短路故障
- 单相接地故障

**测试步骤**:
1. 正常并网运行
2. 在并网点设置短路故障
3. 故障持续0.1-0.5s
4. 故障清除
5. 观察恢复过程

**性能要求**:
- 低电压穿越: 电压跌至20%时不脱网
- 故障电流限制: <2倍额定电流
- 恢复时间: 电压恢复后<1s恢复正常
- 频率稳定: 故障期间频率偏差<2Hz

### ROCOF指标

**频率变化率(Rate of Change of Frequency)**:
ROCOF是评估VSG惯量支撑能力的关键指标。

**ROCOF限制**:
| 场景 | ROCOF限值 | 说明 |
|------|-----------|------|
| 正常运行 | <0.5 Hz/s | 避免设备保护动作 |
| 故障后 | <2 Hz/s | 允许短时越限 |
| 极端情况 | <5 Hz/s | 保护动作阈值 |

**惯量支撑效果**:
- 有VSG支撑: ROCOF降低50-80%
- 多台并联: ROCOF进一步降低
- 惯量常数H越大: ROCOF抑制效果越好

### 同步稳定性指标

**功角稳定**:
- 最大功角偏差: <60°
- 功角振荡阻尼比: >0.3
- 第一摆稳定: 不失去同步

**小信号稳定**:
- 特征值实部: <0 (稳定)
- 振荡频率: 0.2-2 Hz
- 参与因子分析: 关键模态识别

**暂态稳定**:
- 临界切除时间: >100ms
- 暂态稳定裕度: >20%
- 电压恢复时间: <1s

## 研究应用

### 微电网控制

**应用场景**:
- 偏远地区微电网
- 工业园区微电网
- 海岛独立电网
- 应急备用电源

**VSG在微电网中的作用**:
- 作为主电源提供电压频率基准
- 实现即插即用(PnP)的新能源接入
- 提高微电网抗扰动能力
- 支持离网/并网双模式运行

**控制策略研究**:
- 主从控制: 单台VSG主导，其他跟网
- 对等控制: 多台VSG对等并联
- 分层控制: VSG+储能+负荷协调

### 惯量支撑

**惯量缺失问题**:
高比例新能源电网中，传统同步机被替代，系统惯量显著降低，导致:
- 频率变化率(ROCOF)增大
- 频率跌落深度增加
- 系统抗扰动能力下降

**VSG惯量支撑方案**:
- 分布式VSG提供虚拟惯量
- 聚合等效为大惯量电源
- 惯量自适应调节(根据系统需求)

**研究热点**:
- 最优惯量配置: 需要多少虚拟惯量
- 惯量分布优化: 在哪里配置VSG最有效
- 惯量与一次调频协调

### 频率稳定

**频率稳定问题**:
- 新能源功率波动导致频率波动
- 负荷突变引起频率跌落
- 连锁故障导致频率崩溃

**VSG频率控制策略**:
- 一次调频: 下垂控制快速响应
- 二次调频: AGC协调控制
- 紧急频率控制: 低频减载配合

**研究案例**:
- 英国电网频率事件分析(2019年8月)
- 南澳大停电教训与改进
- 中国高比例新能源电网频率特性

### 100%新能源系统

**系统特征**:
- 无传统同步发电机
- 全部由电力电子变流器供电
- 面临严峻的频率稳定挑战

**VSG关键作用**:
- 提供必要的惯量和阻尼
- 建立电压频率基准
- 实现自同步能力
- 支持黑启动

**技术挑战**:
- 变流器过载能力有限
- 储能容量约束
- 多VSG协调控制
- 保护配合问题

**研究进展**:
- 澳大利亚AEMO 100%可再生能源路线图
- 欧盟构网型变流器技术规范
- 中国新型电力系统建设

## 仿真建模

### EMT仿真模型

**详细模型**:
- 开关级变流器模型
- 精确的VSG控制算法
- LC滤波器和线路模型
- 负载和电网模型

**仿真步长**:
- 开关模型: 0.1-1 μs
- 平均模型: 10-100 μs
- 简化模型: 100 μs-1 ms

**常用仿真工具**:
- PSCAD/EMTDC: 电磁暂态仿真
- MATLAB/Simulink: 控制系统设计
- RT-LAB: 硬件在环测试
- PLECS: 电力电子专用

### 平均模型

**模型简化**:
- 忽略开关过程，使用平均等效
- 保留VSG动态和控制特性
- 提高仿真效率，适合大系统

**适用场景**:
- 大电网稳定性分析
- 多VSG协调控制研究
- 长时间动态仿真

### 阻抗模型

**序阻抗建模**:
- 正序阻抗: 对称运行分析
- 负序阻抗: 不对称故障分析
- 零序阻抗: 接地故障分析

**阻抗扫描**:
- 频率扫描分析稳定性
- 阻抗比判据
- 谐振分析
