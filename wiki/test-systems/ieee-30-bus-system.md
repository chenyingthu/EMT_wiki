# IEEE 30节点系统 (IEEE 30-Bus System)

## 概述

IEEE 30节点系统是电力系统优化研究中最经典、最广泛使用的标准测试系统之一。该系统最初由IEEE于1970年代提出，专门用于最优潮流(Optimal Power Flow, OPF)问题的研究验证。作为一个小型到中型的测试系统，IEEE 30节点系统在保持足够复杂度的同时计算效率极高，成为验证各类优化算法、测试控制策略和进行教学培训的首选平台。

与IEEE 39节点系统侧重于暂态稳定分析不同，IEEE 30节点系统的设计初衷是为电力系统经济调度和优化问题提供标准化的测试平台。其网络结构、负荷分布和发电机配置经过精心设计，能够代表典型的中等规模电力系统特征，同时保持数据获取和计算的便利性。该系统包含30个节点、6台同步发电机、41条支路，总发电容量约335MW，总负荷约283.4MW，是研究最优潮流、无功优化、电容器配置和网络重构等问题的理想测试平台。

## 系统特点

### 规模与结构

**基本参数**:
- 节点数: 30个
- 发电机: 6台同步发电机（含1台平衡机）
- 负荷节点: 21个
- 支路数: 41条（输电线路+变压器）
- 基准容量: 100 MVA
- 基准频率: 60 Hz

**网络结构**:
- 主网架: 132kV高压输电网络
- 配电网络: 33kV中压配电
- 辐射状与环网混合结构
- 多电压等级互联

**拓扑特征**:
- 部分网状结构
- 存在并联支路
- 变压器调压支路
- 可重构网络拓扑

### 电压等级

**输电电压**:
- 132kV: 主干网架电压等级
- 主要连接大型发电机和重要负荷

- 33kV: 区域配电电压等级
- 通过变压器与132kV主网连接

**变压器配置**:
- 多台调压变压器
- 固定变比和可调变比混合
- 实现电压等级变换和潮流控制

## 研究应用

### 最优潮流(OPF)

**问题定义**:
最优潮流问题旨在优化电力系统的运行状态，在满足各种约束条件的前提下，使目标函数达到最优。

**数学模型**:
$$\min_{x} f(x)$$
$$\text{s.t. } g(x) = 0 \quad \text{(等式约束)}$$
$$h(x) \leq 0 \quad \text{(不等式约束)}$$

其中:
- $f(x)$: 目标函数（发电成本、网损等）
- $g(x)$: 功率平衡方程
- $h(x)$: 运行约束（电压、线路容量等）

**IEEE 30节点OPF特点**:
- 变量数: 约60个（电压幅值、相角）
- 等式约束: 60个（有功、无功平衡）
- 不等式约束: 约100个（线路容量、电压限值、发电机出力）
- 非凸问题，存在多个局部最优解

**常用算法**:
- 内点法(Interior Point Method)
- 序列二次规划(SQP)
- 遗传算法(GA)
- 粒子群优化(PSO)
- 差分进化(DE)

**标准OPF结果**:
| 目标函数 | 最优值 | 参考算法 |
|----------|--------|----------|
| 发电成本最小 | ~800-804 $/h | 内点法 |
| 网损最小 | ~3.0-3.5 MW | 梯度法 |
| 电压偏差最小 | V偏差<0.05 p.u. | 多目标优化 |

### 无功优化

**优化目标**:
1. **有功网损最小化**: 提高输电效率
2. **电压质量最优**: 减少电压偏差
3. **无功补偿容量最小**: 降低设备投资

**控制变量**:
- 发电机端电压(PV节点)
- 可调变压器变比
- 并联电容器投切组数

**约束条件**:
- 发电机无功出力限值
- 变压器变比范围
- 电容器投切档位
- 节点电压范围(0.95-1.05 p.u.)
- 线路功率传输限值

**IEEE 30节点无功优化特点**:
- 离散变量与连续变量混合
- 非线性、非凸优化问题
- 多目标冲突（损耗vs电压质量）
- 存在多个局部最优解

**典型优化结果**:
- 网损降低: 15-25%
- 电压偏差改善: 30-50%
- 电容器投入组数: 8-15组

### 电容器配置优化

**问题描述**:
确定并联电容器的最优安装位置、容量和投切策略。

**优化方法**:
1. **灵敏度分析法**: 基于网损对无功注入的灵敏度
2. **启发式算法**: 遗传算法、粒子群优化、蚁群算法
3. **混合整数规划**: 处理离散投切问题

**候选节点选择**:
- 高无功负荷节点
- 电压偏低节点
- 网损灵敏度高的节点

**IEEE 30节点典型配置**:
- 最优节点: 3, 10, 24, 29
- 最优容量: 5-20 Mvar/节点
- 网损降低: 20-30%

### 网络重构

**问题定义**:
通过改变配电网络中开关的开闭状态，优化网络拓扑结构。

**应用场景**:
- 故障隔离和恢复供电
- 负荷均衡
- 网损最小化
- 电压改善

**IEEE 30节点重构特点**:
- 存在常闭开关(41条线路)
- 可断开部分支路形成辐射状网络
- 需保持网络连通性和供电可靠性

**常用算法**:
- 分支交换法
- 遗传算法
- 禁忌搜索
- 混合智能算法

### 电压控制

**控制策略**:
1. **分级电压控制**:
   - 一级: 发电机自动电压调节(AVR)
   - 二级: 区域协调控制
   - 三级: 全网优化

2. **无功电压协调**:
   - 发电机无功调节
   - 变压器分接头调节
   - 电容器投切配合

**IEEE 30节点电压特性**:
- 部分节点电压偏低(27-30)
- 长线路末端电压降落明显
- 可通过无功补偿改善

**电压稳定分析**:
- PV曲线分析
- 电压崩溃点计算
- 无功裕度评估

### 机组组合

**问题描述**:
确定各时段发电机组的开停机状态和出力分配。

**优化目标**:
- 运行成本最小
- 启停成本考虑
- 满足负荷需求
- 满足旋转备用要求

**IEEE 30节点机组组合**:
- 6台机组参与优化
- 24小时调度周期
- 启停时间约束
- 爬坡速率约束

## 优化测试案例

### 案例1: 经济调度(ED)

**目标函数**: 发电成本最小
$$\min \sum_{i=1}^{N_G} C_i(P_{Gi})$$

**约束条件**:
- 功率平衡: $\sum P_{Gi} = P_{D} + P_{loss}$
- 发电机出力限值: $P_{Gi}^{min} \leq P_{Gi} \leq P_{Gi}^{max}$

**最优解**:
| 机组 | 最优出力(MW) | 边际成本($/MWh) |
|------|--------------|-----------------|
| G1 | 176.4 | 3.32 |
| G2 | 48.5 | 3.45 |
| G3 | 21.4 | 2.68 |
| G4 | 21.9 | 3.62 |
| G5 | 12.0 | 3.60 |
| G6 | 12.0 | 3.60 |

**总成本**: 803.51 $/h

### 案例2: 无功优化

**优化变量**:
- 发电机端电压: 6个
- 变压器变比: 4个
- 电容器投切: 4个节点

**目标函数**: 有功网损最小
$$P_{loss} = \sum_{k=1}^{N_L} G_k(V_i^2 + V_j^2 - 2V_iV_j\cos\theta_{ij})$$

**优化结果**:
- 优化前网损: 5.85 MW
- 优化后网损: 4.52 MW
- 网损降低: 22.7%

**最优控制变量**:
| 变量 | 优化前 | 优化后 |
|------|--------|--------|
| V1 | 1.05 | 1.06 |
| V2 | 1.04 | 1.042 |
| T6-9 | 1.025 | 1.05 |
| Qc5 | 0 | 15 Mvar |
| Qc24 | 0 | 10 Mvar |

### 案例3: 多目标OPF

**目标函数**:
$$F = w_1 \frac{C}{C_{max}} + w_2 \frac{P_{loss}}{P_{loss,max}} + w_3 \frac{V_{dev}}{V_{dev,max}}$$

**Pareto最优前沿**:
- 成本vs网损的权衡关系
- 多组非支配解
- 根据偏好选择运行点

### 案例4: 含新能源的OPF

**风电接入场景**:
- 母线30接入10MW风电
- 风电出力不确定性处理
- 概率OPF或鲁棒OPF

**优化结果对比**:
| 场景 | 发电成本($/h) | 网损(MW) | 电压偏差 |
|------|--------------|----------|----------|
| 无风电 | 803.5 | 5.85 | 0.12 |
| 确定性风电 | 795.2 | 5.21 | 0.15 |
| 概率OPF | 810.3 | 5.89 | 0.11 |

| IEEE 39 | 39 | 10 | 46 | 6097 | 暂态稳定 |

### 特点对比

**IEEE 14节点系统**:
- 优点: 规模小，计算快，适合教学
- 缺点: 过于简单，缺乏优化复杂性
- 应用: 潮流计算教学、基础算法验证

**IEEE 30节点系统**:
- 优点: 优化特性丰富，计算效率高
- 缺点: 规模较小，动态模型简单
- 应用: OPF研究、无功优化、算法比较

**IEEE 39节点系统**:
- 优点: 完整动态模型，暂态稳定研究
- 缺点: 优化特性不如30节点系统
- 应用: 暂态稳定、小信号分析

**IEEE 118节点系统**:
- 优点: 大规模系统，更接近实际
- 缺点: 计算量大，验证困难
- 应用: 大规模优化、分布式计算

| IEEE 118 | <100 | <500 | 高难度优化问题 |

- **算法开发初期**: IEEE 14节点

```matlab
%% MATPOWER Case Format : Version 2
mpc.version = '2';

%%-----  Power Flow Data  -----%%

    2   2   21.7    12.7    0   0   1   1   0   132 1   1.06    0.94;
    ...
];

%% generator data
% bus Pg Qg Qmax Qmin Vg mBase status Pmax Pmin Pc1 Pc2 Qc1min Qc1max Qc2min Qc2max ramp_agc ramp_10 ramp_30 ramp_q apf
mpc.gen = [
    1   0       0       0       0       1.06    100 1   200     50      ...;
    ...
];
```
