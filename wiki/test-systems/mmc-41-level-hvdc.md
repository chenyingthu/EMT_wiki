# MMC-HVDC 41电平测试系统

## 概述

MMC-HVDC 41电平测试系统是[[mmc-model|模块化多电平换流器]]建模研究中的标准中等规模测试系统，广泛应用于[[electromechanical-electromagnetic-hybrid-simulation|混合仿真]]、[[real-time-simulation|实时仿真]]及[[model-order-reduction|模型降阶]]等研究领域的验证和基准测试。该系统配置40个子模块每桥臂（共41电平），代表实际工程中压/高压MMC换流器的典型规模，是连接小功率实验室样机与大功率工程系统的理想桥梁。

41电平MMC-HVDC系统在以下研究中发挥关键作用：
- **[[average-value-model|增强型平均值模型(EAVM)]]验证**：作为中等复杂度系统，可有效验证平均值模型的精度和适用范围
- **详细开关模型对比**：提供计算量可控的详细模型参考基准
- **[[switch-modeling|闭锁特性]]研究**：分析换流器闭锁后的二极管续流行为
- **[[parameter-identification|参数辨识]]与状态估计**：验证初始条件补偿算法的有效性

该测试系统源自实际工程设计参数，被国内外多篇EMT仿真文献采用，特别是加拿大蒙特利尔理工学院(Polytechnique Montreal)研究团队开发的增强型平均值模型系列研究中作为核心验证案例。

### 调制策略

**最近电平调制(NLM)**:
| 参数 | 数值/说明 |
|------|----------|
| 调制原理 | 选择最接近参考电压的电平 |
| 电平数 | 41 (N=40子模块) |
| 电压分辨率 | 2V_sm/(N+1) = 0.78 kV |
| 优点 | 开关损耗极低，适合HVDC |
| 缺点 | 低电平时谐波含量较高 |

| 参数 | 数值 | 说明 |
|------|------|------|
| 载波频率 | 500-1000 | Hz |
| 载波移相角 | 360°/N = 9° | 40个子模块 |
| 等效开关频率 | 20-40 | kHz |
| 谐波特性 | 载波倍频附近 | 易于滤波 |

**调制方式选择**:
- **高电平数(>20)**: 推荐NLM，损耗优势明显
- **低电平数(<10)**: 推荐CPS-PWM，波形质量好
- **41电平**: NLM为主，CPS-PWM用于精度对比研究

### 子模块电容电压平衡

**排序平衡算法**:
```
输入: N个电容电压测量值，桥臂电流方向
      投入子模块数n_on

步骤:
1. 按电压排序所有子模块
2. IF 充电(电流>0): 投入电压最低的n_on个
   ELSE 放电(电流<0): 投入电压最高的n_on个
3. 更新开关状态
```

**平衡效果**:
| 指标 | 典型值 | 说明 |
|------|--------|------|
| 电压不平衡度 | <5% | 稳态运行 |
| 电压纹波 | <10% | 额定工况 |
| 开关频率 | 基频 | NLM下平均 |

## 典型测试场景

### 启动充电过程

**充电阶段**:
| 阶段 | 操作 | 目标 |
|------|------|------|
| 不控充电 | 所有IGBT关断 | 电容充电至~Vdc/N |
| 限流充电 | 分级投入电阻 | 限制充电电流 |
| 可控充电 | 主动控制充电 | 精确充至额定电压 |
| 解锁运行 | 投入正常运行 | 开始功率传输 |

**充电特性**:
- 不控充电峰值电压: ~0.9 Vdc/N (二极管压降影响)
- 充电时间常数: τ = R_arm × C_sm × N ≈ 40-80s
- 典型充电电流: 限制在0.1-0.2 pu

### 直流极间故障

**故障设置**:
| 参数 | 典型设置 | 说明 |
|------|----------|------|
| 故障位置 | 直流线路中点 | 最严重情况 |
| 故障电阻 | 0.1-10 Ω | 金属性到低阻抗 |
| 故障时刻 | 电压峰值时刻 | 最恶劣条件 |

**保护动作时序**:
| 时间 | 动作 | 说明 |
|------|------|------|
| t=0 | 故障发生 | 直流电压骤降 |
| t=1-5ms | 故障检测 | 过流/欠压判据 |
| t=1200μs | 闭锁信号 | IGBT关断延时 |
| t>闭锁 | 二极管续流 | 故障电流衰减 |

**关键研究点**:
- [[average-value-model|增强型平均值模型(EAVM)]]需准确模拟1200μs闭锁延时特性
- 闭锁后二极管续流通路与子模块电容放电行为
- 故障电流峰值与桥臂电感限流效果

### 换流器闭锁

**正常闭锁**:
| 工况 | 闭锁方式 | 后果 |
|------|----------|------|
| 正常停机 | 有序闭锁 | 功率平稳下降 |
| 紧急闭锁 | 立即关断 | 过电压风险 |

**闭锁过电压**:
- 原因: 感性电流突然中断
- 防护措施: 缓冲电路、避雷器
- 模型要求: EAVM需准确模拟闭锁瞬态

## 研究应用

### 增强型平均值模型(EAVM)验证

**验证目的**:
评估[[average-value-model|增强型平均值模型]]在保持计算效率的同时，能否准确捕捉MMC的关键动态特性。

**验证维度**:
| 特性 | 详细模型 | EAVM | 验证重点 |
|------|----------|------|----------|
| 稳态波形 | 开关细节丰富 | 平均值等效 | 波形吻合度 |
| 闭锁瞬态 | 子模块级精确 | 等效建模 | 1200μs延时 |
| 故障响应 | 完整开关过程 | 简化模型 | 峰值电流精度 |
| 计算速度 | 慢(开关步长) | 快(10-100μs) | 加速比 |

**验证指标**:
- 直流电压误差: <2%
- 交流电流误差: <5%
- 闭锁电流连续性: 无突变
- 仿真加速比: >10倍

### 详细开关模型对比

**对比场景**:
1. **稳态运行对比**
   - 直流电压纹波对比
   - 交流电流THD对比
   - 子模块电压平衡特性

2. **暂态过程对比**
   - 功率阶跃响应
   - 交流故障穿越
   - 直流故障响应

3. **计算效率对比**
   - 仿真步长: 详细模型1-5μs vs EAVM 20-100μs
   - 计算时间比: 通常10-50倍
   - 内存占用: EAVM显著降低

### 闭锁后二极管续流特性研究

**研究背景**:
MMC闭锁后，子模块IGBT关断，电流通过反并联二极管续流，形成不可控整流模式。此阶段的准确建模对故障分析和保护设计至关重要。

**续流模式特性**:
| 阶段 | 电流路径 | 电容状态 |
|------|----------|----------|
| 闭锁前 | IGBT可控 | 正常充放电 |
| 闭锁瞬间 | 二极管接管 | 电压维持 |
| 续流期 | 二极管整流 | 可能过充 |
| 衰减期 | 电阻消耗 | 缓慢放电 |

**关键现象**:
- 闭锁延时: IGBT关断非瞬时，典型1200μs
- 电流连续性: 桥臂电感维持电流
- 电容过充: 续流可能反向充电

### 桥臂电感初始状态补偿验证

**补偿原理**:
增强型平均值模型中，桥臂电感的初始电流状态需要特殊补偿，以确保与详细模型的一致性。

**补偿方法**:
```
i_arm_compensated = i_arm_measured + Δi_initial

其中:
- Δi_initial: 初始状态偏差补偿量
- 基于电容电压和开关状态计算
```

**验证方法**:
1. 从详细模型提取初始状态
2. 应用补偿算法到EAVM
3. 对比暂态响应一致性
4. 验证稳态收敛性

**关键验证点**:
| 时刻 | 详细模型行为 | EAVM行为 | 一致性 |
|------|--------------|----------|--------|
| 闭锁前(t=0-) | 正常可控运行 | 等效运行 | 电流值匹配 |
| 闭锁时刻 | IGBT关断 | 等效关断 | 延时建模 |


**验证结果**:
- 电流突变: <1% (可接受)
- 峰值电流: 误差<3%
- 衰减时间常数: 误差<5%

## 性能指标

### 仿真加速比

**加速比定义**:
```
Speedup = T_detailed / T_EAVM

其中:
- T_detailed: 详细模型仿真时间
- T_EAVM: 增强型平均值模型仿真时间
```

**典型加速比**:
| 仿真时长 | 详细模型耗时 | EAVM耗时 | 加速比 |
|----------|--------------|----------|--------|
| 1s暂态 | 2-4小时 | 10-20分钟 | 10-15× |
| 10s暂态 | 20-40小时 | 1-3小时 | 15-25× |
| 稳态运行 | 基准 | 显著加快 | 20-50× |

**影响因素**:
- 仿真步长: 详细模型1-5μs vs EAVM 20-100μs
- 系统规模: 子模块数越多，加速比越高
- 硬件平台: CPU并行能力

### 波形吻合度

**定量评估指标**:
| 指标 | 计算方法 | 典型值 | 要求 |
|------|----------|--------|------|
| NRMSE | 归一化均方根误差 | 2-5% | <5% |
| 相关系数 | Pearson相关 | >0.98 | >0.95 |
| 峰值误差 | (峰值偏差/额定值) | <3% | <5% |
| 相位误差 | 角度差 | <2° | <5° |

**波形对比方法**:
1. 时域叠加对比
2. FFT频谱对比
3. 包络线对比
4. 统计特征对比

**关键波形**:
- 直流电压: 稳态纹波+暂态响应
- 交流电流: 基波+谐波+暂态
- 桥臂电流: 直流+基波+环流+纹波
- 子模块电压: 平衡特性+暂态

**评估场景**:
| 场景 | 评估指标 | 详细模型 | EAVM | 误差 |
|------|----------|----------|------|------|
| 功率阶跃10% | 响应时间 | 35ms | 36ms | 2.9% |
| 功率阶跃50% | 超调量 | 8% | 7.5% | 6.3% |
| 直流故障 | 峰值电流 | 2.5pu | 2.45pu | 2% |
| 闭锁过程 | 电流连续性 | 连续 | 连续 | 一致 |

**暂态精度关键因素**:
1. **闭锁延时建模**: 1200μs精确模拟
2. **电感初始状态**: 准确补偿
3. **二极管特性**: 导通压降+反向恢复
4. **电容动态**: 充放电特性

**精度提升方法**:
- 子模块数N增加时，平均值假设更合理
- 适当增加EAVM仿真步长可平衡精度与速度
- 关键暂态区域可局部细化模型

### 增强型平均值模型实现

**建模工具**:
| 工具 | 适用性 | 特点 |
|------|--------|------|
| MATLAB/Simulink | 优秀 | 研究首选，灵活实现 |
| PSCAD/EMTDC | 良好 | 支持用户自定义模型 |
| Modelica | 良好 | 方程驱动，物理建模 |

**核心模块**:
1. **平均值阀模型**: 等效受控电压源
2. **桥臂电感模型**: 含初始状态补偿
3. **电容等效模型**: 电压动态方程
4. **控制系统**: 与详细模型一致

**关键方程**:
```
// 桥臂电压平均值
v_arm_avg = n_on × v_cap_avg

// 电容电压动态
dv_cap/dt = (i_arm × n_on - i_load) / C_eq

// 桥臂电感电压
L_arm × di_arm/dt = v_dc/2 - v_arm_avg - v_ac - R_arm × i_arm
```
