# DFIG风电场测试系统 (DFIG Wind Farm Test System)

## 概述

双馈感应发电机(Doubly-Fed Induction Generator, DFIG)风电场是电力系统电磁暂态(EMT)仿真研究中最重要、应用最广泛的测试平台之一。DFIG技术因其独特的变流器配置和灵活的控制能力，自2000年代中期以来成为陆上风电的主流技术路线，在全球风电装机容量中占据主导地位。

作为EMT仿真研究的重要对象，DFIG风电场测试系统广泛应用于以下研究领域：

- **风电并网特性研究**: 分析DFIG风电场对接入电网的功率特性、电压支撑能力和电能质量影响
- **故障穿越能力分析**: 评估低电压穿越(LVRT)和高电压穿越(HVRT)控制策略的有效性
- **等值建模与仿真加速**: 研究大规模风电场的电磁暂态等值方法，降低计算复杂度
- **次同步振荡研究**: 分析DFIG与串补线路或弱电网之间的次同步谐振现象
- **保护配合研究**: 验证风电场内部及并网保护的动作逻辑和配合关系

典型DFIG风电场的规模涵盖从小型示范(5-10台机组)到大型商业化项目(100-200台机组)。在学术研究中最常用的测试配置包括17台和100台机组两种典型规模，分别用于验证控制策略和等值建模方法。

## 变流器系统

### 转子侧变流器(Rotor Side Converter, RSC)

**功能**:
- 控制发电机转子电流，实现有功/无功功率解耦控制
- 提供励磁电流，维持气隙磁通
- 实现最大风能捕获(MPPT)

**技术参数**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 额定容量 | 0.25-0.3 pu | 定子功率的25-30% |
| 开关频率 | 2-5 kHz | 常用IGBT模块 |
| 控制方式 | 矢量控制 | 转子磁链定向 |
| 调制方式 | SVPWM | 空间矢量脉宽调制 |

**主电路结构**:
- 三相全桥电压源型逆变器(VSI)
- 6个IGBT功率模块(带反并联二极管)
- 直流母线电容支撑
- 交流侧LC滤波器

### 电网侧变流器(Grid Side Converter, GSC)

**功能**:
- 维持直流母线电压恒定
- 控制网侧无功功率
- 实现网侧单位功率因数运行(或无功调节)

**技术参数**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 额定容量 | 0.25-0.3 pu | 与RSC匹配 |
| 开关频率 | 2-5 kHz | 与RSC同步或倍频 |
| 控制方式 | 矢量控制 | 电网电压定向 |
| 网侧电感 | 0.1-0.15 pu | 电流纹波抑制 |

**主电路结构**:
- 三相全桥电压源型整流器/逆变器
- 6个IGBT功率模块
- 网侧三相电感(L滤波器)
- 可选择LCL滤波器结构

### 直流母线系统

**直流母线电容**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 额定电压 | 1050-1200 V | 1.5-3 MW机组 |
| 电容容量 | 50-100 mF | 依据功率等级 |
| 电压纹波 | <5% | 正常运行时 |
| 能量时间常数 | 5-15 ms | 直流母线惯性 |

**直流预充电电路**:
- 预充电电阻: 限制上电冲击电流
- 预充电接触器: 预充电完成后旁路电阻
- 直流接触器: 正常运行时接通主电路

### 撬棒保护(Crowbar)

**功能**:
- 电网故障时消耗转子过剩能量
- 保护转子侧变流器免受过电压/过电流损坏
- 实现低电压穿越

**典型配置**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 电阻值 | 0.1-0.5 Ω | 三相或单相结构 |
| 功率容量 | 短时过载 | 持续数秒 |
| 触发方式 | 晶闸管/IGBT | 主动或被动 |
| 动作阈值 | 1.5-2.0 pu | 转子电流或直流电压 |

**动作逻辑**:
1. 检测电网电压跌落或转子过流
2. 闭锁RSC触发脉冲
3. 投入Crowbar电阻
4. 发电机切换至感应发电机模式运行
5. 故障清除后退出Crowbar，恢复RSC控制

## 控制系统

### 最大功率点跟踪(MPPT)

**控制目标**:
- 在不同风速条件下捕获最大风能
- 维持最优叶尖速比(TSR)

**控制策略**:
| 策略 | 原理 | 特点 |
|------|------|------|
| 最佳叶尖速比法 | λ_opt = ωR/v_wind | 理论最优，需风速测量 |
| 功率曲线查表法 | P = f(ω) | 无需风速，工程常用 |
| 爬山法 | dP/dω = 0 | 自适应，响应较慢 |

**典型参数**:
- 最优叶尖速比 λ_opt: 6-8
- 最大风能利用系数 Cp_max: 0.45-0.50
- 功率跟踪精度: >95%

### 桨距角控制

**控制功能**:
- 风速超过额定值时限制捕获功率
- 大风切出保护(通常>25 m/s)
- 参与有功功率调节

**控制结构**:
- 外环: 功率/转速控制，输出桨距角指令
- 内环: 桨距角伺服，液压或电动执行

**典型参数**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 额定风速 | 11-13 m/s | 满功率发电风速 |
| 切出风速 | 25 m/s | 安全停机阈值 |
| 桨距角范围 | 0-90° | 0°为迎风最大 |
| 变桨速率 | 6-10 °/s | 电动变桨典型值 |

### 转子电流矢量控制

**控制目标**:
- 独立控制有功功率和无功功率
- 快速动态响应
- 故障穿越期间的电流限幅

**控制策略**:
- **定向方式**: 定子磁链定向或电网电压定向
- **坐标变换**: 三相静止→两相旋转(dq坐标系)
- **解耦控制**: d轴控制有功，q轴控制无功

**控制框图结构**:
```
功率指令 → 功率外环(PI) → 电流指令 → 电流内环(PI) → SVPWM → RSC
         ↑                    ↑
      功率反馈            电流反馈
```

**典型PI参数**:
| 控制器 | Kp | Ki | 带宽 |
|--------|-----|-----|------|
| 功率外环 | 0.5-2 | 10-50 | 5-20 Hz |
| 电流内环 | 1-5 | 50-200 | 100-500 Hz |

### 无功功率控制

**控制模式**:
| 模式 | 说明 | 应用场景 |
|------|------|----------|
| 恒功率因数 | 维持固定功率因数 | 常规运行 |
| 恒无功功率 | 输出设定无功值 | 电压支撑 |
| 恒电压控制 | 调节机端/并网点电压 | 弱电网 |
| 电压下垂 | 按电压偏差调节无功 | 多机协调 |

**无功调节范围**:
- 迟相运行(吸收无功): 可达0.95功率因数
- 进相运行(发出无功): 可达0.95功率因数
- 动态响应时间: 20-100 ms

### 低电压穿越(LVRT)控制

**穿越要求**(以中国GB/T 19963为例):
| 电压跌落 | 不脱网时间 | 有功恢复 |
|----------|------------|----------|
| 20% Ut | 625 ms | 电压恢复后2s |
| 90% Ut | 持续运行 | - |

**控制策略**:
1. **电压跌落检测**: 监测并网点电压
2. **Crowbar保护**: 严重故障时投入
3. **转子电流限幅**: 限制过电流
4. **无功支撑**: 向电网注入无功
5. **有功恢复**: 故障清除后快速恢复

**改进型LVRT**:
- 转子串联动态电阻
- 定子串联动态电压恢复器
- 直流卸荷电路(Chopper)
- 串联制动电阻

## 典型测试场景

### 风速波动测试

**测试目的**:
- 验证MPPT控制性能
- 评估功率波动特性
- 测试机械载荷

**风速模型**:
| 类型 | 特征 | 参数 |
|------|------|------|
| 渐变风 | 风速缓慢变化 | 0.1-0.5 m/s² |
| 阵风 | 短时风速突变 | 幅值±5 m/s, 持续10s |
| 湍流风 | 随机风速波动 | 湍流强度10-20% |
| 额定风速跨越 | 通过最优功率点 | 10-15 m/s |

**观测指标**:
- 有功功率跟踪误差
- 转速响应
- 桨距角动作
- 变流器功率

### 电网三相短路故障

**故障设置**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 故障位置 | 并网点/集电线路 | 不同严重程度 |
| 故障类型 | 三相接地短路 | 最严重故障 |
| 残压 | 0-50% | 电压跌落深度 |
| 故障持续时间 | 150-625 ms | 保护动作时间 |

**关键观测**:
- 定子/转子电流冲击
- 直流母线电压过冲
- Crowbar动作情况
- 有功/无功功率响应
- 机组稳定性

### 单相接地故障

**故障特征**:
- 不对称故障，产生负序分量
- 故障电流小于三相短路
- 需考虑零序网络

**研究重点**:
- 负序电流抑制
- 不平衡运行能力
- 保护选择性配合

### Crowbar保护动作测试

**测试目的**:
- 验证保护动作逻辑
- 评估能量消耗能力
- 测试恢复过程

**触发条件**:
| 条件 | 阈值 | 延时 |
|------|------|------|
| 转子过流 | 1.5-2.0 pu | 即时 |
| 直流过压 | 1.2-1.3 pu | 10-50 ms |
| 定子过流 | 1.5-2.0 pu | 延时 |

**退出条件**:
- 转子电流降至安全范围
- 定子电压恢复
- 延时计时结束(通常<2s)

### 低电压穿越(LVRT)测试

**测试标准**: 依据各国并网导则

**测试流程**:
1. 稳态运行，输出额定功率
2. 电压跌落至目标水平(如20%)
3. 保持规定时间(如625ms)
4. 电压恢复至正常
5. 观测功率恢复过程

**评价指标**:
| 指标 | 要求 | 测试方法 |
|------|------|----------|
| 不脱网时间 | >规定值 | 时域仿真 |
| 无功支撑 | 按电压跌落比例 | 无功电流注入 |
| 有功恢复 | <2s恢复90% | 功率曲线 |
| 动态电压支撑 | 电压恢复加速 | 电压波形 |

## 研究应用

### 风电场等值建模

**研究背景**:
大规模风电场(100+台机组)的详细EMT仿真计算量巨大，需要等值简化。

**等值方法**:
| 方法 | 原理 | 适用场景 |
|------|------|----------|
| 单机等值 | 单台大功率机组 | 系统级研究 |
| 多机等值 | 按风速/类型分组 | 需要保留多样性 |
| 电磁暂态等值 | 频域拟合 | 高频特性保持 |
| 机电-电磁混合 | 分区建模 | 大规模系统 |

**等值参数聚合**:
```
等值容量 = Σ单机容量
等效惯性 = Σ单机惯性
等值阻抗 = 网络简化计算
控制器参数 = 加权平均或保留主导机组
```

### 电磁暂态同调分群

**同调识别方法**:
- 基于转子转速响应的时域方法
- 基于参与因子的特征分析方法
- 基于电气距离的聚类方法

**分群准则**:
| 准则 | 说明 | 应用 |
|------|------|------|
| 风速分群 | 相近风速的机组归为一群 | 功率特性 |
| 电气距离 | 相近位置的机组归为一群 | 故障响应 |
| 参数分群 | 相同机型的机组归为一群 | 模型简化 |

### 短路电流分析

**分析目的**:
- 风电场保护定值整定
- 并网断路器选型
- 集电网络设计

**短路特性**:
| 故障类型 | 特征 | 计算方法 |
|----------|------|----------|
| 三相短路 | 对称，电流大 | 序网络分析 |
| 单相接地 | 零序通路 | 零序阻抗建模 |
| 相间短路 | 负序分量 | 不对称分析 |

**DFIG短路电流特点**:
- 受变流器电流限幅制约
- Crowbar投入后表现为感应电机特性
- 衰减时间常数与机组参数相关

### 故障穿越特性

**研究内容**:
- LVRT控制策略优化
- 电压支撑能力评估
- 故障后恢复特性
- 多机协调控制

**关键问题**:
| 问题 | 研究重点 | 解决方法 |
|------|----------|----------|
| 转子过流 | 电流限制策略 | 改进Crowbar/串联电阻 |
| 直流过压 | 能量管理 | 直流卸荷/Chopper |
| 无功分配 | 多机协调 | 主从控制/下垂控制 |
| 功率恢复 | 快速恢复 | 改进控制策略 |

### 次同步振荡研究

**振荡机理**:
- DFIG与串联补偿线路的相互作用
- 转子侧变流器控制的负阻尼效应
- 弱电网下的不稳定风险

**分析方法**:
| 方法 | 特点 | 适用阶段 |
|------|------|----------|
| 特征值分析 | 线性化，快速 | 设计阶段 |
| 时域仿真 | 精确，耗时 | 验证阶段 |
| 阻抗法 | 基于频率扫描 | 机理分析 |

**抑制措施**:
- 附加次同步阻尼控制(SSDC)
- 调整锁相环(PLL)参数
- 优化电流环带宽
- 串联补偿度限制

## 性能指标

### 有功功率响应

**稳态性能**:
| 指标 | 典型值 | 说明 |
|------|--------|------|
| 功率跟踪精度 | >95% | MPPT性能 |
| 功率波动 | <10% (10min) | 电能质量 |
| 谐波含量 | <5% | THD要求 |

**动态性能**:
| 指标 | 典型值 | 说明 |
|------|--------|------|
| 功率响应时间 | 100-500 ms | 控制带宽决定 |
| 超调量 | <10% | 阶跃响应 |
| 风速变化跟踪 | 秒级 | 受惯量限制 |

### 无功功率控制

**控制性能**:
| 指标 | 典型值 | 说明 |
|------|--------|------|
| 无功调节范围 | ±0.33 pu | 额定功率的1/3 |
| 响应时间 | 20-100 ms | 电压控制环 |
| 稳态误差 | <5% | 控制精度 |

**电压支撑**:
- 故障期间无功电流注入: 按电压跌落比例
- 恢复后电压调节: 快速响应
- 多机无功分配: 协调控制

### 转子电流波形

**正常运行**:
| 特征 | 典型值 | 说明 |
|------|--------|------|
| 电流幅值 | 0.2-0.3 pu | 与转差率相关 |
| 频率 | s×f_s | 转差频率 |
| 谐波 | <3% | 变流器开关谐波 |

**故障期间**:
- 冲击幅值: 可达2-3 pu
- 衰减时间常数: 与Crowbar参数相关
- 直流分量: 取决于故障时刻

### 故障电流包络

**短路电流特征**:
| 阶段 | 特征 | 时间尺度 |
|------|------|----------|
| 初始峰值 | 最大瞬时值 | 0-20 ms |
| 衰减过程 | 按时间常数衰减 | 20-200 ms |
| 稳态值 | 受限幅值 | >200 ms |

**影响因素**:
- 故障类型和位置
- 机组运行状态
- Crowbar投入时机
- 控制策略
