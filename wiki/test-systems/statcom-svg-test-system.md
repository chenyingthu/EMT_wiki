## 概述

### 基本概念

**STATCOM**（Static Synchronous Compensator，静止同步补偿器）和**SVG**（Static Var Generator，静止无功发生器）是基于电压源型变流器（Voltage Source Converter, VSC）技术的先进柔性交流输电系统（FACTS）设备。它们通过产生可控的无功电流来实现对电网电压的动态调节和无功功率的快速补偿。

STATCOM/SVG的核心工作原理是利用全控型电力电子器件（如IGBT、IGCT）构成的高频开关变流器，将直流电压转换为幅值和相位可控的交流电压，通过连接电抗器与电网并联，实现无功功率的双向流动：
- 当变流器输出电压高于电网电压时，向系统注入感性无功（相当于电容器）
- 当变流器输出电压低于电网电压时，从系统吸收感性无功（相当于电抗器）

### 发展历史

**技术演进历程**:
- 1970年代: 晶闸管控制的SVC（静止无功补偿器）开始应用
- 1980年代: GTO器件使VSC技术成为可能
- 1990年代: IGBT器件成熟，两电平STATCOM商业化
- 2000年代: 三电平NPC和级联H桥拓扑发展
- 2010年代: MMC拓扑在高压大容量领域应用
- 2020年代: 宽禁带器件（SiC/GaN）推动高频化

### 技术优势

相比传统SVC，STATCOM/SVG具有以下显著优势：

**响应速度**:
- STATCOM: <10ms（典型值5-10ms）
- SVC: 20-40ms（晶闸管导通延迟限制）
- 机械开关电容器: 100ms以上

**运行范围**:
- 从额定感性到额定容性连续可调
- 弱电网条件下仍能有效运行
- 不受系统电压跌落影响（恒流特性）

**占地面积**:
- 无需大容量电容器组和电抗器
- 同等容量下占地面积约为SVC的1/3
- 模块化设计便于扩展

**多功能性**:
- 无功补偿（基波正序）
- 不平衡补偿（负序分量）
- 谐波补偿
- 电压闪变抑制

### 应用领域

STATCOM/SVG广泛应用于以下场景：

**输电网应用**:
- 电压稳定控制
- 输电能力增强
- 暂态稳定性改善
- 功率振荡阻尼

**配电网应用**:
- 电能质量治理
- 电压跌落补偿
- 闪变抑制
- 谐波滤波

**工业应用**:
- 电弧炉补偿
- 轧机负荷平衡
- 变频器无功补偿
- 港口岸电系统

**新能源应用**:
- 风电场无功支撑
- 光伏电站电压调节
- 储能系统配套
- 海上风电汇集

## 拓扑结构

### 两电平拓扑 (Two-Level Topology)

**基本结构**:
两电平拓扑是最基础的STATCOM结构，每相由两个开关器件组成半桥，三相构成完整变流器。

```
直流侧
  +---[S1]---+---[S2]---+
  |          |          |
 +||        L1         -||
  |          |          |
  +---[S3]---+---[S4]---+
            N (中性点)
```

**技术特点**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 开关器件数 | 6个/三相 | IGBT或IGCT |
| 输出电平数 | 2电平 | +Vdc/2, -Vdc/2 |
| 直流电压 | 1-3kV | 受器件耐压限制 |
| 开关频率 | 2-5kHz | PWM控制 |
| 适用容量 | <10Mvar | 低压小容量 |

**优点**:
- 结构简单，控制方便
- 开关器件数量少
- 成本相对较低

**缺点**:
- 输出电压谐波含量高
- 需要较大的滤波电抗器
- 开关频率受限（电磁兼容）
- 不适合高压大容量应用

**改进方案**:
- 多重化并联: 多台变流器并联运行
- 多电平PWM: 优化调制策略降低谐波
- 滤波器优化: LCL滤波器设计

### 三电平NPC拓扑 (Neutral Point Clamped)

**基本结构**:
三电平中点箝位（NPC）拓扑通过增加中间电平，有效降低输出电压谐波。

```
        +Vdc
         |
      [S1][S2]
         |
        --- 中点
         |
      [S3][S4]
         |
        -Vdc
```

**工作原理**:
通过不同的开关组合产生三种输出电平：
- 正电平(+Vdc/2): S1、S2导通
- 零电平(0): S2、S3导通（钳位状态）
- 负电平(-Vdc/2): S3、S4导通

**技术参数**:
| 参数 | 两电平 | 三电平NPC | 优势 |
|------|--------|-----------|------|
| 输出电平 | 2 | 3 | 谐波降低50% |
| 器件耐压 | Vdc | Vdc/2 | 可承受2倍电压 |
| 开关损耗 | 高 | 中 | 降低30% |
| 控制复杂度 | 低 | 中 | 直流平衡问题 |

**直流侧电压平衡**:
三电平NPC的关键问题是中点电压平衡：
- **不平衡原因**: 正负半周开关不对称、负荷不平衡
- **影响**: 输出电压波形畸变、器件过压风险
- **控制方法**: 
  - 冗余矢量选择
  - 正负小矢量交替
  - 硬件辅助均压电路

**应用场景**:
- 中等电压等级（3-10kV）
- 容量范围5-50Mvar
- 对谐波要求较高的场合

### 级联H桥拓扑 (Cascaded H-Bridge, CHB)

**基本结构**:
级联H桥由多个单相全桥（H桥）模块串联组成，每个模块具有独立的直流电容。

```
每相结构:
[H桥1]---[H桥2]---[H桥3]---...---[H桥N]---电网
   |         |         |            |
  Cdc1      Cdc2      Cdc3         CdcN
```

**单模块结构**:
每个H桥模块由4个开关器件组成，可输出三种电平：+Vdc, 0, -Vdc

**多电平优势**:
| 模块数N | 输出电平数 | 谐波特性 | 适用电压 |
|---------|------------|----------|----------|
| 2 | 5电平 | THD~15% | 3-6kV |
| 4 | 9电平 | THD~8% | 6-10kV |
| 6 | 13电平 | THD~5% | 10-20kV |
| 8 | 17电平 | THD~3% | 20-35kV |

**模块化特点**:
- **冗余设计**: 部分模块故障时降额运行
- **易于扩展**: 增加模块数提升电压等级
- **独立控制**: 各模块直流电压独立调节
- **维护方便**: 故障模块可热插拔更换

**直流侧均压控制**:
级联H桥的核心挑战是多模块直流电压均衡：

1. **相间均压**: 三相之间总能量平衡
   - 零序电压注入法
   - 负序电流调节法

2. **相内均压**: 同一相内各模块电压平衡
   - 载波移相调制
   - 特定谐波消除优化
   - 功率差异补偿

3. **损耗均衡**: 各模块开关损耗均衡
   - 轮换调制策略
   - 热管理优化

**典型参数**:
| 参数 | 范围 | 典型值 |
|------|------|--------|
| 模块数/相 | 4-20 | 8-12 |
| 单模块电压 | 500-1500V | 1000V |
| 单模块容量 | 0.5-2Mvar | 1Mvar |
| 总容量 | 10-100Mvar | 35Mvar |
| 开关频率 | 500-1500Hz | 1000Hz |

**应用优势**:
- 直接接入中压电网（6-35kV）无需升压变压器
- 输出波形质量高，滤波器体积小
- 电磁兼容性能好
- 可扩展性强

### 模块化多电平拓扑 (MMC)

**基本结构**:
MMC拓扑是高压大容量STATCOM的主流选择，每个桥臂由大量子模块串联组成。

```
        上桥臂                下桥臂
     [SM1]-[SM2]-...      [SM1]-[SM2]-...
相A ----|电感La|----+----|电感La|---- 相A输出
     [SMN]          |          [SMN]
                   ---
                  中点
```

**子模块类型**:
1. **半桥子模块 (HBSM)**:
   - 2个开关器件 + 1个电容
   - 成本低，效率高
   - 不能阻断直流故障

2. **全桥子模块 (FBSM)**:
   - 4个开关器件 + 1个电容
   - 可输出正负电压
   - 具备直流故障穿越能力

3. **钳位双子模块 (CDSM)**:
   - 混合拓扑优化
   - 兼顾成本和性能

**技术参数**:
| 参数 | 典型范围 | 说明 |
|------|----------|------|
| 子模块数/桥臂 | 20-400 | 电压等级决定 |
| 单模块电压 | 1.5-3kV | IGBT耐压限制 |
| 开关频率 | 100-300Hz | 低开关损耗 |
| 输出电平数 | 40-800 | 波形质量极高 |
| 容量等级 | 50-500Mvar | 高压大容量 |

**运行控制要点**:
- **环流抑制**: 二倍频环流抑制控制
- **电容均压**: 排序算法实现电压均衡
- **冗余管理**: 子模块故障旁路处理
- **启动控制**: 预充电和软启动策略

**与CHB对比**:
| 特性 | MMC | CHB |
|------|-----|-----|
| 储能元件 | 多（每子模块） | 少（每H桥） |
| 效率 | 稍低（环流损耗） | 较高 |
| 故障处理 | 灵活（子模块旁路） | 模块冗余 |
| 高压应用 | 更成熟 | 受限于模块数 |

## 系统参数

### 容量等级

**容量分类**:
| 等级 | 容量范围 | 典型应用 | 拓扑选择 |
|------|----------|----------|----------|
| 小型 | 1-5Mvar | 配电补偿、工业 | 两电平 |
| 中型 | 5-30Mvar | 变电站、园区 | CHB、三电平 |
| 大型 | 30-100Mvar | 输电网、风电场 | CHB、MMC |
| 超大型 | >100Mvar | 主网枢纽 | MMC |

**容量配置原则**:
- **感性容量**: 通常为容性容量的50-70%
- **短时过载**: 1.5-2倍额定容量持续1-3s
- **动态范围**: 100%感性到100%容性连续可调

**选型计算**:
无功补偿容量计算公式：
```
Q = P × (tanφ1 - tanφ2)
```
其中：
- Q: 所需补偿容量(Mvar)
- P: 有功功率(MW)
- φ1: 补偿前功率因数角
- φ2: 目标功率因数角

### 电压等级

**直流侧电压**:
| 交流电压 | 直流电压 | 适用拓扑 | 器件耐压 |
|----------|----------|----------|----------|
| 380V | 600-800V | 两电平 | 1200V |
| 6kV | 9-12kV | CHB(6-8模块) | 1700V |
| 10kV | 15-20kV | CHB(10-12模块) | 3300V |
| 35kV | 50-70kV | CHB/MMC | 4500V |

**直流电压设计**:
- **最小值**: Vdc_min = √2 × Vac × m_inv
  - m_inv为调制比，通常0.8-0.95
- **纹波要求**: ΔVdc/Vdc < 5-10%
- **电容选择**: Cdc = Q × T / (0.5 × Vdc² × k)
  - T为响应时间，k为电压波动系数

**交流侧电压**:
- 低压: 380V/690V（工业应用）
- 中压: 6kV/10kV/20kV/35kV（配电网）
- 高压: 110kV/220kV（输电网，需升压变）

### 响应时间

**时间常数定义**:

| 阶段 | 时间 | 说明 |
|------|------|------|
| 检测时间 | 1-3ms | 电压/电流采样和滤波 |
| 控制运算 | 0.5-1ms | DSP/FPGA处理延迟 |
| PWM生成 | 0.1-0.5ms | 调制信号产生 |
| 开关响应 | 0.5-2ms | IGBT导通延迟 |
| 电流建立 | 2-4ms | 电感电流上升 |
| **总计** | **5-10ms** | **典型响应时间** |

**影响因素**:
- 采样频率: 越高检测越快，但计算负担增加
- 滤波器设计: 带宽vs噪声抑制的权衡
- 控制算法: PI/PR/模型预测控制的动态差异
- 开关频率: 影响PWM精度和响应速度

**测试方法**:
- 无功阶跃响应: 从0到额定容量阶跃
- 电压阶跃响应: 系统电压突变
- 频响特性: 扫频测试带宽

**损耗构成**:
| 损耗类型 | 占比 | 影响因素 |
|----------|------|----------|
| 导通损耗 | 40-50% | IGBT通态电阻 |

**效率曲线**:

- 50%负载: 95-96%

**损耗优化**:

## 控制系统

### 电压控制

**控制目标**:
维持接入点电压在设定范围内，典型控制精度为±1%。

**控制结构**:
```
电压给定 Vref
    |
    v
[电压调节器] → 无功电流给定 Iqref
    |            |
    v            v
电压反馈 Vmeas  [电流内环] → PWM调制
                   |
                   v
              电流反馈 Imeas
```

**电压调节器设计**:
- **PI控制器**: 传统方法，参数整定简单
  - Kp: 比例增益（快速响应）
  - Ki: 积分增益（消除静差）
  - 典型带宽: 10-50Hz

- **下垂特性**: 多STATCOM协调控制
  ```
  V = V0 - k × Q
  ```
  其中k为下垂系数，实现无功功率自动分配

**控制模式**:
| 模式 | 描述 | 适用场景 |
|------|------|----------|
| 恒电压模式 | 维持电压恒定 | 电压敏感母线 |
| 下垂模式 | 电压随无功变化 | 多设备并联 |
| 调度模式 | 按调度指令运行 | 电网统一控制 |

### 无功控制

**电流解耦控制**:
在旋转坐标系(dq坐标系)下实现有功和无功电流的独立控制。

**坐标变换**:
```
[id]   [ cosθ  sinθ][ia]
[iq] = [-sinθ  cosθ][ib]
```

**电流环设计**:
- d轴: 控制有功电流（通常设为零，纯无功运行）
- q轴: 控制无功电流（电压环输出）
- 带宽: 100-500Hz（远高于电压环）

**前馈解耦**:
消除d-q轴之间的耦合项：
```
Vd = Vd* - ωLiq
Vq = Vq* + ωLid
```

### 直流电压平衡控制

**必要性**:
多电平拓扑中各模块直流电压必须均衡，否则会导致输出波形畸变和器件过压。

**控制层次**:

1. **总电压控制**:
   - 目标: 维持总直流电压恒定
   - 方法: d轴电流微调吸收损耗
   - 带宽: 5-20Hz

2. **相间均压**:
   - 目标: 三相总电压相等
   - 方法: 零序电压注入
   - 公式: V0 = k × (Va + Vb + Vc)/3

3. **相内均压**:
   - 目标: 同一相各模块电压相等
   - 方法:
     - 排序算法（Nearest Level Modulation）
     - 载波移相优化
     - 无功功率微调

**排序算法**:
```
1. 采样所有子模块电压
2. 按电压值排序
3. 根据电流方向选择投入模块:
   - 充电时: 优先投入电压低的模块
   - 放电时: 优先投入电压高的模块
4. 更新开关状态
```

**控制效果**:
- 电压不平衡度: <3-5%
- 响应时间: <100ms
- 开关频率增加: <10%

### 模块均压（级联H桥）

**相间有功功率平衡**:
由于三相负荷不对称或电网电压不平衡，三相之间可能存在有功功率差异，导致整体电压不平衡。

**零序电压注入法**:
```
Va' = Va + V0
Vb' = Vb + V0
c' = Vc + V0
```
通过注入零序电压，在不改变线电压的前提下，调节各相有功功率。

**优化策略**:
- 在线优化算法
- 模型预测控制
- 智能算法（遗传算法、粒子群）

## 调制策略

### 载波移相PWM (CPS-PWM)

**基本原理**:
多个变流器模块的载波信号按相位移相，使得各模块输出谐波相互抵消。

**移相角度**:
对于N个模块，移相角为：
```
φ = 360° / N
```

**谐波抵消效果**:
| 模块数 | 等效开关频率 | 最低次谐波 | THD |
|--------|--------------|------------|-----|
| 1 | fc | fc | 20% |
| 2 | 2fc | 2fc | 12% |
| 4 | 4fc | 4fc | 6% |
| 8 | 8fc | 8fc | 3% |

**优点**:
- 等效开关频率高，谐波性能好
- 各模块控制独立，易于实现
- 直流侧均压效果好

**缺点**:
- 需要高频开关，损耗较大
- 控制复杂度高
- 对同步精度要求高

**应用**:
- 级联H桥拓扑首选调制策略
- 中压大容量STATCOM

### 最近电平调制 (NLM)

**基本原理**:
根据参考电压瞬时值，选择最接近的实际输出电平，适用于子模块数较多的MMC拓扑。

**调制过程**:
```
1. 根据参考电压计算所需投入子模块数:
   Ninv = round(Vref / Vsm)

2. 根据电流方向选择具体模块:
   - 充电: 选择电压较低的模块
   - 放电: 选择电压较高的模块

3. 输出相应电平
```

**特点**:
- 开关频率低（约100-300Hz）
- 损耗小，效率高
- 谐波性能随电平数增加而改善
- 需要子模块电压排序

**适用条件**:
- 子模块数N > 20
- 高压大容量MMC
- 对效率要求高的场合

### 选择性谐波消除 (SHE)

**基本原理**:
通过优化开关角度，有选择地消除特定次谐波。

**数学模型**:
对于N个开关角，可以消除N-1个特定谐波，并控制基波幅值。

```
基波:  V1 = (4Vdc/π) × Σ(-1)^i × cos(αi)

h次谐波: Vh = (4Vdc/hπ) × Σ(-1)^i × cos(hαi) = 0
```

**开关角求解**:
- 牛顿-拉夫逊法
- 遗传算法
- 查表法（实时查表）

**优点**:
- 开关频率极低
- 特定谐波完全消除
- 损耗最小化

**缺点**:
- 离线计算复杂
- 动态响应差
- 不适合快速变化工况

**应用**:
- 稳态运行工况
- 对特定谐波敏感的场合
- 高效率要求的应用

### 调制策略对比

| 策略 | 开关频率 | THD | 动态响应 | 适用拓扑 |
|------|----------|-----|----------|----------|
| CPS-PWM | 高 | 低 | 快 | CHB |
| NLM | 低 | 中 | 中 | MMC |
| SHE | 极低 | 特定低 | 慢 | CHB/MMC |
| SVPWM | 中 | 低 | 快 | 两电平 |

## 典型测试场景

### 电压跌落测试

**测试目的**:
验证STATCOM在电网电压跌落时的动态响应和电压支撑能力。

**测试条件**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 跌落深度 | 10-90% | 剩余电压百分比 |
| 跌落时间 | 100-1000ms | 持续时间 |
| 跌落类型 | 三相/单相/两相 | 故障类型 |

**性能指标**:
- 电压恢复时间: <100ms
- 无功电流响应时间: <10ms
- 最大无功输出: 1.5-2倍额定
- 低电压穿越: 符合电网规程

**仿真设置**:
```
0-0.1s: 稳态运行
0.1s: 电压跌落发生
0.2s: 电压恢复
仿真时长: 0.5s
```

### 电压骤升测试

**测试目的**:
验证STATCOM吸收无功、抑制电压过高的能力。

**测试条件**:
- 电压骤升幅度: 110-130%额定电压
- 持续时间: 100-500ms
- 恢复过程: 线性和非线性

**控制策略**:
- 快速吸收无功
- 防止过调制
- 直流电压控制

### 无功阶跃测试

**测试目的**:
评估控制系统的动态响应特性。

**测试方法**:
1. 容性阶跃: 0 → +100%额定无功
2. 感性阶跃: 0 → -100%额定无功
3. 反向阶跃: +100% → -100%

**评价指标**:
| 指标 | 优秀 | 良好 | 合格 |
|------|------|------|------|
| 上升时间 | <10ms | <20ms | <30ms |
| 超调量 | <5% | <10% | <15% |
| 调节时间 | <50ms | <100ms | <150ms |

### 不平衡补偿测试

**测试目的**:
验证负序补偿能力。

**测试条件**:
- 电压不平衡度: 5-20%
- 负荷不平衡: 单相大负荷投入

**控制策略**:
- 正序分离
- 负序电流控制
- 相间功率平衡

**评价指标**:
- 补偿后不平衡度: <2%
- 响应时间: <100ms

### 谐波补偿测试

**测试目的**:
验证STATCOM作为有源滤波器的性能。

**测试条件**:
- 背景谐波: 5%、7%、11%、13次
- THD: 10-20%
- 负荷变化: 阶跃和周期性

**控制方法**:
- 谐波检测（瞬时无功理论）
- 多谐振控制器
- 重复控制

**性能指标**:
- 补偿后THD: <5%
- 单次谐波: <3%

## 研究应用

### 电压稳定研究

**应用场景**:
- 负荷中心电压支撑
- 远距离输电电压调节
- 暂态电压稳定

**仿真研究**:
- PV曲线分析
- 暂态电压稳定裕度
- 协调控制策略

### 电能质量改善

**应用领域**:
- 电压闪变抑制
- 谐波滤波
- 电压暂降补偿

**典型案例**:
- 电弧炉补偿
- 轧机冲击负荷
- 电气化铁路

### 新能源汇集

**风电场应用**:
- 电压调节
- 低电压穿越
- 功率波动平抑

**光伏电站应用**:
- 逆变器无功能力利用
- 夜间无功支撑
- 电压质量控制

**技术要求**:
- 响应速度: <50ms
- 容量配置: 风电容量的20-30%
- 控制协调: 与风机变流器协调

### 电弧炉补偿

**负荷特点**:
- 冲击性大（MW-Mvar级）
- 三相不平衡
- 谐波丰富（2-7次）
- 电压闪变

**补偿方案**:
- STATCOM+FC（固定电容器）
- 容量配比: 2:1到3:1
- 响应时间: <20ms

**效果评估**:
- 闪变降低: 50-70%
- 电压波动: <3%
- 功率因数: >0.95

